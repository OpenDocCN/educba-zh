## 下载

### Docker

```
docker pull apachecn0/educba-zh
docker run -tid -p <port>:80 apachecn0/educba-zh
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g educba-zh
educba-zh <port>
# 访问 http://localhost:{port} 查看文档
```
