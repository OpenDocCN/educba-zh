## MATLAB 导数介绍

MATLAB 是一个交互式的编程环境，用于科学计算。它广泛应用于许多需要解决问题、数据分析、算法开发和实验的技术领域。具体学科的软件广泛使用 MATLAB 编写。MATLAB 为我们提供了执行大量数学运算的能力，在这个主题中，我们将学习 MATLAB 导数。

### MATLAB 导数是如何工作的？

微分是一种计算函数微小变化的微积分工具。微分或导数是函数相对于某个变量的变化率。它计算我们的输出对输入的任何变化有多敏感。

<small>Hadoop、数据科学、统计学&其他</small>

数学上可以表示为 f(x)/dx，其中 dx 表示函数 f(x)的值的变化，关于变量 x。

现在让我们了解一下导数在 MATLAB 中是如何工作的。

**导数的语法:**

*   差异(f)
*   差分(f，n)
*   diff (f，var)
*   diff (f、var、n)

### MATLAB 导数的例子

下面举几个例子

#### 示例 1–差异(f)

这个函数将用变量 x 来区分 f

下面是一个使用 diff (f)计算函数微分的例子:

让我们将一个多项式函数定义为:

`x ^ 4 + 3 x ^ 3 + 4 x ^ 2 + 5 x + 1`

**语法**

`df = diff (f, x)`

函数 diff 将返回函数 x ^ 4 + 3 x ^ 3 + 4 x ^ 2 + 5 x + 1 的导数值:

`4x^3 + 9x^2 + 8x + 5`

**输入**:

`syms f(x)
f(x) = x ^ 4 + 3*x ^ 3 + 4*x ^ 2 + 5*x + 1
df = diff (f, x)`

**输出**:

![matlab derivative output 1](../Images/c962341379a86d63463ea130c8433e26.png)

#### 示例 2–diff(f，n)

该函数用于计算输入函数的 n 阶导数。n' here 作为参数传递。

下面是一个例子，我们用 diff (f，n)计算一个函数的‘n’阶导数:

让我们用另一个多项式函数来定义:

`3 x ^ 3 + 2 x ^ 2 + 7 x + 5`

对于我们的例子，我们将计算二阶导数。

从数学上讲，我们的输出应该是:

`18x + 4`

**输入**:

`syms f(x)
f(x) = 3*x^3 + 2*x^2 + 7*x + 5
df = diff (f, 2)`

**输出**:

![matlab derivative output 2](../Images/f9b9fcd53ae290f646967de664bd9832.png)

<noscript><img class="alignnone size-full wp-image-363681" src="../Images/f9b9fcd53ae290f646967de664bd9832.png" alt="matlab derivative output 2" width="351" height="94" srcset="https://cdn.educba.com/academy/wp-content/uploads/2020/05/matlab-derivative-output-2.png 351w, https://cdn.educba.com/academy/wp-content/uploads/2020/05/matlab-derivative-output-2-300x80.png 300w" sizes="(max-width: 351px) 100vw, 351px" data-original-src="https://cdn.educba.com/academy/wp-content/uploads/2020/05/matlab-derivative-output-2.png"/></noscript>

正如我们所看到的，diff (f，n)已经计算了输入函数的二阶导数。

#### 示例 3–差异(f，var)

此函数将查找自变量中变量的导数。

下面是一个使用 diff (f，var)计算函数微分的例子:

让我们将正弦函数定义为:

`Sin(x*t^2)`

对于我们的例子，我们将计算导数 w.r.t 't '

从数学上讲，我们的输出应该是:

`2txcos(t^2x)`

**输入**:

`syms x t
diff(sin(x*t^2),t)`

**输出:**

![output 3](../Images/b467d3597fd854bb0d36666fc967d05f.png)

<noscript><img class="alignnone wp-image-363682 size-full" src="../Images/b467d3597fd854bb0d36666fc967d05f.png" alt="output 3" width="244" height="99" data-original-src="https://cdn.educba.com/academy/wp-content/uploads/2020/05/matlab-derivative-output-3.png"/></noscript>

正如我们所见，导数是按照我们的预期计算的。

[Notice that we have passed ‘t’ as an argument. Diff function will now take the derivative w.r.t ‘t’]

#### 示例 4–diff(f，var，n)

这个函数将计算输入函数对我们作为参数传递的变量的第 n 阶导数。

下面是我们使用 diff (f，var，n)计算函数导数的示例:

让我们将正弦函数定义为:

`Sin(x*t^4)`

对于我们的例子，我们将计算“二阶”导数 w.r.t 't '

从数学上讲，我们的输出应该是:

`12t^2xcos(t^4x) - 16t^6x^2sin(t^4x)`

**输入:**

`syms x t
diff(sin(x*t^4),t,2)`

**输出:**

![output 4](../Images/c71a84acdbd3c0b7e89f81a5fe0bd449.png)

<noscript><img class="alignnone wp-image-363683 size-full" src="../Images/c71a84acdbd3c0b7e89f81a5fe0bd449.png" alt="output 4" width="510" height="110" srcset="https://cdn.educba.com/academy/wp-content/uploads/2020/05/matlab-derivative-output-4.png 510w, https://cdn.educba.com/academy/wp-content/uploads/2020/05/matlab-derivative-output-4-300x65.png 300w" sizes="(max-width: 510px) 100vw, 510px" data-original-src="https://cdn.educba.com/academy/wp-content/uploads/2020/05/matlab-derivative-output-4.png"/></noscript>

正如我们所看到的，我们已经得到了输入函数 w.r.t 't '的二阶<sup>和</sup>导数，正如预期的那样。

[Notice the arguments ‘t’ and ‘2’, for derivative w.r.t and nth derivative respectively]

### 结论

“diff”函数在 MATLAB 中用于计算函数的微分或导数。我们可以计算默认变量或者作为参数传递的变量的导数。此外，微分或导数的程度可以使用自变量来控制。
