## GIMP 字体介绍

在这篇文章中，我们将学习如何在 GIMP 中使用字体。现在使用字体的第一步是知道如何安装字体，为此你应该知道在哪里可以找到它们。所以首先我们会看看各种可以下载字体的网站，然后看看如何在 GIMP 中安装。从技术上来说，你可以使用系统上的字体，但有时我们需要系统上没有的字体，我们甚至可能没有权限在系统范围内安装它，所以在这种情况下，安装 GIMP 字体是必要的。

**第一步:**我们先从下载字体开始。有许多免费和付费的字体网站，你可以从那里下载字体。一些免费网站有 google fonts，dafont，urbanfonts，fontspace 等。我们将看到如何从谷歌字体网站下载如图所示。首先，我们将改变过滤器显示无衬线和显示，这将减少搜索结果。我们不想要衬线，它会在字母的顶部产生闪光。此外，我们不希望手写，这将更像草书，而不是编码字体的等宽。

![GIMP Fonts 1](img/bdb439a2b4505d4f102bfdc7cdd308b3.png)



**步骤 2:** 然后，我们可以根据需要定制文本，或者从下拉列表中选择句子、字母、段落或数字。

![GIMP Fonts 2](img/735ddfab23183fc411e9eec96c6e0a45.png)



第三步:接下来，我们可以使用滑块来改变字体的大小

![GIMP Fonts 3](img/61b775e77da4c3e850296892bd97f8fb.png)



**步骤 4:** 然后我们选择了一种名为 Cairo 的字体，我们可以看到有多种选项或样式可供选择。如果要查找非标准字符，还可以查看字体的字形类型。在“关于”部分，你可以了解字体的历史和作者或设计者的信息。此外，如果你正在寻找用于商业目的的字体，还有一个许可选项，最后一个选项是配对，这是谷歌建议的，如果你想要另一种字体与这种字体相结合。

![GIMP Fonts 4](img/1c076480fbcdff79b28377198249c9e9.png)



**第五步:**现在我们已经选择了常规风格。在这里，我们可以选择查看我们选择的字体，如果您在网站上使用，或者为了我们的目的，如果我们想与 GIMP 一起使用，我们可以选择“全部下载”。此外，如果您想下载这种特定字体的所有字体样式，可以选择下载系列。

![GIMP Fonts 5](img/7ed2b695cfd541970ee1d66ae087958f.png)



**步骤 6:** 一个 zip 文件将被下载到您的计算机上，右键单击该文件可以将其解压缩，所有字体都将位于解压缩后的文件夹中。此过程将在 windows 上运行。

![GIMP Fonts 6](img/ed9e6c893fb6a8fb89f869fa1c35910a.png)



**第七步:**打开 GIMP，然后进入编辑- >首选项。

![GIMP Fonts 7](img/5050190ed78169aec647284de231442c.png)



**步骤 8:** 打开文件夹标签，进入字体部分，选择如图所示的路径，然后点击指示在文件管理器中显示文件位置的按钮。

![GIMP Fonts 8](img/291f8cc9a855c71cad3adb707e6bf8d1.png)



**步骤 9:** 这将打开 GIMP 存储字体的位置。现在我们将把下载的字体复制到这个位置。

![GIMP Fonts 9](img/f04d7ea9cab913518f4083ef9eb60935.png)



**步骤 10:** 然后退出 GIMP 中的首选项窗口，然后从窗口- >可停靠对话框- >字体中打开字体面板。单击“刷新”按钮，在“字体”面板中加载字体。

![GIMP Fonts10](img/75ce0da8a1a37374dac6e5e71fc6fcd0.png)



接下来，我们将从文件菜单中创建一个新文档来测试我们的字体。

![GIMP Fonts11](img/f991cd64e6de3bb60d34c94a699e7774.png)



**第 12 步:**我们将保留所有默认图像尺寸，然后按 OK。

![create file](img/1011262813ea58d606f92bffa85ee64d.png)



第 13 步:使用文本工具，只需拖动一个矩形，键入一个示例文本，并确保字体是我们刚刚安装的 Cairo。

![checking new point](img/16737833b13b8d0eac79b15e79b6f903.png)



第 14 步:现在按 Ctrl + A 选择所有文本，然后你可以将字体改为类似 Cairo Semi Bold 的字体

![checking new point 1](img/5db9cac1e7f6b3fc10fcac248b90c9a2.png)



另一种安装字体的方法不是专门针对 GIMP 的，而是对整个窗口都可用的，你可以双击一种字体，一个预览窗口就会打开。有一个通过点击安装按钮来安装的选项，它将被安装在 windows 上。这种方法只在 Windows 上有效，在 Mac 或 Linux 上无效。

![gimp 15](img/8337dd3a1f50493add80a530c727cc8c.png)



**步骤 16:** 在 mac 中，如果你想安装字体，那么在 finder 中进入- >进入文件夹…然后输入/Library/Fonts，然后点击 Go。这将带你到系统文件夹，在那里你可以拖动下载的字体。这将在系统范围内安装字体。

**步骤 17:** 让我们看看如何使用文本工具的一些基本用法，你可以点击它来添加一个连续的文本，或者点击并拖动它来创建一个文本块。左侧的文本选项中有更改字体的选项，在这种情况下，您不需要选择文本，因为这些选项适用于其中的所有文本。

![testing fonts](img/554b89780014abd27f61e5128f05e076.png)



**步骤 18:** 虽然你可以从浮动选项中改变文本，但那是特定于选择的。

![testing fonts 1](img/c6d614ec5839ffaa8a9354477ba3f709.png)



**第 19 步:**也可以从浮动菜单中更改字体大小。

![gimp 18](img/7942888f0cdbc485f98767d8667907ab.png)



**步骤 20:** 然后有一个使用编辑器的选项，它会打开一个新窗口，您可以在其中添加文本，然后对其进行格式化，但预览将显示在主窗口中，除非您选择了使用选定的字体(如果您喜欢以这种方式工作)。

![testing fonts 2](img/cc0c41637c2adf70df850843dbadda63.png)



第 21 步:下一个选项是抗锯齿，除非你想要一个艺术的外观，否则平滑字体应该一直打开。此选项将减少锯齿边缘。此外，不断暗示，这使字体在小字体大小可读。然后是文本颜色、对齐和缩进、行和字符间距的标准选项。

![testing fonts 3](img/f34d5d3d906bba851bfca37c1c34fed2.png)
