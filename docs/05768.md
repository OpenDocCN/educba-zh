## Tableau Union 简介

Tableau union 用于通过不同的选项追加两个或多个表。这里，要组合的表应该出现在链接到同一数据源的同一数据连接中。如果数据源管理联合，则一旦数据源连接到数据，就会在数据源的左窗格中看到另一个新的联合选项。

### 什么是 Tableau 联盟？

简单地说，tableau union 用于连接存在于同一数据源和连接下的多个表。支持的连接器在 tableau online、tableau server 和 tableau desktop 之间具有各种功能。如果数据源使用 union 选项，那么一旦建立了数据连接，它就会出现在数据源站点的左侧窗格中。为了获得卓越的结果，要在 union 选项下组合的表应该具有类似的结构。每个表都应该有相同的字段计数和一个具有匹配数据类型和字段名的关联字段。

### 如何使用 Tableau Union 数据库？

使用 tableau 数据库或桌面中的通配符搜索选项来联合这些表。

若要实现此方法，请在通配符中使用一个字符，即星号*，它符合工作表名称、excel 工作簿、文本文件中数组的模式或序列。pdf 文件、JSON 文件、google sheets 和数据库表。

单击 New union 选项在数据源页面上建立一个联合。

#### 2.修改联合

用户可以使用下面的步骤通过删除或添加表来自定义 union。

*   在联合数据中，应该双击逻辑表。
*   从联合中的下拉箭头选择编辑联合选项。
*   如果用户想要移除该表，他可以将鼠标悬停在其上并相应地移除它。如果他想添加一个新表，他可以从数据源页面拖动一个表。
*   完成所有修改后，选择确定。

#### 3.移除工会

双击联合数据中的逻辑表，并从下拉箭头中选择 remove。

合并联合中不匹配的名称:

*   当表列在联合中不匹配时，它会在结果表中添加零值。
*   选择需要合并的列。
*   在列下拉选项中，选择合并不匹配的字段。

### Tableau 联合示例

下面是 tableau union 在数据库下实现的简单函数。

*   Join 选项可用于被联合的表中。
*   两个联合表可以用 join 选项再次合并。
*   Union 生成强制字段，如表名和工作表，这些字段可以与连接键一起使用。
*   如果名称范围出现在联合中，则工作表字段中会显示零值。
*   从 merge 中出现的 field 选项可以被更改并用作 pivot、split 或 join 键。
*   在 tableau desktop 中，当用户必须在 Excel 中工作时，他可以使用通配符搜索来包含名称范围，但在解释数据时，它会从表中排除。
*   要使用 union 组合 JSON 文件，它应该有。日志扩展名或。txt，或者。json 格式。
*   当用户使用通配符搜索来合并文件时。输出放在初始的。pdf 文件。
