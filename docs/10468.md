## VMware 简介

VMWare 是戴尔技术公司的子公司。它为云计算和虚拟化提供了一个平台。它是第一家在商业上成功虚拟化 x86 体系结构的公司。它是软件行业中虚拟化的主要提供商之一。它们被分为两个层次，即桌面应用程序和服务器应用程序。VMWare 的[桌面软件](https://www.educba.com/what-is-desktop-software/)兼容所有主流操作系统，包括 Linux、微软 Windows 和 MAC OS。第一个软件产品于 1999 年 5 月发布。后来，他通过发布 VMWare GSX 服务器和 VMWare ESX 服务器进入了服务器市场。

### 什么是 VMware？

它是 VMWare Inc .的一个虚拟机软件。它支持同一操作系统的多个副本，或者可以有几个不同的操作系统同时在同一台基于 x86 的机器上运行。它支持在 Windows 或 Linux PC 上运行多个操作系统。它还有像 VMWare ACE 这样的部署工具。用户的桌面可以存储在 USB 驱动器上以便运输。它是创建基于软件或虚拟表示的过程，包括服务器、存储和不同的网络。它有效地充当了降低 It 开支、提高效率和敏捷性的解决方案。

### 谅解

虚拟化是一个基于软件而不是物理的表示的过程。由程序管理的虚拟化称为管理程序。虚拟机管理程序是一种帮助创建和运行虚拟机的软件。这里有两种类型，如下所示:

1.  类型 1 本机或裸机虚拟机管理程序
2.  类型 2 或托管的虚拟机管理程序

Type 1 提供与硬件的直接交互，它可以直接运行在主机的硬件上并控制它。

在类型 2 中，需要操作系统与硬件交互。虚拟机管理程序随操作系统一起安装。

### VMware 如何让工作变得如此简单？

像 x86 服务器这样的各种服务器都有局限性，这可能会给 IT 组织在多台服务器上部署东西带来问题。这也带来了与高存储和处理需求保持同步的问题。虚拟计算机系统被称为具有软件容器的虚拟机，该软件容器内部具有操作系统和应用程序。所有自含式虚拟机都是完全独立的。这可以是单个计算机上的多个虚拟机，这提供了在一个物理服务器上运行多个操作系统和应用程序的便利。它还提供了分区、隔离、封装和硬件独立性等特性。

### 使用 VMware

这样做很容易，需要遵循以下步骤。首先，你需要[安装 VMWare](https://www.educba.com/install-vmware/) 工作站，然后安装操作系统。您可以命名虚拟机并设置磁盘大小。完成后，用户可以定制虚拟机的虚拟硬件。然后，您现在可以启动虚拟机了。安装完成后，您可以开始浏览 VMWare。您可以轻松地在虚拟机和物理计算机之间移动文件。打印机也可以用你的名字添加。一旦你熟悉了这个系统，你也可以用按键找到快捷方式。

*   你可以很容易地在 Windows 上运行 Linux。
*   打造个人云电脑是可行的。
*   旧硬件的再利用很容易。
