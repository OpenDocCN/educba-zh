## Unix 使用简介

Unix 是一种操作系统。它支持多任务和多用户功能。Unix 最广泛地应用于所有形式的计算系统，如台式机、笔记本电脑和服务器。在 Unix 上，有一个类似于 windows 的图形用户界面，支持轻松导航和支持环境。有了 GUI，使用基于 Unix 的系统变得很容易，但是仍然应该知道在 GUI 不可用的情况下使用 Unix 命令，比如 telnet 会话。

UNIX 有几种不同的版本，但是有许多相似之处。最流行的 UNIX 系统是 Sun Solaris、Linux/GNU 和 MacOS X。

### Unix 的十大用途

在本节中，我们将提到 10 大用途:-

#### 1.多重任务

Unix 是为高性能计算而设计的。它旨在牢记分享的概念。多个人可以同时登录到同一台机器，一些人也可以从不同的位置登录到同一台机器。Unix 比 Windows 系统负担更轻，具有高度的可定制性，并且更安全。它最常用于 web 服务器。

#### 2.现代技术的支柱

Unix 是互联网服务和其他各种服务(如聊天和网飞)的支柱。Red Hat 是 Linux 的一个流行版本，它是 Unix 的一个实例，有命令行和 GUI 可供使用。UNIX 或 Linux 命令提示符工具在灵活性、速度和多功能性等方面比 Windows 工具优越得多。

#### 3.对其他操作系统有影响

在最初的日子里，Unix 对其他操作系统产生了重大影响。诸如交互性、免费使用、象征性收费的教育用途、廉价的硬件、易适应性等成就。是 Unix 的巨大优势。Unix 启发了许多其他计算系统。

#### 4.Linux 的一项发明

1991 年，Linus Torvalds 将 Linux 内核作为自由软件发布。它是在 GNU 通用公共许可证下发布的。Unix 除了用于 Linux 操作系统之外，还提供其他 GNU 包，如 GNU 编译器集合和其他类似的工具链、GNU C 库以及 GNU 核心工具。这些产品在许多其他基于 Unix 的免费系统中也发挥了重要作用。由 Linux 内核和除此之外的大量自由软件组成的 Linux 发行版。Linux 在杰出的开发人员、个人用户、社区和企业中都很受欢迎。有许多流行的 Linux 发行版，包括 Red Hat Enterprise Linux、SUSE Linux Enterprise、Fedora、OpenSUSE、Ubuntu、Debian GNU/Linux、Linux Mint、Slackware Linux、Mandriva Linux、Arch Linux 和 Gentoo 等。

#### 5.一系列服务

与同时代的许多其他操作系统相比，Unix 的使用成功地大大简化了文件模型。这是通过将所有类型的文件都视为简单的字节数组来实现的。这些文件系统层次通常包含机器服务和设备，如打印机、软盘驱动器或终端，并提供统一的接口。所有这些都是以偶尔需要一些额外的机制(如 ioctl 和模式标志)来访问那些不符合简单字节流模型的 Unix 哲学的硬件特性为代价的。

#### 6.命令行功能

Unix 普及了使命令解释器如此简单的思想，以至于任何普通用户都可以使用它。这是通过作为独立程序提供的一组附加命令来完成的。Unix shell 为交互式命令和脚本实现了相同的语言。Unix 有一个创新的命令行语法来创建流水线进程的模块化链，这产生了一个强大的编程范例，也称为协程。后来出现的几个命令行解释器受到了 Unix shell 的启发。

#### 7.基于文本的流程

Unix 的一个简单而基本的假设是为几乎所有文件格式创建一个换行符分隔的文本。在 Unix 的最初版本中，没有二进制编辑器。这意味着整个系统是基于文本 shell 命令脚本的。输入输出系统中的共同点是字节。基于文本的这种过程使得 Unix 管道非常有用，并促进了简单和通用工具的开发，以执行更复杂的任务。Unix 对文本和字节的专注使得它非常具有可伸缩性和可移植性。基于文本的应用程序已经被证明在印刷语言领域非常流行，如 PostScript 或 ODF 等。Unix 也为因特网协议的应用层做出了贡献，如 FTP、HTTP、SMTP、SIP 和 SOAP 等..

#### 8.正规表达式

Unix 普及了正则表达式语法，这种语法后来变得非常普遍。Unix 编程接口的使用现在是设计操作系统接口标准的基础。

#### 9.模块化和可重用性

早期的 Unix 开发人员将模块化和可重用性等概念引入软件工程实践，这实际上是一个巨大的贡献。Unix 的主要开发者后来也建立了一套开发软件的文化规范。这些规范也成为 Unix 不可或缺的、有影响力的指南。

#### 10.网络协议

Unix 也为 TCP/IP 网络协议做出了贡献，这确保了相对便宜的计算机在社区中变得流行和广泛使用。这在全球范围内创造了一个具有实时连接的互联网热潮，也为在如此多的其他平台上的部署奠定了基础。

### 结论

Unix 不可或缺。从简单的命令行应用程序到与服务器连接和对话，Unix 使得基于 GUI 的其他操作系统所不能做到的事情成为可能。Unix 存在于各种应用程序和系统中，包括 Android、iOS、PlayStation 等。

那些将要从事服务器技术和管理工作的潜在候选人，肯定应该学习 Unix，熟悉它的命令、用例以及核心原理。特别是，那些操作 Linux 或 Ubuntu 系统的人，甚至那些想要进行大数据分析的人，肯定应该学习使用 Unix。pwd、chdir、dir、ls、ls-l、passwd 等 Unix 命令的简单应用，应该是所有计算机专业毕业生或者计算机爱好者都知道的。
