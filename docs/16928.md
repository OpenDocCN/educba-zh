## SketchUp 木工入门

SketchUp 木工涉及设计具有所有精确尺寸的木材 3D 模型，以便木工可以为该模型准备切割木材的布局。木工和设计我们在这个软件中创建的另一个模型是一样的，但是在创建这个模型的过程中，你必须记住一些要点。例如，如果您正在制作任何桌子，那么木工在锯木厂创建该桌子时会切割许多凹槽，因此您可以在该软件中输入该凹槽的值并检查装配情况。和这个一样，我们可以做很多事情。

### 如何在 Sketchup 中进行木工？

为了以正确的方式进行木工，你要很好地理解木制模型的尺寸。然后，我就通过做一个开放式的木柜来告诉你木工的知识。

<small>3D 动画、建模、仿真、游戏开发&其他</small>

在 SketchUp 的默认模板中，你会有一个木工模板，这样你也可以使用它。木工模板有两种，一种是英寸，一种是毫米。我将使用英寸模板。

![SketchUp Woodworking Output 1](img/b29908ceb43cf6355e8f7a6386f8e75d.png)



现在选好模板后，我就拿制作橱柜主体的矩形工具。这个橱柜我取的是粗略尺寸，不是标准尺寸。所有的尺寸只是为了给你一个木工的概念。

![SketchUp Woodworking Output 2](img/5a156bda1ae117a9b16f0d3d208e0918.png)



画一个 4 英寸×6 英寸的矩形。

![SketchUp Woodworking Output 3](img/e5a8c17efb39bd9fa8ac3e4380d863b7.png)



借助推/拉工具增加一些高度。例如，我将添加 10 英寸的高度。

![SketchUp Woodworking Output 4](img/8b84fbcb15a4b058a55c97ce53ac5373.png)



现在来到这个模型的正面，使用偏移工具，然后点击这个面的边缘。我将拖动这个工具的鼠标指针到这个面里面，因为我想在里面偏移它。我将输入 0.25 英寸作为偏移距离。这将是制作这个橱柜的胶合板的厚度。

![SketchUp Woodworking Output 5](img/33294cd6aa37779004742b5f8ae085fb.png)



现在用线工具在这个面的中心从上中点到下中点画一条线。

![SketchUp Woodworking Output 6](img/3c46e8c166c3ed846c7e56ea496c7439.png)



现在，我将单击右视图图标来查看右视图，这将显示该橱柜的正面。

![SketchUp Woodworking Output 7](img/55138d20246cd54f9fd023f182cbaeb6.png)



现在用卷尺测量工具点击这条中心线，然后向右拖动鼠标光标，输入值 0.125 ' '，它是 0.25 ' '的一半。

![SketchUp Woodworking Output 8](img/f187e03746a7dba863b44621bba1a835.png)



我将在这条中心线的左边做同样的事情。我画这些指南是因为我想在这个橱柜的中央制作一块 0.25 英寸厚的胶合板。

![SketchUp Woodworking Output 9](img/7cdb5228e73ffecc39300c34d302fde8.png)



现在我想把这个橱柜的左边部分分成两个部分，这样我就可以在这个部分的中心画一条线。

![SketchUp Woodworking Output 10](img/4e4eac47cef025b9f2e18685d5f84520.png)



画两条准则，这样我也可以在这里创建一个 0.25 英寸厚的板。

![SketchUp Woodworking Output 11](img/764151ad4aba7e312e406ea95e6141cd.png)



我将把右边部分分成三部分。你可以看到这个橱柜的高度是 10 英寸，如果我们把胶合板的厚度从两边去掉，即 0.25 英寸+ 0.25 英寸= 0.50 英寸，那么剩下的就是 9.5 英寸，所以你可以把这个左边的尺寸分成 3 个部分，即 9.5/3 = 3.167 英寸。所以我会拿一个卷尺测量工具，在 3.167 的距离处加一个指引。

![SketchUp Woodworking Output 12](img/827e31c61de929cb8c301ad9782c4b69.png)



从下一节的指南开始重复。

![SketchUp Woodworking Output 13](img/a97bc295aa40d11ab846d13b19266137.png)



现在删除我们用线条工具绘制的所有线条。

![SketchUp Woodworking Output 14](img/3ea7f7ea08de5e6eafd22ab286dd1de2.png)



在线条工具的帮助下，画出胶合板的厚度，像这样把橱柜分成几部分。

![SketchUp Woodworking Output 15](img/c436c09614744a3b65ccce63d143ea0a.png)



现在选择所有指南并删除它们。然后，如果你从右向左拖动鼠标光标，它将选择你拖动它的整个对象。

![SketchUp Woodworking Output 16](img/3561a1df8d8914fca743161e1632b728.png)



你可以看到我们的部分已经准备好了。

![SketchUp Woodworking Output 17](img/9eb08bb002166c007cd6f77c1e7bae14.png)



如果您想记住模型的尺寸，则可以使用尺寸工具在模型上标注尺寸。只需从工具面板中选择这个工具。

![Dimensions tool Output 18](img/1ee285ddda0cff4978af0e06325b3e64.png)



首先，单击尺寸的起点，然后单击终点，并向上拖动以正确查看。

![SketchUp Woodworking Output 19](img/52b1e5ce22ec6d1a10e11760e97cd68a.png)



现在，我将使用推/拉工具向内拖动该部分，并输入拉力值 4''- 0.25'' = 3.75 ' '，表示该橱柜中有一个空心部分。

![SketchUp Woodworking Output 20](img/1f44c3449a4b0eb351c35f6207138eec.png)



我将对所有部分做同样的事情。

![SketchUp Woodworking Output 21](img/9575522b065399c1a93319697e2f8bbf.png)



现在，我想在这一部分制作两个抽屉，这样我就可以借助线条工具将这一部分分成两个部分。

![Line tool Output 22](img/48e1055b4fe45a02da6aa460fed65fff.png)



现在我将像这样移动上面的矩形。

![SketchUp Woodworking Output 23](img/e996306c98f43532e9b4a1e51e831488.png)



并在 3.75 英寸处挤压它。

![SketchUp Woodworking Output 24](img/dde3f378bdeb21b8800b7c43c3b5026e.png)



现在点击顶视图图标进入顶视图。

![Top view icon Output 25](img/a437ca04fe8fe72bb449cc9fab4aa29d.png)



并使用偏移工具将其上表面偏移 0.25 英寸的距离。

![SketchUp Woodworking Output 26](img/e2ca420a35ce50ef1296dda3d78a9c8a.png)



您可以使用尺寸工具检查此绘图的高度。

![Dimensions tool Output 27](img/02eb8efaeea6f51ada808d2533392dc7.png)



现在，我将根据尺寸用推拉工具增加这个抽屉的深度。

![Push/pull tool Output 28](img/6861c9631f54379922569e8e5a6529fe.png)



然后我会在抽屉的上表面添加线条。

![SketchUp Woodworking Output 29](img/b8f5de38aa6a780a2c7ebb5578074d3f.png)



然后通过输入 0.25 英寸的向下高度将其向下拉。

![Output 30](img/120899d51ac61a6777bf49ba08a884a3.png)



现在把它移到这个碗橱的中空部分。在移动它之前，将它组合成一个组。

![Output 31](img/942039ab00265eaf452aa146e993d3ee.png)



我会通过画这样的线找到它的最小点。

![Output 32](img/419fb99da2c27cec2d4989a0bd5d0f15.png)



在圆形工具的帮助下，我将在它的中心画一个圆。

![Output 33](img/2bc0d5a422be31fc03ecc80cf058ed9a.png)



用橡皮擦工具一条一条的删除所有线条。

![Output 34](img/55390164713c6bd42fce664f0aafe7eb.png)



并在一定高度上挤压出来，使之成为这个抽屉的把手。

![Output 35](img/4377b154ae851256541042bd68b30b9c.png)



现在按住键盘的 shift 键选择 draw 和这个手柄。在这样做之前，请确保您已经将它们分别分组。现在把它们做成一个组件。接下来，对橱柜的每个独立部分进行同样的操作。

![Output 36](img/05c928b8a472f8bd90a040494ec51112.png)



我将把它命名为抽屉 1。

![Output 37](img/53903cac7d09f5f0e45a2a06ef27deed.png)



现在向下移动它，复制一份。移动时按住 Ctrl 键进行复制。

![Output 38](img/be24fc1b03e55453aa5a4e72e4fe9612.png)



这样，你就可以打造一个各种尺寸的开放式橱柜。

![Output 39](img/5df90e11ac95f8204b6be34f2822f647.png)



您也可以将木质材料指定给木质模型的不同部分。

![Output 40](img/c11390db9fab16b16e8a71373fbf8d1f.png)



### 结论

我已经给了你木工概述，你可以创造自己的设计。你只需要按照你想要的模型的适当尺寸，在为模型切割木材之前，轻松地创建一个布局。在创建您的木制模型后，您可以为此创建 Cutlist，它会自动为您准备切割布局。
