## 3D Max 软件中的快捷键

3D max 软件被不同的公司用于建模，这些公司包括动画公司、视频游戏公司、电视和电影工作室。3D max 中的快捷方式为您提供了项目中的第一个快速工作空间。快捷键是通过使用键盘按钮中的快捷键来激活任何命令的快捷方式。

### 用于在 3D Max 中绘图的视口

3D max 软件中有四个视口工作区，分别是顶视口、前视口、左视口，最后一个是透视视口。我们可以随时选择其中任何一个来绘制我们的模型。

![3D Max Short Cuts](img/97f5264448f4a1e6397f10066bbeaa56.png)



为了在这里使用快捷方式，首先，我们需要在我们的视口中绘制一些对象。

![Objects in Viewport](img/e40efd8a81b0b2df568db92d6fc40339.png)



*   **选择所有快捷方式:**用于选择模型中存在的所有对象。要同时选择所有对象，按键盘上的**Ctrl+A**；所有选择都可以在所选对象周围以天蓝色线条的形式看到。

![Selecting all objects](img/7014c9d05d0ddd07c3fa76072e08ee6f.png)



要取消选择，请按键盘上的 **Ctrl + D** 。

![Deselecting objects](img/ca23273832bc5c869be60dbee862a932.png)



*   **切换锁定快捷键:**用于选择单个对象，并将我们的选择锁定到该对象。激活此选项后，我们将无法同时选择任何其他对象。按键盘上的空格键；单击要锁定的对象以激活此选项。

![3D Max Short Cuts 5](img/a8495bf3b1c4150c35190690d84b4da3.png)



*   **线框模式快捷方式:**按键盘上的 F3 键进入对象的线框模式。

![WireFrame Mode Shortcut](img/494e70d1823a89ae5d62e13ccdce3800.png)



再次按下键盘上的 F3 键，返回到对象的阴影模式。

*   **边缘面快捷方式:**从键盘上按下 F4，同时查看物体的边缘面和阴影面。

![Edge faces shortcut](img/403606274c982ee286a8dfd60263b8f4.png)



*   **选择任何具有名称的对象的快捷方式:**我们可以选择任何具有名称的对象，也可以选择按键盘上的 **H** 。将打开一个名为**的对话框，从场景**标题中选择。

![Select From Scene](img/e96025f1659c4d391f9ee3e82fc97cd0.png)



单击对象名称将其选中。

![Object Name](img/640d4d3e6adc3305aa950a5544acfdc8.png)



按确定。

![3D Max Short Cuts 10](img/228fdcaff914756a50a14249f326b1ba.png)



对于下一个快捷方式，我们将单击盒子模型来选择并右键单击它；将打开一个下拉对话框，选择**转换为**选项。点击**转化为可编辑多边形**选项。

![Convert to Editable poly](img/ac40eec0e8975b8dfa70370f888a94d1.png)



现在我们将在这里看到一些快捷方式。按键盘上的 **1** 可以看到顶点模式。

![3D Max Short Cuts 12](img/a14846f8c4cf108fba9d5c415e70cf67.png)



按键盘上的 **2** 进入边缘模式。

![Edge Mode](img/a3a11cbfb7da96ebe1a63587c3b8613b.png)



按下键盘上的 **3** 键，显示边框。

![Short cut for select any object with name 7](img/0570d888cdaa071874caf4794d9e490a.png)



按下键盘上的 **4** 以显示多边形。

![Short cut for select any object with name 8](img/413ed8d3b8c904f752a29debec41e694.png)



按下元件键盘上的 **5** 。

![Short cut for select any object with name 9](img/8ddd27019c8e258b1612db464efa7345.png)



按下键盘上的 **7** ，在 3D max 中查看多边形建模场景中的多边形数。

![Short cut for select any object with name 10](img/69e1a04f98b67e3ec7e4ff43b9e05338.png)



以上这一点的所有快捷方式将有助于我们在多边形建模期间的工作。

*   **环境和效果快捷键:**按键盘上的 **8** 进行环境和效果；我们可以在这里设置任何位图的背景，改变颜色，给一些效果。

![Environment and effect short cut 1](img/277e5efa30b54e545a2f52a8459318e6.png)



*   在工作区重新定位我们的隐藏对象:当我们绘制的对象不知何故从我们的工作区看不见时，这就成了我们的主要问题。

![Relocating our hidden object](img/a63eda88e7e5e83b7597b39e59838327.png)



要在屏幕上找到或重新定位我们的对象，只需按键盘上的 **Z** 即可。

![Relocating our object](img/77073b5e0e31e92031399572728f5ce3.png)



*   **专家模式快捷键:**按键盘上的 **X** 进入专家模式；这样，你可以在你的模型中搜索你所做的任何动作。

![Expert mode shortcut](img/8c5187fcbee5030936e102c44ca2a908.png)



*   **隔离模式快捷键:**按键盘上的 **Alt + Q** 激活隔离模式。隔离模式为我们提供了任何单个对象的深度视图。

![Isolation mode short cut 1](img/7d7aced290b9900f971147744ceeedee.png)



按“退出”按钮，从屏幕右下角的可用按钮中出来。

![Isolation mode short cut 2](img/73e651e4e18a31829cae058ceebbd67c.png)



*   **增加和减小坐标符号大小的快捷方式:**按键盘上的 **+** 按钮增加坐标符号的大小。

![Increasing and Decreasing size](img/b5f9e6f653e6d00dd08a8e8660b6f8c4.png)



并按下**–**按钮缩小坐标符号。

![Decreasing size](img/6cecdb7b293cee5c7d76100707b5fd69.png)



*   **渲染快捷键:**按键盘上的 **Shift + Q** 查看渲染效果。

![Render short cut](img/f978b75ee9dafb95141aab18f96b8091.png)



*   **渲染设置快捷键:**按键盘上的 **F10** ，渲染设置对话框将会打开。

![Render Set Up short cut](img/6bf474409f9325173fec72a6a87922df.png)



在这里，我们可以根据我们的要求设置一些渲染设置。

*   **素材编辑快捷键:**对于素材编辑按键盘上的**M**；将打开一个对话框或材质编辑器。

![Material Editor 3D Max short cut ](img/e8054462f3c5252b5d8c8798f51209c6.png)



我们可以在这里为我们的对象设置材质，设置对象的颜色和许多其他设置，以给我们的对象一个真实有效的视图。

![Material Editor 3D Max short cut 2](img/10e9324f182c399d7f77d455c0a35a3c.png)



*   **圆锥体快捷方式:**圆锥体是指用相同的大小和形状制作物体的相同副本。选中对象后，按下键盘上的 **Ctrl + V** 键，将对象锥化。

![Cone short cut 1](img/9228c8cf8a33ad71a0fc2242b3879dd0.png)



或者您可以简单地点击对象，并通过按住键盘的 shift 键拖动鼠标指针来使该对象变成锥形。

![Cone short cut 2](img/f08723610db7264de05b7d108e5c3a52.png)



*   **X 射线模式快捷方式:** X 射线模式是指我们可以通过激活该模式来透视物体。透明意味着我们可以看到第二个物体，它在第一个物体的后面。在主动 X 射线模式下，按键盘上的 **Alt + X** 。

![X-ray mode 3D Max short cut](img/43cde336ab1ebac8d8c0c9e822f60da0.png)



*   **切割工具快捷方式:**下一个快捷方式是多边模式，可通过按键盘上的 2 个按钮激活。按下键盘上的 **Alt + C** 激活剪切工具，在我们的对象中进行任何剪切。

![Cut Tool 3D Max short Cut](img/3cd3417ff02b8170d66e42b808f317d7.png)



*   **捕捉工具快捷键:**捕捉使建模变得简单。要激活对象的捕捉，请按键盘上的 **S** 。

![Snap tool 3D Max short cut](img/5d4c76318902bc6928eb77b49ce4d790.png)



这些都是 3D max 软件中经常使用的快捷方式
