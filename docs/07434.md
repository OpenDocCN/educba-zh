### JSON 解析器在 C#中是如何工作的？

使用 JsonConvert 解析 JSON 字符串。DeserializeObject()属于 JsonConvert 类，该方法称为 JsonConvert。用于将 JSON 字符串转换为 C#对象的 DeserializeObject()。这些对象是由 JSON 字符串创建的。
