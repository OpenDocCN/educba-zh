## Perl 推送数组简介

Perl push array 用于将质量的下降推到展览的终点。push()工作经常被用来执行堆栈。push()工作不依赖于那种被传为破败的品质。这些品质可以是字母数字。Perl push()工作用于将价值或质量推送到集群的终点，从而构建组件的数量。此时的新质量成为集群中的最后一个组件。

**语法和参数:**

<small>网页开发、编程语言、软件测试&其他</small>

`Push(array, list)`

在哪里，

*   该数组表示 push 函数中必须添加的值的序列。
*   该列表表示必须添加到特定序列中的值。
*   它返回新发现的数组的各种元素。

### Perl 中如何推送数组？

下面给出了如何在 Perl 编程中推动数组的例子。

#### 示例#1

**代码:**

`@array = ( 5, 6, 7 );
print "First array: @array \n";
push(@array, (9, 10, 11));
print "Newly Discovered Array: @array \n";`

### 结论

因此，我们想以 push()工作恢复集群中新的全部数量的组件作为结论。很难将这种能力误认为是 unshift()作品，它在展览的开始添加了组件。push()工作将新的价值或品质推到展品的正确一侧，并构建组件。展览也可以是一个堆栈的概念。想象一堆编号的盒子，从顶部的 0 开始，向下扩展。每一行代码都应该以分号结束。如果没有，它就不会执行。在本文的堆叠模型中，不带分号的行是包含在一个集群中并被封装在外壳中的值。这并不是分号规则的特例，主要是因为堆栈方法。集群中的质量不是单独的代码行。用一种简单的方式来处理编码会更简单。
