## Inkscape 阴影介绍

在这篇文章中，我们将学习如何在 Inkscape 中创建阴影。阴影是设计师工具箱中的有用工具。它可以用于纯粹的审美目的，或者你也可以用于实际目的，比如如果你创作一个艺术品，但不确定它是用于深色背景还是浅色背景，那么使用阴影是理想的，因为它将创建一个对比，而不管背景图像。Inkscape 中的 Shadows 也用于创建用户界面元素，如按钮和文本，它们看起来凸起，并根据用户交互改变状态。

### 墨迹阴影的步骤

下面是下面提到的步骤

<small>3D 动画、建模、仿真、游戏开发&其他</small>

第一步:用椭圆工具画一个椭圆，并设置填充颜色。

![Inkscape shadow output 1](img/613fac556a5a44b9052fecc8083f2982.png)



第二步:使用 Ctrl + D 复制椭圆，将填充颜色改为之前颜色的较暗阴影，因为它将用于制作阴影。

![Inkscape shadow output 2](img/35f7b0baef9b230bd2821b32e825ec07.png)



然后打开填充和描边窗口，这样我们可以使用它的模糊功能来创建一个阴影。

![Inkscape shadow output 3](img/1d5a6f7d3d52190410c0c9217f4f60af.png)



第四步:给上面的椭圆一些模糊，这将产生阴影效果。

![Inkscape shadow output 4](img/234d1d9e744796c3f8d8deeed3c025d3.png)



**第五步:**然后点击工具栏上的底部按钮或键盘上的结束按钮将其移动到下方，这将显示底部的椭圆，并给它一个阴影的效果。

![Inkscape shadow output 5](img/9d279dfbc2a65180fa0fabc8075f5c9a.png)



步骤 6: 类似的技术可以应用于文本。使用文本工具写任何文本，并增加大小，如 100 磅。你甚至可以根据你的文字加粗。

![Inkscape shadow output 6](img/3bf65a302ce9b5825770aa4f57e4c877.png)



步骤 7: 然后按 Ctrl + D 复制文本，从填充和描边面板中给它一些模糊，比如 20%。

![Inkscape shadow output 7](img/bc52c2164ec328e6caf2d390fb68ed92.png)



**第八步:**然后从底栏开始改变文字的颜色。现在我们可以看到阴影和文字似乎提高了。

![Inkscape shadow output 8](img/a31d6b0e6119519ee4dc0aba85d37f46.png)



**第九步:**然后我们再看一些扁平的阴影。对于使用矩形工具，我们将首先绘制一个矩形。设置绿色填充。

![Inkscape shadow output 9](img/f384f2ce92ddbcc74ce4fa30a8c73d2c.png)



**步骤 10:** 然后按 Ctrl + D 复制，设置黑色的填充。然后把它移到右下角。使用键盘可以准确，因为你可以设置多少像素。你可以用箭头键进行微调。一个专业建议是，如果你在使用箭头键时按 shift，那么你将一次移动对象 5 个单位

![Inkscape shadow output 10](img/3a9d4b59b1c7fe9e1026453f6348b2f0.png)



**步骤 11:** 使用底部选择按钮将选择移动到底部或按键盘上的 end。

![Inkscape shadow output 11](img/80258aebbc95369faf4621824e8d398f.png)



第 12 步:再次按 Ctrl + D 复制黑色方块，将填充设置为白色，并使用将选区提升到顶部或按 Home 键将其上移。

![Inkscape shadow output 12](img/10a945d9a12d3d687047e92f4d711285.png)



**步骤 13:** 我们将再次执行相同的操作，使用箭头键在黑色和绿色矩形之间移动白色矩形。然后移动绿色和黑色矩形之间的白色矩形使用较低的选择一步或使用向下翻页按钮。然后将黑色矩形的颜色改为绿色，与顶部矩形相同。我们的平面阴影几乎完成了。但是如果你在透明的背景下使用它，就会有问题。

![Inkscape shadow output 13](img/caa5e631faf36e248ef424d533608d3b.png)



步骤 14: 我们可以看到，如果我们在绿色矩形后面创建一个红色矩形，阴影和物体之间就会产生一个负空间。

![Inkscape shadow output 14](img/0b78c80347eedf2b0cb4a421884ea865.png)



**步骤 15:** 要解决这个问题，让我们选择底部的两个矩形，然后到路径- >差异或按 Ctrl+–删除白色的矩形。

![Inkscape shadow output 15](img/7039883ac2360a4e92dafd9ea3813ad9.png)



第十六步:正如我们现在看到的，我们可以在任何背景下使用这幅作品，平面阴影会正常显示。

![Inkscape shadow output 16](img/114d079690e68287e3661fb751896bbe.png)



步骤 17: 现在我们来看看如何在文本上应用平面阴影。使用文本工具写任何文本，设置大小为 144 磅左右，并给它红色。

![Inkscape shadow output 17](img/a0262eb5b30d8aabd45afd3043f2375f.png)



步骤 18: 然后按 Ctrl + D 复制选中的对象，给它黑色。

![Inkscape shadow output 18](img/9828130bd2d2f7a6c1bd96fa66b88aaa.png)



**步骤 19:** 然后根据你想要调整阴影的方式，使用箭头键调整黑色文本。

![Inkscape shadow output 19](img/2156cec57e2e1babf939d4b92d2005c2.png)



**步骤 20:** 调整完成后，将选项降低到底部或按结束键。

![output 20](img/cf73f223b915ccd7adb5d27f2647c399.png)



**步骤 21:** 然后按 Ctrl + D 复制黑色文本或阴影，使其变成白色，然后使用箭头键将其调整到红色和黑色文本之间的位置。

![output 21](img/7729d8dc12b27bb5a63b19c3161747a5.png)



**步骤 22:** 然后使用下方的选择前进一步或按 page down，使白色文本位于红色和黑色文本之间。这将创建一个完美的阴影跟随文本。

![output 22](img/34e126f99eb0e9b6068ef40bfdb0fd90.png)



**步骤 23:** 再次清除负空间，我们选择白色文本和黑色文本，然后使用路径菜单中的差异或按 Ctrl+–从黑色文本中移除白色。

![output 23](img/1d6f714b36974e8bbaa1b7968b4437e4.png)



**步骤 24:** 用相同的红色填充阴影，使阴影颜色与文本颜色相同，或者你可以使用“从图像中选择颜色”工具，或者在确保阴影或黑色文本被选中的同时按 D 键，然后点击红色文本，使阴影变为红色。

![output 24](img/72f1e65191860ee5678ab9f5e3cbae75.png)



第 25 步:为了确保我们已经删除了负空间，我们将画一个矩形，并给它黄色填充。

![output 25](img/044fc5b0023a0f0bce86dcc3702a5bcc.png)



**步骤 26:** 然后使用底部选择或按 end 将矩形移动到底部。我们已经成功地消除了阴影和文本之间的负空间

![output 26](img/7dd346ec33952bef90261c5303cdc2b0.png)



第 27 步:还有一个滤镜可以在阴影和发光中使用- >投射阴影。我们将做一个圆并应用于它。

![output 27](img/40131a228d1a9e8066e7bbcb64a0cc04.png)



**步骤 28:** 在这里你可以调整模糊半径、水平和垂直偏移、阴影类型以及改变阴影的颜色。

![output 28](img/d86f0f34f15c20d3c093448b22fa4843.png)
