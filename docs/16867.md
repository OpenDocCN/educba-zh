## Figma 替代品介绍

Figma 是一个基于浏览器的 GUI 设计框架，但它还不止这些。我敢说，这绝对是团队协作设计企业的完美应用。Figma 为该项目的设计过程提供了所有必要的资源，包括全长幻像支持矢量工具、原型制作能力和移交代码创建。虽然 Figma 是基于网络的版本，但也有 Windows 和 macOS 桌面版本。这些可能是内置浏览器的包装应用程序，但 Figma 仍然在这种应用程序中在线运行，不管是不是这样。
