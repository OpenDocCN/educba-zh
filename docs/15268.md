## Lisp 和 Python 的区别

下面的文章提供了 Lisp 和 Python 的概要。一种解释的、面向对象的、具有动态语义的高级编程语言被称为 Python 编程语言，由约翰·麦卡锡创建的第二种高级编程语言被称为 Lisp 编程语言。与 Python 编程语言相比，Lisp 编程语言中的符号计算要容易得多。与 python 编程语言相比，特定领域语言在 Lisp 中是微不足道的。与 Python 编程语言相比，Lisp 编程语言的性能更好，并且用 Lisp 编程语言编写宏比用 Python 编程语言更容易。

### 关键的区别

让我们讨论 Lisp 和 Python 之间的一些主要区别:

*   符号计算在 Python 编程语言中难以执行，而在 Lisp 编程语言中则容易得多。
*   在 Lisp 编程语言中，符号计算是自然发生的，而在 Python 编程语言中，它不是自然发生的。
*   特定于领域的语言在 Python 编程语言中耗时较长，而在 Lisp 编程语言中耗时较少。
*   与 Lisp 编程语言的性能相比，Python 编程语言的性能要差一些。
*   Python 编程语言中没有宏，而 Lisp 编程语言由宏组成。
*   Python 编程语言必须使用预定义的语法，而自己的语法可以用 Lisp 编程语言编写。
*   仅使用 Python 编程语言中的可用功能有一些限制，因为预定义的功能和语法只能在 Python 编程语言中使用。
*   只使用 Lisp 编程语言的可用特性没有限制，因为可以用 Lisp 编程语言编写自己的语法。
*   Lisp 编程语言可以定制，而 Python 编程语言是不可能定制的。
*   Lisp 编程语言包含多行匿名函数，而 Python 编程语言不包含多行匿名函数。
*   与 Python 编程语言相比，Lisp 编程语言更适合元编程。
*   Lisp 编程语言只能在 Emacs 集成开发环境中编写，不能在任何其他集成开发环境中编写，而 Python 编程语言可以在几种不同的环境中运行。
*   Lisp 编程语言具有灵活性，在程序包含错误的情况下，只运行包含错误的函数的一部分，而不是整个程序，而整个程序必须用 Python 编程语言编译和运行以修复错误。
*   Lisp 编程语言不需要测试或持续集成，而 Python 编程语言需要测试和集成。
*   没有预定义的 Lisp 编程语言模式，而 Python 编程语言中有预定义的模式。
*   与 Lisp 编程语言相比，Python 编程语言的处理速度更快。
*   与 Lisp 编程语言相比，Python 编程语言是最适合从事人工智能工作的语言。
*   Lisp 编程语言中没有支持社区，但是 Python 有强大的支持和社区来互相帮助。
*   其他几种编程语言已经采用了 Lisp 编程语言，而 Python 是一种独立的编程语言。
*   Python 是一种更简单的语言，因为一切都是预定义的，而 Lisp 编程语言需要编写我们自己的语法。
*   与 Lisp 编程语言不同，Python 编程语言因其预定义的语法和特性而广泛用于研究和科学计算领域。
