## 玛雅和 ZBrush 的区别

Maya 是领先的高级软件，用于 3D 图形、动画、视觉效果、游戏应用程序等。Alias 系统公司最初将该软件移交给。Maya 用 MEL (Maya 嵌入式语言)、C++和 Python 写过。 [ZBrush 是一款](https://www.educba.com/what-is-zbrush/)高级雕刻工具或应用程序，用于结合 3D/2.5D 建模、纹理和绘画。与其他 3D 软件相比，这款软件最适合数字雕刻。使用 ZBrush 时，您可以转换展示的想象力。它基于像素技术。

#### Autodesk Maya 的应用

Maya 是一款在全球不同行业和不同应用中广泛使用的软件。玛雅用于电影工业、电视工业和电子游戏等行业。你可以在最近上映的好莱坞著名电影中研究玛雅效应，如《卑鄙的我 3》、《超级男人》、《钢铁侠》、《复仇者联盟》、《不可思议》、《侏罗纪世界》、《变形金刚》、《蜘蛛侠蚁人》等等。像《功夫熊猫》、《间谍小子》等电视剧。像怪物猎人，音速部队等电子游戏。

除了这些行业，Maya 还用于建模、渲染、动画、3d 可视化、视觉效果、运动图形等。

### 布鲁什

Pixologic Inc .最初开发 ZBrush。这个软件是 1994 年推出的。这个软件的演示版 ZBrush 1.55 于 2002 年推出；后来，3.1 版本于 2007 年发布。2009 年，他们计划推出支持 Windows 和 Mac 系统的 4.0 版本，但后来推迟了。

与 Pixel 类似的 pixel 技术，除了它之外，还包含关于深度、方向和材质信息的 Z 方向信息，而 pixel 仅包含 X 和 Y 位置和颜色信息。

然后 ZBrush 在同年 9 月推出了 3.5 版本，增加了一些新功能。最后，它的 4.0 版本以 GoZ (Go ZBrush) 4.0 的名字推出。此版本的 ZBrush 提供了与 Autodesk Maya、3dMax、Cinema4D、Lightwave 3D、Poser Pro、Daz studio、Elias、Modo 和 Blender 等流行软件的各种集成。ZBrush launch 拥有 2018 年 3 月的稳定版本。

### 结论

Autodesk Maya 是一个完整的软件包，提供建模、模拟、渲染、视觉效果、运动图形和动画。因为这个软件，只有电影行业发生了翻天覆地的变化。这是市场领先的软件，使用起来更方便快捷。它兼容多种支持平台。

虽然 ZBrush 在一些应用程序中已经足够好了，比如雕刻、纹理和建模，但是 ZBrush 中的数百万个多边形使它在一些文件中更加有效和具体；这就是为什么它被称为数字雕刻软件。
