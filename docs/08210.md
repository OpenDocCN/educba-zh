## HTML 中页脚标记的介绍

页脚表示与作者相关的数据、包含版权信息的数据或与该数据相关的一些链接。这个作者信息包含在

<address>元素中。这种类型的标签在 HTML 中称为

<footer>标签。HTML 中的页脚标签是在

<footer>标签中定义的。</footer>

</footer>

</address>

用户可以在

<article>、、

<aside>、

<section>、<details>、

<fieldset>内或者在

<figure>上使用页脚元素。它总是被放在标签中。它包含在各个块中。HTML 中的页脚标签，用于显示网页或任何部分的页脚部分。

<footer>不作为剖切元素，因此它不会在轮廓中创建新截面。</footer>

</figure>

</fieldset></details> </section>

</aside>

</article>

<small>网页开发、编程语言、软件测试&其他</small>

<页脚>标签的语法:

```
<body>
<footer>
<p>Some text</p>
</footer>
</body>
```

*   <footer>标签可以在标签中表示，用

    <footer>–</footer>

    标签包围，标签中有一些文本，我们将在页脚部分显示。</footer>

*   <footer>是 HTML5 网页或另一个 HTML 文档中最有用的[元素之一。它被视为单个块的一部分。](https://www.educba.com/html5-elements/)</footer>

*   HTML

    <footer>没有任何特殊的属性列表，但是它支持所有的全局属性，比如 accesskey、data、class、contenteditable、dir、draggable、hidden、id、lang、dropzone、spellcheck、style、title、translate 等等。以及类似 onafterprint、onbeforeprint、onbeforeunload、onerror、onload、onhashchange、onmessage、onoffline、onpopstate、onstorage、onresize 等事件属性。</footer>

### HTML 中页脚标记的例子

有多种方法可以将页脚包含到 HTML 文档中；让我们逐一看看其中的一些:

#### 示例#1

简单地在 web 文档的主体中包含

<footer>标签，就像。</footer>

**HTML 代码:**

```
<body>
<h2> Footer Example </h2>
<footer>
<p>Published by EDUCBA</p>
<p>Contact us: <a href=”mailto:info@educba.com”> <u> Email US </u> </a>
</footer>
</body>
```

#### 实施例 2

Footer 对象用于表示最具修复性的 HTML 元素，即

<footer>元素。可以通过 getElementById()访问它；.这个范例说明如何透过物件存取页尾，以及如何建立新的物件来存取页尾资料。</footer>

**HTML 代码:**

```
<body>
<h3>Footer Object Example </h3>
<footer id="footerDemo">
<p>all rights reserved @ EDUCBA </p>
</footer>
<p>Press button to check footer section.</p>
<button onclick=" footerFunction()">Click Me</button>
<p id="footerobject"></p>
<script>
function footerFunction() {
var f = document.getElementById("footerDemo").innerHTML;
document.getElementById(" footerobject ").innerHTML = f;
}
</script>
</body>
```
