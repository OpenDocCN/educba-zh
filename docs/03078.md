## 谷歌云和 AWS 的区别

Google Cloud vs AWS 是分别来自行业领先组织 Google 和 Amazon 的两家云服务提供商之间的比较。谷歌云是谷歌的新产品，而 AWS 是一个已经建立的云服务。被称为 GCP 的谷歌云提供的主要服务是计算引擎。而 AWS 的主要计算服务是弹性计算云，即云计算行业的领导者 EC2。GCP 因其大数据计算的专业服务而越来越受欢迎，如谷歌大查询和大表，而 AWS 则提供广泛的服务来支持企业的所有类型的 IT 需求。

### 什么是谷歌云？

2011 年，谷歌推出了谷歌云平台。这个云计算平台使一个公司得以发展和繁荣。它还使您能够利用和提供智能、安全和极其灵活的谷歌设施。谷歌云平台是由谷歌提供的 CLIC 基础设施服务的集合，用于谷歌最终用户产品的内部化，包括谷歌的谷歌搜索、YouTube 和谷歌照片，以提供计算、网络、存储、数据、更大的数据和管理。

### 什么是 AWS？

亚马逊的网络服务提供了灵活、可靠和可扩展的云计算替代方案，这些方案易于使用且具有成本效益。AWS 云计算平台提供了大量的云服务集合来创建一个全面的平台。它被视为一个发电站，为开发人员提供存储、数据库、分析、网络和部署/交付选项。谷歌云和 AWS 的市场价格在其名下的公共云中占有超过 30%的市场份额。就市场份额而言。谷歌云正在以达到 100%的速度取得巨大进展，但在市场份额方面仍落后于 AWS。根据美国消费者新闻与商业频道的研究，尽管谷歌在 AWS 上延迟了 55 亿美元，但其到 2018 年的收入达到了每季度 10 亿美元。由于谷歌首席执行官桑德尔·皮帅隐瞒了 2019 年的收入披露，尚不清楚 GCP 在 2019 年的市场份额是否会增加，但肯定不会远远落后于竞争对手。

### on Google Cloud 和 AWS 的主要区别

让我们讨论一下谷歌云和 AWS 之间的一些主要区别:

1.AWS 由亚马逊开发和管理，作为一个安全的云服务平台，而谷歌云平台是谷歌公共云的计算机和服务套件。当比较谷歌云和 AWS 的服务时，谷歌云平台提供的 IaaS(基础设施即服务)是谷歌计算引擎，而在 AWS 中，它是由亚马逊弹性计算云提供的。Google 云平台中的 PaaS(平台即服务)就是 Google App Engine。

2.在 AWS 中，PaaS 就是亚马逊弹性豆茎。Google 云平台中使用的容器是 Google Kubernetes 引擎和 Docker 的，而在 AWS 中是 Kubernetes 的。

3.Google 云平台和 AWS 中对象的存储限制是一样的，即 5TB/对象。谷歌云平台的内存卷大小是 1 GB 到 64 TB。另一方面，对于 AWS，内存容量为 1 GB 至 16 TB，而 4GB 至 16 TB 配置了 IOPS。

4.谷歌云平台用的 DNS 是亚马逊 Route 53，而 AWS 用的是云 DNS。此外，两个平台都使用 SSE 256 位 AES 算法进行加密。

5.AWS 使用 AWSLambda 进行大数据分析。谷歌云平台使用 AI-First 进行其服务器数据管理。VPC(虚拟私有云)用于谷歌云平台。AWS 有自己的 VPC，即亚马逊 VPC。
