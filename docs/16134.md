## 什么是 AutoCAD 命令？

AutoCAD 帮助我们以 2D 格式绘制设计概念。在 AutoCAD 中绘制任何对象都使用命令。命令是系统上的快捷键，允许软件绘制您的设计。在本主题中，我们将学习 AutoCAD 命令列表。

### 基本 AutoCAD 命令列表

这里关心的区域是告诉在这个软件中使用的 AutoCAD 命令的列表

<small>3D 动画、建模、仿真、游戏开发&其他</small>

*   **L 命令-** 这是本软件中最常用的命令。要使用此命令，用户必须在屏幕底部提供的任务栏上键入 L，然后按 enter 键。用户必须定义起点和终点。这将允许他画一条简单的线。也可以通过提供角度和半径在极坐标中定义该线。
*   要使用 C 命令，用户必须在任务栏中输入 C。它将允许用户通过提供一个点和半径来画一个圆。
*   **PL 命令-** 这将在你的绘图中创建多边形线。第一条线的端点将作为第二条线的起点，依此类推。
*   **REC 命令-** 该命令用于在 Auto CAD 中绘制一个矩形。要指定矩形的长度和宽度，用户必须指定一个起点和一个与第一个点对角的点。
*   **POL 命令-** 这将允许用户构建一个多边形。要使用该命令，用户必须指定边的数量、多边形的中心，并告知圆将在多边形内接或外接。
*   **弧线命令-** 这将允许用户创建一条弧线。软件将询问用户关于具有两个端点的弧的起点/中心。
*   **CO 命令-** 这是用来在对象上复制的。要使用此功能，用户必须选择一个对象，并将其粘贴到所需的位置。
*   **AL 命令-** 该命令允许用户将一个对象与另一个对象对齐。要使用它，必须指定对象的源点和目标点。源点是要对齐的对象的位置，目标点是所需的位置。用户还可以相对于目标对象缩放源对象。
*   **AR 命令-** 该命令允许我们创建一个对象数组。用户可以根据所需的配置文件创建阵列。
*   **CHA 命令-** 这将在两条不平行的线之间创建一个倒角。
*   **E command-** 这将从 Auto CAD 窗口中删除所选对象。
*   **EX 命令-** 这将扩展轮廓周围的对象。要使用此功能，必须选择要扩展的对象，然后按 enter 键后，选择要扩展的对象。
*   **TR 命令-** 这将修剪对象。要使用此功能，必须选择要修剪的对象，然后按 enter 键。之后，用户必须选择要修剪的对象。
*   **RO 命令-** 该命令将围绕指定点旋转对象。要使用它，必须选择要旋转的对象，然后选择一个旋转对象的基点。
*   **O 命令-** 该命令将对象偏移至所需的距离。要使用该选项，用户必须选择要偏移的对象，然后指定要偏移的距离。
*   J 命令- 该命令将允许对象连接在一起。
*   **M 命令-** 该命令允许移动对象。要使用它，必须选择要移动的对象，然后指定要移动的点。
*   **JPGOUT 命令-** 该命令将保存你的 Auto CAD 绘图。jpeg 格式。
*   **ME 命令-** 该命令将测量您关注对象的尺寸。
*   **颜色命令-** 该命令将打开一个对话框，用于选择所需的颜色。
*   DIV 命令- 这个命令会把你的物体分成不同的部分。例如，一条线要分成几部分
*   **F 命令-** 该命令将在两条相交线之间提供一个圆角/圆弧。要使用此功能，必须选择两条相交的线，然后按回车键。

*   **H 命令-** 该命令将为你的图形的某些特定区域提供一个阴影。使用该命令时，必须注意该区域应完全封闭，否则该命令将不起作用。
*   这个命令将把你的对象组合成一个单独的对象。
*   **I 命令-** 该命令将在你的图形中插入一个块。按下该命令后，用户必须浏览要插入到图形中的块的位置。
*   **LA 命令-** 该命令允许用户打开一个对话框，用户可在其中管理图纸内的图层。
*   **PE 命令-** 该命令将编辑折线。按下该命令后，软件会要求选择所有的折线进行闭合、连接、调整宽度、编辑顶点、拟合直线、将其变为样条曲线、de 曲线。
*   **SP 命令-** 该命令将检查图纸中所有单词的拼写。
*   **TB 命令-** 该命令允许在图形中插入一个具有所需行数和列数的表格。
*   **PO 命令-** 该命令允许用户根据用户要求在对象内插入点。
*   **PU 命令-** 该命令将打开一个对话框，允许用户删除图纸内未使用/不想要的元素。
*   **RE 命令-** 该命令将使用你在图形中所做的更新重新生成你的图形。
*   **REN command-** 这将允许用户重命名你的图层，如果你想改变它们的名字，在图中的块。
*   这将允许用户在你的绘图中插入多行文本。
*   **UN 命令-** 该命令允许选择图纸的测量单位。例如单位为英寸、毫米
*   **U command-** 该命令允许用户撤销之前的所有命令。
