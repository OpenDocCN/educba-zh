## 人工智能工具简介

毫无疑问，人工智能工具和[机器学习工具](https://www.educba.com/machine-learning-tools/)是最近积极占领市场的两个领域。人工智能自 20 世纪 80 年代就存在了，但直到最近几年才看到人工智能及其应用的巨大增长。我们可以说，人工智能是由机器展示的智能，更有可能尝试和创建对人类智能过程的模拟。

#### 1.机器学习

在机器学习中，定义了一个目标，并且机器必须学习达到该目标的步骤。让我们举一个例子，我们有一组猫和狮子的图片样本。该模型的目标是每当屏幕上出现猫的图片时，就说“是”。机器可以通过事先让它接触大量的猫的照片来学习这一点，这样它就可以训练自己，只要猫一出现在屏幕上，它就能识别它。

#### 2.人工智能工具中的机器人技术

机器学习的这个领域专注于机器人的建造和制造。正如我们所见，今天机器人以任何形式存在。我们取钱的自动取款机也是机器人的一种形式，而且还有许多智能工作机器人。亚马逊仓库有超过 10 万个机器人在仓库内完成运输工作。

#### 3.自然语言处理

操纵语音或声音和文本的过程被称为自然语言处理。我们可以从自然语言处理中得出许多重要的结论。例如，我们可以自动执行反馈分类的任务，如果一些用户对服务满意或不满意，我们可以实现一个 NLP，通过 NLP 分析他们的评论来得出结论。

#### 4.人工智能工具中的视觉

这个场赋予机器看的能力。例如，这种能力可以被赋予机器人或汽车，它们可以使用数字信号处理技术通过摄像机进行观察。

#### 5.自动驾驶和车辆

这一人工智能领域的重点是让驾驶和车辆实现自动驾驶。例如，优步已经开始制造无人驾驶汽车，并且在极少数城市运行。

#### 3.人工智能工具中的 Theano

#### 4.人工智能工具中的 Keras

Keras 是一个用 Python 语言编程的开源神经网络库。它有能力运行在其他库之上，如张量流、Theano 等。它是由谷歌的工程师 Francois Chollet 开发的。

Keras 的工作方式是——它不处理任何底层计算，而是使用张量流和 Theano 等其他库来完成。因此，Keras 处理高级 API，并编译具有损失和优化功能的模型。如果我们访问网站，[https://keras.io/](https://keras.io/)，我们可以看到许多教程和学习，任何人都可以获得并开始使用 Keras。

#### 5.sci kit-学习人工智能工具

Scikit learn 也是一个用 python 编程的开源机器学习库。它是由 David Cournapeau 开发的，是 2007 年 Google 代码之夏项目的一部分。Scikit learn 提供了许多监督和非监督的机器学习算法，可以在 python 程序内部使用。

这个库是基于 Scientific Python 的，在我们开始使用 sci-kit-learn 库之前必须安装它。sci-kit learn 提供的一些功能包括:

*   **NumPy:** 包含很多数学函数，可以支持大型[和多维数组](https://www.educba.com/multi-dimensional-array-in-javascript/)。
*   **SciPy:** 该库包含科学和技术计算模块，如线性代数、优化、信号和图像处理、集成等模块。
*   **Matplotlib:** 多用作可视化和绘图库。它可以用来为[可视化机器学习模型](https://www.educba.com/machine-learning-models/)创建大量图形。
*   **IPython:** 它是一个用于交互式计算的控制台，可以与多种编程语言一起使用。
*   **Pandas:** 这个库用于数据操作和分析。

#### 6.人工智能工具中的 Pytorch

PyTorch 是一个基于 Python 的科学包，它使用 GPU(图形处理单元)的能力。它提供了一个易于使用的 API，还提供了一个优秀的平台，提供了可以在运行时改变的动态计算图。
