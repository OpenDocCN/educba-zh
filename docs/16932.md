## SketchUp 取消组合简介

SketchUp 解组是一个过程，通过该过程，您可以在该软件中解组已分组的对象以及组件对象。当在不同的项目中工作时，许多类型在我们绘制的对象中进行不同类型的编辑和更改，我们需要取消分组的对象，这样我们应该知道取消分组的对象。当我们从在线库中导入任何模型时，它也会帮助我们，我们只需要导入模型的某一部分；此时，我们可以从导入的模型中取消不需要的对象的分组。通过遵循一些步骤，你可以非常容易地学会解组。所以让我告诉你吧。

### 如何在 SketchUp 中进行解组操作？

在这个软件中执行解组操作，并不是什么大事；你必须了解如何使用它，以及需要在哪里执行它。首先，我将通过一个例子告诉你所有关于解组的内容。

<small>3D 动画、建模、仿真、游戏开发&其他</small>

在我们开始之前，我认为向您解释一下这个软件的用户界面会很好，这样您就可以很容易地理解这篇文章。如果从顶部开始，则有一个菜单栏，其中有不同的菜单，如文件菜单、编辑菜单、查看菜单等，您可以在其中找到用于执行该软件中不同任务的不同选项。然后，移动到此栏下方的向下方向；有一个完全定制的工具面板，这意味着您可以根据自己的选择在该面板中添加和删除工具。在工具面板下面，有一个显示窗口，我们可以在其中看到我们的工作，在这个窗口的右侧，我们有默认的托盘，其中我们有几个任务合并不同的面板。

![SketchUp Ungroup Output 1](img/5fb5755ab1618def54b60df435cac6de.png)



为了向您解释这个软件的解组操作，我将从 3D Warehouse 中取出一个 3D 模型。3D Warehouse 是一个在线库，通过它您可以拥有不同类型的 3D 模型和其他一些可以在模型中使用的东西。对于 3D 仓库，该选项转到菜单栏的窗口菜单，并从下拉列表中选择 3D 仓库选项。

![SketchUp Ungroup Output 2](img/573e148abcd4ff506c9e6c079e5df62e.png)



我将从这个在线图书馆下载这套桌椅模型。

![SketchUp Ungroup Output 3](img/989d7fea8dcab96df7d837f32247eebb.png)



现在，我将通过单击显示窗口区域的任意位置，将其放置在工作屏幕的地面上。

![SketchUp Ungroup Output 4](img/c19e13bb546dc7817fcccd3f5aaf8cfd.png)



现在，您可以看到该模型的所有组件都被组合在一起，作为一个组件使用，在放置或选择用于不同目的的组件时不会遇到任何困难。

![SketchUp Ungroup Output 5](img/6c89c4d72be7914f7a2702c1b88480d6.png)



我可以双击来改变这个模型所有不同部件的尺寸和材料。

![SketchUp Ungroup Output 6](img/470b59727c13780d73524b18265d9dea.png)



但是如果我想把所有的椅子从桌子上分离出来，放在不同的地方，那么我就把这个模型解组。因此，要取消任何模型的分组，首先选择该模型并右键单击。选择对象后单击鼠标，将会打开一个下拉列表。这里我们有选项的数量，选择爆炸选项。

![Choose Explode option Output 7](img/1cfba06c9f8ca5fa4d1785448aacf950.png)



或者您也可以在选择该模型后进入编辑菜单并单击它。然后转到该下拉列表的组件选项，您可以从组件选项的新下拉列表中选择分解选项。

![Component option Output 8](img/28353d972a825535a04a688c516839ba.png)



一旦你点击“爆炸”，模型将分成不同的组成部分。例如，你可以看到椅子是不同的组件，桌子也是不同的。

![SketchUp Ungroup Output 9](img/3c966075dc58a640b1d2929b6858ce71.png)



现在，您可以根据自己的选择选择任何组件，例如本例中的桌子或椅子。

![SketchUp Ungroup Output 10.1](img/d3090d977dc54bfb55552e4697fbcf37.png)



![SketchUp Ungroup Output 10.2](img/3fcc92c60d53d55b4e5e4e457252ffef.png)



现在我将从工具面板中选择移动工具。您可以按键盘上的 M 键来使用移动工具，我将选择一个计划，我想沿着这个计划移动这个表，然后按住 shift 键，我将像这样移动它，因为现在它已从另一个组件中取消分组。

![SketchUp Ungroup Output 11](img/fc69b6e0118b72e199903be2594c2e9e.png)



您还可以将其放大或缩小，但是如果您对其分组模型进行放大或缩小，则在进入编辑组件模式之前，您无法单独缩放此表。

![Edit Component mode Output 12](img/0ad237a399547c59ace9025addbfa83c.png)



和桌子一样，你也可以移动椅子。

![SketchUp Ungroup Output 13](img/f542fb497aefd021f7068b0795a46f92.png)



现在，如果我想改变这把椅子，你可以看到它也是一个组件，它与另一把相同类型的椅子相连，因为那些椅子是这把椅子的副本。因此，如果您对这把椅子进行任何更改，那么所有其他复制的椅子也会受到影响。

![SketchUp Ungroup Output 14](img/725b7b87691afeb944b9cd134b50e83e.png)



现在，如果我只想在这张椅子上进行更改，我将选择它，然后右键单击并从下拉列表中选择“Make Unique”选项，之后，当我在这张椅子上进行任何更改时，其他人将不会受到该更改的影响。

![Make Unique option Output 15](img/7fdd4d041519b6f5d58893d5e4c3a3d1.png)



或者你可以再做一件事，爆炸这张椅子，就像我们对模型做的那样。选择这把椅子，以同样的方式，点击这个爆炸选项。

![Explode option Output 16](img/cec8d5b6c09aca674c042575c9d035ec.png)



现在您可以在这张未分组的椅子上进行更改，其他人不会受到这些更改的影响。因此，例如，如果我选择这把椅子的底座，并使用推/拉工具像这样挤出它的表面，那么你可以注意到其他复制的椅子是原来的样子，因为我们已经分解了这把椅子，并取消了它与其他椅子的链接。

![SketchUp Ungroup Output 17](img/52f34f3cc612a258a930fa79f773a336.png)



你可以看到，尽管我已经分解了这把椅子，但有些部分是独立的组件。因此，在一个模型中，总有许多组件是作为组件单独制造的。

![Output 18](img/cafb5230fbcce044ede194db850eec40.png)



如果您愿意，您可以像我们在前面的步骤中所做的那样来探索这个工具。对于任何对象或模型的取消分组部分，可以多次使用“分解”命令。

![SketchUp Ungroup Output 19](img/eda490f0f017c9c6fb506d1c8048bf7a.png)
