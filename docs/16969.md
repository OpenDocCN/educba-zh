## GIMP 过滤器简介

滤镜是带有一些可编辑数据的预设计效果，有助于 gimp 软件的图像处理和图像编辑工作，您可以在该软件中对图像或几何形状或文本使用滤镜。我们在 gimp 中有几个滤镜，不同的效果，不同的用途。我们有投影滤镜、长阴影滤镜、新打印滤镜等等，我们将在本文中讨论。您可以在菜单栏的“过滤器”菜单中找到过滤器。所以让我们分析一下 gimp 的一些最常用的过滤器。

### 如何在 gimp 中使用过滤器？

您可以在菜单栏的过滤器菜单中找到过滤器，该菜单位于用户屏幕的顶部。列表滤镜菜单里有很多滤镜，列表的每个滤镜都有几个滤镜效果。因此，首先，让我们为此创建一个新文档或新图像。进入菜单栏的文件菜单，点击下拉列表的新选项，或者按键盘上的 Ctrl + N 键作为快捷键。

![GIMP filters output 1](img/cd6cb7ca5f38d71d1c7242a4b43f2951.png)



现在，在“创建新图像”对话框中设置新图像的所需参数，并点击该对话框的“确定”按钮。

![GIMP filters output 2](img/66f006699d2a4efb3797fa2e31743627.png)



现在转到菜单栏的过滤器菜单并点击它。这是这个菜单的下拉列表中 gimp 的过滤器数量列表。

![GIMP filters output 3](img/f67f462b0942b3b17e4e2042c6f4090c.png)



现在转到 gimp 的工具面板，它在工作屏幕的左边，从这里拿走文本工具或者按下键盘的 T 键作为它的快捷键。

![GIMP filters output 4](img/82ad9fbd4ef526ece9fec7fbbfacd5cc.png)



并在文档区域输入您想要的文本。我会键入 EDUCBA，这是我们的正式名称。你可以选择你想要的字体风格。

![GIMP filters output 5](img/14acffc6176f564f0e16995637e83cc9.png)



现在点击文本工具的颜色框，在文档区域，改变文本的颜色，从“改变所选文本的颜色”框中选择白色，点击“确定”按钮。

![GIMP filters output 6](img/66e536cab9baf974ec9f0e83952b7470.png)



现在转到菜单栏的滤镜菜单，然后转到“光影”滤镜或向下滚动列表，然后选择“投影”滤镜效果的新的向下滚动列表。

![GIMP filters output 7](img/b202e8bdd88e18078431177de560f5f7.png)



你的文字后面会有这样的阴影。

![GIMP filters output 8](img/25efdec31df07165f56da0ed296e89a6.png)



您可以通过此投影参数框更改阴影。

![GIMP filters output 9](img/40e28772a685f0022a800acd2f62e420.png)



通过更改此对话框的“不透明度”选项的值，可以增加阴影的强度。

![GIMP filters output 10](img/16f6ceedca2463af5217eb173e97f7d3.png)



您也可以通过更改此对话框颜色框的颜色来更改阴影的颜色。只需点击颜色选项的颜色框，从“颜色”对话框中选择您想要的颜色，然后点击“确定”按钮。

![GIMP filters output 11](img/5630aeb62d5339944ed39b6a99b0844a.png)



现在让我们将一个过滤器应用于任何图像。所以为了在 gimp 中有一个图像，点击菜单栏的文件菜单，然后点击向下滚动列表的打开选项。

![GIMP filters output 12](img/5420e0ed5323d4fa8bafd519bff2f2eb.png)



然后从这个对话框中选择您想要的图像。一旦你选择了你想要的图像，然后点击这个盒子的打开按钮。

![GIMP filters output 13](img/6e3cc336b96e9504a7932fe8ee7301bd.png)



我会像这样删除这个图像的背景。如果你不知道如何在 gimp 中删除背景，你可以阅读我们的文章“删除背景”

![GIMP filters output 14](img/d6fad0b8fcd784675dd2564e484c35d3.png)



现在点击图层面板的创建新图层按钮来创建一个新图层。我有一个前景色填充层。根据你的图像对象，你可以为你的层选择任何颜色。现在点击这个框的 Ok 按钮。

![GIMP filters output 15](img/231017bb4b8a8aa88812e9b1a216ea9a.png)



在 gimp 的图层面板中，把你创建的图层放在这个家伙的图片下面。

![GIMP filters output 16](img/7e45b656f2f025f7abcc97ee19687510.png)



现在，再次进入滤镜菜单的光影滤镜，在光影滤镜下拉列表中选择长影滤镜。您也可以将投影滤镜应用到此图像。

![GIMP filters output 17](img/7c61ab375966823ad3881d7bfc5a2f36.png)



首先，在长影对话框的颜色框选项中改变长影的颜色。

![GIMP filters output 18](img/f18009d7592ce2d2abf38c770a23b143.png)



你会在这个人的背后看到这种类型的阴影效果。

![output 19](img/8bfe2531fd16965140126833c351545a.png)



改变阴影的角度，得到一个漂亮的阴影效果，就像这样。

![output 20](img/fc80fbb96bce9b3e8fd2681df3780097.png)



您可以从该对话框的合成选项列表中选择任何一个选项。如果你选择“仅阴影图像”选项，它将只给你图像的阴影。

![output 21](img/b0e9e7b0646cc085f3504ff78924b83c.png)



如果你选择'阴影减去图像'选项，那么它将删除主图像，阴影将保持这样。你可以用自己的想法玩这些东西。

![output 22](img/9fd51df86112c9f7cda95bf8d3483337.png)



现在让我们理解另一个过滤器，它是扭曲的。所以去扭曲过滤器，然后点击镜头扭曲的下拉列表中的这个过滤器。

![output 23](img/8a6b60e69b432db8a39defa8bb4cac3d.png)



你的图像会有一个镜头效果，你可以改变这个滤镜的参数来得到好的效果。

![output 24](img/69273774def4907cf8ea23b1a2ef7f61.png)



下一个过滤器是新闻纸，你也可以在扭曲过滤器列表中找到。

![output 25](img/ad64a41073a38a5f29496e2e1fdf4856.png)



一旦你把这个滤镜应用到你的图像上，它会给你的图像一个报纸印刷的效果。这意味着图像看起来像是由像素构成的。

![output 26](img/407d929e8c48e3b7881fd71dc7a2b13d.png)



默认情况下，此滤镜会给你一个黑白效果，但是你也可以使用 RGB、CMYK 模式，所以如果你从列表中选择 RGB 颜色模式。

![output 27](img/1177559f61acfcfbf29a178e1175d881.png)



那么你就会有这种效果。实际上，不同颜色模式的效果会根据您的图像而变化。在我的图像上，我用 RGB 模式实现了这个效果。

![output 28](img/776b57beec07153ec92f47dae0244b49.png)



如果我从列表中选择 CMYK 颜色模式。

![output 29](img/a2089c5f472ab980a8cbd8a751ce0bcd.png)



然后它会给出这种类型的效果。因此，根据您的图像或要求，您可以选择列表中的任何颜色模式。

![output 30](img/b6e3e99e369ca12d739e9740b466e64e.png)



gimp 中有更多的过滤器；你可以通过每一个过滤器来了解他们。
