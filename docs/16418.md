## InDesign 与 Publisher 的区别

InDesign 是 Adobe Inc .推出的桌面出版和页面布局设计软件，我们使用它来创建不同的布局，如传单、海报、杂志、报纸、演示文稿、书籍和电子书。与 Adobe 数字出版套件结合使用，它也适用于出版平板设备的内容。Microsoft Publisher 也是由 Microsoft 推出的桌面出版应用程序，它不同于 Microsoft Word，它强调页面布局设计，而不是文本的合成和校对。我们可以将这两种软件用于相同的目的，但是它们在某些方面有所不同。所以让我们找到这些方面。

### InDesign 和 Publisher 的主要区别

虽然这个软件做的是相同类型的工作，我们知道他们的开发者是不同的，所以有一些事实，使这两个软件在他们的领域特别。所以让我们来看看他们对这个软件的深入分析。

*   InDesign 支持 XML、编码标记和样式表，这使得它可以兼容导出标记文本内容，并且标记内容可以在数字和在线平台上工作，这使得它在版面设计人员中很受欢迎。的确如此。indd 文件扩展名，其互联网媒体类型为 application/x-InDesign。而 LibreOffice 支持专有文件格式(。pub)和 Corel Draw X4 用于只读支持。
*   InDesign 可以以 indd、indt 文件格式保存文件，本软件的文件打开格式为。indd，。indl，。indt，。indb，. inx. idml，.pmd，xqx。而 Publisher 支持多种其他文件格式，包括与多种 windows 平台兼容的增强型图元文件格式(EMF)。这个软件有一个非常好的功能，我们可以查看。Publisher 试用版中超过试用期限的 pub 文件格式。
*   现在，我们将讨论这两种软件的价格，这是一个重要的术语，有助于您了解如何将它们拿在手中。InDesign 的订阅和定价有一些变化，例如，如果我们使用个人包，则您必须每月支付 39.99 美元(这是折扣价)。如果你正在寻找商业套餐，那么你必须每月支付 59.99 美元。对学生来说有一些不同的价格，每月 19.99 美元。因此，根据您的选择，您可以选择您需要的许可证包类型。
