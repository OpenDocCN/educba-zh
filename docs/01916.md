## PHP OOP 面试问答介绍

PHP (超文本预处理器)是一种基于面向对象编程(OOP)语言的用于 web 开发的服务器端脚本语言。它主要是基于 C++编程语言设计的。它也被用作通用编程语言。它最初是由拉斯马斯·勒德尔夫在 1994 年开发的。这个 PHP 代码可以嵌入到 web 开发中的 HTML 代码中。PHP 是一种动态类型的编程语言，支持不同的操作系统，如 Windows、Unix 等。PHP 标准、规范和实现是由 PHP 团队提供的。PHP 代表个人主页。

现在，如果你正在寻找一份与 PHP OOP 相关的工作，那么你需要准备 2022 PHP OOP 面试问题。的确，每个面试都因职位的不同而不同，但是要通过面试，你需要对 PHP OOP 有一个清晰的了解。在这里，我们准备了重要的 PHP OOP 面试问题和答案，将帮助你在面试中取得成功。

### 第 1 部分 PHP OOP 面试问题(基础)

第一部分涵盖了基本的 PHP OOP 面试问题和答案

#### 1.PHP 中的会话是什么？

**答:**
PHP 中的[会话](https://www.educba.com/php-create-session/)用于存储网页浏览器上跨多个网页的信息或数据，而不是存储在本地计算机的内存中。PHP 中的会话是使用函数 session_start()启动的，会话变量可以使用名为$_SESSION 的全局变量来设置。在 PHP 中创建或启动会话的过程中，理想的情况是在会话开始时创建一个惟一的标识符，然后将一个 cookie 发送到用户的机器来标识惟一的会话，这个惟一的 id 存储在远程服务器中，在会话的生命周期中被调用来标识用户。一旦会话超时，整个用户会话数据将被删除。PHP 中默认的会话超时时间是 24 分钟。

#### 2.PHP 中的构造函数和析构函数是什么？

**回答:**
这是在面试中被问到的基本 PHP OOP 面试问题。PHP 中的构造函数用于创建其类类型的对象，该对象用于使用名为 _construct()的方法初始化对象的属性。PHP 中的构造函数可以通过在同一个类声明中用不同数量的参数调用同一个方法来传递不同的参数而被重载。PHP 中的析构函数用于在对象的生命周期将要结束时调用，以释放它所占用的内存或对其进行垃圾收集。可以使用 method _ destrcut()调用析构函数。

#### 3.PHP 的一般用途有哪些？

**答案:**
PHP 的一般用途是——它可以执行创建、读取、打开、关闭以及向计算机内存写入和从计算机内存写入等系统功能操作。它可以访问[cookie](https://www.educba.com/cookie-in-php/)变量并在浏览器中设置这些值。它还可以限制访问某些提供安全性的网页。数据也可以用 PHP 加密。

#### 4.PHP 中的接口是什么？

**答:**
接口是根据用户需求提供不同功能实现的契约。这也可以称为所需结构的蓝图。接口可以在实现类中以多种方式使用和实现。

#### 5.PHP 中有哪些不同的数据类型？

**答案:**
PHP 中不同的数据类型有字符串，浮点数如 double，Integer，Boolean，Array，Object，NULL，Resource。PHP 字符串是一个字符数组。整数是没有小数点的数字。布尔值可以是真或假。对象是一种可以显式声明的类型，通过封装可以在其中存储不同类型的数据。数组是存储在序列中的数据类型的相同列表，可以基于索引对其进行访问。空值也是一种数据类型，它表示没有值并且只有一个空值。

### 第 2 部分 PHP OOP 面试问题(高级)

现在让我们来看看高级 PHP OOP 面试问题。

#### 6.什么是 OOP，PHP 如何成为面向对象的语言？

**答:**
面向对象编程(OOP)语言会有抽象、封装、继承、多态等面向对象的特性。PHP 主要基于 C++编程语言，这是一种面向对象的编程语言。一个 PHP 类包含用来创建该类对象的属性和方法，每个对象都有自己的属性和值。

#### 7.PHP 中的名称空间是什么？

**答案:**
PHP 中的命名空间是用来封装类似于面向对象编程概念中抽象的项。命名空间用于建立类、函数和常数之间的关系。可以使用关键字定义名称空间。名称空间关键字在 PHP 中保留供内部使用，以避免与用户创建的标识符冲突。

让我们进入下一个 PHP OOP 面试问题和答案。

#### 8.PHP 中的 final 关键字是什么，什么时候使用？

**答:**
PHP 中的 final 关键字是用来将一个类或者一个函数标记为 final 的。如果一个类被标记为 final，它就不能被扩展以使用它的属性或方法。它防止其子类重写方法。如果只有一个函数被标记为 final，那么它就不能被扩展。PHP 5 版本中引入了 final 关键字。最终声明可以通过在类名或函数名前面加上 final 关键字来完成。

#### 9.PHP 中的 echo 和 print 是什么？

**回答:**
echo 和 print 是 PHP 中获取输出的两种基本方式。PHP 中获得输出的 echo 和 print 功能是相同的。echo 和 print 之间的区别是:echo 不返回任何值，而 print 返回可在表达式中使用的值 1。Echo 有多个参数，而 print 只有一个参数。Echo 可以通过调用 *echo* 或 *echo()* 来使用，而 print 可以通过使用*一个 print* 关键字*来调用。*

#### 10.PHP 中需要包括什么？

**答案:**
这是高级 PHP OOP 面试中在面试中被问到的问题。PHP 中的 *require* 和 *include* 语句在 PHP 4 及以上版本中均有提供，两者有所区别。Include 语句包含指定的文件并对其进行评估，而需求也具有相同的功能，但它会在失败时产生致命错误，错误级别为 E_COMPILE_ERROR，这是一个编译错误。在这种情况下，它会停止脚本的执行，并暂停运行时流程。将根据脚本中给出的文件路径来包含这些文件。
