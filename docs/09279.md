## JavaScript 职业介绍

1.  首先你应该明白的是，JavaScript 和 Java 是两种完全不同的技能，无论是在概念上还是在设计上。
2.  JavaScript 由 Brendan Eich 于 1995 年开发，并于 1997 年成为 ECMA 标准。
3.  ECMA-262 是该标准的正式名称。ECMAScript 是 JavaScript 的官方名称。
4.  JavaScript 开发人员负责实现描述 web 应用程序图形元素性能的前端逻辑。
5.  JavaScript 开发人员还负责将开发的元素与后端的服务联系起来。
6.  JavaScript 开发人员可以像全栈开发人员一样工作。
7.  JavaScript 开发人员通常由负责所有服务器端应用程序逻辑的后端 web 开发人员支持。
8.  JavaScript 开发人员也可以与其他专门研究应用程序标记和样式的前端 web 开发人员一起工作。

### 教育

要进入任何领域，至少应该满足一些由组织设定的基本资格要求，如:-

1.  对 JavaScript 职业、其怪癖和变通办法有很强的理解。
2.  关于 JavaScript，需要记住的一件很酷的事情是像 [jQue](https://www.educba.com/course/jquery-training-course/) r [y](https://www.educba.com/course/jquery-training-course/) 这样的库的存在(一个插件和扩展的集合，使得在你的网站上使用 JavaScript 更快更容易。)
3.  基本了解 web 标记，包括 [HTML5](https://www.educba.com/html-vs-html5/) 和 [CSS3](https://www.educba.com/css-vs-css3/)
4.  很好地理解(根据具体情况，编码人员应该具备高级 JavaScript 库和框架的知识，如 AngularJS、KnockoutJS、BackboneJS、ReactJS、DurandalJS 等。)
5.  对异步请求处理、部分页面更新和 AJAX 有很好的理解。
6.  精通跨浏览器兼容性问题以及解决此类问题的方法。
7.  熟悉 JavaScript 模块加载器，如 Require.js 和 AMD。
8.  熟悉前端构建工具，如 Grunt 和 Gulp.js。
9.  精通代码版本控制工具，如 Git / Mercurial / SVN。
10.  对浏览器渲染行为和性能有很好的理解

### 工作职位或应用领域

前端开发人员的职责包括

1.  开发新的[软件应用](https://www.educba.com/applications-software-developer/)从概念到完成，定期测试和维护。
2.  为即将到来的使用构建可重用的代码和库。
3.  确保用户界面/UX 设计的技术可行性。
4.  增强应用程序以获得最大的速度和可扩展性。
5.  确保用户的所有输入在提交给后端服务团队之前都经过了身份验证。
6.  与其他团队成员和投资者一起工作。
7.  根据需要评估和实现高级 JavaScript 库和框架。

### 结论或职业前景

JavaScript 开发人员的职业有很多机会，比如:-

1.  建立自己的作品集网站，在正式工作之余做一些自由职业项目。
2.  对高技能人才的需求总是很大。
3.  模糊前端和设计的界限，充分发挥你的想象力。
4.  预计增速高于平均水平(2014 年至 2024 年间为 27%)
5.  高于平均年薪(平均年薪约为 63，490 美元)
6.  专业经验有时可以代替教育。
7.  可能适用于所有类型的行业，如金融、保险或电信。
