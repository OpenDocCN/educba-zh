## 在 Illustrator 中创建表格的介绍？

在 adobe illustrator 中创建表格非常简单容易；在处理基于客户的项目时，提供与该特定项目相关的一些重要参数的信息变得很重要。illustrators 中的表格有助于您以有组织的方式管理和呈现特定项目的任何数据。在本文中，我们将分析在 adobe illustrator 中创建表格的不同方法，并介绍表格的重要参数。

### 创建表的方法

为了在 adobe illustrator 中创建表格，我们必须经历一些简单的步骤。让我们开始学习在 adobe illustrator 中创建表格的方法。这里我们将讨论在 illustrator 中创建表格的两种方法。

#### 方法 1–使用线条工具

**步骤 1:** 首先，从该软件菜单栏部分的“文件”菜单中选择一个“新文档”，该菜单位于工作屏幕的最顶端。

![Create Table in Illustrator - 1](img/9ab62251ab4a2267d249df5e931a4dc5.png)



**第二步:**新文件箱将打开。根据您的要求对新文档的参数进行设置，然后单击此框的“创建”选项卡以应用设置。

![Create Table in Illustrator - 2](img/6088f3edd4380ab23099925814b199a1.png)



第三步:我们将使用线条工具创建表格。为此，去这个软件的工具面板，从这里选择“线段工具”，或者你可以使用快捷键“\”。

![Create Table in Illustrator - 3](img/429ca0392c97c8a144d116694ab8e7b1.png)



**第四步:**现在，按照我画的画两条平行线。画线时按住键盘上的 Shift 键，这样直线就可以是一个画。

![Create Table in Illustrator - 4](img/a7c06db1cb44b1b3ef2832857579460c.png)



第五步:现在选择两条线。

![Create Table in Illustrator - 5](img/330401cdb2523d9a336b7632666add96.png)



**步骤 6:** 按住 shift 和 Alt 键拖动这些线条，以直线距离复制它们。

![Create Table in Illustrator - 6](img/ba1e9e32976bebced52a82d495163ce5.png)



**第七步:**现在按 Ctrl + D 键，将这些线条复制多份。

![Create Table in Illustrator - 7](img/21d0588ce7f060cd8385fb78fd599164.png)



步骤 8: 现在，再次使用线条工具，画一条垂直线，连接水平线的两端，以此制作一个表格。

![Create Table in Illustrator - 8](img/ca3c00e7cd5653839c98e6c7b4f0c3da.png)



**第九步:**现在，通过向另一个方向拖动来复制这条线。按住 shift 键进行正确对齐，按住 Alt 键复制该行。

![Create Table in Illustrator - 9](img/01caaf99135dba4fc41a15f1dddeef78.png)



**第十步:**现在按 Ctrl + D，将制作表格的行复制多份。

![Create Table in Illustrator - 10](img/21f60a62b460dba62f9b45b9c7e425f9.png)



**第十一步:**这样，你就可以利用表格适当的参数来制作表格了。现在从工具面板中选择文字工具。

![Create Table in Illustrator - 11](img/4b17894addb9dbaffd3cefbda073d51a.png)



第 12 步:现在，在键盘的帮助下，在这里输入你想要的文本。

![Create Table in Illustrator - 12](img/4b527619ac0b1c21d334f0a6c061b69e.png)



**第十三步:**复制这段文字。要制作副本，选择该文本，按住 shift & Alt 键拖动它

![Create Table in Illustrator - 13](img/e59ef2dd7c6de66ed9a466847bb9da20.png)



**步骤 14:** 现在按 Ctrl + D 键，将此文本复制多份。

![Create Table in Illustrator - 14](img/65cd3a820ba669ff927d1704f33ef03f.png)



**第十五步:**在表格的水平方向复制数据。

![Create Table in Illustrator - 15](img/1ecb836905c7337d147557bf9c001040.png)



这样，你就可以用这个软件的 Line 工具做一个表格，在文字工具的帮助下把你想要的数据填进去。

#### 方法 2–矩形网格工具

现在，让我们看看在 illustrator 中创建表格的下一个方法。到这个软件的工具面板，从这里选择“矩形网格工具”。

![Create Table in Illustrator - 16](img/a1fcef9addf11d696709413a5df35e78.png)



**步骤 2:** 现在点击文档区域的任意位置，给出表格的参数。单击时，将打开“矩形网格工具选项”对话框进行参数设置。在这里，您可以指定表格单元格的大小，通过给定水平分隔线选项的值来指定行数，通过给定垂直分隔线选项框的值来指定列数。

![Rectangle Grid Tool Options](img/1f1a9948452ae0420dd1bfa1e3ea9ec3.png)



**步骤 3:** 现在按下该对话框的确定按钮，应用设置。

![dialog box](img/9923d7347b96555b3df34dc68030126a.png)



第四步:你的桌子将会是这样的。

![Illustrator Table](img/837ea3d8e48d86bcb6fb62bae330e635.png)



**第五步:**你通过这个网格的笔画选项来增加或减少这个网格的笔画宽度。

![Create Table in Illustrator - 20](img/a45e592be4ac1870b5f41ef10e0c4692.png)



第六步:我将它的宽度设为 1 磅。你可以根据你的情况来拿。

![Create Table in Illustrator - 21](img/e4067e93c062c0d02b65431ea4a7e794.png)



步骤 7: 现在，从工具面板中选择文字工具，输入文字。

![Create Table in Illustrator - 22](img/fbdea5955bb06701e11df1fa2eb4fc17.png)



确保你输入的文本在顶层。要在顶层拍摄，请转到菜单栏的对象菜单。将打开一个下拉列表，转到“排列”选项，并从新的下拉列表中选择“置于顶层”选项。

![Arrange option](img/ebcacb7c39c1fc7e216603d926b37055.png)



**步骤 9:** 现在，将此文本放到表格的一个单元格中，并在此文本的序列中键入更多文本。

![sequence of this text](img/bf284818ad110f2ee9545511167cd5cb.png)



**步骤 10:** 现在选择所有文本，按 Ctrl + T 打开字符属性框。通过增加此特征属性框选项的值来增加文本之间的垂直间距。

![Create Table in Illustrator - 25](img/78aed1d2ec91a221cd50ab527cffe2d4.png)



**步骤 11:** 现在横向复制数据。

![horizontal direction](img/70b09f04e9d23f83abc1990ddbed2e2a.png)



**第 12 步:**按 Ctrl + D 键，用数据填充表格。

![fill the table with data](img/908d27faf14f2fa418200a6b62b21d96.png)



**步骤 13:** 在这里，你可以看到每一列中的数据都不在适当的距离上。那么我们应该做些什么来解决这个问题呢？

![Create Table in Illustrator - 28](img/cf51d641873c07187d9ca77eec856328.png)



**第十四步:**答案很简单。选择所有数据并打开对齐属性框。从这里选择“水平分布中心”选项。

![Align property box](img/85b862e8819fa0f79e29a8f07be8ada6.png)



**第 15 步:**你的数据在表格的每一列都会这样对齐。

![column of the table](img/8650d77b795b65ead6c2f5b08b3bf945.png)



**第 16 步:**现在选择任意数据，用你需要的数据替换。

![data and replace](img/b24f76e038fc8d7ba765a187565b1167.png)



**第十七步:**你可以在这个表格中填充颜色。对于填充颜色，请单击此表格颜色面板的前景色框。将打开一个对话框，从这里选择您想要的颜色。

![filling color](img/ec5d12b52a70c6d01c6a43da05e82e4e.png)



**第 18 步:**按下此框的 Ok 按钮。

![Create Table in Illustrator - 33](img/d0c767f1ac53516f06fd5ee849f88d0f.png)



**第十九步:**你还可以改变这个表格的笔画颜色。只需点击颜色面板的笔画颜色框，并从打开的颜色框中选择您想要的颜色。

![Stroke color box](img/184de6f98955fdcdaf8d6f7a24ab151b.png)



**步骤 20:** 按下此框的 Ok 按钮。

![Create Table in Illustrator - 35](img/50f0a9ffed74f14a87360cd253dce662.png)



**步骤 21:** 你可以单独改变任何特定单元格的颜色。为此，选择表格并从工具面板中选择“实时油漆桶工具”。

![Live Paint Bucket](img/40d56f5f9292d5b93338de7c6053589b.png)



步骤 22: 现在用这个工具点击你想要改变颜色单元格。
