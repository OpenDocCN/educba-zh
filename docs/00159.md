## Python 中插入排序的定义

Python 中的插入排序是一种高效的编程排序算法，用于使整个程序稳定而简单。这种排序算法有很多特性，使它完全灵活和通用。插入排序的整个概念在某种程度上与冒泡排序相关，但不完全相关。它使得排序的整个实现稳定、简单，并且适合数量较少的元素。尽管这种排序模式有很多优点，但它也有一些缺点，这些缺点与数据结构中的元素处理缓慢有关，因为其中存在嵌套循环。

### Python 中插入排序是如何工作的？

插入排序有一个工作的范例和流程，这是非常重要的，因为它有助于在实现时使整个排序技术变得简单和稳定。Python 中的插入排序的工作方式是将这一概念应用于就地和排序算法。它被认为是在初始化时管理数组中相等对象的顺序的稳定算法之一。第二，它也被认为是一种就地算法，它管理并需要额外的空间，而不考虑为它所占用的收集组的大小而给定的输入。

一旦允许执行排序，数组中存在的所有实际元素都会被排序，并重写内存分配和跟踪元素的实际位置。在插入排序中，如果数组中有更多的数字元素，那么它就有能力用所有的元素来调整自己，然后把它们按一定的顺序排列，使之可行并相应地工作。一次只能插入一个元素，并且一次不需要任何预先的知识。

将数组元素的大小保持在 10 或更小是有效且值得的。玩牌可以被认为是 python 中插入排序的最好例子，因为为了对牌进行排序，会对牌进行相互比较。卡片的分类使用插入分类，这使得整个分类过程高效。在应用插入排序后，所有卡片一旦按升序排列，就有助于提高处理效率。

所有的嵌套循环都出现在实现中，因此元素的排序不是很快。基本上，实际的工作流程以这样的方式开始，整个元素数组被分成两个子数组，比如一个未排序的数组和一个排序的数组。排序后的数组由包含初始元素的数组的第一个元素组成，数组的其余部分包含未排序的子部分，未排序的数组集中的元素与排序后的数组进行比较。一旦排序的所有安排都正确地发生，那么它将被正确地放置到一个正确的子数组中。

插入元素的重点是将元素向左侧移动，与右侧的元素进行比较。这意味着右边的值最终小于左边的值。这将重复发生，直到所有的元素都被放置在正确的位置，就像 inplace 概念满足需要一样。插入排序后的主要算法步骤如下:

*   将列表或数组分成两组排序和未排序的数组。
*   从 arr[1]到 arr[n]进行迭代，直到所有元素都按照各自的顺序排列。
*   将当前元素与数组中的下一个元素进行比较。
*   如果当前元素小于另一个元素，则在这种情况下，需要与该元素进行比较，并递增到下一个比比较时的实际元素大的元素。

一个例子将有助于理解更加简化，更容易得到实际的实现。Python 中有一个特别针对插入排序的规定，它允许自定义对象很容易地被排序，并且不会造成太多混乱。

可以创建一个自定义类，在这个类中，可以执行一些算法调整，以获得实际比较参数的细化，并保持与上面相同的代码。为了以不同的方式对对象进行排序，需要使操作符重载。操作符重载可以以这样一种方式执行，即有多种方法来处理使用另一个参数传递给插入排序函数的参数，从而使 lambda 函数集成并正常工作。Lambda 函数非常具有趋势性，它与 python 中的排序方法同时使用，用于调用排序方法。

def insertion_Sort_1(arr_0):
for o in range(1, len(arr_0)):
key = arr_0[o]
j = o-1
while o >=0 and key < arr_0[o] :
arr_0[j+1] = arr_0[j]
j -= 1
arr_0[j+1] = key
