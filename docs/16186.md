## CorelDRAW 版本简介

1989 年发布的 CorelDraw 的第一个版本“CorelDraw 1.0”给计算机图形学带来了一场革命，它是由 Arseng Antonio 和 Vicky De Guzman 开发的。几年后，版本 3 成为捆绑软件的又一个里程碑。从 1989 年到 2019 年，总共推出了 21 个版本。

### CorelDRAW 的版本

以下是 CorelDraw 的几个版本及其功能:

<small>3D 动画、建模、仿真、游戏开发&其他</small>

*   **版本 1.11 (1990):** 该版本支持导入/导出 AutoCAD DXF，可用于 2D 和 3D 设计。
*   第二版(1991 年):这个版本包含了打印合并功能，用于合并文本文件和图形文件并打印结果。其他功能包括包络，混合，挤压和透视工具，用于扭曲和混合对象和形状。
*   两年后(1992 年): Corel Corporation 推出了一个捆绑版本“CorelDRAW All-in-One graphics suite”，其中包括以下软件。
    *   Corel Photo-Paint ASP(位图编辑)
    *   CorelSHOW(创建屏幕演示)
    *   核心图表(图表)
    *   Mosaic 和 CorelTRACE(矢量化位图)
*   **版本 X3 (2006):** 这是第一个包含在名为“Corel PowerTRACE”的新跟踪引擎中的矢量软件，它可以将位图转换为矢量图形；这个版本的另一个重要特点是它有矢量对象裁剪；在以前的版本中，这仅适用于位图。
*   **版本 4 或 X4 (2008):** 更新了“实时文本格式”，在应用前显示文本属性的预览。其他附加功能，如交互式表格，支持 300 个相机的相机原始数据，独立的页面层。
*   **CorelDRAW 2010 版本或 X5:** 这是与 Corel CONNECT 一起发布的，Corel CONNECT 是一个全屏浏览器，允许您仔细阅读套件的数字内容并搜索您的计算机、本地网络或外部存储设备，以快速找到设计的完美补充，增强了具有透明度选项的网格工具。
*   **CorelDRAW 2012 版本或 X6:** 这一增强的&更新了高级 OpenType 支持，它包括四个新的创造性矢量整形工具，用于优化您的矢量对象，如下所列。
    *   **涂抹:**通过沿轮廓拉伸或缩进来塑造对象。
    *   **旋转:**对对象应用旋转效果。
    *   **吸引/排斥:**通过吸引节点或将节点推离附近的其他节点来塑造曲线。
*   **CorelDRAW 2014 版或 X7:** 这些突出的功能是快速定制、喷泉填充、字体游乐场。像“对齐和分布 Docker”这样的新功能，使您可以轻松访问所有可用的对齐选项，以便您可以精确地定位对象。另一个重要的功能是“二维码”与此；您可以通过添加文本、颜色和图像来创建独特、艺术和可扩展的 QR 码。
*   **CorelDRAW 2016 版本或 X8:** 此版本更新了字体列表框、增强的小刀工具、增强的照片编辑功能等功能。增加了对超高清 4k 显示器和实时手写笔(RTS)支持的支持；以下是包含的附加功能
    *   **选择相邻节点:**你现在可以在按住 Shift 的同时使用形状工具选择曲线上的相邻节点。
    *   **高斯模糊羽化投影:**你可以创建具有羽化边缘的真实投影。

*   **CorelDRAW 2017 版本:**此次更新了自定义曲线预览和编辑、LiveSketch 工具、自定义节点形状、突出的交互式滑块等功能
    *   **LiveSketch 工具:**该工具使用一种称为神经网络的人工智能技术，为用户提供了一种全新的方式来实现他们的矢量设计
    *   **触摸友好型用户界面:**平板电脑模式支持，可通过触摸或手写笔支持轻松绘制草图和进行调整

*   **CorelDRAW 2018 版本:**这次发布了一堆附加功能，如对称模式、块阴影工具、虚线中的角控制、轮廓、点画器、冲击工具、点画器效果和光尾效果。
    *   对称绘制模式:这可以让你创建一系列对称的设计，从简单的物体到令人惊叹的曼陀罗和复杂的万花筒效果。
    *   **块阴影工具:**它可以让你给物体和文本添加实心矢量阴影。
    *   **冲击工具:**它有两种模式，平行模式用于添加表示运动的缩放线，径向模式用于添加能量的爆发。
    *   **Pointillizer Effect:** 它可以使用 Pointillizer Effect 从任意数量的选定矢量或位图对象生成高质量的矢量马赛克。
    *   **照片拼贴效果:**这种效果让你更容易创作出令人惊叹的照片拼贴。
*   **CorelDRAW 最新版本:**这是 2019 年 3 月发布的，无损位图效果。
    *   非破坏性位图效果:它显示一个应用到一个选定对象的所有效果的列表，让你通过几次点击来添加新的效果和删除现有的效果。
    *   **像素完美的工作流程:**一系列新功能消除了创建清晰图标、仪表盘图像和网络图形的猜测。
    *   **箭头和尺寸线:**投影绘图增强为您提供了更好的设计组件视图和更理想的结果。现在，当您在投影的绘图平面上绘制时，常规线和尺寸线的箭头会随这些线一起投影。
    *   **新对象 Docker:**Corel DESIGNER、DRAW 和 PHOTO-PAINT 中包含的新功能“新对象 Docker”可让您控制文档及其组件的结构，从而更智能、更快速地工作
*   线条样式集:支持 S1000D 兼容的线条样式集，包括粗线、中线、细线、居中或无线条，现在可以从 object Docker 的样式中访问，或者可以使用键盘快捷键来应用。

### 新的快捷键和修饰键

*   **快捷方式(A):** 您可以使用刀具工具分割对象，并且可以在切割模式、两点线、手绘或贝塞尔曲线之间切换。
*   **修改快捷键(W):** 这允许你沿其推力轴移动椭圆或椭圆组内的椭圆。
*   **快捷方式(Ctrl+Shift+V):** 这使您可以将对象粘贴到绘图可视区域的中心，这样您就不需要缩小和平移来找到它们。
*   **新！查找和替换 Docker:** 新的查找和替换功能提供了一个简单、直观的界面，让您可以更快地定位绘图组件并更改其属性。

### 结论–CorelDRAW 版本

CorelDraw 允许您创建一个新的设计，而不是在现有的内容上工作。如果你想要一张放大或缩小时分辨率不变的图像，这是合适的工具。这个软件的主要用途是在印刷行业的标志，艺术，漫画艺术等。
