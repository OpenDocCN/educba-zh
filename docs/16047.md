## After Effect 字体主题介绍

after effects 中的字体是不同类型的样式，具有输入文本的附加效果。我们可以根据自己的想法在 after effect 中创建一个字体主题，只需使用该软件的一些插件和功能。您可以为您的项目制作一个有效的字体主题，并通过管理 after effect 的字体管理器的一些参数来制作动画。为了用我们自己的想法创建一个字体，我们必须在我们的电脑上安装一个字体管理器来管理字体的参数。为此，我们将使用 Saber 插件。这将为我们的字体主题增加更多的风格。所以为了了解这个话题，让我们开始讨论吧。

### 如何在 after effect 中创建和使用字体主题？

我们可以在 after effect 中创建一个字体主题，只需遵循一些简单的步骤，使用一些重要的功能和插件。现在让我们来学习一篇新的作文。对于新组合，请转到菜单栏的组合菜单并单击它。从下拉列表中选择“新组合”选项。

<small>3D 动画、建模、仿真、游戏开发&其他</small>

![After effect font output 1](img/c749cd9df4e321d72dc0ca31c667ffdc.png)



将打开一个新的组合框。进行所需的设置，然后点击此框中的“确定”按钮。

![After effect font output 2](img/630efc05404cc620bd311a691efe5a3a.png)



现在转到菜单栏的窗口菜单，点击它。从下拉列表中选择字体管理器选项。如果你没有字体管理器，那就从网上下载。

![After effect font output 3](img/2a484e5e3394cef613de5ba54f2b352b.png)



你会在屏幕上看到这种类型的字体管理器对话框。

![After effect font output 4](img/8503a95ea6c8b5229337c15e75918c9c.png)



将此框放在工作屏幕上您想要的位置。我会把它放在这里。

![After effect font output 5](img/6e8bbe2955e107be03f527efdd20e7b7.png)



现在从工具面板中选择文字工具。

![After effect font output 6](img/7c78231b2cb26ef951bfafbdfe914ebf.png)



从文本框中设置所需的字体样式，并键入一个字母。

![After effect font output 7](img/b3c99b4f7488af5d0bda1f5f9a2b853c.png)



通过使用该软件的参数部分的对齐选项卡，将其对齐在合成窗口的中心。

![After effect font output 8](img/40421c1efb0206e70e5a6dcfba2940e9.png)



现在转到字体管理器的选项卡，点击设置选项卡，然后点击该选项卡的快速启动选项。一个对话框会像这样打开。首先，选择“创建新字体”选项，然后点击下一步按钮。

![After effect font output 9](img/9acf03ea25ef1eb27bfda0df918a8e46.png)



在下一个框的“创建角色使用”选项中选择实体层选项，然后点击该框的下一个按钮。

![After effect font output 10](img/104899d92908fafd7a0974d8a0f0185e.png)



这将需要几秒钟来创建一个不同形式的文字层，如符号，数字，字母字体。所有的层都会像这样创建。

![After effect font output 11](img/e7d01158342bcbee1ae063e4d9edd085.png)



现在选择字母 A 的图层，点击这个图层的 solo 按钮，单独制作动画。

![After effect font output 12](img/ac27a75bc7715d8685003f331001c6ba.png)



现在转到参数部分的效果和预设标签，搜索 Saber 插件。现在拖动这个效果到字母 A 层。

![After effect font output 13](img/cb1e48a84400fad25fa09f54060729e7.png)



转到效果控制面板，点击自定义核心选项的下拉按钮。现在选择核心类型的图层蒙版选项。

![After Effect Font output 14](img/2f4bbcc3ea7715cf757d75cf8f30e83e.png)



你可以从 Saber 插件的预设选项中选择任何风格。

![After Effect Font output 15](img/ff6b64526b2c55567d8f1d350f23daf8.png)



我们将继续使用默认选项。

![After Effect Font output 16](img/35ce7c176db723483bc844bece1e7e7b.png)



设置发光强度为 25。你可以根据你的情况来拿。

![After Effect Font output 17](img/e0b052efa6dfd3d92923cec1870c3dda.png)



现在转到渲染设置选项，并在渲染设置选项的复合设置选项中选择透明。

![output 18](img/2a8fdd6c5cbff65385bbe8d5e72d28c3.png)



现在让我们制作这个字体主题的动画。要制作动画，将播放头放在关键帧部分的 1 秒处，并单击开始大小和开始偏移选项的秒表。

![output 19](img/b8bcec874f2072595280f1813698fae0.png)



现在将播放头设置为 0 秒，并将开始大小的值设置为 0%，开始偏移设置为 100%。

![output 20](img/1aad5f3cdaf5cd7e7e687d2d47753d91.png)



现在你可以播放动画了。

![output 21](img/0fbac98c2387ce6ee40d5510fc81e0d1.png)



选择层 A，并按下键盘的 U 键两次，以查看动画关键点。选择动画关键点，并按 F9 键使它们变得容易。

![output 22](img/a6e30811d983cc681c09584b48f101ba.png)



点击该层图形编辑器的图形编辑器图标。

![output 23](img/4022c0ba38a405bba1d6f8b1bb3f06af.png)



从列表中选择速度图。

![output 24](img/36dc97bfafafa06d6c74c9d777763244.png)



现在像这样编辑这个图表。

![output 25](img/ba1c361f37c4359eb087f38f550bc8cc.png)



现在转到效果控制面板，按 Ctrl + C 复制军刀效果。

![output 26](img/df78d0ea512d2596d152952a263d91ef.png)



现在按住 shift 键选择所有图层，然后按 Ctrl + V 按钮将这个效果粘贴到所有图层上。

![output 27](img/b95ace450ddb6bc06e25fb10f470a151.png)



选中所有层，进入这个对话框，点击这个对话框的下一步按钮。

![output 28](img/3e7ca6b796b9a775d2256cb1520baa9d.png)



将打开一个对话框，输入字体设置的名称。我会把它命名为我的字体。

![output 29](img/c75f2a48d4bd4e7e08877e30f8119119.png)



一旦你点击了这个 box 字样文件夹的 Ok 按钮，在这个软件的项目面板部分创建所有大小写的文本。

![output 30](img/b9cf4335f71143af871874fe0a0426fb.png)



现在，从该对话框中选择“全局父属性”选项，然后按“下一步”按钮。

![output 31](img/bc592d3d78efbc5f8f2f9b6a54f2138c.png)



从下一个框中选择“添加快速指标”选项，然后再次按“下一步”按钮。

![output 32](img/3e66859c70f94fa3d62d79e3a44f883e.png)



将你的图层动画粘贴到所有图层上需要几秒钟的时间，一旦完成，你会在图层部分找到这种类型的图层。

![output 33](img/1d9577a103aa9d579df120b42959d3aa.png)



现在进入项目面板部分，从你创建的字体文件夹中选择任意一个组合。

![output 34](img/d284b3cad3b5bc33564a82320edc5d58.png)



从该层中选择要为字样文件夹的所有层进行全局管理的属性。我会选择这 5 处房产。

![output 35](img/1d2c79e5aed9549916db65c0b1d9d579.png)



现在从这个对话框中选择“所有字体我的字体组合”选项，然后按下一步按钮。

![output 36](img/47cf888b3d3576ba3d1f91aa6a985552.png)



现在转到字体主题组合选项卡，这是您的初始文本组合。

![output 37](img/3a17897f5220212eceb58f8b68fc1311.png)



转到字体管理器选项卡，点击这个框的写选项卡，然后在这个框的字体中选择我的字体。

![output 38](img/210829b92a19f8253049b36ca0196fac.png)



在这里输入你想要的文本，你想用你创建的字体主题制作动画，点击这个框的写文本按钮。

![output 39](img/43677c0bab603d331f0e2aa74049af1f.png)



你的字体主题将会应用到你输入的文本中。

![output 40](img/b2934d561dea4667b97dcf89fe35d8ce.png)



您可以从这些选项中调整文本的大小、位置和其他参数。

![output 41](img/61124e4695ae951628316cdc41b2b7b6.png)



设置“偏移动画时间”的值，如 6 帧，以便文本的所有字母将一个接一个地被动画化。

![output 42](img/9e92b09eae679edb756ca5739ea5b1d0.png)



它会像这样活动。

![output 43](img/125130f238f42f0554eb4ea9c492fdc4.png)



您也可以从 Y 偏移选项设置字母在 Y 方向的距离。

![output 44](img/3fef73be904e9c4fb143b44cb7616b12.png)



您还可以从您选择的属性中更改字母的颜色。

![output 45](img/e01644389951fdb81baed686dcc57702.png)



这样，你可以在 After Effects 中创建一个很棒的字体主题。
