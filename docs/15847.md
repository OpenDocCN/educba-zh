## 网页设计与开发导论

好的网页设计和开发不仅仅是好看的颜色、流畅的布局和花哨的元素。这是一门艺术，在它的核心，它旨在提供一种体验，让访问者在第一次点击链接时实现他们的目标。对于一个网上服装店来说，这个目标是让访问者以他们喜欢的价格，以最简单快捷的方式购买他们选择的商品。对于一个健身博客来说，它可以教育访问者关于健身的知识。颜色和布局在网页设计和开发中确实起着重要的作用，但是当它们通过有效的过程被选择和执行时，它们的效果最好。

#### 1.设定你的目标

在你开始实际的网页设计和开发之前，你需要明确项目的目标。你的客户或组织希望通过这个[网站重新设计或新网站](https://www.educba.com/online-website-builder/ "10 Best Online Website Builder")项目实现什么？这个网站的主要目的是什么？是为了销售一些产品，产生品牌的知名度，为潜在客户提供一个接触点，还是其他？一个网站也不必局限于一个。它可以做以上所有的事情，甚至更多。这些目标必须从一开始就很明确。向你的客户或经理询问这些目标，或者问问你自己，这是否是一个你独自承担的项目。如果目标不明确，举行讨论使其清晰并突出显示。前进之前需要一个明确的方向。

一个网站可以有艺术元素，但它本身不是艺术品。它是一个门户和一个具有一组功能的界面。这种功能可以是娱乐、教育、销售、通知或提供对服务的访问。不管是什么服务，网站的设计应该着眼于满足这一点。如果你正在重新设计一个网站，目标尤其重要。你需要问清楚为什么要重新设计。是为了提高网站的效率，还是为了纠正某些错误，或者让网站更接近客户的目标？

网站的目标在网站的设计中起着很大的作用。例如，任何新闻出版物的网站都将是极简主义的，界面的唯一目的是传递新闻内容。相比之下，一个网页设计和开发工作室的目标是用它的设计和创意让访问者眼花缭乱。你可以期待大量的视觉效果，图像和强烈的色彩。

#### 2.了解你的目标受众

在决定网站的外观和功能时，你的受众是一个重要的考虑因素。毕竟，网站是为他们服务的，而不仅仅是为你或你的客户服务的。受众人口统计中的每个元素都会影响设计，包括年龄、性别、技术能力和职业。例如，一个面向房地产经纪人的网站将与面向年轻电子游戏玩家的网站截然不同。如果你的目标受众年龄较大，对技术不太了解，可用性会更重要。除了一般的审美，你关注的目标受众还会决定较小的细节，如字体大小和调色板。

当您定义目标受众时，您需要确定他们:

*   年龄范围
*   性别
*   婚姻状况
*   职业
*   收入水平
*   教育水平
*   价值观念
*   业余爱好
*   害怕

#### 3.将你的网页设计和开发集中在你的品牌形象上

网页设计和开发中的一个常见错误是仅仅实现最新的设计趋势，而没有一个有凝聚力的形象在中心。这些设计趋势之所以流行是因为它们有效，但这并不意味着它们适合所有场合。只需看看颜色这样简单的元素。不管这个季节的趋势如何，并不是每种颜色都很好。相反，更好的做法是考虑你的网站的目标，以及你试图从访问者那里获得的情感。你的网站设计应该以它试图传达的品牌的特征和个性为中心。每个网站都有一个品牌，即使没有正在销售的产品或服务。决定这是什么，并围绕这个品牌形象为你的访问者创造一个持久的，有凝聚力的影响。

#### 4.遵循基于目标的策略

既然你已经确定了网站的目标、目标受众和品牌形象，那么你就拥有了构建网站的所有要素。您的设计决策应该与您的策略同步。例如，如果你的网站的主要目标是为你的服务增加更多订户，你可以做三件事来改变现状:

*   在你的登陆页面上创建一个关于你的服务的简洁明了的片段，这样访问者就能立刻清楚你的网站功能。
*   使用对比和颜色来突出你的注册链接或按钮，让你的访问者容易发现。
*   通过删除所有可选和不必要的元素，使注册过程尽可能简单；人们很容易被长篇大论吓跑。

当然，有更多的方法来实现你的目标，不同网站的目标也不尽相同。然而，策略保持不变。你必须关注和塑造所有的设计元素，以满足你的目标，满足你的目标受众，并投射出一个清晰而恰当的品牌形象。你的网站的美学必须最适合你的目标受众的感受。

如果重点是娱乐，选择大量的图像和颜色，以获得生动的体验。如果网页设计和开发的重点是传递信息，那么网页设计要注重可读性和可用性，尽量减少颜色和图像，因为它们会分散对内容的注意力。如果重点是产品，使用大的图表和图像来展示产品的功能。如果是一项服务，你可以提供一个旅游，并使用描述性的例子来说明你的服务的好处。视频可以很好地让访问者了解你的服务或产品是如何工作的。

#### 5.向访问者提供明确的价值

现在你知道你的目标受众是谁，但是你怎么知道他们想要你提供的东西。你需要有一个清晰简洁的独特的销售主张或价值主张，基本上是对客户的承诺，他们将从你的品牌中获得不同于竞争对手的价值。确保从一开始就传达这一价值主张。让他们明白为什么你比竞争对手强。给他们一个留下的理由。当您考虑您的目标受众时，请回答以下问题:

*   他们有什么问题？
*   你如何解决这些问题？
*   你会为他们节省金钱和/或时间吗？
*   你如何改善他们的生活？
*   他们为什么选择你而不是竞争对手？

#### 6.鼓励分享内容

你有你的目标受众和你的价值主张，他们知道为什么选择你比你的竞争对手更好。但你也应该提供内容和其他元素，让他们愿意与他人分享。好的内容应该吸引观众，提供问题的解决方案，让他们想了解更多并与他人分享。这会引起人们对你的品牌和网站的兴趣，并带来更多有价值的访问者。毕竟口碑是最好的一种营销。

#### 7.用户体验

用户体验是当今的一个关键术语，这是有充分理由的。拥有一个好看的网站来销售好的产品和服务是很重要的，但是你也需要设计一个每次访问都让访问者感觉良好的网站。这可以促使他们更频繁地与你的品牌互动。有很多因素会影响良好的用户体验。首先，网页设计和开发必须是直观的。访问者应该能够轻松地浏览网站，尽可能在最少的点击中找到他们想要的东西。此外，应不惜一切代价避免混乱。从一开始，混乱就一直是 web 设计和开发中的一个主要挑战，直到今天依然如此。请记住使用空白，以便更容易阅读文本，并避免在一页上添加太多内容。

#### 8.质量和观众参与度

对于网页设计和开发来说，质量是非常重要的。当人们访问你的网站时，你最不希望他们认为你的网站看起来不专业，不完整。当你建立你的网站的时候，不要着急，关注每一个元素。不要忽略哪怕是很小的细节。网站必须在主题上保持一致，准确反映你的商业目标。

一个高质量的网站很有趣，但是一旦你的目标受众喜欢阅读、分享内容并与你的网站互动，你如何让他们参与进来呢？实现这一目标的最佳方式是定期添加新的、同样吸引人的内容。找出你的目标受众所面临的挑战，写出对他们有价值的内容。

你也可以通过电子邮件订阅来吸引你的观众。电子邮件是直接的，有针对性的，高度个人化的。收集电子邮件地址，并持续定期沟通。你可以为你的博客设置一个电子邮件活动。

#### 9.衡量结果

一旦网站已经部署，是时候衡量它的工作情况了。直到你测试了你的网站的性能，你才知道它是否有效地满足了你的目标。衡量标准应该根据你的网站目标来确定。如果目标之一是增加订阅量，那么注册人数应该是一个关键的绩效指标。RSS 订阅者的数量也应该是一个关键指标。如果鼓励用户参与是目标，论坛帖子或评论的数量应该是一个关键因素。用户对你的网站设计的反馈也是检验你的网站是否很好地服务于你的目标受众的好方法。然而，这很大程度上取决于你的目标受众是否喜欢参与这些话题。另外，记住不是所有的用户反馈都是有效的。有时候人们有不同的需求和品味，你不可能满足他们所有人。处理用户反馈的最好方法是识别模式和常见问题，然后处理这些问题。

衡量网站指标有很多方法，但重要的是它们应该与你的关键目标相联系。使用这些数据来看看你是否走在正确的方向上，并确定你可以在未来的设计中做出什么样的改变。

#### 10.不断做出改变

这带来了 web 设计和开发的一个关键特征:它永远不会结束。您发布的网站版本不是最终版本。事实上，你不应该认为有最终版本。你可以对它进行改进，网站的设计就是要随着时间不断变化。这不仅仅意味着改变以适应最新的设计趋势，还包括提高网站效率、加快导航速度或修正错误。继续努力逐步更新和改进网站，使其更加高效和有效。使用测量结果识别问题区域。例如，您的跳出率可能太高，一个关键网页可能没有得到足够的访问，或者您的访问者可能会发现很难导航到您的网站的某些点。
