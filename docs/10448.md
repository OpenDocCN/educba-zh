## Linux 职业介绍

如果你计划从事 Linux 职业，但不确定这是否是一个正确的决定，那么你就找对了工作。我们将帮助你解决与选择这项技术的 Linux 职业相关的疑虑和问题。对 Linux 专业人员的需求是永无止境的，尤其是如果你有 Linux 认证的话。现在，公司不再要求 TechOps 资源仅仅是在使用 Linux 技术方面有经验，但是他们更感兴趣的是雇用一个完整的栈开发人员，这个开发人员能够管理服务器、cron 作业、调度批处理、应用负载平衡，并且基本上能够完成使用 shell 脚本应该完成的所有事情。你不需要担心最后一行中使用的术语。当你开始在 Linux 上工作时，你会学到它们。

### Linux 中的职业道路

让我向您展示一下，如果您是这项技术的新手，并且在选择 Linux 职业时要向前看，您可以选择哪条道路。

1.  安装 Linux: 毫无疑问，这是第一步，因为你不会希望最终只获得这个主题的理论知识。
2.  参加 LFS101x: 这个课程将极大地帮助你创建你的 Linux 主干。您可以探索 Linux 管理员和终端用户经常使用的各种工具和技术。本课程将通过提供图形和命令行知识来帮助您。
3.  **Go for LFS201:** 在你完成 LFS101x 之后，建议选修这门课程，学习高级技术和复杂任务。本课程将为你提供详细而有用的知识，从而帮助你获得真正的实际工作经验。
4.  **练习:**没有什么东西能打败这个。你练习得越多，对你来说就越容易。
5.  获得证书:获得证书非常重要，因为这会让你在招聘人员面前引人注目，并帮助他们了解你所具备的知识水平。
6.  **参与其中:**如果你所在的地区有本地用户组(LUG ),考虑加入。这些类型的小组将帮助你与合适的人合作。

让我们也来看看一旦你选择了这项技术，你会有什么样的职业选择。

*   **系统管理员:**专业人员负责管理重要的服务，如 HTTP、FTP、Cron 作业、DNS、创建备份以及其他与服务器相关的任务。他还是组织中负责安全和防火墙相关设置的人。这项工作通常基于面向服务器的发行版，包括 Red Hat Enterprises、Debian、CentOS 和 Ubuntu。
*   **应用程序和 web 程序员:**与 Web 内容相关的出版业务的主要部分是在 Linux、PHP、Apache、MySQL 上运行的，因此能够用多种语言编写代码或能够设计不同数据库和应用程序的程序员比传统的程序员更受欢迎。
*   **技术员工作:**专业人员负责处理工作站和台式机。关键领域包括了解分区、安装、提供服务、排除网络故障、管理用户、基本脚本和操作系统相关问题。在这一领域获得实践机会也很容易，因为学校、大学和组织通常要求候选人具备这一技能。在这里获得经验将为您成为一名高效的 Linux 管理员开辟一条道路。
*   **支持:**您也可以成为那些寻求向全球目前从事 Linux 工作的专业人员提供支持的组织之一。随着 Linux 在企业和公司领域的广泛发展，公司选择 Linux 而不是授权的 windows。
*   教学和写作:这是另一个你可以找到工作的领域。以上有志之士需要导师学习本课程，你可以成为其中一员。你可以选择私下或在任何学校、学院或任何其他机构这样做。写作可能与为大公司提供文档以及与技术论文相关的文档有关。你也可以选择从事教学或写作等自由职业。
