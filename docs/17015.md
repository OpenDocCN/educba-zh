## GIMP 模式介绍

图案可以被理解为从以行或列或以特定顺序排列的任何特定设计中重复出现。图案可以是根据你的想法或任何图像的任何设计，如它可以由不同的自定义形状设计，或任何标志和许多不同的东西的形状。在本文中，我们将学习如何使用 gimp 软件的自定义形状来创建图案设计，以及如何在 gimp 的其他设计工作中使用我们保存的图案。所以让我们通过一个例子来理解这一点。

### 如何在 gimp 中创建和保存模式？

今天我们将通过 gimp 工具面板的椭圆选择工具创建一个图案来学习这个主题。首先，获取一个新文档，它将成为我们模式的参考背景。所以点击菜单栏的文件菜单的新建选项。

![GIMP patterns output 1](img/f6df772b36fccde5bbe4e8b36be91a07.png)



现在，根据您的图案大小设置文档的大小。例如，我将把 100 x 100 像素的尺寸作为我的文档尺寸。

![GIMP patterns output 2](img/6debe34c9277f7ca8751b8584673040f.png)



这是我的文件尺寸。

![GIMP patterns output 3](img/b0d79922747af43a1a3ee2886fc5fe78.png)



现在点击图层面板的“新建图层”按钮新建一个图层。

![GIMP patterns output 4](img/fb775baf139adbc7f240543343706d12.png)



通过单击此对话框的透明度选项创建一个透明层。

![GIMP patterns output 5](img/f9cfd01230e8b3b3b95f8494be5ed86a.png)



现在让我们有指南指出这个文件的中心。因此，请转到菜单栏的图像菜单，转到指南选项，并单击向下滚动列表的新指南(按百分比)选项。

![GIMP patterns output 6](img/9e4bddfc94135c306e2f3ca2d6fa1cd3.png)



现在将位置值设置为 50%,这样我们就可以得到一个指南作为这个文档的一半。这将是一条水平参考线，因为在此对话框的方向选项中选择了水平选项。

![GIMP patterns output 7](img/92351c73d87522ff9973685c8e2d5830.png)



对垂直基准线做同样的操作。

![GIMP patterns output 8](img/dae43490c114110555b0f912ad5534ac.png)



现在你会得到这种类型的指导方针。

![GIMP patterns output 9](img/e3d926478d01a6681e2fb1aa316b02b4.png)



现在从工具面板中选择椭圆选择工具，按住键盘上的 Shift + Ctrl 键画一个圆，然后将它移动到两个参考线的交叉点，这样我们就可以将这个圆放在文档的中心。

![GIMP patterns output 10](img/adee2b9f427dc388a56e7e6d3e01d04a.png)



现在从工具面板中选择桶填充工具。

![GIMP patterns output 11](img/37d07a7260e6b8edf3cd02410478c535.png)



并在工具面板的前景色框中选择您想要的颜色。比如我会选择这种青色。

![GIMP patterns output 12](img/4d1220844bc4523f16ec8877897a5b58.png)



用桶工具在圆圈内点击一下，填充你选择的颜色。

![GIMP patterns output 13](img/b7f0fa7010eb20c13d883748507f2ccf.png)



现在转到菜单栏的选择菜单，点击下拉列表的收缩选项。

![GIMP patterns output 14](img/220d2d96791a12db2c9b6f41e6679555.png)



我将缩小这个圆 8 个像素，这意味着缩小选项将使这个圆的副本比这个圆小 8 个像素。

![GIMP patterns output 15.1](img/00c9784e2f567a6c2bcc7de7c3ee3855.png)



![GIMP patterns output 15.2](img/fa52a9c2359f15a318863bb307277e11.png)



现在按删除按钮删除这个选中的小圆。

![GIMP patterns output 16](img/c8e767c1b980cb7460f40403aa626ecf.png)



再次缩小 5 个像素，用同样的方法填充颜色。我会把黄色填进去。

![output 17](img/833e5b55562fb06f1f3961e91668df1b.png)



现在转到菜单栏的视图菜单，点击显示网格选项，在我们的文档区域显示网格。

![output 18](img/50b19df5783a3a3db26eac0aabc62c38.png)



我们将在文档区域中使用这种类型的网格。

![output 19](img/f9e0b70d0992b6e62f68e7ed41fe9e95.png)



这是一个 1 x 1 像素的网格；你可以放大看。

![output 20](img/0cf14e7deefe8590c195efa867481081.png)



如果您没有这种类型的网格，您可以转到菜单栏的图像菜单，然后单击配置网格选项。

![output 21](img/30ef37a45d2f8160b07dd4ea9e7f6d04.png)



在这个对话框中为网格的水平线和垂直线设置 1 x 1 的值，然后点击 Ok 按钮。

![output 22](img/990cc6acbdee47a4fae14c08d5ed127e.png)



现在转到选择菜单来禁用这个黄色圆圈周围的选择，并点击下拉列表中的无选项。

![output 23](img/5235347645116da08d7d32fc1de4b08a.png)



再次创建一个透明的层，方法和我们之前创建的一样。

![output 24](img/55ef84408fdf587b283d15ba57e87f00.png)



选择这个透明层，并采取铅笔工具从工具面板。

![output 25](img/e6f6e6e1bf9d6eea352a11fcec1089cd.png)



从菜单栏的“视图”菜单中启用“对齐网格”选项，这样当我们在此文档上绘制任何内容时，它都会自动与网格对齐，这样我们就可以在工作中获得准确的结果。

![output 26](img/8e5c3de6a2b3637103121cbc73d6e729.png)



现在放大并用铅笔工具画任何东西。如果你在参数面板中选择 1 x 1 的笔刷大小，铅笔工具会一个接一个的填充这个网格。所以我会用铅笔工具填充这四个方块。

![output 27](img/c465d2c1c6232245de2f3923c8841f59.png)



我将改变颜色框的前景色，然后在铅笔工具的帮助下，在我们的图案中心制作这个形状。

![output 28](img/ee00050da818431818cfb3758a1dd47a.png)



一个图案应该是对称的，所以我在这个圆的反面也做了同样的事情，做出了这种类型的图案设计。

![output 29](img/a3623e7212adcf3e8d81f51ba864ed20.png)



我想要一个透明的图案，这样我就可以删除背景层。你可以根据自己的选择用你想要的颜色保存它。现在关闭网格，从视图菜单中禁用它们。

![output 30](img/7cb93a6d48631a33a1f503cbf1e9e9de.png)



现在通过点击编辑菜单的复制可见选项来复制它。

![output 31](img/c6bd28754195780fcb9688113f7585f9.png)



从窗口菜单的可停靠对话框选项中打开图案面板。

![output 32](img/beae54acc00210609f12c0fd33ef8544.png)



创建一个 1920 x 1080 大小的新文档，您将在图案面板中看到复制的图案，如下所示。

![output 33](img/14f6b4848e87bc948835e9e0039ee0ae.png)



采取桶工具，并启用它的参数面板中的模式填充选项。

![output 34](img/789094ff918048cc25e01ce7a7984cf3.png)



现在用桶工具点击文档区域，你会得到这样的图案。

![output 35](img/549bbd9a0e0d420761629927095b1098.png)



你可以放大看。

![output 36](img/ee0eafd3ff5fe5d26b19f38410c28d89.png)



如果您想要保存此模式以供将来使用，请转到“文件”菜单的“导出为”选项。

![output 37](img/5b7da5c2795d175b1e9e6726f5fcfdd2.png)



并将其保存为一个。pat '扩展到您想要的位置。

### 结论–GIMP 模式

这是通过使用工具面板中的不同工具在 gimp 中创建您想要的设计模式的最简单的方法。当然，你也可以用任何图像来代替这个设计，但是你只需要做一些参数上的改变，你就可以用这个图像得到一个漂亮的图案。
