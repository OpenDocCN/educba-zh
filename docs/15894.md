## 万维网的定义

万维网是 WWW 的简称。万维网的技术概念是:所有使用超文本传输协议的互联网工具和用户，万维网是对网络开放的知识 e 的连续体。WWW 还在互联网上提供计算机之间的知识共享，将它们连接成广泛的交互式多媒体服务。

蒂莫西·伯纳斯·李于 1989 年在日内瓦的欧洲粒子物理研究所建立了万维网。在他的建议下，万维网应运而生，这样研究人员就可以在欧洲粒子物理研究所进行有效的互动。这最终成为了万维网。小型网站将其所有网页存储在一个服务器上，而大型网站或组织将其网页放在不同国家的不同服务器上，以便当一个国家的用户搜索其网站时，他们可以轻松地从下一个服务器检索信息。

### 万维网的历史

英国科学家蒂姆·伯纳斯·李于 1989 年创建了万维网。他在欧洲核子研究中心任职期间。最初，它是为了满足全球科学家之间自动共享信息的需求而创建的，以便于共享实验和研究的信息和结果。CERN 是由 Tim Berners 工作过的 100 多个国家的 1700 多名科学家组成的网络。在他们本国的大学和国家实验室进行研究期间，这些科学家花一些时间在 CERN 网站上，以便他们有有效的交流资源来分享知识。在这个时候，互联网和超文本是可用的，但没有人考虑如何通过互联网连接或交换文件之一。

### 万维网是如何工作的？

我们现在知道，WWW 是一组与互联网相连的网站，用于搜索和共享信息。现在，让我们看看它是什么样的！

当用户请求时，服务器存储并传送网页或信息到用户的网络计算机。web 服务器是一种软件系统，它为 web 用户请求的网页提供指南。客户机是用户向服务器请求文档的机器。建立在用户计算机上的浏览器允许用户访问他们找到的文档。网络服务器保存着所有的网站。就像有人靠租房子住在家里一样，一个网站占用并存储在一台计算机上。如果用户请求他们的网页，网站所有者为同一个域支付托管费用，服务器就托管在该域上。当你打开你的浏览器，在地址栏输入一个网址或者搜索谷歌的时候，万维网就开始工作了。

### 万维网和互联网

许多人将术语“网络”和“万维网”结合在一起我们认为它们是一样的，但它们不是。互联网完全不同于万维网。这是一个由电脑、平板电脑、笔记本电脑等组成的全球网络。它帮助用户向其他用户发送电子邮件和在线聊天。例如，如果你在网上发电子邮件或与人交谈，你就使用互联网。然而，当你打开一个像 google.com 这样的网站了解详情时，你使用的是万维网，一个跨互联网的服务器网络。您通过浏览器从您的设备请求一个网站，该网站由服务器发送到您的浏览器。您的计算机被称为 web 浏览器客户端，它向另一台计算机(服务器)请求所需的信息。

**超文本传输协议:**

正是应用层协议允许 WWW 与超文本传输协议无缝、高效地运行。它是基于客户-服务器模型的。用户是与运行网站的网络服务器交互的网络浏览器。该协议规定了消息的编码和传输，以及 Web 服务器和用户在响应不同命令时要采取的动作。如果您在浏览器中插入一个 URL，HTTP 命令将被转发到 web 服务器，请求的链接将被传输。当我们使用浏览器打开一个网站时，我们打开一个到服务器的链接，并使用 HTTP 向网站提交订单。HTTP 通过 TCP / IP 传输到服务器。服务器处理客户端的应用程序并发送响应，然后关闭连接。然后，客户端检索用户的服务器信息。
