## Java DayOfWeek 简介

Java DayOfWeek 类由包含 DayOfWeekenums 和 API 的包组成，这些包支持不同的时区和各种与根据逻辑要求或代码实现获取特定日期或时间相关的规则。此类对于使用各种字段、值、单位、日期和时间日历来访问不同的日期和时间非常有用，这些日历可以为程序员提供作为实现逻辑的一部分所需的准确的时间相关问题，或者为检索系统状态提供有用的数据。
