## 飞镖和围棋的区别

Dart vs GO 是 Dart 和 GO 两种语言的对比。这两种语言的主要区别和相似之处将在这里注明。Dart 是由 Google 开发的开源编程语言，而 GO language 是由 Robert Griesemer、Rob Pike 和 Ken Thompson 在 2007 年开发的过程编程语言，但在 2009 年作为开源编程语言推出。Dart 既适用于服务器端，也适用于用户端。这里，我们将比较 Dart 和 Golang 这两种语言，并注意这两种语言之间的主要区别。

### Dart 与 GO 的主要区别

Dart 与 GO 之间的一些主要区别如下:

*   Dart 语言中的异步编程是由 Future 对象处理的，Future 是一个在未来某个时间点获取值的对象。当一个函数被未来调用时，它可能导致两种结果，
*   函数不能返回任何值，因此将工作排队并返回一个未完成的 Future 对象。
*   或者当未来值可用时，由函数调用的未来对象将返回值。
*   Golang 的设计方式似乎符合逻辑。语法上得到简化，使程序员和编译器开发人员更容易理解。
*   Dart 开发人员获得核心库:异步、字符串、集合、转换、正则表达式、文件 I/O、格式、类型化数据、数学等等
*   在 Golang 中，API 文档内容丰富，易于记忆。Golang 的文档存放在官方网页上，易于阅读和理解。
*   Dart 在开发中没有编译时。Dart 有一个集成的浏览器 Dartium，与 Dart VM 集成，允许运行和调试 Dart 代码。对于在其他浏览器上的测试，部署被转移到 JavaScript。
*   在 Golang 中，go 编译器立即编译二进制代码，类似于解释脚本语言。
*   Dart 语言有很多工具可用于开发应用程序。如分析器、发布器、开发编译器、测试器、服务器、dartfmt 和观测器。
*   Golang 提供内置测试。所有源代码文件必须与用于测试代码中函数的*_test.go 相关联。
*   Dart 已经开发出可以在几乎所有浏览器上运行的 JavaScript 代码。
*   Golang 非常适合构建网络服务，而 Dart 具有简单的原型
*   Golang 自动为已安装的软件包生成 API 文档，而 Dart 支持 semi-co 例程。
*   Dart 还可以编译为 Android 和 iOS 等移动平台上的本机代码
*   Golang 支持将源代码分割成多个文件，也支持包形式的模块。

### 飞镖与围棋对照表

Dart 与 GO 的比较如下:

| **镖** | **出发** |
| Dart 是一种开源编程语言 | GO 语言也称为 Golang，是一种并发和过程化编程语言 |
| 它是由谷歌开发的 | Golang 也是谷歌开发的，但在 2009 年作为开源软件发布 |
| Dart 拥有自己的 SDK 和编译器 Dart VM | Golang 类似于 C 语言，具有内存安全、结构类型、垃圾收集和使用包组装 |
| Dart 与 flutter 一起用于开发移动应用程序 | Golang 用于跨大规模网络服务器和分布式系统开发 |
| Dart 包含具有构造函数和 de 构造函数的类 | Golang 不包含既有构造函数又有 de 构造函数的类。 |
| Dart 具有编译器 Dart VM，可自动执行垃圾收集 | Golang 为内存分配提供了一种自动垃圾收集方法 |
| Dart 不包含算术指针，只包含指针 | Golang 包含指针，不包含算术指针 |
| 在 Dart 语言中，映射是通过值传递的 | 在 Golang 中，地图是通过引用传递的 |
| Dart 不支持函数重载和用户定义的运算符 | Golang 也不支持函数重载和用户定义的操作符 |
| Dart 也使用包 | Golang 不使用头文件。它使用 use packages 和 import 来导入外部包。 |
| Dart 支持列表中没有的默认值和常量，因为将来 Dart 可以支持非常数默认值 | Golang 不支持可变限定符或常量 |
| Dart 或 Flutter 是单线程的，不可能共享全局变量 | Golang 包含频道和 goroutines。 |
| Dart 有 do while 和 while 语句 | Golang 没有 while 和 do-while 语句。但是 for 循环可以修改为 while 循环 |
| Dart 支持类的继承 | Golang 不支持继承，而是提供了一种替代的嵌入方式 |
| Dart 是一个可扩展的内聚平台，用于构建在 web 服务器或云上运行的应用程序 | Golang 简洁、富于表现力、干净、高效。并发方法使 Golang 用户易于编写程序 |
| 使用 Dart 语言、工具和库，用户可以编写从简单脚本到全功能应用程序的任何代码。 | Golang 语言将代码编译到机器级别，并且仍然有垃圾收集和运行时反射的方法。 |
| Dart 是一种单线程编程语言。如果代码的任何部分阻塞了程序的执行，它将被冻结。 | Golang 拥有运行在 OS 线程之上的轻量级线程，为并发操作让路。 |
| Dart 使用异步操作，让程序运行时没有任何阻塞，这实际上是由 Future 对象处理的。 | Golang 使用通道在 goroutines 之间进行通信，使同步变得毫不费力。goroutine 线程运行的最大数量可以在编译时用 GOMAXPROCS 变量定义 |
| Dart 语言是根据 JavaScript 构建的，但是整个应用程序应该立即根据 JavaScript 构建 | 在 Golang 中，用户发现很难一遍又一遍地编写相同部分的代码。图书馆不提供域名服务。 |
| 在 dart 中，当实现回调时，将类型传递给任何传递给 JavaScript 的函数都会导致 dart2js 编译器崩溃 | Golang 基本上是为大型项目设计的，它有一个弱抽象特性 |
