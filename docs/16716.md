## ArchiCAD 快捷方式介绍

ArchiCAD 快捷方式是加快您在该软件中准确工作的最佳方式，这意味着 ArchiCAD 为不同的功能、选项、工具和另一个重要的命令提供了大量的快捷键，无需前往其位置或点击该命令的图标即可快速访问它们。你只需要记住这个软件不同命令的键盘按键，每个命令都有特定的按键来访问。在这篇文章中，我将告诉你菜单栏菜单的一些快捷命令，工具面板的工具，以及一些其他的 3D 建模命令。

我们将首先带您浏览文件菜单、文件模块、外部参照和文件 GDL 命令的快捷方式。所以让我们先讨论一下。

### 文件菜单选项的快捷命令

*   在开始任何项目或工作之前，我们做的第一件事就是创建一个新文件或文档，这个选项可以在“文件”菜单中找到，要获得快捷方式，可以按键盘上的 Ctrl + N 键。
*   通过按下键盘上的 Ctrl + Alt + N 键，您可以进入该菜单的新建和重置选项。
*   Ctrl + O 键是指用于打开本软件的任何作品、图像或任何支持的文件格式的快捷键。
*   你可以通过按键盘上的 Ctrl + W 键快速关闭这个软件的任何窗口或标签。
*   你要做的最重要的事情是保存你的文档，这个选项也可以在文件菜单中找到，但是你可以按 Ctrl + S 来保存你的工作。
*   Ctrl + Shift + S 是这个菜单的“另存为”选项，这意味着如果你已经保存了你的文档，并且想以另一种格式或另一个名称保存它，那么你可以使用这个选项和它的快捷方式。
*   通过按下键盘上的 Ctrl + P 按钮绘制工作图，对于其设置对话框，按下 Ctrl + Shift + P 按钮。你也可以使用 Ctrl + P 来打印任何作品。
*   Ctrl + Q 是用来退出这个应用程序，这意味着当你想关闭整个软件，然后你可以去。

**文件模块和外部参照命令的快捷方式:**

*   要将模块放入该软件，您可以按键盘上的 Ctrl +'键。
*   通过按 Ctrl + Shift +'可以进入热链接管理器。

**文件 GDL 对象的快捷命令:**

*   按 Ctrl + Shift + N 在工作中创建一个新对象。
*   Ctrl + Shift + O 将有助于打开任何对象。
*   Ctrl + Shift + N 可用于“项目另存为”选项。

这是一些文件菜单的快捷命令，现在让我告诉你一些编辑菜单的快捷方式。

**编辑菜单命令:**

*   我们需要撤销我们的步骤很多时候，所以这个快捷方式是非常有用的，在我们的工作，它是 Ctrl + Z 不仅如此，你可以重做任何你误撤销的步骤，并按 Ctrl + Shift + Z。
*   在我们的工作中可能有很多元素，很多时候我们需要选择它们，所以按 Ctrl + A 通过一个快捷键来选择它们。
*   Ctrl + B 允许您重复上一次使用的命令。
*   要拖动设计组件的任何元素，您可以按 Ctrl + D 键。
*   按 Ctrl + E 旋转对象或元素。
*   Ctrl + F 可用于拆分命令。
*   拉伸可以通过按键盘的 Ctrl + H 键来完成。
*   Ctrl + M 键适用于镜像命令。
*   通过按 Ctrl + K 键可以调整大小。
*   要使用调整命令，请按键盘上的 Ctrl+–按钮。
*   在设计过程中，按 Ctrl + 0 执行“修剪到屋顶”命令。
*   您可以根据自己的需要设置工具，在工具设置框中按 Ctrl + T。
*   对于“编辑选择集”，您可以使用 Ctrl + Shift + T。
*   使用 Ctrl + Shift + D，您可以拖动一个对象或元素的副本，与旋转它一样，您可以按 Ctrl + Shift + E，并使用键盘的 Ctrl + Shift + M 键来镜像对象的副本。

还有一个编辑文本的快捷键。所以让我们也来讨论一下。

**文本编辑命令:**

*   为了查找和替换您设计中的任何特定文本，我们使用查找和替换命令，其快捷方式是 Ctrl + H。您可以按 Ctrl + F 来简单地查找您设计中的任何文本。
*   要在设计的特定部分添加注释，请按 Ctrl + M，按 Ctrl + U 取消注释。

现在让我们讨论一些工具的快捷方式。

**工具快捷键:**

*   Ctrl + G 可用于对多个元素或对象进行分组，如果不需要分组，可以使用 Ctrl + Shift + G 取消分组。
*   Alt + G 将暂停一个组，Alt + Shift + G 可用于自动分组选项。
*   可以通过按 Ctrl +来创建补丁；键盘的按键。
*   Ctrl + =适用于“分解”命令。
*   按下键盘上的 Shift + F6 功能键来选择“前置”选项，按下 F6 键可以使用“前置”选项。F5 可用于“后移”选项，Shift + F5 用于“后移”选项。

### 结论

这个软件中有很多快捷方式，这使得我们的工作很容易，你不需要担心记住它们，只要开始使用这个软件，使用快捷键来执行任何命令，而不是使用传统的方式，随着时间的推移，这将成为你的习惯。
