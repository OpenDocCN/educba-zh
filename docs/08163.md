## HTML 如何工作的介绍？

HTML 是一种基于标签的语言，用于开发网页； [HTML 代表](https://www.educba.com/html-stands-for/)超文本标记语言。超文本是指网页链接在一起的方式。因此，网页上的链接叫做超文本。它是一种标记语言，标记告诉浏览器页面将如何在其上呈现。Berners-Lee 在 1991 年末开发了它，但“HTML2.0”是 1995 年发布的第一个标准规范。后来它的很多 HTML 版本就像 HTML 4.0 一样来了；目前，它的最新版本是 HTML5.0，这是非常著名的前端网站开发。

### HTML 页面如何工作的结构

让我们看看 HTML 页面是如何工作的。

<small>网页开发、编程语言、软件测试&其他</small>

```
<!DOCTYPE html>
<html>
<head>
<title>title tag of html</title>
</head>
<body>
<h1>heading tag of html</h1>
<p>paragraph tag of html<p>
</body>
</html>
```
