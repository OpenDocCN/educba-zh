## 2D 特效动画简介

2d 动画不过是 2 维；对于 2d 动画，需要有良好的画图技巧；另外，我们需要对角色的解剖有一个好的概念。我们需要绘制 2d 动画中的每一帧，Flash 软件主要用于 2d，但在 After effects 中，我们有更舒适的选项来创建 2D 动画，所以许多人开始使用 After effects 制作 2D 动画。有很多方法可以制作 2D 动画。我们将使用一个选项，即步行周期，我们将创建一个卡通行走。
