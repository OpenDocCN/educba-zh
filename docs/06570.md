## ASP.NET 版本介绍

ASP.NET 版本是 ASP.NET 框架的不同版本，从 2002 年开始到现在，该框架已经从其第一个 ASP.NET 版本有了很大的改进，随着时间的推移，它引入了不同的非常令人兴奋的功能，使 web 应用程序的设计和开发更容易和更快，并且通过不同的版本，它提高了网站的安全性，实现了不同的数据加密协议。

### 13 大 ASP.NET 版本

.NET 由各种版本组成，下面是 13 个最好的 ASP.NET 版本。

<small>网页开发、编程语言、软件测试&其他</small>

#### **1)净 1.0**

 **这是 ASP.NET 在 2002 年推出的第一个版本。第一个版本是和可视 Studio.net 一起推出的。它基于面向对象编程，支持继承、多态和其他、类型安全和 DLL 类库。这些类库有助于构建更健壮的 web 应用程序，比简单的 HTML 呈现拥有更多的功能。

#### **2)净 1.1**

 **第二版于 2003 年推出。这个版本主要是为了支持 ODBC 和数据库。移动应用程序开发人员使用该功能来实现更好的 web 渲染。这个版本引入的另一个特性是自动输入验证。ASP.NET 框架的主要特征之一是它提供的安全性。在 ASP.NET 的任何网站作为安全方面的认证，授权，保密性和完整性。使用认证是检查用户是否可以访问网页的过程，而授权是定义用户可以访问网页的哪些部分的过程。这种身份验证在 ASP.NET 可以通过四种方式和自定义身份验证来完成。

#### **3)净 2.0**

 **2005 年推出的第三版是对以前版本的重大改进，增加了许多新特性。在 ASP.NET 2005 中，可以有不同的数据控件，如网格视图、表单视图和详细视图以及导航控件。这个版本还引入了母版页、主题、皮肤、web 部件等等。它还支持 64 位处理器和完整的预编译。

#### **4)净 3.0**

 **ASP.NET 3.0 于 2006 年发布，具有非常重要的功能，如 Windows 演示基础、Windows 工作流基础、Windows 通信基础，使用这些功能可以托管 ASP.NET 网站和 Windows CardSpace，这使用 APS.NET 作为登录角色。

#### **5)净 3.5**

 **3.5 版本随 Visual Studio 2008 和 Windows Server 2008 一起发布。这个版本支持 ASP.NET AJAX 和 LINQ。它还包括更直观的数据控件，如 ListView、数据寻呼机等。这个版本被用来改善用户体验。

#### **6) NET 3.5 服务包 1**

 **Visual Studio 2008 Service Pack 1 包括了此版本。重要的特性包括支持在使用 ASP.NET AJAX 应用程序开发的网站中维护浏览器历史。此外，为了更容易下载，它还包含了将多个 JavaScript 文件合并成一个文件的特性。

#### **7)净 4.0**

 **它于 2010 年推出，包括在 aspx 页面的 Page 类中添加的元数据和元描述等功能，这有助于改善您的网站在互联网上的搜索，使其 SEO 友好。该版本还引入了并行扩展，以支持多核分布式系统中使用的并行计算。

#### **8)净 4.5**

 **Visual Studio 2012 版本附带了 ASP.NET 4.5 版本。它有一些非常令人兴奋的版本，通过创新的数据控制和易于使用的验证，使开发的网站更具可扩展性和更快。

#### **9)净 4.5.1**

 **在 2013 年与 Visual Studio 2013 一起推出，具有实体框架版本 6、SignalR、OWIN 和 Bootstrap 3.0 等功能。SignalR 是一个通信网络构建，用于从服务器到客户端的双向消息流，而 OWIN 是. NET 的开放 web 接口。以前，NET 构建在 IIS 之上，这给在其他 Web 服务器上运行带来了困难。OWIN 创建了一个标准接口。NET web 应用程序和其他 web 服务器。

#### **10)净 4.6**

 **添加了 Visual Studio 2015，它具有实体框架版本 7。该版本在 Windows 10 上运行时提供 HTTP/2 支持。它还为 API 提供了异步调用，使其成为线程安全的。HTTP/2 是万维网中使用的标准 HTTP 协议的一个主要变化版本。web apis 的变化也使得这个版本更加令人兴奋，并且一直被开发人员使用。文化信息。CurrentCulture 和 CultureInfo。Thread 类的 CurrentUICulture 属性也可以读写。

#### **11)净 4.7**

 **它于 2017 年推出，支持 TLS 协议，还包括 windows 10 更新。TLS(传输层安全)协议主要提供通信方之间的隐私和数据完整性。许多应用程序，如网页浏览、即时消息、IP 语音，都使用 TLS 协议服务。

#### **12)净 4.7.1**

 **这个版本是 Windows 10 秋季 Creators 更新附带的。这个版本的主要特点是支持 SHA -2(安全散列算法)。由于 4.7 版支持 TLS，4.8 版支持 SHA-2，web 应用程序现在变得更加安全，因为它们在数据加密中使用加密哈希函数。

#### **13)净 4.8**

 **ASP.NET 的 4.8 版本是 2019 年发布的框架的最新版本。这个版本包括了对 JIT 和 NGEN 编译器的改进。此版本提供了对 WCF 服务的增强，以及对 Web 表单的重大改进和对 WPF 的自动化改进。该版本支持 Zlib(一个数据压缩库)中的更新。ASP.NET 4.8 提供了文件压缩和解压缩的内置功能。
