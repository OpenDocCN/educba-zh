## 玛雅简介

Maya 是一个 3D 计算机图形应用程序，用于开发视频游戏、3D 应用程序、动画电影、电视剧和任何视觉效果。该软件可以创建重模型，这有助于我们提供超 3D 效果，从而在用户端创建逼真的视图。

全名是 Autodesk Maya，但简称为“Maya”。该软件之前由“Alias Systems Corporation”创建，但 Autodesk 以其名义获得了所有权利。它可以在 Windows、MAC OS 和 Linux 上运行。

<small>3D 动画、建模、仿真、游戏开发&其他</small>

### 了解 Maya

这是一个高度开发的软件，具有更高级的功能，解释如下。

*   它具有运动图形功能，如附加 MASH 节点、3D 类型、运动图形工具集。
*   它有 [3D 动画功能](https://www.educba.com/careers-in-3d-animation/)如并行钻机评估，测地线体素绑定，通用动画工具，时间编辑器等。
*   与之相关的 3D 建模功能有 UV 编辑器工作流外观、多边形建模、开放细分曲面支持。
*   它有一些动态和效果，如交互式头发梳理，深度自适应流体模拟，自适应空气动力学解决方案在 Bifrost 等。

### 玛雅是如何让工作变得如此简单的？

它提供了更快的性能，因为它有缓存回放，这使用户能够在视口中检查动画。同样，在许多领域，Maya 的响应速度更快，用户会注意到这一点，并帮助他们更快地工作。
