## Illustrator 中吸管工具介绍

Illustrator 中的吸管工具是最有用和最重要的工具之一。它帮助用户从程序中已经存在或创建的图像或对象中选择任何颜色。滴管工具将帮助用户使用在滴管工具的帮助下选择的相同参考颜色填充图像。吸管工具几乎存在于每一个图形设计程序中，它包含了很多用法和对用户的好处。有时用户不记得他们想要填充的颜色的确切代码。因此，他们使用这个吸管工具挑选和填充图像与他们选择的颜色完全一致。

### 如何使用 Illustrator 中的吸管工具？

让我们继续我们的文章，介绍这个软件的界面。

<small>3D 动画、建模、仿真、游戏开发&其他</small>

**第一步:**在这个软件的界面中，我们在文档区的左侧有一个工具面板，在文档区的顶部有一个菜单栏，在右侧可以从 Windows 菜单栏中使用最常用的工具。

![Eyedropper Tool in Illustrator 1-1](img/38ae8f50f1c397816453d8053e400e9c.png)



**第二步:**现在我们拿一个新文档来了解一下这个软件的吸管工具。对于新文档，请转到菜单栏中的文件菜单，然后单击它。下拉列表将被打开；单击下拉列表中的“新建”选项。

![Eyedropper Tool in Illustrator 1-2](img/ef26670d33aa34da4b05afa8907c61a8.png)



**步骤 3:** “新建文档”对话框将会打开。在此为您的文档设置参数，然后按此对话框的“确定”按钮应用设置。

![New Document](img/bb0a70500ba6db4676fa5e5658244fbb.png)



**第四步:**这是 Adobe Illustrator 软件的吸管工具。

![Eyedropper Tool in Illustrator 1-4](img/c01f7c51bb724927c8fa78adaf48a3a0.png)



**第五步:**让我们为自己的学习做一朵花。在钢笔工具的帮助下制作一片叶子就可以制作一朵花了。

![Using the Pen Tool](img/25252a2d68c7950f440fcef939386b84.png)



**第六步:**现在从这个软件的工具面板里拿出旋转工具，把叶子复制成 30 度角，然后按键盘上的 **Ctrl+D** 键画一朵花。

![leaf at a 30-degree angle](img/2b3fe05c70fa01b94d4793587f6e86dc.png)



**第七步:**现在，画一些圆，在这些圆里，从颜色面板里取不同的颜色。

![color panel](img/7b0def96483d0d6ebd9d7a8f663cdfd3.png)



步骤 8: 现在，从这个软件工具面板中选择吸管工具。

![Eyedropper Tool in Illustrator 1-8](img/bc336d503a2c8cb7a10cc67a0f81f90d.png)



**第九步:**现在按下键盘上的 Ctrl 键[激活选择工具](https://www.educba.com/selection-tool-in-illustrator/)并选择你想要给其中任何一个圆圈涂上相同颜色的叶子。

![Coloring Circles](img/457da5512f4872e75d03389626ca2269.png)



**步骤 10:** 现在松开键盘上的 Ctrl 键，吸管工具将再次被激活。现在点击你想涂上叶子颜色的圆圈的颜色。

![Eyedropper Tool in Illustrator 1-10](img/f92f2f7ecfbdf8ce935b842d29e6d709.png)



步骤 11: 你可以用吸管工具一次改变多片叶子的颜色。要更改多个叶子的颜色，请在选择过程中按住 shift 键，使用选择工具逐个选择叶子。

![selection process](img/95a1c9857bbc8eee3fadeb86678aa499.png)



**步骤 12:** 然后用吸管工具点击你想要给叶子的颜色，如下图所示。

![Eyedropper Tool in Illustrator 1-12](img/db0a91b5dbfe043a60c62a00437746dc.png)



**第十三步:**再次，我用同样的方法选择备用叶。

![Eyedropper Tool in Illustrator 1-13](img/0f96749f8212e78ad025beea9afa73cd.png)



步骤 14: 在吸管工具的帮助下给他们一个绿色圆圈的绿色。

![Eyedropper Tool in Illustrator 1-14](img/b8ac200110325384d6edd134a66e72a4.png)



**第十五步:**你可以随时改变树叶或任何物体的颜色。例如，我想改变绿叶的颜色为蓝色，所以我要选择它，然后在滴管工具的帮助下点击蓝色的圆圈。

![Eyedropper Tool in Illustrator 1-15](img/b7b9f8975394fed864a92b6a3fad2e6d.png)



步骤 16: 你也可以用吸管工具从任何图像中选取一种颜色。让我们为我们的 illustrator 软件拍摄一张图像，以便在 illustrator 中从您的个人电脑或笔记本电脑拍摄图像。转到菜单栏的文件菜单，点击它。将打开一个下拉列表；单击下拉列表中的“放置”选项。

![Eyedropper Tool in Illustrator 1-16](img/17cf86a53566c59ec75b3810a32855c0.png)



**步骤 17:** 再次打开一个对话框，选择图像。进入你想要的文件夹，点击你想看的图片。

![Eyedropper Tool in Illustrator 1-17](img/82ad31f53c8459ac50b29a9be6d29811.png)



**第 18 步:**要在 illustrator 中放置图像，请单击此对话框中的“放置”选项。

![Eyedropper Tool in Illustrator 1-18](img/d917c327a90db80f203c6b21d6758170.png)



步骤 19: 现在，我将使用选择工具选择蓝色的叶子。

![blue color leaf](img/dddad17b68fe4b6a24600eab50fb6a4d.png)



步骤 20: 现在从工具面板中选择吸管工具，点击图像中你想要替换为蓝色的颜色。我将点击图像颜色，如下图所示。

![Eyedropper Tool in Illustrator 1-20](img/80ef518700ede010622e6941c2128608.png)



步骤 21: 你可以使用吸管工具制作任何颜色的新样本。要制作新色板，请借助吸管工具从图像中选取一种颜色。您可以在工具面板的颜色框中看到您选择的颜色。

![New Swatches ](img/4833f3b6f7f027d136cb45c22f003d8e.png)



**步骤 22:** 现在打开样本框，点击这个框的新建样本按钮。

![Eyedropper Tool in Illustrator 1-22](img/3e9047464b4a467e96c0326c699fc802.png)



**第 23 步:**新样本对话框打开。您可以命名新色板，也可以借助 CMYK 颜色模式滑块对所选颜色进行一些更改，然后按此对话框中的“确定”按钮。

![CMYK color mode](img/36d12cdf9ad3f515a8cbbea90c73ef0c.png)



步骤 24: 新的样本将会在这个样本工具箱中生成，颜色与我们正在使用的吸管工具的颜色相同。你可以在下图中看到这一点。

![Eyedropper Tool in Illustrator 1-24](img/8a2203dec02eb2b9a6d0e9ada6882cdc.png)



步骤 25: 吸管工具的另一个用途是制作一个新的笔刷。做一把新刷子。借助吸管工具从图像中选取一种颜色。

![Eyedropper Tool in Illustrator 1-25](img/65655477c4efa5b52a7d7cc47e910606.png)



**步骤 26:** 现在在工作区的右侧工具面板中打开画笔工具框，点击画笔框的新建画笔按钮。

![Eyedropper Tool in Illustrator 1-26](img/0a0869adf9619acba300fcc09bc8c250.png)



**步骤 27:** 将会打开一个对话框，用于创建“新画笔”。它会问你，画笔的类型。从这里选择你想要的笔刷类型，然后在这个对话框中点击 OK 选项。

![Eyedropper Tool in Illustrator 1-27](img/fd934c1ca2a751bdaa34753912b0af7c.png)



**步骤 28:** 再次打开一个新的对话框，设置笔刷参数。您可以根据自己的需要更改笔刷的参数，然后按“确定”选项来应用设置。

![Eyedropper Tool in Illustrator 1-28](img/7d9ac6fb066f156316860989b4fb0028.png)



**第 29 步:**笔刷工具的盒子里会生成新的笔刷。

![Eyedropper Tool in Illustrator 1-29](img/bcd51d74890b569716e7b4834d2bb3c5.png)



**步骤 30:** 现在，从工具面板中取出画笔工具，借助你新设计的画笔，你可以画出任何东西。

![Eyedropper Tool in Illustrator 1-30](img/430816d7e5996d99770c1f4bd827e690.png)
