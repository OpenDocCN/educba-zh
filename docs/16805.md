## Inkscape 圆角矩形简介

以下文章提供了 Inkscape 圆角矩形的轮廓。在这篇文章中，我们将学习在基于矢量的软件 Inkscape 中创建、修改和渲染形状。具体来说，针对矩形和基本形状，我们将学习使用不同的方法来创建和修改形状，使用节点和过滤器。

Inkscape 是一个免费的矢量程序，它有多个动作过程来完成一项任务。我们将学习在节点和路径选择工具的帮助下创建圆形的所有方法。

### 如何在 Inkscape 中创建圆角矩形？

如上所述，用户可以通过三种方式编辑现有节点，并使它们的角变圆或圆角(如程序中所调用的)。就表演和应用效果而言，所有的技术都是相互独立的。

**方法一:**

这是在 Inkscape 中创建圆角矩形的最简单的方法。但是，使用这种方法，用户可以编辑整个矩形的节点，而不是选择要编辑的单个节点，然后修改圆角矩形。

尽管这可能是用户关心的一个问题，但有时它展示了创建圆角矩形的最快和最简单的技术。

以下是步骤:-

1.  启动 Inkscape 并创建一个新文件

![Inkscape rounded rectangle output 1](img/91e0322fc85011ff7d20de4727d5a3e3.png)



2.  创建一个矩形，如下图所示

![Inkscape rounded rectangle output 2](img/efeb254641375bdc2799c52ab41116a3.png)



3.  从工具箱中选择*编辑节点路径工具*，或者从键盘上按 N 键

![Inkscape rounded rectangle output 3](img/b8597d38fe76aff5ca8dbef056e8b13a.png)



4.  从选项面板中选择*制作所选线段曲线*，如下图所示

![Inkscape rounded rectangle output 4](img/06c1fef5c0182c48706f0e256321153f.png)



5.  在这种方法中，我们一次只能修改一个节点，并编辑对象的形状。
6.  向内拖动节点以使矩形变圆。

![Inkscape rounded rectangle output 5](img/31c59d445639bb87a0d172f0d9ab9d0c.png)



7.  我们可以在命令面板中看到 X 和 Y 轴的值，以及当我们向内拖动节点时它们是如何修改的。

![Inkscape rounded rectangle output 6](img/0525e3e4ac137c7046fc037dbaa07bb4.png)



8.  此外，我们可以再次向外拖动节点，像以前一样拉直矩形。

![Inkscape rounded rectangle output 7](img/8f11bdcce05b6f74e162593c86138ea9.png)



**方法二:**

在这种方法中，我们将根据需要使用路径过滤器来编辑形状。这个工具或过滤器的好处是，用户可以通过选择单个节点来编辑矩形。路径滤镜有许多效果，不仅可以应用于基本形状，还可以应用于绘图和复杂的艺术作品。

这是修改对象的最有效的方法，具有完美的准确性和良好的效果。

以下是用这种方法创建圆角矩形的步骤:–

1.  启动 Inkscape 并创建一个新文件

![Inkscape rounded rectangle output 8](img/3cbcfac1602156d7935b03c1a2858353.png)



2.  创建一个矩形，如下图所示。

![Inkscape rounded rectangle output 9](img/05f047b0860f2b0576fd07855c0bb38d.png)



3.  转到路径>路径效果(Ctrl+&)

![Inkscape rounded rectangle output 10](img/4335688e71027b0f511f12480bc26248.png)



4.  将打开一个不同的选项卡来添加/删除路径效果。

![Inkscape rounded rectangle output 11](img/e8d95268c06bd7ef86be4738276022ad.png)



5.  在该选项卡中，点击 **+** 符号，为对象添加路径效果，如下图所示。

![Inkscape rounded rectangle output 12](img/d17fb9a96c2f1c431b111cff423d8325.png)



6.  双击角(圆角/倒角)选项

![Inkscape rounded rectangle output 13](img/b6b03d18779fc2f122431c74af217de8.png)



7.  该效果将应用于如图所示的形状

![Inkscape rounded rectangle output 14](img/24b95f58530b54f3461833c323206c15.png)



8.  但是，现在还看不到。要创建效果，请在“半径”选项中选择一个特定值

![Inkscape rounded rectangle output 15](img/17255d9d1cb50a79ee7a347bcbf6a60f.png)



9.  之后，点击圆角选项

![Inkscape rounded rectangle output 16](img/ec65a73f365a719928086b52aed64062.png)



10.  您将看到形状周围有多个节点。

*   现在，点击工具箱中的路径选择工具

![Inkscape rounded rectangle output 17](img/157529be1e7e97f390058a077c09b1bc.png)



*   分别拖动节点，对矩形的各个角进行圆角处理。

![output 18](img/3a4fab093784835d2d1ac5b8594adae9.png)



*   我们可以从圆角、倒圆角、倒角和倒倒角中选择选项。

![output 19](img/66f0fc18e4a55bfd5fe66cf69ba48bc6.png)



14.  通过应用这个效果，我们还可以编辑半径、角测量和顶点。
15.  用户也可以轻松快速地删除效果，如下图所示。

![output 20](img/f3ee68f1396f134603d21335b32c4c40.png)



16.  同样，用户可以使用 Inkscape 中的路径过滤器工具创建不同的形状。

**方法三:**

这是一种在形状和对象上应用圆角效果的简单方法。我们所要做的就是将对象转换成路径。这种技术将在整个对象上工作，而不是在独立的节点上执行操作。以下是用这种方法创建圆角矩形的步骤

1.  启动 Inkscape 并创建一个新文件

![output 21](img/8647f4451aca2791432a6ee80f7d66c3.png)



2.  创建一个矩形，如下图所示

![output 22](img/3cfe7325e32a4910397d8aca41e87ed9.png)



3.  转到路径并选择对象到路径(Shift +Ctrl+ C)选项

![output 23](img/bbe23a55a758a99c3c02085e80cb9e8d.png)



4.  现在，点击动态偏移(CTRL+J)选项

![output 24](img/13c121cd106e163924a9599d7da7037a.png)



5.  该选项将创建一个节点，用户可以向外拖动该节点以在对象上创建圆形效果。如下图所示

![output 25](img/46d4e49600fc9ce8b3d604c797cfa91a.png)



6.  矩形是圆形的，没有任何复杂的动作。

![output 26](img/40a59dda019a3156d115b7e59600b3e8.png)
