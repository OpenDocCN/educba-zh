## Azure AZ-900 简介

AZ-900 考试被视为 Azure 基础认证，但对于所有其他 Azure 路径来说是可选的。它是为希望获得云服务和 Microsoft Azure 提供的服务结构的基础知识的候选人设计的。该认证的学习计划涵盖了基本的云概念、网络结构、安全策略、azure 管理工具和 Azure 的核心服务。

### Azure AZ-900 是什么？

*   如今，Azure 在云平台中有着快速增长的地位。Azure-900 涵盖了 Azure 云的基本概念，对于那些不熟悉 Azure 世界的人来说，这是一个真正的开始。即使是初级水平的考试，也起到了很难通过考试的作用。
*   该考试测试候选人对 Azure 管理和云概念的基础知识。来自非技术和技术领域的资源可以为该考试做准备，验证他们的云服务技能，并使您为 Azure 角色的下一级认证做好准备。
*   对于有 Azure 服务经验的人来说，这是证明他在 Azure 的核心概念和服务方面的技能的最佳机会，如计算、网络、应用服务、存储、物联网和人工智能。

### 我们如何准备 Azure AZ-900 考试？

*   微软 Azure 的基础是针对那些还没有开始微软 Azure 领域职业生涯的候选人的。
*   首先，从占考试 25%的云概念开始。它曾经发现使用云服务的考虑因素和好处。云计算的好处是可伸缩性、敏捷性、弹性、高可用性和灾难恢复。它用于确定运营支出和资本支出之间的差异，并描述基于消费的模型。它有助于区分不同类别的云服务，并描述了共享责任模型、Iaas、Saas、Paas、基于使用的服务类型以及无服务器计算。它还解释了各种类型的云计算，如混合云、公共云和私有云。
*   然后 Azure 的核心服务覆盖了考试中 20%的题目。azure 的架构组件是可用性区域、资源组、使用和订阅、区域和区域对、Azure 组的管理以及 Azure 资源管理器的好处。虚拟机、azure 容器实例、azure 应用服务、azure 虚拟桌面、虚拟网络、快速路由、虚拟网络对等、blob 存储、存储层、文件存储和磁盘存储的优势。Azure SQL 数据库、Cosmos DB、PostgreSQL 的 Azure 数据库和 MySQL 等数据库。
*   候选人应涵盖 Azure 的管理工具，如物联网中心、Azure sphere、物联网、Azure synapse、Azure 数据砖、HDInsights、认知服务、Azure bot 服务、逻辑应用程序、GitHub、GitHub 的操作、Devtest labs 和 Azure DevOps。其他 Azure 管理工具是 Azure PowerShell、Azure 移动应用程序、Azure CLI、Azure advisor、Azure 资源管理器的模板、Azure 服务健康和 Azure monitor。然后，他应该从一般和网络安全特性开始，这大约占教学大纲的 15%。Azure 的基本功能涵盖了合规策略、安全评分、安全警报和资源卫生。Azure 的重要安全工具有 key vault、Azure sentinel、Azure 专用主机、网络安全组的概念和应用、Azure DDoS 防护、Azure 防火墙。
*   然后，在介绍了 Azure 中的安全管理之后，他应该了解 Azure 的合规性功能、隐私、身份和治理政策，这几乎占了考试的 20%。Azure 的身份服务应该触手可及。候选人应该知道如何区分 Azure active directory 的授权和认证以及功能。单点登录、多因素身份验证和有条件访问的功能应该得到很好的修改。azure 中的治理功能，如资源锁、标签应用、azure 策略、azure 框架、云采用、Azure 蓝图和基于角色的访问控制，应该融入到他的技能中。Azure 中的合规性资源和政策功能，如数据保护修正案、在线服务条款和微软的隐私声明，应该进行修订。候选人应该了解 Azure 中国云服务、Azure 主权区域、信任中心和 Azure 合规性文档的目的。
*   Azure 的服务水平协议和成本管理应该对进出流量、资源类型、位置、服务、混合模型、预留容量、预留实例、总成本所有权计算器进行有效规划。正确定义的服务生命周期是评估的强制部分，因为它直接依赖于客户部分。Azure 中服务水平协议的目的和影响不同可用性区域、一般可用性和公共预览中的 SLA 的操作应该进行修改，因为它们得分最高。

### 学习项目

所以候选人应该在 Azure 的重要服务和相关主题方面熟练。

<small>Hadoop、数据科学、统计学&其他</small>

*   综上所述，考生应该了解云计算。
*   精通重要的 azure 组件，如存储、计算和网络。
*   至少进行三次动手实验体验和练习。
*   通过创建虚拟专用网、虚拟实例、对等、负载平衡器集成和网关来练习。
*   接受创建应用程序服务的培训，并了解最新的定价和可用服务计划。
*   改进存储特性，如 cosmos DB、blobs 和 queue。
*   看看定价模型的概念和安全性。

### AZ-900 认证到期

*   AZ-900 认证没有规定有效期。
*   但另一个相关的 Azure 认证有一个过期日期，它需要更新，每十八个月定期更新一次。

### 结论

考生可以在参加主考之前参加许多模拟考试，这样有助于他计算每个问题的时间分配，并更快地理解问题。因此，在参加考试之前要做好准备并充满信心，这对顺利通过考试有着重要的作用。
