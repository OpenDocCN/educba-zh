## 设计师和建筑师的区别

许多人不知道建筑师和建筑设计师之间的区别。他们的工作都是设计建筑、住宅、工厂和其他结构。然而，如果你仔细观察他们提供的技能和服务，你会发现他们之间有许多不同之处。根据从业人员的技能和资格，可以明确区分这两种职业。建筑设计师的主要职责是提出建筑结构的设计和草图。

建筑设计师将会见客户，并讨论他们的要求和建议，以纳入建筑的设计。在此基础上，设计师提出了建筑的草图和概念。接下来，建筑师考虑这些概念，并确定在项目的可行性、功能性和实用性方面是否符合建筑规范。现在让我们看看下面的比较表，以确定设计师和建筑师之间的基本区别。

### 设计师和建筑师的主要区别

以下是要点列表，描述了设计师和建筑师之间的主要区别:

#### 谁是建筑师？

为了成为一名建筑师，你必须从一个被认可的机构毕业。大部分后来攻读硕士学位，完成建筑师注册考试。建筑师不仅要学习设计建筑背后的艺术过程，还要接受创建建筑规范的培训。他们与设计师、承包商和建筑检查员合作，以确保他们的结构符合所有的法规和规范。他们还可以处理建筑从最初的外观到最后的过程。

为了成为一名职业建筑师，你必须:

*   获得建筑学的有效学位
*   应包括在所需的责任保险中
*   必须注册为建筑师，并符合当地建筑机构的要求

即使一个人有几个建筑学学位和多年的专业经验，如果他们没有正式注册为建筑师，他们作为建筑师进行交易也是非法的。因此，注册政府建筑组织是什么概述了一个合格的和合法的要求的建筑师。从学术上来说，建筑师对建筑规划的工艺和科学都有很深的了解。

建筑师在大学里要经历大约五年的时间，包括技术设计和不同的东西，比如历史和合同法。毕业生通常需要在注册建筑师手下完成两年的实际工作，才有资格成为建筑师。根据客户提供的概要，建筑师和建筑设计师可以将他们的想法应用到项目中。有时，他们甚至可以充当合同管理员，负责整个施工过程。

**建筑师的工作职责包括:**

会见客户，与设计和施工团队成员一起讨论设计策略。

*   给出成本、工作和建造时间的基本评估。
*   间歇访问工地，检查工作和会议计划。
*   利用手绘艺术或计算机支持的绘图软件准备按比例绘制的插图。
*   了解最新的分区法律和指导方针、建筑趋势和设计元素。

#### 谁是建筑设计师？

建筑设计师与建筑师合作，应该以富有想象力的方式处理建筑设计。他们应该理解客户传达给他们的内容，并将这些想法转化为创新的结构草案。对他们来说，掌握建筑法规和建筑行业标准的应用知识也是至关重要的。建筑设计师还应该在数学、物理科学和计算机辅助制图方面有广泛的实践。设计师必须准备好调整设计，以便根据客户的需求准备审阅草稿。

早些时候，设计师被称为绘图员或绘图员，后来成为设计师，以反映他们不断发展的工作和能力范围。如今，设计师的工作已经延伸到了功能性、节能性和令人印象深刻的住宅设计中。建筑设计师可以从各种与设计相关的领域脱颖而出。这包括拥有建筑学学位和实践的个人，他们不是注册的建筑师，也包括在设计领域受过正规教育的人。

**建筑设计师的工作职责包括:**

*   咨询客户以确定建筑用途、规划和大小
*   对建筑的外形和内部设计提出建议，并设计出不超出预算的计划。
*   展示初步和最终设计
*   帮助客户雇佣承包商和建筑公司。

**建筑设计师所需资格**

在许多地方，任何人都可以做建筑设计师，根本不需要任何资格或授权的先决条件。这些地方也不需要注册或许可证。虽然任何人都可以在这样的地方做建筑设计师，但重要的是要记住，他们的工作最终需要符合建筑和规划指南，否则就不会建成。

### 设计师和建筑师对照表

下面是设计师和建筑师之间最顶层的比较:

| **比较的基础** | **建筑师** | **设计师** |
| **教育** | 建筑学学位 | 大专学历优先 |
| **责任** | 与设计师和施工团队合作，使用软件准备设计 | 建筑内部和外部的设计建议 |
| **费用** | 7-10% of the build costs | 3-5% of the build costs |
| **所需经验** | 最少两年 | 不需要 |
| **工资** | 与设计师相比更高 | 与建筑师相比相对较少 |

### 结论

从定义上来说，建筑技能可能涵盖几乎任何程度，并且不属于基本认证的范畴。一个令人震惊的结果是，除了通过他们的工作历史、经验和态度之外，没有简单的方法来识别一个体面的架构师:认证仅仅表明个人有一些概念，但不是真正的实践。

无论是建筑师还是设计师，理想情况下，你都应该根据经验、资历、成就和热情做出选择——并且明确寻找过去的工作符合你要求的人。同样重要或者更重要的是找到一个你确信可以与之共存的合适人选——尤其是如果这个人可能会帮助你管理你的建设项目。你将和他们一起工作很长时间！
