## Oracle Apps 面试问答简介

Oracle app 主要由基于应用或业务的软件组成。Oracle 提供了许多应用软件业务领域，如人力资源管理、供应链管理、客户关系管理以及其他领域。

### 第 1 部分–Oracle Apps 面试问题(基础)

第一部分包括基本的面试问题和答案

#### Q1。解释 Oracle apps 中的 ERP？

**答案:**
在 Oracle 应用中。ERP 代表企业资源规划。在这个企业指的是组织或公司，资源指的是可利用的资源和管理日常资源的计划手段。Oracle applications 为大型组织和公司开发了这种软件。

#### Q2。解释 Flex 字段及其类型？

**答:**
它可以定义为子字段的集合，可以通过添加或删除来更新 flex 字段的结构。这是在面试中被问到的基本 Oracle Apps 面试问题。它不需要任何编程水平。它只是出现一个弹出窗口，可以引起每个片段。以下是弹性域的类型:

*   **键弹性域:**主要用于创建唯一标识符。它主要存储在段内。它正在构建一个定义对象结构的模块。对于键弹性域，有两种不同的属性，即弹性域限定词和段限定词。这些字段足够灵活，无需编程即可使用编码方案。它用于存储与钥匙相关的信息，并有助于显示与钥匙相关的信息。关键字段有助于指定应用于段值的规则，这些段值组合在一起形成有效的代码。
*   **描述性伸缩字段:**主要用于捕捉附加信息，并提供形成的空间，能够使其扩展。它还用于创建新字段。它提供了定制空间的功能。它可以是上下文相关的，其中应用程序的信息取决于在表单中输入的用户。

#### Q3。解释模板和模板中使用的附件？

**答:**
模板是在创建任何其他类型的表单之前需要的表单。它主要具有独立于平台的附件，并且与特定的库相关联。模板表单中有几个附件:

*   APPSCORE: 它是一个附件，由包和程序组成，主要用于创建工具栏、菜单等不同的表单。
*   **APPSDAYPK:** 它是由软件包组成的附件，有助于控制与 Oracle 相关的应用程序。
*   **FNDSQF:** 这是一个附件，它具有用于 flex 字段、配置文件、消息字典和并发处理的各种过程和包。
*   **自定义:**该附件有助于在不修改应用代码的情况下扩展 Oracle 的应用表单。

#### Q4。解释在 PL/SQL 中使用游标的优势？

**答案:**
使用游标的主要目的是处理与 PL/SQL 相关的各种行查询。有两种类型的游标，即隐式游标和显式游标。Oracle 中已经提供了一个隐式游标，可用于处理所有与 Oracle 相关的查询。Oracle 使用的未命名隐式游标会占用内存空间。显式光标是由用户定义的光标。

#### Q5。解释 Oracle 中的验证类型？

**答案:**
验证的类型如下:

*   **无:**主要表示最小验证。
*   **相关:**输入值根据之前的值与值的子集进行比较。
*   **独立:**在这种类型的验证中，输入值应该在已经定义的值列表中。
*   **特殊:**在此验证中，值使用弹性字段。
*   **Pair:** 在这种情况下，值对使用 flex 字段。
*   **表:**已根据应用表中已有的可用值检查了输入值。
*   **可翻译相关:**在此输入和子集的值与先前的值列表进行比较。
*   **可独立翻译:**在这种情况下，该值与之前定义的列表值进行比较。

### 第 2 部分–Oracle Apps 面试问题(高级)

现在让我们来看一下高级 Oracle Apps 面试问题。

#### Q6。有哪些不同类型的触发器可用？

**答:**
可用的各种类型的触发器有按键触发器、消息触发器、错误触发器、基于查询的触发器、导航触发器和事务触发器。

#### Q7。你说的自主交易是什么意思？

**答案:**
本交易主要独立于另一交易。它允许你暂停交易。它有助于执行 SQL 操作、回滚操作以及提交那些事务。这些自治事务不支持资源、锁或任何类型的提交依赖，而提交依赖是事务的主要部分。

让我们进入下一个 Oracle Apps 面试问题。

#### Q8。解释将报表附加到 Oracle Apps 的不同步骤？

**答:**
它们是在将报表附加到 Oracle apps 时需要注意的某些步骤或规则，Oracle apps 设计报表，生成与报表相关的可执行文件，将可执行文件和源文件移动到产品的适当文件夹，以并发可执行文件的形式注册报表，为可执行的已注册程序定义并发程序，最后添加用于请求责任组的并发程序。

#### Q9。解释程序中强制的参数？

**答案:**
这是在面试中被问得最多的 Oracle Apps 面试问题。在声明过程时需要注意一些参数，这些参数被分配了某种作业。例如，参数是 Errobuf，这是用于在 oracle apps 中返回错误消息并将其发送到日志文件进行跟踪的参数。另一个参数是 Retcode，它主要用于显示与 0，1，2 这样的过程相关的状态代码。这些是该参数显示的不同状态。0 定义正常状态，1 主要定义完成警告状态，2 定义程序中发生的错误。

#### Q10。定义值集？

**答案:**
是包含数值的集合。这些值集主要与发送给用户的报告参数和值列表一致，以便以参数值的形式接受不同值中的一个。
