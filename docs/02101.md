## Azure 与谷歌云的区别

Azure cloud 是由微软提供支持的云平台，提供了许多云服务，包括存储、网络、分析和计算服务。用户可以将这些云平台服务用于他们现有的应用程序，或者可以使用这些服务开发新的应用程序。它使用 IaaS 和 PaaS 网络环境。Google cloud 是另一个由 Google 支持的云平台，它为应用程序提供了使用云服务的基础设施。谷歌云提供的服务是机器学习、分析工具、存储和数据分析。谷歌云使用的网络环境是平台即服务(PaaS)、无服务器计算和基础设施即服务(IaaS)。

### Azure 与谷歌云的主要区别

让我们讨论一些主要的关键差异:

1.  Azure 云平台和 Google 云平台的关键区别在于两个云平台的用例。Azure 云平台更侧重于企业服务，为各种企业提供服务，而 Google 云平台不侧重于为企业服务，而是侧重于为云服务的本地用户提供服务。Azure 云平台相比谷歌云平台提供了更多的服务。Azure 云平台提供的服务包括物联网集成、大数据、区块链技术、游戏开发、可扩展数据类型仓库。谷歌云平台提供的服务有应用开发、数据管理服务、数据存储服务、工作负载管理和生产力工具。
2.  Azure 云平台和 Google 云平台的另一个重要区别是数据库、存储和备份服务。Azure 云平台使用的数据库有 Redis 缓存、SQL 数据库、Cosmos DB、数据工厂、服务器拉伸数据库、数据仓库。Google 云平台使用的数据库是 cloud SQL、cloud datastore、cloud spanner、Cloud Bigtable。Azure 云平台提供的备份服务是站点恢复、归档存储和恢复备份，Google 云平台备份服务是 nearline 和 Goldline。Azure 云平台提供的存储服务是磁盘存储、blob 存储、队列存储和文件存储。谷歌云平台提供的存储服务是传输服务、云存储和持久磁盘。
3.  这两个云平台最重要的区别之一是服务的定价。这种差异决定了哪个云平台更好。Azure 云平台提供各种具有复杂结构的服务，这增加了服务的成本，使其不太适合。另一方面，与微软 Azure 云相比，谷歌云平台以非常低的成本提供服务。谷歌云平台为其服务提供各种折扣。Azure 云平台的定价结构要复杂得多，需要适当的文档，还需要丰富的经验，以便用户在使用服务之前能够正确理解。与另一个云平台相比，谷歌云平台是最新的，因此它提供各种折扣来建立市场。

### Azure 与 Google Cloud 的对比表

让我们来看看排名靠前的比较:

| **比较因子** | **蔚蓝色** | **谷歌云** |
| **建立** | Azure 云服务是微软在 2010 年推出的。Azure 云平台提供了几种用于商业目的的云服务。 | 谷歌云平台是谷歌为提供众多云服务而推出的。它使用谷歌用于 YouTube、谷歌搜索等产品的相同基础设施。 |
| **可用区** | Azure 云服务遍布全球约 140 个国家，覆盖全球 54 个地区。 | 谷歌云平台覆盖全球近 20 个地区，其中三个地区即将推出。谷歌云平台是新的，这就是为什么与 Azure 云平台相比，它的区域更少。 |
| **市场份额** | Azure 云平台的市场份额约占整个云市场的 16%。它比 AWS 云平台少，比 Google 云平台多。 | 谷歌云平台的市场份额约占整个云市场的 10%。与 Azure 云平台和 AWS 云平台等其他云服务相比，它更少。 |
| **增长率** | Azure 云平台的增长率为 75%，高于 AWS 云平台的 41%，低于谷歌云平台的 83%。 | 谷歌云平台的增长率为 83%，远远高于 Azure 云平台 75%的增长率。 |
| **服务用户** | 目前，Azure 云平台正在慢慢占领市场，并已覆盖 80%的财富 500 强公司客户群。一些客户是霍尼韦尔、Polycom、Controls、苹果、惠普等等。 | 由于谷歌云平台使用类似谷歌产品使用的基础设施，如 YouTube 和谷歌搜索，增加了谷歌云平台的客户群。一些客户是彭博，汇丰银行，多米诺骨牌，贝宝和更多。 |
| **数据库使用** | Azure 云平台使用 SQL 数据库。 | 谷歌云平台使用谷歌云 SQL 作为其数据库。 |
| **存储服务和 DNS** | Azure 云平台使用磁盘存储作为对象存储。并使用 Azure 文件存储作为文件存储服务。Azure 云平台使用 Azure DNS 服务。 | 谷歌云平台使用谷歌云存储作为对象存储。并使用 Avere /ZFS 作为文件存储服务。谷歌云平台使用谷歌云 DNS 服务。 |
| **价格** | Azure 云平台与谷歌云平台相比成本更高。此外，Azure cloud 向客户收取每分钟的费用。 | 与 AWS 或 Azure 等其他云服务提供商相比，谷歌云平台更便宜。此外，谷歌云平台按秒向用户收费。 |

### 结论

Azure 云平台和 Google 云平台提供了众多云服务，各有利弊，选择合适的云平台完全取决于组织需求。当它用于提供云服务时，有许多因素可以区分这两个平台。
