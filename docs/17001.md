## GIMP 法线贴图介绍

法线贴图是 GIMP 的一个特性，通过它，我们可以增加任何普通纹理的细节，以便在该文本的材质上获得自然的高光，我们稍后将在其他 3d 设计软件的 3d 模型中使用它。在 GIMP 中，我们没有法线贴图的任何内置特性，所以我们必须为 GIMP 中的法线贴图获得一个插件。今天在这篇文章中，我们将了解如何使用一个法线贴图插件来增加大理石砖墙上的纹理细节，我们也将学习如何在 GIMP 中安装这个插件。那你还在等什么让我们开始吧。

### 如何在 GIMP 中执行法线贴图？

在学习如何使用法线贴图之前，我们必须在 GIMP 中有一个法线贴图的插件。所以先放开浏览器，搜索 GIMP 法线贴图插件’。

<small>3D 动画、建模、仿真、游戏开发&其他</small>

![GIMP normal map output 1](img/bef66b2d2b664480b6d7882f3d8bc438.png)



点击搜索的第一个链接，转到下载标签，然后根据你的电脑操作系统下载这个插件的设置。

![GIMP normal map output 2](img/5062db900fbf5117ba9e7efa48fd60b5.png)



您将拥有这个插件的 zip 文件。

![GIMP normal map output 3](img/bb679d6983c7dce4011035403c3b5b0f.png)



现在将它解压到 GIMP 的 Plugins 文件夹中。为此，进入 c 盘>程序文件> GIMP 2 > lib > gimb > 2.0 >插件，点击 Ok 按钮。

![GIMP normal map output 4](img/20a6e728503b7a5a1dffbdbcf87b360a.png)



现在从 GIMP 的 plug-ins 文件夹中选择这三个文件并复制它们。

![GIMP normal map output 5](img/c5e602b708fcd149c340631b9daa2f36.png)



并粘贴到 GIMP 的 bin 文件夹中。

![GIMP normal map output 6](img/fffdb0e7f2fde2477aed2bc06b744178.png)



现在重启 GIMP 软件。

现在让我们有一个在浏览器上为纹理搜索纹理应用法线贴图的纹理。

![GIMP normal map output 7](img/3dc6b0c6a6e9bc842eaebffc6c7755c0.png)



我将下载这个大理石砖块。

![GIMP normal map output 8](img/232bb726e3c9d74dcfe13f05ab4e1c9f.png)



现在进入 GIMP 用户界面，点击菜单栏的 File 菜单。点击打开选项，在 GIMP 中打开这个纹理。

![GIMP normal map output 9](img/dd1d1cf07b1e85978a335d96017d8ed2.png)



选择你想要的文件夹，然后点击打开按钮。

![GIMP normal map output 10](img/d1796d19435861f8184c111d59adcd8f.png)



现在我们有了这个纹理。

![GIMP normal map output 11](img/874566fc5f8d786a182ad63dbc37fb03.png)



让我们把它弄得更黑更白，看看上面的法线贴图效果。转到菜单栏的颜色菜单，然后去饱和，并点击新的向下滚动列表的去饱和选项。

![GIMP normal map output 12](img/b5ad641f2d17ab40a917befb1f72fa42.png)



现在在去饱和对话框的模式选项中选择亮度选项，点击 ok。

![GIMP normal map output 13](img/66665bb68fd20403af0f272b9d5acb6c.png)



现在让我们使这个纹理图像的亮区更亮，暗区更暗。你可以选择任何一个选项来做这件事，那就是亮度对比度、色阶或曲线。我会选择曲线型。

![GIMP normal map output 14](img/106e1ea7c8cd776b3a0aacc41d46076e.png)



现在向上拖动曲线的这个区域，增加图像明亮区域的亮度。

![GIMP normal map output 15](img/43ee113fe2eb86ce631c1dfdfca625e5.png)



向下拖动曲线的左侧区域，使图像的暗区变暗。我们这样做是为了在这个图像的黑白颜色之间有良好的对比度。

![GIMP normal map output 16](img/73f8b251617d016a9dd3ab87d6010f14.png)



现在转到菜单栏的过滤器菜单和地图选项，然后点击新的向下滚动列表的法线地图选项。如果你已经在你的电脑上安装了这个法线贴图插件，你将只有在这个列表中有这个法线贴图选项。

![GIMP normal map output 17](img/a09b7c50e8c3c54fc8a2944e214e3a17.png)



一旦你点击这个选项，你会看到这个法线贴图对话框。

![GIMP normal map output 18](img/d6d00c4ff4f329b8a173831793ddfaac.png)



你可以看到 Alpha 通道和高度源选项在这个对话框中是禁用的，所以要启用它们，请转到 GIMP 的图层面板，右键单击纹理层，然后单击下拉列表中的添加 Alpha 通道选项。

![GIMP normal map output 19](img/8c61dee0b8acfc7d9ce87466a52eb8f9.png)



现在重新打开这个对话框，这一次您将像其他选项一样激活这两个选项。

![GIMP normal map output 20](img/836b79077da2dfa93a25885d436b96e3.png)



现在点击这个对话框的 3d 预览按钮来查看这个纹理的 3d 预览。

![GIMP normal map output 21](img/9d785c8928577993f2d40c24286f69b8.png)



这是 3d 预览对话框。

![GIMP normal map output 22](img/66309f3c70909c6f3d4d7967395b382f.png)



在 3d 预览框中按住该块并旋转，以查看该平面上的纹理。你可以看到在你的纹理平面上有一个砖块形状。

![GIMP normal map output 23](img/31bc1abeedffe63c76a53a210729f8b4.png)



现在你可以使用这个列表中的任何一个法线贴图框的过滤器来增加纹理的细节。你可以看到这个列表的每一个滤镜在你的纹理上一个接一个的效果。

![output 24](img/77dd2b25fa4291830dd4c2b6cbdf6bcc.png)



我将选择一个 3 x 3 的过滤器，它将调整这个纹理的 3 x 3 像素的 RGB 值，并对其细节进行浮雕。增加“缩放”值以增强更多细节。

![output 25](img/3da080691d895fd68ab042a8dd8dc513.png)



可以按住键盘的 Alt 键，然后做鼠标右键按住；然后移动鼠标在预览窗口中放大或缩小该对象。现在启用此对话框的反转 X 选项，并查看其对浮雕细节的影响。它将沿着平面的向内方向运动。

![output 26](img/606ec73a18dd5744c4191db0c407a2ff.png)



一旦你禁用了它，你可以看到它会从飞机的向外方向来。所以你可以根据自己的选择来选择这个选项。

![output 27](img/bc18f8c0f288fb31917f53f878a2c5e3.png)



我们在这个预览窗口的顶部有一些选项。让我们也分析它们，以便在法线贴图上更好地处理。第一个按钮是“旋转对象”；点击启用它，你可以旋转你的纹理对象。

![output 28](img/71e6891d028a24468947608a3ca2cb6a.png)



接下来是旋转灯按钮；点击它来选择它。

![output 29](img/9fc7d358991ff8c54dbac0db74fea638.png)



你可以在你的纹理上增加或减少光线。

![output 30.1](img/e9dd7e7fa4c43c71d28e62f4284cfd02.png)



![output 30.2](img/f53037e531d94baea9ebb4fae49ebc72.png)



您也可以在另一个 3d 形状中查看预览。从列表中选择任何人。

![output 31](img/240a2712cd2d159cd09e06d94e36d34a.png)



你可以选择立方体、球体或任何你想要的东西。它会让你知道当你在 3d 模型上使用它时会是什么样子。

![output 32.1](img/8d6a31093f832602fd7a93ce59cb394c.png)



![output 32.2](img/b9a062526de7662fb9749ca1dbad4da9.png)



一旦你完成了所有你想要的设置，点击法线贴图对话框的 Ok 按钮。

![output 33](img/379f881a4af27443e082b2741d7b060c.png)



你将有你的法线贴图在你的纹理上，并且有你设置的法线贴图的参数。现在将它导出为 PNG 或 JPEG 文件，以便在任何其他 3d 建模软件中使用。为此，请转到菜单栏的文件菜单，然后单击导出为选项。

![output 34](img/358bc66564c7b7720eb3466f937e462b.png)



使用所需的文件扩展名将其保存在所需的位置。

![output 35](img/1bbcaa11561a886ca85ce5fcc689b172.png)
