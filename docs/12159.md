## 在 Windows 上安装 Python 概述

*   调用了具有 windows 和管理员权限的系统。
*   命令提示符
*   如果 python 安装在远程服务器上，那么远程桌面连接应用程序。
*   最少 25 兆字节的磁盘空间。
*   如果重新安装 python，预计还会增加 90 兆字节。

### Python 在 windows 上的安装

以下是在 Windows 上安装 Python 的步骤:

#### 第一步:

下载 python 软件的官方页面链接如下，

<small>网页开发、编程语言、软件测试&其他</small>

链接:[https://www.python.org/downloads/](https://www.python.org/downloads/)

![Python installation on windows Step 1](../Images/c0fc158b2627961460847c355c66d611.png)

#### 第二步:

选择 windows 链接将带您到下面的子链接。在下面的子链接中，我们需要选择 python 版本。

![Python installation on windows Step 2](../Images/9d5ce624a1200a025a944c5cc92d7120.png)

<noscript><img class="alignnone wp-image-211046 size-full" src="../Images/9d5ce624a1200a025a944c5cc92d7120.png" alt="Python installation on windows Step 2" width="557" height="295" srcset="https://cdn.educba.com/academy/wp-content/uploads/2019/09/pyt.png 557w, https://cdn.educba.com/academy/wp-content/uploads/2019/09/pyt-300x159.png 300w" sizes="(max-width: 557px) 100vw, 557px" data-original-src="https://cdn.educba.com/academy/wp-content/uploads/2019/09/pyt.png"/></noscript>

下载软件并保存到 PC。名为 python-3.7.4-amd64.exe 的文件已经开始在你的标准下载文件夹中下载。该文件以 30 兆字节为主题，确保在进入软件安装之前完成软件的完整下载。如果需要，我们可以将这个文件移动到一个不同的位置，对于 python 安装来说，这个位置可能更永久。

#### 第三步:

以下是 python 的安装说明:

从安装位置，点击下面的图标 python-3.7.4-amd64.exe。点击此图标会弹出 python 的安装仪表盘。这个仪表板也称为 python 设置弹出窗口，下面粘贴了一个相同的快照；此快照适用于 python 3.7.4 版的弹出窗口。确保在安装设置弹出窗口中勾选了下面列出的复选框。

*   "为所有用户安装启动器(推荐) "
*   "将 Python 3.7 添加到路径"

如果我们的系统中已经安装了 Python 的现有版本，那么命令提示符中声明立即安装的消息将作为立即升级发布。这意味着最好在当前用户系统中安装最新版本的 python，或者重新安装现有软件。

#### 第四步:

此外，定制安装选项将允许用户定制与 python 核心版本一起安装的各种组件。下面的屏幕显示了定制安装屏幕中可用的各种选项。

![Python installation on windows Step 4.1](../Images/0b9baf23ef01bf7757630f6ef11c73e6.png)

<noscript><img class="alignnone wp-image-211055 size-full" src="../Images/0b9baf23ef01bf7757630f6ef11c73e6.png" alt="Python installation on windows Step 4.1" width="559" height="340" srcset="https://cdn.educba.com/academy/wp-content/uploads/2019/09/pty4.png 559w, https://cdn.educba.com/academy/wp-content/uploads/2019/09/pty4-300x182.png 300w" sizes="(max-width: 559px) 100vw, 559px" data-original-src="https://cdn.educba.com/academy/wp-content/uploads/2019/09/pty4.png"/></noscript>

**选项包括**

*   核心 python 文档文件的安装。
*   Pip 安装程序是帮助安装和卸载面向 python 的包的附加软件。
*   TCL / TK 和 IDE:这允许我们为 python 安装 IDE 设置。
*   python 的测试套件。
*   python 启动器使得 python 的开始变得容易得多。

在这个定制屏幕上单击“下一步”,将把用户带到 python 安装中的进一步高级选项。这些包括以下项目:

![Python installation on windows Step 4.2](../Images/8469a549c360dbdb6c58bb352a6662fb.png)

<noscript><img class="alignnone wp-image-211056 size-full" src="../Images/8469a549c360dbdb6c58bb352a6662fb.png" alt="Python installation on windows Step 4.2" width="552" height="344" srcset="https://cdn.educba.com/academy/wp-content/uploads/2019/09/pty5.png 552w, https://cdn.educba.com/academy/wp-content/uploads/2019/09/pty5-300x187.png 300w" sizes="(max-width: 552px) 100vw, 552px" data-original-src="https://cdn.educba.com/academy/wp-content/uploads/2019/09/pty5.png"/></noscript>

*   是为该系统上的所有用户安装 python，还是考虑为某个特定用户安装。
*   使用 py launcher 关联多个 python 文件。
*   创建必要的快捷方式。
*   自动将 python 添加到系统环境变量中。
*   与 python 相关的所有可用标准库的预编译。
*   与 python 环境相关的所有调试符号的下载。
*   下载调试二进制文件。

我们还可以注意到，这个定制部分包含 python 定制安装位置的选项。因此，在这里设置位置允许 python 软件安装在非常需要的位置。

#### 第五步:

在这种情况下，用户第一次在系统中安装 python 软件，单击 install now 消息，甚至在 customization 透视图中，单击安装设置弹出窗口底部的 install 按钮。这将在用户屏幕上显示安装进度设置。安装进度设置显示了用户当前系统中包含的所有软件相关文件的总体进度，以及预期包含的进度。这是通过进度屏幕上绿色刻度的进度来通知的。
