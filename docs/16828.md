## SVG 墨水匣简介

以下文章提供了 Inkscape SVG 的概要。Inkscape SVG 是一种文件格式，这种 SVG 代表可缩放矢量图形，这意味着您可以编辑该文件的对象，以改变它们的大小和属性，而不会损失其质量。

### 过程

在这篇文章中，我们将看到如何将 Jpeg 或光栅图像转换成矢量文件，并保存该矢量文件，以便在其他设计平台上使用。这个过程包括参数调整位图描摹的特点，该软件的菜单栏，我们也经历了一些其他参数。

<small>3D 动画、建模、仿真、游戏开发&其他</small>

因此，让我们深入研究这些特性，以获得我们想要的结果。

为了开始我们的过程，我们需要一个光栅图像。所以在网上搜索一下。我们将通过搜索“卡通人物剪贴画”关键字来搜索卡通形象，然后选择这个卡通人物。

![Inkscape SVG 1](img/f0f36f50d629254324f342621e21a009.png)



现在切换到 Inkscape 软件，要打开它，请单击文件菜单，然后选择打开选项，向下滚动列表，或者您可以按 Ctrl + O 作为快捷键。

![Inkscape SVG 2](img/61a32c9764479508c9808d83ffa6ffd3.png)



我们将在保存的文件夹中导航我的图像，然后选择它。现在点击这个对话框的打开按钮。

![Inkscape SVG 3](img/52c6c0397e1a09c901209eaa174f4e75.png)



现在选择这张图片，我们将跟踪它。所以为了追踪它，去菜单栏的路径菜单然后点击它。点击此菜单下拉列表中的描摹位图选项，或者使用快捷键 Shift + Alt + B。

![Inkscape SVG 4](img/45640b2041159734dc38a9c9771ceb1c.png)



或者，您可以右键单击，然后从弹出列表中选择描摹位图选项。

![Inkscape SVG 5](img/de1f5e6e4125c5a45614bea5939e9a80.png)



这是用于跟踪图像的对话框。这里我们有许多方法来跟踪位图(如果你想知道这个对话框中跟踪位图的每一种方法，那么你可以搜索我的文章“Inkscape 跟踪位图”),但是我们将使用第一种方法，即亮度截止。

![Inkscape SVG 6](img/32e2691d4ebf87bfc990c359b95ea260.png)



此选项有阈值限制，通过更改其值，您可以在位图跟踪中获得不同的结果。

![Inkscape SVG 7](img/5d2425fdbab169b921284e83604a86dc.png)



单击此对话框的更新按钮，查看跟踪结果的预览。

![Click on Update button of this dialog box](img/2d4fcab37da6632c786b1898de026102.png)



在这个框的预览部分，你可以看到结果，但我们只是想跟踪这个字符的轮廓，所以我们将调整跟踪只是轮廓的阈值限制。

![trace outline](img/263a6b90e16d6766303e05c5e26981a5.png)



对于图像阈值将是这样的。现在点击确定按钮来追踪它。

![click on Ok button for tracing it](img/6096f3c646a874c83779dd7c9a7bb677.png)



你可以移动你的跟踪图像到原始图像旁边，这个跟踪图像将会是矢量格式。

![move your traced image](img/ca872258597ca7024b9e2567f8983157.png)



现在，我们将填充颜色到这个跟踪的图像，并在用户界面的底部点击箭头按钮。一旦你点击箭头按钮，你会有一个弹出列表，从这个列表中选择滴管。

![Inkscape SVG 12](img/aa6e67d7e350f3dfc585e377ba0cac71.png)



在使用滴管之前，请确保我们的文档区域没有被选中。然后将光标移动到你想追踪的颜色上。我们将移动滴管工具光标在这个卡通人物的肤色是黄色。

![Inkscape SVG 13](img/e9f25129b0da6094f46d173d33ff6d16.png)



然后从工具面板中选择“填充有界区域”工具，将我们选择的颜色放入我们想要的描摹图像区域。您也可以按 Shift + 7 来使用此工具。

![Inkscape SVG 14](img/80113c3c3734c1621507fd448f17e9c5.png)



用这个工具的光标点击描摹图像的皮肤，它就会这样填充。

![click on the skin of traced image](img/7d931b3c5ce2979d4174b6059a259e96.png)



为了选择下一种颜色，取消选择描摹图像的前一部分，否则在您下一次选择新颜色时，将会像这样替换前一部分的填充颜色。

![Inkscape SVG 16](img/a03516f6a343cceb1f9e1008d139bd3c.png)



重复上面的步骤，填充这个角色的所有部分。

![repeating same process](img/a6604506245e042cb20d874592cb6df6.png)



现在你可以看到我们角色的皮肤已经准备好了。

![Inkscape SVG 18](img/45466d4bee53694cf212416e07d9a9b9.png)



我们将画一个深色的矩形来解释你在填写时应该记住的事情。

![Inkscape SVG 19](img/3706a0ea6d5392160cfc468c6d72d66f.png)



现在这是这个字符的白色区域，你也应该填充它，因为你可能会因为我们的背景是白色而误将其留空。如果有一部分是黑色的，对黑色也做同样的操作。简而言之，你必须用颜色填充所有部分，不管是白色还是黑色，或者和你的背景颜色一样。

![Inkscape SVG 20](img/3d294501940939def7bb738ebc24dd99.png)



现在我们将统一这个角色的相同部分。因此，选择这个唇部分，然后进入路径菜单，点击下拉列表中的联合选项，或按 Ctrl +快捷键。

![Inkscape SVG 21](img/be21a4d05e36edde13e172ee26cf4bb3.png)



你可以看到它现在是一部分。

![Inkscape SVG 22](img/86e30c588cc6f3e4b8dd431bd6ac7b5f.png)



对你想做成一个零件的所有零件做同样的操作。

![Inkscape SVG 23](img/8b29cb77f9f13d5a63e4280313f09722.png)



你应该分析所有零件，这样就不会留下任何零件。

例如，你可以注意到这是它皮肤的一小部分，与全身皮肤是分开的，所以我将它与全身皮肤结合在一起。

![Inkscape SVG 24](img/d5736d2b160129c7884be99aa1ce96a8.png)



现在这个角色的所有部分都像这样准备好了。每个部分都在不同的层上。

![all part of this character is ready](img/0f69a5b19d558ba663b5e9d309c6aac1.png)



现在选择这个字符的轮廓，并按下键盘上的 Ctrl + d 键复制它，以便在你未来的工作中使用。

![make duplicate copy](img/47e796a7a201aa6d26b201138557eb14.png)



你可以改变轮廓的颜色以及你的选择

![Inkscape SVG 27](img/a2dd1cc98b2c3e544b8f049d217ba342.png)



现在，我们将保存为 SVG 文件，所以去文件菜单，并点击下拉列表中的另存为选项。

![Inkscape SVG 28](img/8fa008a0918eaf61c9fb832f5cfc0b69.png)



在文件扩展名列表中选择普通 SVG 选项，并选择你想保存它的位置，然后点击保存按钮。

![Inkscape SVG 29](img/6488cb21d25404017bfdcccc525864ab.png)



并且我们的 SVG 文件可以在其他设计软件中使用。

![Inkscape SVG 30](img/805bf45c160368f4d7e6f1c7f2af0e38.png)



您会注意到我们的轮廓字符没有保存，因为它超出了您的文档区域。所以要确保 SVG 文件中的任何内容都应该在文档区域内。

![notice our outline character has not saved](img/f6dd3618b0578081e7eb3cc5feec2c17.png)



保存后就准备好了。

![it is ready](img/2263ee67b91a5aef6958b10ba8c0df90.png)
