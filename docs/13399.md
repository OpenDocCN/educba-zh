## UI 设计工具的定义

用于软件应用程序的用户界面(UI)设计的工具，即它是软件工程的一部分，帮助 UI 设计人员开发显示屏设计、重要功能的设计、软件应用程序用户界面的原型或蓝图。它提供软件应用程序视图屏幕设计的副本或模型、原型，或者与开发人员交换应用程序功能，并跟踪产品设计工作流程。

### UI 设计工具有哪些？

有多种工具用于软件应用程序的用户界面(UI)设计。它帮助开发者根据设计者制定的设计程序开发用户界面。根据基于向量的学习曲线、与团队共享工作流、组件的使用、工作的容易程度、成本因素和保真度水平，我们选择用户界面(UI)工具来设计软件应用程序的 UI。一些工具用于线框化(用户界面的结构级设计的蓝图),而其他工具用于 UI 设计和软件应用的用户界面设计的原型。但是我们讨论一些具有高级特性的重要工具。这些工具的例子有:

### 9 大用户界面设计工具

泽普林(苹果操作系统，视窗系统，网络)

#### 1.素描

它是一个类似 Photoshop 的工具，设计者最喜欢用它来为软件应用程序设计用户界面(UI)。它用于为 mac 操作系统(macOS)设计 UI，并提供标准功能，如画板、符号和镜像效果等。这是一个基于矢量的绘图工具，允许编辑和操作照片，改变风格，布局和大小，可重复使用的元素使用应用程序用户界面内的复制和粘贴。

#### 2.视觉

它是一个基于 web 的原型工具，主要用于 UI 设计，并在 mac 操作系统(macOS)中使用。它提供了先进的原型，工作流程状态，拖放设计组件，添加静态图像，添加动画图片和上传文件。它在设计者和开发者之间架起了一座工作流的桥梁，从而设计出更好、更快、更实时的用户界面(UI)。它也被称为 InVision Studio。

#### 3.Adobe Xd

它也被称为 Adobe 体验设计。它是一个基于向量的设计和原型工具，用于用户界面设计。它支持 macOS 以及 Windows 操作系统。Adobe Xd 提供屏幕设计和重用、原型制作、开发线框、实时查看设计、导入文件以及集成一些基于 Adobe 的产品/工具，如 Photoshop。

#### 4.菲格玛

它是第一个 UI 设计工具，表示设计功能、实时协作、屏幕设计、UI 的实时镜像、原型制作、屏幕大小更改、收集反馈、高级动画和设计元素的重用。它将支持 Web、macOS、Linux、Windows 操作系统。

#### 5.弗林托

它是一个交互式原型工具，用于软件应用程序的用户界面设计。它提供了一些高级功能，如设计之间的微交互，屏幕切换过渡，通过拖动视频文件或其他视频文件(如 GIF 文件)到我们的用户界面设计中来添加视频文件，添加声音效果和滚动页面效果等。它将只适合 mac 操作系统。

#### 6.巴尔萨米克

它是最好的线框(网站的结构层次设计)工具，它在设计师、产品负责人、项目经理和开发者之间建立了联系。这里，不同的用户界面元素出现在 UI 库中，并且符合我们的要求；我们选择元素并将其拖放到线框中。所以这是设计师准备 UI 原型的简单方法。我们可以在交给开发人员进行可用性测试之前对其进行测试。它既适用于 macOS，也适用于 Windows 操作系统。

#### 7.折纸工作室

这是脸书使用的用户界面设计工具，具有多种功能，如添加不同的交互规则和逻辑，动态动画，屏幕可见性标准，实时视图等。它支持所有类型的操作系统，如 mac OS、Windows、Linux 和 Web。

#### 8.原则

当我们设计一个多屏幕应用程序或新的动画时，这个工具最适合 UI 设计。它支持 mac OS 的额外功能，如屏幕连接、艺术板创建、对齐、UI 的实时预览和屏幕镜像，但不支持团队协作。

#### 9\. Zeplin

它不是一个 UI 设计和原型工具，但是，它用于 UI 设计原型的后期设计环境。它是设计者和开发者之间的一个中介，将 UI 转换成开发的应用程序，也就是说，如果我们将设计的用户界面添加到 Zeplin，那么它将转换成开发者环境来开发项目。它将支持 mac OS，Windows，Web。
