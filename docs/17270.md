## ZBrush 雕塑简介

在这篇文章中，我们将学习 ZBrush 雕刻。在 Zbrush 中雕刻主要使用笔刷，但是许多修改器影响笔刷和我们雕刻的方式。可以使用几个控件来修改笔刷，如重力、包裹模式、密度等。除此之外，还有一个 alt 模式，可以改变或逆转画笔的行为，这可以改变雕塑的效果。还有许多改变笔刷行为的笔画模式。在雕塑中，当你有一些锯齿状的边缘时，你可能想要平滑它们，并且有各种各样的方法来平滑几何体。\

### ZBrush 雕刻的步骤

下面是下面提到的步骤

<small>3D 动画、建模、仿真、游戏开发&其他</small>

**步骤 1:** 转到菜单栏中的文档，然后选择打开 WSize 的新文档，这将使画布大小与视口大小相同。

![ZBrush sculpting output 1](img/d0a2e9431141f38c5416be61138888dd.png)



**步骤 2:** 从加载工具选项开始。

![ZBrush sculpting output 2](img/f373718180a1068971fa84f4df3ab8e0.png)



第三步:如果你有任何工具，导航到存储的工具，或者你可以选择一些原始的工具，如球体，因为这个工具可以更好地演示各种笔刷。

![ZBrush sculpting output 3](img/edf84de2baa0a53e85e429e9c7b0a839.png)



**步骤 4:** 然后在视口内拖动工具

![ZBrush sculpting output 4](img/9e9ddbbc70ea66f67ce7ff52cb21589f.png)



**第五步:**创建球体后，进入编辑模式。此模式将允许您雕刻 3D 对象。当您将基本三维对象置于编辑模式时，它将在工具选项板中自动关闭。

![ZBrush sculpting output 5](img/9560efd09ecd57febb182410810cf76b.png)



**步骤 6:** 启用透视按钮，该按钮将对 3D 对象应用透视缩短。此外，按 Floor 键激活透视地板栅格，可以按轴激活，默认情况下，Y 轴处于启用状态。

![ZBrush sculpting output 6](img/ac32c279e35e028702baa2d1f78de0dc.png)



**第七步:**默认选择标准笔刷，我们用它开始雕刻，看看一些设置。绘图大小决定了画笔的大小。另一方面，z 强度决定了深度。而焦点偏移决定了画笔的衰减

![ZBrush sculpting output 7](img/1e1dea8cf5c72bc0bf730f1b1936d767.png)



**步骤 8:** 接下来是笔画模式，当我们切换到手绘模式时，虽然它通过放置多个实例类似于点模式，但它也通过减少点之间的间距来连接这些实例，并且它可能比点笔画慢。

![ZBrush sculpting output 8](img/298ba1d02e5db0c269f69db931119174.png)



**第 9 步:**喷涂笔划用于在光标拖动路径上应用不同大小和颜色强度的随机图案。可以根据您想要的外观调整位置、比例、颜色和流量参数。

![ZBrush sculpting output 9](img/5d52fc2dbb7779bb135b3e8b25e4e100.png)



**步骤 10:** 让我们切换到粘土笔刷或按 B–C–L，当我们填充耳朵附近的区域时，它会给我们背面的伪像。

![ZBrush sculpting output 10](img/53f340aac27be5813ccbba7c1f788fa9.png)



**步骤 11:** 为了避免这个问题，进入笔刷- >自动蒙版，打开背面蒙版。

![ZBrush sculpting output 11](img/28bc543309588dd666f71d6cfa4508ae.png)



**第十二步:**现在，当我们灌满耳朵，问题就没了。

![ZBrush sculpting output 12](img/e795e4564a2ecc15526bde3b0510a397.png)



**步骤 13:** 在“笔画”菜单中，有一个“懒惰鼠标”选项，它使您能够创建精确度较低的装饰性形状，因为您可以沿着您绘制的曲线看到一条线。可以调整懒人半径来增加效果。

![ZBrush sculpting output 13](img/886ccf8cd55e0fe74fa7929dba4fbf8c.png)



**步骤 14:** 现在，让我们来看看可以使用 X 键打开和关闭的笔刷对称性。如果你想知道你是否处于对称模式，而不看另一边，那么你可以看到，当我们处于对称模式时，主圆内有一个额外的圆，当对称模式打开时，那个圆就消失了。

![ZBrush sculpting output 14](img/e7e30298c700c178c32d0193fa02b301.png)



**步骤 15:** 当我们在变换调色板中看到，我们可以看到，我们也可以激活沿 Y 轴和 Z 轴的对称性。

![ZBrush sculpting output 15](img/8479c77ce29f5f08608c10aaeb8fe69a.png)



**步骤 16:** 为了演示径向对称，我们可以选择一个球体工具，然后在任意轴上激活径向对称，以获得一些有趣的效果。

![ZBrush sculpting output 16](img/cfcde92072f3dd6497a7c7290882ad4f.png)



**步骤 17:** 回到我们的面工具，当我们看绘制尺寸的时候，有一个动态的选项。如果启用，那么笔刷将根据模型缩放。

![output 17](img/13156ddc0dca1ab51b8c5e355f15390b.png)



第 18 步:如果我们双击单词 dynamic，它就会切换它，然后绘制的大小将取决于屏幕，如果我们想快速平滑模型的部分，这是很有用的。

![output 18](img/d8da52fdd3099fa134bb52f7c4bd3ca9.png)



**第 19 步:**还有一个选项是打开每个笔刷的记住动态模式，它会记住你是否在那个笔刷上关闭或打开了动态模式。

![output 19](img/0ac372967c1adc68b67706ee3715cf5a.png)



**步骤 20:** 同样的，你也可以记得每刷画尺寸。

![output 20](img/f55f768a46bbb878de735e78bdff7bec.png)



步骤 21: 现在我们知道雕刻的时候按住 shift 键可以平滑几何体，但是我们可以改变这个笔刷。让我们选择平滑峰刷或按 B–S–p。会有一个警告，然后跳过它，直到重新启动。当我们开始平滑颈部时，它不起作用。

![output 21](img/2351a1df52fbc3d7cf95f4d6336d1373.png)



第 22 步:但是当我们在下巴上下功夫的时候，它就起作用了。

![output 22](img/2167aaa11a63c0c455bc92d9825b98e4.png)



**步骤 23:** 假设你要平滑颈部，我们可以边按边按 alt。按 alt 也叫反转笔刷效果。

![output 23](img/9d7c6307be41883882779df4330d7b60.png)



**步骤 24:** 大部分画笔都有 alt 模式；正如我们在绘制时在标准画笔中看到的，第一个两笔处于正常模式，这是建立，第三个处于 alt 模式，这是删除。

![output 24](img/6c003e54b105dba0882c578b9883373b.png)



**第 25 步:**但也有一些例外，比如 DamStandard，默认情况下会深入，然后在 alt 模式下，会在相反的方向创建一个折痕。

![output 25](img/6ccd3403f841ca68301c315c18440068.png)



**步骤 26:** 当我们在这个 alt 模式下使用移动笔刷时，它直接从表面而不是相对于相机视图来绘制几何图形。

![output 26](img/d4c9ea3019751781d21fdb25a15d9de4.png)



然后，我们选择一个阿尔法

![output 27](img/185cfa235a9bc592d91dbdb0f9ff976b.png)



**步骤 28:** 打开 alpha，它沿着曲线给同一个图像盖章。

![output 28](img/374a82b13e6999fe1c9a975572d98aca.png)



**步骤 29:** 现在打开“滚动笔划修改器”

![output 29](img/2163fbe712921eb82ccd75d5396433e5.png)



**步骤 30:** 所以画笔先完成一份，再完成另一份。

![output 30](img/d946011eeceb96b481c713f3d551996f.png)



**步骤 31:** 如图所示，滚动修改器与具有连续性的笔刷配合使用效果更好。

![output 31](img/f013a4c191c688fdd3e24fd1e07303b8.png)



**步骤 32:** 另一种方法是选择 DragRect 笔画类型，放置一个笔画实例，你可以用光标控制大小。

![ZBrush sculpting output 32](img/3a379a0d5840d42eb2660f9da89945fe.png)
