## Python 正则表达式测试器简介

在本文中，我们将讨论一个用于测试正则表达式的工具或编辑器。在 Python 中，regex 定义了正则表达式，它是一个字母或符号序列，定义了一个逻辑模式，用于将字符串与 regex 定义的逻辑模式进行匹配。所以要在 Python 中测试这些正则表达式，有不同的正则表达式测试器。这些正则表达式测试器是用于正则表达式编辑的工具。这些测试人员充当编写和测试正则表达式的编辑器。很多在线正则表达式测试器测试的正则表达式有 Pythex，PyRegex，Regex Planet 等。
