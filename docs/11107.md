## 网络攻击类型介绍

网络攻击是针对计算机网络、计算机系统、信息基础设施或个人计算机设备的任何类型的恶意攻击，使用各种方法来更改、窃取或破坏数据。有两种类型的网络攻击，例如主动攻击，它们试图改变系统资源或改变数据的改变和破坏。被动攻击试图使用系统中的信息，但不影响系统资源。窃听或监控传输是被动攻击的例子。

### 什么是网络攻击？

*   网络攻击可以定义为攻击互联网上可用的系统，以任何方式给组织或企业带来损失。网络攻击的结果可能是业务损失、声誉损失、金钱损失等等，无论如何都会危及业务。
*   恶意用户通常在未经系统所有者事先同意的情况下发起攻击。有很多方法可以发动攻击。虽然可以保护系统免受组织外部恶意用户的攻击，但是必须保护系统免受内部恶意用户的攻击。
*   网络攻击一词指的是对互联网上的系统发起的攻击。根据系统被滥用的方式，我们将在下一节看到各种各样的网络攻击。
*   必须定期评估系统，以抵御强大的攻击。攻击者有时会攻击其他系统来获得名声，而有时他们这样做是为了金钱利益。

### 网络攻击的类型

因此，让我们来看看那里可用的、经常使用的网络攻击类型——

**勒索软件攻击—**勒索软件攻击可以定义为一种网络攻击，在这种攻击中，所有系统文件都被加密，黑客要求组织付费才能重新访问这些文件。一旦使用这种攻击模式成功地攻击了系统，剩下的唯一选择就是向黑客支付费用，这可能会使组织付出巨大的代价。可以使用最好的防御系统并保持系统评估的频繁性来保护它。

**中间人(MitM)攻击–**在这种攻击中，黑客试图在信息在互联网上传输时捕获信息。他们试图嗅出它，然后非常仔细地研究信息，以检查它是否有助于破坏系统。为了保护您的系统免受这种攻击，您的设备与服务器的通信应该加密。例如，如果您想使客户机和服务器的通信安全，您可以选择 SSL 服务。使用安全套接字层加密的数据不太容易受到这种攻击。

**拒绝服务攻击–**它可以被定义为一种网络攻击，即主机向目标服务器发送过多的请求，从而导致其崩溃。无法处理这么多请求的服务器会停止工作一段时间，并且不允许任何用户使用它，因为一切都已恢复。为了保护服务器免受这种攻击，使用了负载平衡器。负载平衡器的作用是在服务器开始收到太多它无法单独处理的请求时为其提供支持。这种攻击的扩展形式是 DDOS(分布式拒绝服务)，其中许多主机向目标服务器发送请求，每个主机发送足够的请求以使目标崩溃。

**跨站脚本攻击—**跨站脚本攻击是一种利用 JavaScript 来利用 web 应用程序的网络攻击。通过填充和提交文本字段，并检查它们是否正在运行正确的检查，或者只是让脚本运行而不进行任何清理，可以手动检测易受此类攻击的 web 应用程序。这种攻击的严重程度足以让黑客获得一些有用的信息。为了对抗这种攻击，应用程序必须始终被开发为检查用户提交的值，而不是处理它们。

**网络钓鱼攻击–**这是一种非常常见的攻击，属于社会工程的范畴。在这种攻击中，黑客向经过身份验证的用户发送一个看起来与合法网站完全相同的网页 URL，并诱骗用户提交他们的凭据。黑客使用凭证获得系统的完全访问权，并进行他们想要的更改。一些网站已经开发了该系统来保护他们的[系统免受网络钓鱼攻击](https://www.educba.com/what-is-a-phishing-attack/)，但是在任何地方提交他们的凭证之前，必须非常小心。
