## LotusScript 简介

LotusScripting 遵循 OOP 的概念，即面向对象的编程语言。它使用 Lotus notes 进行 LotusScripting。它将使用 4.0 版的 Lotus notes。它在 IBM Louts 软件产品中也很有用。LotusScripting 类似于 VB，即 visual basic。在 LotusScripting 中，有两个要点，第一个是集成开发环境，第二个是特定于产品的对象类。LotusScripting 将提供一组丰富的应用程序或特定于应用程序的类。它将为 Lotus word pro、Lotus Notes、Lotus 1 2 3 等提供良好的平台支持。在 LotusScripting 中，我们可以使用 OLE 自动化。我们也可以在 Lotus notes 生态系统中使用它。尽管如此，我们也可以在 Visual Basic 中使用它。

LotusScript 的一些优点是:

*   类似于任务集成，我们也可以将待办事项也集成。
*   在 LotusScript 中，我们可以使用电子邮件集成以及电子邮件警报。
*   在 LotusScripting 中，我们也可以轻松地集成地址簿。
*   为了数据的可靠性和可用性，我们需要启用复制因子。
