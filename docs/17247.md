## Photoshop 中线条工具介绍

线条工具可以帮助 Photoshop 用户使用线条来组合和定向他们的作品或对象，以创建一个干净和结构良好的设计布局，线条工具还可以在各种项目领域中派上用场，从创建简单的线条，添加流动推车，在设计文档中添加分区，为建筑绘图创建设计元素-线条工具满足所有这些和更多。

这个工具也很有趣，其属性将在内容中详细描述。对于学生和初学 Photoshop 的人来说，了解线条工具如何用于不同的行业项目是非常有用的。线条工具用于绘制线条、制作流程图、将工作空间分成相等或可调整数量的分割部分、绘制土木、电气和管道布局以供施工使用等等，这取决于我们使用它的项目，并且易于使用。

Before starting to draw a line on the workspace, check and configure its properties shush as Arrow if needed at start or end, lines length %, width % and concavity % of the arrows, and the weight of pixel count of the line. 长按键盘上的 shift 键有助于准确地从头到尾绘制直线。shift 键技术也可以用来绘制 45 度角的箭头在任何方向上根据你的鼠标光标结束。这个技巧有助于绘制布局，报纸上的文章边界，工程技术图纸等等。

箭头可以在绘制前根据需要进行设置。有 4 种类型的箭头设置，我们的箭头只是一条线，箭头只在线的开始，箭头在线的两侧，箭头在线的结束。下面是所有这些的例子。

一些更高级的选项是模式，不透明度和反锯齿。模式有助于在图层上创建线和线性空心(擦除)用户必须测试所有图层，以了解何时有效地使用它们。不透明度处理基于%集合的线的视觉显示。必须始终打开抗锯齿，因为它有助于对绘制到其他像素的线条颜色进行非像素评级，这会影响图像的质量。

### 带截图的步骤

*   打开 Photoshop 后，“U”是激活形状工具和选择线条工具的热键。在工作区排队。该工具位于工具菜单栏中的文本工具下方。这包含了中枢中所有其他的整形工具。选择线条工具后，线条工具的属性将显示在应用程序的顶部菜单栏上。

![line tool 2](img/88e776fe481c320455228b6be7877885.png)



*   单击下拉箭头以获取线条的箭头属性列表。我们可以根据需要对它们进行配置。请确保在绘制前配置线条属性，因为第一次正确操作比使用其他工具交替设计更准确。

![line tool 3](img/ee083f55fe280ee61352d2532a38ed23.png)



*   我们有线属性，重量像素配置，因为每计数的像素可以做。我们可以根据需要设置线条的模式，如果需要的话使用不透明度，并激活反锯齿以避免在创建线条的边界处出现线条像素模糊的问题，从而提高工作质量。

![line tool 4](img/88bcf05f62a578503a6fefb06d32fcb7.png)



*   让我们尝试使用 shift 键没有任何箭头的线，开始和结束线拖动鼠标 courser 从起点到终点画一条直线。使用 shift，可以向工作区域的任何一侧画出 45 度角位移的直线。
*   徒手画图也可以不用 shift 键，这样会导致像素化的变化，很多 Photoshop 的专业用户都不推荐。

![line tool 5](img/8f02b8a1bd889606ac7c030338a0a031.png)



*   起点带有箭头的线条，具有所需的宽度、长度和凹度。我们可以看到箭头的起点是活动的。

![line tool 6](img/be03d22b770be863c008605d0ad5a00a.png)



*   该线用两个方向的箭头绘制。我们可以看到，在画线之前，起点和终点都处于活动状态。类似地，我们可以关闭起点箭头，只绘制直线的终点箭头。

![line tool 7](img/8b1eb199a0c4519bd29d1c183c52f2da.png)



*   使用不同的凹度百分比绘制不同的线条。

![line tool 8](img/f211c442e55fb3830d72a9bdcee58d58.png)



*   激活抗锯齿功能有助于避免线条边缘出现线条像素。

![anti alias](img/6b2b6354ebc76d9b0b0c60e025937de4.png)



*   可以为参考边界设置所绘制线条的不透明度。

![opacity](img/193cabb20dfaa2d7c9a9c347c03c835d.png)



*   模式有助于获得实线，擦除线条形状的数据等。

![modes](img/c29e9d2441da90317dcca347e8d08a44.png)



*   线的权重无非是在工作空间上绘图时线像素计数的占用。

![weight of line](img/bb220e7635416ae047060a0b24659d3a.png)
