## Photoshop 混合笔刷工具简介

Photoshop 混合笔刷工具提供了大量的功能和效果，如创建美丽的纹理和 3D 效果的选项，混合背景色。我们可以选择背景图像作为画笔笔尖，使它们从 Photoshop 中提供的其他画笔工具中脱颖而出。它还详细介绍了如何使用 Photoshop 混合笔刷工具——一旦用户完全掌握了这些工具，就可以方便地创建出令人惊叹的效果。

### 使用 Photoshop 混合笔刷工具

下面详细解释了使用 photoshop 混合笔刷工具的步骤，

<small>3D 动画、建模、仿真、游戏开发&其他</small>

**第一步:**在 photoshop 中打开一张图片。我们可以从文件打开并点击打开，或者我们可以使用 CTRL + O。

![Photoshop Mixer Brush Tool 1-1](img/4bf1f8a1f9c1cd671cfd385632f03f72.png)



**第二步:**从工具栏中选择混合笔刷工具；按住画笔工具，我们可以得到三个布什选项第四个选项是混合画笔工具，我们可以使用 B 作为这个工具的快捷方式。

![toolbar](img/194a20e6099ead324fb7ba7a8af7a5f4.png)



**第三步:**我们可以通过点击这两个'[ ]'键来控制画笔的大小。

**第四步:**新建一层；我们将在一个新的图层中绘制它。

![new layer](img/e6a9564ca82b6e4d4f24b57b5acd58b1.png)



**第五步:**这里是当前刷加载的选项。

![current option](img/5ba2c6cec1d983ad06a9af8ae571f0a3.png)



我们从这里选择画笔的颜色。

![Photoshop Mixer Brush Tool 1-5](img/2b53c82eeb2ba63890752eb75a48f42c.png)



第六步:我们要用画笔在这里画画，但是它比普通的画笔移动得太慢了。

![Photoshop Mixer Brush Tool 1-6](img/7a8a1cfa814071915344f39856884ce3.png)



**第七步:**这里有一个控制混合笔刷工具的选项。切换画笔面板。

![Toggle](img/11281fcec9107b115da3eea299a83017.png)



当我们选择那个选项时，我们将得到一个菜单和画笔控制选项。

![controlling](img/d606e1b823958b469903043c04cc1f4a.png)



**步骤 8:** 我们需要禁用形状动态选项。

![shape dynamics](img/b02bd991b15d4e19133da485ec6067da.png)



然后选择选项画笔笔尖形状，并减少间距。

![Photoshop Mixer Brush Tool 1-11](img/767118cd3b73659ee7b942434e935bbf.png)



如果我们控制了这两个选项，笔刷将会比以前移动得更快。

**第九步:**现在，我们来看看加载画笔工具，

![Photoshop Mixer Brush Tool 1-12](img/c40c99bd4f4cf6cc5a4d24d9f7ccd64b.png)



我们可以用这个选项给笔刷加载一种颜色。这里我加载 2%。我们用这个画画吧。
![Photoshop Mixer Brush Tool 1-13](img/448360023d954262fde9f2e8515fe59a.png)



在这里，如果我们看上面的图像，我们可以发现油漆正在慢慢流失，所以我们能涂多少就涂多少。

**步骤 10:** 切换笔刷面板，将笔刷间距减小到 1。

![Photoshop Mixer Brush Tool 1-14](img/3f33e9286240b741a5b504164e37732a.png)



第 11 步:我们只能画一条小线，因为我们在增加画笔笔尖的空间，而且我们的画笔只有 2%的负荷。

![Photoshop Mixer Brush Tool 1-15](img/46f002dcad713d6491a1e362dd6ab531.png)



**第 12 步:**现在，让我们尝试一个湿选项；100 的时候我们会给 wet，给 wet 的时候它会要求一个 mix 选项；我们也给 mix 100。

![Photoshop Mixer Brush Tool 1-16](img/1620c91d64e45eb7a0d564c6e2b98556.png)



我们可以将颜色选项改为黑色。

![Photoshop Mixer Brush Tool 1-17](img/952fd1114f63cdf489a62918b7f0ee8e.png)



**第 13 步:**我们看上面图像中的颜料是怎么画的，如果我们画的话，会把背景图像颜色拿到颜料上；它会给笔刷一点 3D 深度。

![Photoshop Mixer Brush Tool 1-18](img/ad057702398451ce0341cafa5ec14eea.png)



第 14 步:我们已经给了一个 100%的混合，让画笔为背景上色。

![Photoshop Mixer Brush Tool 1-19](img/74cb68f9b5aceef6b4bd66a31df86a64.png)



现在让我们改变混合的值。我已经给了 10%。

![Photoshop Mixer Brush Tool 1-20](img/5e10f925969d4d005eb6c824143e3a50.png)



我们现在可以看到，画笔用来绘制我们选择的颜色，并混合背景一点点。

**第十五步:**现在我们做一个改动，给湿 0%，混合值为 100%；如果我们在这些值中绘画，绘画将是非常正常的。

![Photoshop Mixer Brush Tool 1-21](img/c35f7e1d8b20542d0b1c88b8fb4a3945.png)



**第十六步:**我们这里有一个有趣的选项，只要按住 ALT 键，走到某一点，点击颜色；然后画笔会观察那个颜色，如果我们画，它会用特定的颜色来画。

![Photoshop Mixer Brush Tool 1-22](img/c4e9dbfea729c58a0647c788667d2345.png)



它将被涂上我们通过按住 ALT 键选择的颜色，

第 17 步:现在，我们将使用此选项尝试另一种方法。只需禁用加载纯色。

![Photoshop Mixer Brush Tool 1-23](img/461b8fa686a8316b09f182d20dc6c20e.png)



当我们激活此选项时，我们只能选择和绘制纯色。但是当我们禁用这个选项并按住 Alt 的时候，它可以捕捉到我们用 Alt 点击的位置的图片。

![Photoshop Mixer Brush Tool 1-20](img/33a2a754ec0a2dc1656e98f319234f65.png)



在这里，我点击一棵树，我们可以看到画笔的尖端像树叶符号一样发生了变化。

![leaf](img/08db32bba3d7d2b310378e36fd6e02ee.png)



在以前，我们用纯色绘画，但是现在我们要用新的画笔笔尖。

![Photoshop Mixer Brush Tool 1-35](img/55ea2f2802f2179e8a2b9785ec8e7912.png)



你看到了吗？这是如此有趣的痛苦；收尾好顺利。

笔画和混合非常棒，将所有的内部线条保持在一起，给这幅画一种三维的感觉；这是惊人的，绘画风格看起来不同，我们用这个选项获得了 3D 深度。

**第 19 步:**我们可以在这里更换画笔；我们有一些笔刷选项。

![ 3D depth](img/3a48c17ee50a03662caa4e4d3387d888.png)



从菜单中选择另一个画笔。

我们可以试着用新的笔刷来画画。

![Photoshop Mixer Brush Tool 1-27](img/5be20d7cb9ccecab5cb3564c95f0f6bf.png)



**步骤 21:** 现在，我们可以在笔刷的帮助下混合背景，只需更改 wet 和 mix 的值；在两个选项中给出 50 和 50 的值。现在通过拖动来绘制它；我们可以看到混合背景。

![Photoshop Mixer Brush Tool 1-28](img/a402ba96652cbd7e8b59dedac7a162b2.png)



第 22 步:如果我们给湿为 0 %，我们可以得到可怕的效果原始笔刷阴影。

![Photoshop Mixer Brush Tool 1-29](img/025f69930db01f04d79783bdf7f70280.png)



**步骤 23:** 选择一个新的笔刷，进入散射选项，增加计数值。

![count value](img/df58db68282776ae91052157cb77e05e.png)



如果我们痛苦，我们现在可以得到一幅不同的画。

![painting](img/41941e6b475207436588f8a6f6b6914d.png)



我们可以用选项得到油漆符号；我们可以用这些选项绘制不同的符号，

**第 25 步:**通过改变颜色，我们可以用这支画笔创造出不同的美丽符号。

![Photoshop Mixer Brush Tool 1-32](img/c77af413099c878868ffaad5b64fd098.png)



我们可以使用这个笔刷工具来创建一个漂亮的纹理。

**步骤 26:** 在这里，我们可以尝试使用一个新的笔刷工具，选择一个看起来像树叶的笔刷，点击它将绘制三个符号。

![Photoshop Mixer Brush Tool 1-33](img/9d75c4344bbd235f4123e20a1abd6370.png)



**第 27 步:**现在，只需随便点击一个地方，图像上就会出现一个漂亮的符号。

![beautiful symbol](img/9795ba163868562408430459b79c1419.png)



我们在这个过程中讨论了混合笔刷工具，我们导入了一张图片，我们用不同的笔刷和阴影进行绘制。
