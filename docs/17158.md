## Photoshop 阳光效果简介

Photoshop 是一种图像编辑软件，用于处理光栅图像，并被专业的图像编辑人员广泛用于他们的专业工作。我们可以在这个软件上创建一个非常好的效果，给我们的图像一个自然的外观。阳光效果是一种效果，通过它你可以在你的图像上添加太阳光或者阳光效果，给它一个非常自然的阳光效果。所以在这篇文章中，我们将学习如何在我们的图像中创建阳光效果，以及创建这种类型的效果需要什么参数。所以让我们开始学习这个话题。

### 如何在 Photoshop 中创建阳光效果？

我们可以很容易地在我们的图像上创建这种类型的效果。我们只需要遵循一些简单的步骤。所以让我们来讨论它们，但是在开始之前，我认为我们应该看看这个软件的工作屏幕来非常容易地理解这个话题。因此，让我们先了解一下这个软件的用户界面。

**步骤 1:** 在工作屏幕的顶部，菜单栏有多个菜单，用于处理该软件的不同类型的参数。在该菜单栏下方，有一个活动工具或图像的属性栏，用于更改它们的参数。在这个属性栏下面，我们在这个软件中有三个部分，在左侧我们有工具面板，在中心我们有显示窗口，你可以在其中看到你的工作，在右侧我们有一些标签，如颜色标签，层面板，和其他一些。你可以根据你的需要调整所有这些部分。

![menu bar](img/ecb1be66d489ec735728f67fa5cdfca7.png)



第二步:现在，让我们对这个话题有一个形象的了解。要在该软件中放置图像，请转到菜单栏的文件菜单并单击它。单击下拉列表中的打开选项。

![Open option](img/c49b8e5ffdb14819dbbab737a1f6484d.png)



**第三步:**点击此选项，会打开一个对话框。转到您个人电脑的目标文件夹，然后点击您想要的图像。选择您想要的图像后，单击此框的打开按钮，在 Photoshop 中打开此图像。

![dialog box](img/98a291eab372ff4fd7824ef165304c1e.png)



**第四步:**它会像这样在这个软件的一个新标签页中打开。您可以通过变换框来调整此图像的大小。现在转到这个软件的图层面板的底部，点击“创建一个调整图层”按钮。从下拉列表中选择渐变选项。

![layer panel](img/06c16e4b2a0f06016198e397374d8ae0.png)



**第五步:**现在，一个渐变框会打开。点击盒子的渐变比例，一旦你点击了盒子的渐变比例，一个渐变编辑框就会像这样打开。现在，将此框渐变刻度的右侧颜色停止滑块拖动到此刻度的左端。

![Gradient Editor box](img/2dadd6325558028f50ed572aa10de339.png)



**第六步:**现在点击该比例尺下方的颜色框。将会打开一个颜色选择器框。从这里选择这种颜色，这样你可以创建一个阳光效果。现在逐个按下两个框的 Ok 按钮。

![color picker box](img/1df09bfc818718c2e2921428d1b56ac8.png)



**第七步:**现在，在渐变框中选择径向渐变。

![Radial gradient](img/4239f7dc77c69a8d97dc3a1b1c04c364.png)



**第八步:**将你图像中的这个渐变移动到你想要的地方。

![Photoshop Sun Effect - 8.1](img/1ccfc6216d550cdb34be4a13eb49d970.png)



**第九步:**您可以放大或缩小这种渐变效果，以增加或减少它的扩散面积。

![Photoshop Sun Effect - 9](img/f071c447dd22265f94e0dca46a700cae.png)



**第十步:**现在在这一层的模式选项中选择柔光选项。

![Photoshop Sun Effect - 10](img/414f0f117cdcaaef681a52524a4d1e15.png)



**步骤 11:** 现在，右键点击渐变图层，点击复制图层选项，进行复制。

![Photoshop Sun Effect - 11](img/f676024d50519bac5899c5c80668d532.png)



**第十二步:**现在，在这个复制图层的模式下，选择屏幕选项。

![Photoshop Sun Effect - 12](img/32cdf1a364c5d82a346659ba342411c9.png)



**步骤 13:** 现在双击这个复制图层的渐变框，将渐变比例降低到 70%，点击这个框的确定按钮。

![Photoshop Sun Effect - 13](img/d896a3b83aa88bba5e24512072bd27cf.png)



现在，再次从这个列表中选择渐变选项。

![Photoshop Sun Effect - 14](img/262e2d5bb4b811bfe2066ddc7e5edddc.png)



**步骤 15:** 并在渐变编辑框的渐变类型中选择噪波选项。

![Photoshop Sun Effect - 15](img/03780bca5fffdd8f3ac258ad524781be.png)



现在，从颜色模式选项中选择 HGB 选项。

![Photoshop Sun Effect - 16](img/1be4299d531ee7870e15b0e59c3b04be.png)



**步骤 17:** 将饱和度标尺的右滑块滑动到左侧。

![Photoshop Sun Effect - 17](img/5543cf440f52718087493d908c12bcc0.png)



步骤 18: 现在反复点击这个框的随机按钮来获得这种类型的噪声效果，然后按下确定按钮。

![Photoshop Sun Effect - 18](img/a0ddf011441a80904625a0251b8f9faf.png)



**步骤 19:** 现在，在渐变风格中选择角度选项。

![Photoshop Sun Effect - 19](img/6f90c31e14397a621d8c562c51478015.png)



**步骤 20:** 现在在和之前渐变相同的地方调整这个渐变。

![Photoshop Sun Effect - 20](img/95d910104fb01ba6ca694bc85ac5354e.png)



**步骤 21:** 现在，在这一层的模式选项中选择屏幕选项。

![Photoshop Sun Effect - 21](img/eb0fc26f1fbd831c94ae76cab7318d58.png)



**步骤 22:** 现在再次打开这一层的渐变框，根据你调整角度。

![Photoshop Sun Effect - 22](img/7d0c23f47a21fbb05e3d572180a52e2b.png)



**步骤 23:** 现在，拿一个笔刷工具，调整笔刷的大小。在前景色框中选择黑色。

![Photoshop Sun Effect - 23](img/fcf5fbc3b95b422101b0983561b0ae92.png)



步骤 24: 现在清除图像中不想要的区域的渐变。

![Photoshop Sun Effect - 24](img/000930b06e4565a5c9868c558336f251.png)



第 25 步:现在，从列表中选择级别选项。

![Photoshop Sun Effect - 25](img/c163e5e77b866604c97715e3c7305c75.png)



**步骤 26:** 移动水平标尺的滑块，增强阳光效果。

![Photoshop Sun Effect - 26](img/e2a1e39c75364ba9a3e0beee3c61ff06.png)



**步骤 27:** 现在，从列表中选择渐变贴图选项。

![Photoshop Sun Effect - 27](img/38d49f114114c14d0e9573c296268ccb.png)



**步骤 28:** 点击打开框的渐变比例，然后从渐变编辑框中选择这种渐变类型。

![Photoshop Sun Effect - 28](img/e355f889d5267f38820f7da922d24d3d.png)



**步骤 29:** 现在，在这个渐变贴图层的模式选项中选择柔光选项。

![Photoshop Sun Effect - 29](img/99f5767b92a31ebc365d7ee58c456bc4.png)



步骤 30: 现在，将这一层的不透明度降低到 75%。

![Photoshop Sun Effect - 30](img/996da521a981d2dfeb2b9c3bdc1f21bb.png)



步骤 31: 点击图层面板的“新建图层”按钮，新建一个图层。

![Photoshop Sun Effect - 31](img/821ea042f5792455f53960eef80f0403.png)



**步骤 32:** 现在转到菜单栏的图像菜单，点击下拉列表的应用图像选项。

![Photoshop Sun Effect - 32](img/af7f7a05306a0c4fea7a0d2cfa5822ea.png)



**步骤 33:** 现在，一个框将被打开，点击这个框的 Ok 按钮。

![Photoshop Sun Effect - 33](img/ae42dc297083e8cea5b417fcce5b5b37.png)



**步骤 34:** 现在进入菜单栏的滤镜菜单，点击列表中的‘相机原始数据滤镜’选项。

![Photoshop Sun Effect - 34](img/f98c586452bf8e08ccbfbd37dd46dbdf.png)



**步骤 35:** 现在相机原始数据盒将被打开。根据您的需要，在此框中调整参数以获得最佳的阳光列表效果。您可以通过更改参数值来查看该框中每个参数的效果。

![Photoshop Sun Effect - 35](img/1fd9b9a91b1bfdedd9be5fffaa641d0b.png)



第 36 步:现在按下这个框的 Ok 按钮。

![Photoshop Sun Effect - 36](img/0ac9a97bd1721a188ff8239299a29766.png)



第 37 步:你最终的阳光效果会是这样的。

![Photoshop Sun Effect - 37](img/008ba40555ab09663ff520e8ab1d8034.png)



这样，你就可以在 Photoshop 软件中创建一个阳光效果。
