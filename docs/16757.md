## Cinema 4d 快捷方式介绍

4D 电影院允许设计师创造和振兴复杂的三维图形模型。用于 3D 图形的程序是通用的，具有许多功能。这简化了设计师对 4D 电影院的操作。由于 3D 图形工具、高分辨率纹理和复杂的动画，该程序具有简化 3D 图形工作流程的灵活性。现在我们会看到一些影院 4d 的快捷方式。

### 4d 影院快捷方式

让我们看看 4D 电影院的捷径

<small>3D 动画、建模、仿真、游戏开发&其他</small>

*   **CTRL + N:** 此键用于 4D 电影院的新项目
*   **CTRL + O:** 该快捷键将打开弹出或对话框，允许用户打开 4D 电影院中已有的项目
*   **CTRL + S:** 该键用于保存 4D 电影院中的项目或任务
*   **CTRL + Shift + S:** 该键用于在 4D 电影院以指定的名称保存当前项目
*   **CTRL + Shift + W:** 用于关闭所有工作项目
*   **CTRL+C；**该键用于用户选择复印
*   此键将关闭 4D 电影院的窗口
*   **退格:**退格用于删除 4D 影院中的对象
*   **CTRL+Q；按下这个，你将退出 4D 电影院**
*   **CTRL+X；**该键用于用户选择切削。
*   **CTRL + P:** 该键用于粘贴用户的选择。
*   **Shift A:** 使用该键，可以反转当前选择
*   **CTRL+A；使用此键，您可以选择 4D 电影院的所有作品。**
*   **CTRL+shift+A；使用此键，您可以取消选择 4D 电影公司的所有作品**
*   **CTRL+Z；使用此键，您可以撤销当前项目**
*   **Shift + Z:** 使用该键，可以撤销上一次操作。
*   通过使用这个，你将得到偏好的弹出窗口
*   通过使用这个关键字，你将获得关于你的项目的信息
*   **Shift + F1:** 此键将帮助您激活 4D 电影院中的对象管理器
*   **Shift+ F2:** 此键将帮助您激活 4D 电影院的材料管理器
*   使用此键，您将获得 4D 电影院的摄影表
*   **Shift + F4:** 这个键会帮助你打开 4D 电影院的图层管理器
*   **Shift + F5:** 此键将帮助您打开 4D 电影院中的属性管理器
*   **Shift + F6:** 使用此键，您可以在 4D 电影院打开图片视图
*   **Shift + F7:** 这个键将帮助你打开 4D 电影院的坐标管理器。
*   **E:** 如果您按下 E 按钮，它将帮助您移动刀具
*   **R:** 如果您按下 R 按钮，它将帮助您在 4D 电影院旋转项目
*   **T:** 此按钮将帮助您打开缩放工具
*   **Shift + F:** 这将帮助你启动项目中的动画。
*   **Shift + G:** 这将帮助你结束项目中的动画。
*   **Shift + Q:** 该键用于在 4D 电影院启用量化。
*   **Shift + S:** 此键将帮助您在 4D 电影院启用组件捕捉。
*   **CTRL + Shift + Y:** 通过使用此键，您可以在项目中重做您的视图费用。
*   **CTRL + Shift +Z:** 通过使用此键，您可以撤消您在项目中的查看费用。
*   **CTRL + R:** 使用此键，您可以渲染 cinema 4d 中的活动视图。
*   **Alt + D:** 该键将帮助您查看显示轴。
*   **Alt + v:** 通过使用这个，您可以配置 cinema 4d 中的所有视图。
*   使用此键，您可以在 cinema 4d 中配置您的视图。
*   **CTRL + G:** 在这个键的帮助下，可以切换到下一个关键帧。
*   **CTRL + O:** 在这个键的帮助下，可以切换到最后一个关键帧。
*   **CTRL + P:** 在这个键的帮助下，可以切换到第一个关键帧。
*   **CTRL + U:** 通过使用该键，您可以在时间线中看到过滤器。
*   使用此键，您可以全屏查看图片。
*   **空格:**使用空格按钮，可以向前播放
*   **Ctrl + shift + N:** 使用此键，您可以在 4D 电影院创建新的 PBR 素材
