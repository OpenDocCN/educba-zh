#### 1.数学动画引擎 Python GitHub 项目

*   数学动画引擎是一个解释数学概念的视频动画引擎。它主要用于以编程方式生成动画。Python 3.7 用于运行数学动画引擎。
*   我们必须首先使用 pip 命令安装 manimlib 模块来运行这个项目。如果不建立这个命令，我们就不能在我们的系统中运行上面的项目。

以下步骤显示了如何使用 github 运行数学动画引擎项目:

*   第一步，我们将项目上传到 git 存储库；现在上传项目后，我们在系统中创建父目录。创建目录后，我们使用 git 命令克隆项目。
*   克隆项目后，我们安装所需的包；在安装了所需的包之后，我们运行项目。
*   该项目与 python 3.7 和更高版本兼容，因此我们可以使用 python 3.7 版运行该项目。

**代码:**

```
$ mkdir manim
$ git clone https://github.com/3b1b/manim.git
$ cd manim
$ pip install -e .
$ manimgl example_scenes.py OpeningManimExample
```

#### 2.谷歌图片下载的 Python GitHub 项目

*   这是一个命令行程序，可以搜索和下载数百张谷歌图片。这个 Python 脚本允许我们在图像上搜索关键字和关键短语。这个程序兼容 Python 2.x 和 Python 3.x。

下面的步骤展示了如何使用 github 运行 google 图片下载项目。

*   第一步，我们将 google 图片下载项目上传到 git 存储库；上传项目后，我们在系统中将父目录命名为 google。创建目录后，我们使用 git clone 命令克隆项目。

在下面的例子中，我们已经安装了 python 包，所以我们不需要在我们的系统上再次安装它。

**代码:**

```
$ mkdir google
$ cd google/
$ git clone https://github.com/hardikvasa/google-images-download.git
$ python setup.py install
```
