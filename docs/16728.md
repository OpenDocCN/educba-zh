## Pro 工具快捷键介绍

Pro tools 快捷方式是在该软件中执行特定任务的键盘按键组合，通过使用这些快捷方式，我们可以节省时间并提高工作效率。Pro tools 提供了许多快捷方式，用于基本到高级级别的命令或选项，并处理它们以使这些命令在处理任何项目时易于访问，例如，您可以使用快捷方式快速保存任何项目，只需按下键盘上的几个键即可开始录制或停止录制，用于导入音轨，创建组和许多其他重要的命令。所以让我一个一个的告诉你。

在这篇文章中，我将把这些快捷方式分成三个部分，让你更容易理解。这些部分分别针对初级水平、中级水平和高级水平。因此，让我们以一种非常激动人心的方式来看看这些部分。

### 初级水平的快捷方式

当您开始使用该软件工作时，这些快捷方式有助于提高您在该软件中的工作技能和效率，这些快捷方式如下:

注意:在快捷键中处处使用 Mac 上的 Command 键代替 Windows 上的 Ctrl 键。

*   保存命令用于保存本软件中的任何项目或工作，您可以按键盘的 Ctrl + S 或 Command + S 键作为快捷键。
*   要开始录音或打卡，请按键盘的 Ctrl +空格键、F12(键盘的功能键)或 3(键盘的数字键)。
*   对于停止录制或放弃录制选项，您可以按 Ctrl +句点(。)键盘的键。你应该记住的一件事是，放弃录音不是一个不可撤销的命令。
*   要将轨道设置为记录安全，您可以按住 Ctrl 键并单击轨道记录启用按钮，即按住键盘上的 Ctrl 键，然后单击工作窗口区域上的轨道记录启用按钮来执行该命令。
*   通过按键盘的 Ctrl + Shift + I 键，可以快速导入本软件中的任何音频。
*   要打开所选剪辑的“渐变”对话框，可以按键盘上的 Ctrl + F 键。
*   键盘的 Ctrl + E 组合键将把所选区域与其他区域分开。
*   通过按键盘上的 Ctrl + G 键，可以为选定的元素创建一个组。
*   Ctrl + H 键将做愈合分离。

我认为这些命令对于一个初学者来说已经足够了，可以很好地开始使用这个软件。现在让我们向前迈进一步，讨论其他一些处理下一级命令和功能的快捷方式。

### 中级的快捷方式

一旦你掌握了初学者的快捷方式，那么你将需要这些快捷方式来提高你的技能，这些快捷方式如下:

*   要使用水平放大，您可以按 Ctrl + [或 Ctrl + ]作为快捷键。Ctrl + [将用于放大，Ctrl + ]用于缩小。
*   按 Ctrl + Alt + [或 Ctrl + Alt + ]选择振幅缩放选项。放大和缩小与水平命令的作用相同。
*   通过按下键盘的 Ctrl +向上箭头键(用于增加)或 Ctrl +向下箭头键(用于减少)，可以增加或减少编辑窗口中任何有光标指针的通道的高度。
*   按 Ctrl + Alt +向上箭头键(用于增加)或 Ctrl + Alt +向下箭头键(用于减少)将增加或减少编辑窗口中所有轨迹的高度。
*   通过按下键盘上的 Alt + K 键，可以对录音中的音轨进行仅输入监控或自动输入监控。
*   按下键盘的 Ctrl + Shift + N 键，调出工作中的新轨道对话框。
*   当您录制或播放任何内容时，可以通过按下数字小键盘上的 Enter 键来将标记放在时间线部分。
*   切换编辑窗口模式可以通过按键盘的 F1–F4 功能键来启用，或者也可以按键盘的 Alt + 1 到 4 数字键。
*   按键盘上的 F5–F10 功能键或使用键盘上的 Ctrl + 1 到 6 数字键来切换编辑窗口工具。
*   合并剪辑选项可以通过按键盘的 Alt + Shift + 3 键来启用。

现在，让我带您通过高级水平的快捷方式命令，了解该软件的高级功能，以实现高效的工作输出。

### 高级别的快捷键

*   键盘的 Shift + Alt + ]键在该软件的工作窗口上启用导入会话数据选项。
*   键盘上的 Alt + A 组合键将重置缩放选项，以便整首歌曲能够适合屏幕。
*   打开速度操作窗口可以通过按键盘上的 Alt + 2(数字键盘按钮)键打开。
*   Ctrl + Alt + shift + B 键将用于反弹轨迹选项。
*   弹回磁盘可以通过按键盘的 Ctrl + Alt + B 键来完成。
*   对于半速播放，您可以按下键盘的 Shift +空格键。
*   半速录音可以通过按 Ctrl + Shift +空格键来完成，或者也可以按键盘的 Shift + F12 功能键。

这些都是关于 Pro tools 软件快捷键的可能信息，从基本命令到高级命令。您可以根据自己的工作要求，从这里指出一些快捷方式，使工作更加顺利。
