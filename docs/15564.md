## xlrd 与 openpyxl 的定义

Xlrd 和 openpyxl 都是在 python 中使用的。Openpyxl 试图在功能和性能之间取得平衡。通过使用 xlrd 和 openpyxl，我们将功能优先于优化。与其他库和应用程序相比，内存使用率相对较高。我们使用“pip install openpyxl”命令在我们的系统中安装 openpyxl 模块。要安装 xlrd，我们使用“pip install xlrd”命令在我们的系统上安装 xlrd 包。

### xlrd 与 openpyxl 的主要区别

*   Python xlrd 模块用于从 excel 文件中读取数据，而 python openpyxl 模块用于从 excel 文件中读取和写入数据。
*   我们正在使用 pip 命令安装 xlrd 模块，而 openpyxl 模块也是使用 pip 命令和源代码安装来安装的。
*   我们可以使用 openpyxl 来读取 xls 文件。但是不可能使用 python xlrd 读取 xls 文件。假设我们运行的 xlrd 版本是 1.2.0，那么就有可能读取 xls 文件，但是如果我们运行的是 xlrd 的更新版本，就不可能读取 xls 文件。
*   Python xlrd 用于从电子表格中检索信息；另外，python openpyxl 从电子表格中读取和写入信息。
*   Python openpyxl 使用 excel 格式，即 A1:B2，或者按列或行读取数据。Xlrd 只通过使用列和行来读取数据。在 xlrd 中不可能使用 excel 格式读取数据。
*   Python xlrd 允许我们对数据进行切片。Openpyxl 允许我们使用 excel 格式的范围来获取数据；此外，openpyxl 允许通过使用切片来存储数据。
*   通过使用 python xlrd，我们可以根据代码中使用的标准检索数据。此外，在 openpyxl 中，可以按照我们在代码中编写的代码检索数据。
*   从电子表格中提取和写入信息，Python xlrd 是非常有用和重要的。而 openpyxl 在 python 中从 excel 文件中读取和写入数据也很有用和重要。这两个模块都用于从 python 中的 excel 文件中检索数据。
