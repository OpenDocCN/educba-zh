## 函数求导的 MATLAB 介绍

MATLAB 包含各种命令和函数，有许多实用程序。这篇文章的重点是了解如何 MATLAB 命令' diff '可以用来计算一个函数的导数。MATLAB 中的 diff 命令用于计算符号导数。在其最简单的形式中，我们希望计算其导数的函数作为参数传递给 diff 命令。

**对微分或导数的理解**

<small>Hadoop、数据科学、统计学&其他</small>

微分是一种基本的微积分工具，它表示数量上相当小的变化。微分是函数随其中一个变量变化的速率。它计算输出值相对于其输入值变化的敏感度。

例如，一个物体的速度是该运动物体的位置对时间的导数。这里的速度导数显示了当时间移动时，物体的位置变化有多快。用数学术语可以表示为 dx，dy，dz 等。可以写成 f(x)/dy，其中 dy 将代表 f(x)相对于 x 的微小变化。

既然我们已经更新了微分的概念，现在让我们了解它是如何在 MATLAB 中计算的。

**导数的语法:**

1.  差异(f)
2.  diff (f，var)
3.  差分(f，n)
4.  diff (f、var、n)

### MATLAB 中函数求导的例子

现在我们将借助各种例子来理解上面的语法

#### 1.差异(f)

diff (f)将用 symvar (f，1)标识的变量来区分“f”

下面是一个使用 diff (f)计算函数微分的例子:

让我们将正弦函数定义为:

罪恶(x ^ 3)

**代码:**

`syms f(x)
f(x) = sin (x ^ 3);
df = diff (f, x)`

**输出:**

![Matlab derivative function- example 1](../Images/bc8f3e04d1cb530bedd5aaec8c15f14e.png)

<noscript><img class="alignnone size-full wp-image-309690" src="../Images/bc8f3e04d1cb530bedd5aaec8c15f14e.png" alt="Matlab derivative function- example 1" width="343" height="100" srcset="https://cdn.educba.com/academy/wp-content/uploads/2020/02/Matlab-derivative1.png 343w, https://cdn.educba.com/academy/wp-content/uploads/2020/02/Matlab-derivative1-300x87.png 300w" sizes="(max-width: 343px) 100vw, 343px" data-original-src="https://cdn.educba.com/academy/wp-content/uploads/2020/02/Matlab-derivative1.png"/></noscript>

该函数将返回函数 sin (x ^ 3)的微分值:

3 x^2 cos (x ^ 3)

#### 2\. diff (f，var)

diff (f，var)将区分作为参数传递的变量(在命令中称为“var”)。

下面是一个使用 diff (f，var)计算函数微分的例子:

让我们将一个正弦函数定义为:

罪恶(x * t ^ 4)

**代码:**

`syms x t
diff (sin (x* t ^ 4))`

**输出:**

![Matlab derivative function- example 2](../Images/f883a83f762d5dafcbbd13b300999e3c.png)

<noscript><img class="alignnone size-full wp-image-309692" src="../Images/f883a83f762d5dafcbbd13b300999e3c.png" alt="Matlab derivative function- example 2" width="215" height="109" data-original-src="https://cdn.educba.com/academy/wp-content/uploads/2020/02/Matlab-derivative2.png"/></noscript>

该函数将返回函数 sin (x * t ^ 4)的微分值:

`t ^ 4 cos(t ^ 4 x)`

正如我们所注意到的，这个函数是关于 t 't '的微分

#### 3.差分(f，n)

diff (f，n)将计算函数“f”对使用 symvar 确定的变量的 n 阶导数(如参数中传递的)。

下面是一个使用 diff (f，n)计算函数微分的例子:

让我们把一个函数定义为:

4t ^ 5

我们将计算我们函数的 3 次<sup>、4 次<sup>和 5 次<sup>导数。</sup></sup></sup>

**代码:**

`syms t
d3 = diff (4 * t ^ 5, 3)
d4 = diff (4 * t ^ 5, 4)
d5 = diff (4 * t ^ 5, 5)`

**输出:**

![Using diff (f, n)](../Images/acf7b126257ed1052f4093022078303a.png)

<noscript><img class="alignnone size-full wp-image-309696" src="../Images/acf7b126257ed1052f4093022078303a.png" alt="Using diff (f, n)" width="275" height="163" data-original-src="https://cdn.educba.com/academy/wp-content/uploads/2020/02/Matlab-derivative3.png"/></noscript>

该函数将返回函数 4t ^ 5 的 3 个 <sup>rd</sup> ，4 个<sup>th</sup>T6】5 个 <sup>th</sup> 导数，如下所示:

d3 = 240 吨^ 2
d4 = 480 吨
d5 = 480

#### 4\. diff (f、var、n)

diff(f，var，n)将计算给定函数“f”对参数中传递的变量的第 n 阶导数(在语法中称为“var”)。

下面是一个使用 diff (f，var，n)计算函数微分的例子:

让我们把一个函数定义为:

x * sin (x * t)

**代码:**

`syms x t
diff (x * sin(x * t), t, 3)`

**输出:**

![Using diff (f, var, n)](../Images/a6e815347aa428f1ad54f93792acceda.png)

<noscript><img class="alignnone size-full wp-image-309700" src="../Images/a6e815347aa428f1ad54f93792acceda.png" alt="Using diff (f, var, n)" width="209" height="112" data-original-src="https://cdn.educba.com/academy/wp-content/uploads/2020/02/Matlab-derivative4.png"/></noscript>

该函数将返回函数 x * sin (x * t)的 3 个 <sup>rd</sup> 导数，对 w.r.t 't '的微分如下:

-x^4 公司

正如我们所注意到的，我们的函数是对 w.r.t. 't '求导的，我们已经得到了 3 <sup>rd</sup> 导数(根据我们的论点)。因此，正如我们所了解的，在 MATLAB 中可以使用“diff”命令来计算函数的导数。我们还可以通过将‘n’(n 阶导数)作为参数来控制我们想要计算的导数的次数。
