### 如何登录 WordPress？

一旦 WordPress 被安装在网络服务器上，我们可以从不同的 URL 使用 WordPress 登录页面。登录有几种方式；让我们看看登录方式，

*   #### Log in to

    through direct link.

访问 URL 的初始方法如下所示:

website.com/wp-login.php 网站

此外，我们还可以通过

website.com/wp-admin 网站

*   #### Log in to

    through subdirectory.

通过子目录文件夹登录。为了将相同的 url 用于子目录，URL 将如下所示，

website.com/subdirectory/wp-login.php 网站

*   #### Log in

    through subdomain

与子目录一样，在网站 url 中添加子域，以访问 WordPress admin 登录页面。url 将如下所示，

Subdomain.demo.com/login

Subdomain.demo.com/wp-login.php

### 如何找到 WordPress 登录的网址？

第一步是登录 WordPress 网站，这保证你已经为安装在 WordPress 的主机支付了费用，并获得了管理员的详细信息，包括用户名和密码等登录凭证。安装 WordPress 时，我们可以通过各种方法登录网站，包括通过 web 主机仪表板登录和通过后端访问 WordPress 网站。让我们用如下简单的方法来看看后端通过站点的日志记录，

*   **浏览 WordPress 登录页面–**后端的 WordPress 登录将是域名，如结尾的/wp-admin。考虑一个例子，如果我们有一个 yourwebsite.com[的网站](http://www.yourwebsite.com)，那么 WordPress 的登录页面将是 www.yourwebsite.com/wp-admin。
*   **将书签网址包含到浏览器—**将 WordPress 登录页面包含到浏览器的书签栏，无需每次都输入登录网址。它有助于防止不正确的登录链接。每个浏览器都有自己想要的书签处理过程。
*   **在网站菜单上添加登录 URL**–另一种找到登录方法的方法是通过 WordPress 管理区将登录添加到 URL 菜单。它将直接出现在 WordPress 网站上。如下图所示，

仪表板会喜欢菜单

要在自定义链接中包含 url 并添加链接文本，如下所示，请将其添加到菜单中，然后单击保存菜单。

*   **通过输入 WordPress 凭证—**一旦我们在网站上看到登录页面，就可以输入用户名和密码等凭证了。当你注册一个虚拟主机账户并安装 WordPress 的时候，在设置你的用户名和密码的时候记下它。它将通过电子邮件发送登录凭证，因此如果您忘记了凭证，可以很容易地在电子邮件框中进行搜索。
*   **存储登录凭证–**有时，当凭证很复杂但密码安全时，很难记住。
*   **点击登录–**一旦我们输入登录凭证，只需点击登录按钮，也可以使用 WordPress 帐户登录。请记住，指定的 WordPress 仪表盘将根据我们的帐户和安装的插件而有所不同。当第一次登录时，在“开始”旁边会有一个欢迎信息，比如“欢迎使用 WordPress”。

**WordPress 登录仪表板和提示**

大多数完整的登录方法都可以访问 WordPress 仪表盘；一些功能有助于优化 WordPress 网站。它们如下:

*   **Posts**–用于将新的博客文章添加到 WordPress 网站。
*   这些页面包括添加和管理 WordPress 网站上的各种页面
*   **媒体**–媒体是所有媒体类型的集合，从视频和图像到音频文件。
*   **评论**–评论是用来管理你从 WordPress 网站访问者那里收到的评论。
*   **外观**–用于定制 WordPress 网站设计和功能。
*   **用户**–它用于控制访问你的 WordPress 网站的各种用户。
*   **插件**–它用于在使用 WordPress 插件时获得额外的工具和特性。
*   工具用于检查 WordPress 网站的信息，这是一种健康的方法，也用于导入和导出数据。
*   **设置**–用于改变仪表盘和 WordPress 网站的设置。
*   **访问 WordPress 仪表盘–**WordPress 仪表盘有各种各样的东西；让我们看看仪表板的基本元素，它们有助于设置和执行仪表板，如下所示:

定制站点:当点击“定制你的站点”按钮时，首先登录到 WordPress 仪表盘，它会重定向到另一个窗口。屏幕左侧允许定制网站的不同视觉元素，包括以下内容:

*   站点标识(Site identity)–这个站点标识是 WordPress 站点的名称、你的 favicon(一个出现在浏览器标签上的小图标)和你的标语。
*   **颜色**–颜色用于修改 WordPress 设计主题中与品牌匹配的任何颜色。
*   **主题选项–**仅在此选项中，我们用于为我们的网站选择主题。
*   Widgets 对 WordPress widgets 进行修改
*   **封面模板**–封面模板选择背景图像并定制不透明度、颜色等。
*   **主页设置**–在这里，我们可以从菜单列表中选择我们想要的主页。
*   **背景图片—**这些图片用于上传网站的背景图片。
*   **菜单**–我们在这里添加菜单时使用了该菜单。
