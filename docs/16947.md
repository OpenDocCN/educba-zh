## GIMP 魔棒工具简介

魔棒工具用于快速选择任何图像中相同颜色范围的区域，通过它，您可以从图像中剪切特定区域，或者通过更改该工具的参数以及使用该软件的其他选项来移除所选对象的背景，但是在 GIMP 中，您会发现该工具的名称为模糊选择工具。在本文中，我们将学习模糊选择工具和工作算法的不同用途，因为该工具对图像的任何区域以及其他一些参数有不同类型的选择方法。

### 如何使用魔棒工具？

您可以非常容易地使用这个工具，因为它有非常方便的参数和它们的值。所以让我们来分析一下。

### 魔术棒工具的使用

一旦你选择了这个工具，你将拥有一个类似光标的魔法棒。现在用这个工具点击你想要选择的区域，你会有一个和这个颜色范围相同的选择被激活。

![GIMP magic wand tool output 3](img/0fe3ee59e54c1c3a46c643d61cac2600.png)



*   **模糊选择工具的参数:**

现在，让我们了解该工具的每个重要参数，这些参数都显示在参数框中。

*   **选择模式:**

我们在选择中有四种模式，第一种是替换当前选择。

![GIMP magic wand tool output 4](img/11fc5ce67ab2688a5b98374175578f78.png)



启用此选项并点击图像的所需区域。我将点击这些花的花瓣，并有这个选择。

![GIMP magic wand tool output 5](img/c4fbdd7331668d18fc5e9aa81ce70938.png)



现在点击另一个区域，你会注意到之前的选择会自动删除。因此，使用此选项，对于每个新的选择，先前的选择将自动删除。

![GIMP magic wand tool output 6](img/fe38d9a5e81f450d60360a441a91864d.png)



下一个模式是添加到当前选择。现在再一次，像这样在花上选择一个区域。

![GIMP magic wand tool output 7](img/5e21f3c03b106bd47e538bb7b2a59ed5.png)



并再次点击以进行选择。在这里你会发现新的选择增加了前一个。因此，它将继续添加每个新的选择与前一个选择。我们通常在选择过程中使用这种模式。

![GIMP magic wand tool output 8](img/39c38219e6e51b7e8ecb9d340795e5b0.png)



下一步是从当前选择中减去。这有助于移除你不需要的选择。只需点击不想要的选择。为此，您可以多次单击。

![GIMP magic wand tool output 9.1](img/b5119b5efe66096366fcab81aa6a46bd.png)



![GIMP magic wand tool output 9.2](img/41d8b3313446a5492f2a68a4e70cb74f.png)



*   **绘制蒙版**

如果你想在点击区域之前看到选择区域，那么启用绘制蒙版选项。

![GIMP magic wand tool output 10](img/eeb66acc1253d587f5257781c01e9a00.png)



只需点击你想要的区域，并保持点击看到选择区域。它会像这样用洋红色高亮显示。

![GIMP magic wand tool output 11](img/b1114ce1134c1a970fa38351cc15bd26.png)



一旦你释放它，点击它，使你选择周围的地区。

![GIMP magic wand tool output 12](img/3a02e281af7e54b47270cf1be54ad423.png)



*   **阈值:**

阈值是该工具的一个非常重要的参数。在选择过程中，我们必须选择合适的阈值来进行正确的选择。

![GIMP magic wand tool output 13](img/14e9af1af7fdbd217e3c29fcc616e46b.png)



例如，增加阈值的值，并在这个花图像的背景上单击，您将有这种类型的选择。

![GIMP magic wand tool output 14](img/b2ebb8ba19d4d85f7c872e6d93b7201b.png)



现在减小阈值的值，再次进行选择，您会注意到通过改变阈值的值，选择已经改变。

![GIMP magic wand tool output 15](img/6c742f5c7dc537cae713045d13c73716.png)



所以像这样检查你选择的值。正确选择的阈值随着每幅图像而变化。

![GIMP magic wand tool output 16](img/29b9a006150de61e1caf6cbcd76f9fbe.png)



你可以看到这个选择在这个阈值上非常好。

![GIMP magic wand tool output 17](img/bffa74755bde37cb8ef249b006246c74.png)



现在让我们从图像上剪下这朵花。这个选择在背景上，因为我们点击了背景。现在反转这个选区，围绕这朵花进行选择。所以转到选择菜单，点击向下滚动列表的反转选项。

![GIMP magic wand tool output 18](img/446909ff84bea01626e83243124eeabf.png)



现在你可以注意到花周围的选择。你可以通过图片背景周围的黄色虚线来识别它。这条黄线表示图像的画布区域。

![GIMP magic wand tool output 19](img/08f14338cbaae1b1e10fb30cbbf96ba9.png)



现在点击创建新层按钮。

![GIMP magic wand tool output 20](img/9a1a1151d9479af130b6e37d080be53e.png)



在此对话框中选择透明度作为该层的填充，然后单击确定。

![GIMP magic wand tool output 21](img/fccfae3b705cd1c5669e34faf9e844a6.png)



现在来看看你原来的花层，复制你的选择。要复制它，进入编辑菜单，点击复制选项。

![GIMP magic wand tool output 22](img/cd199816a5ac210a005c772925eeb94a.png)

![GIMP magic wand tool output 24](img/12c04e1dc5b659433a67094c462ffa6b.png)
