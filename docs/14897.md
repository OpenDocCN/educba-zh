## JavaScript 和 VBScript 的区别

VBScript 是由微软在 1995 年开发的。VBScript 有更多的通用属性，使用非常局限于 Excel。语言不区分大小写，语句必须写在单独的行中。为了创建一个变量，语言中需要两条语句。JavaScript 在使用中更加通用，不区分大小写。变量创建很容易，只需要一条语句就可以完成。JavaScript 更容易使用，每个语句都用分号隔开。JavaScript 比 VBScript 更复杂。两种语言中都使用了几个语言运算符。

### JavaScript 是什么？

JavaScript 是一种面向对象的脚本语言，允许您在网页中创建具有交互式效果的动态 HTML 页面。在谷歌 chrome 发布其 JavaScript 引擎“node.js”之前，浏览器中一直有 JavaScript。JavaScript 允许您在浏览器中处理和维护数据。我们可以在 node.js 的帮助下跨全栈开发使用 JavaScript，帮助前端开发者处理后端逻辑和应用编程接口，无需学习新的编程语言。Brendan Eich 在网景公司创建了 JavaScript，它于 1995 年问世。它只能在网络浏览器上运行，并且是一种解释型编程语言。JavaScript 允许客户端脚本与用户交互，异步通信，控制浏览器和改变网页内容，这些都可以通过 JavaScript 实现。

### VBScript 是什么？

VBScript 是在 visual basic 上开发的通用脚本语言，由微软开发。它也被称为活动脚本语言。它是一种 visual basic edition 脚本语言，因为它是基于 visual basic 开发(建模)的，是一种具有快速解释器的轻量级脚本语言。只有浏览器 Internet Explorer 支持 VBScript，它主要用于 Microsoft 环境和 visual basic 语言产品中。默认情况下，它将安装在每个 windows 桌面版本中。VBScript 可以通过 windows 脚本宿主执行，也可以通过 Microsoft script 控件嵌入。VBScript 使用公共对象模型来访问环境变量；用于读、写、更新、删除操作的文件系统对象，以及通用脚本语言。VBScript 主要被系统管理员使用，测试自动化脚本语言用于不同的工具，如 QTP。VBScript 是一种容易学习的脚本语言。

### JavaScript 和 VBScript 的主要区别

两者都是市场上的热门选择；让我们讨论一些主要的区别:

1.  JavaScript 不是真正的面向对象的脚本语言，因为它不支持继承概念、子程序。尽管如此，它支持对象的使用，为后续的对象创建定义类。相比之下，VBScript 也不是真正的面向对象的脚本语言，因为它不支持继承、对象使用和类使用，但支持可重用的函数和子例程。
2.  JavaScript 是大多数浏览器的默认脚本语言，而 VBScript 不是默认的脚本语言，我们需要将它作为脚本语言提及。
3.  JavaScript 是区分大小写的脚本语言，而 VBScript 不是区分大小写的脚本语言。
4.  JavaScript 语法类似于 C 编程语言，而 VBScript 语法类似于 visual basic，因为它是 Visual Basic 的一部分，并且遵循 Visual Basic 的语法。
5.  JavaScript 用作客户端脚本语言，而 VBScript 既可以用作服务器端脚本语言，也可以用作客户端脚本语言。
6.  JavaScript 对不同的操作使用相同的操作符，而 VBScript 对不同的操作使用不同的操作符。
