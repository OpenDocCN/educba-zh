## InDesign 段落样式介绍

InDesign 段落样式可以理解为对文本内容的段落样式进行参数调整，我们在该软件中单独提供了一个完整的段落样式面板。这个面板有许多不同类型的功能，可以帮助我们非常快速地完成排版工作，并且在对段落面板进行设置后，我们不需要将我们想要的样式应用到段落或文本内容的每个部分。它对每种类型的文本段落内容的几个样式层起作用，并帮助我们理解段落样式的不同属性。所以今天我们要从这一段的风格面板中学习一些有效的技巧。

### 段落样式在 InDesign 中如何工作？

在 InDesign 中使用段落样式是一个非常好的功能，在我们使用它之后，我们可以非常轻松地以专业的方式组织文本内容。所以让我给你解释一下这个软件的段落样式面板。为此，我将首先创建一个新的文档页面，为此，我将单击 InDesign 欢迎屏幕上的“新建”按钮。

<small>3D 动画、建模、仿真、游戏开发&其他</small>

![InDesign paragraph styles Output 1](img/81d4121b472fbe7ac76de82972723053.png)



我将从这个新文档对话框中获取文档页面的默认自定义大小，然后单击“创建”按钮。

![InDesign paragraph styles Output 2](img/4238cc2a9734ec459e60c6ee4f3be954.png)



现在我将从工具面板中选择文字工具，或者你可以按 T 键作为快捷键。

![InDesign paragraph styles Output 3](img/840d48cd4d6d997b825aeaf13ba92ffe.png)



现在，我将从工作区类型中选择排版选项，将工作区更改为排版模式。我将切换到这种模式，以便在工作屏幕的顶部显示所有需要的文本格式面板。你可以在任何你觉得舒适的工作空间工作。

![InDesign paragraph styles Output 4](img/c0b5a1c4dc8229ff4efbc2b63b1e50d4.png)



我将在绘制的文本框中粘贴一些文本内容。

![InDesign paragraph styles Output 5](img/8b52d8661601a03b4aad0f779c46f64a.png)



您也可以转到文件菜单下拉列表中的放置选项，将文本放置到文本框中，放置选项的快捷键是键盘的 Ctrl + D 按钮。

![InDesign paragraph styles Output 6](img/409d07fc914c83555c85d260afcfef8a.png)



在此排版工作区布局中，工作屏幕的右侧会有一个段落样式框。

![InDesign paragraph styles Output 7](img/c9fa0b0aec4bb9a22610f5561de047d1.png)



如果您的工作屏幕中没有这样的面板，那么您可以从菜单栏的类型菜单的下拉列表中选择它，或者您可以简单地按键盘的 F11 功能键在工作屏幕上选择它。

![InDesign paragraph styles Output 8](img/218d14c1a5d8fd811c615e21c4d420bb.png)



在这个工作屏幕的顶部，您会发现字符格式控制。

![InDesign paragraph styles Output 9](img/5a424b8fa2bce4c2420d7b8bac2056e8.png)



在它的正下方，有段落格式控件，我们将在使用段落样式面板的过程中使用这两个控件。

![InDesign paragraph styles Output 10](img/081afdf5ce1b41f142064bfce0e201fe.png)



现在，我将选择此内容的第一行作为此文本内容的标题，并从字符格式栏的字体大小选项中增加其大小。

![InDesign paragraph styles Output 11](img/67ca54276efbc4711febb26e2cfecac7.png)



我也将从这个列表中改变它的字体样式。

![InDesign paragraph styles Output 12](img/f8d5dbf344daa0ce98f8ea8f18e2a597.png)



现在我将把这个颜色文本框的颜色改为绿色。

![InDesign paragraph styles Output 13](img/1d167fd75cd8060a8d821f15ebcd4403.png)



现在来到段落样式框，点击这个框底部的创建新样式按钮。确保文本被选中，你不需要选择整段文字，只要文本工具的闪烁光标在那里就可以了。

![InDesign paragraph styles Output 14](img/3d16790bf43a98119a71629294c281e9.png)



现在，我将双击这个层的名称，并命名为标题。

![InDesign paragraph styles Output 15](img/73d0a3a365d6137dd486880bc6024637.png)



当你双击这个样式层时，它会显示你选择的段落当前的格式。

![InDesign paragraph styles Output 16](img/9688ec858e317e8c797d7b4409b8c224.png)



现在，这个段落样式面板的神奇之处在于，如果你想对文本内容的其他标题应用相同的格式，只需点击一下就可以做到。所以选择你想用相同格式做标题的段落，或者你可以把闪烁的光标放在那里。

![InDesign paragraph styles Output 17](img/9efaa9ec667a8a8f167aa1cfd72db3c1.png)



并在段落样式框中点击前一个标题的图层样式。在我的例子中，我将点击标题层，它将应用所有相同的格式与选定的段落。

![InDesign paragraph styles Output 18](img/eefac3a34102ebb4ca6d23765874e624.png)



同样，我们可以制作一个副标题层，并将其应用于两个或更多的副标题。

![InDesign paragraph styles Output 19](img/63c538c88c66a3b5b4ac64f40df86038.png)



现在，我想在这个文本内容的标题中做一些其他的格式化，让我们看看如何将这个改变的格式应用到所有相同类型的标题中？我会选择这一段。

![InDesign paragraph styles Output 20](img/6063c328c8a42724893ba07fb3570c62.png)



并将其颜色变为红色。

![InDesign paragraph styles Output 21](img/398e3bc6cc31004b0e86ad2bf6b36bb4.png)



您可以看到，当我们对某个特定的段落样式进行任何更改时，其图层中会出现一个加号，表示该特定图层有一些其他格式。

![InDesign paragraph styles Output 22](img/0c7b7818a92feaa7a4ed54b11716342c.png)



要将此应用于所有相同类型的标题，只需右键单击该层，然后单击重新定义样式选项。

![InDesign paragraph styles Output 23](img/6a1574a8a8208112c742f1b64460f18d.png)



所有相同类型的标题都会随着格式的改变而自动更新。

![InDesign paragraph styles Output 24](img/b6f0b0c0eeac810218074eab9c7932da.png)



让我们做一个正文样式，为此，我将选择这一段。

![Output 25](img/adab33303d116642f487737b43da2080.png)



并更改其字体样式。

![Output 26](img/001eb0e715618f9f390296b037646f02.png)



并为它创建一个新的样式层。

![Output 27](img/1d591f2aa19d9cfa92b249aeac153897.png)



我将把它命名为身体复制。

![Output 28](img/eba8a0d2f1fd4524222a5bf4a258ad1a.png)



并将其应用到另一个段落，就像我们在标题情况下所做的一样。

![Output 29](img/da36d8e0bc8258030057760f8c73c2ec.png)



并且它根据正文的格式而变化。

![Output 30](img/e5591956b026e1eb8d7536463f75c4a6.png)



但是当我将它应用于段落的项目符号时，您可以看到它只更改了一行并删除了项目符号，因为我们的正文中没有项目符号样式，它只更改了所有项目符号段落中的一行，因为一行项目符号本身就是整个段落。

![Output 31](img/15de3b75dbd4bf6244fc989ad1eaff83.png)



您可以通过从“类型”菜单的下拉列表中启用“显示隐藏字符”选项来检查段落，或者您可以简单地按 Ctrl + Atl + I 作为其快捷方式。

![Output 32](img/b394c3b162830ac526cd439e7db81d1b.png)



这个符号表示段落到此结束。

![Output 33](img/0286fa44a92c74da36d935e5eb197cb1.png)



在段落样式框的帮助下，我们可以像这样对文本内容进行有效的格式设置。

### 结论

我相信现在您可以开始使用段落样式框在 InDesign 项目中制作不同类型的段落格式了。有了段落样式面板，你可以在更少的时间里做更多的工作，也可以摆脱一次又一次地对每个段落应用你想要的格式。
