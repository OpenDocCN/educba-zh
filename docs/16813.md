## Inkscape 弯曲文本简介

Inkscape curved text 是一种样式，通过该样式，您可以将任何文本设置在曲线路径上，以便用它进行有效的排印安排，并且为了获得曲线文本，我们将使用 Inkscape 软件的许多工具和功能，我们将在该工具的用户界面上找到这些工具和功能。我们还可以编辑曲线文本来改变曲线路径形状。因此，今天我们将通过两种方式来解释 Inkscpace 软件的这一技术，并根据我们的需要进行编辑。所以让我们开始讨论这个话题。

### 如何在 Inkscape 中创建弯曲文本？

在 Inkscape 中创建弯曲的文本并不是什么大不了的事情，我们只是很好地掌握了正确的方法。你要做的第一件事是从用户界面左侧的工具面板中选择一个圆或椭圆工具，来制作一个圆，作为文本的曲线路径。您可以按 F5 作为该工具的快捷键，或者通过单击其图标直接从工具面板中取出。

![Inkscape curved text output 1](img/9dc59b4da3b3c38f75d8db6187b3f638.png)



现在按住 shift 和 Ctrl 键来画一个圆，就像这样。你可以画一个椭圆，让你的文字有弯曲的路径。这完全取决于你想走哪条路。

![Inkscape curved text output 2](img/c2e748f722fd5287da56235ce19f2238.png)



现在再次导航到工具面板，从工具列表中选择文本工具，或者点击键盘上的 F8 激活这个工具。

![Inkscape curved text output 3](img/59d49caa41f0aa4595e15a51f94d75a4.png)



现在为你的文本选择文本样式，然后在菜单栏下面的文本工具的属性面板中设置文本的大小。我将输入 eduCBA 作为我的文本，这是我们的正式名称。您可以键入任何其他文本。

![Inkscape curved text output 4](img/8214d797e27babd65bd303460f0155c8.png)



现在从工具面板中选择工具，或者按键盘上的 F1 键快速切换到这个工具。

![Inkscape curved text output 5](img/2b4af37f1cc0eb20bd7e31a713245075.png)



按住键盘的 Shift 键，然后通过逐个点击来选择文本和圆。现在转到菜单栏的文本菜单，它位于该工具用户界面的顶部，单击它。点击此菜单下拉列表中的“放入路径”选项。

![Inkscape curved text output 6](img/7794836b31a82ab4a3c777feb92ecf0c.png)



现在你选择的文字会像这样环绕这个圆。

![Inkscape curved text output 7](img/908f5be5b8edfa1d209d4af75022261d.png)



点击这个选择，你将有围绕这个选择的结合框旋转手柄。按住鼠标点击丁波盒子的旋转箭头。

![Inkscape curved text output 8](img/c83c3bbce865e51bf778161c574828a0.png)



然后旋转此选项，将文本在圆形路径上移动到您想要的位置。如果你想删除这种扭曲的文本，然后再次进入文本菜单，点击“删除”从路径选项的滚动列表。

![Inkscape curved text output 9](img/80027d648e55c5ef522c4b7ba56c710a.png)



文本将通过在其位置上留下圆形路径来恢复其原始形状。

![Inkscape curved text output 10](img/3d48b60fb211898a9ff8c044bfbf021d.png)



或者你可以用选择工具选择这个圆，然后按键盘上的删除键来释放扭曲。

![Inkscape curved text output 11.1](img/5316cadfd2b533df10a7ce2974fb8eee.png)



![Inkscape curved text output 11.2](img/8e6e666263e855748917c8c27e1955f8.png)



现在让我们移除这个圆的填充，为了理解弯曲文本的下一步，进入菜单栏的对象菜单，然后点击向下滚动列表的填充和描边选项。

![Inkscape curved text output 12](img/d97708f1b529b99e2daf6ff96242b568.png)



填充和描边面板将在用户界面的右侧打开。现在选择填充表格，点击十字按钮来移除这个圆的填充，然后点击描边标签，选择平面颜色作为这个圆的描边。

![Inkscape curved text output 13](img/4b4ff669be2dfc3b4ed27169ecd83a24.png)



用和上面步骤一样的方法选择文字和圆圈。

![Inkscape curved text output 14](img/591195ae93a4d8eb5c1d2253058e8e43.png)



在选择工具的帮助下，旋转这个圆底部的文本。

![Inkscape curved text output 15](img/b1db7fe37558de55fcc013b09d6ad84f.png)



现在点击属性栏的“水平翻转所选对象”选项或按 H 作为快捷键。

![output 16](img/5f9e80d5a9d960d79ebd49def255241f.png)



我们的文本将进入这个圆圈。

![output 17](img/3bc204965e55b52695d3f91799a5961b.png)



现在转到菜单栏的路径菜单，点击“对象到路径”选项，向下滚动列表。

![output 18](img/cad743f0874436a0954c1a47df391a5c.png)



现在你可以把你的文字从这个圆圈中分离出来了。

![output 19](img/ec5363a6fdc03900471e06ce8299480d.png)



而你的弯文就这样准备好了。

![output 20](img/808ab429650d7370d3e2dc5e046cc4b2.png)



现在让我们看看做这件事的另一种方法。对于这种方法，从工具面板中选择贝塞尔工具，画出你想要的曲线路径。我会画一条像这样的曲线路径。

![output 21](img/02cb7d5c41e2928e6abfc87ce55bd099.png)



现在再次采取文字工具，键入一个文本。这次我将键入一个句子，并通过选择工具选择文本和路径。

![output 22](img/aaaf2706458e0024c0eb4945f8e6e59c.png)



并通过相同的选项，为这个文本分配路径，我们使用了上述弯曲的文本。

![output 23](img/8b008bbf67f7faddbdcff09e26d85eb3.png)



你可以看到我们的文本不适合这个路径。您可以增加绑定框的面积，以容纳您的全文，因此，为此，单击此绑定框的箭头，并拖动它以增加框的大小。在可以看到整个文本的地方增加文本框的大小。

![output 24](img/edefdb5f8a88d8ec3a58820f76386809.png)



您也可以通过旋转此绑定框来旋转此文本。

![output 25](img/3b39164b109d2176030d22f71e731241.png)



你也可以编辑这个弯曲的路径，从工具面板中选择编辑路径工具或者按 F2 作为快捷键。

![output 26](img/ffe12a9871219cc8c6dd1353cc9d837e.png)



并从这条曲线路径的锚点编辑它。

![output 27](img/d4bdea0c840009efd74dac6c2d98cb86.png)



现在选择文本和曲线路径，然后进入路径菜单，点击下拉列表中的“对象到路径”选项，或者你可以使用快捷键 Shift + Ctrl + C。

![output 28](img/7bd1262d6dcbf3cae5e3f32d41a0502b.png)



现在通过移动将文本从路径中分离出来。

![output 29](img/55cdb2c67cfaba63a58e7f9f616353b7.png)



现在你可以删除这个路径，我们的两个完整的弯曲文本就准备好了。你可以在你的任何一个设计作品中使用它们。

![output 30](img/6350921c41c0520e88aa1398e4ab7004.png)



这是在 Inkscape 中创建曲线路径的两种方法。
