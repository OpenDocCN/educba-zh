## HTML 中颜色名称介绍

HTML 中的颜色名称在网页和应用程序设计过程中起着非常重要的作用。它有助于改变我们的网站或应用程序的外观和感觉，也使我们的应用程序更具吸引力和互动性。有多种方法可以将 HTML 中的颜色名称定义为 HTML 编码格式。可以使用[附加的 CSS 属性](https://www.educba.com/css-font-properties/)来更有效地使用它。我们能够在页面级别使用 HTML 颜色名称，方法是将它包含到正文部分，还能够为特定元素设置单独的颜色。HTML 中的颜色名称专门用于文本颜色、背景颜色、边框颜色、阴影、渐变等等。

### HTML 中颜色名称的类型

在 HTML 中有不同的方法来定义颜色名称。很难记住不同颜色的不同名字。所以要决定给不同的颜色赋予一些值。我们一个一个来看。

![Types of the Color Name in HTML](../Images/40489f443aacc7a1c2256fc6b28dc402.png)

<noscript><img class="alignnone size-full wp-image-246158" src="../Images/40489f443aacc7a1c2256fc6b28dc402.png" alt="Types of the Color Name in HTML" width="535" height="491" srcset="https://cdn.educba.com/academy/wp-content/uploads/2019/11/Types-of-the-Color-Name-in-HTML.png 535w, https://cdn.educba.com/academy/wp-content/uploads/2019/11/Types-of-the-Color-Name-in-HTML-300x275.png 300w" sizes="(max-width: 535px) 100vw, 535px" data-original-src="https://cdn.educba.com/academy/wp-content/uploads/2019/11/Types-of-the-Color-Name-in-HTML.png"/></noscript>

#### 1.RGB 值

*   在红、绿、蓝颜色格式值的组合中定义 HTML 颜色名称是其中一种方式。哪个用于定义数字 0 到 255 之间的颜色强度？
*   如果我们想用 RGB 组合定义一个黑色，它会喜欢 RGB(0，0，0)。这里所有的颜色参数将设置为 0。所以它会显示黑色。
*   像另一个 RGB(0，0，255)将显示蓝色，由于 255 被设置为蓝色的最高值而另一个为 0。
*   如果我们要 3 个相等的 RGB 值，那么它将产生灰色的阴影。

**示例:** RGB (0，0，0)表示最暗的颜色，RGB(255，255，255)表示最亮的颜色。

#### 2.十六进制值

*   这是 HTML 中最常用的颜色名称格式之一，其中颜色名称将以十六进制值的格式定义。
*   #rrggbb 格式包括红色的 RR 值、绿色的 gg 值和蓝色的 bb 值，这些值位于 00 toff 十六进制值之间。
*   让我们考虑一个定义颜色#00ff00 的例子。这些值将生成绿色，因为绿色包含最高值(ff ),而其他值设置为 00。

#### 3.HSL 值

*   另一种以色调、饱和度、亮度的形式定义 HTML 颜色的格式。
*   现在我们将逐一查看所有术语。色调被定义为 0 到 360 之间的颜色程度。红色为 0，绿色为 120，蓝色为 240。
*   饱和度将颜色值定义为百分比，其中 0%表示灰色阴影，100%表示全色。
*   亮度也是百分比，0%定义为黑色，50%定义为既不亮也不暗，100%定义为白色。

#### 4.RGBA 值

*   这种类型的 HTML 颜色定义了 RGB 颜色的扩展，包括 alpha 值，其中 alpha 用于定义颜色的不透明度。
*   Alpha 值介于 0.0(完全透明)和 1.0(完全不透明)之间
*   例如 rgba(255，99，71，0)

#### 5.HSLA 值

*   它的工作原理与 HSL 颜色值附加 alpha 值相同，用于定义颜色的不透明度。
*   hsla(8，100%，80%，0)为全透明色。
*   hsla(8，100%，80%，0.9)不那么透明的颜色。

### 使用 HTML 颜色名称

在你的网页上实现颜色很容易，既可以内嵌也可以通过单独的 CSS 样式表实现。

当你打算在一个页面上多次使用相同的 HTML 颜色名称时，最好使用 [CSS 工作表来定义和设置颜色](https://www.educba.com/css-text-formatting/)。相反，如果你只需要使用一次或两次颜色，使用内嵌方法被认为是更好的。

#### 1.使用 HTML 颜色内嵌

```
<body>
<h3 style="color: Brown">Brown H3 Heading</h3>
<p style="color: Gainsboro">Gainsboro colored paragraph text</p>
</body>
```

#### 2.在 CSS 中实现

```
h1 { color: Brown; }
p { color: Gainsboro; }
```
