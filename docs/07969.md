## 道德黑客软件简介

就道德黑客而言，软件意味着获得对系统或计算机中数据的未经授权的访问。实施网络黑客行为的人被称为黑客。下面提到了三种不同类型的道德黑客软件:

1.  白帽黑客
2.  灰帽黑客
3.  黑帽黑客

**白帽黑客**是非恶意破坏安全的人。可能是为了测试自己的安全系统。这些人会发现道德黑客软件或系统中可能存在的漏洞，并报告给他们以修复这些漏洞。他们也被称为“道德黑客”。例如，公司雇佣的认证道德黑客(CEH)来寻找道德黑客软件中的缺陷。而那些经过认证的道德黑客都是拥有 CEH 认证或者道德黑客认证的，这是通过清除 CEH 考试获得的。

<small>网页开发、编程语言、软件测试&其他</small>

**黑帽黑客**是那些出于恶意或个人利益而破坏系统安全或道德的黑客软件的人。这些人通常组成非法黑客团体，并试图闯入任何安全网络，以破坏、修改甚至窃取机密数据，如信用卡资料、银行资料等。他们也被称为“饼干”。

**灰帽黑客**人介于白帽和黑帽黑客之间。这些人调查系统，发现漏洞或任何安全缺陷，并向管理员报告。有时灰帽黑客向全世界而不是一群人报告这些安全缺陷。在某些时候，他们可能会提出收费纠正缺陷。这些人进行黑客攻击，没有任何私利。

所以有各种各样的道德黑客项目培训或 CEH 培训来教授这些类型的道德黑客。

### 什么是道德黑客软件？

让我们详细看看什么是黑客软件:

#### 形容词（adjective 的缩写）网络伦理

网络道德是使用互联网的行为准则。我们已经看到了黑客和他们的类型是什么。现在，我们将研究与网络道德相关的其他术语。

*   Phreaker: 这是一个被创造出来的术语，用来描述一种亚文化的活动，他们研究、试验或探索电话连接网络。由于电话网络已经计算机化，Phreaking 与计算机的道德黑客基础密切相关。它有时被称为 H/P 文化，其中 H 代表 Hacking，P 代表 Phreaking。

*   脚本小子:这些人使用他人开发的脚本或道德黑客基础知识来攻击计算机系统和网络。它也被称为脚本兔子，斯基迪，或脚本运行少年(SRJ)。

*   黑客行动主义者:术语“黑客行动主义者”是两个词“黑客”和“积极分子”的组合。这些人出于政治原因进行诸如丑化网站之类的道德黑客活动。他们的活动包括政治观点和问题。

#### B.情报收集

就道德黑客基础和调查而言，信息收集是初始过程。这涉及到一些过程，例如使用某个已定义的过程来分析任何组织、系统、服务器或个人。这通常被攻击者和/或调查机构用来获得关于受害者的附加信息。

有不同的方法可以收集道德黑客信息。其中一些列举如下:

**1。搜索引擎的使用**

一般认为搜索引擎会提供受害者的某些信息。关于使用互联网的道德黑客基本原则是“一个人在网上冲浪时会在任何地方留下足迹/信息。”

攻击者和黑客都使用这一原理。攻击者将收集关于系统、现有系统中的任何漏洞以及利用它的可能方式的信息。调查人员将收集信息，例如攻击者访问系统所使用的方法。最强大的搜索引擎是谷歌、雅虎搜索、MSN live 搜索、AOL 搜索、Ask 搜索。

**2。使用关系搜索引擎**

关系搜索引擎不同于普通的搜索引擎。它从不同的搜索引擎获得结果，并在这些结果之间建立联系。

**一、Whois 查找:** WHOIS 发音为“WHOIS”，是一种查询协议，广泛用于查询官方数据库，以确定域名所有者、IP 地址等详细信息。

**二。Maltego:** 这是一个开源的情报和取证应用程序，允许挖掘和收集信息，并以有意义的方式表示这些信息。图形库允许你识别信息之间的关键关系。

**三世。反向 IP 映射:**此方法用于查找托管您的软件/功能的同一服务器上托管的多个网站

**四。TraceRoute:** 它给出了有用的信息，比如你的计算机和远程计算机之间的服务器数量。这对于调查以及不同类型的攻击都很有用。您可以使用 NeoTrace 查看您的系统和攻击者系统之间的路由，它提供了攻击者和受害者之间所有节点的地图视图或节点视图。

**v .电子邮件蜘蛛:**这些是自动化的道德黑客程序，它们使用蜘蛛捕获电子邮件 id，并将其存储在数据库中。垃圾邮件发送者(向大量人发送垃圾邮件的人)正在使用电子邮件蜘蛛收集一千封电子邮件来发送垃圾邮件。

#### 碳（carbon 的缩写）扫描

扫描是找出远程系统、服务器和网络中任何打开或关闭的端口、任何漏洞的过程。它有助于获取受害者的详细信息，如 IP 地址，

使用的操作系统以及在远程计算机上运行的服务。

有三种不同类型的扫描。它们是端口扫描、[网络扫描](https://www.educba.com/network-scanning-tools/)和漏洞扫描。

端口扫描是黑客在道德黑客程序中最常用的方法，用来寻找系统中的漏洞。所有连接到 LAN(局域网)或使用调制解调器连接到互联网的系统都运行许多道德黑客服务，这些服务监听众所周知和不为人知的端口。计算机上总共有 1 到 65535 个可用端口。端口扫描允许攻击者找到哪些端口可用。

开放扫描(也称为 TCP 扫描)通常用于编程套接字。这是一种非常古老的道德黑客技术，在与服务器建立完全连接时更加正确。为了建立连接，它使用三个数据包进行身份验证。这种身份验证机制被称为三次握手。

对于*打开端口*:

*客户端—>SYN—>*

 **客户端—>确认—>*

对于*关闭端口*:

*客户端—>SYN—>*

*<–RST<–服务器*

这种开放式扫描的优点是易于编程。但是，它的局限性是很容易检测到每个连接并记录日志。

TCP 连接() :

*   由道德黑客操作系统提供的 connect()系统调用用于打开到机器上每个感兴趣的端口的连接。
*   如果端口正在监听，connect()将会成功；否则，该端口不可访问。

同步扫描:

*   这种扫描技术被称为半开扫描，因为 TCP(传输控制协议)连接没有完成。
*   SYN 数据包被发送到远程计算。
*   目标主机响应一个 SYN+ACK 数据包，表示端口正在侦听，RST 表示端口不侦听

零扫描:

*   空扫描不使用 TCP 报头的标志，它发送到目标主机
*   关闭的端口用 RST 回复数据包
*   开放端口忽略数据包

NMAP(港口扫描仪):

*   [NMAP 是一个强大的实用工具](https://www.educba.com/what-is-nmap/)来扫描大量合乎道德的黑客工具
*   它有 GUI(图形用户界面)和命令行界面
*   它受到多种操作系统的支持，如 Linux、Mac OS、Windows
*   它可以进行同步扫描，手指扫描，隐形扫描，半开扫描，以及许多其他类型。

#### D.病毒、蠕虫、特洛伊木马和病毒分析

[病毒(特别代表被围困的重要信息资源)](https://www.educba.com/viruses-function/ "Viruses and Trojans: We Live in Your Wires")是一种应用程序或一段代码，它通过将其代码注入其他数据文件或道德黑客程序来复制自身，并产生有害影响，如破坏系统或破坏数据。

蠕虫是一种独立的恶意软件计算机程序，它通过道德黑客基础进行自我复制。与病毒相比，它不需要将自己附加到现有的程序中。

特洛伊木马(在计算环境中)是任何恶意的计算机程序，它将自己伪装成有用的，以诱使受害者在他/她的系统上安装它。

这些病毒、蠕虫或特洛伊木马有几种不同的传播方式。其中一些列举如下:

1.  互联网中继聊天
2.  ICQ(我找你-即时消息程序)
3.  电子邮件附件
4.  物理访问(例如连接受感染的 USD 驱动器或硬盘)
5.  受感染的浏览器
6.  在线广告(比如声称你中了彩票的横幅)
7.  网络基本输入输出系统(Network Basic Input / Output System)

### 病毒运行阶段

大多数病毒分两个阶段运行，即感染阶段和攻击阶段。

1.  **感染期**

*   在这个阶段，病毒决定何时感染哪些程序
*   有些病毒会在系统上安装病毒文件后立即感染系统
*   一些病毒会在特定的日期、时间或特定的事件感染系统
*   TSR 病毒被加载到内存中，然后感染系统

2.  **攻击阶段**

*   在此阶段，病毒将删除文件，将自身复制到其他系统，并且只破坏目标。

### 病毒感染系统的症状

病毒感染系统的症状如下。

*   文件/文件夹的名称不同于正常名称(例如，文件名为%$#%%)
*   文件扩展名也可以更改。
*   该程序的加载时间比正常情况下要长
*   系统硬盘不断耗尽可用空间
*   您将无法打开某些程序
*   程序无缘无故地被破坏
*   系统运行非常缓慢，有时会意外重启

### 病毒类型

–宏病毒:传播并感染数据库文件

–文件病毒:感染可执行文件

–源代码病毒:影响和破坏源代码

–网络病毒:通过网络元素和协议传播

–引导病毒:感染引导扇区和记录

–外壳病毒:在目标主机的正版程序周围形成外壳，并将其作为子例程托管

–终止病毒:在工作会话期间永久保留在内存中

### 避免在道德黑客软件中检测到病毒的方法

下面我们列出了一些有用的方法来避免病毒的检测。

*   #### Keep the same "last modified" date.

I .为了避免被反病毒软件和用户发现，一些病毒使用不同类型的欺骗手段，例如它们与其他正版文件或文件夹具有相同的最后修改日期。

二。有一些病毒(尤其是在 MS-DOS 系统上)确保当病毒感染文件时，宿主文件的“最后修改”日期保持不变。

*   #### Avoid bait file

一、诱饵文件(或称山羊文件)是杀毒软件或反病毒专业人员自己专门创建的被病毒感染的文件。

二。有许多反病毒程序使用诱饵文件对自己的代码进行完整性检查。

三。因此，感染此类程序会增加病毒被检测到的机会。

*   #### Killing activities of anti-virus software

有一些病毒通过杀死与反病毒软件相关联的任务来避免被反病毒软件检测到。

*   #### Making invisible virus

有一些强大的病毒通过拦截反病毒软件的请求来欺骗反病毒软件。

二。然后，病毒可以将文件的未感染版本返回给反病毒软件，这样它就认为该文件是“干净的”。
