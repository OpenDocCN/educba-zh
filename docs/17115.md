## Maya 3D 模型简介

3d 模型是分别用 x、y 和 z 轴创建的三维对象。Autodesk Maya 是一个流行的三维建模和动画程序。用户可以创建非凡的 3D 模型，例如人物模型、建筑模型以及基本的 3D 对象。

Autodesk Maya 支持许多插件和功能，可帮助用户创建简单的 3d 模型和对象。

<small>3D 动画、建模、仿真、游戏开发&其他</small>

### 如何在 Maya 中创建 3D 模型？

在 Autodesk Maya 中，动画师和建模师通常在程序中创建基本模型或导入复杂的 3d 模型。由于 Autodesk Maya 主要用于平滑动画和渲染，因此用户可以在 Autodesk Max、AutoCAD 或 Blender 等软件上创建 3D 模型。

但是有些用户也使用 Autodesk Maya 进行 3D 建模和动画制作。要在 Maya 中创建 3d 模型，用户可以使用三种基本功能或工具。

**1。NURBS** : NURBS(非均匀有理基础样条)3D 建模基于创建平滑对象，并用于借助贝塞尔工具创建拱形表面。NURBS 聚合物建模通常用于创建内部和外部设计，也用于机械工程中设计汽车 3d 模型。

### 如何在 Autodesk Maya 中使用 NURBS 创建 3D 模型？

*   安装并打开 Autodesk Maya

![Maya 3d models output 3](img/ababc3e832d4fbc488fd7d7a84f9d200.png)



*   单击您想要创建的对象。
*   现在，例如，我们想要一个球体，所以点击它

![Maya 3d models output 4](img/b0499af9523ea21844b84ace033fd14b.png)

**2。多边形**:创建任何 3D 模型的最基本和第一步是借助多边形建模。多边形建模基于更简单的数学计算，包括边、顶点和面。用户可以使用具有该建模功能的形状来创建简单的模型。

在修改或编辑对象时，这是建模的最佳选择。这种建模的唯一缺点是它只能处理粗糙或坚硬的边缘。对于窒息和弯曲的边缘，用户应该使用 NURBS 建模。

**3。细分曲面:**这些对象同时使用 NURBS 和多边形建模图元。这些对象有边、顶点以及面，但是它们内部有曲线作为 NURBS。在 Maya 上创建这些对象要复杂和困难得多。

在这些原始工具的帮助下，用户能够根据需要创建基本和复杂的模型。考虑到这一点，Autodesk Maya 是一款昂贵而周到的软件，仅用于创建 3D 模型。因此，设计师有时会从在线网站下载 3D 模型，并将其导入软件，然后在他们的帮助下创建项目和动画。

### 结论

总之，在 Autodesk Maya 上创建 3D 模型需要大量的工作和软件知识。尽管用户可以使用上述步骤创建 3d 基本形状和几何图元。
