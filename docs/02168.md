## 什么是大数据技术？

一种分析、处理和解释大量结构化和非结构化数据的软件工具被称为大数据技术，这些数据无法通过人工或传统方式进行处理。这有助于形成结论和对未来的预测，从而避免许多风险。大数据技术的类型有运营型和分析型。操作技术处理日常活动，如在线交易、社交媒体互动等，而分析技术处理股票市场、天气预报、科学计算等。大数据技术存在于数据存储和挖掘、可视化和分析中。

### 大数据技术

在这里，我列出了一些大数据技术，并对其进行了清晰的解释，以使您了解即将到来的趋势和技术:

<small>Hadoop、数据科学、统计学&其他</small>

#### 阿帕奇火花

它是一个快速的大数据处理引擎。这是在牢记数据实时处理的基础上建立的。它丰富的机器学习库很适合在 AI 和 ML 领域工作。它在集群计算机上并行处理数据。Spark 使用的基本数据类型是 RDD(弹性分布式数据集)。

#### NoSQL 数据库

它是一个非关系数据库，提供数据的快速存储和检索。它处理结构化、半结构化、非结构化和多态数据的能力是独一无二的。

没有以下类型的 SQL 数据库:

1.  **文档数据库**:它以文档的形式存储数据，可以包含许多不同的键值对。
2.  **图形存储**:存储社交媒体数据等通常以网络形式存储的数据。
3.  键值存储:这些是最简单的 NoSQL 数据库。数据库中的每一项都存储为属性名(或“键”)及其值。
4.  **宽列存储**:该数据库以列格式存储数据，而不是基于行的格式。[卡珊德拉和 HBase](https://www.educba.com/hbase-vs-cassandra/) 就是很好的例子。

#### 阿帕奇卡夫卡

Kafka 是一个分布式事件流平台，每天处理大量事件。由于速度快且可扩展，这有助于构建实时流数据管道，从而在系统或应用程序之间可靠地获取数据。

#### 阿帕奇 Oozie

这是一个管理 Hadoop 作业的工作流调度系统。这些工作流作业以有向无环图(Dag)的形式被调度用于动作。

#### 阿帕奇气流

这是一个计划和监控工作流程的平台。智能调度有助于高效地组织和执行项目。Airflow 拥有在出现故障时重新运行 DAG 实例的能力。其丰富的用户界面使得在生产等不同阶段运行的管道可视化、监控进度以及在需要时解决问题变得非常容易。

#### 阿帕奇波束

这是一个统一的模型，用来定义和执行包括 ETL 和连续流的数据处理管道。Apache Beam 框架提供了应用程序逻辑和大数据生态系统之间的抽象，因为不存在绑定所有框架(如 Hadoop、spark 等)的 API。

#### 每个人都离开了

ELK 因 Elasticsearch、Logstash 和 Kibana 而闻名。

Elasticsearch 是一个无模式的数据库(索引每一个字段),具有强大的搜索能力和易伸缩性。

Logstash 是一个 ETL 工具，它允许我们获取、转换和存储事件到 Elasticsearch 中。

Kibana 是一个用于 Elasticsearch 的仪表板工具，在这里你可以分析所有存储的数据。从 Kibana 中提取的可操作的见解有助于为组织制定战略。从捕捉变化到预测，Kibana 一直被证明非常有用。

#### 停靠站(k)

这些是帮助应用程序在 Linux 容器中运行的新兴技术。Docker 是一个开源工具集，可以帮助你“在任何地方构建、发布和运行任何应用”。

Kubernetes 也是一个开源的容器/编排平台，允许大量的容器和谐地一起工作。这最终减轻了操作负担。

#### TensorFlow

这是一个开源的机器学习库，用于设计、构建和训练深度学习模型。所有计算都在 TensorFlow 中完成，并带有数据流图。图包括节点和边。节点代表数学运算，而边代表数据。

TensorFlow 有助于研究和生产。它是在牢记这一点的基础上构建的，它可以在多个 CPU 或 GPU 上运行，甚至可以在移动操作系统上运行。这可以用 Python、C++、R 和 Java 来实现。

#### 很快

Presto 是由脸书开发的开源 SQL 引擎，能够处理数 Pb 的数据。与 Hive 不同，Presto 不依赖于 MapReduce 技术，因此检索数据更快。它的架构和接口很容易与其他文件系统交互。

由于低延迟和简单的交互式查询，它现在在处理大数据方面越来越受欢迎。

#### 聚合碱

Polybase 工作在 SQL Server 之上，访问存储在 PDW(并行数据仓库)中的数据。PDW 专为处理任意数量的关系数据而构建，并提供与 Hadoop 的集成。

#### 储备

Hive 是一个用于大型数据集的数据查询和数据分析的平台。它提供了一种类似 SQL 的查询语言[，称为 HiveQL](https://www.educba.com/hiveql/) ，它在内部被转换成 MapReduce，然后被处理。

随着数据的快速增长和组织对分析大数据技术的巨大努力，市场上出现了如此多的成熟技术，了解这些技术将带来巨大的好处。如今，通过提高运营效率和预测相关行为，大数据技术正在解决许多业务需求和问题。大数据及其相关技术的职业生涯可以为个人和企业打开许多机会之门。

从今以后，是时候采用大数据技术了。
