# 如何在 Illustrator 中添加纹理？

> 原文：<https://www.educba.com/how-to-add-texture-in-illustrator/>

![How to Add Texture in Illustrator](img/e930254d23f2c99c26e7141e5964ded3.png)



## Illustrator 中什么是纹理？

纹理意味着在任何物体上添加额外的亮点，这增强了该区域的细节，并有助于理解艺术品中该特定区域的用途。您也可以在 Illustrator 中使用纹理，让您的想象力在您的作品中实现，并带有自己的想法。您可以将 illustrator 预先设计的纹理用于您的作品，或者您可以从互联网上的许多网站获得它，或者您可以按照一些简单的步骤制作自己的文本。

### 如何在 Illustrator 中添加纹理？

您可以在 Illustrator 中以非常简单的方式添加纹理。这里我们将通过一个在艺术品中添加纹理的例子来理解它。所以让我们以循序渐进的方式开始我们的学习。在我们开始学习纹理之前，让我们先制作一个简单的艺术品来学习。你可以带自己的作品去完成这个任务。

<small>3D 动画、建模、仿真、游戏开发&其他</small>

**步骤 1:** 从工作屏幕顶部的该软件菜单栏的文件菜单中选择一个新文档。

![New document](img/be682d9b80954c94728fb8b2845e7ba1.png)



**第二步:**根据您的要求设置文档的参数，点击该对话框的创建选项卡。

![Setting the parameter](img/ee8c720c4e01659706f7a73280c6a52b.png)



**第三步:**现在，我将使用钢笔工具、形状工具和颜色面板制作一个像这样的简单艺术品。使每一个形状在不同的层，这样你就可以给每一层不同的纹理和管理纹理参数。

![How to Add Texture in Illustrator - 3](img/3fdc59949bfeab26a5eb2ee29c50bf71.png)



第四步:现在给每一层命名，这样我就不会混淆层了。命名层，只需双击该层，并根据你的名字命名。

![naming each layer](img/f5d8dd731f8565853ea52a433cff9415.png)



第五步:现在，为这幅作品添加纹理，新建一层。要创建一个新的层，请转到该软件的层部分，然后单击该部分底端的“创建一个新层”图标。

![new layer](img/431520b5f7fbfec23fc9954412f5a60c.png)



**第六步:**如果这一层不在最上面，就把它放在所有层的最上面。

![layer at the top](img/daae2c54b5feb5a6806dab4ae444eb64.png)



**第七步:**现在，在这一层做一个长方形。到这个软件的工具面板，选择一个矩形工具来制作一个矩形。从颜色面板中选择此矩形的颜色。矩形的颜色将成为你的作品纹理的颜色。你也可以使用这个软件的眼药水工具从你的作品中选择任何颜色。

![How to Add Texture in Illustrator - 7](img/237a116abbbc1714d4378268331af8ee.png)



第八步:当你制作矩形时，确保矩形能恰当地覆盖你的作品。

![How to Add Texture in Illustrator - 8](img/8979e081625faedb7e0484e574ff7ab7.png)



**第九步:**现在进入该软件菜单栏的效果菜单。将会打开一个下拉列表。转到该层的纹理选项，点击新建下拉列表的纹理选项。

![Effect Menu](img/d49246fe4cd5add986265b4a55469e8a.png)



**第十步:**一个纹理对话框会打开。在这个盒子里，我们有不同类型的纹理。你可以逐个看到每种纹理的效果。

![type of textures](img/dfe66bad91099778f0d3e07b91cb97cf.png)



**步骤 11:** 比如马赛克瓷砖给出这样的效果

![Mosaic Tiles](img/cf537f9fe135b7e54b6421ea40598c38.png)



现在，在这个下拉列表中，我们有不同类型的纹理图案。你可以选择其中任何一个适合你的作品。

![How to Add Texture in Illustrator - 12](img/0354b196969dbc424de7aa0a9786d6b9.png)



第十三步:我将从这里选择纹理作为我们作品的纹理。

![How to Add Texture in Illustrator - 13](img/4609fd01843a6bf4706c09e5a152215b.png)



**第十四步:**如果你用图形笔去，那么效果会是这样的。

![How to Add Texture in Illustrator - 14](img/5d9d8d869a401057fac1684b9d363255.png)



**第十五步:**一旦选择了纹理的类型和图案，进入所选类型的参数区。在这里，您可以通过增加和减少该选项的值来增加和减少纹理的强度(纹理密度)。

![How to Add Texture in Illustrator - 15](img/292dbbb667b5d71663c3a0ac8ab3ae9c.png)



**第十六步:**你还可以改变对比度的数值。对比度增加和减少纹理颜色的亮度。

![How to Add Texture in Illustrator - 16](img/85b12a317fafa1aac2cf70552b81f8f1.png)



**第十七步:**下一个选项是颗粒类型。在固有类型中，我们基本上设置所选纹理的方向和图案，以增强纹理的细节。

![How to Add Texture in Illustrator - 17](img/c043951ea1c639d6b6e9964508b04648.png)



**第十八步:**设置好纹理的参数后，我会点击这个对话框的确定按钮。

![How to Add Texture in Illustrator - 18](img/259ca9037fa3909cb7d9e9d7fe621685.png)



第 19 步:现在来看看你的作品的纹理效果。转到位于工作区顶部的该层的属性栏。点击不透明度选项，并点击混合模式的向下箭头按钮。

![How to Add Texture in Illustrator - 19](img/a9a5811a48aff94c7ee5ce8a545b2aae.png)



**第 20 步:**一个下拉列表将被打开。从这里选择覆盖选项。

![How to Add Texture in Illustrator -20](img/0730457b72e0c847e63a08a0ceceb4b0.png)



第 21 步:纹理将被应用到你的作品上，就像这样。

![How to Add Texture in Illustrator -21](img/81a357f19339bdece3a4ab1bcef50666.png)



第 22 步:你可以给不同的艺术品添加不同的纹理。例如，我想在太阳层添加不同的文字。然后选择太阳层，用这个软件的 circle 命令给太阳盖上一个圆。

![How to Add Texture in Illustrator -22](img/5b8645023e9773fb1e0d4b7fe35c9200.png)



步骤 23: 现在进入效果菜单，进入纹理，然后像之前一样选择纹理。

![How to Add Texture in Illustrator -23](img/d968961c432c9735fc421778ffc21307.png)



步骤 24: 现在，为这个图层选择你想要的纹理，并设置它的参数。

![How to Add Texture in Illustrator -24](img/effb82cc8aa91a8e885d0ac44040b5e4.png)



**步骤 25:** 点击该对话框的确定按钮，应用设置。

![How to Add Texture in Illustrator -25](img/b0b4bc58bb8865ba9816cd80f95a1f75.png)



第 26 步:新的纹理会像这样添加到这个太阳层中。

![How to Add Texture in Illustrator -26](img/5865c425683d78e68d68b21f3278b981.png)



这样，您可以为每个图稿的对象添加不同的纹理，并赋予该图稿逼真的外观。在获得纹理方面的专业知识后，你可以轻松地制作出专业的艺术品。
