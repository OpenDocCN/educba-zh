## GIMP 文本阴影介绍

文字阴影是一个非常有效的功能，可以在你的文字上为你的浏览者创造出 3d 的错觉，同时也给你的文字增加了很好的重量；在 GIMP 中，我们使用绘图工具、滤镜选项和 GIMP 软件的许多其他重要功能为文本添加阴影，这种阴影通常出现在文本后面，但是您可以根据自己的需要设置阴影角度。我们将为我们的文本手动创建一个阴影，并使用过滤器作为阴影的预设。所以让我们找出两种方法，通过这两种方法我们可以为我们的文本添加阴影。

### 如何在 GIMP 中创建文本阴影？

首先，让我们对我们的文本有一个非常漂亮的背景，因此，采取一个新的文件。进入文件菜单，点击“新建”选项。

<small>3D 动画、建模、仿真、游戏开发&其他</small>

![GIMP text shadow output 1](img/49d9f3bde5f634ccf5cf1548518cd441.png)



在此设定您想要的文件尺寸。我要 1920 x 1080 的。

![GIMP text shadow output 2](img/d4809708f80fa75c8f5fecd3e33dab84.png)



我们将对背景应用渐变，因此在工具面板的前景和背景色框中设置两种渐变颜色。我要这种深浅不一的蓝色。

![GIMP text shadow output 3](img/a79e1e26eb2186b0c2b99782d0962367.png)



现在从 GIMP 的工具面板中选择渐变或混合工具。

![GIMP text shadow output 4](img/e8258beae90d491ba85cd79d887364a6.png)



在渐变参数中将 FG 设置为 BG 颜色模式，也采用径向渐变，然后将渐变光标从文档的中心向外拖动。

![GIMP text shadow output 5](img/68f76eda1705b947fd968df0cb2e19b8.png)



我将从渐变参数中反转我的渐变颜色，并在我的背景上获得这种渐变的平滑混合。

![GIMP text shadow output 6](img/5e4e2cd216e0ab73baefd8506c04e9cc.png)



现在从工具面板中选择文字工具。

![GIMP text shadow output 7](img/b7894e923597d07d014ded7007a8170c.png)



将文本的颜色更改为您想要的颜色，并从它的参数中选择一个好的字体样式。

![GIMP text shadow output 8](img/9269d6c6947db09067dcd720b4d9fa1e.png)



现在在这里输入你想要的文本。我会键入 EDUCBA，这是我们网站的正式名称。

![GIMP text shadow output 9](img/00ce41f53126bdc82d4fcf1bb453dc0c.png)



现在根据你的需要增加文字的大小。

![GIMP text shadow output 10](img/842706e10ec7e98551f99f889f70f1ce.png)



转到菜单栏的图像菜单，然后转到向下滚动列表的参考线选项，并单击新的参考线(按百分比)选项，在文档的中心设置垂直和水平参考线。

![GIMP text shadow output 11](img/bf2ac69daad4cd1310c2b2249798073f.png)



在文档尺寸的 50 %处选择水平导轨，然后点击确定。

![GIMP text shadow output 12](img/0fd77ce20fbd4cc7bf03a997890b3315.png)



对垂直参考线进行同样的操作，您将在文档的中心获得参考线。现在，在移动工具的帮助下，将文本放置在参考线交叉处的文档中心。

![GIMP text shadow output 13](img/eca3c4f963e36d67f55ab996b565a957.png)



在工具面板的旋转工具的帮助下，旋转这个文本一点点，让你的文本有一个好的阴影效果。

![GIMP text shadow output 14](img/fce83b71d512452da3263b5aa7a7d9fd.png)



现在右击文本层，点击下拉列表中的 Alpha to 选择选项来选择文本。

![GIMP text shadow output 15](img/971c0001a73034c13197daf83fee6ba6.png)



通过单击菜单栏的“编辑”菜单中的“复制”选项或按键盘上的 Ctrl + C 按钮来复制此选择。

![GIMP text shadow output 16](img/656871b686556d2556b6925eaee70aad.png)



点击图层面板的新建按钮，在图层面板新建一个透明图层。

![GIMP text shadow output 17](img/2a49a16939ec189fd97803cfa84580db.png)



点击编辑菜单下拉列表中的粘贴选项，或者按键盘上的 Ctrl + V 键，将选区粘贴到新的透明图层上。

![GIMP text shadow output 18](img/4478dc36f02df5683bc58558c27a5f20.png)



现在点击图层面板底部的锚按钮，为下面的透明图层添加浮动图层。

![GIMP text shadow output 19](img/ab0b7514152bc36c140bd2bf0c9e5c33.png)



现在把下面这一层移到原来的层。

![GIMP text shadow output 20](img/9cc6f63dba2fdec91645ed4e8bd50c20.png)



再次选择并设置黑色为颜色框中的前景色。通过点击图层面板中的眼睛按钮隐藏顶部原始图层。

![GIMP text shadow output 21](img/c2fa63a106f4d052f24a36ed9edad857.png)



现在用桶工具点击选区填充黑色。

![GIMP text shadow output 22](img/9edb5bfde33bc29e86eaf3cacf50fef1.png)



通过单击菜单栏选择菜单的“无”选项启用选择。

![GIMP text shadow output 23](img/5a9b4ba89b4385bc79ef7fc25792c091.png)



现在取消隐藏最上面的图层，将黑色文字图层向下移动一点，这样你的文字就会有一个很好的阴影效果。您可以降低黑色层的不透明度，以获得软阴影或硬阴影效果。

![GIMP text shadow output 24](img/9e0ecb793b78a88e3e5e9434b7100a76.png)



现在让我们使用 GIMP 的滤镜选项来制作文字阴影。为此，通过点击图层面板的创建副本按钮创建一个副本图层。

![GIMP text shadow output 25](img/bc21036080dae937ed2d098eed8e2955.png)



并将其移动到原始层的下方。

![GIMP text shadow output 26](img/9ebea65c9b46ca50121835b16c8016bb.png)



现在选择一个复制的图层，点击菜单栏的滤镜菜单。转到下拉列表的光影选项，然后点击新的下拉列表的投影选项。

![GIMP text shadow output 27](img/d29080da1356d02a5c91b710ed34e0fd.png)



你会得到这个漂亮的阴影。我们有一个带有这个滤镜的投影对话框。您可以增加或减少模糊半径，以增加或减少阴影的暗度。

![output 28](img/e56f1d1366056e68b10a53c26c444ec7.png)



您也可以增加或减少发光半径来改变阴影的效果。

![output 29](img/c606842b8a9a394b0cdf21ac00212afc.png)



现在，让我们尝试另一个过滤器的阴影，使一个以上的原始层复制层，并将其移动到下面的原始层。

![output 30](img/81abe34cd3753bc6c41681846d404d03.png)



再次进入滤镜菜单，然后选择光影选项，并从新的下拉列表中选择长影选项。

![output 31](img/5f4ee21338e300dc737e94947d16c599.png)



你会得到这种长长的阴影效果。

![output 32](img/161d25ec02927eb15606b9ce36ce9d70.png)



您可以从该对话框的“样式”选项中为长投影效果选择不同的预设选项。

![output 33.1](img/771da6df6e67ec6f5f9039de12efc3e8.png)



![output 33.2](img/169352db46b191832b25f749994cb8bb.png)



![output 33.3](img/7ab7e08a0dddc0e4f0c7aae5c652531a.png)



![output 33.4](img/dc00230ea3f3a48fea905021ef8a5757.png)



以上效果谁都可以去。在这里，我们也可以通过改变这个对话框的角度选项的值来设置阴影的角度。

![output 34](img/23ec3bb35afb62eceefaafa2dbfb544c.png)



此对话框还允许您更改投影的颜色；只需点击色标并从颜色框中改变颜色。

![output 35](img/e8a84b207a705eadec7418883ee6a078.png)



从该对话框的“合成”选项中选择“仅阴影”选项，以便在应用的文本图层上有唯一的阴影。

![output 36](img/c2e85d42d82a1bb2506fed06656afd1b.png)



你可以通过点击顶部文字层的眼睛按钮来显示它。

![output 37](img/99192dedc5306f92469560c6793542c3.png)



现在合并上面的图层和下面的阴影图层，使它们在一个图层中。为此，点击 GIMP 图层面板的合并按钮。

![output 38](img/78bda7c1c182e5ad8ed05a95c10311e2.png)



这样，你可以为你的文本创建一个漂亮的阴影。
