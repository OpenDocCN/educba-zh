## Godot 发动机简介

Godot Engine 是一款免费、开源、跨平台的游戏引擎，适用于 2D 和 3D 游戏设计，在 MIT(麻省理工学院，自 20 世纪 80 年代末以来一直提供免费软件许可)许可下发布。Juan Linietsky 和 Ariel Manzur 为多家公司开发了该软件，直到它发布供公众使用，公开发布日期为 2014 年 1 月 14 日。它兼容微软 Windows，Mac OS，FreeBSD，NetBSD，OpenBSD，你可以在很多不同的语言中找到它。如果你想要，你可以去 www.godotengine.org。

### 我们为什么需要戈多发动机？

*   如上所述，它是 2D 和 3D 游戏设计软件，所以我们需要它来设计游戏，它提供了许多内置的设计节点，使设计工作变得容易。
*   这款游戏的建筑部分的创新结构也可以由它来设计，同时渲染也可以以最高的效率来完成。
*   所以这是一个合适的免费游戏包，可以用来设计游戏资产和它们的环境。

### 戈多发动机是如何工作的？

*   如果我们谈论 Godot 引擎的工作，那么我们可以说它是一个跨平台的游戏引擎，这意味着它既可以在手机上工作，也可以在 PC 上工作。它涉及签名编码语言，GDScript，这有助于在这个引擎的游戏开发，用户谁没有适当的 C++语言知识。C++是它的基础语言。
*   它使用后处理效果使其游戏环境更加有效。除了后期处理效果，您还可以在处理过程中添加效果。
*   在这些效果的形式中，它为你提供了一个用于管理色调的色调映射器，不同类型的曲线支持，雾效果，等等。它还与 2D 灯光和地图效果一起工作，使设计的游戏更加真实。

### 戈多发动机的优缺点

下面给出了提到的优点和缺点:

#### 优势:

*   它的第一个优点是它提供了一个灵活的屏幕系统，这有助于为合成创建节点。
*   在 Godot 引擎的下一个优势中，你可以有许多内置的节点，通过这些节点，游戏设计变得非常方便，它还通过其令人难以置信的功能使我们用于游戏设计的工具的处理变得非常容易。
*   如果你在任何一个设计好的游戏中做了直播剪辑，那么在停止游戏后，改动会保留，也就是说不会被打扰。
*   你可以在一个令人兴奋的工具系统的帮助下创建你自己的定制工具。
*   渲染可以通过完全的 MSAA 和 FXAA 支持来完成，这是一种非常好的渲染方式。
*   它有一个全局照明系统，通过它你可以为游戏目的进行实时图形设计，并获得一个漂亮的结果。

#### 缺点:

*   它在 2D 物理学上有一个弱点，这意味着它在 2D 游戏物理学上的能力比其他免费游戏引擎要弱。
*   你可能会发现一些场景和资产的标签有时会与戈多混淆，在作品的上下文中没有合适的押韵。
*   Godot 提供的大片游戏较少，这分散了一些人对它的选择。

### Godot Engine 的职业道路

*   一个游戏开发者可以在其中创造一个好的职业生涯，因为它是免费软件，一个学习者可以开始学习 2D 和 3D 游戏。
*   对于游戏设计者来说，它确实是一个很好的基础开发者，因为它在 PC 和移动上都可用，这使得它可以跨越游戏引擎。
*   掌握了这个引擎后，你可以开始用它来创建 2D 和 3D 游戏，从这里你可以开始赚钱和探索你的兴趣。

### Godot 发动机的使用

*   它用于开发 2D 和 3D 游戏设计，您可以在其中制作角色骨骼和所设计游戏对象的动画。
*   它也有助于创建带有曲线和补间动画以及更多你可以拥有的特性的极端水平的动画。
*   因此，作为免费软件，它是许多开发者设计和使用 2D 和 3D 游戏的一个很好的选择，因为它的执行只需 60 MB，不需要安装。
