### 第 1 部分——科特林面试问题(基础)

第一部分包括基本的面试问题和答案

#### Q1。科特林是什么？

**答:**
Kotin 是一种开源编程语言，以 Java 虚拟机为平台执行，由 JetBrains 开发，相对来说比 Java 简单，提供了几个 Java 没有的特性。它是过程化和面向对象编程语言的结合。

#### Q2。科特林是如何工作的？

**答案:**
Kotlin 运行在 Java 虚拟机上执行。所有的 kotlin 代码都被转换成本地代码，这个字节码转换的过程也可以通过使用其他相关的虚拟机来实现。

让我们转到下一个科特林面试问题。

#### Q3。java 中没有的 kotlin 的关键特性？

**答案:**
有一些关键特性是 java 所没有的。

*   扩展功能
*   运算符重载
*   智能转换
*   数据类别
*   范围表达式
*   零安全
*   协同程序
*   伴随物体

#### Q4。科特林的重要好处？

**答案:**
这是科特林面试时问的基本问题。Kotlin 优点如下

*   语法上类似于 java，学习语言也很容易
*   一种函数式语言，内置对 JVM 的引用
*   代码可读性和可理解性非常高

#### Q5。在 java.io.file 列表中，kotlin 提供的扩展方法？

**答案:**
一些扩展方法提供了:

*   **bufferedReader():** 读取文件到 bufferedReader
*   **readBytes():** 将文件读取到 ByteArray
*   **readText():** 将文件读入单个字符串
*   **forEachLine():** 用于在 Kotlin 中逐行读取文件
*   **readLines():** 将文件读入列表
