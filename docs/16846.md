## Inkscape PNG 到 SVG 简介

Inkscape PNG to svg 可以理解为将任何图像的 PNG 文件转换为 SVG 文件格式，这里 SVG 代表可缩放矢量图形，这意味着您可以缩放 SVG 图像而不会损失其质量，并且它不会像素化。在 Inkscape 中，我们可以将我们的作品保存为 svg 文件格式，不仅是我们的作品，我们还可以将任何 png/jpg 图像文件保存为 SVG 文件格式。但是为了保存 png，我们必须将图像转换成矢量格式，在 Inkscape 中有一个非常快速的方法可以将 png 转换成矢量格式。因此，让我们找到我们将如何做到这一点。

### 如何在 Inkscape 中将 PNG 转换成 SVG？
