## Inkscape PNG 到 SVG 简介

Inkscape PNG to svg 可以理解为将任何图像的 PNG 文件转换为 SVG 文件格式，这里 SVG 代表可缩放矢量图形，这意味着您可以缩放 SVG 图像而不会损失其质量，并且它不会像素化。在 Inkscape 中，我们可以将我们的作品保存为 svg 文件格式，不仅是我们的作品，我们还可以将任何 png/jpg 图像文件保存为 SVG 文件格式。但是为了保存 png，我们必须将图像转换成矢量格式，在 Inkscape 中有一个非常快速的方法可以将 png 转换成矢量格式。因此，让我们找到我们将如何做到这一点。

### 如何在 Inkscape 中将 PNG 转换成 SVG？

在将 png 图像转换为 svg 之前，我们必须将 png 转换为矢量文件。让我给你解释一下你是如何做到这一点的。在 Inkscape 中获取您的 png 图像，为此，我将转到文件菜单，并单击其下拉列表中的导入选项。

![Inkscape PNG to SVG 1](img/4f1a37cb26dbf94d25d021b86df0629d.png)



现在，我将选择 EDUCBA 徽标的图像。

![Inkscape PNG to SVG 2](img/cb49c919fc5af144c8d8a63c27e48c82.png)



有了它之后，我将根据徽标图像的大小来调整文档页面的大小，为此，请再次进入文件菜单，并单击向下滚动列表的文档属性选项。

![Inkscape PNG to SVG 3](img/3065aab2a7ff1f3d22893e74c8b7b733.png)



现在，在文档属性对话框中，单击“根据绘图或选择调整页面大小”选项卡。

![Inkscape PNG to SVG 4](img/7fbbe78478ddbf58636ce2f280ab4eeb.png)



它会根据这张图片调整页面大小。

![Inkscape PNG to SVG 5](img/68be92b511d35b20472f6e08193e4050.png)



现在，如果我缩放它，你可以看到它是一个光栅图像，如果我们保存为 SVG，然后缩放它，它会像素化。首先，我们将把这张图片转换成矢量格式。

![Inkscape PNG to SVG 6](img/1aff9b930381b6488a29431185c6f5a5.png)



要将此图像转换为矢量格式，请转到路径菜单并单击描摹位图选项。

![Inkscape PNG to SVG 7](img/aa688b29276b94fea1dd9b95a560f0a1.png)



工作屏幕右侧将打开一个跟踪位图选项卡。这里我们有两个按钮用于扫描图像，即单次扫描和多次扫描。您可以对黑白图像进行单次扫描，或者如果图像是多色的，则可以进行多次扫描。为了深入理解跟踪位图面板，你也可以阅读我的“图像转矢量”文章。

![Inkscape PNG to SVG 8](img/8b8f9da24b99328ee2e7c6b51acd6dc4.png)



还有一个自动跟踪选项，所以我会点击它。

![Inkscape PNG to SVG 9](img/793d63a346c74a97e8714acdc92ab6c7.png)



并单击该对话框的更新按钮。你可以在这里看到描摹图像的预览。

![Inkscape PNG to SVG 10](img/75b33418e84bbee0f87454a25f559b03.png)



现在点击“应用”按钮，你将在源图像的顶层描摹出图像。我移动它，你可以看到这是一个很好的矢量轨迹。最好的部分是它有一个透明的背景。

![Inkscape PNG to SVG 11](img/6db4a6381d4c0921f248c25c1d23c09a.png)



如果您对这种跟踪不满意，您可以选择多次扫描并设置扫描次数。

![Inkscape PNG to SVG 12](img/fec3a1c28295cfdb4f2619e2efcb1ab4.png)



现在选择列表中的颜色选项。

![Inkscape PNG to SVG 13](img/95c38a6b101a4f3b24f783f61bb6621c.png)



点击多次扫描按钮后，点击更新按钮，如果你对结果满意，然后点击应用按钮。我将移动跟踪的图像，你可以注意到它的颜色几乎与我们的图像标志相同。

![Inkscape PNG to SVG 14](img/2cf72be631d06ae31026d4f1cbb5a9b4.png)



现在你可以通过放大两张图片来看出它们之间的区别。我们追踪的图像是矢量格式的。

![vector format](img/4f85de014a2f027c6a622c04baaf7a6e.png)



这个跟踪的图像有一组 7 层，所以你可以根据你的要求取消分组。

![requirement](img/1ed4d693115868a9b179c9f72a7b6f10.png)



把他们分开。

![Inkscape PNG to SVG 15](img/94e4607c9250bc66eefee2dd2b1078eb.png)



如果您的图像有一个复杂的设计，那么不能通过跟踪位图选项跟踪，但您可以使用贝塞尔工具手动跟踪您的图像。所以首先去填充和描边面板。您将在对象菜单的下拉列表中找到此面板。

![image](img/0a75bdb655800a9267f7f1f977d19d63.png)



在这个面板中，我将降低这个图像层的不透明度，这样我们就可以很容易地用贝塞尔工具描绘出它周围的路径。

![image 1](img/02bc6468f132d7ef71272850f4a07933.png)



现在我将使用贝塞尔工具，你可以按 B 作为快捷键。

![tool](img/92ed65abe26246d2fe18815a47550ff0.png)



现在我将选择“创建 b 样条线路径”选项，这样当我们使用贝塞尔工具时，它将在跟踪过程中自动绘制路径曲线。

![tracking](img/43e888927d038b1aa1f518ad99558121.png)



我会这样追踪它。

![image 2](img/c50496fcb0ba27273f5aee565363a8e3.png)



如果有任何尖角，按住键盘的 Shift 键，你可以在这里像这样转动曲线。

![image 3](img/bd2b8f5fbb302abca5258dcf0c2fb093.png)



当你到达路径的终点时，点击它的起点来关闭路径。

![image 4](img/bc605f2a5489e8c61b4e72becb34b357.png)



我们将根据我们的标志得到这种类型的形状。

![logo](img/be523ac26dbe6e4e7a6668441b87dd82.png)



现在你可以为这个创建的形状选择相同的颜色，并使用“从图像中选择颜色”工具。确保您已经选择了绘制的徽标形状。

![image 5](img/2a770d2f4d1df3bd6c4d03671f07b595.png)



然后点击图标的图像，它会从你点击的图像区域选择一种颜色。这是平面颜色，你可以根据自己的选择添加渐变。

![image 6](img/7bbd6999b59406a45888bb745783e129.png)



现在我将关闭填充和描边面板中的描边。

![stroke panel](img/10f2e73744ce5259173a356d23848345.png)



如果我画一个图形，并把它放在这个矢量图形标志的下面，你可以看到两个图形都有一个透明的背景。

![image 7](img/2d8ec0ad0384a3505765bec42b729393.png)



现在，我将删除其中一个矢量标志，这一个是准备保存为 svg。为此，我将再次转到“文件”菜单，单击它，然后单击“另存为”选项。可以按 Ctrl + Shift + C 作为其快捷键。

![image 8](img/b372dfca442194d426c6d9fd29e8ab93.png)



将会打开一个保存对话框。在您的计算机上选择一个位置来保存此徽标文件，并根据您的选择为其命名。我将把它命名为 EDU 标志。在保存之前你必须做的一件事是从保存类型列表中选择一个 SVG 文件格式，因为我们想将这个文件保存为 SVG，然后点击这个对话框的保存按钮。

![image 9](img/716303dbe8b1af2c571af486e0f9e71a.png)



您可以在保存文件的位置找到 SVG 格式的文件。

![image 10](img/7cd022acefa1fe16720fc0d35d10206c.png)



### 结论

现在，您可以在 Inkscape 中将任何 PNG 文件转换为 SVG 文件格式。不仅仅是 PNG，你还可以用本文中讨论过的方法将任何 jpg 文件格式转换成 SVG。我建议你用不同类型的 PNG 图像来尝试这种方法，这样你就可以彻底地分析这种方法。
