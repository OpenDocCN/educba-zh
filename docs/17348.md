# 危险保险

> 原文:[https://www.educba.com/hazard-insurance/](https://www.educba.com/hazard-insurance/)

## 定义

风险保险保护房屋或其他财产免受自然和人为灾害的损害，如火灾、风暴、故意破坏和盗窃。业主根据[保单](https://www.educba.com/fiscal-policy/)的条款和条件支付所需的年度保费。如果保险单承保损失，车主将得到赔偿。例如，2021 年飓风艾达造成的保险损失估计约为[36000](https://www.iii.org/publications/a-firm-foundation-how-insurance-supports-the-economy/supporting-resiliency-and-disaster-recovery/hurricanes)美元。拥有危险保险的业主可以根据保单条款索赔。

### 主要亮点

*   它为房屋结构提供保护，抵御从地震和洪水到盗窃和故意破坏等自然和人为灾害。
*   抵押贷款贷方有时需要业主购买风险保险。保险范围必须取决于影响业主区域的灾害类型。
*   人们通常在容易发生洪水和地震的地区购买它来保护他们的家园。然而，它只涵盖某些种类的灾难。
*   成本取决于各种因素，如业主的信用评分和房产的位置。
*   所有者不支付单独的保险费。家庭保险政策包括它。

### 危险保险是如何运作的？

*   由于灾害保险涵盖范围广泛的灾害，选择正确的政策是至关重要的。例如，如果业主住在山里，保单必须包括山体滑坡险。
*   一旦业主决定了最好的风险保险，他们应该联系特定的公司。
*   然后，保险公司会派一名检查员检查财产，并确定其市场价值。车主必须缴纳的保费就是以此为基础计算的。
*   当这样的不幸事件发生时，车主必须提出保险索赔，并向保险公司提供有关事件的详细信息，包括照片和视频(如果有的话)。
*   在检查损失后，保险公司向车主支付修理费。
*   然而，车主必须自掏腰包支付免赔额。免赔额是您同意支付的索赔部分。这通常是索赔总额的 5%。

假设你的危险保险单有 500 美元的免赔额，一场飓风造成了 3000 美元的损失。在这种情况下，你必须支付 500 美元，而你的保险公司支付 2500 美元。

### 危险保险包括什么？

危险保险为几种不同的灾害提供保险。这包括自然灾害、人为灾害和家用设备故障。其中包括:

*   洪水和滑坡损害
*   火灾和烟雾损害
*   风、冰雹和龙卷风的损害
*   盗窃和故意破坏
*   短路、过热设备和管道爆裂

### 危险保险的类型

有两大类:

1.  指定风险保险-如果保单中包含损坏原因，则提供保险。
2.  开放式风险保险——针对保单中未列明的风险提供保险。

这些事件必须意外发生且不合时宜，保单才有效。例如，如果由于电线故障超过一个月而发生短路，所有者将没有资格索赔。

同样重要的是，要了解政策不涵盖的内容。保险范围根据房产所有人的居住地而变化。任何个人或家庭物品的损坏或在灾难中遭受的伤害不在本保险范围内。它只保护主人的家，而不是里面的东西。

某些高风险领域有其特定的覆盖范围。例如，如果该地区容易发生洪水，业主将不得不购买单独的洪水保险。其他情况下，政策可能不适用的是虫害，霉菌和真菌攻击，或一般磨损和撕裂的房子。

### 危险保险与房屋保险

| **危险保险** | **家庭保险** |
| 它涵盖了灾害造成的结构性破坏。 | 它涵盖了家庭的各种责任。 |
| 它涵盖整个物业，而不是物业内的内容。 | 它包括个人物品、法律费用和其他结构，如围栏和车库。 |
| 这是家庭保险政策的一个子部分。 | 这是一个完整的政策，保护家庭免受各种责任。 |
| 抵押贷款人要求在[放贷](https://www.educba.com/lending-vs-borrowing/)之前，风险保险到位。 | 抵押贷款人不需要其他种类的保险。 |
| 条款和条件因地区而异。 | 条款和条件在不同的地方保持相似。 |
| 它不包括灾难或财产损失情况下的日常生活费用。 | 如果房主因灾难或其他不太可能发生的事件而无法住在自己家里，它会提供日常开支(酒店住宿和餐饮)。 |

需要注意的一个重要方面是，在某些地区，危险保险被排除在家庭保险之外。这通常发生在物业区域特别容易受到某些天气事件影响的时候。在这种情况下，对于业主来说，在房屋保险中包含危险保险就太贵了。

### 什么是风险保险抵押？

如果房主想把他们的房子抵押，在他们的家庭保险政策中有风险保险是强制性的。这是因为它涵盖了贷款人关心的整个财产结构。

一般来说，家庭保险政策满足抵押贷款人的需求。然而，根据地理区域的不同，贷方可能会要求额外的灾害保险，如飓风。此外，贷方需要它，因为损坏会降低财产的价值。

### 结论

底线是，对任何有房子或财产的人来说，购买风险保险是明智的。业主必须与保险代理人交谈，为他们的房屋获得正确的保护。

### 常见问题解答

#### Q1。危险保险是强制性的吗？

**回答:**拥有危害保险并不是强制性的。只有当业主决定抵押财产时，它才变得必要。在易受灾地区有房子的人应该准备好保险以防灾难。

#### Q2。危险保险*不*覆盖什么？

**答:**危险保险不承保任何个人物品或因电器故障造成的损坏。洪水和地震需要单独的保险，不属于危险保险。

#### Q3。保险的价格取决于什么？

**回答:**保单的价格很大程度上取决于房产的价格以及与该地区相关的[风险](https://www.educba.com/prepayment-risk/)水平。对于更昂贵和高风险的财产，利率往往更高。信用评分也可以是一个重要因素，而决定价格。

#### Q4。危险保险是否附有房主保险单？

**答案:**意外险是家财险的一个细分领域。它通常与家庭保险政策，虽然条款和细节的变化从政策到政策。一些政策可能不包括危险保险。它必须单独购买。所以政策一定要针对业主所在区域。
