## SharePoint vs OneDrive 简介

SharePoint 和 OneDrive 是微软的文件管理平台。这两个应用程序有很多共同点，用户可以使用任何工具共享和存储高达 15 GB 的文件，同时共同编辑文档，并访问文档的版本历史。SharePoint 和 OneDrive 都是移动设备，支持设备之间的本地文件共享。它们还与 Office 365 链接，使工作变得简单，这样当你进入云时，你就可以立即同步所有通知和离线更改。

### 什么是 OneDrive？

Microsoft OneDrive 是一个安装、存储和与用户共享信息的程序。它只是一个文件和纸张的在线存储文件夹结构。对人有欲望，对公司有驱动力。在本文中，我们将集中讨论市场推动。任何单个用户都可以在 OneDrive 中为公司保存文件，除非用户希望与整个团队共享文件。在与团队交换文件之前，任何团队成员或实体都可以访问该文档。每个用户都有一个万亿字节的磁盘，所以当设备出现故障或记录时，用户不会因为任何原因丢失任何细节。两个 OneDrive 记录都保存并备份在 Microsoft Cloud 中。这是因为 OneDrive 存储可以保持文件整洁，提高设备输出，并减少 IT 票证数量。

### OneDrive 和 SharePoint 的主要区别

您可以在企业中免费访问的 SharePoint 网站上构建您的内容。它是专为以最佳方式与公司团队合作和交流而构建的。但 OneDrive 是为一次性使用而设计的，这意味着用户可以与其他人交换他们的文档。大多数企业将 SharePoint 用于面向客户的企业网站，并直接在网站上发布员工或客户的信息。但是，内容不能写入 OneDrive 上的一个网页。
SharePoint 的功能远不止被企业用作备份库。您可以使用 SharePoint 查看日程安排、工作列表、更新和新闻，这使得与最终用户的合作和交流更加频繁。当您与朋友分享您的信息时，OneDrive 提供了大量的合作选项，但 SharePoint 中不包括某些额外的功能。
除了品牌、内部团队网站，一些企业现在使用 SharePoint 创建和管理他们的商业网页。SharePoint 的 CMS 框架将在线发布存储在网站上的任何文档。同样，整合的社交媒体促进了与公众交流亮点，并跟踪参与程度。企业还可以利用详细分析为员工或消费者构建个性化应用。

### SharePoint 与 OneDrive 对比表

|  | **SharePoint** | **OneDrive** |
|  | 虽然没有以市场为中心的版本，但可以在个人基础上访问。 | 作为独立服务和同名的消费者版本提供 |
|  | SharePoint 应用程序云版本可追溯到 Office XP | 以前基于 SharePoint Workspace 2010 和 Groove 2007 构建 |
|  | 从浏览器正常访问所有功能，但可以访问本地文件夹中的文件 | 根据用户选择，您可以访问标签或本地文件夹或应用程序 |
|  | 用户将使用 SharePoint 作为一个管理运行的品牌企业网页，作为新闻表，时间表等。 | 用户注册自己 OneDrive 的公司帐户，无需共享 GUI。 |
|  | 上传团队文件或资源以供共享和/或签入工作流的最佳场所 | 发布私人工作论文或受限范围或生命周期的文本的最佳位置 |
|  | 可以被视为替代的互联网站点和/或文件服务器 | 可以被认为是你的计算机的“我的文档”文件夹的云等价物 |

### 结论

在本文中，我们看到了 SharePoint 和 OneDrive 在文件管理方面的主要区别。您可以根据自己的需求选择任何一种。
