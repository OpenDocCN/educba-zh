## GIMP 文本效果介绍

在这篇文章中，我们将学习如何在 GIMP 中创建文本效果。网站中一般使用文字效果来创建横幅图像；它们也可以用于印刷媒体的小册子。基本上，文本效果用于使重要信息可见。除此之外，文字效果也可以用于纯粹的审美目的，并做到这一点；当我们有 GIMP 这样完全免费的程序时，我们就不需要昂贵的图形编辑程序了。我们可以结合股票网站上一些免费的资源和画笔，在 GIMP 中创建令人惊叹的文本效果。

### GIMP 文本效果的步骤

下面是下面提到的步骤

<small>3D 动画、建模、仿真、游戏开发&其他</small>

**步骤 1:** 首先，我们将使用图像和笔刷创建一个文本效果，我们将访问以下网址并从 Behance 下载所需的笔刷和图像，或者您可以随意使用自己的资源–https://www . Behance . net/gallery/31115821/Powder-Explosion-brush-set

**第二步:**让我们从进入文件菜单打开一张图片开始，然后打开或按 Ctrl + O。

![GIMP text effects output 1](img/40b54d744a4f31ac6da597610b2318b7.png)



**步骤 3:** 然后导航到下载的图像并在 GIMP 中打开它

![GIMP text effects output 2](img/b8b566422222a1bfaad979cc0206dfe8.png)



**步骤 4:** 然后选择裁剪工具或按 Shift + C，这将删除层或图像的边缘区域。

![GIMP text effects output 3](img/9e64cb1537d00cfb39135a224908ee1d.png)



**第五步:**拖动一个正方形或任何形状，将图像的爆炸聚焦。你可以在边缘调整图像，然后拖动。当您对裁剪选择满意后，双击中心创建一个裁剪。

![GIMP text effects output 4](img/0c0ea3d9857d3167948876658e150a12.png)



**第六步:**然后使用文字工具或按 T 键，用来创建和编辑文字层

![GIMP text effects output 5](img/4f441c2ce39825101e751d8b8a14cb39.png)



**第七步:**点击一次创建文字层，给它一些文字。我们在 Nexa 中使用了粗体字体，并给出了适当的像素大小。

![GIMP text effects output 6](img/1dc86d95949a12e8b46953a461920ec6.png)



第八步:接下来，我们可以改变颜色，使其成为白色

![GIMP text effects output 7](img/555fa7bb8e81685e1fb82f494f2664da.png)



**第九步:**选择所有文本，调整字体大小和颜色，改变字距，增加字符间距。

![GIMP text effects output 8](img/6b4f01f9aa032ce94a62edf4457f2687.png)



**第 10 步:**点击移动工具，这将带来对齐工具或按 Q，对齐或排列层和其他对象。然后在对齐面板中，选择居中对齐或目标对齐，并对齐目标的中间，这将水平和垂直对齐图像。

![output 9](img/20c3540d4ace310a3c994154b6bf8144.png)



**步骤 11:** 接下来，我们给文字添加图层蒙版；它将做的基本上是，我们将在图层蒙版上涂上黑色，这将显示文本层下面或爆炸图形的一些区域。这也被称为非破坏性编辑，它允许绘制文本，擦除它，如果你犯了一个错误，然后你可以重新绘制。要添加图层蒙版，右键单击并选择添加图层蒙版。

![output 10](img/4058d5c9b05d37dc3ee71cf23473adf6.png)



步骤 12: 保持初始化图层蒙版为白色，因为我们想看到所有的文字，除了我们将用黑色颜料蒙版的文字。

![output 11](img/205736eca786f7be541585bd00e29d71.png)



第 13 步:然后使用缩放工具调整缩放级别或按 Z 键，这样我们就可以更好地处理文本。

![output 12](img/189ba9f2e6210fff27c917db12847088.png)



**步骤 14:** 我们导入一些笔刷来工作，导入一个笔刷；从编辑转到首选项。

![output 13](img/de34388d5243ad7bb633a4b2b30454ab.png)



**第十五步:**在笔刷文件夹中，你会找到文件夹的位置。选择可写文件夹(由十字图标表示)，然后选择图标以显示文件管理器中的文件位置。

![output 14](img/7da5cad84035fef3818d8a950359b783.png)



**步骤 16:** 将下载好的笔刷拖动到这个文件夹中，这个文件夹就是 PowderExplosion.abr

![output 15](img/2f1727ef95b7a68b75b49a3ae4392382.png)



**第十七步:**检查刷面板；如果不在右边，那么进入窗口- >可停靠对话框- >画笔或者按 Shift + Ctrl + B

![output 16](img/201ac5bcb8dc49825b8bfaee9a122db2.png)



**步骤 18:** 点击右下角的刷新画笔图标，将加载新的画笔。如图所示，我们在粉爆包中有多个笔刷。这些是 photoshop 笔刷，通过扩展可以看到。abr，但是这些是兼容 GIMP 的。

![output 17](img/01e9a71ebaebc944def1dced3a25df29.png)



**步骤 19:** 然后选择画笔工具或者按 P 键，使用画笔创建平滑的笔画。从导入的笔刷包中选择任意一个笔刷，首先，它的尺寸会很大，可以通过尺寸选项来减小。您也可以通过选择画笔图标来更改画笔。

![output 18](img/073c345d5ab5ad5ac4e1383214b483ac.png)



**步骤 20:** 确定前景色是黑色，这是很重要的一步，后面会看到为什么。

![output 19](img/5966642c251d32fe321054ffe3cf2263.png)



**步骤 21:** 笔刷大小、长宽比、角度、间距、硬度、力度都可以改变，得到如图所示的想要的效果。您可以使用图形输入板来获得更微妙的效果。开始绘画时，请确保选择了蒙版，而不是文本。如果您想正确地观察颜色，可以调整蒙版层的不透明度。快速改变画笔大小的一个快捷方式是使用“[”来减少和“]”来增加，因为这是一个经常需要的功能。你还必须定期更换刷子，因为图像上的粉末是不一样的。您也可以进一步放大以获得更好的视图，并使用类似于您想要绘制的区域的画笔。

![output 20](img/dc5dec3bf576caf7dbf6a578375be9ea.png)



**步骤 22:** 假设你不小心过度使用了蒙版的某些部分，那么简单的方法就是撤销或者按 Ctrl + Z，但是如果你采取了太多的步骤并且不可能撤销，那么还有另一种方法。通过按双箭头图标将活动前景色更改为白色，这将交换前景色和背景色。然后把它涂在你想去掉的部分。

![output 21](img/41b640a18ff2e0495194413fb3c5f9c5.png)



**步骤 23:** 这是最后一件艺术品，GIMP 中的一个文字效果已经做好了。

![output 22](img/d90b5bb3e397b535158ecf3d7d0be82a.png)
