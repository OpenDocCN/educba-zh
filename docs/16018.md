## 3ds Max 软件简介

3ds Max 是一款专业的计算机设计软件，被视频游戏开发人员、电影和电视编辑用来为他们的模型提供三维可视化效果。3ds max 软件中有不同类型的命令和功能工具。样条就是其中之一；它是 3ds max 软件中制作 2D 图形的一个非常重要和基本的工具。在我们的 3D 建模中，我们需要一些基本的 2D 形状来根据我们的用途制作不同类型的对象，通过样条，我们可以很容易地制作 2D 形状。

*   我们在 3ds max 中有四个视口:顶视口、前视口、左视口和透视视口；您可以使用它们中的任何一个在 3ds max 中绘制任何对象。

![viewports (Spline in 3ds Max)](img/5dc42d377ce9b828079023d692f68b04.png)



*   在 3ds max 的公共面板中，单击“样条线”选项的“形状”选项卡。

![shape option (Spline in 3ds Max)](img/2e22651738c441827cae125180b991e8.png)



*   样条曲线中有不同类型的 2d 形状，如直线、矩形、圆形等等。我们将一个一个地看到它们。

![2d shape (Spline in 3ds Max)](img/79aa105b92e3a120b597e0a4cd38e0a6.png)



*   我将在顶视图中绘制这些形状——按下 **Alt+W** 进入全屏模式。

![top viewport (Spline in 3ds Max)](img/4dc0143c307f2410c281422568c2d839.png)



*   现在，点击公共面板中的线命令选项卡，激活线命令。

![line command2 (Spline in 3ds Max)](img/3e9e028bd66372b5812ebf8ea532c0d3.png)



*   我们在线命令的参数部分中有一个创建方法属性，它是拐角和平滑的。

![creation method (Spline in 3ds Max)](img/f01bb2fb73a972ee3baa2aac532a1351.png)



*   在拐角法中，我们会在线段中找到一个尖角来留下任何命令按下鼠标右键或按下键盘上的 **ESC** 键。

![corner method (Spline in 3ds Max)](img/2a4549e4100a9f92d903a5c7372eff61.png)



*   但是在平滑法中，我们在线段的拐角处找到平滑的曲线。

![smooth method (Spline in 3ds Max)](img/0547a45d891e4cab45025473c83ab622.png)



*   另一个属性是拖动类型，这意味着当你拖动鼠标指针时，它会画出什么？它可能是绘图角、平滑线段或贝塞尔曲线。

![data type (Spline in 3ds Max)](img/fd4f9f126ee8224f4b6df6dd651d81b1.png)



*   当您在“拖动类型”中选择“拐角”选项时，它将拖动一条线，如下所示。

![corner type (Spline in 3ds Max)](img/b599a2947ca4ebc31081f69c1d31befd.png)



*   当我们选择平滑选项时，它会像这样绘制。

![smoot type (Spline in 3ds Max)](img/297a7605456948b13226d40122c1bc1f.png)



*   从拖动类型中选择贝塞尔曲线。在贝塞尔曲线中，我们有一个曲线手柄，通过它我们可以随时改变线条的曲线形状。

![bezier type (Spline in 3ds Max)](img/e8998995aa9920fb87fd0a52c01cd383.png)



*   可以用键盘输入法拖动线条；只需在此填写所需数据，然后点击该选项的添加点选项卡。

![keyboard entry (Spline in 3ds Max)](img/673501103cedaf45098fa7395a4b2cdd.png)



*   另一个属性是插值属性。你可以看到平滑选项画出的线实际上并不平滑。

![Interpolation ](img/978940fdb6dc934bb84d14ba35c5f0a6.png)



*   为了使其平滑，增加插值选项中的步长。

![increase interpolation](img/598d860c5c58eaa7a5595d096ee0dfda.png)



*   您可以通过双击名称和颜色部分的颜色框来更改线条的颜色。

![color line](img/3cfc70df1d9e00ebbc00e63d63ec37a3.png)



*   将会打开一个对话框。从这里选择您想要的颜色，然后按 Ok。

![choose color line](img/0ed65828039172308e3701a511fc7b50.png)



*   你可以看到，当我们在这里画多条线时，所有线的颜色都是不同的。

![multiple line](img/12baf27fd0f1e141baca75352b6fbce4.png)



*   如果你想用同样的颜色画线，那么双击颜色框的名称和颜色部分一个对话框将会打开。在这里取消选择分配随机颜色选项，然后点击对话框中的确定选项卡。

![Color Box](img/8f5415bb6e7f75a075c35389849b9894.png)



*   现在我们将从公共面板部分获取矩形命令。单击矩形选项卡。

![rectangle tab](img/20d8a401a0b22b5f358b84148e02e947.png)



*   我们有两种创作方法。一个是 Edge 现在，在这里，我们将通过拖动边缘来绘制一个矩形。

![draw rectangle](img/2c07afa9868d5aef94be4184d090cb9b.png)



*   当您选择中心方法时，将从矩形的中心绘制一个矩形。

![center method](img/134de434cd13d9f015766edfd8b5eb43.png)



*   你可以从这里给你的矩形一个圆角半径。

![corner radius](img/1f3428a20030cdbe646b9bb390f99af9.png)



*   矩形的另一个参数设置与直线相同。
*   现在从公共面板中选择 circle 命令。我们有两种方法画圆，也和画矩形一样；这两种方法都可以逐一检查。这里你可以直接从半径参数中给出圆的半径。

![circle command](img/f30ad79ee9908db8cc41f05d3f248fed.png)



*   椭圆非常类似于圆；你可以用同样的方法画出来供你学习。

![ellipse method](img/4c24f1cd991433ab746b197f4d30c8d7.png)



*   接下来是 Arc。从公共面板中选择 Arc 命令。这里有两种弧-端-中方法；一种是你应该先取第一个端点，然后再取第二个端点，把它往你想画圆弧的方向拖。

![arc command](img/4687f940dca13e3688065ee6a4b72ada.png)



*   第二种方法是中心-端点-端点法，即先单击圆弧的中心点，然后给出第一个端点，并将鼠标指针拖动到下一个端点。

![center end method](img/2d371cc382fa42662c59687e817612fb.png)



*   下一个是甜甜圈。从公共面板中选择圆环命令。这里还有两种甜甜圈的制作方法。一个是边缘，第二个是类似矩形的中心。要绘制它，请单击网格中的任意位置，向外拖动鼠标指针以绘制圆环的外圆，再次向内拖动以绘制圆环的内圆，然后单击离开。

![donut method](img/3022150768cc3bdd4db214e9fa76ec06.png)



*   下一个是恩贡。从公共面板中选择 NGon 命令，这里我们也可以找到两种创建 NGon 的方法；一个是边缘，第二个是类似矩形的中心。它类似于多边形，但在 NGon 中，你可以根据你的需要给它任意数量的边。您可以从参数框中手动给出半径。

![Executing NGon command](img/b4020a0cd5b9bf523ec4c4857ca75918.png)
