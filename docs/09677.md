#### Q1。JS 是什么？

**答:**
[JavaScript 是](https://www.educba.com/what-is-javascript/)一种脚本语言，主要用于创建网页以及为 web 应用程序添加交互性。

#### Q2。JavaScript 是如何工作的？

**答案:**
这是面试中常见的 JS 面试问题。每个浏览器都有三个主要组成部分。第一个是 DOM(文档对象模型)解释器。这将获取您的 HTML 文档，并将其转换和显示在浏览器中。作为浏览器一部分的另一个小程序是 CSS 解释器，它将设计页面的样式并使其看起来更好。最后一个是浏览器中的迷你程序，称为 JS 引擎。

*   浏览器加载 HTML 文件/JS 文件
*   JavaScript 是一种解释型语言(意味着不需要编译)
*   浏览器(JavaScript 引擎)逐行执行并等待事件(如点击、鼠标悬停等。)发生

#### Q5。定义 JavaScript 中出现的常见错误？

**答案:**
一般来说，我们在 JS 中发现的错误有以下三种。

*   运行时错误:这是在 HTML 语言中误用命令的结果
*   **加载绑定错误:**这是一个语法错误，是动态生成的
*   **逻辑错误:**当功能的逻辑执行不当时，会出现此错误。

#### Q6。解释一下为什么 JS 是区分大小写的语言？

**答案:**
JS 是一种区分大小写的编程语言。在 JS 中，我们使用不同类型的变量、函数和其他各种应该始终一致的标识。

现在让我们来看看高级 JS 面试问题。

#### Q7。列出 JavaScript 的一些优点和缺点？

**答案:**
**优点:**

*   丰富的用户界面
*   增强的交互性(当鼠标悬停在按钮或键盘辅助功能等元素上时)

**缺点:**

*   缺少多线程活动
*   不适合网络应用
*   客户端 JavaScript 无法读取或写入

#### Q8。JS 中对象的类型并定义它们？

**答案:**
JS 中有 2 类对象:

1.  **日期对象:**这是在 JS 编程中构建的。这些都是使用新的日期创建的，一旦创建好，就可以在一系列可用方法的帮助下进行操作。这包括日期对象的年、月、日、小时、分钟、秒甚至毫秒。这些是在当地世界时标准的帮助下设定的。
2.  **数字对象:**这些包括日期，因为它只由整数和分数表示。数字的文字会自动转换成数字类别。

#### Q11。提到一些 JavaScript 数据类型？

**答案:**
这些数据类型一般持有值。在 JS 中有两种类型的数据类型。

*   原始数据类型
*   非原始数据类型

#### Q12。JavaScript 中的函数是什么意思？

**答案:**
函数是一段可重用的代码。这允许用户编写特定的代码，并通过调用函数根据需要多次使用它。JS 函数不一定要返回值。JS 支持两种类型的函数

1.  匿名函数
2.  命名函数

JS 函数的语法–

```
Function functionName (parameter1, parameter2, …..parameter n)
{//statement of the functions
}
```

要声明一个函数，我们必须使用函数，后跟函数名和括号。在括号内，我们必须指定函数参数(可以有多个参数)。

要调用这个函数，我们必须简单地指定函数的名称，并在括号内指定参数的值(传递值)。

addNumbers(x1，x2)——这里我们给出了值并调用了函数。

**Note:** If in the code there are 3(let’s say) parameter and we pass 3 or more parameter values. In this case, JS will simply ignore the additional parameter values.
