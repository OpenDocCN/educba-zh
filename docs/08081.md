## GitHub xmrig 简介

xmrig 是 GitHub 平台中托管的中央处理器(CPU)和图形处理器(GPU)挖掘器。xmrig 利用 CPU/GPU 的高性能计算能力来解决复杂的数学过程，同时在将区块链事务提交到区块链域之前对其进行验证。xmrig 是一个开源平台，它在其挖掘过程和随机 X 基准测试中支持多种算法，如 KawPow、AstroBWT、CryptoNight 和 RandomX。它已经发布了 Windows、Mac、FreeBSD 和 Linux 等主流操作系统的可执行二进制文件。xmrig 支持英特尔、AMD、Nvidia、Broadcom、高通、摩托罗拉、TSM 和德克萨斯制造的所有 CPU/GPU 处理器，并对其进行基准测试。

### GitHub xmrig 的特性

xmrig 基本上围绕在高性能计算环境中挖掘 CPU/GPU，以实现数字货币提取中的卓越处理。

#### 1.CPU 挖掘

这是一个利用中央处理器的能力从区块链账本中挖掘加密货币的过程。它指的是如何定义一个哈希，分配给一个块，并集成到加密货币世界中，以最快的速度提取货币。中本聪在一个普通的低端 CPU 中发明了它，他成功挖掘了一百万个比特币。

在最初的比特币时代，CPU 可以进行挖掘活动，但随着区块链技术的发展和回报变得越来越少，CPU 难以满足需求。它让位于 GPU 进行加密挖掘。尽管如此，CPU 还是设法在特定情况下生成了比特币，Monero 就是这样一个例子。但是比特币一代的产量无论如何都比不上 GPU。人们可能会认为 CPU 直接从计算机中挖掘加密货币。事实并非如此，他们在短时间内利用 CPU 的能力使用不同的算法产生散列。

#### 2.GPU 挖掘

CPU 可以以非常高的时钟速度处理任务，但它们在处理数据时缺乏并发性。但是 GPU 可以以并行模式处理多个数据流，并呈现高分辨率的视频和图像。CPU 的时钟速度可能很高，它可能有一个大而宽的指令集，但 GPU 由于其并行处理而在短时间内处理的数据量超过了 CPU 的任何标准。这是因为 GPU 在给定时间以并行操作的方式在数千个处理器的帮助下处理数据，每个核心专注于一个单独的计算。

除了图像和视频渲染，GPU 还可以处理人工智能学习、模拟、加密挖掘和许多其他科学计算中的任务。虽然 CPU 和 GPU 可以通过一系列交易来处理加密货币挖掘活动，但 GPU 凭借其独特的大规模计算能力，能够以更快的速度生成数字货币。它在以太坊、Ravencoin 和 Beam 比特币环境中得到有效使用。

#### 5.标杆管理

任何用于加密货币挖掘的机器都应该测试其使用不同功能生成用于挖掘货币的散列的能力。它能产生的货币量在这个基准中被测量。此外，还有一些模型可以根据不同显卡的操作性质来比较它们的性能。

哈希速率是它每秒可以执行的操作数，该指标通常用于 CPU/GPU 处理器的基准测试。此外，还有像 Cudo Miner 这样的软件，当安装在机器上时，会持续进行这种基准测试，以便用户可以确保他们挖掘的比特币是最有益的。

一些基准测试工具托管在云中，当客户端软件安装在一台机器上时，该工具通过互联网定期获取该机器的数据进行基准测试，并自动建议应该生成哪些比特币，以及这对用户有多大好处。

### 结论

CPU/GPU 挖掘是一项如此重要的活动，有几种算法可用于根据比特币类型的要求以任何给定的速率生成哈希。此外，有不同的基准测试工具可以监控任何机器的挖掘功能的性能，并建议应该生成的比特币以优化产量。
