## GIMP 像素艺术简介

正如你从像素这个名字可以理解的，像素艺术是一种在任何文档区域的每个像素上创建的艺术，它通常用于设计视频游戏的角色和对象。我们有一个非常有限的颜色范围，从调色板使用它们来创建像素艺术字符和对象。在 GIMP 中，pixel art 也可以通过使用工具面板的一些工具来创建，我们还需要做一些设置来在文档区域获得 1 x 1 的像素。所以让我们好好讨论一下。

### 如何在 GIMP 中创建像素艺术？

让我们有一个制作像素艺术字符的文档，因为我们将通过设计字符来理解像素艺术。所以去菜单栏的文件菜单，点击它，然后点击“新建”选项，向下滚动列表。

![GIMP pixel art output 1](img/3c1ce9d9131f9900a445126b85a3dba3.png)



单击后，您将看到此对话框。此对话框的设置在像素艺术中起着主要作用，所以请注意。

![GIMP pixel art output 2](img/67ca9427d30142cf5da8041cb6c4486a.png)



对于像素艺术，我们一般采用 60 x 60 或 64 x 64 的文档尺寸。你也可以拿小号的。我们这样做是为了让我们的文档上有像素，一旦我们有了像素，我们就可以轻松地一个像素一个像素地绘制艺术作品。我将把 60 x 60 作为我的文件尺寸。

![GIMP pixel art output 3](img/163e60400d26609ca992507162d1e5c9.png)



并确保选择以文档尺寸值为单位的像素。

![GIMP pixel art output 4](img/e074922187ee806281e33bfdf4a24b27.png)



现在转到这个盒子的高级设置，设置分辨率为 72。

![GIMP pixel art output 5](img/47833639ab5648dd8ae17a5408f1d71d.png)



这些都是非常重要的设置。所以要非常小心。现在我们将得到这种类型的文档，它非常小。

![GIMP pixel art output 6](img/489f5ab298e47eef1d5342c3d4af8e9f.png)



我们将在一个透明图层上创建我们的像素艺术，这样我们就可以在任何背景上使用它。所以点击图层面板的新建按钮，在这个白色填充的图层上新建一个图层。

![GIMP pixel art output 7](img/1615de534f651f82cda1cbf83dffbca6.png)



现在选择这个对话框中的透明度选项，点击确定。

![GIMP pixel art output 8](img/f2dc282f0020219e6a63cbbf5e4a8b46.png)



我们使用白色背景层只是为了在绘制像素艺术的过程中具有适用性。现在让我们放大一个文档，以便正确地看到它。

![GIMP pixel art output 9](img/84f82753aada6183a1457c98dda4968a.png)



像素艺术有任何对象的对称设计，所以让我们有一个网格线将这个文件分为两半水平和垂直。因此，进入图像菜单，进入指南选项，然后点击新指南(按百分比)选项。

![GIMP pixel art output 10](img/d8e7c1a4d169c14caa25c629173dd664.png)



然后将水平导轨的位置值设为文件尺寸的 50 %,并点击确定。

![GIMP pixel art output 11](img/b98c965afb5326348ea1d885c8f20e2b.png)



你现在会有这种类型的指南。

![GIMP pixel art output 12](img/e3ceb7cb51f46a01cdb4d1e9b2e40616.png)



再次使用相同的选项，这一次，在这个框中选择垂直，50 %的值，点击确定。

![GIMP pixel art output 13](img/ba6348e34b41f6a293cc961fd8cc22ff.png)



现在我们的对称向导准备好了。

![GIMP pixel art output 14](img/88c858122a1060282a8ee6c57676a4b8.png)



现在转到视图菜单，单击向下滚动列表的显示网格选项，在我们的文档上显示一个网格。

![GIMP pixel art output 15](img/f2294996d65065c8f0aa5a91a3e5aa97.png)



你会得到你的网格线。这不是像素艺术需要的网格大小。我们的要求是像素艺术的 1×1 像素大小，所以如果你得到这种类型的大指南。

![GIMP pixel art output 16](img/0afa91957ba490eea6a647f3c90ff468.png)



然后转到编辑菜单的首选项选项。

![GIMP pixel art output 17](img/b8f16ca9a55add71d46861576d594dfd.png)



点击这个对话框的默认网格选项，设置水平和垂直值为 1，然后点击确定。

![GIMP pixel art output 18](img/e0c66a6f41cd797cad8ec7a32aa3968c.png)



如果你没有得到你设置的网格值，那么关闭 GIMP 并重启它；然后，你将有这种类型的网格是 1×1 像素大小。

![GIMP pixel art output 19](img/5188d700cb721693d85ef5f669b28142.png)



我们使用铅笔工具来绘制像素艺术像素艺术，因为如果你使用画笔工具，它会给你一个羽毛的笔画，这不会创建一个像素。所以拿个铅笔工具。

![GIMP pixel art output 20](img/d2d2c52ede8b44b31562f59a17b5f053.png)



将铅笔尖的大小设为 1，这样我一次点击只覆盖 1 个像素。如果您增加铅笔尖的大小，它将根据大小一次覆盖多个像素。所以这是像素艺术工作中非常重要的一点。

![GIMP pixel art output 21](img/6ca498937c5e3105820da1acb8a952f4.png)



现在点击这个网格的方块。这些块表示像素。一个方块代表一个像素，一旦你用铅笔工具点击，它就会像这样画出来。

![GIMP pixel art output 22](img/271711d8db41c7b9c4170915f594535e.png)



如果你想删除任何绘制的像素，然后采取橡皮擦工具。

![GIMP pixel art output 23](img/dd4a0c6dd7a87bb1807d28225a1c01af.png)



并点击不需要像素，但是你会注意到它不会一次点击就清除像素。它只是淡化它，当你一次又一次地重复时，点击那个像素，它就会从那个区域消失。

![GIMP pixel art output 24](img/c6d028fdfe9b24b1451f0cc15303eb8d.png)



擦除一次点击擦除工具只需在橡皮擦工具的参数中进行一些设置。使其参数的渐变长度值为 0。

![GIMP pixel art output 25](img/7d70f5c7b309314a32badb8d3dfaafb5.png)



并启用“硬边”选项。

![GIMP pixel art output 26](img/4a0d7678c708c0ac7f7e2fbddf54ad52.png)



现在再一次，点击你想要移除的像素，这一次像素会在橡皮擦工具的一次点击中移除。

![GIMP pixel art output 27](img/f2425ba3fb2c4206f606d3b75afb9181.png)



现在让我们用像素艺术的方法画一个字符。采取椭圆选择工具。

![GIMP pixel art output 28](img/cd720e7cecb5f054655b8b7b7a50c9ba.png)



并启用此工具的抗锯齿选项，因为我们不需要像素艺术作品的平滑度。

![GIMP pixel art output 29](img/cb3707a01bb0c6cc3e0959f9d7a8750b.png)



现在做这个圆，在网格交叉点的中心调整。

![output 30.1](img/05cb0a7f800ae62a49c95887c574c555.png)



![output 30.2](img/15212f8f8b2b8a090b18aa9e368a1bd4.png)



现在转到路径选项卡，点击路径选项卡的选择路径按钮，在这个圆上选择一个路径。

![output 31](img/6786beaec0aa431916ed7c2c978882d1.png)



并启用路径选项卡中的眼睛按钮，以查看围绕此圆的路径。

![output 32](img/38571d0b7255a16f1d6a9c4fd726c3b5.png)



现在使用路径工具。

![output 33](img/c74d55c1f83dae065b568b75fa046940.png)



用此工具点击圆形路径，然后点击此工具参数的笔画路径选项。

![output 34](img/49f7b61b1db5e7971a26ca36a1a5320e.png)



在绘画工具选项中选择铅笔。

![output 35](img/493af00cb24e38464230bcad2dd9f448.png)



你可以在点击 Ok 之前选择你想要的颜色。

![output 36](img/6511358062e4593bdc51fd3b5c259965.png)



现在你会有一个像素的笔画，就像这样。

![output 37](img/4f7b2f94f2cb3a8718834e23921ab9be.png)



你可以注意到，除了一个像素之外，网格两边的像素数量相等；这是由于圆的曲线。可以删除，因为像素艺术需要对称。

![output 38](img/c3d707702d67fab74dac709165c615ac.png)



现在只需在像素上移动铅笔工具，通过保持对称性来填充颜色。
