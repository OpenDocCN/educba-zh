## Windows 简介

Windows 是一个管理个人电脑的操作系统，被称为一系列程序。它是微软开发的图形操作系统。这提供了一种数据、应用程序运行、玩视频游戏、流式视频和互联网接入的方式。微软最初成立于 1985 年 11 月，随着计算机内存的扩大、芯片处理速度的提高，当然还有互联网的发明，它也频繁地被修改。在 Windows 出现之前，个人电脑使用一套文本命令。之后发布了十几个版本，包括最新的 Windows 10 版本。

### Windows 的应用

Windows 已经开发了一系列有用的应用程序。这样的应用程序将帮助你进行许多流行的活动，包括网上冲浪、日历管理和音乐购物。点击开始按钮，然后点击用户图标，就可以打开这些应用程序中的大部分。若要查找您正在搜索的应用程序，您可能需要导航开始屏幕。

#### 1.网络浏览器

Internet Explorer 是最著名的互联网应用程序之一，主要是因为在过去 20 年中，许多 Windows 型号都捆绑了它。GUI 和最常见的 web 浏览器功能包括您最喜欢的书签页面。微软 Edge 取代了 Windows 10 的 ie 浏览器，但提供相同的功能。它通过 Internet Explorer 具有一些附加功能，包括易于阅读的选项以及将网页与其他 Windows 10 组件集成。

#### 2.生产力

Windows 8 和 10 包含多种生产力功能，如处理电子邮件和日程安排、遵循指示和组织笔记。邮件和日历是将你的账户链接到微软、谷歌和苹果的软件。在这些服务中，您可以同步邮箱和日历，并将它们显示在电脑的单个窗口中。
地图，阿炳地图应用程序，允许您使用所有相同的导航和搜索应用程序。它还引入了一些额外的功能，比如锁定和保存所有 Windows 工具上的指令。
最初随微软 Office 附带的一个 Note 是高级通知管理程序，但现在它是 Windows 10 的默认功能。OneNote 帮助您创建笔记列表，您可以搜索和共享该列表。

#### 3.设置

您可以在控制面板中更改计算机的设置和配置。Windows 8 设置应用程序为您提供了大部分设备设置，但不是传统的 Windows，而是现代的平板电脑模型。

### 优势

现在，我们将首先看到它的优势:

*   易于使用:对于熟悉以前版本的用户来说，更现代的版本可能也很简单。也许这是由于几乎所有系统的统一外观和感觉，以及自微软视窗时代以来文件系统的实现方式。
*   **可用软件:**配备了大范围的应用。这是微软在桌面操作系统和办公应用程序领域处于世界领先地位的原因。当您搜索您的业务程序时，如果该产品存在，Windows 版本很可能在某处可用。
*   **向后兼容性:**如果您目前使用的是旧版本，并且需要更多最新的详细信息，但又不想牺牲对您的业务需求很重要的任何旧的仅窗口程序的使用，那么这些程序可能会在新版本上运行。
*   **Plug & Play:** Windows 也是 PC 硬件 plug & play 支持的专家，作为普通家庭用户的操作系统。只要你安装了正确的驱动程序，Windows 通常能出色地识别新硬件。许多操作系统确实有即插即用&功能，但是需要手动控制的程度更低，频率更高。
*   游戏:如果你想要电脑游戏的最新创新，你需要 Windows。它提供了大量的游戏以及其他不同的支持游戏硬件。也有一些最常见的 Linux 游戏和一些 Mac 游戏，但没有可比性。然而，所有为 Windows 95 和 98 编写的过时游戏也不能在 XP 上运行。

### 不足之处

缺点如下:

*   **高资源需求:**相比之下，与其他系统的制造商相比，微软希望其消费者在计算机硬件上花费最多:更快的处理器(CPU)、更多的内存和更大的磁盘。微软也坚持认为这是因为它的客户需要所有的额外功能。
*   **Closed Source:** 如果用户和支持人员确切了解发生了什么，修复 Windows 问题将会简单得多。不幸的是，只有微软能够直接访问其软件的源代码，而且由于没有创建日志文件，其用户可以通过反复试验来推断是什么导致了他们的问题。
