## 后效梯度介绍

后效中的渐变是一个超级平滑混合模式系统中不同颜色的组合。因此，您可以为您的项目创建一个非常有效的颜色组合动画，它将作为任何特定工作的背景。为了在这个软件中制作这种类型的渐变效果，我们必须对这个软件的一些参数进行管理，例如效果和预设，使用不同的工具，制作预合成和一些其他重要的操作术语，以便在其中获得这种效果。我们将在 after effects 中通过一个令人兴奋的渐变颜色的运动背景的例子，利用这个软件的一些特性来理解这个效果。所以让我们来看看吧。

### 如何在 after Effect 中使用渐变来制作运动背景？

我们可以按照一些简单的步骤，通过使用 after effect 的渐变功能来创建一个好的背景，但在开始之前，让我们看看这个软件的工作屏幕，以便我们可以很容易地理解我们在学习这个主题时将使用的不同术语。

<small>3D 动画、建模、仿真、游戏开发&其他</small>

在工作屏幕的顶部，有一个我们称之为菜单栏的栏，它有许多菜单；在这下面，我们有一个工具栏及其右侧的属性栏，在这个工具栏下面，我们有三个部分，在左侧我们有项目面板和效果控制面板，在中心我们有显示窗口，我们可以在其中看到我们当前的工作，在右侧我们有一些参数选项卡，如预览选项卡，对齐选项卡，效果和预设选项卡和其他一些。您可以根据自己的选择在工作屏幕的任何其他位置重新排列所有这些部分。

![After Effects Gradient output 1](img/caef74b37e356f8c836f6c6abb8dc5e1.png)



现在让我们来学习一篇新的作文。对于新组合，请转到菜单栏的组合菜单并单击它。然后，单击合成菜单下拉列表中的“新合成”选项，进行新合成。

![After Effects Gradient output 2](img/f3a7f30e68162bcce8ca717ce6c1887e.png)



在“合成设置”框中为该合成进行所需的设置，然后点击该对话框的“确定”按钮。这将是我们的主要组成部分。

![After Effects Gradient output 3](img/8307e87f727558e169da1d8346b65a24.png)



再次制作另一个构图，命名为渐变，然后点击这个构图框的 Ok 按钮。

![After Effects Gradient output 4](img/97a1a2943afe0aed233973f9f6faa140.png)



选择渐变组成，并前往该软件的工具栏。然后从它的椭圆工具点击它。

![After Effects Gradient output 5](img/d38d715551fdb963eef8516060bedd03.png)



现在用这个工具画一个任意大小的圆。

![After Effects Gradient output 6](img/ab58aa7ab4cf7a37b5d4b8cb59cc3cf9.png)



转到这个软件的参数部分的效果和预设选项，搜索填充效果，双击这个效果。

![After Effects Gradient output 7](img/1f043f111aa2c7a6a361836fa3349241.png)



现在在效果和预设选项中搜索投影效果，双击它。

![After Effects Gradient output 8](img/46d1c1eda3a8b1603c495f29711cd024.png)



从“效果控制”选项卡中的填充参数更改填充的颜色。

![After Effects Gradient output 9](img/13e7634783b53cbb5f1124f856dd4a55.png)



请改变阴影的颜色，并增加它与圆圈的距离。

![After Effects Gradient output 10](img/d0fde21e63754279f602f0e8f22b2850.png)



现在进入图层菜单，点击下拉列表中的新选项。然后，从“新建”下拉列表中选择“实体”选项。

![After Effects Gradient output 11](img/cfd3f543e07bc47e9e35c8b0858aac39.png)



将其命名为背景，然后点击对话框中的确定按钮。

![After Effects Gradient output 12](img/3059f92c144738ac271699cf64ab8705.png)



再次从效果和预设选项添加填充效果到这个固体层。

![After Effects Gradient output 13](img/680727cd596df133bcf00c759fff9f80.png)



选择实心层的填充颜色。我会选择这种颜色。

![After Effects Gradient output 14](img/667e6793c8df92a0e9222bdaa68e8213.png)



现在，通过按 Ctrl + D 按钮并改变新椭圆的颜色，创建一个椭圆图层的副本。

![After Effects Gradient output 15](img/ed631ce1b3b045618c14ceb6cfaff241.png)



像这样用不同的颜色和大小复制这个图层。

![After Effects Gradient output 16](img/93057f205dc577cc7655e1a8100b6d99.png)



在 Comp 1 中放置渐变构图，这是我们的主要构图。

![After Effects Gradient output 17](img/ff7d8e95cfb7511cafd737cd7d3d5607.png)



通过按键盘上的 Ctrl + X 键从渐变构图中剪切背景层。

![After Effects Gradient output 18](img/695ce60908e5e9adc3a9e41473124828.png)



并将其粘贴到主成分中。

![After Effects Gradient output 19](img/de30bb59d13e2b2cf5bfb715ff1c8a5b.png)



现在选择渐变层，在效果和预设选项中搜索湍流效果，双击它，你就会得到这种效果。

![After Effects Gradient output 20](img/064ec79523578c990d5acc58d8e5f2e0.png)



将金额设置为 20。

![After Effects Gradient output 21](img/2893a52c1cd65e97850d1f2bab98d801.png)



并且对于该效果中的平滑度，复杂度为 2。

![After Effects Gradient output 22](img/82c589921131c27fbe4e68385a98db83.png)



现在在效果和预设选项中搜索方向模糊效果，双击它。

![output 23](img/43f2b24cd17797907e319427c8fcf6e3.png)



将模糊长度设置为 600。

![output 24](img/a430856ad790ab88520fb1301e50763e.png)



现在在效果和预设选项中搜索旋转效果并双击它。

![output 25](img/3f060d6044f2816926601262a0df2ab4.png)



根据您的选择，从参数中更改旋转选项的角度。

![output 26](img/3b2ec476f185788200f980d529d123c6.png)



将旋转半径设定为 100。

![output 27](img/647eda3856f152a7bd4a41dc6853a1cd.png)



在效果和预设选项中搜索快速模糊效果并双击它。

![output 28](img/8fd95c7b6190e8944d56542f99b89c00.png)



将模糊度设置为 10，以平滑所有颜色的混合。

![output 29](img/1aab0789c91b9dec5ab5f397150429fb.png)



现在点击这个组合的锁定按钮来锁定它。

![output 30](img/484df880bdfc143eace99e9a994bacc8.png)



现在转到“视图”菜单，单击“新查看器”选项。

![output 31](img/9d6b326753a129be6b2aff30445ce7b7.png)



对于每个合成，您的合成窗口屏幕将像这样分成两个窗口。

![output 32](img/54ebfb1bd5c4810550fe5d0095420f76.png)



现在，通过按下键盘的 Ctrl + D 键复制这些圆，并将它们放置在不同的位置，你会看到更多的线条将添加到主构图中。

![output 33](img/1acd4428ba3bfc124120062bd0f34edb.png)



请一遍又一遍地做，然后根据你的情况展开所有的圆圈，以便在主构图中产生良好的旋转渐变效果。

![output 34](img/6bf0b9dab053c659effc36c21a5a08e2.png)



现在通过按 Ctrl + A 按钮选择形状的所有层，然后右键单击并选择预合成选项进行预合成。

![output 35](img/d2f80c003cbfd31ea9cfb462592d8a4a.png)



命名为渐变形状，然后点击确定按钮。

![output 36](img/30b61ff4ca50445d69b8a67d322ad159.png)



现在点击 P 按钮的位置属性的渐变形状层。我们会让它充满活力。点击这个属性的秒表图标，然后放置关键帧播放头为 10 秒，并像这样改变这一层的位置。

![output 37](img/61ce3c689d466520aa10f050ed3592eb.png)



现在在效果和预设选项中搜索 CC 重复效果并双击它。

![output 38](img/18e2678d1408251f6c79f7f1a16c544d.png)



对于填充形状的左侧，区域设置“向左扩展”值为 800。

![output 39](img/51d9511a832779de4f742b1a76cca752.png)



为所有形状的平滑连接设置平铺展开选项。

![output 40](img/bb7adfec33d79229c11ddae9ccb64c50.png)



现在你可以播放动画了。

![output 41](img/a4a9642a3af04a665a4be872d70e8364.png)



这样，你就可以在这个软件中使用渐变效果了。
