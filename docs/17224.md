## Photoshop 中吸管工具介绍

吸管工具是 Photoshop 中一个非常有用的工具，用于在图稿中获得精确的颜色；使用吸管工具，用户只需将工具放在源对象上，就可以将特定的对象颜色与新对象匹配，即使不知道颜色值，也可以轻松快速地完成任务；此外，当用户链接文件中已经可用并可以在图像中任何其他地方使用的打印时，他们可以使用滴管将该细节阴影分配给调色板。

### 使用滴管工具的提示

下面是一些有用的建议，让你在一个地方使用吸管吸色器，而在其他地方使用:

1.  从可访问的任何类型的图片中，选择您想要的任何颜色。如果您关闭了许多图片，您可以在没有操作的图像中按。即使有额外请求或来自您计算机的信息，您也可以增加任何类型的颜色。只需按下并把滴管带到你想从图片面板显示的颜色。确保可以看到两个 app house 屏幕。
2.  选择您所在地区的样本。使用滴管设备时，您只有两种选择(在选项栏上)。可以选择您刚刚点击的像素的颜色(采样点)。或 Photoshop 标准，用于距离为 3-x-3、5-x-5、11-x-11、31-x-31、51-x-51 或 ten1-x-101 的像素颜色。您也可以决定目前只对您的能量涂层或所有零件进行取样。
3.  右击鼠标创建网页颜色。如果您的图片被右键单击(在 Mac 上按住 control 键单击)以增加背景产品的范围，您也可以使用复制颜色 HTML 选项来添加到您的页面。它将样本的颜色转换为十六进制颜色代码，并将其复制到剪贴板，以确保代码可以粘贴到 HTML 信息中。您也可以使用复制颜色的十六进制代码来处理 CSS 人标签的“=”部分。

![Eyedropper Tool in Photoshop (HTML Color)](../Images/e82eca55f843430f6e2740d40808c9ed.png)

<noscript><img class="alignnone wp-image-201963 size-full" src="../Images/e82eca55f843430f6e2740d40808c9ed.png" alt="Eyedropper Tool in Photoshop (HTML Color)" width="621" height="326" srcset="https://cdn.educba.com/academy/wp-content/uploads/2019/08/HTML-Color.png 621w, https://cdn.educba.com/academy/wp-content/uploads/2019/08/HTML-Color-300x157.png 300w, https://cdn.educba.com/academy/wp-content/uploads/2019/08/HTML-Color-620x326.png 620w" sizes="(max-width: 621px) 100vw, 621px" data-original-src="https://cdn.educba.com/academy/wp-content/uploads/2019/08/HTML-Color.png"/></noscript>

4.  在滴管和其他工具之间翻转。为了有效地完成绘画，在使用画笔、铅笔、颜色替代品、渐变、油漆桶或成形工具时，按住 Alt 键(Mac 上的选项)可以让您快速使用滴管。将按钮移至初始仪器。
5.  在底部和前景之间切换。如果第一种颜色的实例是鲜艳的，滴管工具会通过按住 Alt 键单击(可选单击 Mac)来提升新的背景色。当色样背景很鲜明时，全新的前景颜色会随着单击 Mac 的选项而提升。

### 我们怎样才能使用滴管？

吸管工具指定第一种/第二种颜色。

1.选择工具箱中的工具。

2.若要指定第一种颜色，请单击图片上的某个位置。看看第一个标记是如何针对工具包进行调整的。

3.要按背景色，请按 Alt(按住[ Alt]按钮)。查看工具箱设置上下文标记。

4.样本大小(滴管范围)可以在选项栏中转换:-

*   “点示例”记录了您点击的像素的颜色。
*   所有其他的例子从更广泛的领域检查典型的数字。

### 结论

对于您想要使用的每种颜色，您可能知道准确的 RGB 或 CMYK 值。这种情况下不需要滴管。但是如果您不知道确切的颜色值，滴管会立即为您找到这些值。你可以用这个仪器来确定你的精确颜色。您可以匹配现有图片的颜色。在相应的颜色中，您可以插入文字或图像。

Photoshop 确实有几个有用的工具，但滴管是对一些客户最有用的工具。滴管使用方便，意义重大。

使用滴管工具对给定颜色进行采样。你要做的应该是在滴管的正下方用一个图章或修复刷。选择上述选项之一后，选择笔的类型并转到示例区域，然后单击 Alt+。现在你可以在纸上的任何地方画出同样的装置。
