## Inkscape 图像转矢量简介

Inkscape Image to Vector 是 Inkscape 的一项非常有用的技术，通过使用这项技术，我们可以将任何图像格式转换成矢量图像。图像可能是 png 或 jpeg 文件格式。在 Inkscape 中，我们可以通过使用菜单栏的路径菜单的跟踪位图功能将任何图像转换为矢量格式，并且我们还必须对该功能的参数进行一些设置，以获得我们想要的矢量图像结果。有了追踪图像的矢量格式后，我们可以借助工具面板的节点编辑工具对其进行编辑。所以让我告诉你这个过程的步骤。

### 如何在 Inkscape 中将图像转换成矢量？

Inkscape 有一个功能，通过它我们可以在几秒钟内将图像转换成矢量，所以让我们试试这个软件的这个功能。为此，我将首先在这个软件的工作界面中放置一个图像。所以转到菜单栏的文件菜单，点击下拉列表的打开选项。

![Inkscape image to vector output 1](img/cdd1587e6592e9ab18e1ae8dc5df311f.png)



我将用这张熊猫图像来解释我们如何追踪黑白图像？

![Inkscape image to vector output 2](img/2eccc136a83cba3ed34549608a131475.png)



现在点击导入对话框的 Ok 按钮。

![Inkscape image to vector output 3](img/eb637df06f3e00b1d34ebfcee5424ddf.png)



你可以看到，如果我放大这个图像，它会像素化，因为它是一个光栅图像。所以现在，我将把它转换成矢量图像。

![Inkscape image to vector output 4](img/ed7a4d03db4dbf771ba68923133662ab.png)



现在选择你想要的图像，进入路径菜单，然后点击下拉列表中的跟踪位图选项。我将用这个熊猫图像来做这件事。

![Inkscape image to vector output 5](img/d8b11abc938d3c05fb994f33ac300b5e.png)



将在工作窗口的右侧打开一个跟踪位图对话框。

![Inkscape image to vector output 6](img/b5bee070aa87412db79eed377d942fe2.png)



在此对话框中，我们有不同类型的参数，通过对它们进行更改，我们可以获得不同的矢量图像跟踪结果。有一个“单次扫描”按钮，通过它我们可以开始跟踪选定的图像。所以我会点击它。

![Inkscape image to vector output 7](img/7d1a64c48f1bc89d163acae3022af317.png)



在“亮度阈值”选项中，您可以在描摹过程中控制图像中像素的密度值。所以根据需要我们可以增加或减少它。

![Inkscape image to vector output 8](img/ad46777a12ce7f7172d4f9630649c61a.png)



一旦你完成了设置，然后点击这个对话框中的预览部分的更新按钮，它会显示一个带有所选参数设置的描摹图像的预览。

![Inkscape image to vector output 9](img/57e7cfaf1f6071ed9db8de5b1578d0ec.png)



获得所需的跟踪结果后，单击此对话框的应用按钮。现在，我们将在不同的图层中的源图像上描摹图像。您可以通过从源图像移动它来查看它。

![Inkscape image to vector output 10](img/649289f0eeb03b01becdbb7248fe85cf.png)



你可以看到追踪层是矢量格式，因为我从来没有像素缩放。

![Inkscape image to vector output 11](img/4fe66b659b6238f2903ab336979de1c1.png)



如果需要，您可以反转跟踪的图像，为此，您必须在此对话框中通过单击反转图像选项的复选框来启用反转图像选项。

![Inkscape image to vector output 12](img/32b549b90a601a386eb68d961d5e1fca.png)



然后点击更新按钮，你将得到你所选择的图像的反转轨迹，你可以在你的插图工作中使用不同的目的。

![Inkscape image to vector output 13](img/4f7a1f7013a171192d10916b5c5de4be.png)



现在让我们将一幅彩色图像转换成一个矢量，为此，我拍摄了这幅美丽的树叶图像。

![Inkscape image to vector output 14](img/22e851234e836feecaaddb70f8e1e9dc.png)



首先，单击跟踪位图对话框的单次扫描按钮。

![Inkscape image to vector output 15](img/99fe3b66f6edd8cd2a69fac579d9c0de.png)



一旦我们点击更新按钮，它会给你这个美丽的图像跟踪结果。

![Inkscape image to vector output 16](img/b9b0e81b4c23d18b30d290b2061a73dd.png)



应用这个跟踪结果后，你会发现这个彩色图像的黑白矢量形式。

![Inkscape image to vector output 17](img/f38daa30cc0e68e935cdf1e4e97362cd.png)



您可以调整阈值，使跟踪的图像有变化。

![Inkscape image to vector output 18.1](img/d57ecf61665e12c6ac9593a5ad84f346.png)



![Inkscape image to vector output 18.2](img/a8cd475d62a8cdec106d14799057c4b3.png)



您可以使用这些参数来获得更精确的结果。

![Inkscape image to vector output 19.1](img/81c08db63088f6b8bb981c6f56e1b1e7.png)



![Inkscape image to vector output 19.2](img/b57027e3715e68408de99c3faeabd8c5.png)



有一个边缘检测选项，通过它我们可以只扫描图像的边缘，并获得这些边缘的矢量形式。因此，我将选择此选项，然后单击“单次扫描”按钮。

![Inkscape image to vector output 20](img/54175c0be59650562fae2486181c1506.png)



你可以看到它给了我们这个图像边缘追踪的结果。

![Inkscape image to vector output 21](img/88df40d39edef360137976560501cd47.png)



您还可以改变边缘阈值，以获得更精确的结果。

![Inkscape image to vector output 22](img/2c9daa4318c6a2436b6f3aa8768b1db8.png)



看，这是一个非常好的边缘追踪结果。

![Inkscape image to vector output 23](img/cd926b60d9c2083fb2def40c98df569a.png)



还有自动跟踪选项，它会跟踪你的图像。所以点击它。

![output 24](img/1e993518147e719c7337efde6fcb8737.png)



点击更新按钮后，我们将得到彩色跟踪图像，因为这是彩色图像。

![output 25](img/e0bd8802592c439bc3a46222c7e852df.png)



如果您对这种颜色跟踪不满意，那么您可以选择“多次扫描”选项。

![output 26](img/1b399e3ac88f028c9081d15e8efaa9f3.png)



现在点击“多重扫描”按钮。

![output 27](img/455e7d05ae5b922c8d0ea48eed21e375.png)



在单击“多次扫描”按钮之前，您可以通过增加和减少扫描框中的扫描值来管理扫描值。这意味着，如果扫描值为 8，那么它将跟踪您选择的图像 8 次，并为您提供图像中的 8 层不同深浅的颜色。

如果您增加扫描次数，那么您将获得更精确的结果，但是根据您的计算机系统的容量，它可能会花费时间或使您的计算机更慢。

![output 28](img/9abd10040dfbe30ef31e0b7839a6b5cf.png)



现在，在“多次扫描”中，我们在亮度等级选项的下拉列表中有颜色选项。所以我会选择颜色选项。

![output 29](img/4f8648d95696728a0445bdc4f2800c0b.png)



启用平滑选项以平滑跟踪图像的路径。

![output 30](img/7e469a2824485b790d230be7499cf898.png)



如果您想要移除跟踪图像的背景，可以启用此移除背景按钮。

![output 31](img/e6c6b57f5dc759a4f344dd6201cea5d5.png)



现在点击更新按钮，它会给你跟踪彩色图像预览。

![output 32](img/de44d0d6cbe7cc7cd05b478dbcc5ea44.png)



点击应用按钮，你可以看到这是最准确的图像转换成矢量格式的跟踪。

![output 33](img/92726606faf9223f102ab96b47a22fed.png)
