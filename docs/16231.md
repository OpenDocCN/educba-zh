## Autocad 和 Archicad 的区别

由匈牙利公司 Graphisoft 设计的建筑信息建模(BIM)平台是 ArchiCAD。它最初是在 1982 年为苹果麦金塔开发的。这个软件基本上被城市规划师、建筑师、设计师、室内设计师使用。ArchiCAD 是构建建筑物、室内、城市区域等模型所需的许多工程方面的解决方案。ArchiCAD 旨在满足 2D 或 3D 设计和 BIM 功能的需求，从基础到计划执行再到资源管理。Windows、Mac 和移动设备支持 ArchiCAD。

AutoCAD 是三维计算机辅助设计软件。AutoCAD 是为制造、规划、建筑、设计和施工而创建的。Autodesk 在 1982 年开发了这个软件，作为在微型计算机上运行的桌面应用程序。2010 年，AutoCAD 以移动和网络应用的形式发布，通常称为 AutoCAD 360。AutoCAD 是一个令人惊叹的软件，是许多建筑师、娱乐业、教育工作者、学生、自由职业者，甚至是大一新生的指南。AutoCAD 是制作 2D 图纸、草图、文档的高手；它还具有创建 3D 模型和可视化的能力。

### Autocad 和 Archicad 的主要区别

两者都是市场上的热门选择；让我们讨论一些主要的区别:

*   AutoCAD 是一种 CAD 系统，有助于绘图、三维建模和渲染。也可以用各种语言的脚本即兴创作。而 ArchiCAD 则是为 BIM 和虚拟建筑建模而设计的。它基本上用于创建建筑模型。
*   在 ArchiCAD 中，图形在模型中混合或网格化。而在 AutoCAD 中，用户将独立创建图纸。
*   有了 ArchiCAD，用户可以用真实的建筑元素如墙、窗、屋顶、板等虚拟地建造建筑物。而在 AutoCAD 中，用户从 2D 线创建绘图。
*   ArchiCAD 极大地提高了生产率，并具有更好的协调设计和基于建筑过程的计算机模型。而 AutoCAD 缺乏这一功能。
*   虽然 AutoCAD 可能易于使用和理解程序，但 ArchiCAD 需要更多时间来理解，因为选项设置太多。
*   AutoCAD 通常可以称为基于二维表示的电子绘图板，但也可以创建三维几何图形。而 ArchiCAD 是用来创建虚拟电子建筑的。应用程序可以理解建筑元素的属性以及如何连接它们。

### 结论

我们无法比较 Autocad 和 Archicad。两者都非常不同。Autocad 和 Archicad 在各自的风格中都有非常有用的特点。一方面，AutoCAD 是几乎所有技术人员和专业人员用于 2D 和 3D 建模的 CAD 软件，ArchiCAD 是使用实时编辑的虚拟建筑建模器。这两个软件都有一个干净的工作空间和界面，但仍然用于不同类型的工作。

Autocad 和 Archicad 各有优缺点，但不同的群体广泛使用两者。
