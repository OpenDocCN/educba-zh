## SketchUp 花园设计简介

SketchUp 花园设计是该软件中景观设计的一部分，我们可以用创建景观设计的相同方式创建花园设计。为了创建一个花园设计，我们需要了解一些工具和他们的工作，以及不同菜单的其他选项。您可以从外部资源导入一些 garden 人员，这意味着您可以使用一些从互联网上下载的 3D 建模组件。在 Sketchup 中，我们有一个在线的 3d 组件库，你可以找到这个名为 3D Warehouse 的库。所以我用一个简单的园林设计来给大家讲解一下园林设计。

### 如何在 Sketchup 中进行园林设计创作？

这完全取决于你想如何创建你的花园，它看起来是什么样子，根据这一点，你可以为你的模型创建一个美丽的设计，以及使用该设计作为景观设计。

首先，我会采取矩形工具是在工作屏幕的顶部，或按下 R 作为快捷键。

![SketchUp garden design output 1](img/27f0b3b1807b9c703c049159a3e29ca1.png)



现在我将绘制这个矩形作为一个花园的一小部分的布局。我没有为这个设计使用任何特定的尺寸。

![SketchUp garden design output 2](img/9d63134b928194049141555b78f36e81.png)



现在我将采取两点圆弧工具。

![SketchUp garden design output 3](img/4212ae1455f6c52469e73e4eb90302b7.png)



现在我将在这个矩形的一个角上画一个这样的圆弧。

![SketchUp garden design output 4](img/7dec1ea0a0b26572f5e19c2109390d14.png)



并通过删除不需要的弧形区域来创建这种形状。对于删除不需要的区域，只需选择并按下键盘上的删除按钮。现在我将使用选择工具选择这个形状的两条边。要选择两条边，请在选择一条边后按住 shift 键，然后单击另一条边。

![SketchUp garden design output 5](img/4838d78c309c91921721dd7d1ab687d4.png)



现在我将使用偏移工具，像这样将这些边偏移一段距离。

![SketchUp garden design output 6](img/34c4b32a85b7256a011e5aa93590aafe.png)



现在使用推/拉工具，给它增加一些高度。

![SketchUp garden design output 7](img/d56df36778b5b9655a7bee6c5a044be9.png)



现在选择整个对象并创建组，这样它就不会在创建过程中干扰其他对象。要创建组，请在选择后右键单击，然后单击下拉列表中的“创建组”选项。

![SketchUp garden design output 8](img/268bb77f2c6abcbc3fc2d6def9f0cb17.png)



现在让我们给地面添加一些设计来制作水池的侧面物体，这将在下一步中创建。为了创建这个设计，我将再次使用两点圆弧工具，并像这样绘制它。

![SketchUp garden design output 9](img/160e61f4189ee4713fb9c091cfe211f7.png)



现在再次使用推/拉工具，我将增加地板设计的高度。

![SketchUp garden design output 10](img/0aba56a5af2b114a1a0892c1bb0096bf.png)



选择地面矩形的其他两条边，并像我们在上一步中偏移矩形的第一条边一样偏移它们。

![SketchUp garden design output 11](img/15e9ad905a69627db1f4609c6fc74d57.png)



我将增加同样的高度，因为我们已经增加了花园的地面设计。

![SketchUp garden design output 12](img/9d4a651da8fce080ca365adac8125dcd.png)



现在用橡皮擦工具擦掉多余的线条，使这些物体成为一个单一的物体。

![SketchUp garden design output 13](img/55b6ac3fe89a6b00a8591428c86fdf59.png)



现在选择这些对象，然后将这个设计组合成一组，这样它将与其他设计分开。

![SketchUp garden design output 14](img/d21dcd14a90295d3ae2337ac4cd697a3.png)



现在让我们将材质应用到我们的设计中，要应用材质，请转到默认托盘选项的材质面板，它位于工作屏幕的右侧。

![SketchUp garden design output 15](img/fb25d20cb7cd5e716c4af9602e0ddf23.png)



如果您的工作屏幕上没有物料选项卡，您可以在窗口菜单中找到它。所以点击窗口菜单，然后转到默认托盘，并从下拉列表中选择材料选项。

![SketchUp garden design output 16](img/08e22df63a5b460a060ef5a8c911fc37.png)



现在从材料列表中，我将选择景观，围栏选项。

![SketchUp garden design output 17](img/5bbcf9bc99d87f4356771628cd828369.png)



现在，我将从景观选项的材质列表中为所选的地面区域选择岩石材质。

![SketchUp garden design output 18](img/111118b732c8d8c80b5aee15225a3bc6.png)



并且像这样应用它。

![SketchUp garden design output 19](img/2e5c9e0ac7dcfb6f7bc4142d6bb1d869.png)



我将在这个区域的深处应用混凝土块材料。

![SketchUp garden design output 20](img/37d3e8a757cd54417e8480006a02a5cc.png)



现在，我将选择这个花园设计一侧的墙状结构，然后对其应用木栅栏，为此，我将从材质面板中选择木栅栏，并使用铲斗工具单击选定的区域。

![SketchUp garden design output 21](img/5fcb77ff7c722c43759a68c92f271877.png)



现在，让我们将池水应用到这个选定的区域，我将再次转到“材质”选项卡，从材质列表中选择“水”。

![SketchUp garden design output 22](img/12fcc668e13898cdccb5421cf7fae695.png)



我将从水的材质列表中选择这个水池材质。

![SketchUp garden design output 23](img/528b70e570b19399f55c33fd87088aec.png)



并在铲斗工具的帮助下将其应用于选定的区域。

![output 24](img/910f85827a971bd621bea03a52fca3d7.png)



现在让我们在这里放置一些园艺人员，为此，我将转到 3D 仓库，您可以在窗口菜单的下拉列表中找到它。所以去那里点击它。

![output 25](img/9b5d610f86fd36c405d81a44c0a4bbcf.png)



一旦你点击它，一个 3D 仓库对话框就会像这样打开。在这里的搜索框中，我将寻找花园花盆。

![output 26](img/41e2f371c50533cab6970cfba260f819.png)



并从锅的结果中下载这个。你可以根据你的设计选择任何其他的。

您可以使用工具面板的缩放工具，根据设计中所需的大小对其进行缩放。你也可以把植物放在这个花盆里。

![output 32](img/f9a4b7a10ff390670d2403f47bc2e725.png)
