## Python NOT 运算符简介

Python 中的 NOT 运算符属于逻辑运算符的范畴。

Python 中的逻辑运算符用于返回布尔值的条件语句，该值可以是真或假。根据这些条件的真值，程序决定其执行流程。

<small>网页开发、编程语言、软件测试&其他</small>

NOT 运算符在操作数为 false 时返回布尔值 true，在操作数为 true 时返回布尔值 false。

用外行人的话来说，一个 NOT 操作者可以被认为是一个成功的悲观者和一个失败的乐观者。

一个悲观的个体可以被认为是一个陈述，他的消极思想可以被认为是操作数。然而，尽管有消极的想法，个人还是获得了成功，这可以被认为是返回的布尔真。

### Python NOT 运算符的示例

让我们举一些 Python 中 NOT 操作符的例子:–

#### 1.直接使用布尔值

**代码:**

```
a = True
b = False
print('Result of not a is : ', not a)
```

**输出:**

![Python NOT Operator output 1](../Images/56309af87d9dd627668c72cfc18fe623.png)

<noscript><img class="alignnone size-full wp-image-327683" src="../Images/56309af87d9dd627668c72cfc18fe623.png" alt="Python NOT Operator output 1" width="353" height="64" srcset="https://cdn.educba.com/academy/wp-content/uploads/2020/03/Python-NOT-Operator-output-1.png 353w, https://cdn.educba.com/academy/wp-content/uploads/2020/03/Python-NOT-Operator-output-1-300x54.png 300w" sizes="(max-width: 353px) 100vw, 353px" data-original-src="https://cdn.educba.com/academy/wp-content/uploads/2020/03/Python-NOT-Operator-output-1.png"/></noscript>

变量 an 用布尔值 True 初始化

用布尔值 False 初始化变量 b

“a”上的非条件运算符反转 a 的布尔值；因此，结果是错误的

#### 2.对整数使用比较运算符来获取布尔值

**代码:**

```
a = 2>3
b = 3==3
print('result of not a is : ', not a)
```

**输出:**

![Python NOT Operator output 2](../Images/c7dcff417368dec9787a3dbd4b9644a1.png)

<noscript><img class="alignnone size-full wp-image-327684" src="../Images/c7dcff417368dec9787a3dbd4b9644a1.png" alt="Python NOT Operator output 2" width="296" height="66" data-original-src="https://cdn.educba.com/academy/wp-content/uploads/2020/03/Python-NOT-Operator-output-2.png"/></noscript>

变量 an 用条件 2>3 初始化，结果为假

变量 b 用条件 3==3 初始化，结果为真

“a”上的非条件运算符反转 a 的布尔值；因此，结果证明是真的

#### 3.在列表上使用成员运算符获取布尔值

**代码:**

```
a = 2 in [3,6,8,9,10]
b = "p" in "programming"
print('result of not a is : ', not a)
```

**输出:**

![Python NOT Operator output 3](../Images/d7569f146275689d7b55c8e4e9865a2d.png)

<noscript><img class="alignnone size-full wp-image-327686" src="../Images/d7569f146275689d7b55c8e4e9865a2d.png" alt="Python NOT Operator output 3" width="296" height="66" data-original-src="https://cdn.educba.com/academy/wp-content/uploads/2020/03/Python-NOT-Operator-output-2-1.png"/></noscript>

变量 a 用使用成员运算符确定布尔结果的条件进行初始化。2 不在列表中；因此，结果 if 为假

“a”上的非条件运算符反转 a 的布尔值；因此，结果证明是真的

#### 4.在字符串上使用标识运算符获取布尔值

**代码:**

```
a = "python" is "python"
print('result of not a is : ', not a)
```

**输出:**

![Examples of Python NOT Operator 5](../Images/a3631583aa00be8679f2d620f092038b.png)

<noscript><img class="alignnone wp-image-327689 size-full" src="../Images/a3631583aa00be8679f2d620f092038b.png" alt="Examples of Python NOT Operator 5" width="353" height="64" srcset="https://cdn.educba.com/academy/wp-content/uploads/2020/03/Python-NOT-Operator-output-1-1.png 353w, https://cdn.educba.com/academy/wp-content/uploads/2020/03/Python-NOT-Operator-output-1-1-300x54.png 300w" sizes="(max-width: 353px) 100vw, 353px" data-original-src="https://cdn.educba.com/academy/wp-content/uploads/2020/03/Python-NOT-Operator-output-1-1.png"/></noscript>

变量 a 用使用标识运算符确定布尔结果的条件进行初始化。“python”和“python”是一样的；因此，该条件为真。

“a”上的非条件运算符反转 a 的布尔值；因此，结果是错误的

#### 5.在 NOT 运算符上使用逻辑运算符(AND、OR)

**代码:**

```
a = 2 in [3,6,8,9,10]
b = 3 == 3
print('result of not a is : ', not a and b)
```

**输出:**

![output 5](../Images/0fc39d73b50cfd3d6c1390c633af9a91.png)

<noscript><img class="alignnone wp-image-327693 size-full" src="../Images/0fc39d73b50cfd3d6c1390c633af9a91.png" alt="output 5" width="296" height="66" data-original-src="https://cdn.educba.com/academy/wp-content/uploads/2020/03/Python-NOT-Operator-output-2-2.png"/></noscript>

如上所述，变量 a 采用布尔值 False

如上所述，变量 b 将布尔值设为真

执行的优先顺序不是，然后和。所以首先 a 不为真，然后 b 为真，结果为真

#### 6.使用多个 NOT 运算符

**代码:**

```
a = 2 in [3,6,8,9,10]
print('result of not a is : ', not not a)
```

**输出:**

![output 7](../Images/b5c4ded6a446680fb142af2a97b364c4.png)

<noscript><img class="alignnone wp-image-327699 size-full" src="../Images/b5c4ded6a446680fb142af2a97b364c4.png" alt="output 7" width="353" height="64" srcset="https://cdn.educba.com/academy/wp-content/uploads/2020/03/Python-NOT-Operator-output-1-3.png 353w, https://cdn.educba.com/academy/wp-content/uploads/2020/03/Python-NOT-Operator-output-1-3-300x54.png 300w" sizes="(max-width: 353px) 100vw, 353px" data-original-src="https://cdn.educba.com/academy/wp-content/uploads/2020/03/Python-NOT-Operator-output-1-3.png"/></noscript>

**Note:** Applying for NOT operation odd number if times provides us with the opposite of the initial Boolean value, and applying NOT even number of times gives us the initial value itself
