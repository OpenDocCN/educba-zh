## 树大小替代的定义

TreeSize 是一个硬盘分析器工具，用于分析磁盘中的所有文件和文件夹，以确定每个资源占用的空间量。它有助于用户通过易读的图表和表格查看完整的信息和完整的数字。只需点击一下鼠标扫描系统和磁盘，在单个屏幕上显示结果，并相应地删除不必要的文件以释放内存空间，从而使决策过程变得简单。人们可以很容易地管理所有的硬盘驱动器，而不用做任何复杂的计算。市场上有许多具有各种特征的树尺寸的替代物。

### 树木大小的替代方案

TreeSize 工具的基本功能是分析每个文件夹及其子文件夹，并根据大小、文件类型、类别等显示它们。TreeSize 是一个功能强大的硬盘工具，被广泛使用，但市场上有许多替代产品。其中一些列举如下:

#### 1.WinDirStat

这是一个用于磁盘存储统计的开源工具，可用于不同版本的 Windows。它以表格格式表示数据，并以简化的视图显示各种文件夹的详细信息。在详细视图中，一列包含根据文件大小排序的目录，整个内容显示在彩色矩形中。它还根据各种文件类型显示数据，如音频、视频。pdf 等。此外，为了目录结构的清晰视觉，它使用了适当的阴影。

#### 2.大视角

该实用程序适用于 Mac OS。它允许用户查看硬盘存储在一个很好的图形格式，这使得它很容易和清楚地理解用户了解每一个文件的细节。它允许用户为文件名、扩展名、临时文件、文件创建等分配不同的颜色，以使采取行动更简单。它为用户提供了从界面上删除文件、文件夹等的便利，只是为了节省磁盘存储空间。

#### 3.太空堡垒

这个磁盘分析器使用户能够清楚地了解所有文件是如何分布在整个 pc 上的。除了文件和文件夹的各种数据，如类型(视频、音频、Docx 等)、文件大小，它还显示文件位置，用户可以通过它了解大文件夹及其下的小文件夹的位置。用户可以在显示的数据中应用各种过滤器，并可以通过双击显示的文件/文件夹来更深入地了解信息。

#### 4.磁盘分析器专业版

该实用程序仅适用于 Windows 操作系统。它的基本版本可以免费下载和使用，提供所有必要的功能。虽然这是付费的高级版本，向用户提供一些高级选项。它显示了计算机中磁盘空间消耗的详细情况。用户可以在几秒钟内轻松删除较大的文件、临时数据和垃圾文件。人们可以根据文件大小、文件所有权、文件日期、时间和类型来查看文件和文件夹的数据。用户可以坚持自己的搜索条件，可以找到重复的文件、文件夹、图片等。

#### 5.猴面包树

它是一个为 Linux 用户设计的磁盘分析器，用于分析计算机硬盘，并以图形格式显示磁盘信息。它分析磁盘存储并显示文件夹结构及其详细信息。它使用完整的文件系统扫描来检索系统中每个文件和文件夹的详细信息及其性能。它允许扫描在 2 个选项，即，e 完整的文件系统扫描，和单个文件夹扫描。进行完整的文件系统扫描是为了获取所有文件的信息，尤其是内存的大分区(删除其中一些并释放空间)，而单个文件夹扫描允许根据用户要求扫描每个文件/文件夹。

#### 6.菊花盘

此实用程序仅适用于 Mac OS X 用户。它分析磁盘存储，并以交互式地图的形式显示各种文件和文件夹的数据。通过这些地图，文件和文件夹占用不同的内存在磁盘空间是清晰可见的，查看哪个用户可以采取相应的快速行动。它使用一个非常快速的引擎来收集磁盘数据，并在交互式的高质量图表中显示数据。

#### 7.磁盘检查器

这是一个强大的磁盘管理工具，广泛用于管理不必要占用磁盘空间的大文件。它执行高速扫描，以分析各种文件和文件夹的磁盘存储，并以图形和图表的形式显示数据及其详细信息。除了对可用的空闲和占用空间进行常规的 PC 检查之外，它还为用户提供了各种磁盘管理功能。

#### 8\. Xinorbis

Xinorbis 是一个功能强大、简单易用的磁盘分析器，广泛用于扫描和分析硬盘存储。执行扫描后，它会提供硬盘驱动器的图形深度细节，并以表格和图形的形式向用户显示结果。当用户希望在文件分发环境中根据内容、大小和结构检索数据时，使用 XInorbis 被认为是最好的方法。这取决于用户分析整个硬盘驱动器或他/她想要分析的特定文件夹的要求。它允许根据文件类型和大小进行分发，并利用自己的搜索引擎搜索和保存报告。
