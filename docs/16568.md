## XMP Lightroom 简介

Lightroom XMP 可以理解为一个文本文件，其中包含元数据或我们对原始照片文件所做的任何更改，以便对它们进行不同的操作。Lightroom 本身将数据存储在目录中，但它仅对 Lightroom 可读，并且只作为一个整体存在于我们的目录中，但 XMP 文件为我们提供了最好的功能，可以将我们编辑的图像更改保存在一个单独的文件中，以便于检索和使用它们。所以今天，我将告诉你 XMP 的所有重要方面，并告诉你如何保存它们，以便你也可以使用其他编辑软件的 Lightroom 文件。

### 如何保存 XMP 文件？

在您进行以下设置之前，Lightroom 不会自动保存 xmp 文件。这只是两步 0 设置，让我告诉你这一点。首先，转到菜单栏的编辑菜单，您可以在用户屏幕的顶部找到它，并从它的下拉列表中选择目录设置选项，或者您可以按 Ctrl + Alt +来快速访问该选项。
