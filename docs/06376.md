## 有角物质形态的定义

Angular material form 用于包装 Angular 的几个材料组件，并应用常见的测试字段，如浮动标签、消息提示和下划线。基本上表单域指的是包装器的组件。我们可以将组件编写为 mat-form-field，在设计应用程序时，我们需要使用多个材料组件。

### 角形材料形状概述

*   我们使用 mat-select 表单控件从选项集中选择值，这类似于本地 select 元素。
*   每个 mat-option 将包含 value 属性，该属性用于设置当用户选择指定选项时所选择的值。mat 选项的所有内容将由用户显示。
*   正如我们所知，角状物质将支持所选择的元素，它是天然的，并在垫子-形态-场的组成部分中。
*   本机控制将包含可访问性、性能和可用性优势。在内部使用本地 select 将导入 mat 输入模块，并将 mat 本地控件属性添加到 select 元素中。
*   mat-select 和 select 将支持对核心表单和反应式表单模块的指令。本机模块 select 和 mat-select 也支持 compareWith 函数。
*   我们可以使用与 select 和 mat-select 组件一起使用的 mat 表单字段中的许多特性。这包括前缀、后缀、主题和错误消息。
*   为了将选项添加到 select 中，我们需要将 mat-option 元素添加到 mat-select 元素中。
