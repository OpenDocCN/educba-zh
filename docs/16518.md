## InDesign 链接文本框简介

InDesign 链接文本框可以理解为通过链接线相互连接的溢出文本的文本框架。在这个软件中，我们处理大量的文本内容，所以当它们不适合我们绘制的文本框架时，我们需要对它们进行排列。我们可以通过许多方法将文本排列到链接的文本框中，我们用来排列它们的方式完全取决于文本内容的密度。在这里，我将告诉你如何链接少量文本内容和大量文本内容的文本框，使它们具有不同的用途。

### 如何在 InDesign 中链接文本框？

我们需要将文本框链接起来，以不同的方式排列我们的文本内容，我将告诉你如何在这个软件中做到这一点。因此，我将创建一个信纸大小的文档，其中只有一页。我将从工具面板中选择文字工具，然后画一个这样的文本框。

![InDesign link text boxes output 1](img/f56deeb96cf9e1d9e3c13d24cb1e1687.png)



现在，我将把一些文本内容放入这个文本框架，并选择这个文本到菜单栏的文件菜单并点击它。在这里，在它的向下滚动列表中，我将选择放置选项，或者你可以按键盘的 Ctrl + D 键作为它的快捷命令。

![InDesign link text boxes output 2](img/d3d83eb4baf56703d61b27eb36bb687b.png)



将打开一个放置对话框。我将在我的计算机中导航我的文本内容，并从它的保存位置选择它，然后单击此对话框的打开按钮。

![InDesign link text boxes output 3](img/9c3453b8f98918f2dd4d96ec84fa0ff3.png)



我把文本内容放在文本框里，就像这样。

![InDesign link text boxes output 4](img/c840695e0a4f037e5ed990c5124ee267.png)



如果您想在放置文本内容之前不需要绘制文本框架，只需从 place 选项中选择您想要放置在该软件中的文本内容，一旦您选择了文本内容，它的指示将出现在鼠标光标上。然后你可以绘制鼠标路线来绘制文本框架，它会自动放置在该框架中。

![InDesign link text boxes output 5](img/2693641d64eab18d10911991e0c6e873.png)



在这里，您可以看到文本框右下角有一个红色的加号按钮，表示文本内容不适合当前文本框。在这个软件中，我们称之为溢出文本。

![InDesign link text boxes output 6](img/f087c117f03f1aaddb96684e25f926b0.png)



我将单击红色按钮，鼠标光标上将出现额外的文本指示，符号将变为类似按钮的箭头。

![InDesign link text boxes output 7](img/9612d5b38c7c7a3914e0caa153f2757b.png)



现在，用带有额外文本指示的相同鼠标光标，我们可以绘制新的文本框架。

![InDesign link text boxes output 8](img/a2f4b2a4bf9b876efd44fbf6f54dba4c.png)



一旦我们在绘制了这个框架后释放鼠标光标，前一个文本框架的额外文本将进入绘制的文本框架。

![InDesign link text boxes output 9](img/6b8eb8fd5965b44511a330ead86f42ef.png)



您也可以将这些额外的文本链接到已经绘制的文本框架中。例如，这一次我将从工具面板中选择文字工具，并绘制一个文本框。

![InDesign link text boxes output 10](img/78b6a1a1f29f2b3a001ec1924770a841.png)



现在点击第二个文本框的溢出文本按钮。

![InDesign link text boxes output 11](img/5590e4a0bbaaf65414b3bfdbfebdce6c.png)



并点击第三个绘制的文本框架，一旦我点击该框架的额外文本进入这个框架。

![output 12](img/ac3774b07bc3c35801df2c89596a6b78.png)

您可以看到第三个框架中的文本在第二个框架中上移，这表明这些文本框架是相互链接的，当我们在任何链接的框架中进行更改时，其他框架的文本内容都会受到影响。

![output 15](img/a605070a899008ca9c085ce87d856348.png)



如果你想看到这些文本框之间的链接，你必须到菜单栏的查看菜单，并到下拉列表的附加选项。在此选项的新下拉列表中选择“显示文本线索”选项。

![output 16](img/6a60ac4e33a8b173555cbd1cd846c0f8.png)



你可以看到这些文本框像这样相互连接。

![output 17](img/f7ca7cb9e8f1ac6e6a0d647c4886d786.png)



如果你不想一次又一次点击溢出文本的红色加号按钮，你可以做一件事，首先点击溢出文本框架的红色加号按钮，然后按住键盘的 Alt 键，然后绘制下一个要放置额外文本的文本框架。当您在绘制文本框架后释放鼠标按钮时，额外的文本将出现在绘制的文本框架上，如果有更多的额外文本，鼠标光标将自动显示额外的文本。

![output 18](img/7b482d8aeb6d08b86d99adcec842e2e3.png)

如果您的工作屏幕上没有这样的面板，那么您可以在该软件的窗口菜单的下拉列表中找到此页面面板，或者您可以按 F12 作为其快捷键。

![output 22](img/32adc1a3780c375b066c07e5dd8744e1.png)
