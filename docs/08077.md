## 什么是 GitHub？

GitHub 和许多 IT 巨头一起向学生社区免费提供一套工具和服务，以发掘他们的才华，让他们尽早进入 IT 领域，并在年轻时塑造他们的思想/职业生涯。这些工具和服务包括桌面集成开发环境(IDE)、域和 Web 托管、云访问、AWS 云服务培训、与专业人员的协作、开发和托管应用程序的 Azure 云服务。年满 13 岁及以上的附属学院或学校的学生有资格注册并利用这一机会塑造他们的未来。学生可以在云中使用 Azure 或 AWS 或 DigitalOcean 服务器构建他们的应用程序，并通过 GitHub 环境创建高度安全的基于 SSL 的网站。在这篇文章中，让我们深入了解细节。

### 学生如何使用 GitHub？

GitHub 及其合作伙伴为学生提供了一个挖掘潜力和证明自己的绝佳平台。下面是一些详细的使用案例。

<small>网页开发、编程语言、软件测试&其他</small>

*   #### Develop impressive projects

学生可以利用 GitHub 为他们提供的设施，在他们感兴趣的领域的开源环境中创建一个端到端的项目。他们可以创新地做一些对开发者有用的事情。例如，他们可以尝试创建一些具有更多功能的替代工具来替代数据科学、语音识别、视频匹配、自然语言处理等领域的现有工具，

*   #### publicize the work done

GitHub 为学生在公共媒体上展示他们的作品提供了一个完美的平台。他们还可以从开发人员社区获得对他们工作的改进。他们的编码能力、创新技能、表达能力和其他技能将会被他们未来的招聘人员看到，这将比获得一份好工作的学历证明更有说服力。

*   #### Contribute to the community

学生可以通过 GitHub 平台为开源社区做出贡献，并向开发人员社区展示他们的技能，甚至在他们开始职业生涯之前就获得认可。学生们在人工智能、数据科学和区块链方面有更多的途径来追求他们的兴趣。

*   #### Acquisition of repurchase

GitHub 保存了学生在项目和学术课程中所需的大量有用信息。学生可以参考 GitHub 中大量可用的资源库并从中受益。

*   #### Cooperation

GitHub 为学生提供了与其他项目成员合作的绝佳机会，无论是其他学生还是专业开发人员。它教会他们协作的艺术、团队合作的力量以及许多其他软技能。

### 它是如何工作的？

学生必须在 GitHub 注册，并获得一个登录帐户，该机构提供的电子邮件 id 应与该帐户相关联。学生可以使用帐户凭据登录教育包，输入他们所在机构的详细信息，并配置他们将使用的服务。获得 GitHub 团队的批准后，他们就可以开始在 GitHub 中工作了

### 它是如何为学生工作的？

获得登录详细信息和 GitHub 批准后，学生可以在以下位置开始活动:

*   他们可以开发自己的项目，并邀请他人来改进它
*   他们可以访问许多存储库中的信息
*   他们可以作为团队成员在许多其他项目中合作
*   将他们的作品发布到 GitHub 上，让其他人观看和欣赏
*   加入 GitHub 中的任何社交团体，并积极参与 GitHub 门户网站内的信息交流

### 它是如何为老师工作的？

作为 GitHub 教育平台的一部分，GitHub 教室的特点是:

*   教师可以为学生的作业创建不同的工作流
*   GitHub 是一个面向教师的工具，它生成初始代码，设定任务的方向
*   工作流中定义了截止日期、里程碑、仪表板和其他测试框架
*   作业或课程在 GitHub 中开始
*   根据学生的意见，评估作业的进度
*   教师可以专注于他们的核心教学活动，同时工作流报告作业状态，仪表板显示进度。
*   当 GitHub 负责后续活动时，教师可以花更多的时间与学生在一起
*   老师能很好地洞察学生的细微活动，一切进展顺利
*   GitHub 专家提供故障排除来解决任何问题
