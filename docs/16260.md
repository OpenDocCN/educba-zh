## Inkscape 和 Illustrator 的区别

在本文中，我们将了解 Inkscape vs Illustrator。Inkscape 软件和 Illustrator 软件通常用于与矢量图形相同的目的。图形设计人员使用这两者来制作矢量图形并对其进行编辑，但两者各有特点，互不相同。这些不同的特性使它们与众不同。在本文中，我们将对这两个软件进行观察，以了解它们在我们的项目中的用途。我们将逐一查看这两个软件，并制作该软件的对照表来分析它们。

#### 什么是 Inkscape？

这是一个矢量图形软件，图形设计师使用它来从他们的项目中获得高质量的结果。这是一个由不同操作软件支持的软件，这些操作软件包括 Windows、Mac OS X 和 Linux。它是免费的开源软件，支持 SVG(可缩放矢量图形)和 W3dC 文件格式。您可以在此软件中导入和导出不同格式的文件，如 PDF(可移植文档格式)、PNG(可移植网络图形)、SVG(可缩放矢量图形)、EPS(封装 PostScript)和 AL。

#### 什么是 Illustrator？

它也是一个矢量图形软件，但由世界著名的 Adobe systems 开发，是其 Creative Cloud desktop 应用程序家族的一部分。它还兼容不同的操作软件，即 Windows 10 和 Max OS，这是 Adobe 公司于 1987 年以 C++计算机语言推出的。您可以使用该软件设计矢量图形，即 T2 标志设计、横幅设计、模板设计和不同类型的插图等。它还支持 Inkscape 核心格式 SVG(可缩放矢量图形)。

### Inkscape 和 Illustrator 的主要区别

让我们讨论一下 Inkscape 和 Illustrator 之间的一些主要区别:

*   Inkscape 的工作原理是节点编辑方法，对于编辑节点，我们使用其中的节点工具，而 Illustrator 使用直接选择工具来处理任何图形路径的节点。两者都有做这些工作的专业工具，并从他们的工作中给出最好的结果。
*   我们有不同的快捷方式在两个软件中应用相同的命令。例如，在两个软件中，撤消命令是通过按下 **Ctrl+Z** 来完成的，但重做命令是通过在 Inkscape 中按下 **Ctrl+Y** 按钮来完成的，而在 Illustrator 中，我们必须按下 **Ctrl+Shift+Z** 来执行重做命令。
*   在 Illustrator 中，我们必须按下**Alt+鼠标滚轮**来放大或缩小，而在 Inkscape 中，我们必须按下**Ctrl+鼠标滚轮**来放大或缩小。
