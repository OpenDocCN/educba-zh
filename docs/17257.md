## ZBrush 热键介绍

在这篇文章中，我们将了解 ZBrush 热键。热键不仅可以节省你的时间，还可以省去浏览多个嵌套菜单的麻烦。现在想象一下，如果你一遍又一遍地使用同一个命令，那么使用快捷方式是正确的，因为它肯定会节省你的时间。Zbrush 不仅提供了默认的热键，而且你可以用适当的修饰符自定义每个热键。还有画笔的热键，这是选择画笔的字母序列，如果你反复使用特定的画笔，这也很方便。热键还可以提高生活质量，比如弹出窗口，让工作流程变得更快。

**第一步:**我们先从第一个热键或快捷方式开始，就是打开一个项目。按 Ctrl + O 并导航到一个. zpr 文件，该文件将打开项目。

<small>3D 动画、建模、仿真、游戏开发&其他</small>

![ZBrush hotkeys 1](img/2c8e5969a6e5d7e078996c0d30a5c676.png)



**第二步:**接下来是保存热键为 Ctrl + S 的项目，系统会询问您要保存的位置。

![ZBrush hotkeys 2](img/89b2797af0826f0b2e951d324c443603.png)



**第三步:**现在假设你画了一个笔画，你想撤销它，只需按 Ctrl + Z 来删除我们已经创建的笔画。如果你想重做你已经撤消的操作，那么按 Ctrl + Shift + Z。

![ZBrush hotkeys 3](img/5a3ae0a86cbeda15209b48de1faf385b.png)



**步骤 4:** 现在让我们看看弹出的选择热键。要打开工具弹出窗口，请按 F1。

![ZBrush hotkeys 4](img/39599d531dd19bc08885f376b7392348.png)



第五步:然后按 F2 打开画笔面板，或者你也可以按 b。

![ZBrush hotkeys 5](img/1dd7e73a4b4491101eba59a167669a16.png)



**第六步:**然后 F3 会打开你需要的笔画类型的弹出框。

![ZBrush hotkeys 6](img/0030f62fda00fa7d34f7ed9bdc1978d4.png)



第七步:然后 F4 是阿尔法弹出窗口的热键。

![ZBrush hotkeys 7](img/39b70c73594c0baa694e3a1dd881fd80.png)



第八步:F5 弹出纹理。

![ZBrush hotkeys 8](img/194ec08922d25a979eb46fa29480df94.png)



第 9 步:顶部打开材料弹出按 F6，但要确保你离开画布。

![ZBrush hotkeys 9](img/2d3b32143d0ab39c02db8a9ab75f9513.png)



**第十步:**然后按空格键或右键将显示快捷菜单。

![ZBrush hotkeys 10](img/458a1117497dcdc491686efb05c0c3c0.png)



步骤 11: 然后使用 Tab 热键将显示或隐藏浮动调色板。

![ZBrush hotkeys 11](img/3c577d51b1f76346758fb40e1476ab26.png)



**步骤 12:** 现在对于导航，有一些热键在编辑模式下适用于 3D 网格。要自由旋转，请单击并拖动背景，要移动，请按住 alt 键并单击并拖动背景。如果要将旋转限制为 90 度，请在单击并拖动的同时按住 Shift 键。现在要缩放，首先你必须按住 alt 键并点击鼠标，然后松开 alt 键并拖动背景。您也可以通过按住 shift 键并单击鼠标，然后松开 Shift 键并拖动鼠标来沿 Z 轴旋转。

**步骤 13:** 使用【逗号键】来显示或隐藏 Lightbox，然后通过双击任何一个缩略图就可以加载它。

![step 13](img/33c49534cc27c5ec8a9f41f0f1e58364.png)



**步骤 14:** 现在使用热键 B–S–t 加载标准笔刷。同样，我们可以选择其他笔刷，如使用 B–M–E 移动弹性笔刷和使用 B–C–l 绘制粘土笔刷。

![step 14](img/db585ff6b935729ffe1f0eeb3cbd30ef.png)



**第 15 步:**画图时按住 Ctrl 会使用蒙版笔。

![step 15](img/8919a0b46f336fb0b52ab2399203cd01.png)



**步骤 16:** 按 Ctrl + Shift 将使用如图所示的剪贴画笔刷。

![step 16](img/255f2b93a70ffbb59426f622eaa30e80.png)



**步骤 17:** 现在，假设我们要为这个弹性笔刷指定一个自定义热键，按 Ctrl + Alt 并单击，状态提示将是按任意组合键来指定自定义热键，或者按 Esc 或鼠标键来取消，或者如果您之前已经指定了热键，那么您可以按 DELETE 键来删除自定义指定。

![step 17](img/e6eb654b2ab3d2e67635648099b35f19.png)



**步骤 18:** 所以现在我们给弹性画笔分配了一个自定义热键 Alt + S。

![step 18](img/efa65889b22b90f2f0520ee874be3dee.png)



**第 19 步:**进入首选项菜单，在热键子菜单中，有 4 个选项。首先是恢复，将热键恢复到 Zbrush 初始热键设置。要存储热键，请按 store，以便它们总是可用。Load 将加载您自己的自定义配置。然后有一个保存选项，它会保存到文件中，但不会使其可用。

注意:与其他 ZBrush 配置文件一样，StartupHotkeys.txt 文件存储在计算机上的一个公共文件夹中。

在 Windows 上，这是:Users \ Public \ Documents \ ZBrushData \ ZStartup \ Hotkeys

在 Mac OSX 上是:用户/共享/ZBrushData/ZStartup/热键

![step 19](img/df1ace952b8fc276c4d9d7fef398b11f.png)



**第 20 步:**在文档调色板中，按 0 查看实际尺寸，按 Ctrl + 0 查看一半尺寸，如图所示抗锯齿。然后使用+[加号]放大，使用-[减号]缩小。

![step 20](img/0d28ad83f22bc30872feda1999a24889.png)



**步骤 21:** 那么 W、E、R 分别是移动、缩放、旋转的快捷键。您可以按 Q 键再次返回绘图模式。

![step 21](img/8bc48f48c70e6cf57828f88fcdc2540c.png)



**步骤 22:** 然后复制 Ctrl 并点击拖动手柄。

![step 22](img/dfe34fc9426814da4fbc59ec04829cb1.png)



**步骤 23:** 现在，对于雕刻和绘画选项，您可以通过按 s 来设置绘画尺寸。

![step 23](img/f8ccb8662379910b95381c8792d52bd9.png)



**步骤 24:** 然后按 o 就可以进行焦点转移了。

![step 24](img/1e20028902be77bfd1834c1c5ed43a3a.png)



**第 25 步:**然后您可以通过按 u 来设置 Z 强度。

![step 25](img/49256cc098672a4178ab09bc4d5cb9b7.png)



**步骤 26:** 你可以通过使用【设置单位】来增加绘制大小，通过使用【设置单位】来减少绘制大小(你可以在 Zplugin - >杂项实用程序>笔刷增量滑块中设置增量)。

![step 26](img/a423b94a3fd8066b24400af780ccf731.png)



**步骤 27:** 按 Ctrl + D 分割几何体，按 Shift + D 降低分辨率。如果想要更高的分辨率，则按 D，对于循环边，按 Ctrl + E，这适用于部分隐藏的网格。

![step 27](img/1eec12e3600e66aa15baa227797fbfb1.png)



**步骤 28:** 加载一个 ZScript 按 Shift + Ctrl + L，重新加载一个 ZScript 按 Ctrl + L，显示/隐藏 ZScript 窗口按 h。

![step 28](img/7a21e39677c80eb317c456b470b3efa0.png)



**步骤 29:** 你可以使用 Shift + R 使用最佳预览渲染(BPR)，使用 Shift + Ctrl + R 渲染场景中的一切，或者使用 Ctrl + R 进行光标选择性渲染。

![step 29](img/5cee771f26f5ddeb9d70de5f986aa6cf.png)
