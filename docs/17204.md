## Photoshop 中的塑料效果介绍

在 Photoshop 中，塑料效果是指我们的文本或图像可以看起来像是由塑料制成的，或者在上面放了一张塑料片，给它一种类似塑料的效果。我们可以将这种效果添加到许多图像和文本中，其中大部分你可能已经在广告和社交媒体中看到过。现在这种效果可以用几种不同的方式来使用。因此，第一种方法将是经典的使用图层样式和工具箱。第二是艺术滤镜的使用。那么什么是艺术滤镜呢？Photoshop 提供了各种现成的编辑效果，可以很容易地应用到我们的图像或文本中，从而给它带来想要的改变。保鲜膜效应就是这样一种。今天我们将看到两者的例子，以便更清楚地理解。

### 在 Photoshop 中应用塑料效果

我们将用两种不同的方法应用塑料效果，一种在图像中，另一种在图层中:

<small>3D 动画、建模、仿真、游戏开发&其他</small>

#### 方法 1–图像上的塑料效果

我想说，这种效应在我们周围的任何地方都很常见。我们将挑选两幅图像，以便我们有一个清晰的概念。

**第一步:**用第一张图片！.我选择了一个花卉的风景图像，因为花卉很容易显示体积。因此，请执行以下操作。文件>打开。现在选择所需的图像，并让它在工作区中打开。

![first image (Plastic Effect in Photoshop)](img/a678af8ad2f9710aa6151400607ec629.png)



第二步:我建议复制下面的图层。因此，右键单击背景层，选择复制层。

![create duplicate (Plastic Effect in Photoshop)](img/2473522bc3589ea4e353e74eaae5668f.png)



*   现在很容易在复制层中进行更改，这样原始图像就不会受到影响。当选择背景复制层，去过滤器，然后艺术在最后塑料包装。

![background copy layer (Plastic Effect in Photoshop)](img/0b140458f52e6d98252ee4a230769475.png)



第三步:你会看到一个窗口，它会打开并显示预览，这样你就可以进行修改了。您可以根据自己的喜好和偏好调整这些值。我用了下面的。高光强度:10，细节:12，流畅度:15。点击确定。

![value to plastic image (Plastic Effect in Photoshop)](img/49681d210d443c535a3a1cc956687671.png)



*   你会看到一张类似塑料的纸盖住了我们的原始图像。我们通常在准备购买相框或商店中看到这种情况。

![plastic wrap frame (Plastic Effect in Photoshop)](img/efc7ff922f97a5cee4ca764557b93b01.png)



**第四步:**(用下一张图！)下面我们来打开下图。转到文件和打开，选择图像并点击打开。

![second image (plastic effect in psotoshop)](img/8352476c6b751623c1efe2441d653299.png)



*   再次，创建下面的层的副本，并保持该层被选中。

![create duplicate2 (Plastic Effect in Photoshop)](img/81e4eb609b814b82576718aa665eef0a.png)



**第五步:**从左侧选择快速选择工具。只需在背景中拖动除电话部分以外的粉红色区域。

![quick selection tool (Plastic Effect in Photoshop)](img/314bdacc991e533ceaffb373d05d393e.png)



*   现在去选择>反向选择。您将看到电话被选中。

![inverse selection (Plastic Effect in Photoshop)](img/4ab8c760fdaa7ad814daf23492c949ac.png)



**第六步:**现在去过滤>艺术>保鲜膜。由于我们之前已经使用过塑料包装，一个快速提示是键盘上的快捷键 Ctrl+F。您将再次看到预览框。如下所示更改以下值。硬度:7，细节:8，光滑度:12。点击确定。

![plastic wrap (Plastic Effect in Photoshop)](img/903bb1e08de1cf16989f3e98e8cbcaee.png)



![preview effect (Plastic Effect in Photoshop)](img/e41876209767d79f0ee1c94fbff46d7d.png)



**第七步:**你会在手机上看到一种类似塑料的浮雕效果，让它看起来像是被塑料包裹着。

![plastic wrapped (Plastic Effect in Photoshop)](img/bf3e3aea55189e15954d2ae0372c4cb2.png)



*   因此，我们可以根据自己的喜好和需要在图像中使用塑料效果。我们还可以使用屏蔽选项，并执行以下步骤。

#### 方法 2——在文本层

**第一步:**我们从一个新鲜的 Photoshop 文档开始。

![new document photoshop (Plastic Effect in Photoshop)](img/f013be4bb5af617b8c4495401bb60b1b.png)



*   现在使用文字工具，键入塑料。现在，选择你选择的字体风格和大小。我选择了 **Humnst777 Blk BT** 。使用移动工具选择塑料层来调整单词“塑料”的大小。或者使用“编辑”菜单中的“自由变换”选项。

![type tool (Plastic Effect in Photoshop)](img/a60eae0f2d5199304bab4654a102c355.png)



![text in plastic (Plastic Effect in Photoshop)](img/c1f8bfda161f5299d5ec21239a79a92f.png)



第二步:现在，让我们从图层样式开始。右键单击层命名为塑料，并选择混合选项。选择彩色叠加。在混合模式选项旁边，请选择颜色选择器。现在，更改以下值 R: 88，G: 211，B: 252。

![text layer style](img/5e4ea055e4697208a8288b15ee8c952e.png)



*   如果你喜欢更深或更浅的版本，你可以做一些变化。

![color plastic](img/196f62fa5c38d42e107767f9b6bb2be5.png)



第三步:现在点击左边的内发光选项，改变下面的值。混合模式:乘，不透明度:50 %，大小:13 像素。

![inner glow option](img/193b8d16e9669f4b42231f96bb0791c9.png)



*   现在在杂色选项下面的颜色选择器选项中，你可以选择一个深绿色来给它一个更暗的内部发光，如下所示。

![green shade](img/20a3b45c3a025087c93b1e41fe174843.png)



**第四步:**点击选项斜面和浮雕，选择玻璃轮廓。在这个对话框中，你会看到一个小箭头。请单击箭头，您将看到一个下拉菜单。选择轮廓选项。在对话框中，通过单击“追加”接受操作。在这里你会看到轮廓会有自己的名字和形状。选择双环选项。

![bevel and emboss](img/98e1f9a58adaf1ed3d2ed47ab85c29bf.png)



![append](img/5f867e9276fc5858ca2c5b92b8980187.png)



![ring double](img/a996f32faffff70de20575e5e9008226.png)



**第五步:**让我们改变斜面&浮雕中的参数。同样，这些值可以根据需要进行更改。更重要的是要记住，创造塑料效果的基本概念和方法是相同的。所以深度:130，大小:18 px，高度:60 度，高光模式不透明度:100 %，阴影模式不透明度:0 %。

![change parameter](img/69d3b913d7edb99f3f76d30e416ad48b.png)



**第六步:**现在选择等高线图层样式，在下拉菜单中选择浅坡山谷选项。检查反走样，使范围为 75%。您会看到以下效果。

![contour layer style](img/b1cf7dbdf88723b5f7a5f52dbe73447c.png)



![anti contour](img/c4612d31e19bf1e5b04a51703ffb9be0.png)



第七步:现在，这个效果看起来不错，但是让我们把它做得更有可塑性。复制一层塑料。右键单击它，选择复制层，命名为塑料副本。

![plastic copy](img/3109c258dce0684c0db4112a49b70126.png)



**第八步:**现在选择原来的塑料层，双击进行修改。选择图层样式，设置大小为 10 像素。在色样中，借助滴管选择塑料的蓝色，如下所示。

![select stroke](img/f273c70af6baffe257034bcd2819e0dc.png)



步骤 9: 现在再次，去斜面和浮雕图层样式，并选择如下所示的中风浮雕选项，并改变如下:玻璃轮廓:浅坡谷和不透明度为 40%。

![stroke emboss](img/220d1ee75f13814f5da04307a6ec9c69.png)



![final (Plastic Effect in Photoshop)](img/6357e0e3ed8c341ef5b3635051aab8da.png)



因此，塑料效果是通过使用两种不同的选择，都完全不同，但有趣。
