## 网络技术概论面试问答

Web 技术被定义为两台计算机之间的通信过程，两台计算机之间使用标记语言和不同的多媒体包进行通信的方式称为 web 技术。Web 技术也可以定义为服务器(web 服务器)和客户端(web 客户端)之间的中介或媒介。Web 技术是用于 web 开发(一个网站的前端开发)的前端编程语言[。这些语言是 HTML、CSS 和 JavaScript，其中 HTML 是](https://www.educba.com/career-in-web-development/)[超文本标记语言](https://www.educba.com/uses-of-html/)，CSS 级联样式表用于样式化网页，JavaScript 用于 web 开发的后端。web 技术用于创建、维护和开发 Web 应用程序，它统称为万维网。开发人员会认为这些是编程语言、框架和数据存储机制服务，有助于建立平台和 web 应用程序，它们可以是使用万维网的 web 上的主机。Web 框架，如 [JavaScript 和 Python](https://www.educba.com/python-vs-javascript/) 以及客户端技术或编程语言，如 HTML、CSS 和 JavaScript。在目前的就业市场上，有许多网站开发人员的职位空缺。因此，我们将简要介绍一下网络技术和人们在面试中可能遇到的不同话题。

#### Q1。CSS3 中的 Webkit 有什么用？

**答:**
Webkit 是一个核心软件组件或模块，它将在 Chrome 和 Safari 等不同浏览器上呈现 [HTML 代码和 CSS](https://www.educba.com/html-vs-css/) 。在市场上，有一些其他浏览器的渲染工具引擎，如 Opera 的 Presto，Mozilla Firefox 的 Gecko 和 Internet Explorer 的 Microsoft Edge。为了在网页中使用 Webkit，我们需要在 HTML 代码中呈现“-webkit”和 [CSS。让我们看一个如下的例子:](https://www.educba.com/how-to-add-css-in-html/)

`-box-shadow {
-webkit-box-shadow : 0px 5px 0px 5px #ffffff;
-box-shadow : 0px 0px 5px 5px #ffffff;
}`

#### Q2。CSS 盒子模型及其组件有哪些？

**答:**
盒子模型在 HTML 中很常见，因为所有的东西大多只用盒子的形式来表示。类似地，CSS 中的盒模型也用于建模和设计其元素及其主要组件的布局，如下所示:

*   **Border** :边框组件会受到边框周围使用的填充和内容字段的影响，我们也可以改变边框的背景颜色。
*   **Margin** :该组件是 CSS 盒子模型的[顶层。](https://www.educba.com/css-box-model/)
*   **Padding** :该组件定义了盒子模型中的间距，也就是盒子中元素的位置。
*   **内容**:该组件表示需要在框中显示的内容。

#### Q3。JavaScript 中未定义值和空值的区别？

**答:**
一般来说，当一个变量被声明但没有给它赋任何值时，一个未定义的值就会被赋给一个特定的变量，而不是 JavaScript 引擎给那个变量赋一些垃圾值。而当在代码中声明变量并为其赋值 null 值时，则为变量赋值 Null 值。空值是一个对象，而未定义的值是一个类型。

#### Q4。提到 HTML5 最近批准的 API？

**答案:**
这是面试中问的基本 Web 技术面试问题。HTML5 批准的 API 有媒体 API、应用缓存 API、文本跟踪 API、历史 API、数据传输 API、命令 API、用户交互 API、约束验证 API 是 HTML5 最近批准的 API[。](https://www.educba.com/html5-interview-questions/)

#### Q5。JavaScript 如何处理变量的自动类型转换？

**答:**
由于 JavaScript 是一种符合 ECMS 标准的动态语言，并且是按周输入的，这意味着在 JavaScript 中，我们可以使用一个函数作为另一个函数的参数(一个函数接受另一个函数作为参数)。JavaScript 处理自动类型转换，因为它支持自动类型转换，只要变量、表达式或语句没有将其类型作为输入，JavaScript 就会执行自动类型转换以匹配变量数据类型。

#### Q6。HTML 中的会话存储和本地存储对象有什么区别？

**答案:**
会话存储和本地存储的区别描述如下

*   **本地存储:**在本地存储中，对象对存储在其中的数据没有任何有效期，并且在浏览器关闭后，本地存储对象没有删除数据的权限。
*   **会话存储:**而会话存储将对象保存在其中，直到会话处于活动状态，并在会话关闭时清除存储在其中的数据。

#### Q7。JavaScript 中有哪些不同类型的弹出框？

**回答:**
在 JavaScript 中，我们可以使用三种类型的弹出框，如下所示

#### Q8。CSS 中的 Z 索引是什么？

**答:**
CSS 中的 z-index 是一个属性，用于定义元素的堆栈顺序，基于该顺序元素将显示在网页上(即，在网页中，较高顺序的元素将显示在较低顺序的[元素之前)，我们只能将 z-index 应用于位置元素，如 position: absolute、position: relative、position: fixed 等。](https://www.educba.com/web-page-design-layout/)

例如:
`div {--
Position: absolute;
Left: 10px;
z-index: -1;
}`

#### Q9。Canvas 和 SVG 有什么区别？

**答案:**
这是在面试中被问得最多的 Web 技术面试问题。 [Canvas 和 SVG](https://www.educba.com/svg-vs-canvas/) 的区别定义如下:

Canvas 不提供对事件处理程序的支持，依赖于分辨率，非常适合图形密集型游戏开发，并且仅适用于网页的小渲染区域，而 SVG 不依赖于分辨率，并且提供对事件处理程序的支持，不适合游戏，并且对于像 [Google map](https://www.educba.com/career-in-google-maps/) 等大区域的渲染表现良好。

#### Q10。HTML5 中有哪些新的表单元素？

**回答:**
[html 5](https://www.educba.com/html5-elements/)中引入的新表单元素如下:

*   **Output** :这个表单元素定义了一个表达式的结果。
*   Datalist :该表单元素用于指定输入控件的选项列表。
*   **Keygen** :这个表单元素用于生成加密的密钥。
