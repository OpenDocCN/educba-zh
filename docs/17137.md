## Photoshop 分割调色简介

分割色调是一种效果，通过调整色轮或色阶的补色，您可以看到您的图像。分割意味着将某物分成两部分，所以在 Photoshop 中，我们将使用这种效果来调整图像的色调和阴影。您将在图像的色调区域获得您选择的色调，并且该选择的色调的补色将在该图像的阴影区域起作用。我们可以通过许多方法在 Photoshop 中分割色调，如相机原始滤镜、曲线调整、实心层调整、渐变贴图和其他一些选项。所以让我们来讨论这些方法。

#### 使用相机原始数据滤镜选项进行分割调色

第一种方法是相机原始数据滤镜。对于这种方法，请转到菜单栏的“滤镜”菜单，并从下拉列表中选择“相机原始数据滤镜”选项。

![Photoshop split toning output 4](img/e67d71907b93193e50cd8cf681b00b88.png)



相机 raw 选项的弹出窗口将会打开。

![Photoshop split toning output 5](img/78b0518db436d3b598434b55e0a21b76.png)



单击此对话框的“单音”按钮。

![Photoshop split toning output 6](img/9a664679144e2d7d9c87d4a0c98b8165.png)



在这里，您将获得分割色调选项中高光和阴影的色调和饱和度。根据您的选择，在色调范围内选择任何色调。我将为我想要的色调输入 170 个值。

![Photoshop split toning output 7](img/19b7218b0eeca4714d9c4153ff1f41e5.png)



现在设置饱和度的值为 100，也可以根据你自己取。

![Photoshop split toning output 8](img/f3570a5a6771ba64961e05f39a4c7aef.png)



在色轮上，任何选定颜色的互补色都位于与之相对的 180 度角上。所以在你选择的颜色值 170 上加上 180。现在我们将得到 350 的阴影色调值，这意味着我们选择的颜色的补色将位于色轮上的 350 度。

![Photoshop split toning output 9](img/75470c1b9720577e11c67ee6e9496fae.png)



现在根据你的选择调整阴影的饱和度值。

![Photoshop split toning output 10](img/6404dc932ed164f9665658914e733d4c.png)



现在，您可以检查分割调色后图像的效果。图像的阴影区域具有该选项的阴影值的色调，浅色区域具有高光颜色的色调。您可以在此处找到分割色调前和分割色调后图像上的色调差异。

#### 使用纯色调整层进行分色

现在让我们转到我们的第二个方法，这是通过使用纯色调整层。到这个软件的图层面板的底部，点击调整图层按钮。从列表中选择纯色选项。

![Photoshop split toning output 12](img/cc5bb87d8e0fa65e9fabfd7378edcb0d.png)



现在从色标中选择你想要的纯色。从色标中选择该颜色较亮或较暗的色调。我会选择这种红色的浅色调。

![Photoshop split toning output 13](img/78b8b31c8446c1de0efdbf428ef84b5a.png)



现在转到这一层的混合模式选项，并从混合模式选项下拉列表中选择排除选项。

![Photoshop split toning output 14](img/fb8fcf5535578abdc63bf2fae438b6d8.png)



现在你将得到这种类型的效果，将图像的色调区域转换成高光，将阴影区域转换成你所选颜色的补色。

#### 使用渐变映射的分割色调

现在，让我们讨论下一个方法的分裂色调，这是通过使用梯度地图。对于这种方法，转到这个软件的图层面板的底部，点击调整图层按钮。从列表中选择渐变贴图选项。

![output 19](img/a9c70f5755c498b70b2078484cf5ba0b.png)
