## React 本机使用状态简介

对于功能组件中的状态变量，使用 usestate。我们可以在函数中传递初始状态，函数可以返回一个变量以及当前状态的值和另一个用于更新该值的函数。Usestate 在 React Native 中用于将组件的状态更改为不同的状态。在制作允许用户改变函数状态的应用程序时，这个概念非常有用。本文用不同的例子解释了使用状态的概念，有助于读者理解这一概念。
