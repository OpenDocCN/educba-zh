## 安装搅拌机简介

Blender 是一款 3D 建模、动画和渲染软件。最重要的是，Blender 是一个免费开源的 3D [计算机图形程序](https://www.educba.com/what-is-computer-graphic-design/)。Blender 最常用于创建动画电影、艺术、3D 打印模型、视觉效果和视频游戏。Blender 最初是由 Ton Roosendaal 在 1998 年发布的。程序用 C、 [C++和 Python 语言](https://www.educba.com/python-vs-c-plus-plus/)编写。

### 搅拌机的特点

Blender 的一些最重要的功能如下:

<small>3D 动画、建模、仿真、游戏开发&其他</small>

1.  UV 展开
2.  织构化
3.  三维建模
4.  索具和剥皮
5.  粒子模拟
6.  软体模拟
7.  火柴移动
8.  运动图形
9.  视频编辑
10.  堆肥。

Blender 与 Microsoft Windows、Mac OS 和 Linux 操作系统兼容。该程序有 32 位和 64 位版本。

Blender 支持内部渲染引擎，有助于扫描线渲染、环境遮挡和间接照明。这些效果可以在各种平台上以多种格式导出。Blender 还有一个名为 Cycles 的路径追踪渲染引擎。该引擎专门用于在渲染过程中获得图形处理单元的帮助。渲染器循环也支持开放着色语言。

Blender 集成了许多不同的编辑器，用于查看和修改各种信息技术。

该计划包括关键帧动画工具集。这些工具专门用于反向运动学、曲线和基于晶格的变形以及形状动画。这些工具还帮助用户创建非线性动画。在与 Open CL 一起工作的渲染过程中，该程序也与基于节点的合成器同化。该程序使用 [Python 脚本语言](https://www.educba.com/python-scripting-language/)来开发工具、游戏逻辑、任务自动化和原型制作。Blender 具有修改器，有助于在 3D 环境中为 2D 动画执行建设性的效果和油脂铅笔工具。

### 安装搅拌机的系统要求

以下是安装 Blender 的系统要求:

1.该程序至少需要 32 位双核 CPU，支持 SSE2，主频为 2GHz，大多数情况下建议使用 64 位四核 CPU。

2.该软件至少需要 2GB 内存。建议使用 8GB 左右的 RAM。

3.一个开放的 GL 2.1 显卡是 512 MB 显存的最低要求。虽然强烈建议使用 2GB 显存的 Open GL 3.2 显卡。

4.1280 x 768 像素的显示器是 24 位彩色处理的最低要求。

5.安装 Blender 还需要 Visual C++ 2013 可再发行软件包。

### 兼容的文件格式

*   Blender 将多个场景存储到一个文件中。这个文件称为混合文件。
*   的”。Blend”文件是独立于平台的，并且与 Blender 的不同版本兼容。
*   Blender 将信息排列为不同类型的“数据块”，如对象、网格、灯光、场景、材质和图像。
*   用户还可以使用具有 Blender 功能的脚本来协作或修改 3D 工具，Blender 支持这一功能。
*   Blender 还帮助导入和导出许多 3D 文件格式。这些格式是 3DS、FBX、DXF、SVG、VRML、STL 和 X3D。这些文件格式是各种 3D 程序的通用格式。
