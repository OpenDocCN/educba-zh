## DirectX 版本介绍

DirectX 版本是 DirectX 中新功能的更新集合，通过它可以灵活地使用新技术，您可以拥有不同的版本来使用改进的环境。DirectX 是一个应用程序编程接口(API)的集合，专门用于处理游戏编程。在本文中，我们将了解 DirectX 的版本，并向您介绍每个版本的重要功能。所以让我们来找一下 DirectX 的版本，分析一下每个版本的优点。

### DirectX 的顶级版本

Direction X 于 1995 年 9 月开发，与 Microsoft Windows、Dreamcast、Xbox、Xbox 360、Xbox One、Xbox X 系列和 X 系列兼容。自开发以来，它发生了许多变化，并以不同版本的形式出现。我会告诉你一些版本，让我们开始吧。

<small>3D 动画、建模、仿真、游戏开发&其他</small>

#### 1.DirectX 9

在推出时，它兼容 Windows 98 和 XP，其推出时间是 2002 年，但在 2007 年晚些时候，它被改进为具有与 Windows 2002 或 Xp 的兼容性。它是通过着色器模型引入的，该模型具有像素着色器 2.0 和顶点着色器 2.0。

#### 2.DirectX 10

它附带了 DirectX API 中的主要更新，并与 Windows Vista 及其更高版本兼容。DirectX 10 的独占版本无法在旧版本的 Windows 上运行，如 Windows XP。这个版本做了很多改变，比如用 XInput 改变了上一个版本的 DirectInput，跨平台的 DirectSound，音频创作工具系统，以及对硬件加速音频的额外支持。还有一件事被 dplayx.dll 取代了，那就是 DPLAY.DLL。

#### 3.DirectX 11

DirectX 11 带来了重大的变化和非常好的功能，并在西雅图的 Gamefest 08 活动中推出。它有许多 GPGPU 支持的功能，如果我们谈论 Direct3D 11，它带有镶嵌支持。它还改进了多线程支持，这有助于开发视频游戏，并以更好的方式利用多核处理器，这种多线程吸引了视频游戏开发人员。您可以在 Windows 平台的不同操作系统上运行此版本，如 Windows Vista、Windows 7、Windows 8 和 Windows 10。它需要 Direct3D 11 支持硬件镶嵌和着色器模型 5.0 的硬件。微软还推出了 Direct3D 11 的技术预览版。10.1 版的其他硬件和 API 功能处于搁置状态，仅在需要增加此版本的功能时添加。后来随着 2009 年 Windows Vista 的最终更新，Direct3D 11 有了四个更新。让我们讨论一下更新是什么。

第一个是 DirectX 11.1，与 Windows 8 兼容，支持 WDDM 1.2 和 WDDM 1.2，以提高性能并改善 Direct2D、Direct3D 和 DirectCompute 以及 XNA 框架的 DirectXMath、XInput 和 XAudio2 库的集成功能。DirectX 11 的这次更新也用 Windows 7 平台的一些更新部分支持了 Windows 7。它还具有支持游戏和多媒体视频的立体 3D 功能。

DirectX 11.2 在 Windows 8.1 和 Windows Server 2012 R2 中推出，并具有 Direct2D 的新功能，如几何实现。它还引入了交换链合成功能，为您提供低分辨率场景的渲染，然后通过硬件叠加以更高的分辨率合成该场景。第三次更新是 DirectX 11。x，是作为 DirectX 11.2 的下一步推出的，它运行在 Xbox One 上。它包含了随后在 DirectX 12 版本中引入的 draw bundles 功能。下一次更新是在 GDC 上用 DirectX 12 宣布的，更新是 2015 年发布的 DirectX 11.3。

#### 4.DirectX 12

DirectX 的下一个版本是 DirectX 12，由微软于 2014 年 3 月 20 日在 GDC 上推出，并于 2015 年 7 月 29 日与 Windows 10 一起正式发布。第一个可以突出显示的功能是 Direct3D 12 的低级编程 API，该功能减少了驱动程序开销。通过并行计算，资源利用变得更加高效，开发人员可以实现自己的命令表，并用 GPU 缓冲该命令。

我要告诉你的是，开发者 Max McMullen 表示，Direct3D 12 是为了在 PC、手机和平板电脑上实现控制台级别的效率，而 Direct3D 12 的发布附带了一些举措，如 AMD 针对 AMD 显卡的 mantle、Khronos Group 的跨平台 Vulkan 以及苹果针对 iOS 和 Mac OS 的 Metal。

DirectX 12 中有多适配器支持，允许用户在系统上同时使用多个 GPU，而在以前，多 GPU 支持依赖于供应商的实施。所有 Fermi 和 Nvidia GPUs、AMD 基于 GCN 的芯片、英特尔的 Haswell 及其更高版本的处理器都支持 DirectX 12。

在 2014 年由英特尔发布的计算机生成的英特尔 SIGGRAPH 小行星场，DirectX 12 在渲染速度和 CPU 功耗方面比 DirectX 11 提高了 50%至 70 %。公开推出的第一款利用 DirectX 12 的游戏是“奇点之灰”,在 2015 年 ArsTechnica 的测试中，Nvidia GeForce 980 Ti 的 DirectX 12 性能比 DirectX 11 有所下降，而如果我们谈论 AMD 镭龙 R9 290x，则 DirectX 12 的性能持续改善，最高可达 70 %。

#### 5.DirectX 12 旗舰版

这是 DirectX 的最新版本，将于 2020 年 3 月推出。它兼容 Windows 10 和 Xbox series X 以及其他第九代 Xbox 主机。此版本中包含的新功能是可变比率着色和光线跟踪 1.1，它提供了对着色细节级别的完全控制。这取决于设计选择、采样器反馈和网格着色器。
