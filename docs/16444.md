## 草图插件简介

Sketch Plugins 是它的助推器。通过拥有不同类型的 Sketch 插件软件，我们可以增强其工作能力，并添加高水平的功能，以便在矢量图形编辑工作中获得最佳输出。您可以找到该软件的许多插件，可以轻松安装并用于特定类型的工作。我们将告诉你一些流行的 Sketch 插件及其重要的方面，以及如何让它们开始工作。所以让我们开始一个一个地讨论这些插件。

### 草图插件

我们将告诉你一些草图插件，如草图运行器，草图插件管理器，草图工具箱，零高度，链和草图材质。

#### 1.速写赛跑运动员

*   有了这个插件，你可以在你的设计中应用共享的风格，运行和安装其他插件，在你的设计作品中插入符号，还有很多你可以用它做的事情。
*   你可以用它在草图中更快地完成设计工作，拥有它，你可以访问 sketchrunner.com。在这里，您可以拥有与之相关的所有其他重要细节。
*   你可以在有限的时间内免费试用，或者如果你想购买，你可以访问它的官方网站。
*   控制你的符号和样式是相当容易的，因为它为每一个提供了一个单独的窗口。
*   它具有管理您的设计系统的功能，例如它提供批量重命名组件，您可以跳转到任何符号进行编辑等等。
*   您可以轻松地搜索任何符号，并将其插入到您的项目工作中。

#### 2.草图插件管理器

*   它通常用于保持所有已安装的插件最新，目前仅支持 OS x 10.12 Sierra。
*   它有目录功能，将这个管理器与 Sketch 插件目录连接起来，这样你就可以在它的官方网站上浏览和安装可用的插件。
*   有了这个软件，你不需要检查你安装的插件的更新；每当您启动 Sketch 时，它会检查所有已安装的插件，您可以在接受它们之前检查所有更新。
*   它就像一个中枢，你可以从这里管理所有已安装的插件，并获得关于它们的特性的信息。

#### 3.草图工具箱

*   使用 Sketch Toolbox，您可以浏览流行的插件，这些插件可用于草图和下载，并且只需单击一下就可以安装它们。
*   它帮助你浏览插件，并自动保持所有插件的更新，这样你就可以接触到所有插件的最新功能。
*   在这个插件即将到来的特性中，你将有更好的能力来组织不同的已安装插件和每个插件的元数据；你可以根据最新的，当前的趋势和最开始的插件来分类。
*   键盘快捷键的定制也将出现在即将到来的更新中。您可以访问 sketchtoolbox.com 了解更多相关信息或下载。

#### 4.零高度

*   它为工程师、设计师、产品和营销团队创建了一个中心枢纽，这样他们就可以将参与该设计系统的每个人联系起来。由于更好的联系，你可以在工作的产出中获得最大的效率。
*   它为设计提供同步，并将工具集合在一起，这是您的团队所需要的。在处理互联项目时，每个人都可以轻松使用工具。
*   它根据团队共享的指导原则转变您的工作，并为最佳实践提供良好的合作环境。
*   这个软件的优点是，它是一个免费的插件，允许你创建资源库和风格指南，以实现良好的协作。

#### 5.链子

*   正如它的名字所显示的，它在事物之间创建了一个链；是的，它可以在巨大的草图文件中创建动态颜色关系，以便在全局范围内改变颜色，并节省您在每个文件中逐个进行这种改变的时间。
*   使用它时，您只需选择要链接的图层、参考颜色和要在所有文件中应用的变换，然后通过管理色调、饱和度、亮度和 alpha 值的参数进行颜色校正。
*   使用动态连接，您可以在多个链接层中进行更改。您只需在参考层中进行所需的更改，您所做的更改将影响所有链接的层。

#### 6.草图材料

*   有了这个插件，你可以从 Google material design 系统中生成任何组件，复杂的材料组件的生成对你来说变得很容易，比如桌子，芯片等等。
*   这个插件提供了许多高级功能，可以与我们用来设计不同对象、组件或元素的材料协同工作。
*   您可以安装它，因为它拥有各种不同类型的材料库，从而为您设计的零部件提供逼真的外观。

#### 7.自举网格

*   这是一个组件生成器插件，在草图文件中生成 Bootstrop 网格。
*   你只需要画一个矩形来画网格，这个插件会自动在这个形状里面创建一个网格。
*   你可以使用热键或快捷键来执行这个任务，例如对于选项窗口按 Ctrl + Shift + W，然后设置网格与最后的配置按 Ctrl + shift + G，或者设置默认的引导网格按 Ctrl + Shift + D
*   您可以使用默认的网格设置，也可以根据您的需求进行自定义设置，以获得您想要的结果。
*   它还有不同类型的好特性，让用户感觉很舒服，比如网格的多项选择。
