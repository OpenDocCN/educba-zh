## CNN 和 RNN 的区别

在这篇文章中，我们将讨论 CNN 和 RNN 的主要区别。[卷积神经网络](https://www.educba.com/convolutional-neural-networks/)是信息技术领域神经网络家族中的特殊版本之一。它从底层的隐藏层中提取其名称，隐藏层由汇集层、卷积层、完整的、互连的层和规范化层组成。它是使用正常的激活方法设计的；卷积、汇集函数被用作激活函数。[递归神经网络](https://www.educba.com/recurrent-neural-networks-rnn/)是一种定义的方差，主要用于自然语言处理。在常见的神经网络中，输入通过有限的输入层进行处理，并在假设输入层完全独立的情况下生成输出。

### CNN 和 RNN 的主要区别

让我们来讨论一下 CNN 和 RNN 的顶级对比:

*   数学上，卷积是一个分组公式。在 CNN 中，卷积发生在两个矩阵之间，产生第三个输出矩阵。矩阵只不过是存储在列和行中的数字的矩形阵列。CNN 利用卷积层中的卷积来分离输入信息并找到实际的信息。
*   卷积层从事计算活动，如卷积神经网络中的高度复杂，卷积神经网络充当数字滤波器，帮助计算机找到图片的角落、集中和褪色区域、颜色收缩以及其他属性，如图片的高度、深度和分散的像素、图像的大小和重量。
*   汇集层通常建立在卷积层之间，卷积层用于减少卷积层设计的表示结构，卷积层用于减少允许许多卷积层的存储器组件。
*   归一化是为了提高神经网络的生产率和稳定性。它倾向于通过将所有给定的输入改变为相应的平均值 0 和 1 的变型(其中这些输入被认为是规则化的数据),来对单个层进行更具适应性的输入。完全互连的层有助于将每个神经元从一层连接到另一层。
*   CNN 是专门为计算机视觉设计的，但用所需的数据引导它们可以修改它们，以获得图像、音乐、语音、视频和文本的高级形式。
*   CNN 包含无数层隐藏和优化的过滤器或神经元层，在检测图像时效率很高，并且该过程从相互连接的层开始。由于这一普遍特性，它们被称为前馈回路。
*   RNN 具有与人工神经元网络和 CNN 相同的传统结构。他们有另一个可以作为反馈回路的内存分区。同样，就像人类的大脑一样，特别是在对话中，数据冗余被赋予很高的权重，以联系和理解句子及其背后的含义。RNN 的这一独特特征被用来预测下一组或下一个单词序列。RNN 还可以接收一系列不同长度和大小的数据，而 CNN 只处理固定的输入数据。
*   现在 CNN 的例子是图像识别。这台计算机能阅读数字。但是用 1 和 0 的图片表示和 CNN 的很多层。卷积神经网络的窥视深度有助于学习更多的技术。
*   通过分析每一层的数学计算，帮助计算机一次一位地定义图像的细节。这有助于通过逐层读取来识别特定对象。
*   RNN 是一种具有主动数据内存的神经网络，俗称 LSTM，可应用于一系列输入数据，帮助系统预测下一步。一些互连层的输出通过创建反馈回路再次反馈到前一层的输入。下面解释了 RNN 的最佳方案。
*   跟踪酒店中一周内不应重复的主菜，如周一的墨西哥玉米卷、周二的汉堡、周三的意大利面、周四的披萨、周五的寿司。在 RNN 的帮助下，如果输出“比萨饼”被再次输入网络以确定星期五的菜肴，那么 RNN 将让我们知道下一道主菜是寿司，因为该事件在过去几天中被周期性地执行。
*   在这些现代日子里，被配音的 KITT 将从卷积网络和递归神经网络中进行深度学习，以看、说和听，这是通过 CNN 作为用于视觉的图像处理器和 RNN 作为实现语言模式的耳朵和嘴巴的数学引擎来实现的。

### 结论

CNN 是自动驾驶汽车、聚变能源研究和石油勘探的愿景。它也比医学成像更有助于更快地诊断疾病。RNN 被用作亚马逊 Alexa、苹果 Siri 和谷歌助手的语音控制，它理解人类语言处理，并基于基于语音的计算革命原理工作。今天，自动驾驶汽车可以在上路前进行测试。基于人工智能的机器和技术正在与 CNN 和 RNN 一起设定未来的趋势。
