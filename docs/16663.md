## 什么是 Autodesk Navisworks？

Autodesk Navisworks 是一个软件包，其中包含对 Autodesk Revit、AutoCAD 和 MicroStation 等软件的三维设计审阅。它允许用户打开和组合 3d 模型，并在标准工具的帮助下查看模型。不仅如此，我们还可以查看注释、红线批注、测量和视点。模型的实时导航也是其功能之一，除此之外，还有许多插件可以增强包的功能，因为添加插件后，我们可以处理不同的因素，如 4D 时间模拟、干扰检测、真实感渲染、PDF 发布。现在您已经很好地了解了什么是 Autodesk Navisworks？

### Autodesk navisworks(工作原理)

Navisworks 以前称为 JetStream，有一个核心模块，我们称之为 Roamer。为了确定它的工作方法，我们需要了解它的内置功能。通过研究它的功能，我们可以了解它是如何在其工作领域提供最佳输出的。所以让我们好好讨论一下它的功能性吧:

*   这是 Naviswork 的核心部分，在它的帮助下，用户可以打开设计软件的模型。这些模型来自一系列的 3D 设计和激光扫描格式，之后它给出了一个单一的 3D 模型，作为我们的 3D 模型范围的结果。
*   发布者:它帮助用户创建一个 3D 模型的单个 NWD 文件，我们可以在 Freedom 的帮助下免费打开它，Freedom 是一个免费的浏览器应用程序。
*   渲染器:它以前被称为 Presenter，在它的帮助下，用户可以通过应用材质和照明来制作逼真的模型(图像和动画)。
*   **Clash Detective:** 这个帮助用户启用干涉检测功能。干涉检测可以理解为在模型的特定选定部分检测几何冲突，并获取有关特定问题的信息。
*   **TimeLiner :** 它增加了 4D 模拟，帮助用户将几何图形与时间和日期联系起来，以便模拟模型的构建或拆除。
*   **量化:**帮助用户自动估算材料，测量面积，统计建筑的组成部分。
*   动画师:就像它的名字一样，动画师我们可以理解它的意思是帮助用户制作 3d 模型的动画，这样人们就可以很容易地与他们互动。
*   **Scripter:** 它帮助用户建立一个行动集合，当一个特定类型的条件进入工作阶段时，用户想知道这些行动集合。
*   它可以将其工作文件保存为多种文件格式，例如
*   本机文件格式:它包含。nwd，。nwf，。nwc
*   AutoCAD file : .dwg, .dxf
*   3D Studio Max: .3ds。prj(前情提要)
*   MicroStation: .dgn, .prp, .pr
*   .exp，。型号，. dlv3，.CATPart，。CATProduct，。Catia 的 cgr 文件格式
*   .rvt
*   .右室粘液瘤(right ventricular myxoma)
*   .skp of sketchUp
*   .wrl，。VRML 的 wrz
*   .fbx of FBX
*   .prt，。sldprt，。asm，。土壤工程的 sldasm 文件格式
*   所有版本的 pdf
*   Rhino 的. 3dm 文件

**Autodesk Navisworks 提示和技巧**

在使用 Navisworks 的过程中，您必须了解一些提示和技巧，以便使用该软件成为一项愉快的任务。所以我要告诉你一些你需要记住的重要提示和技巧。

*   **圆形物体测量:**很多时候，您可能会在精确测量圆形物体时遇到问题，例如管道的底部圆形侧面。因此，要解决这个问题，请转到 Navisworks 选项编辑器，然后转到界面的捕捉选项下。此处启用“对齐边缘”选项。一旦您启用了此选项，它将允许您的测量工具捕捉圆柱形对象的底部和顶部。
*   **使用测量工具快捷键:**为了方便您的工作并节省时间，您可以使用测量工具的快捷键，为此，激活测量工具，然后按 Ctrl + Shift + Z，它允许工具与 Z 轴对齐，Ctrl + Shift + Y 允许工具与 Y 轴对齐，Ctrl + Shift + X 允许工具与 X 轴对齐。
*   **使用选择工具快捷键:**可以按 Ctrl + 1 切换或激活选择工具。
*   **使用选择框工具快捷键:**使用选择框工具，可以非常快速地选择多个对象。因此，如果您按住键盘的 Shift 键并拖动选择框工具的鼠标光标，那么与框接触的对象也会随着对象进入选择框工具的选择区域而被选中。
*   **单个/多个对象碰撞分组:**在碰撞检测过程中，您将面临多个对象与单个对象发生碰撞的问题，您需要立即将它们分组。因此，转到“clash detective 结果”选项卡中的项目下拉窗口，然后右键单击包含多个碰撞的单个元素。右键单击后，您将有一个选项可以对与该对象相关联的所有碰撞进行分组。

在开始使用这个软件后，你会了解到更多的技巧和窍门。这里我只是给你一些你可能经常用到的重要事实的信息。
