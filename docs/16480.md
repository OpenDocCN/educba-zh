## InDesign 数据合并简介

InDesign 数据合并是该软件的一个非常好的功能，它帮助我们创建文本数据的多种变体，每个条目都有独特的数据集。你可以把它理解为微软 word 的邮件合并，因为它为我们的创意版面执行相同类型的工作。通过数据合并功能，我们可以轻松地用每个不同的条目创建设计布局的多个副本。在这个软件中，我们有一个带有几个参数的完整的数据面板，我们将根据我们的要求对其进行操作。关于使用数据合并，您还需要了解一些事情。所以让我们开始吧。

### 如何在 InDesign 中使用数据合并？

为了开始我们的学习，我将通过单击 InDesign 欢迎屏幕上的“新建”按钮来创建一个新的文档页面。

<small>3D 动画、建模、仿真、游戏开发&其他</small>

![InDesign data merge output 1](img/47e387c9d9c39f3908c44717c6fc7c8e.png)



我将使用该软件的数据合并功能创建名片的多个副本，因此我将使用名片模板并单击创建按钮。

![InDesign data merge output 2](img/15b4e17594f86d10e8b2f79294548b19.png)



现在我将在这里放置一个插画文件，这是我为名片设计的。为此，我将转到“文件”菜单，单击“放置”选项，或者按键盘上的 Ctrl + D 键。

![InDesign data merge output 3](img/eae1844c00eebaef9195e9ea0fa98826.png)



我从这里选择我想要的文件。您可以在 InDesign 中置入此列表中的任何文件格式。

![InDesign data merge output 4](img/6e439a2b0dc2909ff7080f6ac2304c09.png)



这是一个中心有 EDUCBA 标志的设计，我将把它用作名片设计。

![InDesign data merge output 5](img/a3e37936ef7b4988bc6851d893d31fa6.png)



现在，我将从工具面板中选择文字工具，绘制一个文本框。

![InDesign data merge output 6](img/7e89dad917c4a309046b13207969069b.png)



现在我们将打开数据合并面板，为此，请转到菜单栏的窗口菜单，转到下拉列表的实用程序选项，然后选择数据合并选项。

![InDesign data merge output 7](img/09d0c98f5a79f0cfe2f6934c31970fc1.png)



在开始使用这个面板之前，我将向您展示我将在这里用作名片条目的数据。我在记事本里有我的数据。你可以使用 Excel 表，谷歌表，文本编辑器数据，也根据你的选择。

你要记住的一件事是，你要在名片上分别输入的每一个数据都要用逗号书写，第一行应该是条目的标题，名片上会有雇员的名字，然后是职务，然后是电话号码等等。所以我在我的第一线有这些东西。

![InDesign data merge output 8](img/ac1d5f49f5c379a357131f97a45d9299.png)



然后根据标题用逗号输入员工的详细信息，我这里有四个条目。

![InDesign data merge output 9](img/37f7c031575a4d4f252cffb370a8f83d.png)



现在进入 InDesign，单击“数据合并”面板右上方的选项卡，然后单击下拉列表中的“选择数据源”选项。

![InDesign data merge output 10](img/295fd939f21563a897f6ad681a55d777.png)



从打开的对话框中的保存位置选择数据文件。你可以选择。txt 或者。csv 文件格式。我这里有一个. txt 数据文件。

![InDesign data merge output 11](img/67834ea92a15b2f822da207ccb8cbfb2.png)



现在，数据合并会自动将数据文本文件的第一行理解为主条目，并像这样显示在它的面板中。这些条目将是源条目，或者您可以将它们称为主条目。

![InDesign data merge output 12](img/6b91645c0a462e9ca5dacd5ad1b00876.png)



现在来到我们之前画的文本框，点击这里，然后在数据合并面板上点击名称层，这样它将作为条目的源文本进入文本框。

![InDesign data merge output 13](img/0bd884fb00876c3c7695155d5b4ef010.png)



我将按下回车键，然后点击指定层，因为我想在下一行。

![InDesign data merge output 14](img/ca85b29d85477c0f6af67515fa0e583c.png)



我将对剩余的条目进行同样的操作。

![InDesign data merge output 15](img/6ec5360aa56c9f10ada3a9ca5a2ec194.png)



您可以通过点击数据合并面板的预览按钮的复选框来查看预览。在文本框的右边缘，您可以看到一个红色的加号，表示我们的文本不完全适合此文本框。

![InDesign data merge output 16](img/cdd89e1145cb73124bc013111993a91c.png)



因此，我们可以改变文本框的大小，以适应它；它不会影响条目。

![InDesign data merge output 17](img/1fcbb1eedfa140d09a57104c3a9ea904.png)



您可以通过单击数据合并面板的“下一步”按钮来查看下一个条目。

![InDesign data merge output 18](img/a69346b71bf7ccd40da92339a1b23a5b.png)



现在，我将更改这个名称条目的字体样式和颜色。

![output 19.1](img/950c0980ab91942ce6ba4c87b7cdbe8c.png)



![output 19.2](img/fbc70f465b168e1bbc9a5a018b1a5670.png)



并且在其他文本条目中的段落样式绘画也是如此。

![output 20](img/47712e8815d8193b78469771c48f1542.png)



现在，您可以再次看到预览。如果你在数据中犯了什么错误，比如约翰的拼写错误。因此，您无需再次经历所有步骤。

![output 21](img/307ba5bc12a82b5fe851a2f9ab75b562.png)



只需在数据表中进行修改并保存更改。

![output 22](img/79cbcbe1edf3046a74695dce459616d8.png)



然后点击数据合并面板右上角的按钮，然后点击更新数据源。

![output 23](img/0f8b44cc77eba140b24593a6becaaad3.png)



您所做的修改将在此自动更新。同样的，你可以增加条目的数量。只需添加其他员工数据，并从数据合并面板中进行更新。

![output 24](img/a7b56dbf72a67c2495053db6fe8cf454.png)



现在禁用预览模式，并点击数据合并面板的创建合并文档按钮。

![output 25](img/0bba9d3e81715aa192a5309988ac4e65.png)



此选项的对话框将会打开。在这个框中，您可以选择要创建多少个数据条目。因此，举例来说，您可以通过此选项仅为单个条目创建一个合并文档。

![output 26](img/0ed41e5dd2c5ee08d934e2e0bc9b7530.png)



或者从范围选项中决定它们的范围。

![output 27](img/ee28a9a9a33f742e68418ccdaf1c46de.png)



我将选择“全部”,然后单击该对话框的“确定”按钮，当我单击“确定”按钮时，如果没有溢出文本，则会显示一条消息，这意味着我可以创建一个合并文档，而不会出现任何问题。如果你想了解溢出文本，你可以去看看我在 InDesign 的文章“溢出文本”。

![output 28](img/54a99cf56c8335c88a562208692a233e.png)



一旦我点击“确定”按钮，它将在每个单独的页面上创建所有条目的名片。所以现在我们可以把它保存为。pdf 并直接拿去打印。

![output 29](img/dedeaafb594901a1f47957bfdc03b8f5.png)



### 结论

我希望你发现数据合并是这个软件的一个非常好和有效的功能，它节省了你为不同的布局设计制作这些类型的条目的大量时间。你可以用。psd 文件设计也在 InDesign 中，数据合并功能也将与。psd 设计。
