## InDesign 栏简介

InDesign Column 是该软件的一个非常有用的功能，通过它我们可以将任何设计布局的文本内容排列成列，为它们提供有效的外观，并将其管理到给定的空间中。在这个软件中，我们可以通过一些技巧或者使用菜单栏的选项来实现。您可以创建多个栏，以便更好地排列文本内容。我将告诉您柱选项的所有重要参数，以便您可以顺利地处理它们，获得您想要的结果。在 InDesign 的最新版本中，我们多了一个选项，通过它我们可以添加规则(分隔线)作为列的设计元素。所以让我告诉你所有这些功能。

### 如何在 InDesign 中创建和使用分栏？

我们可以通过几个简单的步骤在 InDesign 中创建分栏，然后开始使用分栏以易读的方式排列文本。

<small>3D 动画、建模、仿真、游戏开发&其他</small>

因此，让我们首先通过单击该软件欢迎屏幕上的“创建新文档”按钮来创建一个新文档。在“新建文档”对话框中，您可以看到有一个列选项，但这次我将使用默认设置。

![InDesign columns output 1](img/50e147c83315d180fba2e09ba0e52aa9.png)



您可以看到我们的文档页面区域没有列。它只有边缘。

![InDesign columns output 2](img/5c3fb9be55227dc25b61d46c4563e7b5.png)



现在，我将再次通过单击“文件”菜单的“新建”选项中的“文档”选项来获取文档。

![InDesign columns output 3](img/9c04ea2be873df0e39146a19c44206da.png)



这一次，我将在此对话框的列选项中输入 2 作为列号，因为我希望文本内容有两列。

![InDesign columns output 4](img/8fd699ab4a402b2c753d2f251485051a.png)



现在，我们将文档分成两列。

![InDesign columns output 5](img/697819924cadaa70a077e5ee2a3b9304.png)



现在，我将使用工具面板中的文字工具，在空白区域内绘制一个覆盖整个文档页面的文本框架。

![InDesign columns output 6](img/30c0017bd420f3b30c18ed116d00bd9d.png)



我将此文本内容粘贴到这里，但是您可以看到它并没有排列到这两列中。

![InDesign columns output 7](img/a4694a0f7f299ee37215b30c8ffc6f85.png)



因此，我们可以根据列设置文本框。

![InDesign columns output 8.1](img/d37f8993951b77204eb1ecea35c86f2e.png)



![InDesign columns output 8.2](img/8274c98b293b54debb9b56ccbcb1f3cb.png)



您可以看到文本处于溢出状态。

![InDesign columns output 9](img/9c4be7d20b0ec821b5d808c28805bc68.png)



为此，我将创建一个新的文本框，并单击这个红色的加号按钮。

![InDesign columns output 10](img/f4b90390636a5808667b5ec6a31c4b05.png)



一旦我们点击这个按钮，溢出的文本就会出现在鼠标的光标上。

![InDesign columns output 11](img/74e05fc7598df850f569edf0b9ecd7ef.png)



我将单击下一个文本框来调整其中的额外文本。这种方式可以将文本设置成列，但这是一种传统的方式，需要大量的调整。

![InDesign columns output 12](img/e1c035b85cde7c8a65721ff7795a912c.png)



因此，让我们转向智能的方式，让设计自动地为我们服务。这里我有一个新文档的文本内容，没有任何列部分。

![InDesign columns output 13](img/fcec78154e3dede78e18435df193ec6a.png)



现在选择这个文本框，在这个软件的属性栏的右上角，我们有“列数”选项。

![InDesign columns output 14](img/7b5befee61acb8318f934fa83cd4e936.png)



如果我们增加这个数字，InDesign 会自动在文本内容中添加列，并根据列设置进行排列。

![InDesign columns output 15](img/437316e1d51ddfd868fe67a7e30c4b69.png)



您可以通过菜单栏对象菜单的“文本框选项”对列设置进行更多控制。从该菜单启用该选项，或者您可以按 Ctrl + B 作为快捷键。

![InDesign columns output 16](img/6351aacf3949ddde20c8a3b3b8e6c83c.png)



一旦你点击它，这个选项的对话框将被打开，在常规选项中，我们有参数栏。您可以看到，我增加了列号的值，但是文档区域的文本内容没有变化。原因是我的文字内容这次没有入选。所以要小心这些愚蠢的错误。

![InDesign columns output 17](img/8b4ffe2abdd9e1d024124a34e5f858b7.png)



现在，我将选择文本内容并增加列数，您可以看到 InDesign 会自动将列添加到该文本内容中。

![InDesign columns output 18](img/22d30d097a92a9b7ef8c7b3140c0810f.png)



您可以根据自己的选择增加列数，它会自动调整给定列数中的文本。

![InDesign columns output 19](img/de0c6ecb8ac171c606b170064967b054.png)



本栏目的下一个选项是装订线。装订线是两列之间的间隙区域，因此您也可以通过此对话框的装订线选项对其进行调整。

![output 20](img/1d247e5cdf92209450f523d9a815eb34.png)



这完全取决于你的设计布局，你希望在你安排文本内容的栏之间有多大的间距。

![output 21](img/d799f2cbe8a1ada7c196ef1e461fd8e8.png)



此对话框中有一个平衡列选项。

![output 22](img/acec3363ba4e1cf51442391b2fb2385f.png)



如果我们通过点击它的复选框来启用它，那么它将平衡所有列中的文本内容，使它们的高度在末尾相等。

![output 23](img/a530d20c34b723c9dc302c39ab3baf64.png)



这里还有一个与专栏相关的令人兴奋的特性。这是文本框选项的同一个对话框中的列规则选项。所以点击它就可以得到它的参数。

![output 24](img/487a2ef1b8cf317f8c72057fea0fcb38.png)



现在通过点击它的复选框来启用插入列规则选项。您可以看到，一旦我们启用它，一条基准线将出现在列的装订线区域。

![output 25](img/3db594d019f11fdf1197f89bfa26d46a.png)



我们可以从起点调整这条线的长度，也可以从嵌线长度的起点和终点选项调整它的终点。

![output 26.1](img/70151403f147650aba07b5be64223eb5.png)



![output 26.2](img/a57882e356a8e25fb06740398c99f4b6.png)



这两个选项都有一个链接选项，所以我们可以根据需要分别从起点或终点改变嵌线长度。

![output 27](img/734c4079cc995c90305bad3a0e73fe8f.png)



我们可以设置嵌线与所在列之间的偏移距离。

![output 28](img/4f492634163997d69517b7a830119ef0.png)



通过更改此对话框的宽度选项值，我们可以更改基准线的粗细。

![output 29](img/b6fc557bab8093e52bbd52100d4a4ef3.png)



如果需要，我们也可以从颜色选项中更改基准线的颜色。

![output 30](img/a4ad6e0acff1ba6e73a718f4ec94f443.png)



这条规则线的笔画有几种样式，所以我们可以根据自己的选择从中选择任何人。

![output 31](img/64dff6eac9b889f3a3c3b2b2cfa88599.png)



现在让我们通过点击这个预览按钮切换到预览模式，或者你可以简单地按 W 作为它的快捷键。

![output 32](img/d20138adb7ba99b5625d79b2ac90bae1.png)



你可以看到我们的文本被漂亮地排列成列。

![output 33](img/e8110d0047835f2d8e111a07e1757ac6.png)
