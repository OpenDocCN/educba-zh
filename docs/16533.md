## indesign 网格简介

InDesign 网格可以理解为指导方针，通过它我们可以根据我们的要求适当地调整我们的工作，因为在 InDesign 工作期间，我们必须管理大量带有文本的图像，以使它们具有良好的组合。在这个软件中，我们有两种类型的网格，一种是文档网格，另一种是基线网格。我们将这两个网格用于不同类型的目的，这两个网格都有自己的参数面板，我们可以根据工作要求在其中进行调整。让我们一个接一个地浏览两个网格。

### 怎么用？

我们可以在 InDesign 中使用网格来完美对齐项目工作中的文本和图像。所以我要告诉你关于网格的事情，这样你就可以用它在这个软件中进行有效的布局设计。

首先，我将从该软件欢迎屏幕的“新建”按钮中获取一个默认大小的新文档。

![InDesign Grid output 1](img/da74e42fad9fe68648b60bb2e7d98a72.png)



您可以从菜单栏中选择一个网格面板，转到编辑菜单，然后转到下拉列表中的首选项选项，然后在新的首选项选项下拉列表中单击网格选项。

![InDesign Grid output 2](img/fe50608cbca23f5aa4e61c66ad700c9c.png)



在网格的对话框中，我们有基线网格和文档网格的参数，我们可以根据自己的需要操作它们。我将立即使用两个网格的默认参数设置，并单击此对话框中的“确定”按钮。

![InDesign Grid output 3](img/1652cae53c44695847695a8dff9edb13.png)



要查看文档页面上的网格，请转到“视图”菜单，然后转到下拉列表的“网格和参考线”选项，然后在新的下拉列表中，我们有“显示基线网格”和“显示文档网格”两个带有快捷键的网格选项。

![InDesign Grid output 4](img/f38fe1a916e13d2889709fa043f93ecf.png)



我将首先通过单击“显示文档网格”选项来启用文档网格。

![InDesign Grid output 5](img/481f4f540bab6a95bffd8e3062097813.png)



我们在文档页面上有这样的网格。

![InDesign Grid output 6](img/192802379c221748a544ad51e6392795.png)



### 如何在 InDesign 中使用带有图像的网格？

现在让我们用一个图像来理解网格的目的。我将转到文件菜单，单击下拉列表中的放置选项，在该软件中放置图像。

![InDesign Grid output 7](img/ffb4525a007040440711a90e8f6299e5.png)



我将选择这个 EDU 证书格式的图像。

![InDesign Grid output 8](img/86c971d46d1d6449466c49da82de47d4.png)



我会把它放在这里。

![InDesign Grid output 9](img/ceacc4e0a9a013989916f0364cb67205.png)



在使用网格之前，请确保您已经启用了“对齐网格”选项，要启用它，请再次转到“视图”菜单，并单击“网格和参考线”的“对齐文档网格”选项。

![InDesign Grid output 10](img/6ce7c5912b135091e5df107f7934e752.png)



我已经做了一份证书的副本，我们可以很容易地将这个图像与网格角对齐。例如，我想将此图像的右边缘与这些网格块对齐，当我将它移动到这里时，它会自动与它们对齐。

![InDesign Grid output 11](img/86d8fac97c2ec132b864b645b830734c.png)



因为这个网格在它的所有块之间具有相等的距离，所以我们可以非常容易地考虑块来保持两个或更多图像之间的间距。

![InDesign Grid output 12](img/da85bbf7a8b45897d976d038b98265b2.png)



如果你愿意，你可以增加或减少网格块的大小。目前，区块为 72 x 72 磅。

![InDesign Grid output 13](img/128b7f9565c017367dad476dcaf9639a.png)



如果我把它改成 100 点 x 100 点。

![InDesign Grid output 14](img/2792a09b1493540f0eae100b9a606396.png)



也禁用'网格在后面，对话框的一个选项，然后点击它的确定按钮。如果我们禁用此选项，那么网格将出现在文档页面所有内容的顶部。

![InDesign Grid output 15](img/546bb2dd7632e3fd48e966413d684735.png)



现在，您可以看到块的大小增加了，网格现在位于图像的顶部。因此，根据您的适合性，您可以将网格放在内容的上面或后面。

![InDesign Grid output 16](img/5aa12599f4976808b11213ab3f240252.png)



### 如何在 InDesign 中对文本使用网格？

现在我将使用文本工具创建一个文本框。现在我们明白网格是如何处理文本的了吧？

![output 17](img/12f3d309e9b3fea581b59b5f81c36af3.png)



现在，我将在这个文本框中粘贴一些文本。

![output 18](img/723b5cc287c4d80e07632c8432c44927.png)



您可以沿网格线移动文本框架，使其在所需位置与图像正确对齐。

![InDesign Grid output 19](img/9cf5c301ab4bc9505eb55c06103c9dc9.png)



现在让我们来看看另一种类型的网格，即基线网格。我将首先通过单击“隐藏文档网格”选项，从菜单栏的同一菜单中禁用此文档网格。

![InDesign Grid output 20](img/e738eebb21b72aaca1adc688c243e865.png)



现在，在相同的选项中，我将通过单击“显示基线网格”选项来启用基线网格。

![Show Baseline Grid output 21](img/202dcdde1568a7f06be344d9361ca0b9.png)



我们的文档页面上会有这种水平网格线。我们通常使用基线网格来对齐文本。

![output 22](img/9c40333ed167bbf1fd1995f47eab588d.png)



您可以看到文本没有完全与基线对齐，因此您不需要通过移动它来手动对齐它。

![output 23](img/1593d6a3fbaa239e071368406e2cce22.png)



只需从工具面板中选择文本工具，并选择此文本。您不需要选择整个文本，只需单击鼠标，文本的闪烁鼠标光标就会出现在文本框中。然后转到段落格式控制面板。您可以在工作屏幕的顶部找到此面板。如果您的工作屏幕上没有这样的面板，那么您可以将该软件的工作区更改为排版模式。

![output 24](img/917451d08591ac415251df4aedccfdf7.png)



并单击该面板的“对齐基线网格”选项卡。

![InDesign Grid output 25](img/7b9765bd99857e8af506c0d652c0b121.png)



一旦你点击这个选项，它会自动对齐我们的文本与基线，但你可以看到它对齐我们的文本在一个基线的差异。如果您希望文本与每条基线对齐。

![output 26](img/40622bd28030920c53b2ed6e88320b8c.png)



然后，您可以再次转到首选项选项的网格面板，并在此对话框中更改基线网格部分的增量间隔选项的值。我会把它换成 8 点。然后点击确定按钮。

![output 27](img/856856aa9b5b1311c513229f5bc5aa12.png)



您可以看到，与之前的对齐方式相比，它更紧密地对齐了我的文本。所以你可以根据自己的选择来设置网格面板的参数。

![output 28](img/9b6517855f4300c868becc11ef3d94ad.png)



### 结论

因此，今天我们讨论了 InDesign 中的网格及其类型，您会发现它在使用该软件进行任何创造性设计布局时都很有用。该软件中网格的优点是两种类型的网格都有自己的面板，可以对它们的参数进行一些调整，以获得不同的结果。
