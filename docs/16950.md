## GIMP 3D 文本简介

您可以将 3d 文本理解为具有三维外观的文本，它为查看者创建了 3d 对象的幻觉，在 GIMP 中，我们可以通过使用画笔工具的参数，通过在路径工具的帮助下创建路径，以及通过使用文本工具(它是一个非常必要的部分)来非常容易地创建它。我们也使用这个软件的图层面板的一些功能来获得我们文本的 3d 效果。使用 3d 文本，您可以在我们的图像处理工作和作品中添加效果和引人注目的功能。

### 如何在 GIMP 中创建 3d 文本？

只要和我们呆在一起，在 GIMP 中创建 3d 文本对你来说将不再是一个非常困难的任务。你只需要按照一些简单的步骤来设置工具面板中不同工具的不同参数。

<small>3D 动画、建模、仿真、游戏开发&其他</small>

你要做的第一件事是创建一个新文档。为此，进入文件菜单，点击它，然后点击“新建”选项，向下滚动列表。

![GIMP 3D Text output 1](img/f0bd04509385b99077f16939b2dc494b.png)



根据您的文本，您可以从该列表中选择任何预设的文档。

![GIMP 3D Text output 2](img/4865abe26dbb82f5315921c63e3b91f8.png)



或者设置文档大小的值。我要一份 1280 x 720 像素的文件。

![GIMP 3D Text output 3](img/929358d799e76f1fdbeb62fc6de2e487.png)



现在让我们在我们的文档上填充一个好的对比颜色，这将是你的文本的背景。所以点击前景色框。

![GIMP 3D Text output 4](img/d15302122810514f2da90bc988ad043f.png)



选择这种深绿色。

![GIMP 3D Text output 5](img/c8970b0aeefff17eddd688392cd5dbee.png)



现在进入编辑菜单，点击它，然后点击填充 FG 颜色选项，用我们选择的前景色填充文档。

![GIMP 3D Text output 6](img/eee8215f4c119dd02a1e0f78558ab180.png)



这是我们选择的颜色。

![GIMP 3D Text output 7](img/49bc4c8b166108e7a8981d5d7803f987.png)



现在转到工具面板，并从那里采取文字工具。

![GIMP 3D Text output 8](img/a2e80db877b9beee62b3d6f95cfcf2a5.png)



将前景色切换到亮色，这样我们就可以很容易地在深色背景上看到我们的文本。

![GIMP 3D Text output 9](img/46b89af55ba2244f7aa7475769580018.png)



现在为 3D 文本选择一个好的字体。我会选择这个脚本 MT 粗体字体。

![GIMP 3D Text output 10](img/a14ad77b7fb113817fd223b0ea99dfc7.png)



在此键入您的文本。

![GIMP 3D Text output 11](img/e4df0461652d632314c6d3d4161d7b7a.png)



根据您的选择放大或增加字体大小。

![GIMP 3D Text output 12](img/86dbb1deb00376426001044ce26c2a76.png)



使用移动工具将其移动到文档区域的中心。

![GIMP 3D Text output 13](img/cf3fe8dbce659cd0e70d1bb20f58daa3.png)



现在通过图层面板的不透明度比例值来降低图层的不透明度。

![GIMP 3D Text output 14](img/17c4037fb2bb450194949c563ff25b8a.png)



现在转到图层面板，点击创建新图层按钮，创建一个新图层。

![GIMP 3D Text output 15](img/c2b3c8f44382995eed8dadb1099c6284.png)



现在将打开一个对话框，用于设置新层的参数。不要更改此框的参数；只需选择透明度选项作为有一个透明的层的'填充'选项，然后点击确定按钮。

![GIMP 3D Text output 16](img/f1e416165faa8bbf1c51f1d19f4c5b93.png)



现在让我们选择前景和背景颜色，这将是我们的 3d 文本的渐变颜色。

![GIMP 3D Text output 17](img/94fe59f1306a0d56e94d7e9800913f4a.png)



一个一个选吧。我将从这个颜色选择框中选择亮黄色作为前景色。

![GIMP 3D Text output 18](img/fe96ef8f05a1ffbd312403c4c2bf07a8.png)



然后选择这一点点暗黄色作为背景色。你可以根据你的情况选择其他颜色。这完全取决于你的创造力。

![GIMP 3D Text output 19](img/88c099166b017854358cd36f7ea593f9.png)



现在，再次进入工具面板，点击椭圆选择工具来画一个圆。我们做这个圆是为了用它作为画笔工具提示的图案。

![GIMP 3D Text output 20](img/93d93fb0e68e7fbff54931693f7ec9e1.png)



确保你选择了一个透明的图层。现在按住键盘的 Ctrl 键画一个圆。

![GIMP 3D Text output 21](img/c45903605c96f5fd7ef526bc27a5a9a5.png)



从工具面板中选择渐变工具或混合工具。

![GIMP 3D Text output 22](img/6b448f19e541c5fba45b81c403b5fa59.png)



在画笔工具参数中选择“从底色到底色”选项，这样你就有了你选择的前景色和背景色的渐变效果。

![GIMP 3D Text output 23](img/78a48795a945f0009f0b3ed48054649b.png)



现在像这样在这个圆上画一个渐变。

![GIMP 3D Text output 24](img/64a2e7aa655df21d276e26e10c681fd1.png)



去选择菜单，然后点击“无”选项，使选择周围的圆圈。

![GIMP 3D Text output 25](img/d83c6d9a9f3b30e88aea8f792c9d88a1.png)



现在从工具面板中选择缩放工具来缩小这个圆，因为我们想让它成为画笔笔尖图案。

![GIMP 3D Text output 26](img/2f92447cfbf89b0cbbbc51c7264d92ce.png)



像这样缩放它。

![GIMP 3D Text output 27](img/e3f41252815388715015cea9c878faec.png)



然后按键盘上的回车键。

![GIMP 3D Text output 28](img/d902063953dd0da4fcf8f9e3c2ceaf9b.png)



现在去图层面板，右键点击一个透明图层。从下拉列表中选择“选择的 Alpha”选项。

![GIMP 3D Text output 29](img/0c3f122335b5e93cc25bc31ffc10583b.png)



现在转到菜单栏的‘编辑’菜单，点击复制选项或者按 Ctrl + C 键作为快捷键在 GIMP 软件的剪贴板中复制这个圆。

![GIMP 3D Text output 30](img/ce2e5716ca7d1da42f2c12b4380a3eca.png)



现在启用这个圆周围的选择，所以点击菜单栏的选择菜单的'无'选项。

![GIMP 3D Text output 31](img/af7ac933ce9d2ab45b59b0e20159332e.png)



现在通过点击眼睛按钮使这个图层不可见，并从工具面板中选择画笔工具。点击笔刷工具参数的图案选项。你将在列表中复制一个圆形作为笔刷图案。选择它。

![GIMP 3D Text output 32](img/64cdc1fa73c3f01eda6998f2d0f3cd70.png)



启用平滑描边选项，使文本平滑描边。

![GIMP 3D Text output 33](img/f8ff0c3d57bb1d9caa4017d276b31011.png)



现在你可以用这个图案画出这种类型的 3d 形状。

![GIMP 3D Text output 34](img/513093ef8d342cf9f5ded0d3d3543be7.png)



如果你能很好地掌握它，你可以写一篇自由的文章。

![output 35](img/00db03baf8181f1beca0dd90206f3382.png)



但是如果你没有，那么不要担心，让我们做一些不同的事情。从工具面板中选择路径工具。

![Output 36](img/78a09a0784c164f3de741bbe45987555.png)



像这样画一条直线路径。

![Output 37](img/d0e0bb112b0daaabe2ccfae5993976d9.png)



现在像这样调整它的曲线。

![Output 38](img/69e639e80943a71c933a9fa475beffea.png)



我是这样做的。你照你的课文做。

![Output 39](img/9830a388d45826e896ba636e75f5791b.png)



你已经误入歧途，那就不要担心；去路径标签，这是下面的层面板，并点击它。你在这里会有你的路；只要点击一下，你就可以再次拥有它。

![Output 40](img/578de4f41e1d1b2434e7b2ea37460633.png)



现在选择它，点击它，并点击画笔工具的笔画路径选项。

![Output 41](img/7783c28cab4e325effddbfe561ded877.png)



现在点击这个对话框的“用绘画工具描边”选项的单选按钮。

![Output 42](img/8f9dd95754ae8763f1a5ef1860b6ce90.png)



在绘画工具中选择画笔选项。

![Output 43](img/23af89178a99b2b3f2f8c4e2aa098cf8.png)



启用此对话框的模拟笔刷动态选项，并点击笔画按钮。

![Output 44](img/cb3e87e3608149938a5e1dcf447b8f76.png)



你会在你的文字上有这样的笔触。这是 3d 文本。

![Output 45](img/56b2281ec7ad1fb0b050eddd438f560e.png)



如果你对路径的曲线不满意，去编辑菜单，点击撤销描边路径选项。

![Output 46](img/54fc0db4c2641abb0f6b6e05c94e81b0.png)



并根据你来编辑。

![Output 47](img/6a2a81954ff948da3e41b08869ad613f.png)



这样，您可以在 GIMP 中创建 3d 文本。

![Output 48](img/5ab4a0c1466e44b8ab9a553d324f1938.png)
