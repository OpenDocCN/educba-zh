## 计算机内存导论

内存是计算机的必要组成部分。存储和分析数据并相应地对用户在计算机上推断的命令做出响应是一种基本需求。内存是计算机功能的存储部分。立即使用的信息和计算机硬件设备及其功能也存储在存储器中。为了处理计算机中的信息，高速运算发生在存储器中。与人脑相似，计算机上也有不同类型的存储空间。随着时代的变迁和技术的不断发展，内存也在升级。

### 计算机中不同类型的内存

计算机中主要有两种内存——主内存和辅助内存。主存储器包括只读存储器和随机存取存储器。辅助内存包括单独包含的硬件存储设备，如 HDD(硬盘驱动器)、SSD(固态驱动器)、光盘和其他设备。计算机系统在这些存储器类型的帮助下操作和运行。每种类型的内存都有自己的优势和劣势，并不断进行研究和升级，以使世界各地使用计算机的用户受益。由于在计算机存储器领域成功地进行了研究以增强体验和存储最大数据，计算机技术已经取得了巨大的成功。一些计算机内存可以存储数据更长的时间，即使没有电源。它们被称为非挥发性的。有些只能在短时间通电的情况下存储，它们被称为易失性的。如果在计算机的配置中使用这两种类型的存储器的非常好的组合，它提供极好的结果。让我们来看看两种类型的记忆:

#### 1.主存储器

这种类型的存储器包括 RAM(随机存取存储器)和 ROM(只读存储器)。它存储非常少量的数据，并且当计算机在电源模式下运行时可以非常快速地访问。

##### 随机存取存储器

可以以随机方式访问存储在 RAM 存储器中的数据。不需要特别的顺序。这是最昂贵的内存类型。它速度非常快，但是运行、交换数据和共享存储在其存储器中的数据需要电力。任何需要立即处理的数据都被移动到 RAM 中，在 RAM 中，数据的修改和访问被快速完成，因此没有等待 CPU 开始工作的时间。它提高了 CPU 的处理速度和用户给它的多种不同命令的处理速度。任何不再使用的数据都将被转移到二级存储器中，RAM 中的空间将被释放以供进一步处理。

两种类型的 RAM 是:

**a. DRAM**

动态随机存储器是计算机中常用的随机存储器。与早期计算机使用单数据速率(SDR) RAM 不同，现在它们使用具有更快处理能力的双数据速率(DDR) RAM。DDR2、DDR3、DDR4 是 DRAM 的可用版本，根据它们的数量每个都是有效的。在安装之前，需要检查 DRAM 与操作系统的兼容性。

**b. SRAM**

静态 RAM 比动态 RAM 快，因此更贵。它的每个单元中共有六个晶体管。因此，它通常被用作 CPU 和高端服务器中的数据缓存。它有助于提高计算机的速度。

##### 二。只读存储器

数据只能读取，不能写入。这是一种快速记忆。它是一种非易失性存储器。在没电的情况下，数据存储在其中，然后在通电时由计算机访问。它通常包含引导代码，计算机需要这些代码来与硬件系统进行交互，并根据给它的命令来理解它的操作和功能。在通常使用的较简单的设备中，有 ROM，其中存储了固件数据，以使它们根据需要运行。

#### 2.辅助存贮器

它在物理上被放置或位于一个单独的存储设备中，然后连接到计算机进行访问。硬盘和固态硬盘通常是计算机中最常用的辅助存储设备。它们不像 RAM 和 ROM 那样昂贵，而且价格合理。它们的读写速度也相对较低。因为它是一个完全独立的设备，所以它可以在不需要电源的情况下保存存储在其中的数据。交换和存储的数据首先发送到 RAM，然后发送到 CPU 进行进一步处理。它们有可能永久存储数据，同时，你可以很容易地在任何地方使用它们，并随身携带。

### 计算机内存的使用

计算机内存是 CPU 及其正常运行计算机的基本需求之一。计算机内存加速了 CPU(中央处理器)的数据处理。临时和永久数据的存储对于计算机运行其处理过程以及通过与支持计算机及其运行的所有硬件设备通信来响应用户命令是必要的。计算机的效率需要内存作为其系统中的基本便利。记忆力越好；计算机运行得越快。

### 结论

存储器在计算机系统的运行中起着至关重要的作用。随着技术的进步，更好的存储速度的发明肯定会即兴发挥计算机性能的质量，今天在全球范围内用于不同行业所有类型的所有主要编程和过程。
