## 密码学简介

术语“密码术”指的是保护重要数据和通信免受敌人攻击的策略的练习和详细学习。按照[计算机科学](https://www.educba.com/what-is-computer-science/)的说法，加密就是保护数据和通信策略，这些策略来自基于规则的计算网络和被称为“算法”的数学理论这个过程导致消息以不同的方式转换，不容易解码。这些确定性算法在生成密钥、数字签名和验证方面起着至关重要的作用。这最终屏蔽了数据，在互联网上浏览。它还有助于保护机密通信，如通过信用卡、电子邮件等进行的交易。

### 什么是密码学？

通过采用代码创建一个平台来防止数据和通信，该平台的信息应该能够被读取和处理，这种概念称为密码学。在这里，术语“地穴”指的是“隐藏/金库”，而术语“图形”指的是“书写”。加密技术是一种应用程序，用于实现安全可靠的信息和通信流程，防止众多可疑和恶意的第三方——对手。这里，选择加密和密钥将输入的明文转换为加密的输出密文。因此，只要使用相同的密钥，特定的算法就会通过有效地转换相同的明文来生成相同的密文。

### 密码学的目的和优势是什么？

在保护数据和其他机密问题时，传统加密技术无法满足组织的需求。

但是新时代的加密技术有能力屹立不倒并熟练地实现以下四个目标:

#### 1.最好的隐私

该数据可以被任何其他人承认，无论该数据是对谁而言都是无意的。保护敏感数据，甚至包括人们的个人信息，是使用密码学的意义之一。这牢牢地控制了隐私和其他与匿名相关的问题，从而减少了犯罪分子和假冒监控实体解码机密信息的机会。

#### 2.保持诚信

在没有检测到任何可疑活动的情况下，对于发送者和预期接收者来说，数据不能在存储或传输中被改变。通过识别机密信息，黑客可以滥用它并驱使它实施严重的欺诈。加密技术使拥有相关技能的专业人员能够修改加密信息。这种药物一旦被修改，就能被接受者识别出来。这有助于维护整个企业的完整性，并使 it 部门能够应对这种网络犯罪。

#### 3.突出的不可否认性

使用加密技术，信息源永远不能否认其在以后阶段创建/传输数据的意图。

#### 4.跨设备保护身份验证和数据

加密技术为发送方和接收方提供了识别和确认身份以及数据来源/目的地的能力。在当今的现代世界中，移动设备通过为人们提供信息传输和存储功能，已经成为人类不可或缺的一部分。加密有助于在所有类型的移动设备之间传输信息时保护和存储信息。除此之外，它还支持高级身份认证等安全程序，有助于阻止未经授权的客户。

### 密码术的类型

随着密码学从计算机科学、数学、电子工程等领域发展而来，它被应用于各个领域，如电子商务、数字货币、基于芯片的支付卡等等！

根据功能，有不同种类的加密算法和密钥，讨论如下:

*   **秘钥密码术:**主要用于维护隐私和安全，它使用单个密钥进行加密和解密。这通常也称为对称密钥加密。在这个过程中，发送方可以使用密钥对作为密文的明文进行加密。在接收者端，使用相同的密钥，消息被解密为明文。
*   **公钥密码学:**主要用于身份验证的目的；公钥或非对称加密是通过使用一个密钥进行加密，另一个密钥进行解密来执行的。在这种情况下，公钥用于加密消息，而接收方拥有私钥用于解密消息。
*   **哈希函数:**主要是为了消息完整性而实现的，该功能使用数学算法来加密消息，而不是密钥。当今环境中最常见的哈希算法是 SHA-1 和 MD5。

### 例子

让我们构建一个场景来理解密码学概念。假设一个人，A 已经通过一个公众平台给他在异地生活的朋友 B 发了一条短信。然而，一个名为 C 的人破解了通信课程的安全性，并更改了来自 A 的消息，将其重定向到 b。这就是密码学概念的由来。

现在 A 用一个密钥加密同一个消息，通过同一个公共平台发送；然而，这一次，没有人能够解码信息，因为它是加密的。接收者 B 将使用他的解密密钥将消息再次转换成可读格式。即使 C 中途攻击了通信信道并修改了消息，B 在试图解密相同的消息时也会得到错误信息。

### 密码学是如何使用的？

加密的基本目标是保护消息的真实性和完整性。在通信网络中使用电子钥匙的概念是为了确保传输的数据没有任何中断地到达接收器。在安全计算机系统的帮助下，发送者现在可以用密钥和算法解密信息。

### 为什么我们需要密码学？

每当通过电子网络进行通信时，都必须通过加密技术来满足强制性的安全要求。

创建加密平台的基本要素包括:

*   隐私
*   不可否认性
*   证明
*   完整

不同的密钥和算法可用于满足不同操作的需要。金融交易、与安全相关的信息、行政管理、法律等等，无论电子通信发生在哪里，加密技术都发挥着重要的作用。

### 这项技术将如何帮助你在职业生涯中成长？

在过去的 4-5 年里，密码学的概念蓬勃发展。像优步、易贝、GEICO 等大公司正在寻找在这一技术领域有专长的专业人士。从经理、开发人员、数据科学家等等开始，这项技术为年轻的专业人士提供了广泛的就业渠道。

### 结论

我们的世界比以往任何时候都发展得更快，我们很容易忽视那些让我们以现在的方式生活的技术基础元素。密码术从一种用于与军方通信的技术发展成为现代世界的基石之一。随着创新和进步，密码学有潜力将世界推向新的高度。
