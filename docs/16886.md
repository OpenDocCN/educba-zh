## Figma 快捷方式介绍

在这篇文章中，我们将看到现在经常需要的最有用的快捷方式；想象一下，如果你重复使用同一个命令，那么使用快捷方式是可行的，因为它肯定会节省你的时间。Figma 的快捷方式是直观的，如果你使用过像 photoshop 这样的程序，那么这应该感觉就像在家里一样，因为许多快捷方式都是相同的，例如，钢笔工具。

### Figma 快捷方式的步骤

下面给出的是 Figma 快捷方式的步骤:

<small>3D 动画、建模、仿真、游戏开发&其他</small>

第一步:我们可以在网上使用 Figma，也可以下载应用程序。在这篇文章中，我们做的是应用。首先，我们将开始寻找查看所有键盘快捷键的位置。所以你要么按？或者按 Ctrl + Shift +？查看所有快捷方式。

![Figma Shortcuts 1](img/5279a9c36043bac36ca40808bdc98132.png)



**第二步:**首先你可以看到，UI 和颜色的必备键盘快捷键，以及快速动作和其他快捷键，都是按照功能来划分的。

![Figma Shortcuts 2](img/83ba95e9ca124d9e8bc5397a5a4bade0.png)



**第三步:**当您使用快捷方式时，它会变成蓝色，表示它已被使用。

![it turns into blue](img/a08bcbb11328a0e453f97f975e40e354.png)



**第四步:**我们用快捷键 r 从一个矩形开始。

![keyboard shortcut R](img/7e8e6fd3752c74f036baebaa27c75f8d.png)



**第五步:**同样，我们有 L 代表直线，O 代表椭圆，T 代表文字，F 代表框架。

![we have L for line, O for Ellipse, T for Text and F for Frame](img/7c7981993e0bb990dfbcbcb5fc39e4ab.png)



**第六步:**假设我们放大了，这可以用 Ctrl 和鼠标滚轮来完成，我们需要平移，所以我们要么用 H 的手形工具，要么你可以用空格键点击平移，释放空格键回到之前的命令。

![Figma Shortcuts 6](img/66002bacc77a7075f77af839d65e48e8.png)



**第七步:**让我们用红色填充圆圈，如果你想改变不透明度，然后按数字；例如，按下 4 会将其变为 40%不透明度的红色。

![press the number](img/2856406e347757307f3b1c583731b3e9.png)



**第八步:**快速按 2 个数字，像 45，不透明度为 45%。

![Figma Shortcuts 9](img/3cb9213f4af46a83e49d33e2a5ef1f06.png)



**第九步:**可以按 Shift + 1 缩放至适合。

![press Shift + 1 to zoom to fit](img/4b5a5df2e87b943a698e73e5aa21ecc0.png)



**第十步:**可以按 Shift + 2 缩放到选区)。

![Figma Shortcuts 10](img/2d22085b5e300f73ba88bb2409e1a509.png)



**第十一步:**可以按 Shift + 0 缩放到 100%。此外，您可以按 Z 键并拖动以放大，按 alt + Z 键以缩小。

![Shift + 0 to zoom to 100%](img/6e7add8e89e2e6cc604e878408dea5b9.png)



第 12 步:如果你想做一些更精细的调整，你可以用箭头键移动物体，移动 1 像素。按压移动将移动 10 像素。

![Figma Shortcuts 12](img/feaf5d8677f7749c72a19365af0b4563.png)



**步骤 13:** 类似的在轻推的同时按 Ctrl 会改变矩形的大小 1 px，按 shift 会改变 10 px。

![Figma Shortcuts 13](img/23bfb01e3eca710e6718c13393c33e06.png)



**第十四步:**按 Ctrl + G 可以创建一个组。

![create group by pressing Ctrl + G](img/643affbb07f84dc118fdb98d299a4866.png)



**第十五步:**接下来是快捷键 I 可以使用的滴管工具，首先选择你要填充颜色的对象，然后点击你要填充颜色的矩形。

![Figma Shortcuts 15](img/8a64114e2cffb815644dccf280c2c81e.png)



**步骤 16:** 我们可以看到，圆形填充的是矩形的颜色。

![Circle is filled with the color of rectangle](img/58dd9fe5600fbbce1279348b06e41d1c.png)



**步骤 17:** 现在，当网格添加到框架中时，可以通过单击眼睛图标来显示和隐藏网格，但有一个使用 Ctrl + Shift + 4 切换显示和隐藏的快捷方式。

![Figma Shortcuts 17](img/663774299710997c3578d913fa9e6b7e.png)



步骤 18: 现在，如果你有一个多页的文档，你可以使用 Page up 和 Page down 键在它们之间移动。

![Figma Shortcuts 18](img/5f100feafc50aecd5ae3bb39193688f4.png)



让我们创建一个正方形，然后我们可以使用 Ctrl + D 复制并移动它，如图所示。

![then we can duplicate using Ctrl + D](img/807de4b73d17030e0640bad0c59e9699.png)



**步骤 20:** 现在，如果您再次按 Ctrl + D，那么它被复制，并且位置或偏移被自动应用。

![Figma Shortcuts 20](img/6a1c970b40c2f5c991077666b9065548.png)



第 21 步:我们也可以复制一组对象，我们为其创建一组正方形，然后复制它。

![Figma Shortcuts 21](img/77387bfe17fa7960c4dad89ef107cec2.png)



**步骤 22:** 接下来，要对齐垂直和水平中心，我们有一些便捷的快捷键 Alt + H 和 Alt + V。

![Figma Shortcuts 22](img/2bf0a96244939f4ce7f2d57769e2de04.png)



![Figma Shortcuts 23](img/ccbb947b1c6ed9e9dc5657c472030d27.png)



**步骤 23:** 然后，我们可以使用钢笔工具，使用快捷键 p，创建带有锚点的自由流动的形状。

![Figma Shortcuts 24](img/6e819372af984b5b7ecd2e6a2c5b89ea.png)



**步骤 24:** 使用快捷键 Shift + P，我们可以使用铅笔工具使用快捷键 Shift + P 创建没有锚点的自由流动的形状。

![Figma Shortcuts 25](img/d3a01612065a2bd5005618602dbd6d9b.png)



**第 25 步:**如果要看标尺，那么按 Shift + R。

![Figma Shortcuts 26](img/0927f0459bbae15042be641d68be5e41.png)



步骤 26: 如果你只想查看轮廓，按 Ctrl + Shift + 3，去掉所有的颜色和阴影。

![Figma Shortcuts 27](img/527ce9a31a69399ca8af2d8c1b151d28.png)



**步骤 27:** 如果你想要像素级的控制，那么按 Ctrl+’。

![Figma Shortcuts 28](img/801c32ddb543b3384e32dd2ce2527a6d.png)



**步骤 28:** 你也可以使用 Ctrl + Alt + Y 打开像素预览，它会以曲线的形状显示。

![Figma Shortcuts 29](img/7d635cd1e476a1f29e1a6b8d6e9e79d6.png)



关于格式化文本，我们可以将 Ctrl + B 设为粗体，将 Ctrl + I 设为斜体，Ctrl + U 将加下划线。我们可以用 Shift + Alt + <和>来调整行高。我们可以使用 Alt + <和>来调整字母间距。我们也可以使用 Ctrl + Alt + <和>来控制字体的粗细。

![Figma Shortcuts 30](img/827aa05207f43b30cdca0415084c2f43.png)



**步骤 30:** 文本对齐可以使用 Ctrl + Alt + L 调整到左边，使用 Ctrl + Alt + T 调整到中间，使用 Ctrl + Alt + R 调整到右边，如果你想两端对齐，那么使用 Ctrl + Alt + J。

![Figma Shortcuts 31](img/da9c1088d8de5f936840e2e9e7c537d6.png)



**步骤 31:** 假设我们要移除填充，然后按 Alt + /并移除笔画，然后按/。

![Figma Shortcuts 32](img/5e68ec9a0af77f2ea224be2bf9ef2435.png)



使用 Shift + X，我们可以交换填充和描边。

![Figma Shortcuts 33](img/72e85305568eff3561a30a23fafc6f25.png)



**步骤 33:** 按 Ctrl + A 会选择所有内容，按 Ctrl + Shift + A 会反转当前选择，按 Esc 会不选择任何内容。

![Figma Shortcuts 34](img/272081e3e96ccaa712694ba153883793.png)



**第 34 步:**如果你有一个组，想要选择其中一个元素，那么使用 Ctrl + click 的深度选择。

![use Deep select by Ctrl + click](img/5d022c21127ccd4c5e89e3b97574d128.png)
