## 什么是 Matlab 3D Plots？

MATLAB 是一种用于技术计算的语言。我们大多数人都会同意，一个易于使用的环境是集成计算、可视化和最终编程任务所必需的。MATLAB 通过提供一个不仅易于使用，而且我们得到的解以我们大多数人都熟悉的数学符号的形式显示出来的环境来做同样的事情。

### MATLAB 中三维图形的类型

下面我们讨论了 MATLAB 中用于计算的 3D 图形的类型。

#### 1.图 3(线形图)

Plot3 有助于创建 3D 线或点图。

**Plot3(x，y，z):** 如果 x，y，z 是等长的向量，那么这个函数会创建一组由线段连接的坐标。如果我们指定 x、y 或 z 中的至少一个作为向量，它将为同一组轴绘制多组坐标。

**代码:**

`A= 0: pi/100; 50*pi;
sa= sin(a);
ca=cos(a);
plot3(sa, ca, a)`

**输出:**

**plot3(** **X** **，** **Y** **，** **Z** **，** **LineSpec):** 该函数将使用指定的线型、标记和颜色创建绘图。

**代码**:

`A= 0:pi/20:2*pi;
sa= sin(a);
ca=cos(a) ;
plot3(sa, ca, a, ’o’)`

**plot3(X1，Y1，Z1，…，Xn，Yn，Zn):** 该功能将为同一组轴绘制多个坐标。

**代码**:

`a= 0:pi/100*pi;
xa1= sin(a).cos(10*a);
ya1=sin(a).*sin(10*a) ;
za1=cos(a);
xa2= sin(a).cos(15*a);
ya2=sin(a).*sin(15*a) ;
za2=cos(a);
Plot3(xa1,ya1,za1,xa2,ya2,za2)`

#### 2.散点图 3(三维散点图)

**散射 3(** **X** **，** **Y** **，** **Z** **):** 该功能将在 X、Y、Z 的矢量位置创建圆

**举例**:

x，y，z 是向量球。

**代码:**

`[X,Y,Z] = SPHERE(10)
x = [0.5*X(:); 0.25*X(:); X(:)];
y = [0.5*Y(:); 0.25*Y(:); Y(:)];
z = [0.5*Z(:); 0.25*Z(:); Z(:)];`

**输出:**

![3D Plots in Matlab 4](../Images/cb0bdd7c7ec8d88e7796d6c5f92cccf6.png)

<noscript><img class="alignnone wp-image-240058 size-full" src="../Images/cb0bdd7c7ec8d88e7796d6c5f92cccf6.png" alt="3D Plots in Matlab 4" width="513" height="388" srcset="https://cdn.educba.com/academy/wp-content/uploads/2019/11/3d-plot4.png 513w, https://cdn.educba.com/academy/wp-content/uploads/2019/11/3d-plot4-300x227.png 300w" sizes="(max-width: 513px) 100vw, 513px" data-original-src="https://cdn.educba.com/academy/wp-content/uploads/2019/11/3d-plot4.png"/></noscript>

**scatter3(** **X** **，** **Y** **，** **Z** **，** **A** **，C):** 该函数将创建一个带有一个圆的图，该圆的大小为 A。如果 A 是标量，圆的大小将是相等的。对于圆的具体大小，我们将不得不把 A 定义为一个矢量。

**代码:**

`[X, Y, Z] = sphere(20);
x = [0.5*X(:); 0.75*X(:); X(:)];
y = [0.5*Y(:); 0.75*Y(:); Y(:)];
z = [0.5*Z(:); 0.75*Z(:); Z(:)];
scatter3(x, y, z)`

定义一个向量来指定标记大小。

`a = repmat([50,10,2],numel(X),1);
C = repmat([1,2,3],numel(X),1);
a = a(:);
c = c(:);`

**输出:**

#### 3.轮廓 3(三维轮廓图)

**contour3(Z):** Z 是一个矩阵，该函数将创建一个三维等值线图，其中包含矩阵 Z 的等值线以及 x 和 y 平面的高度细节。平面上的 *x* & *y* 坐标是 z 的行列索引，等高线由 MATLAB 自动选取。

**代码**:

`[X,Y] = meshgrid(-1:0.20:2);
Z = X.^2 + Y.^2;
contour3(Z)`

**输出:**

![3d plot 6](../Images/4bc57441979c3d12e701726f3c6f39e2.png)

#### 4.QUIVER3(速度图)

3D 颤动图在点(x，y，z)创建具有分量(u，v，w)的向量，其中 u，v，w，x，y 和 z 是实数值。

**代码**:

`x = -3:0.5:3;
y = -3:0.5:3;
[X,Y] = meshgrid(x, y);
Z = Y.^2 - X.^2;
[U,V,W] = surfnorm(Z);`

**输出:**

#### 5.FILL3(三维填充多边形绘图)

这个功能有助于创建平面形状和 Gouraud 多边形(以获得不同的光线阴影)。

**fill3(X，Y，Z，C):** 它帮助创建具有顶点 X，Y，Z 的填充多边形。X，Y，Z 值可以是数字、持续时间和日期时间等。

**代码:**

`X = [1 2 3 4; 1 1 3 2; 0 1 2 1];
Y = [2 2 1 1; 1 2 1 2; 1 1 0 0];
Z = [1 1 1 1; 1 0 1 0; 0 0 0 0];
C = [0.5000 1.0000 1.0000 0.5000;
1.0000 0.5000 0.5000 0.1667;
0.3330 0.3330 0.5000 0.5000];`

**输出:**
