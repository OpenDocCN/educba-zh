## 红移日期简介

Redshift 提供了不同类型的内置函数来根据用户需求对数据执行不同的操作。例如，dateadd()是 Redshift 提供的函数之一。基本上，dateadd()函数用于通过添加所需的日期和时间戳来返回新的日期-时间值，或者我们可以根据用户要求指定日期和时间戳。换句话说，我们可以说 dateadd()函数用于根据添加的时间间隔返回特定的日期和时间。通常，当我们需要根据需要指定未来和过去的日期和时间时，会使用 dateadd()函数。

**红移日期添加的语法:**

<small>Hadoop、数据科学、统计学&其他</small>

`dateadd (specified datepart, specified interval, {specified date| specified time|specified timetz| specified timestamp })`

**说明:**

在上面的语法中，我们使用 dateadd()函数和不同的参数，如下所示:

*   **指定日期部分:**指定日期部分是指我们可以将数据分成年、月、日等不同的部分，也可以说是小时，也就是说我们可以根据自己的需要来划分日期。
*   **指定间隔:**间隔是指指定的间隔，或者我们可以说我们需要的天数，可以是正数，也可以是负数。这也取决于用户需求。
*   **specified date | specified time | specified timetz | specified timestamp:**基本上是用来根据用户需求转换日期、时间或者时间戳，但是这个表达式需要一个 specified date 部分。
