## InDesign 圆角简介

InDesign 圆角是形状或图像的一个特征，通过它我们可以使它们的角变圆，这意味着我们可以在形状的角上提供圆弧。InDesign 中有一个功能，通过它我们可以为不同目的制作圆角。不仅仅是形状，我们还可以用放置的图像和其他东西来做这件事。我们有两种方法可以做成圆角。第一种是使用角点选项，第二种是直接使用形状或图像的锚点。让我来告诉你如何做到这一点？

### 如何在 InDesign 中创建和使用圆角？

在这个软件中，为有角的形状或图像创建圆角是非常简单的任务。我们只需要了解这个软件的角选项的一些参数，我会告诉你一个更好的方法，就是通过处理形状或图像的锚点来做到这一点。

首先，我将创建一个新文档，为此，我将单击该软件欢迎屏幕的“新建”按钮。

![InDesign rounded corners output 1](img/d5368b8282048d5b055830d3afdb3c70.png)



我将使用新建文档对话框的默认设置。

![InDesign rounded corners output 2](img/5c69de53866ad65ee20015f737ba2915.png)



现在，让我们创建一个矩形形状的角落，我们将转换为圆形。为此，我将从工具面板中选择一个矩形工具，或者你可以按 M 键作为快捷键。

![InDesign rounded corners output 3](img/5045c88d2ebcb6d4f61a32d6b02ff9d4.png)



我将创建一个像这样的矩形。

![InDesign rounded corners output 4](img/fc81f302018a3d7328117ab99c219b91.png)



这个矩形没有填充颜色，只有线条。

![InDesign rounded corners output 5](img/011325cfa6bf92c604fabfca2b53ca71.png)



我将增加笔画宽度为 2 点，这样我们可以看得更清楚。

![InDesign rounded corners output 6](img/7644afd8c5b7a4078b58ec4d7f8cd16b.png)



我将打开角选项的属性面板，它位于面板部分工作屏幕的右侧。

![InDesign rounded corners output 7](img/af1cd98e647389d230505e98bdfad803.png)



如果您的工作屏幕中没有属性面板，那么您可以在窗口菜单中找到它。所以去窗口菜单并点击它。在此菜单的向下滚动列表中，您可以找到属性面板。

![InDesign rounded corners output 8](img/a43fc616288c559a78793d3bf1af2dee.png)



在属性面板中，我们有一个角选项，所以单击它。一旦你点击它，一个角选项对话框将会打开。

![InDesign rounded corners output 9](img/e5f2853552de43bee44ce2d90394dee4.png)



您也可以从菜单栏的“对象”菜单中打开此对话框。所以去对象菜单并点击它。在它的下拉列表中，我们有角选项。点击它启用它。

![InDesign rounded corners output 10](img/7b65e33187ba8958ff661ca50351eeb8.png)



在这个对话框中，我们有四个选项用于这个矩形的所有四个角，所以让我们来讨论一下。如果我们改变任何一个角的值，在中心有一个链式链接图标显示所有四个角的值成比例地改变。我们也可以根据我们的要求取消链接，我将在本文的后面部分告诉您。

![InDesign rounded corners output 11](img/c7efbfb86029456e828054ffdb8a35e3.png)



我将点击第一个角框选项的向下箭头按钮，在弹出列表中，我们有一些角的样式，如花式，斜面等。“四舍五入”在列表的最后，所以我要点击它来选择它。

![InDesign rounded corners output 12](img/56f036862f9b5d07f36712c892a85755.png)



你可以看到这个矩形的角变成了圆形。所有的角都是相连的，所以如果我们改变一个角，其余的也会随之改变。

![InDesign rounded corners output 13](img/0919573018c9d1b7d154c732d03ef7eb.png)



也可以在值框中输入所需的值，以便在拐角处创建所需的圆弧。

![InDesign rounded corners output 14](img/fe562e0eb4437a71fcf0cf3219ae8835.png)



圆角还有一个选项，但它会以相反的方向圆角。这次我将从转角样式列表中选择它。

![InDesign rounded corners output 15](img/a462de228c2c6efcac40a113de525020.png)



你可以看到它会反方向绕过拐角。您也可以在这里更改弧线值。

![InDesign rounded corners output 16](img/2ebc850970726e2a151dddf1f1d5edf7.png)



现在让我们看看如果我们解除这个链图标的链接会发生什么。要取消链接，请单击此图标。

![InDesign rounded corners output 17](img/ec845e60cf3b4fd9fbb592bc155d635d.png)



现在，我将只更改第一个角的圆角值，您可以看到这次更改只应用于所选的角，因为链图标未链接。

![InDesign rounded corners output 18](img/825cf28a2426dd20fe6713e992b13a47.png)



现在我将复制这个矩形并填充一些颜色来解释制作圆角的其他方法。您也可以将此方法应用于笔划。

![InDesign rounded corners output 19](img/ed4670bd83c51517544a79bd71c20abf.png)



在这个矩形的边缘有一个黄色的正方形锚点。

![InDesign rounded corners output 20](img/50d11cc8d9c1c83a1b7240f15b4c3975.png)



点击定位点，它将变成四个定位点，将出现在所有角落。

![InDesign rounded corners output 21](img/01796d4b5476d690f44cd7b704a69a88.png)



用选择工具按住黄色锚点，向内拖动。

![output 22](img/d46d574f98c4a36ecd5b5dc333c77c23.png)



当你向内拖动定位点时，它会绕过拐角。

![InDesign rounded corners output 23](img/8d801d8e5f7b4287600f25cd3cf41bfa.png)



如果你只想弄圆一个角，按住键盘上的 shift 键，向内拖动这个黄色的定位点。

![output 24](img/4a770be2677007ab2d63ebd4ce3e3d63.png)



它会把选择的角变成圆角。

![output 25](img/9f45cdafbad1c2f02411dc34a5c72888.png)



如果你按住键盘的 Alt 键，然后点击这些黄色的定位点，那么它将改变角的样式，在这个序列中，你可以反转圆角。

![InDesign rounded corners output 26](img/5adac6f1fa1309a6b855431a8f2bced4.png)



现在让我告诉你这个圆角如何处理图像。为此，我将在该软件中放置一个图像，因此我将单击文件菜单中的放置图像选项，或者您可以按 Ctrl + D 作为快捷键。

![output 27](img/4346b445ac641c2309755c0e6939de8f.png)



我会选择这个图像。

![output 28](img/be87d522d2e3b018cae8d20045a9b6ea.png)



然后像这样放在文档区。

![output 29](img/0f9518c25d9001d11372ca1b56a3db16.png)



您可以看到有一个与我们形状相同黄色锚点。所以点击它。

![output 30](img/f9987c79c938521c2c73f717e65f1494.png)



我会像这样向内拖动它。

![output 31](img/3c0ce8aa39a00c127ba503850e45dbdf.png)



它变成了圆角。

![InDesign rounded corners output 32](img/3e5f6efd90a3146ebf3a11f753fb7a22.png)



您也可以从角选项中完成此操作，因为我们已经将此角选项用于形状。

![output 33](img/2b1c04cf173a7056b613649172bc52a3.png)
