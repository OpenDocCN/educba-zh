## 建筑设计工具介绍

在这篇文章中，我们将学习什么是架构设计工具。架构师使用的主要工具是绘图，但是正如我们从这篇文章中所了解到的，这些绘图是如何制作的已经发生了变化，并且还在不断变化。各种建筑设计软件的使用取决于情况的需要和要求，因为你现在会发现所有的要求你需要一个软件，由于缺乏功能或兼容性或成本因素。你对设计工具的选择也取决于你愿意花在绘图过程上的时间，因为一些软件有一个陡峭的学习曲线，如果你打算长时间使用它，你需要投入时间。

#### 1.Revit

![Architectural design tools output 1](../Images/583295c899e2d58b3dc63da88e56387d.png)

<noscript><img class="alignnone size-full wp-image-446437" src="../Images/583295c899e2d58b3dc63da88e56387d.png" alt="Architectural design tools output 1" width="750" height="422" srcset="https://cdn.educba.com/academy/wp-content/uploads/2021/01/Architectural-design-tools-output-1.jpg 750w, https://cdn.educba.com/academy/wp-content/uploads/2021/01/Architectural-design-tools-output-1-300x169.jpg 300w" sizes="(max-width: 750px) 100vw, 750px" data-original-src="https://cdn.educba.com/academy/wp-content/uploads/2021/01/Architectural-design-tools-output-1.jpg"/></noscript>

Autodesk Revit 是一种 BIM(建筑信息建模),被从建筑师到结构工程师的各种学科所使用。除此之外，它在机械和电气工程领域也有它的用途。使用 Revit，可以创建结构构件，并在三维空间中对其进行可视化。

它提供了一个完整的包，从最初的概念化到最终的建设以及维护任务。Revit 可用于创建整个建筑以及装配。也可以单独创建 3D 形状。

建模可以通过内置资产来完成，或者您可以导入自己的几何资产。Revit 的一个缺点是它不能操纵单个多边形，因为它缺乏 NURBS 建模功能。

#### 2.Autodesk 公司的计算机辅助设计软件

![Architectural design tools output 2](../Images/a40dc0093f7416d5417b90a452336f52.png)

<noscript><img class="alignnone size-full wp-image-446438" src="../Images/a40dc0093f7416d5417b90a452336f52.png" alt="Architectural design tools output 2" width="1029" height="584" srcset="https://cdn.educba.com/academy/wp-content/uploads/2021/01/Architectural-design-tools-output-2.jpg 1029w, https://cdn.educba.com/academy/wp-content/uploads/2021/01/Architectural-design-tools-output-2-300x170.jpg 300w, https://cdn.educba.com/academy/wp-content/uploads/2021/01/Architectural-design-tools-output-2-1024x581.jpg 1024w, https://cdn.educba.com/academy/wp-content/uploads/2021/01/Architectural-design-tools-output-2-768x436.jpg 768w" sizes="(max-width: 1029px) 100vw, 1029px" data-original-src="https://cdn.educba.com/academy/wp-content/uploads/2021/01/Architectural-design-tools-output-2.jpg"/></noscript>

AutoCAD 是计算机辅助设计和绘图应用程序。与 Revit 类似，建筑师和项目经理以及城市规划者也使用它。

AutoCAD 与 Revit 的主要区别在于，它用于创建精确的 2D 和三维草图，而 Revit 适用于大规模应用。AutoCAD 也有其变种，专门从事各种学科，如电气，测绘，结构细节。

AutoCAD 也有移动和网络解决方案，可以随时随地工作。还有一个学生版，对学生和教育工作者都是免费的。

AutoCAD Architecture 是一个专用于建筑目的的应用程序，用于绘制 3D 对象，包括门、墙和窗，并且许多智能数据也与之相关联。因此，通过创建 2D 图纸，我们可以创建立面图和剖面图。同样，civil design、civil design 3D 和 pro 版本支持有助于工程计算和表达的数据对象。端口是为 windows 和 mac 版本创建的，具有完整的功能。

#### 3.ArchiCAD 系统

![Architectural design tools output 3](../Images/4fae65c33dd20f4be75d67528494e0a2.png)

<noscript><img class="alignnone size-full wp-image-446439" src="../Images/4fae65c33dd20f4be75d67528494e0a2.png" alt="Architectural design tools output 3" width="698" height="460" srcset="https://cdn.educba.com/academy/wp-content/uploads/2021/01/Architectural-design-tools-output-3.png 698w, https://cdn.educba.com/academy/wp-content/uploads/2021/01/Architectural-design-tools-output-3-300x198.png 300w" sizes="(max-width: 698px) 100vw, 698px" data-original-src="https://cdn.educba.com/academy/wp-content/uploads/2021/01/Architectural-design-tools-output-3.png"/></noscript>

ArchiCAD 是由 Graphisoft 开发的软件，提供 BIM 和 CAD 的功能。ArchiCAD 的主要特点是它与参数化对象或智能对象一起工作，这些对象是数据增强的对象，因为它用于创建更引人注目的虚拟建筑，如墙、板、门、窗、屋顶和家具等结构。内置了许多预先设计和定制的对象。

协作是通过远程访问完成的。还使用 API(应用程序编程接口)和 ODBC 连接为第三方开发人员开发的附加组件提供 API 和脚本。

与 AutoCAD 等其他商用软件的数据交换选项。除了设计之外，还有可视化和渲染工具，可以创建照片级的图片和文档管理。他们有自己的服务器，这为备份和恢复创建了版本控制，以防出现故障。

#### 4.BricsCAD BIM

![Architectural design tools output 4](../Images/ff4d189c8744ab3b79cd4d67307dd054.png)

<noscript><img class="alignnone size-full wp-image-446440" src="../Images/ff4d189c8744ab3b79cd4d67307dd054.png" alt="Architectural design tools output 4" width="1599" height="899" srcset="https://cdn.educba.com/academy/wp-content/uploads/2021/01/Architectural-design-tools-output-4.jpg 1599w, https://cdn.educba.com/academy/wp-content/uploads/2021/01/Architectural-design-tools-output-4-300x169.jpg 300w, https://cdn.educba.com/academy/wp-content/uploads/2021/01/Architectural-design-tools-output-4-1024x576.jpg 1024w, https://cdn.educba.com/academy/wp-content/uploads/2021/01/Architectural-design-tools-output-4-768x432.jpg 768w, https://cdn.educba.com/academy/wp-content/uploads/2021/01/Architectural-design-tools-output-4-1536x864.jpg 1536w, https://cdn.educba.com/academy/wp-content/uploads/2021/01/Architectural-design-tools-output-4-1280x720.jpg 1280w" sizes="(max-width: 1599px) 100vw, 1599px" data-original-src="https://cdn.educba.com/academy/wp-content/uploads/2021/01/Architectural-design-tools-output-4.jpg"/></noscript>

BricsCAD 是另一种 CAD，它被 AEC 行业以及汽车和制造行业使用。它还拥有人工智能工具，可以增加工作流程并节省时间，从而增强 2D 和 3D 模型。BricsCAD 形状用于建模 3D 资产，这很容易访问。

你还可以访问免费的 3D 组件库，比如树、人类交通工具。然后，BricsCAD 的 BIM 对应部分使用人工智能来对模型进行 BIM 化。

BricsCAD Mechanical 用于创建机器零件和组件，并生成各种视图(等轴测视图、正交视图、截面视图)，而物料清单则根据设计变更进行视图更新。

#### 5.FreeCAD

![Architectural design tools output 5](../Images/0a2f30970bed8e744d45bb6b2bd2008f.png)

<noscript><img class="alignnone size-full wp-image-446441" src="../Images/0a2f30970bed8e744d45bb6b2bd2008f.png" alt="Architectural design tools output 5" width="698" height="393" srcset="https://cdn.educba.com/academy/wp-content/uploads/2021/01/Architectural-design-tools-output-5.png 698w, https://cdn.educba.com/academy/wp-content/uploads/2021/01/Architectural-design-tools-output-5-300x169.png 300w" sizes="(max-width: 698px) 100vw, 698px" data-original-src="https://cdn.educba.com/academy/wp-content/uploads/2021/01/Architectural-design-tools-output-5.png"/></noscript>

FreeCAD 是用于建模和 BIM 的开源软件。虽然它主要集中在机械工程设计，它也可以应用于建筑和电气。

由于本质上是开源的，FreeCAD 的功能可以使用 Python 编程语言进行扩展。唯一的问题是不支持像 DWG 这样的专有文件格式。它包括许多类似于用于 BIM 的 CATIA、Inventor、Revit 的功能和工具。

![output 7](../Images/b6867520ac45418d10a456b5213d0c13.png)
