## Inkscape 图层简介

在使用该软件的过程中，Inkscape 层对于生成一个用户友好的环境起着非常重要的作用，换句话说，您可以完全控制您的插图对象，并根据您的要求安排它们的序列结构。你可以把它理解为面包片，如果你把某样东西放在最上面的面包片上，你首先会看到它，就像在 Inkscape 中一样，我们使用层来使不同层上的对象具有不同的属性，这样我们就可以管理它们，而不会干扰我们插图中的其他对象。

### 如何创建图层并在 Inkscape 中使用它们？

默认情况下，Inkspcae 软件的用户界面左侧会有图层面板。

<small>3D 动画、建模、仿真、游戏开发&其他</small>

![Inkscape layers output 1](img/e01d9f78a00e42572570e20a41e3959f.png)



让我们结束这一切。

![Inkscape layers output 2](img/554ceb4f28bb2f5950b3d19d991e1d7e.png)



现在，您将从该列表中再次拥有该面板。你可以通过点击箭头键打开这个列表。

![Inkscape layers output 3](img/ec39933d61e59c902f52de56830203f7.png)



还有一个选项，通过它你可以打开图层面板，为此你必须去菜单栏的图层菜单。在向下滚动列表中，我们有层选项，点击它，你可以有层面板。

![Inkscape layers output 4](img/0bb82ce709996947a0c482365fb232d7.png)



现在无论你创建什么，它都将在图层上创建，现在我们在图层面板中有一个图层，所以我的创建圆将在这个图层上。

![Inkscape layers output 5](img/47546af88f584b025589231cea63c134.png)



我们有眼睛按钮层面板中的层。

![Inkscape layers output 6](img/3ffb52f62891c3cd3a3a9806b2da5581.png)



通过禁用它，我们可以隐藏该层的对象。

![Inkscape layers output 7](img/311e9e1a25aee6846a352ffcaae20e5f.png)



在眼睛按钮旁边，图层上有一个锁图标。

![Inkscape layers output 8](img/1162bd059ed9493be7229d37bc6b236b.png)



如果我点击它，那么它将锁定选定层的对象，现在我们不能在锁定层的对象中进行任何类型的编辑。

![Inkscape layers output 9](img/52df031d4e47351c40e91f9ec56fec38.png)



你可以根据自己的需要重命名图层，只需点击图层名称并输入你想要的图层名称。

![Inkscape layers output 10](img/1ac137c789302c9a082683edf5e0e38c.png)



“图层”面板中可以有许多图层。让我们在图层面板中创建一个新图层，然后点击这个面板的“创建新图层”按钮。

![Inkscape layers output 11](img/05678d46c03ea2a22fcc5f9532eaa454.png)



当你点击这个按钮时，一个对话框将会打开。在这里，您可以根据自己的选择再次命名该层。它将采取的名称在序列中已经有层，例如我有一个层的名称为“第一层”在图层面板中，现在你可以看到它建议我下一层的名称为第一层 1。

![Inkscape layers output 12](img/28757d23bdeafacdd68582f4818c399d.png)



我将命名为第二层。这里，我们在框中有位置选项，这也意味着您可以添加新层到当前层的上面或下面，或者您也可以将其添加为选定层的子层。

![Inkscape layers output 13](img/11ae64165d2171b5ad2f7a346b4a76f7.png)



点击这个对话框的添加按钮后，你将在图层面板中有一个新的图层。

![Inkscape layers output 14](img/7bc488f7d5466a838fc81f0c3f483908.png)



你可以通过用鼠标移动任何层来上下移动该层。

![Inkscape layers output 15](img/294338b5f45902c309862986ff9e52cc.png)



“将当前图层提升到顶部”选项将帮助您将所选图层放置在图层面板的所有图层的顶部。

![Inkscape layers output 16.1](img/f88d67580207fdbefb49edcb82e0a0e3.png)



![Inkscape layers output 16.2](img/38ea5a7124e3c72b6358937ef62200b9.png)



如果你想在层列表中上移一层，你可以选择“提升当前层”选项。

![output 17.1](img/384319f24baaa657bb2cb5d38a5a8e24.png)



![output 17.2](img/92dbf30e027c00fafaaca0efc86752b3.png)



与上面的选项相同，我们有两个选项用于列表中的下一层。为此，您可以尝试这些选项。

![output 18.1](img/c690c50153b3e8a7b847b22afc666182.png)



![output 18.2](img/bd44e6ea26f08dcbac2bb5e7cde91934.png)



您在工作过程中创建的任何对象都将位于您选择的层上。例如，我将选择图层面板中的第二层，并使用工具面板中的矩形工具绘制一个矩形。

![output 19](img/1c855a5d28c11a4cfee60a84de69b774.png)



你可以看到，如果我隐藏第二层，那么只有矩形我们被隐藏，因为它是在这一层上创建的。所以在创建你的对象时，确保你在正确的图层上。有一点我也会用这个例子来解释，你可以理解图层面板的重要性，当我隐藏第二层的对象时，它不会干扰其他层的对象。因此，由于图层面板及其功能，您可以分别处理不同的对象。

![output 20](img/bf3fb42d0622c06d70f3618706c6d98e.png)



该对象将位于文档区域中其他对象的下方，该层位于其他层的下方。这里我有第二层低于第一层，所以第二层的矩形低于第一层的圆形。如果你想让它在上面画圈，你可以在图层面板里移动它的图层。

![output 21](img/11e5019193c73e9fe86d8fb72729f629.png)



现在，当我选择圆形时，我会自动移动到图层面板的第一层。我可以从图层面板的不透明度选项中减少任何图层的不透明度。不透明度可以定义为任何对象的不透明度。

![output 22](img/c13162e5e90f9fcf66c06cde42e3dc1f.png)



使用此面板的模糊选项，您还可以使图层中的任何对象变模糊。

![Inkscape layers output 23](img/74de11f61ceec09c978ad1e1300f7c7b.png)



有一个混合模式选项，通过它你可以改变选中层的对象。例如，我选择了圆形，然后将其模式改为屏幕，你可以看到，由于两个对象的颜色混合，我们有其他颜色。

![output 24](img/6f2998614ceab9471d909bc74da5d787.png)



我们经常面对的一件事是，我们错误地在错误的图层上创建了对象，Inkscape 对此也有解决方案。这个蓝色的矩形在第一层，我想把它移到第二层。所以我会选择它。

![output 25](img/accca1e4f0b5afd5c7fa69196eaa50aa.png)



因此，去层菜单，并点击'移动选择层以下'的选项。

![output 26](img/e632c1becdefa0718c9b9c89630ebc7a.png)



现在它在第二层。

![output 27](img/f78169492fb90fe93b3f125175a0b049.png)



你可以记住“切换到层”选项的快捷键，这样你可以快速地从一层跳到另一层，或者你可以浏览这个菜单的这些选项。

![output 28](img/4cf88ea086a986b1edbda224a8b02cc6.png)



通过“复制当前图层”选项，您可以在图层面板中复制选定的图层。

![output 29](img/dca2f2e52a84de74356777f85c8efb00.png)



![output 29.2](img/3988ee700c96e54052f8e74c0b7bf5c2.png)
