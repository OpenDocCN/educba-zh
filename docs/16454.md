# InDesign 对齐

### InDesign Align 简介

InDesign Align 是该软件非常有用且必要的功能，可帮助我们正确对齐设计布局内容的元素。我们在这个软件中进行两种类型的对齐，第一种是对齐设计元素，用于创建效果布局设计，第二种是对齐文本内容，用于管理空间以及以良好的方式排列文本。InDesign 为我们提供了一整套“使用一些可调参数对齐”选项，还提供了一个“文本框架”选项，允许我们将文本与文本框架对齐。所以今天，在这篇文章中，我将通过一个例子告诉你这两种方式。

### 如何在 InDesign 中使用 Align？

在 Indesign 中对齐是一项非常有用的功能，因为它可以在排列元素时节省大量正确对齐元素的时间。这也有助于我们在元素之间保持相等的距离。让我告诉你 InDesing 的 align 特性的所有重要方面和要点。

<small>3D 动画、建模、仿真、游戏开发&其他</small>

让我们先创建一个新文档，为此，我将从“新建文档”对话框中选择“信纸大小”页面。

![InDesign align output 1](img/856ea43609095c742b5d43a88957b2bf.png)



我创建了这些形状作为这个主题的例子。我在工具面板的矩形工具的帮助下绘制了这些形状，并将这些矩形的一个角变成圆角。如果你想知道如何在 InDesign 中圆角，可以去我的文章《InDesign 圆角》。我还使用工具面板的文字工具在这里键入了一些文本。

![InDesign align output 2](img/8bbad63deeb6a8adc39d938311ff0623.png)



现在我将打开对齐面板来解释对齐。如果你没有在你的用户屏幕上对齐面板，你可以去菜单栏的窗口菜单，在这里的下拉列表中，去对象和布局选项。在这个选项的新的下拉列表中，我们有对齐选项。通过点击它，我们可以在我们的工作屏幕上启用它的面板，或者我们可以简单地按下 Shift + F7 作为快捷键。

![InDesign align output 3](img/cf3d23e08ccb351c5fdd542331684c3b.png)



这是对齐面板，其中有许多参数。所以让我们逐一讨论。

![InDesign align output 4](img/273629eac1b87059824e9deed084e5a2.png)



您必须选择那些要对齐的元素。例如，我想对齐这些矩形，所以我将选择它们，然后单击对齐面板的“对齐左边缘”选项。它将对齐所有元素的左边缘。

![InDesign align output 5](img/8e18065f246e3aae72e2572a977c65fc.png)



下一个选项是“水平居中对齐”，这意味着它将元素的水平中心对齐，或者你可以说它将元素从它们的中心对齐。

![InDesign align output 6](img/d82190387b45d6ddf0878666929e304f.png)



与左边缘对齐相同，我们可以通过单击对齐面板的此选项来对齐元素的右边缘。

![InDesign align output 7](img/9af230b664c139bf45ecae70cc4e58a2.png)



如果我们单击“对齐上边缘”选项，那么它将对齐所有元素的上边缘。这里，顶层元素对您不可见，因为它位于底层。

![InDesign align output 8](img/1880afb1497f987f4298b7ea2ab2ce24.png)



让我把这三个形状的不透明度降低一点。

![InDesign align output 9](img/9d4284ad6def2b2236279e7537a8b60b.png)



再次对齐它们，你可以看到所有的元素从顶部边缘对齐。

![InDesign align output 10](img/f21d28d83af46a673658e26a5808c119.png)



同样，我们可以通过下一个选项从中心对齐。

![InDesign align output 11](img/53b39cdd9609123f8f3f3f0e1e7f3eda.png)



或者从下一个居中选项开始从底部对齐。

![InDesign align output 12](img/d14ab07a996faf79335465ec2cce9224.png)



如果您愿意，可以将任何元素指定为关键元素，这意味着所有其他对象都将根据所选的关键元素对齐。例如，首先，选择所有要对齐的元素，然后单击要成为关键元素的元素。单击后，关键元素周围会出现一个蓝色的矩形选择。

![InDesign align output 13](img/5dd3d686c3d46bfc4fb74f3dda23d2af.png)



现在，我将选择第一个对齐选项，它会将所有其他元素与所选关键元素的左边缘对齐。

![InDesign align output 14](img/03587246a8a904094729310108f1568d.png)



现在我将这些元素放置在不同的距离。

![InDesign align output 15](img/c85054c0d6da573de7d5c4501881f47a.png)



选择它们后，我将点击“分布上边缘”选项。

![InDesign align output 16](img/4677497fb0d607b1f9cef6ff39e74189.png)



并且它会以相等的距离对齐所有元素的上边缘。如果我们选择“分布垂直中心”，那么它会以相等的距离对齐元素的中心。

![InDesign align output 17](img/c8f30f6e808c560dd09f173d2ea25ef1.png)



与“分布下边缘”相同，以相等的距离对齐元素的下边缘。

![InDesign align output 18](img/7fc6ea9c6ac5b63f3f766afe4c6c5ae4.png)



现在我将横向排列这些元素。

![InDesign align output 19](img/d8838dc277cc651fead6a5cd341e5972.png)



所以在选择之后，我将点击“分配左边缘”选项。

![InDesign align output 20](img/59c2d8bf7cfa558d14c8dbd241b63f3d.png)



并且它将在水平方向等距离分布左边缘。所以“分布水平中心”和“分布垂直中心”的作用是一样的。

![InDesign align output 21](img/ad1da4363aed0b81017b0d51c76d3d6c.png)



“分布右边缘”将使右边缘水平等距对齐。

![InDesign align output 22](img/c7869fa1fa39339e3415c51287349c63.png)



您可以为分布距离指定任何值。为此，通过点击复选框启用“使用间距”选项，并输入您想要的值。这一次，我将输入 1 英寸作为它的值。

![InDesign align output 23](img/a79c6dbe6e2d15f6e7773aaad423204e.png)



现在，如果我单击任何分布对齐选项。它会将它们以 1 英寸的距离对齐。

![InDesign align output 24](img/1c2a7b6757bb1c7bde882ae677f42a3f.png)



在下面的间距选项中，我们有'对齐'选项，通过它我们可以选择根据哪个我们想要对齐其他对象。

![InDesign align output 25](img/7d80bc5e4695b9c0fc992d368771878c.png)



现在，我将选择这个绿色的文本，并使这个形状成为一个关键元素。

![output 26](img/6d156779089819d2280e6e42d551c3b1.png)



然后将此文本在此形状的中心对齐。之后，我将点击分布垂直空间选项。

![output 27](img/b47148de0a5e66e0f2376f7c08ad79d9.png)



它会在这个形状的某个距离处垂直对齐这个文本。

![output 28](img/2f08eafb6600d61fabd001d9ae5f6e47.png)



如果我在这里输入一些间距值。

![output 29](img/c4591128733429dad6fc73ba5a5b6775.png)



然后，您可以看到，如果我们重复同样的选项，那么它将在一定距离内与我们的文本正确对齐。

![output 30](img/f57373bec2e7d0209d551e513c302c74.png)



我们有一个水平对齐的水平间距选项。

![output 31.1](img/992c68792175b8a77d7f28f391eb303b.png)



![output 31.2](img/befe62f939a96e007cf48978067e2153.png)



您可以看到文本没有完全与这个粉红色形状的中心对齐。原因是文本没有与文本框架对齐。

![output 32](img/445ab6f4e7e51414a190e208dc3d5087.png)



为此，请从“对象”菜单中选择“文本框架选项”。

![output 33](img/70e47a30d3745f7e0065a1cde3b9783a.png)



并从该面板的“对齐”选项中选择“居中”选项。

![output 34](img/190c3720b1c56f6350651ad72f325a7d.png)



确保预览已打开。

![output 35](img/88247f1fe0979df9a72909b7b8049cdb.png)



这样，我们可以很好地对齐。

![output 36](img/c8665c51826d613db5cd897df03e49a0.png)
