## C++面试问答介绍

C++被称为中级语言，意味着高级和低级编程语言功能的结合。它是由比雅尼·斯特劳斯特鲁普在 1979 年发明的。它也被称为面向对象的编程语言。它是编译的、区分大小写的、静态类型的。

让我们看看面试中被问到的 2022 C++面试问题。这些问题分为以下两部分:

### 第 1 部分–c++面试问题(基础)

这第一部分涵盖了基本的 C++面试问题和答案。

#### 1.定义阶级？

**答案:**
类一般指定数据的结构。它可以被定义为描述受支持类型的状态的蓝图。例如，学生班级中学生的详细信息，如学生的年龄、姓名、地址等。，在一个块中，这是类。

#### 2.定义对象？

**答案:**
这是面试中被问到的基本 C++面试问题。对象是类的实例，它可以有状态和行为。一旦在类中创建了一个对象，您就可以在对象的帮助下轻松地访问该类的已定义成员。

#### 3.解释面向对象编程(OOP)的特征或概念？

**答案:**
OOP 的特点或概念如下:

*   抽象
*   包装
*   遗产
*   多态性

**抽象:**用于隐藏内部实现，只向用户显示所需的细节。抽象可以在抽象类或接口的帮助下实现，例如计算器。在计算器中，无论给定什么输入，它只显示输出。对于用户来说，如何进行计算不是必需的细节。

**封装:**简称数据隐藏。它是一种将数据和操作绑定在一起并对用户隐藏细节的机制。封装可以通过访问说明符来实现，这些说明符在 C++中是公共的、私有的和受保护的。在他们的帮助下，我们可以直接向用户提供访问权限或阻止访问权限。

**继承:**它指的是从现有类创建新类的过程。意思是由子类继承父类的属性。父类也称为基类，子类也称为派生类。继承主要用于代码重用和扩展父类。

**多态性:**指多种形态。它主要描述当一个成员函数被调用时；它将根据调用函数的对象类型执行不同的函数。

#### 4.C++中的访问说明符是什么？

**答案:**
在面向对象语言中，访问说明符被称为关键字，用于设置类、函数、方法和其他成员的可访问性。C++只有三个访问说明符:1 .公共 2。二等兵 3。保护

*   **Public:** 声明为 Public 的成员或字段也可以在类外部访问。
*   **Private:** 声明为私有的成员或字段不能在类外访问。它们只能在类内部访问。
*   **Protected:** 声明为 Protected 的成员或字段可以在类外部访问，但只能在从该类派生的类中访问。

#### 5.抽象类和接口有什么区别？

**答案:**
两者的区别如下:

| **序列号** | **抽象类** | **界面** |
| One | 它可以有实例方法，也可以有实现。 | 它是隐式抽象的，不能有任何状态或实现。 |
| Two | 它可以扩展其他类和多个接口。 | 它只能扩展接口。 |
| Three | 它不能被实例化。 | 它是绝对抽象的，不能实例化。 |
| Four | 成员可以声明为公共的、私有的或受保护的。 | 默认情况下，成员是公共的。 |
| Five | [抽象类](https://www.educba.com/abstract-class-in-c-plus-plus/)简称 fast。 | 它比较慢。 |
| Six | 变量或字段可以声明为非最终的 | 变量或字段是最终的 |

### 第 2 部分–c++面试问题(高级)

现在让我们来看看高级 C++面试问题。

#### 6.什么是继承并解释它的类型？

**答案:**
传承前面解释过；请检查问题 2。有不同类型的继承:

*   **单一继承:**指只有一个父类和一个子类时的继承。
*   **多重继承:**指派生类继承两个或多个类的属性时的继承。
*   **多级继承:**是指当派生类是另一个类的基类时的继承。
*   **层次继承:**指多个派生类继承同一个基类的属性时的继承。
*   **混合遗传:**指虚拟遗传。它是多级继承和层次继承的结合。

#### 7.C++中的数据类型和变量有哪些？

**答案:**
数据类型是用来定义变量的类型，相应的，OS 分配内存空间和可以存储什么。声明了将被赋值的变量，并且在声明时，它保留内存空间来保存这些值。例如，在本例中，int X 是数据类型，X 是变量。一些数据类型的例子是 int，char，float，long，bool，double，void。

#### 8.C++中的构造函数和析构函数是什么？

**答案:**
在创建类的新对象时，构造函数已经被执行。它将具有与类名完全相同的名称。构造函数可以是默认的构造函数，这是不需要提及的，它没有任何参数。构造函数可以是我们需要在类中提到或声明的参数化构造函数，参数需要初始化。当一个类的对象不在作用域内或者不再被使用时，析构函数被执行。它将具有与类名完全相同的名称，但应该以(~)符号为前缀。它主要用于释放资源，如关闭文件等。

#### 9.C++中有哪些虚函数？

**答案:**
这是高级 C++面试中在面试中被问到的问题。它主要用于替换父类的实现。这些是在类中用关键字“virtual”声明的函数。当用子类类型的对象初始化父类类型引用，并且使用父类引用调用重写方法(声明为虚拟)时，将调用子类的方法。

**答案:**

| **序列号** | **C** | **C++** |
