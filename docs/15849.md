## 网页开发简介面试问答

Web 开发广义上是指与开发通过互联网或内部网托管的网页相关的任务。网站开发过程包括网站设计、网站内容开发、客户端/服务器端脚本和网络安全配置。

Web 开发层次结构如下。

<small>网页开发、编程语言、软件测试&其他</small>

*   客户端脚本
*   服务器端脚本
*   数据库技术

### 客户机-服务器体系结构

**客户端**在客户端/服务器模式类型的系统中，是服务的接收端或服务的请求者。该客户端通常位于可以通过网络访问的另一个系统或计算机上。

**服务器**是专用于管理网络资源的计算机、设备或程序。服务器通常被称为专用服务器，因为除了服务器任务之外，它们几乎不执行任何其他任务。

客户机/服务器体系结构是一种计算模型，在这种模型中，服务器托管、交付和管理客户机使用的大部分资源和服务。这种类型的体系结构有一台或多台通过网络或互联网连接到中央服务器的客户端计算机。这个系统共享计算资源。

客户端-服务器架构也称为网络计算模型或客户端/服务器网络，因为所有的请求和服务都是通过网络交付的。

### 网站架构

一个**网站**是共享一个域名的可公开访问的、相互链接的网页的集合。网站可以由个人、团体、企业或组织创建和维护，以服务于各种目的。

网站架构是在设计、开发和部署之前，对网站的技术、功能和视觉组件进行规划和设计。它是网站设计者和开发者用来设计和开发网站的一种手段。

### 网站的 Web 开发类型

**静态网站**

也称为平面网站或固定网站。它们被加载到客户端的浏览器上，就像它们被存储在 web 服务器上一样。用户只能使用信息，但不能对信息进行任何修改或与之交互。静态网站仅使用 HTML 创建，并且仅在不再需要修改信息时创建。

**动态网站**

动态网站在不同的时间点显示不同的信息。可以在不加载整个网页的情况下改变网页的一部分。AJAX 技术的使用使得[成为可能。](https://www.educba.com/ajax-interview-questions/)

网页剖析

网站由以下组件组成

*   **包含块**

容器主要是页面的主体标签，也是所有的 div 标签。

*   **标志**

徽标定义了网站的身份。

*   **导航**

网站的导航系统应该容易找到和使用。

*   **内容**

网站上的内容应该与网站的目的相关。

*   **页脚**

通常包含版权、合同和法律信息以及网站主要部分的链接。

*   **空格**

也称为负空间是指没有被类型或插图覆盖的区域。

对于一个成功的 web 开发人员来说，应该具备以下技能

*   彻底理解客户端和服务器端脚本。
*   测试跨浏览器的不一致性。
*   传导观察用户测试，
*   用 JavaScript，PHP，JQuery 进行编程交互。

一旦网站被设计、构建并测试成功，它就应该被发布以使其在网络上可用。为了发布您的网站，您需要以下先决条件。

*   **网页开发软件:**用于构建你网站的网页。例如 WordPress、Dreamweaver。
*   **互联网连接:**需要访问远程网络服务器。
*   **网络服务器:**你的网站所在的实际位置。

**答案:**

**回答:**
这是面试中被问到的基本的 Web 开发面试问题。CSS3 转场有助于创建简单快速的动画效果。它们不仅给我们改变财产价值的控制权，而且让它在给定的期限内缓慢进行。

#### 5.CSS3 中的伪类是什么？

**答案:**
伪类是一种 CSS 技术，用来设置元素改变状态时的样式。

例如:

*   编辑鼠标悬停事件时的样式。
*   当元素获得焦点时设置样式。
*   对已访问/未访问的链接应用不同的样式。
