#### 1.Adobe Audition

Adobe Audition CC 是一个强大的音频编辑器，符合行业标准，并有一个定价原型作为补充。该应用程序每年都会更新新的属性，并扩展与 Windows 和 macOS 新版本的相似性。Audition 可以作为一个单声道声音编辑器，也可以作为一个多声道混音控制台来录制和编排声音。该软件可以作为一个完整的 DAW 来使用，可以同时记录多个信号源，也可以使用外部插件(AU、VST 和 VST3)。Adobe 还包括最好的音频修复工具，允许您在录音中捕捉噪音样本，然后从整个文件中消除不需要的声音范围。您还可以利用灵活的降噪和斑点修复来消除咔哒声和爆裂声。

#### 2.逻辑 **Pro X**

Logic Pro X 包含一些非常好的功能，通常将其定义为初学者和经验丰富的消费者的惊人的音频编辑软件。它带来了您在音频编辑器中需要的所有基本属性，并附带了高端功能，包括能够自动利用智能速度来补充项目中的各种轨道。该软件还附带 flex-time，允许您逐个操纵波形中音符的时间度量，而无需将其从片段本身中移除。它也有 flex pitch，可以为单拍执行类似的事情，但音高是定时的替代选择。它还有一个自动琶音器，可以自动将和弦转换为琶音，让你的音乐有更复杂的感觉。许多预先录制的音频是可用的。您还可以自由地利用系统中的补丁，以及 Logic Pro X 附带的大量 Apple 插件。

#### 3.大胆

Audacity 是一款非常优秀的在线免费声音编辑器。它为消费者提供了一系列完整的母带制作和编辑工具，包括有害波形和不止一个音轨的编辑。因为它是免费的，所以对于刚开始音频编辑的人来说，它是最容易获得的应用程序。它有许多效果，包括高音、低音、噪音消除、失真等。它还带来了分析工具，如静音查找器，节拍查找器，声音查找器等。Audacity 几乎支持所有的音频文件格式，如 WAV、MP3、PCM、AIFF、OGG Vorbis、AC3、FLAC、WMA、AMR、MP4 等。

#### 4.收割机

Reaper 是一个音频编辑应用程序，非常强大，功能丰富，相比之下，比其他一些可用的替代产品便宜。Reaper 为多个音轨提供了辅助功能，并且在一个音轨中拥有 64 个声道的出色的多声道辅助功能。它有可能直接将声音录制到立体声、单声道甚至是多声道的音频文件中，如果你愿意的话，还可以同时录制到多个磁盘上进行数据存储。Reaper 允许你以一种实时的、无害的方式注册效果，几乎可以放入任何 MIDI 或第三方插件，甚至可以将实时时间延长和音高移动能力融入混音中。

#### 5.Ableton **现场直播**

Ableton Live 以电子音乐制作和音乐家而闻名。Ableton Live 配有无限的 MIDI 音轨和音频，因此您可以根据自己的要求将众多音轨叠加到作品中。它还有 256 个单声道输入、MIDI 采集和输出通道。此外，您还可以在项目中使用 70 GB 的内建声音资源库，其中包括 15 种应用乐器和大约 46 种声音效果。音高修正不可用，附加效果如淡入淡出也不像其他软件如 Pro Tools 那么简单。
