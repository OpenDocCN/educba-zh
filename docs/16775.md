## Krita 插件简介

Krita 插件是额外的软件或程序，旨在提高准确性以及 Krita 的工作能力。有一些免费的插件和付费的插件，你可以根据你的工作需要在这个软件中同时拥有免费和付费的插件。当你下载插件安装到你的系统中时，你会有一个插件的 zip 文件，通过解压，你可以把它安装到 Krita 的文件夹中。今天插件的安装不是我们主要关心的；事实上，我会告诉你一些最新的插件，以便你可以在使用这个软件时使用它们。

### Krita 插件列表

在 Krita 最新的插件列表中，我们可以包含颜料。o、Buli Notes、Krita 参考插件、space otter code/Krita oot 等等。因此，让我们以一种非常激动人心的方式逐一讨论它们。

#### 1.颜料。英语字母表中第十五个字母

我们有许多好的特性使得这个插件最受 Krita 的支持，这些特性是:

*   定义:这个插件拥有颜色选择器和混合器工具的专长，并提供了许多工具来实现这个好的结果。
*   **颜色管理:**可以显示你当前的颜色，也可以显示你之前的颜色。你也可以用这个插件锁定亮度。
*   **和谐色彩管理:**和谐色彩显示在许多类别上，如活动色彩和 ARD、HSL、HSV、色调面板。
*   **和声规则:**有单色、互补、类比、双互补、分裂互补双等和声规则。
*   **多个面板:**可以显示多个面板，如 FGC 面板、RGB 面板、ARD 面板、HSV 面板、HSL 面板、色相面板、YUV 面板、Gam 面板、点面板、Obj 面板。所有这些面板用于不同的目的，例如使用 FGC 面板来显示色温和协调设置。
*   **十六进制编码:**提供独立颜色的十六进制颜色编码，让您轻松识别颜色；不仅如此，您还可以锁定 CMYK 通道以稳定混合颜色。
*   可调整和可扩展 UI: 它提供的特性对于 UI 来说是可调整和可扩展的，为 UI 提供了一个流畅的工作环境。
*   **设置/快速访问选项:**不同参数的设置可以在其中轻松完成，并且具有快速选项功能，可以节省您在使用该软件时的时间。

#### 2.布利笔记

Buli Notes 还有一些功能和特性，可以在 Krita 文件中添加注释，这些注释将以 Krita 的本地文件格式保存。

*   **定义:**可以认为是 Krita 的一个笔记管理器插件，可以管理针对不同类型组件的笔记，它会嵌入。kra 文件。
*   **Notes Docker:** 该插件在。kra 文件将通过 docker 提供，在 docker 中，您将拥有一个已定义注释的列表、添加/编辑/删除注释选项、锁定/解锁注释选项、固定/取消固定注释等等。
*   **三种类型的注释排列:**它提供了三种类型的注释:标题注释、描述注释(您可以在此注释中描述任何内容)注释和颜色注释。
*   **自动设置日期/时间:**有一个好东西是针对上次修改和创建的，它在 notes 上自动设置日期/时间。
*   **文本编辑器:**使用文本编辑器，您可以选择字体系列，设置字体大小，文本粗细，文本样式，文本对齐，还可以添加项目符号。
*   手写笔记:它还为您提供了在便笺簿上设置手写笔记的便利。

#### 3.Krita 参考插件

你也将在这个插件中拥有一些特殊的功能，如下所示:

*   **定义:**与 Krita 4.0.0 兼容，用于在该版本 Krita 中快速替换移除的参考图像 docker。有一点你必须记住，这只是暂时的删除。
*   **参考图像工具:**在 Krita 4.1 中，参考图像 Docker 已经被替换为参考图像工具。
*   **放大/缩小功能:**如果你想放大或缩小，那么你可以按住键盘的 Ctrl 键滚动鼠标滚轮。
*   使用这个插件，我们可以从引用中选择一种颜色，为此，我们只需点击引用中我们想要的部分，它的预览就会显示在 Docker 的顶部。

#### 4.space otter code/krita oot

Kritatoot 是 Krita 软件最好的插件之一，具有许多高级功能，包括:

*   **定义:**这个插件用于将我们的图像从 Krita 内部发布到 Mastodon，它已经准备好使用这个插件了，因为它不需要外部依赖。它有麻省理工学院的执照。
*   **将文档直接发布到 Mastodon:**Krita 4 . x 允许您将当前文档的副本直接发布到 Mastodon 上。
*   **直接导出并提交文档副本:**它始终为您提供轻松导出并提交当前文档副本的功能。如果当前文档从未保存过，导出的图像将默认为 PNG 格式。
*   **隐私和消息设置:**有一个指定消息的安排，一个你的 toot 的隐私设置，如公开、未列出、直接和仅关注者。
*   限制:这个插件也有一些限制，你可以每只脚贴一张图片，如果你想指定一个焦点，它不会实现。没有仅用于替代文本的输入。
*   **特性依赖于乳齿象服务器:**关于它有一点很重要，就是乳齿象服务器设置了文件的最大大小和类型。
