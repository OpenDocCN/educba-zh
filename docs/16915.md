## SketchUp Kitchen 简介

SketchUp 厨房是建筑设计的一部分，这意味着我们需要为任何建筑设计的 3d 模型设计厨房的结构。厨房设计包括如何在任何建筑单元的平面布局中为厨房分配面积，以及你在厨房中添加了哪些东西来赋予它模块化的外观。你可以在厨房设计中加入很多东西，比如平板的形状、冰箱的面积、抽屉的设计和摆放等等。您可以从 sketchup 的 3D 仓库中下载不同的组件，作为厨房的附件，如灶台模型、水槽模型和其他您希望在设计的厨房中使用的对象。

### 如何在 Sketchup 中创建厨房设计？

为了开始创建厨房，我将从网上导入一张厨房的布局图，然后进入文件菜单，选择导入选项。

<small>3D 动画、建模、仿真、游戏开发&其他</small>

![SketchUp kitchen output 1](img/06b9368ed7738028cd93c90b483f6cd4.png)



现在，我将从保存的位置选择这个厨房布局，并确保在此框中选择图像选项，因为我们要将此布局作为图像导入。

![SketchUp kitchen output 2](img/bf1638b96f73d829d2e8ce734be78ba9.png)



现在它会像这样导入。我将点击放置它。

![SketchUp kitchen output 3](img/cb7d2caf3e17c871c851a27231a4448e.png)



现在，让我们首先改变到俯视图，并采取卷尺测量工具。

![SketchUp kitchen output 4](img/50e231eab772888de5cc904960f41694.png)



现在测量布局的任何尺寸，检查它是否是实际尺寸，或者我们必须按比例缩放。在这里你可以看到它不是实际的比例，所以首先用卷尺点击尺寸的起点，然后点击尺寸的终点。现在，在测量框中输入该尺寸的准确值，即 2.3 米，然后点击键盘上的 enter 按钮。

![SketchUp kitchen output 5](img/5a4001a8ed2fcf8dade2f6afb34380d8.png)



一旦你点击输入按钮，它会问你是否要缩放它，所以点击这个弹出框的是按钮。

![SketchUp kitchen output 6](img/d7759b638177dd1337bbca3ad2076221.png)



现在让我们分解这个图像布局，并为分解它选择它，然后右击并选择分解选项。

![SketchUp kitchen output 7](img/9a385721be6b405bc62c69e6161eadc8.png)



分解后，您可以将其用作该布局的基础矩形。在做任何事情之前，先把它组织起来。

![SketchUp kitchen output 8](img/33e20284221ebb5135ef69c6f213d519.png)



现在你可以使用线条工具来创建厨房的墙壁，或者使用矩形工具来创建厨房的墙壁。

![SketchUp kitchen output 9](img/a7995371fe38be7e56a57de6a3c95e24.png)



创建后，我将使用推/拉工具，并根据需要的尺寸为这面墙增加高度。

![SketchUp kitchen output 10](img/1e074c17493c1be9b834bababee05c75.png)



我会对另一面墙做同样的事。

![SketchUp kitchen output 11](img/1df67d4f2f6017d31b5e94c45b981f9f.png)



一旦你做了，确保你把墙分成一组，把它和其他设计对象分开。因此，为了进行分组，在这面墙上双击三次，将选择这面墙的整个几何图形，然后右键单击选择“进行分组”选项。

![SketchUp kitchen output 12](img/5d9ef054d011f234a891724c7a4601ae.png)



您可以隐藏任何绘制的截面，以正确创建其他组件，并隐藏组件选择该组件，然后右键单击并选择隐藏选项。

![SketchUp kitchen output 13](img/15275bd35e66f962ea4705f02fc31e46.png)



现在，在制作完墙的其他部分后，我将使用与之前相同的方式将它分组。

![SketchUp kitchen output 14](img/2c45e5cb13dd2580d38949c8fb6599e2.png)



如果你想取消隐藏对象，然后去编辑菜单，去下拉列表的取消隐藏选项，并选择全部或最后。

![SketchUp kitchen output 15](img/c68b89685e969238609c018f44695630.png)



现在我将根据布局在这个区域制作窗口。为此，我将借助卷尺测量工具来标记窗户离地面的高度。

![SketchUp kitchen output 16](img/48cf4aa7afbfcc22a1c06393beeea317.png)



然后再用同样的方法标出窗户本身的高度。

![SketchUp kitchen output 17](img/9a99ea6ae72a3a9c1b34b5884313e54f.png)



然后在这个区域画一个矩形，用推拉工具推拉。

![output 18](img/77e1d547d3df26009eb157bf643440b7.png)



现在让我们把厨房打扫干净。您可以使用布局图中的线条工具选择该区域。

![output 19](img/ffbfc6419d92b4439d3d7ea63c35f9de.png)



像这样增加高度。

![output 20](img/bec57af45053c93a26a27d6878c336b2.png)



或者也可以像这样用线条或矩形工具创建一个新的区域。

![output 21](img/e0129631c3c2bf2051a8df895999488c.png)



现在我将根据需要的尺寸给它增加一些高度。

![output 22](img/4010f69c12a3bb7ee5677efa1571776f.png)



现在让我们看看 3D 仓库里的一些厨房配件的 3D 模型。为此，进入窗口菜单，并点击三维仓库选项。

![output 23](img/aa07d0ef1e63f164bb955dc2da8a6379.png)



我将首先在此对话框的搜索框中搜索厨房炉灶面。

![output 24](img/8964e9acb70acd45cc9500016eee3731.png)



下载这个炉灶面。

![output 25](img/fc2b4cd383179cf72f73394da06240d7.png)



现在，我将把它放在工作屏幕上的任何地方，就像这样，然后切换到 x 射线模式。

![output 26](img/39fa5d2a6e2257baf325451bf649c0c5.png)



并隐藏侧壁，这样我就可以正确地把这个灶台放在平板上。对于隐藏墙，我将再次选择此墙，然后右键单击从下拉列表中选择隐藏选项。

![output 27](img/c148d2eaf700596e4a00136032278405.png)



现在我将改变到这个厨房模型的俯视图，然后从工具面板中选择旋转工具。将旋转平面指定给此炉灶面，并沿楼板方向旋转。

![output 28](img/3069f0ebd7441b862eac7df18e7917ff.png)



然后根据平面图将其移动到灶台位置。为了移动它，我将使用移动工具并指定移动平面，然后像这样移动它。

![output 29](img/d0919060067e71cdc383963ea48ceb63.png)



你可以在平板上划分区域，作为放置厨房用品的抽屉。要分割，请双击平板区域，这样它将转换为编辑模式，因为我已经将它作为一个组。你必须记住，每次你制作任何对象时，首先要使它成为组或组件。

现在，我将使用线条工具，从板的顶部边缘到底部边缘绘制线条，这样划分截面。我没有考虑他们的具体维度，只是为了解释你。

![output 30](img/e74a02f83e10dd579240135c4eb6a79b.png)



现在选择截面的任何一个，并偏移一定的距离来制作正面。

![output 31](img/14fbd4dfe3f142eb4c5b16f98ef9728f.png)



像这样推/拉这个区域。

![output 32](img/bb3b724ec86cd356334011db13bf237a.png)



现在你也可以用玻璃做拉门，这样来抵消。

![output 33](img/83bf11997c33a5a3a8f598745360b8de.png)



然后移除中心区域。

![output 34](img/0d0e12fbac4d3c9eece6595ad61c6cb9.png)



在这里为玻璃画一个矩形，你可以根据选择的玻璃给它提供厚度。

![output 35](img/158b46c8acecb3efdd98f1a33cd7922b.png)



完成你的厨房布局后，你可以在每个部分填充材料，就像我在笔记本上填充的一样。

![output 36](img/def7f723ac4e8c62166150b27a98c6c2.png)
