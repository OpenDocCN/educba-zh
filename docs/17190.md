## 裁剪工具简介

裁剪工具只不过是选择一个我们需要的区域，并从图像中移除剩余的区域；当我们选择一个区域时，剩余的部分将是透明的，如果我们单击进入，剩余的取消选择的区域将消失，所选区域将成为完整的图片，我们有一些类型的裁剪选项裁剪用于每个编辑软件，如视频编辑和照片编辑，在 Photoshop 中它主要用于照相馆的目的，他们裁剪普通照片到护照和其他照片大小。当我们需要改变图像的大小时，我们通常会使用裁剪，如果我们认为图像的某一部分足够满足我们的要求，那么我们会借助裁剪工具删除其余部分；有许多作物可供选择，这对我们帮助很大。

### Photoshop 中的裁剪工具怎么用？

下面是在 photoshop 中使用裁剪工具的步骤:

<small>3D 动画、建模、仿真、游戏开发&其他</small>

**第一步:**打开桌面上的 Photoshop 应用程序。

![Photoshop app (Crop Tool in Photoshop)](img/63af1bb01ea7b2596108e92166d90a19.png)



*   这里是 Photoshop 应用程序；我们可以在左边找到工具，在右边找到图层。

**第二步:**打开任何文件，进入文件，点击打开。或者我们可以使用快捷键“ctrl+o”来打开。另一种方法是在 photoshop 窗口上双击打开文件。

![open file (Crop Tool in Photoshop)](img/2bfb550c6de55f0e09aef0b759d9d83f.png)



**第三步:**当我们点击打开时，我们可以找到一个窗口，我们需要从这个窗口中选择我们需要编辑的图像。

![select image (Crop Tool in Photoshop)](img/551b6982b3d5684404ce4a0d53f66fba.png)



**第四步:**这里，我们打开一张图片。

![working image (Crop Tool in Photoshop)](img/7eaf99fe58f6e9b1737aa65b6e725be6.png)



*   在这里我们可以发现一只蝴蝶在花上；现在，我们将在这张图片上工作。

**第五步:**现在转到工具栏，在这里我们可以找到作物符号，工具栏左侧的第三个<sup>符号</sup>；我们也可以使用 C 作为裁剪的快捷方式。

![select crop (Crop Tool in Photoshop)](img/3373c6e96aa64a334177b8df4e3de577.png)



*   一旦我们选择了裁剪工具，符号将会改变。

**第六步:**现在在图像中画一个矩形选框符号。

![rectangle symbol (Crop Tool in Photoshop)](img/98fa9f6e89e12bf01cf4c0dbc534cdaa.png)



*   在这里我们可以找到图像中的矩形选框符号；其余区域看起来是透明的，这意味着我们只需要选定的区域，其余的可以禁用，如果我们点击进入；所选区域的其余部分将会消失。

**第七步:**如果我们需要放大选中的区域，我们可以拖动矩形框，这里我们有 8 个点来增加大小；这是一个自由大小的作物，所以我们可以裁剪多少。

![drag rectangle (Crop Tool in Photoshop)](img/14baa500f99d89088dffff0f0d460088.png)



![set rectangle (Crop Tool in Photoshop)](img/b1e825938d36ada2720b2df95d274df5.png)



![set rectangle2 (Crop Tool in Photoshop)](img/2433d3fbdba8583814e875c19b005a2f.png)



**第八步:**这里我们有盾色选项；我们可以改变取消选择区域的颜色。

![sheild color (Crop Tool in Photoshop)](img/279dba5717b5dacd806b3c10da8c5b13.png)



第九步:在这里，我选择绿色作为盾牌的颜色。

![background green (Crop Tool in Photoshop)](img/7d22d0afb020b1d8691fc7ef7fd97e8d.png)



*   我们发现背景颜色变成了绿色。

**步骤 10:** 现在，我们看到取消选择区域中填充的颜色。

![clolor filled (Crop Tool in Photoshop)](img/d49397f1a9552afe4389f2d6f0a9d242.png)



**步骤 11:** 在这里，我们可以找到一个盾牌的不透明度选项；使用此选项，我们可以更改取消选择区域的透明度。

![opacity in crop (Crop Tool in Photoshop)](img/5a5d9c0aa489b77f62dc07c2d2e413f2.png)



*   这里我们给了 100%的透明度，所以背景隐藏，只有选定的区域是可见的。

**步骤 12:** 如果我们降低透明度，我们也会找到背景图像，这可以帮助我们找到目标来增加裁剪区域的大小。

![transpency 50% (Crop Tool in Photoshop)](img/7d598f224cb9e3782d6e6e64607cc082.png)



*   我总是建议保持 50%的透明度，这样，如果我们需要增加作物面积，就可以通过找到目标来找到其余的选定区域。

**第十三步:**如果我们给 1%的不透明度，我们没发现有什么区别；我们只能找到裁剪矩形框。

![opacity 1% (Crop Tool in Photoshop)](img/361138e03fbe0c887df5b681a1f6b3b0.png)



**第 14 步:**这里，我们有一些裁剪选项；在左上角点击向下箭头；我们可以找到子菜单。

![crop submenu (Crop Tool in Photoshop)](img/35ac73f2497ec477fb15e7ede943daaf.png)



*   在子菜单中，我们可以找到 5 种类型的作物选项，如果我们下载预设，我们会获得更多；每个裁剪选项都有不同的尺寸。

**步骤 15:** 在这里，我们尝试 4×6 英寸的裁剪选项，让我们看看它是如何工作的。

![4x7 crop (Crop Tool in Photoshop)](img/d45357f69bd993fd873c8e1beec3e81a.png)



*   如果我们用 4×6 的裁剪来拖动，我们只能得到标准的照片尺寸；我们不能随意增加规模；我们只能增加标准尺寸。

![4x6 crop](img/ab6dc135a205039d6e141013d0d23485.png)



*   实际上，4×6 是一个基本的标准照片尺寸，它有 35mm 标准，35mm 是 2:3 比率的旧胶片格式，这是一个可打印的格式，所以 6×4 接近于这种格式，这就是为什么它被称为标准。

**第 16 步:**这里，我们尝试 5×7 的裁剪。

![5x7 crop](img/0db9592a87acb1fed5ebd647965e6c17.png)



*   如果我们需要 1500 x 2100 像素，我们可以使用这个裁剪选项，它给出了 7:5 的比例。

**步骤 17:** 让我们试试 8×10 的裁剪选项。

![8x10 crop](img/ad0ec2db701ab04150b37c3e03496f85.png)



*   8×10 是 5:4 比例的大照片尺寸；它也称为插图尺寸，它有 2400 x 3300 像素，但打印分辨率应为 1536 x 1024 像素。

![big photo size](img/c28a7235327b6385e1629b6e31d9c4d4.png)



**第十八步:**我们可以根据需要给作物我们的尺寸。

![set dimension crop](img/169065bce10863e0094ba10098ab1c94.png)



*   我们可以改变宽度和高度的值来裁剪图像。

![set to crop](img/8cec82381b57342126e5200c395cbf93.png)



**步骤 19:** 如果我们需要旋转裁剪，只需选择并旋转矩形框。

![rotate rectangle ](img/81617f27865c85c4a5c30bd1b3107f4d.png)



*   同样，我们可以得到旋转的作物图像。

![rotate crop](img/edaa661ec31de4d4b37473da9d969ab7.png)



**步骤 20:** 如果我们需要让两幅图像的裁剪尺寸相同，我们需要使用正面图像选项。

![drop image](img/fac04f04ed4021ed9051ed5040ad5c23.png)



*   在这里我们可以找到正面图像选项。
*   在这个方法中，我们需要导入两幅图像，裁剪第一幅图像，选择前面的图像选项，然后点击第二幅图像；相同的裁剪将应用于第二个图像。

![import two image](img/d413da04a1fa875ff0d5e8abc105bbf7.png)



*   当我们对多幅图像进行裁剪时，这种方法非常有用。

**步骤 21:** 进入裁剪选项，点击播放图标，我们会得到一个子菜单。

![new tool option](img/19f78267a8e9dbf4bd3b502e365d424b.png)



*   在子菜单中，我们可以获得很多选项，如创建新的工具预设；从这个菜单中，我们可以导入新的预设，也可以删除旧的预设。

**步骤 22:** 我们可以创建自己的 also 预置，进入裁剪选项，点击图片图标；然后，我们可以得到一个询问预置名称的菜单。

![present image](img/640a659f0509a07653435a06a981171c.png)



*   我们可以用我们使用的尺寸给预置命名。

### 结论

在这个过程中，我们导入一个图像，并在 Photoshop 中进行裁剪；我们使用不同类型的作物选项，如 6×4、5×7 和 8×10；我们学习什么是正面图像，我们可以对两个图像使用相同的裁剪尺寸，如何导入预设以及如何添加预设等。我希望我们进行了一次有趣的讨论，并学到了一些重要的东西。
