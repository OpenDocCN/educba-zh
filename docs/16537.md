## InDesign 高亮文本简介

InDesign 高亮文本可以理解为为文本框架中文本或字符或字母创建背景，以指示用于特定目的的选定文本或字符或字母。通过突出显示文本，我们可以将该文本与文本框架中的其他文本区分开来，或者使用该荧光笔作为设计元素。我们可以为文本使用任何颜色的荧光笔，但没有专门的工具来突出显示文本。事实上，我们通过调整文本参数的一些特征来做到这一点。在这篇文章中，我将告诉你如何突出显示文本，并在不同页面的文本上突出显示。

### 如何在 InDesign 中高亮显示文本？

我们在该软件中突出显示文本，以便为该文本提供特殊指示，或者使用荧光笔作为设计元素。所以让我告诉你这个软件中高亮文本功能的所有重要方面。

<small>3D 动画、建模、仿真、游戏开发&其他</small>

我将首先从“新建文档”对话框创建一个新文档。

![InDesign highlight text output 1](img/053002b407f17f5d433d5f8549bac257.png)



现在我将使用文字工具在这里画一个文本框。

![InDesign highlight text output 2](img/f14d1e193a1f2dadb8ee1a9a5e496db8.png)



我将在这个文本框架中粘贴一些文本内容，并用它做一些格式化。

![InDesign highlight text output 3](img/07382774279482dbfc7544750a5b280a.png)



现在，我将选择此文本并突出显示它。你必须首先选择你想要突出显示文本或字符或字母。

![InDesign highlight text output 4](img/ca20077f452609b9b92da02235c3df80.png)



现在，在 InDesing 的用户屏幕中导航字符面板。在这个软件的面板部分，我有一个角色面板。

![InDesign highlight text output 5](img/3cf6d374da92f839e87c3f522c51dcc2.png)



如果你的用户界面中没有字符面板，你可以进入窗口菜单，在下拉列表中你会找到类型和表格选项。在新的“类型和表格”下拉列表中，有一个字符面板，或者您可以按键盘上的 Ctrl + T 键作为快捷键。

![InDesign highlight text output 6](img/82d11d8f36c67aa4040be2e6e76e9eea.png)



点击这个面板右上角的按钮，你会看到一个弹出列表。在列表中点击“下划线选项”。

![InDesign highlight text output 7](img/7339670201384519474f09696525f20d.png)



一旦你点击它，一个下划线选项的对话框将会打开。在此对话框中,“下划线打开”选项被禁用。

![InDesign highlight text output 8](img/74f36bc43cc2f20eb051b6a85220ad98.png)



单击复选框启用它。一旦你点击它，一条下划线将出现在你选择的文本上。

![InDesign highlight text output 9](img/e3e4517cca4299d1a5ebf6c1f80b6fd6.png)



我将增加权重选项的值，您可以看到，如果我们增加该选项的值，下划线的宽度也会增加。

![InDesign highlight text output 10](img/7d00c9c1ed654f0d29883151b37479c9.png)



我的字体大小是 12pt，所以我将采用比这更高的权重值，这样我的文本就可以完全覆盖。您可以根据需要设置下划线宽度。没有必要完全覆盖文本。

![InDesign highlight text output 11](img/f478449cfcb6d6316c06b4b8c21ba939.png)



现在我将开始改变最大值。通过改变偏移值，我们可以向上或向下移动下划线。如果我们正向增加偏移值，它将向下移动。

![InDesign highlight text output 12](img/0ec0aa1d0b773b643c58a7228ed9761e.png)



或者如果它是负值，那么它会向上移动。

![InDesign highlight text output 13](img/ac52ce84bccec3354ddf51bb7fde0c4f.png)



现在从颜色选项，我们可以选择下划线的颜色，这将是我们的荧光笔的颜色。我将选择黄色，然后单击确定按钮。

![InDesign highlight text output 14](img/3f3ef3cd9bfd86b9d4f176fc228232c3.png)



现在你可以看到我们已经突出显示了我们选择的文本。

![InDesign highlight text output 15](img/493bb296f393cd26035297af64c29488.png)



如果您想在文档的其他文本上也有某种类型的高亮文本效果，该怎么办？它可能在同一页上，也可能在文档的任何其他页上。出于这个目的，我将使用字符样式面板。因此，我将从该软件的同一面板部分打开该面板。

![InDesign highlight text output 16](img/ca2e8856877e3cf6857ca51ffaf827a7.png)



您也可以在菜单栏的“窗口”菜单中找到此面板。所以进入这个菜单，然后进入下拉列表的样式选项。在新的样式下拉菜单中，你会找到字符样式选项。您还可以按键盘的 Shift + F11 功能键作为其快捷命令。

![InDesign highlight text output 17](img/65b69d95bc500e4f72548e65ea8c064e.png)



我将选择此文本，然后单击此面板的“创建新样式”按钮。你会在这个面板的右下角找到这个按钮。

![output 18](img/25e4116710b6c401ce77d7cb9a2da901.png)



一旦一个新的字符样式层被添加到面板的图层部分，点击它，并确保您的文本被选中。现在这个字符样式层将被分配给选定的文本。

![output 19](img/b1c9138fe3da3ea12b3dda2134e546ec.png)



现在右键点击选中的字符样式 1 层，从下拉列表中选择“编辑字符样式”选项。这一次你不需要选择文本。

![output 20](img/ad6a46704f998d88df87405abd85b071.png)



点击此选项后，将会打开一个字符样式选项对话框。从列表中选择“下划线选项”,并通过点击该选项的复选框启用“下划线开启”选项。你可以看到，一旦我们启用这个选项，下划线将添加到我们的文字这个字符样式层。

![output 21](img/407b3b99486a76867ce9fb002911a4bd.png)



这里我将增加下划线的宽度，方法和上面讨论的步骤一样。

![output 22](img/1883b1c274bfc9c58147bf8628bf46ba.png)



我还将根据我的字体大小设置偏移值。

![output 23](img/a2f8ecf098835d17598c254a47a346a0.png)



然后给它指定一种颜色。确保此对话框的“预览”按钮处于打开状态，以便您可以看到实时更改。

![output 24](img/99114da41db0774fa76e9a8a4c8cfcde.png)



这是色调百分比选项，通过改变它的百分比，我们可以增加或减少所选颜色的不透明度或亮度。

![output 25](img/3ba132cb4f22ea18b600c0584bc915d6.png)



从笔画的类型来看，我们可以从这个列表中选择不同的风格。

![output 26](img/9ea078a5936be538138504980cfdaed4.png)



你可以看到，如果我选择这种风格，那么荧光笔会像这样变化。

![output 27](img/958f23ae6bd8fdc830cceaa563a922d7.png)



您也可以选择这种点画来创建一个好的高光效果或设计。

![InDesign highlight text output 28](img/3429ebc87492ce95f74d1f7b307a5cb8.png)



如果你用虚线，你也可以像这样添加间隙颜色。

![InDesign highlight text output 29](img/837c7e057af1eb2f520a44e360e4ecc8.png)



现在你可以将这个高亮文本效果应用到其他文本中，只需选择该文本即可。

![InDesign highlight text output 30](img/45bdebea89003f04bc2c7a4881110d7c.png)



并点击字符面板的字符样式层。

![InDesign highlight text output 31](img/a835f85e47a376766ab95554696a8608.png)



它也将应用于选定的文本。

![InDesign highlight text output 32](img/2c69eebf641926f1b84a02f11a5d1e00.png)



### 结论

这样，您可以为不同用途的所需文本创建高亮文本效果，并将相同的效果应用于文档不同页面的其他文本。您可以分析此软件的下划线面板的不同参数，以获得更多关于此高亮文本功能的命令。
