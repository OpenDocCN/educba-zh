## 概率神经网络导论

概率神经网络可以是前馈神经网络；它广泛应用于分类和模式识别问题。PNN 有三层节点。在 PNN 算法程序中，每个类别的亲本似然分布性能由 Parzen 窗口和非参数性能来近似。

### 概率神经网络的层次

PNN 平方测量的操作被组织成具有完全不同层的多层前馈网络。这个正方形有四层。它们是:

*   输入层
*   图案层
*   求和层
*   输出层

**输入层:**我们预测一个值，它被给到输入层，在那里它减去中值。这些输入神经元被输入到下一层，这是每个隐藏节点的隐藏层。
预测变量非常非常单元是由输入层圈定的。然后，输入神经元每隔一段时间将这些值提供给隐藏层中的每个神经元。

**模式层:**该层在每种情况下每隔一段时间包含一个单元指导工作数据值。它存储目标价值方面的预测变量的值。隐藏单元计算行为与神经元中心的几何距离，然后应用径向基函数执行内核执行牺牲字母值。

**求和层:**在目标变量中，PNN 的每个类别都有一个模式。每个辅导工作案例的实际目标类别都包含在每个隐藏单元格中。隐藏单元开始的加权值被单独提供给对应于隐藏神经元类别的模式单元。模式神经元将它们所代表的类的值相加。

**输出层:**输出层比较模式层每隔一段时间累积的每个目标的加权投票，并使用最前面的必要投票来预测目标类别。

### 概率神经网络的应用

*   它被用于模拟雨水管道的结构退化。
*   它用于发现完全不同的模式分类问题。
*   它有助于群体药理学。
*   它有助于遥感图像分类。
*   船舶识别受害者。

### PNN 算法

以下步骤设置了 PNN:

*   将它们分类成 N 个不同的集合，其中每个集合包含一个向量类别。
*   对于每一组，我们都倾向于勾勒出数学家的表现。
*   对于每一个集合，我们都倾向于勾勒出数学家输出性能的总和。

一旦勾勒出 PNN 的轮廓，我们就可以将矢量输入其中，并对它们进行如下分类:

*   读取输入向量，并将其提供给每个数学家在每个类别中执行。
*   对于每个隐藏节点群，计算所有数学家在隐藏节点上的有用值。
*   对于隐藏节点的每个集群，将其所有数学家有用的值提供给该集群的一个输出节点。
*   在每个类别输出节点，对所有输入求和，然后乘以常数。
*   在输出节点上找出所有有用值的最大值。

我们主要使用 PNN 而不是多层感知器，通过牺牲 PNN，我们得到了一些好处和坏处。我们已经提到了一些。

在上图中，它代表了 PNN 的建筑。概率神经网络有三个节点。上图代表了 2 类 PNN，但它可以扩展到 X 类数量。输入层有 N 个节点。这 N 个节点通过分支连接到隐藏层，以便隐藏节点接收完整的输入特征。

隐藏节点被收集到每个 X 类的组中。类 X 的组中的每个隐藏节点对应于以该类中与其相关联的特征为中心的高斯函数。这些高斯函数值被发送到输出层，它们被称为 X 输出节点。

### PNN 的优势和劣势

下面是概率神经网络的优点和缺点。

**优点:**

*   PNNs square 比多层感知器网络快得多。
*   PNNs 可能比多层感知器网络正确很多。
*   PNN 网络平方测度对异常值相对不敏感。
*   PNN 网络产生正确预见目标的可能性。
*   PNNs 接近 Bayesoptimum。

**缺点:**

*   与多层感知器相比，PNN 较慢。
*   PNN 需要大量的内存区域来存储。
