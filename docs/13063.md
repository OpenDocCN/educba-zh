## 浏览器简介

浏览器是一种应用程序，它提供了一种查看和交互所有万维网信息的方式。这包括网页、照片和视频。在网络出现之前,“浏览器”一词起源于用户界面的通称，使你能够在线访问文本文件。今天，许多人可以使用网络浏览器访问互联网，这几乎被视为许多人管理日常生活的必需品。

### 浏览器是如何工作的？

Web 浏览器充当混合客户机/服务器的一部分。客户端是运行在用户计算机上的应用程序，它向 web 服务器发出请求，而服务器端是向应用程序发回详细信息的 web 服务器。然后浏览器解释用户的设备并显示信息。网络浏览器通常由许多互通的部分组成。这包括 UI，它是用户与浏览器交流的点。浏览器引擎查询呈现，呈现引擎将 XML 或 HTML 文档解释为所请求的网页。网络是管理互联网内的安全性和连接性的东西。用于 Javascript 解释的解释器用于网站上的 Javascript 应用解释和执行。UI 的后端用于小部件和窗口。为了保存数据，包括书签、缓存和 cookies，称为数据存储/规范的存储层管理数据。

**网络浏览器的组件如下:**

*   **用户界面:**用户界面是用户可以使用不同选择的环境，如窗口地址栏、后退和前进按钮、菜单和书签。
*   **浏览器引擎**:UI 和渲染马达作为桥梁链接在一起。它查询再现引擎，并基于来自多个用户界面的输入来操纵它。
*   **联网:**URL 是通过 HTTP、FTP 等互联网协议获取的。它管理互联网上所有形式的通信和安全。此外，检索到的文档可用于存储网络流量。
*   **数据存储**:为了将某种信息作为 cookies 存储在本地，数据存储是浏览器使用的持久层。浏览器支持许多存储过程，包括 IndexedDB、WebSQL、localStorage 和 FileSystem。
*   **JavaScript 解释器:** JavaScript 解释器将文档中使用的 JavaScript 代码进行转换，顾名思义执行。返回引擎将在系统显示屏上显示翻译的结果。

### 浏览器的功能

以下是浏览器的常见功能:

1.  **书签:**这允许用户选择其他用户预定义的网站。
2.  **标签:**这允许用户在一个窗口中打开多种网站。
3.  **后退和前进按钮:**这将用户带到上一个或下一个标签。
4.  **Stop** 阻止网页加载页面中断与网页服务器联系的按钮。
5.  **刷新:**此功能将重新加载页面。
6.  **主页按钮:**当选择时，将用户带到预选的主页。
7.  **URL:**URL 是键。您在地址栏中输入的网址或 URL 会通知浏览器可以从哪里访问一个或多个页面。例如，当你在地址栏输入 https://www.educba.com/的网址时，你会被带到 educba.com 的主页。在两个关键部分，用户看到这个特定的 URL。第一个是 http:/ element 协议。HTTP 是一种标准协议，用于请求和分发互联网文件、大量网页和项目。由于浏览器知道协议是 HTTP，它知道如何查看前面斜线右边的任何内容。

### 网络浏览器如何帮助你的生意？

所有有效的企业都采取某种形式的数字营销。在设计公司网站时，您必须确保其在所有主要平台上的可访问性。如果没有，你可能会失去流量和公司，并无意中转移到你的竞争对手的网站客户。食品生产商在很大程度上依赖吸引人的照片和视频来使他们的产品和材料(以及声望)富有魅力。美容和设计公司会让他们出售的服装看起来同样时尚。如果网络浏览器不能正确加载图像，你可能会失去潜在的客户。在一种浏览器类型下加载很快，而在另一种浏览器类型下加载很慢的网站可能会失去客户。如果人们感到不安，人们想要快速的答案，迅速关闭你的网站，关注你的竞争对手。点击量很重要，所以跨浏览器强制执行对你的公司来说比你想象的更有用。
