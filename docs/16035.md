## 钢笔工具简介

钢笔工具，通常我们在 photoshop 中看到，它帮助我们创建一个路径和选择字符。钢笔工具也是 after effects 中一个很棒的表现工具；使用钢笔工具，我们可以轻松地绘制形状，并创建不同的遮罩；遮罩用于在后期效果中合并多层。在钢笔工具的帮助下，我们可以绘制形状和创建动画的路径；我们现在可以用钢笔工具制作动画、文本动画和形状动画，让我们看看钢笔工具帮助下的一些动画。

### 在后期效果中使用钢笔工具

首先，我们看看如何用钢笔工具画一条线，并制作一条线的动画。

<small>3D 动画、建模、仿真、游戏开发&其他</small>

**第一步:**新建一个构图。

![new compostion in pen tool](img/96051a738f4dc177918f8eb1db559002.png)



第二步:创建一个形状层，在这里我们可以画出线条的形状并制作动画。

![creating shape layer](img/933d55f3348359189f61cee3d6258f62.png)



**步骤 3:** 选择钢笔工具，这里我们不需要填充，因为我们只画一个形状，填充将用于遮罩的情况下，这里有两个选项一个是遮罩，另一个是绘制形状，所以我们选择形状选项。

![select pen tool](img/e62e55f8c6298e959078f6a882cf763b.png)



**第四步:**选择你喜欢的颜色的股票；总是选择吸引人的颜色，因为它给动画带来美感。

![choose color ](img/ac7230e3c44208ed2aa550b37f4d07d8.png)



第五步:现在我们开始绘画，借助钢笔工具画出你想要的形状，画一条像曲线一样的线即使你没有得到正确的形状也没问题，因为我们有另一个选项来纠正形状，我们可以在下一步看到。

![draw with pen tool](img/4db830975922441e25e7db2afdf24da3.png)



每次我们都不会得到一个好的绘图，但是，我们使用钢笔工具；当使用钢笔工具时，在某些地方，我们不能管理曲线和形状；在这种情况下，我们可以选择修正形状和曲线；让我们看看如何

**第六步:**到形状层，内容，形状。如下图所示选择路径。一旦我们选择了路径，我们就可以在绘图中找到这些点。

![select path and draw](img/ea3f79ad90a5ac35859210e7ac59fcf2.png)



**第七步:**现在选择[选择工具](https://www.educba.com/selection-tool-in-illustrator/)调整图中的点；通过调整这些点，我们可以根据我们的需要得到好的形状。

![adjusting point](img/177f41a0f71bd01bbc6f64b2472a3b77.png)



**第八步:**我们现在可以看到图纸中的锚点；我们可以选择每个点并调整绘图。

![anchor points](img/4924d5785b1b4859a4f27eff3b5b310f.png)



**第九步:**通过调整点，我们可以得到我们的形状。

![adjusting the point](img/987c70c030bfbc40af83bd25930d8887.png)



**第十步:**在这一步中，我将解释非常重要和有用的选项修剪路径；我来讲讲修剪路径；一些很少使用 aftereffects 的用户可能不知道这个，因为它隐藏在形状子菜单下，这个工具允许你调整图层路径。我们可以用这个修剪路径做多个动画，像地图路线，信息图，文本笔划，文本框，圆圈爆发。

![the trim path](img/1fda40782f4b3ffec3584824d8f32aa3.png)



**步骤 11:** 在这个修剪路径中，有两个主要选项开始和结束。

![Trim paths](img/5e4f7ed6de8ce6d6aea7230400e7ea1f.png)



**第十二步:**现在，更改 start 和 end 的值。

![change value](img/aa20dd9b48cad935442120e105fc5288.png)



**步骤 13:** 这里我们改变 start 和 end 的值。

![changed value (pen tool in after effects)](img/2da7d61a7c49f34e8b3e9f5ade1a527f.png)



**步骤 14:** 在这里，我们需要[通过将动画](https://www.educba.com/animation-in-flash/)应用到起点和终点来启动动画。第一个开始是从 0，0 开始，结束值转到下一个需要的点，并在开始值和结束值中给出值 100，100。

![start animation](img/39ce72694168374213361f694738f1fd.png)



**第十五步:**在起点，现在把 End 的值增加到 20；我们可以找一条小线。

![end point](img/f1504c6b7a58ae6f9fc2ecd8d15e118f.png)



**步骤 16:** 当我们从起点移动到终点位置时，可以看到直线像蛇一样从起点移动到终点的动画。

![moving animation](img/0f9164d90374ea3525504a64a8d32d82.png)



**步骤 17:** 在这里，我们用矢量模式来编辑数值，使我们的动画更加流畅。

![vector mode (pen tool in after effects)](img/83e59d5aef53fd393a6d326ef579f6b1.png)



*   在这个过程中，我们看到了一个形状是如何在钢笔工具的帮助下动画化的，这里我们用钢笔工具画一条线；在路径的帮助下，我们创建一个形状，并使用修剪路径通过改变起点和终点来动画显示该线条。在钢笔工具的帮助下，我们可以制作一个粒子的动画，这不是唯一的形状。

### 使用钢笔工具制作粒子动画

下面是用钢笔工具[制作动画](https://www.educba.com/animation-in-after-effects/)的步骤。

**第一步:**导入任何你想要制作动画的物体；这里，我正在导入一个球。

![import a ball](img/94c7cd1970eb8a755f13099e68038cbc.png)



**第二步:**我们可以通过改变开始和结束的位置，用关键帧动画来动画化物体；这是任何人都能做到的基本正常程序。

![normal keyframe](img/e5633245c5ef64b683895135dfce70b1.png)



但是，如果你需要做一些不同的或有创意的动画，只需通过钢笔工具。

第三步:选择我们要为动画绘制路径的形状层。

![select shape layer](img/78d7acb91ad6b1fef415678205e80947.png)



**第四步:**选择钢笔工具绘制形状。

![use pen tool](img/4f6da8ea6dadb6e0322c5d905ad32fbe.png)



**第五步:**现在根据我们的动画要求绘制一个形状，我们已经导入了一个球对象，所以我绘制了一个上下的形状，因为我觉得球需要从上往下下落并一步一步地跳跃，所以我从顶部的第一个点开始，绘制了一个上下的形状，并在球需要停止的角落处结束。

![animate with pen tool](img/9a22dd5a7854de2702057a79e54cfee0.png)



这里你可以看到形状是怎样的；在这个形状中，球将要运动，你可以找到球运动出屏幕的第一个点。

**第六步:**对于每一个物体，都会有一个中心点；在这个过程中，我们需要将中心点改为底部，因为我们有一个对象球；如果球在接触地面时在中间有一个中心点，它的中心点将在看起来不专业的地方遇到地面，所以我改变了中心点。如果我们有任何其他对象，我们可以根据要求改变中心点。

![change center point](img/b4f00e886e7e57d255c6a4552ad4ed39.png)



**第七步:**到形状层，内容，形状-路径 1-路径。选择路径并复制，确保复制时选择了两个路径。

![copy the path](img/957825b80aa82ce972ff5bdcaaba720b.png)



**第八步:**到对象(球)上，点击键盘上的(p)选择位置，粘贴。

![select position (pen tool in after effects)](img/a7ec473b3ff59a8c5a66c7c56b63ecbd.png)



现在球是隐藏的，因为在将路径复制到位置应用到球时，它在该路径中移动。

**第九步:**通过调整时间线中的关键点，我们可以对动画进行快开头慢结尾之类的管理。

![adjust key points](img/fe295f33aa9a19d98de65b2a42673578.png)



### 结论

在这个过程中，我们使用钢笔工具制作一个球的动画；我们绘制了一个形状，创建了一个路径，并复制了形状层的路径，并粘贴在对象的位置，使对象跟随形状的位置。通过使用这个作为样本，我们创建了许多其他的动画。同样，我们可以使用钢笔工具来制作任何对象的动画。
