## Illustrator 中的形状生成器工具简介

Adobe Illustrator 是一个包含效果和高光的选项和工具的发电站，Illustrator 中的 shape builder 工具是一个重要且有价值的工具，可供用户学习和实践，因为他们可以使用形状和图案制作各种艺术作品和矢量设计，而不需要了解钢笔工具和画笔来创建他们的理想结构，Shape Builder 工具很像 Pathfinder 设备，因为 Pathfinder 工具为用户提供了五个选项，在 Shape Builder 工具中，有一个工作路径。

### 形状生成器工具

Illustrator 中的形状生成器工具是一个重要且有用的学习和实践工具。在实践中使用这个工具，用户可以使用形状和图案制作不同的艺术品和矢量图形。用户不需要精通钢笔工具，画笔可以利用形状生成器工具创建他们想要的设计。形状生成器工具很像[探路者工具](https://www.educba.com/pathfinder-tool-in-illustrator/)。而在 Pathfinder 工具中，用户可以从其他五个选项中进行选择，如相交、相减、相加、合并工具。在形状生成器工具中，我们创建一个路径，然后按照用户的要求去做。

### 使用形状生成器工具的步骤

**步骤 1:** 新建一个 A4 大小的文件。

![Shape Builder Tool A4 Size](img/d579a0685b19d0538ab8d8926bc84c46.png)



**第二步:**创建一个椭圆。确保“吸附到点”和“智能参考线”已打开。

![Shape Builder Tool step 1.1](img/f9873322a16c61b57361e9492f9ded17.png)



![step 1.2](img/0f55d0c7e2a322c3dc8c72a55ccd5278.png)



**第三步:**做好椭圆后，按 Alt +鼠标键捕捉到中心，创建一个相同的副本。

![Shape Builder Tool step 3](img/253fc4abcb6997eac101c2bce43dec33.png)



第四步:现在，我们有两个椭圆。选择两个圆，复制两个圆，并将它们放在上面两个圆的下面。

![Shape Builder Tool step 4](img/e9903ea6d10acb9ae6c7f1262619b2f3.png)



**步骤 5:** 选择所有四个圆，使用工具箱中的形状构建工具。一旦我们选择了形状生成器工具，我们就创建了一个要构建的路径。

![Shape Builder Tool step 5](img/ef5e5a99760d33b313f0caa2ebc45856.png)



第六步:最终的图像将会是这样的。

![Resulted Image](img/2effa0eff67b8e804ed8ec90c842406d.png)



步骤 7: 使用矩形工具，为已经创建好的花朵创建一个茎。

![Shape Builder Tool step 6](img/254ad075d7041ba00a569e8d4ee30971.png)



**第八步:**现在，旋转花朵形成合适的形状，重新上色。创建茎，并使中心与花对齐。

![step 7](img/6966a41f9adb2c50c1c574031be67b0a.png)



**第九步:**再创建一个椭圆，把这个椭圆放在花的中心。现在，选择花和椭圆，用形状生成器工具，点击椭圆的中心来创建一个形状。

![step 8.1](img/3afa3d31ff243558ba84e1a81094f885.png)



![step 8.2](img/af4818fffea2682fff2b43e2d971bbb6.png)



第十步:创建另一个椭圆，复制同样的椭圆。使用形状生成器工具，单击两个椭圆的相交部分。去掉形状的中心部分。

![Shape Builder Tool step 9](img/2eb643123560a6c5cf88188e86199bb2.png)



**步骤 11:** 旋转，将叶子贴在茎上。创建一个相同的副本，并将其附加到另一边。这将在形状构建器工具的帮助下完成并创建一朵花。

![Final Shape Tool](img/e3e34cdb5e4ea36dc41a7b3c74a425a9.png)



### 结论

有时，用户会发现形状生成器工具有点令人困惑和棘手，但是当我们不断练习和使用这个工具时，我们会发现使用它很容易也很有趣。同样，路径查找工具也可以在 Adobe Illustrator 中使用。形状生成器工具是一种快速简单的工具，用于使用形状和线条制作简单的艺术品。

使用形状创造任何东西都需要大量的逻辑思维和耐心。但是，与使用钢笔工具和画笔创作艺术品相比，它速度更快，需要的精力更少。虽然钢笔工具和刷子也很平滑和容易，但创建有形状的东西会很快，需要更少的时间。

在上面的例子中，我们看到了一朵简单的花是如何用圆形和矩形创建的。同样，我们使用形状生成器工具创建任何东西。图形设计师从使用这个工具中受益匪浅，因为它节省了大量的精力和时间。
