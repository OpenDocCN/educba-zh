## AutoCAD 中的拉伸简介

AutoCAD 是由 [Autodesk](https://www.educba.com/what-is-autodesk/) 开发的最常见、最好的计算机辅助绘图和设计软件。我们知道这个软件在设计领域的重要性。随着 [AutoCAD](https://www.educba.com/what-is-autocad/) 的引入，设计领域发生了巨大的变化和技术升级。我们可以把设计分为两类，2D 绘画和 3D 绘画。今天我们将讨论 AutoCAD 中的一种 3D 设计工具。使用 [AutoCAD 中的挤压功能](https://www.educba.com/autocad-vs-solidworks/)通过挤压一个封闭的表面来制作三维物体。顾名思义，该工具通过给定高度尺寸来拉伸 2D 曲面。例如，如果我们想做一个圆柱形的酒吧，我们必须首先绘制圆形表面，并将其应用到挤压工具，并给予适当的挤压尺寸，我们可以将其更改为一个三维对象。

### 在 AutoCAD 中创建拉伸效果

拉伸是一个简单的命令，在 AutoCAD 中给出了很多效果；让我们一步步来看几个例子。

**第一步:**在 AutoCAD 软件中打开一个新的图纸页面，从下拉菜单中选择**三维建模**选项，如下图所示。

![3d modelling](../Images/c862e8641a95585a654568827359313f.png)

**步骤 2:** 下一步是将绘图页更改为等轴视图。为此，你必须旋转导航工具，它被称为**视图立方体**。通过长按鼠标左键旋转 view cube，并设置适用于等轴测绘图的绘图页。制作 3D 对象的 2D 绘图总是在等轴测视图中绘制。

**第三步:**下一步是开始绘制要挤压的 2D 设计。挤出效果仅适用于封闭的形状，如圆形、方形、矩形、多边形或任何不成形的封闭图形。所以在这里，作为第一个例子，我们要挤出一个矩形。为此，我们必须绘制一个矩形，如下图所示。要画一个矩形，我们可以用线条或矩形工具来画。画出矩形并给出适当的尺寸。

![extrude a rectangular shape](../Images/a2a2419629cf1ad7c65372d4b0ca7922.png)

第四步:要把这个 2D 矩形变成一个三维物体，我们必须应用挤压命令。对于该类型，请在命令框中拉伸或选择工具栏左上方的拉伸工具。然后，您可以在命令框中看到一些选项，如下图所示。您必须选择任何一个命令。这里我们选择路径命令。

#### 使用挤出工具创建圆柱体

**步骤 1:** 步骤 1 和 2 与我们在第一个例子中所做的相同。打开软件，如前所述设置等轴测视图。

**第二步:**如果你想做一个圆柱体，首先要画出 2D 圆。在命令框中键入 circle，或者从工具栏中选择圆形工具，并根据您的要求绘制圆形，如下所示。

### 结论

拉伸是一个简单的命令，它在 3D 设计平台中为您提供了强大的功能。这是 3D 建模中最基本也是最重要的功能。使用“拉伸”功能可以制作任何复杂的形状。我们只看到了这个功能的几个例子，你可以多练习，通过做更多的任务，你会更熟悉和擅长这个。通过学习 AutoCAD 中的拉伸命令，您将了解该功能的基本功能。它将帮助您与其他 3D 建模软件一起工作，因为挤出的基本概念在所有应用程序中都是相同的。
