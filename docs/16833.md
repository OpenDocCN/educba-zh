## Inkscape 渐变简介

在这篇文章中，我们将学习如何在 Inkscape 中使用渐变。渐变效果用于给物体一种立体感，通常通过混合多种颜色或不同的阴影来创造立体感。渐变也可以用于纯粹的美学目的，但渐变的常见用途是在网页中用于各种形式和按钮。它也用于桌面应用程序。在 Inkscape 中，我们可以使用各种方法来创建渐变，这些方法不仅可以应用于填充，还可以应用于笔画，以创建一些令人惊叹的效果。

### 墨迹渐变的步骤

下面是下面提到的步骤

**第一步:**我们先用创建矩形和正方形工具或者按 r 键创建一个正方形，默认情况下，它会有黑色填充，没有描边。

![Inkscape gradient output 1](../Images/44448efd73587731d6527515f706c0dd.png)

<noscript><img class="alignnone size-full wp-image-453460" src="../Images/44448efd73587731d6527515f706c0dd.png" alt="Inkscape gradient output 1" width="698" height="567" srcset="https://cdn.educba.com/academy/wp-content/uploads/2021/02/Inkscape-gradient-output-1.png 698w, https://cdn.educba.com/academy/wp-content/uploads/2021/02/Inkscape-gradient-output-1-300x244.png 300w" sizes="(max-width: 698px) 100vw, 698px" data-original-src="https://cdn.educba.com/academy/wp-content/uploads/2021/02/Inkscape-gradient-output-1.png"/></noscript>

**第二步:**然后使用创建和编辑渐变工具或按 G 键，然后拖动到正方形对象上，这将创建一个渐变。首先在你想要开始渐变的地方点击，当你想要结束渐变的时候释放它。默认情况下，将创建一个线性渐变。您可以通过向不同方向拖动来更改渐变的角度。但是为了获得更精确的度数，您可以按住 Ctrl 键以 15 度为增量。

![Inkscape gradient output 2](../Images/2f8d3030b9c765df0911b23b14713015.png)

**步骤 3:** 然后使用创建圆形工具或按 E 键，我们已经创建了一个圆形，然后再次使用渐变工具，但这次我们将从控制栏中选择创建径向椭圆或圆形渐变。从中间拖动并释放。虽然这种方法不会像你猜测的那样从中心产生渐变。

**步骤 4:** 我们已经完成的渐变是渐隐到白色，但是如果我们想要改变渐变的颜色以及它渐隐到的颜色，那么我们选择正方形的节点，顶部节点将改变它开始的颜色，底部节点将指示原始颜色渐隐或混合到的渐变的颜色。在圆形的情况下，它是径向渐变，所以它有两个端点，但您可以选择任何一个来改变渐变的颜色。

步骤 5: 现在，如果你想要一个不同的渐变颜色，而不是从两个颜色中创建的颜色，那么你可以从控制栏中插入新的停止点。这将添加一个菱形的新节点，因为起始节点是方形的，结束节点是圆形的。您也可以拖移色标，并将位置更改到需要颜色过渡的位置。在线性渐变和圆形渐变中添加停止点的方法是相似的。关于颜色选择，我们在底部有一个预置颜色栏，但是如果你想从另一个图像或对象中选择，你可以直接使用从图像中选择颜色工具或按 D 键

第七步:在选择任何颜色之前，确保你已经选择了渐变中的任何一个节点，如果没有，那么就用一种颜色填充整个形状。因此，在选择任何节点后，它可能是开始、结束或中间，您可以使用可用的颜色模式(如 RGB、HSV、HSV、CMYK、Wheel、CMS)来更改平面颜色的填充。我们将使用 HSL，即色调、饱和度和亮度。色相决定颜色，饱和度决定颜色的深浅，明度决定你想要的明暗。如果您想要删除任何特定的颜色，请按平面颜色前的 x 图标
