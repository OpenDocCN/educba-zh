## 什么是 GitHub 回购？

Github 存储库是可以灵活存储代码的地方，它们允许存储库托管服务。要特别提及的面向网络的基于网络应用的接口用于该过程。使用 GitHub repo-like 接口的一些关键优势是它提供访问级别控制的能力，还可以执行与协作级别相关的功能。这是一个基于存储库的托管服务，但除此之外，它还有很多自己的特色。这些 GitHub 存储库非常灵活，可以存储各种项目。通过 source projects 选项，可以设计和引入更可靠的软件。人员级别的协作也可以通过 GITHUB repo 的方式引入，工作的跟踪过程可以以非常高效的方式进行。

### 回购示例

GITHUB 存储库上 python 的一些关键趋势任务如下，Manim:对于数学视频，允许以编程方式创建相应的动画，它本质上在 Python3.7 上执行。Python 主要完全基于包来生成所创建的动画。

气流:工作流程布局过程。这些管道允许动态地执行代码的编写。Python 2。五用于此过程。工作流程的计划和控制可以在这里完成。这里是英里，就像最初的简单看板，现在不再有很大的区别，要考虑的最简单的本质区别是在这些结构中，扑克牌可能会例行公事地从一种声誉转移到另一种声誉。

Ansible:一个简单高效的基于 IT 的自动化平台，托管在基于 GitHub 的服务上，这个库的主要优势是它能够维护和允许软件部署到系统上。软件应用程序开发的所有过程都可以通过这些自动化存储库复杂地自动化。例如云的管理、网络管理和产品的安装等都可以轻松安装。这些都是可转换元素的优点。

Peregrine:这是 GitHub 上另一个著名的资源库，它是一种叫做 perigrine 的语言的来源。这种语言与 python 编程关系非常密切，在某种程度上也以同样的方式使用。
