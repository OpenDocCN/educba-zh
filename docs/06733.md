## 引导所见即所得简介

bootstrap 有许多高级工具和类，用于制作用户友好的高级网站和 web 应用程序。它是一个轻量级的高级工具，用于操作和设计信息。这是一个网站用户的文本编辑器。用户可以根据需要制作精美的信息或文字。这不过是使用 JavaScript 和引导技术的富文本编辑器。
