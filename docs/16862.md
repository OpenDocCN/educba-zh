## Figma 字体介绍

在这篇文章中，我们将学习如何在 Figma 中使用字体。现在使用字体的第一步是知道如何安装字体，为此你应该知道在哪里可以找到它们。因此，首先我们将看看各种网站，在那里我们可以下载字体，然后如何在 Windows 中安装，这将在 Figma 中使用。Figma 有一套强大的工具，可用于根据输出格式进行适当的类型设置，这些输出格式与效果和定制选项一起出现在流行的文本编辑程序中。

### Figma 字体的步骤

以下是下面要提到的步骤

<small>3D 动画、建模、仿真、游戏开发&其他</small>

第一步:让我们从下载字体开始。有许多免费和付费的字体网站，你可以从那里下载字体。一些免费网站有谷歌字体、dafont、urbanfonts、fontspace 等。我们将看到如何从谷歌字体网站下载如图所示。首先，我们将改变过滤器显示无衬线和显示，这将减少搜索结果。我们不想要衬线，它会在字母的顶部产生闪光。此外，我们不希望手写，这将更像草书，而不是编码字体的等宽。

![Figma fonts output 1](img/e33811635c81bd877a0fb4756153b5c9.png)



第二步:然后，我们可以根据需要定制文本，或者从下拉列表中选择句子、字母、段落或数字。

![Figma fonts output 2](img/9994ca9c35d6293342a99561507b7961.png)



第三步:接下来，我们可以使用滑块来改变字体的大小

![Figma fonts output 3](img/4ed7c278c2ac92f89560bc8bf241822c.png)



第四步:然后我们选择了一个名为 Ubuntu 的字体，我们可以看到有多个选项或样式可用。如果您正在查找非标准字符，您还可以看到该字体的字形类型。在“关于”部分，你可以了解字体的历史和作者或设计者的信息。此外，如果你正在寻找用于商业目的的字体，还有一个许可选项，最后一个选项是配对，这是谷歌建议的，如果你想要另一种字体与这种字体相结合。

![Figma fonts output 4](img/117d62f2a64d36151e8df11524bbae50.png)



**第五步:**现在我们已经选择了常规风格。在这里，我们可以选择查看我们选择的字体，如果您在网站上使用，或者为了我们的目的，如果我们想与 Figma 一起使用，我们可以选择“下载全部”。此外，如果您想下载这种特定字体的所有字体样式，可以选择下载系列。

![Figma fonts output 5](img/1b60877054fcf2f474f0eb1f7284fe12.png)



**步骤 6:** 一个 zip 文件将被下载到您的计算机上，右键单击该文件可以将其解压缩，所有字体都将位于解压缩后的文件夹中。此过程将在 windows 上运行。

![Figma fonts output 6](img/16bfe13fea55686f94d881c961f47557.png)



**步骤 7:** 接下来，我们将右键单击所有字体，并选择为所有用户安装。

![Figma fonts output 7](img/ff515ce0846f553c0faad41851812fb3.png)



**步骤 8:** 然后打开 Figma，从预置的空白画布开始。

![Figma fonts output 8](img/baae07e7b64d85326b465cebe3295f73.png)



第九步:你可以从工具栏的中间重命名你的文档

![Figma fonts output 9](img/f370d24ce75d1ca0a7f3c9594ea8d9ad.png)



**第十步:**先来一个边框或者按 F，选择 iPhone 11 Pro 的设计。

![Figma fonts output 10](img/eb44ca290795d6dbe00649af15179ce0.png)



**步骤 11:** 然后使用矩形工具或者按 R，做出如图所示的矩形。

![Figma fonts output 11](img/716cb71c40787886539f4a185be3f9ef.png)



**步骤 12:** 你可以在设计面板中更改填充颜色。

![Figma fonts output 12](img/38a70a196061cbfec6b04f26e60d74eb.png)



步骤 13: 现在使用文字工具或按 T 键点击矩形，这将创建一个文本框，然后添加文字，改变字体填充颜色为白色，大小为 32，字体样式为我们刚刚安装的 Ubuntu。

![Figma fonts output 13](img/de980a900d76d62441c66764b56cfbbc.png)



**步骤 14:** 再次使用文本工具将大小设置为 20，其他选项应与最后一个文本相同，除了填充是黑色的，单击一次并键入这个创建文本框，自动调整属性为宽度，允许文本框的宽度随着文本增长。

![output 14](img/7e31db3f2d469b656d79651c801b601c.png)



**步骤 15:** 接下来，我们单击并拖动一个文本框，创建一个固定大小的文本框，自动调整大小是固定的，一旦文本到达文本框的末尾，该框将不会垂直调整大小。

![output 15](img/59a1fcd0577648d8d4f9d997af50d3cf.png)



**步骤 16:** 因此，在文本部分的设计面板中，我们可以为该文本框设置自动高度，现在我们可以键入更多内容，高度将根据文本进行调整。

![output 16](img/239aeb5d1ef6293aa9501048c274ea45.png)



步骤 17: 接下来，我们创建三个文本字段，使用 Ctrl + B 演示粗体，使用 Ctrl + I 演示斜体，使用 Ctrl + U 演示下划线

![output 17](img/fbe885aa6f5915b8d4a0108eee37b961.png)



**第 18 步:**然后我们调整行高为 140%，默认为 100%，它改变了行间距。因此，通过按住 shift 键并点击向上箭头键，我们可以以 10%的增量增加行高，并将字母间距设置为 2px(默认情况下为百分比),它会改变字符之间的间距。

![output 18](img/741dcf87b21fbf65b08f8bca4f155bfe.png)



**步骤 19:** 创建另一个框架或通过复制粘贴复制现有框架，并保留标题。然后通过拖动创建一个文本框，并放入一些随机文本。通过在句子末尾按回车键来创建段落。

![output 19](img/0f66ae64f7a1aedd60a98df267083d79.png)



步骤 20: 现在我们将段落间距设置为 10，段落缩进设置为 20，这样每个段落的开头都有一些空间。

![output 20](img/02ca0a30a5800798cb2bf14753cdda52.png)



步骤 21: 现在我们已经创建了另外两个段落，首先我们可以将水平文本对齐方式改为左对齐、居中或右对齐。在第二个较大的文本框中，我们可以将垂直文本对齐方式更改为顶部、中间和底部。

![output 21](img/29f65ba3183c7164a2ce377f4befd0aa.png)



接下来我们将看看一些文字效果。首先，我们可以将文本的填充改为红色，如图所示。

![output 22](img/98e58efaff27dc8383d52806cf12effd.png)



第 23 步:接下来我们添加一些文字，填充为灰色。然后我们使用 stroke 属性在外侧添加一个 1pt 的笔画。

![output 23](img/96bbbfd22defd72a2a53d098cbdfb928.png)



最后，我们可以添加阴影效果。我们可以叠加多个阴影来突出效果。

![output 24](img/c15394fdf04f8ac3e1c92166e13cbd80.png)
