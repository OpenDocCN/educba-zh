## InDesign superscript 简介

InDesign superscript 是一种文本样式，我们使用它将任何文本以较小的尺寸放置在其他文本的顶部，这意味着我们通常在编写数学公式或所有这些类型的内容时使用它，但在 InDesign 中，我们将它用于多种用途。该软件中有许多选项，通过这些选项，我们可以在任何文本内容中获得我们想要的结果。InDesign 提供了一些功能，通过这些功能，我们可以为上标创建一种字符样式，只需单击一下就可以使用它。所以让我告诉你这个特性的一些方面，以便你能理解它。

### 如何在 InDesign 中使用上标？

我们对任何字母或整个文本使用上标风格，用于许多不同类型的目的，你可以通过几个简单的步骤学会它。因此，让我们首先创建一个新文档&我将从工具面板中选择文字工具，以便在我的文档区域中包含文本内容。

你可以看到一些文字需要在一个上标的形式，在第一' st '应该像 1 <sup>st</sup> 。这是上标形式。

你总是注意到，在阅读一本书的过程中，在某一行的末尾有一些上标形式的数字，表明该行的细节在该数字的特定部分之下。

![InDesign superscript output 4](img/2c890ced1c0b12fcd26797b8fa4e737f.png)



现在选择你想转换成上标的文本，然后进入用户界面顶部的字符格式控制面板。

如果没有这种类型的面板，您可以转到窗口菜单，并启用该菜单下拉列表中的控制选项。

一旦你点击这个选项，它会变成这样，这是一个上标形式的文本。

![InDesign superscript output 8](img/031180439a5fc68b7aa1d335faeb4189.png)



您也可以在另一个面板中找到此选项。所以打开用户界面右侧的字符面板。

![InDesign superscript output 9](img/ddc6c42a22a16455edcd57ead9d89e9a.png)



您可以在同一个窗口菜单中找到此面板，在“类型和表格”选项中，我们有“字符”选项。

![InDesign superscript output 10](img/2a4a7e3afcb4dd43a6f80b4651666dfd.png)



如果我们点击这个字符面板右上角的按钮，我们会在列表中看到一个上标选项。你可以注意到这个上标有一个快捷键，就是键盘的 Ctrl + Shift + =按钮。

![InDesign superscript output 11](img/10a322f38af26189d4c6990f98eeb156.png)



同样，我会把这个文本也在上标。但是一次又一次这样做似乎太无聊了。所以你可以为此创建一个字符样式。

![InDesign superscript output 12](img/ac75a4b4f0c20b2607889393b0ffac9a.png)



让这个面板转到菜单栏的窗口菜单，并转到这个菜单的下拉列表的样式选项。在这个菜单的新下拉列表中，我们有字符样式选项，或者你可以按 Shift + F11。

![InDesign superscript output 13](img/31eca1cf02b4e6336e49f6b1325f7acf.png)



在这个面板中点击创建新样式按钮。

![InDesign superscript output 14](img/0ab62efc97333bab12450630f32c8049.png)



在这个面板中我们有一个新的字符样式。现在我将双击新图层，一个对话框将会打开。

![InDesign superscript output 15](img/d0e4d2e6e76865dcb335ff9889cad92d.png)



在这里，我将命名这一层上标。

![InDesign superscript output 16](img/2a9491e7276bd9c8669ff87d1e1a6e23.png)



现在点击对话框列表中的基本字符格式选项。

![InDesign superscript output 17](img/d849aacd239ddb250d1fcdc773cb3dd9.png)



点击位置选项的下拉箭头按钮，从列表中选择上标选项，然后点击此对话框的确定按钮。

![InDesign superscript output 18](img/ed4c5f145851bd4e9f6825b092e9758e.png)



现在我们在这个角色面板中有了一个新的角色风格。我将选择我想要上标的文本。

![output 19](img/6be44d9b273e9abafe3f8285d6dac2da.png)



并点击这个上标字符样式层。你可以看到它把选定的文本变成上标形式。

![output 20](img/6723629e7442496a6ec285d8c6abb9ad.png)



现在让我告诉你另一种方法，通过它你可以在几个步骤中把文本的数字变成上标。为此，请转到菜单栏的“编辑”菜单，并单击下拉列表中的“查找/更改”选项。

![output 21](img/f037b8e0bace6120c90e425d43c52f31.png)



一旦你点击它，一个查找/更改对话框将被打开。

![output 22](img/690121996e6ebeb35b22f91b3b65d861.png)



我想把这些类型的文字数字变成上标。因此，我将单击此对话框“查找内容”选项的@符号箭头按钮。我将转到打开列表的符号选项，并从新的符号选项列表中选择“左括号字符”。

![output 23](img/f242b9156697b818c65c61a5197509e1.png)



接下来，我们选择的文本中有一个数字，所以这次我将从同一列表的通配符选项中选择任意数字选项。

![output 24](img/9c35342ff76b8041e066c9452dafb14e.png)



现在再次选择“右括号字符”选项。

![output 25](img/bd247adb1400724ef3caac09496cd072.png)



现在，这将找到我们想要找到的文本语法。

![output 26](img/a5db41ab1c21f2cbf2627e74f0a3d626.png)



单击此对话框的“更改格式”选项的“指定要更改的属性”按钮。

![output 27](img/e6f248e3ec46e83097764b5e0acd001c.png)



在这里点击字符样式选项的卓尔箭头按钮，选择上标样式。如果你已经创建了这样的样式，这个上标样式只在这个列表中。如果你还没有创建过这样的样式，你也可以在这里创建一个新的样式。你只需要点击列表中的“新字符样式”选项。

![output 28](img/cac680bd94a75630af85cb788422b261.png)



现在单击该对话框的“查找下一个”选项。

![output 29](img/5d4118691a7cbc559800bc4e5b74fe9e.png)



它会在您的文档中找到相似的文本，因此单击“更改”按钮。

![output 30](img/7f2bebfb9b44f16eb6da61ebd8defedc.png)



它会将我们选择的文本变成上标形式。

![output 31](img/4bc8513d4b04ead526db0974266a9150.png)



您可以单击“查找下一个”按钮再次执行此操作。

![output 32](img/a05e70c67fee7841c335a667f12abf69.png)



或者，如果您想要更改文档中该样式的整个文本，请单击此对话框中的全部更改按钮。

![output 33](img/f2e9ea1f5e939c5f6aab89ad9ce9ee92.png)



它会显示你的文档中相同样式的文本数量。点击确定按钮。

![output 34](img/22524daa5eae1e41f82eba8965efc046.png)



只需点击一下，它就能把所有相同类型的文字变成上标。

![output 35](img/239c22a0723e572309aa045f091f6937.png)
