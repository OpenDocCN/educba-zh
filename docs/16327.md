## 暗台和灯箱的区别

[Lightroom 是 Adobe 打造的软件](https://www.educba.com/what-is-adobe-lightroom/)程序，主要面向摄影师。它允许用户在不破坏最初设计的情况下创建和完成艺术品。这个软件的奇妙之处在于它可以组织艺术品。我们可以组织所有的艺术品来完成完整的目录，并保存里面的一个。LRCAT 文件。该软件提供的关键字进一步帮助组织数据。像旗帜、星星和颜色这样的命令可以帮助用户根据需求编辑设计。人们通常使用 Photoshop 软件来完成主要的设计工作，而 Lightroom 用于小型编辑任务。将 Lightroom 的数据导出到 Photoshop 软件中相当容易和舒适。

Darktable 也是一款免费的图片编辑软件。这是一个类似 Lightroom 的替代软件。它的特点就像 Lightroom。这是 Adobe 提供的一个开源程序[。在市场上所有可用的软件中编辑任何 RAW 文件都是困难的。有了 Darktable 软件，我们可以编辑来自 400 多台相机的原始文件。像 Lightroom 一样，我们可以在不破坏原始设计的情况下编辑任何图像。我们可以编辑图像并将其还原，而不会降低其质量。因为几乎相似的功能，我们可以使用 Lightroom 和 Dark table 的任何软件。它有一个很长的清单，可以将其数据导出到其他软件中。我们可以将图像直接导出到脸书和 Flickr，并通过电子邮件直接发送给任何人。这两个软件的区别之一是，我们不能用 Lightroom 软件自动完成重复的任务，但用 Darktable，我们可以做到。通过自动化的重复性任务，我们可以加快编辑任何图像的过程。](https://www.educba.com/adobe-editing-softwares/)

### 暗台和灯箱的主要区别

让我们讨论一下暗室和亮室之间的一些主要区别。

*   Lightroom 桌面界面对用户友好。它的界面就像 Photoshop。Lightroom 使用 7 个标签的名称作为模块。所有主要工作将在“图库”模块中完成，而另一项编辑任务将在“开发”模块中完成。所有的功能都被很好地放置在界面中，这使得在整个软件中导航很容易。Darktable 界面几乎和 Lightroom 一样。
*   Darktable 有很多工具可以使用，但是所有的工具都是集群的，这对于第一次使用的用户来说很困难。所有的特征，比如颜色等等。，没有组织好。这个软件的导入功能很快。
*   Lightroom 在组织工作方面是其他任何软件都无法比拟的。我们可以使用文件夹、智能收藏和许多其他方式来排列图像。该软件支持的关键字也有助于用户在该软件中方便地搜索图像。在导入任何图像的过程中，都可以使用关键字。这将节省大量的时间和工作。这些关键字可以在使用后添加到库模块中。Lightroom 不仅提供关键字建议，还允许用户从关键字集中进行选择。
*   在暗表软件中，关键字被称为“标记”。如果用户更频繁地使用 Lightroom 软件，他/她在使用关键词时会感到困难。我们在标签区使用标签。我们可以给图像附加标签，而不是添加标签。为此，双击图像直接在图像标记区域添加标记。这个软件没有一个明显的方法来找到带有标签的图像。我们找不到任何可以直接指向所选图像的关键字列表。
*   Lightroom 处理时间非常短。我们可以根据需要使用对比度、曝光和色调曲线滑块来调整图像。我们可以在图像周围使用各种颜色。白色的平衡以及 HSL(色调饱和发光)区域的使用提供了良好的对比图像。我们可以在“开发”模块中找到所有这些功能。
*   该模块还允许用户调整功能。我们可以在特征名称之前或之后使用 X 和 Y 模式，通过它们我们可以一起比较图像。它的功能创建了一个虚拟复制选项，允许我们复制用户的图像。暗台的处理几乎和 Lightroom 一样。只有当用户熟悉任何软件时，他们才会体验到处理速度。
*   与 Darktable 相比，在 Lightroom 软件中编辑任何图像的颜色都更好。在 Lightroom 软件中，图像质量的清晰度和对比度也很好。用暗桌软件制作的图像不那么强烈，也不那么饱和。
*   在 Lightroom 的帮助下产生的黑白图像比黑暗的桌子更有力量。它有更多的对比。
*   随着 Creative Cloud 的推出，Lightroom 软件可以在线下载，每月收费。

### 暗台与亮室对比表

让我们来看看黑暗之桌和光明之屋之间的比较。

| **暗表** | **灯箱** |
| Darktable 软件对其用户是免费的。 | Lightroom 软件在市场上不是免费的。 |
| 它的用户界面对用户不太友好。用户必须搜索命令来编辑任何图像。 | 在 Lightroom 中编辑任何图像都很容易，因为用户界面对用户非常友好。 |
| 用这个软件生成的图像模糊不清。 | 用这个软件生成的图像对比度更高，清晰度更好。 |
| 这是跨平台软件，用户可以在其中更改他们的软件。它不仅适用于基于 Linux 的软件，也适用于其他开源程序。 | 对软件做一些修改是非常困难的。 |
| 在黑暗中做重复性的工作对用户来说是容易和舒适的。 | Lightroom 通常用于图像编辑的新任务。 |
| 在黑表中使用关键字很困难。 | 在这个软件中制作关键字既简单又容易。 |
| 暗盘使用起来有些复杂。 | 这个软件很容易使用，因为命令很容易使用。 |

### 结论

这两款软件都允许用户编辑图像，各有优缺点。人们普遍喜欢对用户更友好的软件。暗桌软件免费提供给它的用户，这让更多的人使用这个软件，增加它的市场份额。Adobe 的 Lightroom 生成了对比度更高的图像。
