## VMware 替代产品简介

虚拟机(VM)是模拟软件环境的计算机系统。它使真正的计算机的替代品变得更简单。它们模拟机器架构，具有与物理计算机相同的功能。以下是 Windows、Mac 和 Linux 顶级虚拟机应用程序的精选列表。该列表包括开源软件(免费)以及商业软件(付费)。

### VMware 替代产品列表

让我们看一下 VMware 的替代产品。

<small>网页开发、编程语言、软件测试&其他</small>

#### 1.Oracle VM VirtualBox

来自 Oracle 的 Virtualbox VM 是另一个很酷的免费虚拟机程序，它是为 VMware 工作站替代方案而设计的。对于初学者来说，这是一个完美的选择。VirtualBox 安装 Windows、OS X、Solaris 和 Linux，用于实验室测试和桌面运行。这个免费软件允许从 ISO 或进口创建新的虚拟机，而不是虚拟机播放器。大多数操作系统作为虚拟机运行在 32 位和 64 位上。

#### 2.VMware 工作站

VMware Workstation 是一个全面且经过认证的虚拟机计划。这个软件是为需要几个系统操作的高级用户设计的。例如，对于同一个虚拟空间中需要数据库、web 和应用程序服务器的开发人员来说，它工作得很好。它还支持保存和克隆项目。VMware Workstation 还使用 API 与 vSphere 进行通信。从我的经验来看，这是一个多功能的解决方案，可以在桌面上管理虚拟机，有太多的功能值得一提。

#### 3.XenServer

XenServer 是 Citrix 收购的一款常见的虚拟机开源软件。但是 Citrix 后来犯了一个错误，过于努力地利用 XenSource，失去了许多主要支持者。此后，Citrix 发现了这个错误，并重新开放了源代码。XenServer 是免费的，但是也可以购买完全支持的版本。在许多商业环境中，这是托管服务器和桌面操作系统的通用方法。它几乎可以做 vSphere 与 XenCenter 一起可以做的所有事情，例如实时迁移、HA 以及将虚拟机导入和导出到云中。

#### 4 .千伏计（kilovoltmeter 的缩写）

KVM 被称为内核驱动的虚拟机。这是第二好的选择。KVM 是一个完整的 Linux 虚拟化解决方案，已经在大多数 Linux 发行版中可用。与 VMware 或 XenServer 相比，它更适用于 Linux，并与 OpenStack 完美融合。免费或完全支持的 RedHat 版本都有，好东西都说完了。但是 KVM 对于 Windows 虚拟化有局限性，就像 Hyper-V 对于 Linux 一样。

#### 5.VMware ESXi

所有这些以及更多工作都由 VMware ESXi 完成。在我看来，VMware ESXi 是大型公司在 Windows 或 Linux 上部署虚拟服务器的最佳解决方案。我要提醒您的是，一开始这并不是一个便宜的解决方案，但是 VMware 有一个免费的 ESXi 版本可以帮您搞定。不要对免费版本期望过高，因为最重要的功能集中在 vCenter、共享存储集成、HA、vMotion、DSR 和 storage DRS 许可证上。自 20 世纪 90 年代以来，VMware 一直在发展壮大。它最初是一个安装在 Windows 工作站上的笨拙的小程序。然后，它修改为 Windows 服务器进入 GSX 从那里。它也被转换回 ESX，消除了 Windows 的需要，并引导到一个内核红帽。在版本 4 之后，它变成了 ESXi，现在运行在定制的 Linux 内核上。

#### 6.超 V

Hyper-V，历史上被称为 Windows 服务器的虚拟化。它是一个虚拟机管理程序，旨在 x86-64 系统虚拟机上构建。Hyper-V 服务器设备可以被设置为将各个虚拟机暴露给一个或多个网络。Hyper 提供了许多 Linux 版本，也帮助运行多台机器。

#### 7.Red Hat 虚拟化

Red Hat 是一个软件定义的开源平台。该虚拟机可帮助您管理和供应新的虚拟机，克隆现有的虚拟机，并了解它们如何协同工作。它还包括针对 Linux 和 Microsoft Windows 操作系统的虚拟化。

#### 8.VirtualBox 便携式

VirtualBox Portable 是一个应用程序包。这是一个开放的免费工具，允许你从一个 USB 驱动器运行任何操作系统，不需要单独安装操作系统。VirtualBox 路由配置被修改并解包。它通过虚拟网络适配器提供网络支持。使用可移植 VirtualBox 的主要优点是它是一个可移植的程序。这种便携式软件允许开发人员和 IT 专家在任何其他操作系统上控制任何操作系统，如 u 盘或任何其他外部硬盘。
