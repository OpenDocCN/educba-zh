## Illustrator 中的平滑工具概述

Illustrator 中的平滑工具是用户广泛使用的工具之一。平滑工具用于使通常出现在基于矢量的程序中的艺术品和对象的弯曲和粗糙边缘平滑和有光泽。平滑工具还帮助用户利用可用于编辑对象的属性和功能来控制需要添加到对象的平滑量。使用这个平滑工具，用户可以使铅笔草图和图纸看起来更平滑，更有光泽，没有任何粗糙的边缘。

### 如何使用 Illustrator 中的平滑工具？

下面是把直线变成美丽曲线的步骤。

<small>3D 动画、建模、仿真、游戏开发&其他</small>

**第一步:**打开一个新文件。

![Open New File](img/38c79956deb73a054b62c3284a8c8429.png)



**第二步:**进入工具，选择铅笔工具；我们也可以使用 **N** 作为铅笔工具的快捷键。

![Pencil Tool](img/0a966f10834040335a00565a9f7973bd.png)



**第三步:**在这里，我将借助铅笔工具绘制线条。

![Draw Line with PT](img/cd58dc9d89a3013dfcbe952bc7dd3173.png)



我们画的线是如此不均匀，形状和边缘都不好，所以我们要清除粗糙的边缘，并在平滑工具的帮助下制作这个好的形状。

**第四步:**画好线条后，我们可以在线条中找到一些点。

![Points in the Lines](img/c6dcba28d4105f3ff70616ad82bc7873.png)



**第五步:**在这里，我们有另一个选项，借助直接选择工具获得精细的边缘；这个工具的快捷方式是 **A** 。

![Fine Edges](img/2d4224530bb1143cb8281c207bdb64d6.png)



通过使用这个工具，我们可以移动和调整点，以获得良好的边缘

![Fine Edges 2](img/6a420b4e9e4f75670c2e62208df64cd5.png)



**第 6 步:**我们不能说平滑工具和直接选择工具是一样的，因为我们可以用直接选择工具调整边缘平滑，但我们不能平滑线条和粗糙的边缘。在这个过程中，我们需要使用平滑工具，所以选择平滑工具，转到工具并按住铅笔工具，我们可以得到选项，我们可以选择平滑工具。

![Process of Smooth Tool](img/fab4e43b03ef171f94ddda11992a8b82.png)



**第七步:**我们有另一个选项来选择平滑工具，从工具栏中选择一个笔刷工具，按住 **alt，**然后它就作为一个平滑工具，直到我们完成平滑。我们需要按住 ALT 键，一旦我们释放 ALT 键，它就变成了笔刷工具。

![Brush Tool](img/715de1f5d41f504788f962ae665727e2.png)



现在按住 ALT，然后它将作为一个平滑工具。

![ALT - Smooth Tool](img/26ac6fc0839baa25d790a75c276242d1.png)



**第八步:**我们来看看如何使用平滑工具。只要按住点击并在粗糙的边缘上画线，然后我们就可以找到平滑的线，继续这样直到我们得到平滑的边缘。

![Smooth Edges](img/3e2a7eee2b91e951987d8b1dc8d04abe.png)



![Smooth Edges - 2](img/b3067b6938a46ae7cafe290ba202cc86.png)



在起跑线上，我们可以找到不同之处。

步骤 9: 在这里，我正在绘制顶部边缘，以获得一个平滑和精细的边缘，我继续绘制这个过程，直到我们获得正确的边缘。

![Smooth and Fine Edge](img/e64f1751a87f365548f1a91c568ab826.png)



每当我们使用平滑工具绘制线条时，我们可以找到调整位置并使边缘平滑的锚点，一些新的锚点被创建，一些点在我们使用平滑工具绘制时被删除。

![Adjusting the Positions](img/d751c632dd9d4be25955fe89ec3ae94d.png)



**步骤 10:** 在这里，我对整个线条连续画出平滑的线条，直到我得到平滑的好的形状；我们可以看到两幅图像的不同之处。

![Draw Smooth Line](img/c4b5adaeb7fba484a8796b2b1dfe8655.png)



上面的图像是我们首先绘制的原始线条，下面的图像是应用平滑工具后的。

![Applying a Smooth Tool](img/76266394dc284e3106e402a23e5f821a.png)



仔细看两幅图像，我们可以发现应用平滑工具后的差异；我们得到了平滑和直线的圆角边缘，在使用平滑边缘之前，我们有粗糙的边缘和角不均匀的不均匀线条，而使用平滑工具绘制时，锚点调整并给我们平滑的线条，精细的边缘和每条曲线都是如此平滑和均匀。

**第 11 步:**让我们试着借助钢笔工具画一个签名。

![Drawing Signature](img/dfdb8b9dbf681326b1501e6cb52121e5.png)



步骤 12: 我们得到了不想要的曲线，错过了中间的形状，所以选择平滑工具来学习这个。

![Select the Smooth Tool](img/8df8048b741260ef004ea3769a194471.png)



**步骤 13:** 缩放图像，找到曲线。

![Zoom the Image](img/751b5e053e21a5254b58e87a8e15b93e.png)



**步骤 14:** 开始在每条路径和曲线上进行绘制，得到平滑。

![Drawing on Each Path](img/5bab3b89616c6b47609162a42f06b39d.png)



**第 15 步:**继续画，直到我们得到完整的签名，看起来清晰好看。

![Drawing on Each Path - 2](img/38e6093e3241c3f77e7eb9001bb1b3aa.png)



到目前为止，我们现在尝试绘画让我们看看它如何与钢笔工具；当我们使用钢笔工具创建一些图形或字符时，我们会忽略其间的形状，在完成绘图后，我们选择平滑工具，我们可以清楚地调整形状，也可以在平滑工具的帮助下重新连接或断开形状。

让我们看看它是如何工作的。

**步骤 16:** 选择工具栏中的钢笔工具；钢笔工具的快捷键是 **P.**

![Selecting Pen Tool - Smooth Tool in Illustrator](img/ffb8bc9a1192fa12f1cf0598a8082156.png)



步骤 17: 用钢笔工具画一个形状，希望如果你有空闲的手，你可以画得很好，我画了一个粗略的形状，只是为了实验。

![Shape - Pen Tool - Smooth Tool in Illustrator](img/8523fa36415b5a416d209f487661cc06.png)



这是一个有很多曲线和凹凸不平的粗糙形状，我们将使用平滑工具来改善这个形状。在这里，我用平滑工具绘制；形状已经画了一半；让我们看看区别。

**步骤 18:** 选择平滑工具。

![Select Smooth Tool - Smooth Tool in Illustrator](img/74d329be0572d3479404d0a33a8242a6.png)



**步骤 19:** 这里，我用平滑工具绘制；形状已经画了一半；让我们看看区别。

![Half of the Shape - Smooth Tool in Illustrator](img/f7dc71a72d12c20181afc03adc31a879.png)



![Smooth Tool Half 2 - Smooth Tool in Illustrator](img/3446e1979058396bd563f0fd418fb54d.png)



我们可以发现我应用了平滑工具的底部形状的不同，调整了关键点，做了一些平滑和精细的边缘；第一幅图像中的锐边经过了调整，我们可以在第二幅图像中看到不同之处。

**步骤 20:** 这里，我们现在完成了整个形状的平滑；让我们看看区别。

![Smoothing Whole Shape - Smooth Tool in Illustrator](img/e7b50f342c0d456f26c23df36116593e.png)



![Smoothing Whole Shape 2 - Smooth Tool in Illustrator](img/55a9636b5a154fbafc4e4060f2686c67.png)



查看平滑前后两个图像的差异；光滑后可以看到很多变化，现在每个锋利的边缘都变得光滑了，结构看起来很好。

这样的魔术可以在一个容易使用的平滑工具的帮助下完成。

### 结论

在这个过程中，我们使用铅笔工具绘制一个形状，并在平滑工具的帮助下使形状平滑，我们尝试平滑使用铅笔工具绘制的签名，最后，我们看到如何使用平滑工具使钢笔工具形状平滑。
