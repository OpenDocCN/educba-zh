## 3ds Max 界面简介

3ds Max 是一款 3d 建模软件，被许多 3d 建模设计专业人士使用。这个软件有一个非常用户友好的环境。在本文中，我们将以非常简单的步骤讨论 3d max 软件的用户界面，并了解该软件用户屏幕的各个部分。一旦你熟悉了这个软件的用户界面，你就可以毫无问题地成为 3d 建模设计专家。

### 如何在 3ds 中创建 Max 界面？

在这个软件的用户界面上，我们有许多不同类型的部分。我们将以一种非常有趣的方式逐一讨论它们。因此，让我们开始讨论这一背景。

<small>3D 动画、建模、仿真、游戏开发&其他</small>

现在让我们从该软件工作屏幕的顶部开始。在顶部，我们有两条丝带，分别是:

*   **菜单栏**:在工作屏幕的顶部，我们有一个功能区，它有许多不同的菜单，可以在这个软件中进行不同类型的调整。

![Menu Bar](img/a3b202321c8ea05f509d45db0ba2d56a.png)



*   **工具栏:**在这个软件的菜单栏下面。在工具栏中，我们有一些重要的工具，可以帮助我们在这个软件中制作不同类型的模型。

![Toolbar](img/ab21967e720fe8af07fc9512519b5c63.png)



**步骤 1:** 在此之下，我们将工作窗口区域分为四个视窗。视窗是指工作窗口的一部分，你可以从不同的角度看到你的对象。我们在这个软件的工作区域中有四个视口:顶视口，你可以从你的对象的顶部看到你的对象；前视口，你可以从你的对象的前面看到你的对象；左视口，你可以从你的对象的左侧看到你的对象；透视视口，你可以看到那个对象的 3d 模型。

![working window area](img/f5ecc39d29fdca34c0073a96d3e156d1.png)



**步骤 2:** 您可以在工作的任何时间看到任何视窗的完整视图。要查看完整视图，请转到您想要查看完整视图的视窗，然后按下键盘上的 Alt + W 按钮来查看完整视图。我将转到“透视”视口，然后按 Alt + W 按钮查看“透视”视口的完整视图。如果想回到原来的视口，那么再次按 Alt + W 键。

![Perspective viewport](img/8c86f23f40dead069f058084df1e2ad3.png)



第三步:在这个软件的工作区的右边，我们有一个公共面板，在这个面板里我们有一些标签用来创建模型，我们设计中的角色。

![Common Panel](img/c5f4406f48562b048f4f4efafb500463.png)



第四步:我们必须在这个软件的公共面板中创建一个选项卡。在创建选项卡中，我们有不同的命令选项卡，用于创建二维形状，三维形状，如盒，圆柱，线，圆等。

![3d shape](img/a5f58b09b2c0bc4d5f1617c9f7faab9c.png)



**步骤 5:** 在“创建”选项卡的“几何图形”列表中，我们有一些形状选项。我们可以使用这个列表在这个软件中制作一些不同的 2d 和 3d 形状。

![Geometry tab](img/cd0426338b59e0b71fb197a495787428.png)



**步骤 6:** 在创建选项卡旁边，我们有修改选项卡，用于在任何模型的绘制的 2d 或 3d 形状中进行任何修改。

![Modify tab](img/6e2230390e8c6627933b59aa190010dc.png)



**第七步:**在这个软件的工作区底端，我们有一个时间线部分，用于 3d 建模设计的模型动画制作。在时间线部分，我们有一些处理选项卡，用于在该软件中制作模型动画。

![Timeline section](img/70ee95b685307a80dfcd4d44e9cd9fad.png)



**第八步:**在下端右侧，我们有一些处理我们工作的导航工具，比如缩放全部按钮，缩放到扩展等。

![Zoom to Extend](img/2a84fae0663667b87b5501be948526e5.png)



**第九步:**您可以更改该软件用户界面工作屏幕的主题。要更改屏幕主题，请转到菜单栏的自定义菜单，它位于该软件工作屏幕的顶部。将会打开一个下拉列表。单击下拉列表中的“自定义用户界面和默认 s 切换器”选项。

![3ds Max Interface - 11](img/b36d04692e57d8d72b8f2599d651d0ad.png)



**步骤 10:** 设置用户屏幕主题的对话框将打开。您可以从 UI 方案选项中选择主题。

![3ds Max Interface - 12](img/c2b9b19e80c3741e5755173c2db25f0a.png)



**步骤 11:** 我们有三个选项:默认 UI，亮模式，暗模式。你可以选择其中任何一个。

![3ds Max Interface - 13](img/8482c1eb9f956dab6835f8874795f11b.png)



**第 12 步:**如果你想要一个灯光主题，然后点击它，显示这种类型的对话框需要几分钟来调整新屏幕的设置。

![3ds Max Interface - 14](img/fdc6ead296eefabbb6f6a76cbcba63a7.png)



第 13 步:它还会显示一条信息，提示您下次启动软件时，该屏幕将成为您的默认屏幕。完成后，单击此对话框的设置按钮应用设置。

![3ds Max Interface - 15](img/760f8de9a1b2a1d4d50d1e2518eff431.png)



**步骤 14:** 现在，再次点击该下拉列表中的偏好选项。

![3ds Max Interface - 16](img/5e9ef8966604fc88efb256fdb33ff4ed.png)



**步骤 15:** 一个偏好设置框将被打开。现在启用此框的“将本地文件路径转换为相对路径”选项。您必须启用此选项，因为当您在其他软件中的任何地方打开此软件的文件时，此软件中使用的所有材料也将在该软件中相同，然后按下此对话框的“确定”按钮。

![3ds Max Interface - 17](img/57d108b6eebc6d216aeb2cc3e95d221c.png)



第 16 步:我们可以使用文件菜单打开一个新文件或者保存我们在这个软件中的工作。

![3ds Max Interface - 18](img/fa29b69bf749e9e1a918d1fda1856b4e.png)



**第十七步:**在文件菜单中，我们有导入导出选项，可以导入导出任何文件。

![3ds Max Interface - 19](img/308910d3a6c424e95230140c1f862590.png)



**步骤 18:** 你可以拥有任何工具的单独面板。对于任何工具的单独面板，右键单击菜单栏的自定义菜单。将会打开一个下拉列表。单击要分离面板的任何工具。我将单击捕捉工具。

![3ds Max Interface - 20](img/2c2bb72834c1065f6860ff09793de7f5.png)



**步骤 19:** 一旦你点击这个工具，这个工具的一个单独的面板就会出现在你的工作屏幕上。您可以在工作屏幕上的任何地方调整它。

![3ds Max Interface - 21](img/9cf1be37dbc86d821bd816f07b504020.png)



**步骤 20:** 你可以通过点击同一个列表中的一些工具选项来关闭这个面板。

![3ds Max Interface - 22](img/6b80a65427fbebc21c3c88b1b948c1af.png)



### 结论

现在看完这篇文章，你就可以了解这个软件的用户界面，按照你的处理。您可以通过对该软件的选项进行一些更改来制作自己的工作屏幕。经过练习，你可以成为使用这个软件的专家。
