## 什么是 InDesign Extension？

InDesign extension 可以理解为帮助我们增强 InDesign 功能的编程软件。在任何特定依恋的帮助下，我们可以比传统上花费更多时间更直接地完成特定任务。在该软件中，您可以从 Adobe 网站下载 Adobe 扩展来使用它们。我们在 Adobe Photoshop 和 Adobe Illustrator 中使用的几乎每个扩展都可以与 Adobe InDesign 一起工作。今天在这篇文章中，我将告诉你如何安装它们，如果你不想使用它们，你可以删除或卸载它们。

### 何时使用 InDesign 文件扩展名？

在了解了 InDesign 扩展之后，您现在想知道我们为什么在这个软件中使用该扩展。原因是在开发的时候，特定软件的开发者将所有的特性都嵌入到当时最好的软件中。尽管如此，经过一段时间后，我们需要改进它们来快速有效地完成特定的任务，所以使用扩展。

<small>3D 动画、建模、仿真、游戏开发&其他</small>

为了满足 InDesign 的这些要求，我们也需要这些类型的改进。因此，我们对该软件进行了扩展，并安装了它们，以便在我们的工作中发挥它们的优势。现在，我将告诉您如何在 InDesign 中安装所需的扩展并开始使用它们。

### 如何安装 InDesign Extension？

我们可以出于多种目的在 InDesign 中安装扩展，要安装您想要的扩展类型，您需要访问 adobe 的官方网站。在这里你必须和 exchange.adobe.com 一起去。

![Install InDesign Extension 1](img/73b224b4d4a3c14319d90c2e52f9a3b9.png)



一旦你搜索它，你就会看到这个 Adobe 页面。转到本页的“创意云”标签。

![Install InDesign Extension 2](img/c1cdcdc4a2a6e425540b3c91d1708391.png)



现在向下滚动到此页面，并从产品列表中选择 InDesign 选项。一旦您决定使用 InDesign，它会向您显示所有与其兼容的扩展。

![Install InDesign Extension 3](img/43c62d07f5075c48dc997cd8fcfeae0f.png)



或者，您可以打开 InDesign 软件，转到“窗口”菜单。您可以在下拉列表中找到“在 Exchange 上查找扩展名”选项，所以请单击它。

![Install InDesign Extension 4](img/8f11f7726c0d94ecbadc14c21f5b01d5.png)



单击该选项后，InDesign 会自动将您带到 Adobe for InDesign 的扩展页面。

![Install InDesign Extension 5](img/f3f41f441a0d23353ba354eb03358db1.png)



这里从列表中选择您想要的扩展名。我会选择这个。你可以看到它是免费的。所以有很多免费的扩展，也有根据你是否适合和任何人去而付费的。

![Install InDesign Extension 6](img/fcef617cec9e4bf1cf924efce16895a4.png)



如果您没有登录，它会要求您登录。

![Install InDesign Extension 7](img/1646b2f65a8ac011474fc7e51ac5d02b.png)



之后，点击免费按钮，使其可以安装。它让你接受条款和条件，所以接受它。

![Install InDesign Extension 8](img/45abd511665be55f868c65e117b3d470.png)



你会看到一个安装按钮，所以点击它。

![Install InDesign Extension 9](img/7cb0765f267e6ef9b16d6299d9f66033.png)



如果您遇到任何问题，请单击此页面上的安装帮助选项。

![Install InDesign Extension 10](img/4374d3d9449e554089bb0fd3ddd68846.png)



一旦点击帮助选项，它会带你到这个页面。在这里你可以直接下载选择的扩展。因此，通过点击下载按钮，你会有一个安装文件。然后，您可以按照下面的说明正确安装您想要的扩展。

有了所选扩展名的 raw 文件后，借助 adobe 的 Creative Cloud 桌面 app 进行安装。

![Creative Cloud desktop 11](img/1a091efc9dab205aad1f235ddfdd43fd.png)



如果你在安装中仍然遇到问题，你可以向下滚动这个下载页面，你会看到 Anastasiy 的扩展管理器的下载链接。所以下载吧。

![Install InDesign Extension 12](img/03d4f9e726fdc97f4c2637ae9f66e87c.png)



现在它将带你到这个扩展管理器的下载页面。点击下载按钮。

![Download button 13](img/b6f7eab7ac5b4f02b5f4fc9a1db95632.png)



下载完成后，安装它。

![Install 14](img/f563713249167a39714d0a92d6a878d6.png)



一旦你安装了它，它就会像这样打开。在这个对话框的左上角有一个安装标签，所以点击它。

![Install tab 15](img/1daf3f7597a2a9971f1960c77c16c520.png)



一旦你点击安装按钮，一个对话框将再次打开，选择你想要的扩展名的安装文件。我将选择这个文件。

![Choose the file 16](img/b0cdffc875374813ddf122a41ad96371.png)



一旦扩展管理器完成安装，它将向您显示安装成功的消息。

![Install InDesign Extension 17](img/bb9c3c019186cfce391d17128b23d34e.png)



您安装的扩展可能位于 InDesign 软件中的不同位置，因此请返回所选扩展的安装页面。向下滚动时，您会发现一个标题“在哪里可以找到它”,单击它可以阅读其内容，在这里您可以阅读该扩展在 InDesign 软件中的位置。

![Install InDesign Extension 18](img/1f0ec37f6b36c3ca010b4e67310b0601.png)



如果您已经打开了 InDesign 软件，请重新启动它以查看更改并转到“窗口”菜单。在它的下拉列表中，我们有扩展选项。在它的子下拉列表中，你会找到你安装的扩展。

![Install InDesign Extension 19](img/8c0da86e7d2ba2663030cb2aafc0f46a.png)



### InDesign 扩展移除

有些扩展对我们来说已经过时了，或者出于其他原因，我们需要卸载它们或者从软件中删除它们。所以我要告诉你如何做到这一点。

如果通过 InDesign creative cloud 应用程序安装了扩展，请打开该应用程序，或者打开扩展管理器。一旦你打开它们中的任何一个，你就会看到一个已安装的扩展列表。选择要删除的内容。例如，在这里，我目前只安装了一个扩展。因此，我将通过单击来选择它。

![InDesign Extension Removing 1](img/99e251eb282d6fd40a9bf39f14a043de.png)



选择所需的选项后，单击此对话框中的“删除”按钮。

![Remove button 2](img/204cc514b9dd916fad4a012263987e31.png)



卸载您选择的扩展需要一些时间。

![InDesign Extension Removing 3](img/ea24f5fd3524159c24d381bf721a3172.png)



一旦完成，它会向您显示这样的确认消息。

![Confirmation message 4](img/f3fa72b62583766b7ad8ebc8984f7ff2.png)



如果您在 InDesign 软件中重新检查它，它可能仍会显示已删除的扩展在已安装的扩展列表中。这是因为您必须重启软件才能看到更改。

![InDesign Extension Removing 5](img/00bfbfcdd4c37ae71fe262afd1f4acff.png)



现在，重启我的软件后，没有安装扩展。

![InDesign Extension Removing 6](img/b4e6177b12ab960fc206093f4a05f1f8.png)
