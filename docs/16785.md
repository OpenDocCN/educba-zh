## 亲和照片简介

Affinity Photo 是与 Affinity Publisher 和 Affinity Designer 一起为 iOS、Mac OS 和 Microsoft Windows 开发的光栅图形软件。它是 2009 年为 Mac OS 开发的光栅图形编辑器，第一个版本于 2015 年发布供公众使用。后来，Serif 为 Windows 推出了这个版本。该软件主要关注色彩校正、raw 处理、图像拼接、黑点补偿、alpha 合成和光学像差校正。

#### 1.菜单栏快捷方式

*   **新建文档(Ctrl + N):** 按 Ctrl + N 在您的画布区域创建一个新文档。
*   **打开文档(Ctrl + O):** 按 Ctrl + O 打开您保存的文档。
*   **保存文档(Ctrl + S):** 按 Ctrl + S 将任何文档保存到您想要的位置。
