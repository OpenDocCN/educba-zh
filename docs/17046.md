## Illustrator 重复动作简介

illustrator 中的重复动作是一个非常好的功能，通过它，您可以记录在 Illustrator 中制作任何特定插图的步骤，并在其他对象上使用记录的动作，只需单击或按下一个按钮即可完成相同的工作。我们将了解预设动作按钮，并创建一个动作按钮，通过使用该软件的动作选项的不同参数来记录创建插图的每个步骤。因此，为了理解 illustrator 的这个主题，让我们以一种非常令人兴奋的学习方式来讨论它。

### 如何在 illustrator 中创建重复动作？

我们可以在这个软件中创建一些操作按钮来简化我们的工作，您也可以通过为不同的插图创建操作按钮来节省时间，但在开始之前，让我们看一下这个软件的工作屏幕，以便我们可以更好地理解这个主题。

在工作屏幕的顶部，有一个菜单栏，其中有许多菜单，如文件、编辑、对象和其他一些重要的菜单，使我们在这个软件中的工作变得容易。在这下面，我们有三个部分，在左侧，我们有工具面板；在中心，我们有一个文档窗口，你可以看到你的工作；在右侧，我们有一些标签，如层，颜色，属性，和其他一些。您可以在工作屏幕的任何地方根据您的需要设置所有这些部分。

![Illustrator Repeat Action output 1](img/f4703a855a1821ea0c9968908c59b082.png)



为了学习这个主题，让我们有一个新的文件。对于菜单文档，单击该软件欢迎屏幕上的创建按钮，或单击新文档的文件菜单。单击“创建”按钮后，将会打开一个新文档对话框。在那里为你的文档做你想要的设置，然后点击创建按钮。

![Illustrator Repeat Action output 2](img/a3d13db346a5a7190748841954617b0b.png)



现在转到菜单栏的窗口菜单，点击它。接下来，单击操作对话框下拉列表中的操作选项。

![Illustrator Repeat Action output 3](img/df967d007a2ac9ec84e6d080595c75f6.png)



您会发现这种类型的操作对话框。我们在这个盒子里有一些动作预置，你可以用它来节省你的工作时间。

![Illustrator Repeat Action output 4](img/80de8824f38c6f311bf8e6b9f10d3c0d.png)



让我们通过使用列表中的一个预置来理解动作预置。从工具面板中选择矩形工具，像这样画一个任意大小的矩形。

![Illustrator Repeat Action output 5](img/2a9adcd7cd14ab15a7a9da0a2c0c8a08.png)



再次制作另一个矩形，像这样覆盖第一个矩形。现在选择“不透明度 60”动作按钮，点击动作对话框的播放按钮。

![Illustrator Repeat Action output 6](img/88ad5da28c660bf28ec9cc5c81cb11c5.png)



一旦你点击播放按钮，它会将所选对象的不透明度降低到 60%。

![Illustrator Repeat Action output 7](img/53a63cdfa5dc69c67dd3f18b1471cb2c.png)



您也可以使用此对话框的按钮模式选项。点击此对话框的选项按钮，并从列表中选择按钮模式选项。

![Illustrator Repeat Action output 8](img/4be428f273a6306ef098dcb5de9abc9e.png)



现在它会转换成这样的按钮。

![Illustrator Repeat Action output 9](img/c3e505cb7723300fe32c5d9cc24a9010.png)



现在只需点击'不透明度 60 '的行动按钮，它会把你选择的对象不透明度转换为 60%，只需点击一下。

![Illustrator Repeat Action output 10](img/c59af93baa53879f283bfff30486a2e2.png)



现在让我们创建自己的动作按钮。单击此对话框的“创建新集合”按钮，它将作为您创建的操作的文件夹。

![Illustrator Repeat Action output 11](img/ca8e6593d0d274be516135294b7027df.png)



根据您的选择命名您的动作按钮。我将把它命名为“重复动作”。

![Illustrator Repeat Action output 12](img/f884b24d68c560fd2999b4a13efe08ed.png)



“重复动作”文件夹将创建一个动作对话框。现在像这样做一个长方形。

![Illustrator Repeat Action output 13](img/2046264fb9c35c64b108a6accf463382.png)



现在，单击此对话框的“创建新操作”按钮来创建您的操作。

![Illustrator Repeat Action output 14](img/5225135294902bd80bf8ecc5e73f2b10.png)



根据你的选择命名你的行动。我将命名为重复形状。

![Illustrator Repeat Action output 15](img/8b5ef8cd027c588e010a88e2bad50a8f.png)



您可以为您的操作指定任何快捷键。从列表中选择任意功能键。

![Illustrator Repeat Action output 16](img/9cf923d442bbc91bdfcdf35520b7c757.png)



如果您愿意，让我们选择 F2 并启用除此功能键之外的 Shift 按钮。

![Illustrator Repeat Action output 17](img/3e9c02445f0247dbaf0022b4f99b992c.png)



从列表中选择动作按钮的颜色。例如，我将选择黄色作为我的动作按钮颜色。

![output 18](img/18b16633de7685468be470900ee494b8.png)



现在点击记录按钮开始记录你的行动步骤。

![output 19](img/00ea6abf2a89e99247b4cfea0c0e9334.png)



一旦开始录制，您可以在操作对话框按钮处看到这个红色的录制符号。

![output 20](img/9127ec51b9abd180d53c0c0a6a167667.png)



现在开始你的插图，它会记录。首先，转到菜单栏的对象菜单并点击它。接下来，进入转换选项，点击新列表中的“转换每个”选项。

![output 21](img/dc7608e346f2a02a1cdf9384ad0ab40a.png)



我希望这个矩形的大小增加 200%。因此，在变换每个框时，将水平和垂直选项的值设为 200%。打开预览按钮，这样您就可以看到放大后的预览。

![output 22](img/d1e39e558d5dda4b7c3019e1ef05b3e7.png)



现在指定要偏移该矩形的距离。例如，我将给出 400 px 作为水平方向的距离。

![output 23](img/65ef95c19f9c5a2331d4032c510c5aac.png)



现在点击这个对话框的复制按钮，因为我想复制这个矩形增加尺寸。

![output 24](img/0406dec364f2d7854e6904277cce1062.png)



您的步骤将记录在这样的动作对话框中。

![output 25](img/3428be37fe42580d2a3c8d522e48f86c.png)



现在，如果您想再次将这个矩形变大，请按下键盘上的 Ctrl + D 按钮。

![output 26](img/13e93bba44dcf250b4e14931aac8d22d.png)



如果你移动这些物体，那么它也会记录这些步骤。

![output 27](img/f94b03cb748cf89886d715be9c47515c.png)



完成后，点击此按钮停止记录。

![output 28](img/d655adf50732ee95cba16b9486b13912.png)



现在让我们将我们创建的动作应用于不同的对象。从工具面板中选择椭圆工具，像这样画一个圆。

![output 29](img/c5e6c7edc2cda23b0815afde1ed2e156.png)



从动作对话框中选择您创建的动作，然后单击播放按钮。

![output 30](img/540b9c0a2da245fa2ae8666e6b5904c8.png)



它将转换一次，并通过显示此对话框要求您进行下一次转换。

![output 31](img/54d18314db5ddebc34a0bf2a7e6ea703.png)



如果您不想在每次转换时都显示此对话框，请禁用此切换/对话框按钮。

![output 32](img/7f05d77eded961278bab9335ec9c3d5e.png)



现在，再次单击播放按钮，将您选择的操作应用到该对象。

![output 33](img/b9552c6cee380eb15694d513a31d6b7a.png)



它会重复你的动作，而不会出现像这样的对话框。

![output 34](img/dcf51db1743b97bd28ff7fd324c014a3.png)



您可以从创建的操作中删除任何步骤。只需选择您想要的步骤，然后单击删除选择按钮。

![output 35](img/e2128c2b484177b23fa2cad85245481d.png)



你可以进入按钮模式，你创建的动作会以你选择的颜色显示，就像这样。
