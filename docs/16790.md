## Vectorworks 简介

VectorWorks 是一家 2D 和 3D 软件设计公司，为建筑、景观、工程和娱乐行业创建计算机辅助设计(CAD)和建筑信息建模(BIM)程序。它是由理查德·迪尔在 1985 年创立的。

### 我们为什么要使用 Vectorworks？

设计师和建筑师基本都用 VectorWorks 软件。它有很好的用户界面和渲染能力。此外，该软件还提供了一个平台，设计师可以理解分析产品创作的初始阶段和最终阶段的过程。

<small>3D 动画、建模、仿真、游戏开发&其他</small>

它具有详细的 BIM 功能和集成的工具集；这个软件使工作变得有创造性和精确。它拥有用于导入和导出、2d 绘图、装配功能、创建计算报告、绘制概念、与云服务共享文件、渲染音频/视频输出的一致工具，以及庞大的 3D 建模工具集。

### 向量工作是如何工作的？

VectorWorks 软件公司为其目标行业中的每一个目的创建了一个程序

1.VectorWorks 架构师，负责创建 BIM 模型、蓝图、建筑设计和艺术品。

2.用于景观设计和结构网格的 VectorWorks 地标

3.用于动态渲染效果和照明设备的 VectorWorks Spotlight

4.创建 2D 和 3D 模型和文档的 VectorWorks 基础。

5.VectorWorks Designer 可在一个平台下创建、建模和渲染一切。

6.视觉软件，用于在活动开始前预测灯光效果。

7.Braceworks 基本上是为 Spotlight 和 Designer 软件提供一个附加的当前模块。这个项目是为设计者创建设计、产品和索具材料而创建的。

8.ConnectCAD 作为 Spotlight 和 Designer 的附加预设模块，用于将音频和视频通用代码转换为可读格式。

### 优点和缺点

它有一些独特的功能，使该程序不同于其他 CAD 程序。每个程序都有自己的能力和功能。以下是 VectorWorks 计划的一些利弊列表

**优点**

1.  VectorWorks Fundamentals 在创建 2d 绘图方面有大量的工具。这些工具可以修改，并具有详细的 BIM 功能。
2.  VectorWorks Architect 拥有最简单的 2d 工具来创建 2d 艺术品，使用和学习起来都很简单快捷。工具有多种快捷方式，使工作变得快速和友好。
3.  Braceworks 软件是一个额外的优势，可以创建数学报告，以适当的精度研究 3D 模型，并以 DSTV 格式导出文件，可以与不同的队友安全地共享。
4.  ConnectCAD 是 Spotlight 的一个有用模块，可用于创建详细报告、创建复合 2D 和 3D 氛围、重新编辑设计以及与活动策划人联系。
5.  VectorWorks 最值得称道的优点是，来自公司的客户支持是顶尖的，对用户确实有帮助。

**缺点**

1.  VectorWorks Fundamentals 有时会导致崩溃，用户界面变得太慢而无法工作。用户发现在使用软件的不同阶段，程序是不均衡的。
2.  这个节目的观众是有限的。由于这个原因，用户发现很难容易地与其他行业成员共享或联合。
3.  对于一些用户来说，该程序的成本似乎很高。对他们来说，购买其他廉价产品比购买 VectorWorks 更容易。
4.  虽然，VectorWorks 在 Mac 和 Windows 操作系统上都可以工作。但是，有时，系统之间的协调是一个缓慢而麻烦的过程。
5.  VectorWorks 程序在学习过程中有一个陡峭的学习曲线。用户可能需要额外的时间来熟悉程序的工具和工作过程。
6.  为用户提供的关于 VectorWorks 程序的在线教程有限。由于软件很难学，用户需要教程来学习软件。

### Vectorworks 的用途

1.  VectorWorks Architect 具有深厚的 BIM 属性和 CAD 功能，可帮助设计师毫无困难地设计和绘制 2d 图纸、创建 3d 模型并进行渲染。
2.  该软件由西门子 Parasolid 3D 引擎支持，可创建无障碍 3D 模型。
3.  该程序的设计非常精确，可以在 MAC、Windows、iOS 和 Web 操作系统上很好地运行。
4.  VectorWorks Vision 是一个方便的程序，可通过 3D 渲染视图结构预测设计。用户可以在视觉软件的帮助下预先可视化设计结构。
5.  购买 VectorWorks Designer 产品，在一个平台上获得基础、聚光灯、建筑师和地标功能的组合。
6.  使用聚光灯相机和渲染功能创建逼真的模型和绘图。可调和多种选择可用，这个程序本身是惊人的。

### 结论

总而言之，VectorWorks 提供了一系列程序，让设计师和建筑师为创造一些令人惊叹的艺术品做好准备。它还支持 Braceworks 和 ConnectCAD 形式的附加模块。该软件还支持与团队成员共享云。

用户可以根据自己的需求购买软件。总的来说，它是一个很好的 CAD 程序，有无限的工具和功能，一点也不会让人失望。
