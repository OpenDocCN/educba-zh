## SketchUp 架构介绍

Sketchup Architecture 是一款 3d 建模软件，可处理广泛的绘图应用，如室内设计、建筑绘图和建模、景观建筑元素设计、机械和土木工程绘图、视频游戏设计以及电影 3D 模型等。它是基于网络的应用程序，免费使用，但也有付费版本，其中有一些额外的功能，你可以找到命名为 SketchUp Pro。它有几个 3D 建模的功能和工具，为它提供了高级别的工作能力和工作效率。所以让我们好好讨论一下这个软件。

我要告诉你的第一件事是使用这个软件的目的，这样你就可以理解 SketchUP 架构真正代表了什么。

<small>3D 动画、建模、仿真、游戏开发&其他</small>

### 为什么我们需要 SketchUp 架构？

它由 Trimble Inc .于 2000 年 8 月开发，用于处理建筑设计的不同组件和元素的 3D 建模。它为我们提供绘图布局，也允许渲染不同风格的表面，因此通过该软件，我们可以有精确的绘图布局，也可以通过该软件提供的渲染过程在现实世界中成像这些布局的对象。如果你想将不同的纹理应用到你的建筑或其他绘图的对象或模型上，给它们一个真实世界的外观，那么你可以从它的库中获得一些纹理(你可以在库中上传你自己的纹理，并下载其他的供你使用)。您可以管理模型上的灯光效果，因为有许多灯光，如聚光灯、点光源等。，并使用它的动画功能根据您的要求制作模型动画。

### SketchUp 建筑软件是如何工作的？

Sketchup Architecture 是一款软件，通过专利的推拉方法，您可以创建和编辑 2D 以及 3D 模型。推拉工具允许您挤出不同的平面，以赋予它们 3D 形状。要做到这一点，你只需要选择你想要挤出的物体，然后开始向你想要的方向拉这个物体。

当您启动 SketchUp 时，您可以为您的模型选择模板或设置默认模板，打开最近的文件，从它的欢迎屏幕浏览任何现有的文件。在预设集模型中，它为您提供了不同类型的预先设计的部分，您可以在您的项目工作中使用。

### 优势

该软件对 3D 建模的不同领域都有很好的影响，因为它有许多有效的功能和工具，但也有一些缺点。所以让我们讨论一下它的优点和缺点。

当你开始使用它时，你会注意到它有许多优点，如下所示:

*   您可以在网上找到自由模型组件库，如窗户、门和汽车等。不仅仅是模型组件，你还将拥有一个 3D 仓库，任何用户都可以向其中贡献模型，这意味着仓库就像一个盒子，你可以在其中放下(上传)你的模型或选择(下载)其他人的模型。
*   它是多语言软件，有多种不同的语言版本，几乎每个地区的用户都可以使用，这些语言包括英语、意大利语、法语、德语、西班牙语、韩语、日语、中文(简体和繁体)、葡萄牙语和巴西语。这是这个软件支持的相当多的语言，这使得它在各种用户中很受欢迎。
*   它与许多操作系统兼容，如 Windows 7 及其更高版本、OS X 10.9 及其更高版本。
*   它有组织良好的用户界面，并有每一个重要工具的部分，以及它的功能，如有，菜单栏，工具栏，绘图区，测量框，状态栏和默认面板。这些部分为您提供了平稳的工作环境。

### 不足之处

虽然我告诉你这个软件的一些优点，但是这个软件也有一些缺点，所以让我们也来看看。

*   SketchUp 不能导入 CAD 文件格式，这是它最大的缺点，从这个软件中获得多种 3D 文件格式有点困难。
*   这个软件的免费版本缺少内置的模型和元素，要获得更多的模型和元素，你必须使用它的付费版本。
*   在使用该软件期间，您不能在项目工作中添加指定的尺寸和细节，因此，在复杂的项目中添加几个元素的细节对我们来说变得很困难。
*   您可能会发现，它对于文档工作(如绘制比例图和所有此类事情)没有用处，因为缺少文档功能。

现在让我告诉你这个软件的一些应用，你可以理解它的商业目的。

### SketchUp 建筑软件的使用

作为一个建筑领域的三维建模软件，它在其他不同的相关三维建模领域有许多应用，如室内设计，景观建筑领域的组件和元素的设计等。它对媒体、电影和视频游戏行业产生了影响，为这些行业创建了 2d 和 3D 模型。
