## Google 开发工具简介

世界上最大的产业之一是博彩业。它在全球拥有 10 万名员工，从 2017 年起每年收入超过 1000 亿美元。这就是为什么许多开发者和爱好者想为他们自己的游戏赚钱。我们已经编制了一个有用的工具列表，它们简化了游戏开发过程，并优化了市场上众多移动游戏开发工具的结果。

### 谷歌开发工具

让我们一个一个来看看游戏开发工具。

1.  #### consistent

Unity 的流行度和虚幻引擎一样快。Unity 比 unreal 有几个优势，这可以作为你想要创建的游戏类型的决定因素。游戏的开发更快更容易。对于多平台游戏来说也是更好的——Unit 通过更少的移植和更多的调整转移到一个新的生态系统。

#### 2.戈多

二维和三维游戏都可以很好地配合 Godot 引擎。Godot 拥有广泛的常用元素和工具，以便用户可以完全专注于制作游戏。该引擎是开放的，可以免费使用，没有任何附加条件。用户创造的东西完全是他们的。一个大型在线社区向用户提供新功能和与错误相关的问题支持和更新。

#### 3.GNU 工具链

Visual Studio 的开源等价物是 GNU 编译器集合。如果你在 Linux 生态系统中开发游戏，你可能已经安装了大量的工具链。它可以和你能想象到的几乎任何编码语言一起工作。它不仅仅是一个 IDE，而是你和一个好的编辑器所需要的，无论你是编译、链接还是调试。

#### 4.米夏莫

Adobe 角色和动画的在线平台是 Mixamo 动画软件。用户在注册 Mixamo 动画软件后，可以免费下载动画和角色。用户将能够选择角色，添加动画，然后将其作为套件下载。此外，在插图、互动体验、游戏和电影中，用户可以添加动画、骨架或 3D 角色。

#### 5.螺旋芯

Perforce 的 Helix Core 是一个旨在让团队保持一致的应用程序，从存储源代码到共享数字资源，没有 Google Drive 或 GitHub 的伤害。任何与团队一起参与任何项目的人都知道，额外的劳动力可能很棒，但他们可能会有自己的问题。源代码管理对我管理每个团队项目来说是必不可少的。Helix Core 允许您轻松维护项目版本，并帮助您在不中断团队工作流程的情况下跟踪修改。

#### 6.胡迪尼 FX

胡迪尼 FX 是我个人生成程序和模拟的方式。胡迪尼的产品组合和大型社区汇集了具有复杂视觉效果的团队项目。Houdini Core 为参与制作专题、商业或视频游戏的艺术家提供了独特的 3D 动画体验，使其成为市场上最通用的游戏开发工具之一。Houdini 的基于节点的方法允许节点网络被共享成定制节点。带有视觉节点系统的程序很常见，但是 Houdini 以一种更紧凑和组织项目的方式建立了这个概念。

#### 7.速度树

SpeedTree 是一个行业标准，通过强大的定制系统在几秒钟内创建树木、灌木和其他植被，并被大片和游戏所使用。最近，通过新的工作流程，SpeedTree 已经转移到 PBR 材料，在那里可以在不离开应用程序的情况下更改纹理的颜色、色调和对比度，以及新的 150 个树木、灌木和植物库，使用北卡罗莱纳州植物园的真实世界示例进行扫描。

#### 8.奎克塞尔的巨型扫描仪

Quijel 的 Megascan 图书馆提供了华丽、逼真的 3D 游戏和电影资料。所有模型都有精细的紫外线和完美的纹理，并从 3ds Max 无缝导入到 UE4 及更高版本。Quixel 很快成为许多开发人员的“一站式商店”，他们正在从他们的综合照片扫描库中寻找真实的材料和 3D 资产。Quixel 的 Bridge 很容易将这些资产快速下载并导出到您的选择引擎中。您还可以使用 bridge 来录制和编辑纹理，以适合诸如 Substance Painter 或任何 3D 渲染软件之类的程序。

#### 9.粉笔吗

Krita 是一个免费的开源 Photoshop 替代品。它不是一个精确的克隆体，有自己的技能和画笔，背后是一个庞大的社区。我喜欢开源思维，我也喜欢 Krita。但是对于每个开发者来说，Adobe 套件是必须的。Krita 不能使用市场上可用的 Photoshop 的大量画笔和动作。
