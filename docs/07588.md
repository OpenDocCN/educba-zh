## 大型计算机功能介绍

拥有更大 CPU 的计算机，如存储所有数据以供处理和实时计算的家庭橱柜，被称为大型计算机。它是多处理器的同义词，因为几个任务是以并行方式执行的，所以没有一个任务在中间停止。这些功能强大的计算机需要大量的空间，公司或住宅的几层楼应该专门用来放置这种计算机。计算机的存储器和许多其它处理器被组合在一台大型计算机中，在那里任何工程问题都可以在几秒钟内解决。

### 特征

更长的性能:当我们安装并开始使用大型机时，这些系统可以保证我们在更长时间内的性能。不是几个月而是几年。适当的维护和用新软件更新系统将使计算机工作更长时间。同一系统中存在不同的处理器，在处理数据时，工作负载由所有处理器分担。这使得系统工作得更快更有效率。

#### 储存；储备

这些系统具有良好的存储容量，使系统能够在需要时处理大量数据。它可以存储大量数据，并根据客户要求进行处理。处理完数据后，系统可以显示准确的结果，而不会出现任何数据计算错误。

#### 集中式服务器

该系统具有中央服务器，该中央服务器具有多个操作系统，并且允许以中央方式管理操作。这使得系统高效，具有更高的性能和对单个操作系统更少的工作负荷。此外，它还具有分时功能，允许多个用户同时在同一个系统上工作。无论如何，我们不能称大型机为一台计算机，因为它有多个操作系统和不同的处理器。虚拟机通过系统中存在的应用程序的高度兼容性和复杂性来帮助系统实现这一点。

#### 肾素血管紧张素系统

大型机中的所有应用程序都是基于可靠性、可用性和可服务性而设计的，这使得该系统不同于其他计算机。在这些计算机的帮助下，数据处理可以很容易地完成，公司利用系统中的可伸缩性来操作不同的存储容量。所有这些应用程序的处理能力都由系统中的 CPU 维持。当系统中发生软件更新时，工作负载在处理器之间共享，因此工作不会在处理器之间中断。在某些情况下，系统不能停止工作，因为这对整个公司来说成本很高。申请的处理将无法进行，如果该公司是一家银行，国家安全可能会受到影响。大型机在使系统全天候可用方面起着重要作用。这三个特征是大型机的主要特征，系统基于这些特征工作。

#### 可量测性

大型机本身成本很高，如果无法实现同样的规模，公司将蒙受巨大损失。可扩展性可以在大型机中完成，这有助于公司对大型机的需求增加。扩展可用的大型机不需要单独的费用，因为它是具有额外容量的相同系统。这有助于公司利用可用的基础设施管理额外的资源。

#### 安全性

多个用户通过同一个操作系统或不同的操作系统访问同一个系统，这使得系统中的数据易受攻击。数据被视为最重要的资产，在系统中受到各种方式的保护。在这里，大型机只允许授权用户访问数据并在其上执行分析。其他人不允许查看或访问数据，因此对隐私的担心在大型机中是不需要的。由于数据库被设计为以完全安全的方式工作，大型机可以免受病毒攻击和其他恶意软件的攻击。

#### 和睦相处

大型机系统与其他系统一起使用类似应用程序的能力称为兼容性。大型机中有许多应用程序的设计成本很高，如果与其他系统不兼容，用户将无法使用操作系统。由于大型机系统不经常更改，应用程序很久以前就被开发了，并且这些应用程序会根据新的需求随时进行改进。这使得该系统能够与任何其他操作系统兼容。此外，这些系统是向后兼容的，因此应用程序可以更改为旧模式，也可以通过相应的更改更改为新模式。

#### 吞吐量计算

大量的输入和输出数据被传递给系统，这使得系统能够轻松地管理系统的所有工作。大型机不关心从系统传输或传输到系统的数据量，因为工作可以通过系统中的服务器和中央 CPU 的数量来管理。此外，系统确保在数据库中传输大量数据时不会出错。它被称为容错计算。高速处理器和大容量存储使用户能够连续地在大型机上工作，因此以准确的方式输出数据对他们来说非常重要。

#### 交易处理

其他特征包括它的事务处理能力，在事务处理中，针对库存或票据或任何类型的商品更新数据库。这种处理有助于用户更新单个数据库中的大量数据，该数据库有数百万条记录，可能是银行或机票预订，甚至是铁路预订。内存和存储容量有助于大型机在这方面击败其他计算机。

大型机与超级计算机并不相似，但它们有一些共同的特征，例如处理和存储。一个系统不能替代另一个系统，它的运行独立于任何其他系统。越来越多的公司使用大型机来实现流畅的工作流程，而以研究为导向的公司则依靠超级计算机来提高计算的准确性和速度。
