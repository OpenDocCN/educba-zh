## 电源 BI 中的连接

有许多方法可以将数据连接到 Power BI。Power BI 中用于连接数据的工具种类比市场上的任何工具都多。除了 Excel、文本和 CSV 文件等常规连接，我们还可以使用查询、数据库、SQL 服务器、云以及网站来获取数据。提到的平台是除了 Excel 和文本文件之外常用的平台。在 Power BI 中，所有可能的连接源都按照其应用进行了分类。我们将在本文中详细讨论所有这些联系。

### 如何从网站获取数据？

### 电源 BI 连接的优点

*   Power BI Connections 拥有种类最多的电源，可通过正确的连接过程获得。
*   除了常用的连接和文件源，Power BI 中的其余连接类型也连接到文本、Excel 或 PDF 文件。

### 要记住的事情

*   尽管 Power BI 中提供了各种各样的数据库资源，但我们仍然不能使用或链接超过 5 个资源。
*   最常用的数据库源位于文件类别下。
*   一年或两年前，当前添加的数据库列表是当前列表的一半。微软一直致力于将越来越多的数据库添加到 Power BI 中。
