## Inkscape 渐变简介

在这篇文章中，我们将学习如何在 Inkscape 中使用渐变。渐变效果用于给物体一种立体感，通常通过混合多种颜色或不同的阴影来创造立体感。渐变也可以用于纯粹的美学目的，但渐变的常见用途是在网页中用于各种形式和按钮。它也用于桌面应用程序。在 Inkscape 中，我们可以使用各种方法来创建渐变，这些方法不仅可以应用于填充，还可以应用于笔画，以创建一些令人惊叹的效果。

### 墨迹渐变的步骤

下面是下面提到的步骤

<small>3D 动画、建模、仿真、游戏开发&其他</small>

**第一步:**我们先用创建矩形和正方形工具或者按 r 键创建一个正方形，默认情况下，它会有黑色填充，没有描边。

![Inkscape gradient output 1](img/44448efd73587731d6527515f706c0dd.png)



**第二步:**然后使用创建和编辑渐变工具或按 G 键，然后拖动到正方形对象上，这将创建一个渐变。首先在你想要开始渐变的地方点击，当你想要结束渐变的时候释放它。默认情况下，将创建一个线性渐变。您可以通过向不同方向拖动来更改渐变的角度。但是为了获得更精确的度数，您可以按住 Ctrl 键以 15 度为增量。

![Inkscape gradient output 2](img/2f8d3030b9c765df0911b23b14713015.png)



**步骤 3:** 然后使用创建圆形工具或按 E 键，我们已经创建了一个圆形，然后再次使用渐变工具，但这次我们将从控制栏中选择创建径向椭圆或圆形渐变。从中间拖动并释放。虽然这种方法不会像你猜测的那样从中心产生渐变。

![Inkscape gradient output 3](img/d887f37680aea9e0827cbe8270840f40.png)



**步骤 4:** 我们已经完成的渐变是渐隐到白色，但是如果我们想要改变渐变的颜色以及它渐隐到的颜色，那么我们选择正方形的节点，顶部节点将改变它开始的颜色，底部节点将指示原始颜色渐隐或混合到的渐变的颜色。在圆形的情况下，它是径向渐变，所以它有两个端点，但您可以选择任何一个来改变渐变的颜色。

![Inkscape gradient output 4](img/943ac7c1641eed2d4fbe92f56c16f59c.png)



步骤 5: 现在，如果你想要一个不同的渐变颜色，而不是从两个颜色中创建的颜色，那么你可以从控制栏中插入新的停止点。这将添加一个菱形的新节点，因为起始节点是方形的，结束节点是圆形的。您也可以拖移色标，并将位置更改到需要颜色过渡的位置。在线性渐变和圆形渐变中添加停止点的方法是相似的。关于颜色选择，我们在底部有一个预置颜色栏，但是如果你想从另一个图像或对象中选择，你可以直接使用从图像中选择颜色工具或按 D 键

![Inkscape gradient output 5](img/3ab8236df5d0b551effa1f8c1e0e3ab2.png)



第六步:接下来我们打开填充和描边面板，或者按 Shift + Ctrl + F 来改变颜色，进行更多的控制。

![Inkscape gradient output 6](img/e927d0323e9768c56b7de987193dbcc9.png)



第七步:在选择任何颜色之前，确保你已经选择了渐变中的任何一个节点，如果没有，那么就用一种颜色填充整个形状。因此，在选择任何节点后，它可能是开始、结束或中间，您可以使用可用的颜色模式(如 RGB、HSV、HSV、CMYK、Wheel、CMS)来更改平面颜色的填充。我们将使用 HSL，即色调、饱和度和亮度。色相决定颜色，饱和度决定颜色的深浅，明度决定你想要的明暗。如果您想要删除任何特定的颜色，请按平面颜色前的 x 图标

![Inkscape gradient output 7](img/0184622442191e4bc0ba07c4a761b9a0.png)



到目前为止，我们已经在填充上使用了渐变，但是我们也可以在笔画上使用渐变。要给对象添加描边，首先选择它，按住 shift 键单击底部栏的颜色，或者右键单击您想要的颜色，然后选择设置描边。这将给我们一个平坦的行程。

![Inkscape gradient output 8](img/657ed99fde2bc104024fbf36d5ca987e.png)



步骤 9: 现在选择渐变工具，我们将创建一个新的线性渐变，但重要的是，我们将创建一个笔画渐变，而不是填充渐变。然后拖动圆的一端到另一端。

![Inkscape gradient output 9](img/73b5710fec8e0eb196c3cf7bf6e9c29a.png)



**步骤 10:** 然后你可以选择节点，再次改变开始和结束的颜色，你也可以插入新的停止点

![Inkscape gradient output 10](img/71e0b7e3110defd373a01771533ffe78.png)



步骤 11: 现在，如果你减小渐变的大小，你可以得到一些有趣的效果，如图所示。

![Inkscape gradient output 11](img/b2a7d3ed34dd5d1039f4a12c92468cc0.png)



现在让我们看看重复和反向是如何工作的。首先，我们在一个正方形上创建了一个径向渐变

![Inkscape gradient output 12](img/d3649c1c6f7d515c9ee9721b1a5bf5f6.png)



**第 13 步:**如果你想反转渐变方向，点击反转渐变方向按钮。

![output 13](img/bd7d0f5cd7c217c2622d6484ec0b665f.png)



**步骤 14:** 默认情况下，repeat 为 none，这意味着受影响的区域仅在线内，因此两端前后的颜色将是一致的。如果你想重复这个渐变，那么我们可以设置重复模式为反射。

![output 14](img/d26427b3579f3938cf1562d71bc1a4db.png)



**步骤 15:** 我们可以更好地看到，如果我们减少梯度线的大小，因为一个重复的模式，反映了开始和结束的颜色。

![output 15](img/fb37e9fa632def85dc79a66b907a387e.png)



第 16 步:另一个选项是直接，阴影将直接重复，并给我们颜色效果的突然变化。

![output 16](img/d2d8189823c47d14db5ec42e13cda103.png)



第 17 步:我们可以重用我们的线性或渐变填充和笔画的渐变。首先，使用星形和多边形工具创建一个形状。并选择任何现有的渐变作为填充

![output 17](img/26412bb18c822917baa04789eb98f051.png)



类似地，我们也可以在笔画上使用现有的渐变。

![output 18](img/a8e8fc39ee0db56aa32e3e636dd608be.png)
