## Inkscape 箭头简介

Inkscape arrow 不是什么特别的东西，它与其他设计软件的箭头一样，但我们在 Inkscape 中以不同的方式使用它，我们在不同类型的指示工作以及许多不同的艺术工作中使用箭头，所以今天的文章我们将学习不同的方法，通过这些方法我们将在这个软件中创建箭头，这些方法将包括工具面板的工具，以及我们将通过菜单栏的菜单获得我们的最终结果。我还会告诉你如何在箭头和箭头上有两种不同的颜色。那么学习这些方法你兴奋吗？让我们这样做吧。

### 如何在 Inkscape 中创建和使用箭头？

我们有两种方法可以让你在 Inkscape 中创建 arrow，但是你可以在自己的基础上找到更多的方法。第一种方法是使用任何路径的笔画样式。

#### 使用笔画样式创建箭头

为了学习这种方法，我们需要一条线，这条线应该是直的，或者是倾斜的，但不是弯曲的。我们将在本文后面讨论曲线箭头。因此，从工具面板中选择贝塞尔工具，或者按键盘上的 Shift + F6 键。

![Inkscape arrow output 1](img/302d25551c6e3f311e01bc87b32c4a81.png)



现在按住键盘的 Ctrl 键像这样画一条直线。

![Inkscape arrow output 2](img/fac3f0457ef4cc5dd7f1de508df74bef.png)



一旦你画好线，进入工具面板，选择工具或者简单的按键盘的 F1 功能键来切换这个工具。

![Inkscape arrow output 3](img/3aae691b97a94ccf147e5a097d7a6e57.png)



现在转到用户屏幕顶部菜单栏的对象菜单，然后点击填充和描边选项或按键盘的 Shift + Ctrl + F 键。

![Inkscape arrow output 4](img/6c60683a4f929962a646e22446e7db16.png)



在填充和描边面板中，点击描边样式标签，我将增加我们绘制路径的描边宽度，这样我可以清楚地显示箭头。这个宽度可以根据你的工作需要而定。我采取这个宽度只是为了解释它。所以我将 3 作为笔画宽度的值。

![Inkscape arrow output 5](img/9b2cc8316d3c2b0fb9dc50b81fc3dbab.png)



现在点击这个标签的标记选项的向下箭头按钮，在这里你填写查找箭头的数量。你可以和你喜欢的任何一个一起去。

![Inkscape arrow output 6](img/1fe7e7c6b1c8700b6b215cf9b0900ed7.png)



我将选择这个箭头，它将与这条线的起点相连。

![Inkscape arrow output 7](img/ea9f33f8c34b017bdb6fd9525060b32c.png)



在标记选项的第三个选项卡中，您将有相同的箭头，但这些箭头将像这样连接在线的端点。

![Inkscape arrow output 8](img/3d9f9f6bf4a80f247085d7b6593cff8b.png)



您可以使用此选项在任何其他节点或线的中间添加箭头。

![Inkscape arrow output 9](img/e70eb6597ec2228759cd1a66f8d6fb7e.png)



您也可以通过这些选项更改箭头的线条样式，这意味着您可以将线条更改为虚线或此列表中的任何其他线条。

![Inkscape arrow output 10](img/d0224f403224bc0cf13f35ca433fe434.png)



现在让我们深入分析一下，进入菜单栏的查看菜单，导航显示模式选项，然后点击新下拉列表的轮廓选项。

![Inkscape arrow output 11](img/b6b9d7475840a1382b93fd79b3ca5733.png)



现在你可以看到这个箭头是一条简单的线。

![Inkscape arrow output 12](img/09e4c53c59a1b8400c5e4d7e7d655417.png)



因此，让我们把它转换成形状，并为去路径菜单的菜单栏，并点击'中风的路径'选项，或者你可以按下 Ctrl + Alt + C 键的键盘为这个选项。

![Inkscape arrow output 13](img/f76efb2d2dd871a8506993eec6dcca0a.png)



你会看到这个箭头的形状像这样。

![Inkscape arrow output 14](img/768a25e7580e355072464aae4d9c91d1.png)



现在，通过点击对象菜单的取消分组选项或按键盘的 Shift + Ctrl + G 键来取消分组。

![Inkscape arrow output 15](img/1c6510ea7fb6a35d5ddd69775b512e68.png)



现在你可以单独移动箭头。

![Inkscape arrow output 16](img/170fa8d48e01b200e3eca1f26cbb4fea.png)



为了正确对齐，您可以通过对齐面板。

![Inkscape arrow output 17](img/de214550199a96d1a774ebbe70c54136.png)



将显示模式从轮廓模式更改为正常模式。

![Inkscape arrow output 18](img/c4929ba77f4b7ad78f5b84ac8cbbd887.png)



你可以分别改变线条和箭头的颜色。这里我们有箭头笔画。

![output 19](img/c1326874338c89f3bc4063776336a9b5.png)



如果你愿意，你也可以删除箭头的线条，或者给箭头的线条添加线条。

![output 20](img/38218b0168e10b161547ba0506380c1f.png)



您还可以从标记列表中添加带有箭头终点或起点的不同形状。只是我在这里。

![output 21](img/11b19b70fc7b7e299fadc6992eadae76.png)



下一个方法，我们可以通过使用形状工具来创建一个箭头。

#### 使用 Inkscape 的形状工具创建箭头

首先从工具面板中选择矩形工具，或者按下键盘的 F4 功能键。

![output 22](img/2583b065ce8a2862b03730f523e20321.png)



像这样画一个矩形，然后从工具面板中选择“开始或多边形”工具，或者按键盘上的*键。

![output 23](img/ab22e02367341c3236f752d907abdf9e.png)



现在在这个工具的参数栏的角选项中给多边形的角值 3。

![output 24](img/b8308d8070d50a7940d220ae5119e34d.png)



现在画这种类型的三角形。

![output 25](img/6c250946b5f5df690676f6536db05807.png)



使用“对齐”面板的“对齐”选项卡，像这样对齐这两个形状。

![output 26](img/dedf9fb586ca46d1e42fabb5f2a5f3e5.png)



现在我将它做成一个形状，然后点击下拉列表中的联合选项。

![output 27](img/412ca022124a9557b3c0ddf4890cd86a.png)



现在你看，如果我选择它，将只有一个选择框，这意味着它是一个对象。

![output 28](img/d88e47df04116a953b4c671baea84ae0.png)



你也可以画曲线箭头，这样画一条曲线。

![output 29](img/9b237520ec91edda7b33b60b70a966e1.png)



将箭头应用于线条样式选项中曲线的任何一点。你可以看到这不是曲线的正确方向。

![output 30](img/52e60ed2ea7f6d4b71ff0c0706900c6b.png)



只要按照我们在直线的情况下做的相同的步骤，在取消组合两个形状后，你可以根据曲线设置箭头的角度。

![output 31](img/0f36eed7720e313ca6527bebf4df758a.png)



你也可以用 shape 方法来做这个，我们在上面的步骤中为矩形创建了箭头，然后通过旋转来调整三角形的角度。

![output 32](img/9aa1c4ef94e4dd8fc73a4c6f27520caf.png)



我在这里向你解释了在 Inkscape 中创建箭头的两种方法。现在你可以和他们中的任何一个一起去。
