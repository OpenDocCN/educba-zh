### 句法

在 HTML 中，我们有一个 CSS 样式，用于突出网页的可定制性和用户友好性，在浏览器和用户视图中具有吸引力。HTML 对齐文本值与正确对齐的文本或值对齐；默认情况下，对齐文本应该位于网页的左侧。让我们看看网页中 HTML 对齐的一般语法。

```

</head>
<body>
<p align="justify">Welcome To My Domain.</p>

</body>
</html>
```
