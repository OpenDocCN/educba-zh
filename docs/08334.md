### 第 1 部分 HTML 面试问题(基础)

这第一部分包括基本的面试问题和答案。

#### 1.你说的 HTML 是什么意思？

答案:
HTML 被称为超文本标记语言。这种语言用于万维网。这是一种用于创建网页的标准语言。

#### 2.HTML 页面由哪些部分组成？

**答案:**
基本上，网页有两部分:内容和标签，它们负责一个 HTML 页面的格式。

让我们转到下一个 HTML 面试问题。

#### 3.标签是什么意思？

**答案:**
在 HTML 中，页面内容放在标签之间，标签基本上负责页面的格式化。标签写在小于符号(<)和大于符号(>)之间。例如:上面例子中的<h1>text</h1>
h1>是开始标签，< /h1 >是结束标签。

#### 4.所有的 HTML 标签都是成对写的吗？

**答案:**
这是在面试中最常见的 HTML 面试问题。不，有一些 HTML 标签可以单独使用。例如:
< img >，< br >。

#### 5.HTML 中有哪些列表类型？

**回答:**
下面给出了 HTML 中常见的列表类型:
**–有序列表
–无序列表
–定义列表
–菜单列表
–目录列表**

#### 6.举一个在 HTML 页面上放置评论的例子？

**答案:**
<！—供评论的文本—>

#### 7.如何在 HTML 文件中插入版权符号？

**答:**
对于版权符号，我们可以用&式复制；或者

#### 8.如何在 HTML 页面中应用超链接？

**回答:**
我们可以对 HTML 页面使用< ahref >标签。例如:< ahref > Text < /a >

#### 9.如何改变 HTML 页面的字体颜色？

**答:**
我们可以给出这个命令来改变字体颜色
<font color = " color ">…</font>

#### 10.什么是字幕标签？

**回答:**
对于滚动文本，我们[使用一个字幕标签](https://www.educba.com/html-marquee-tag/)。例如:<字幕>文本</字幕>

让我们转到下一个 HTML 面试问题。

#### 11.如何在网页上创建支持电子邮件的文本？

**答:**
对于邮件启用文本，你要在网页上写下以下命令。<A HREF = " mailto:email address ">要点击的文本< /A >

#### 12.HTML 页面上的段落怎么写？

**答:**
对于段落标签< p >你要显示的文字段落< /p >都会用到。

#### 13.你如何将图片作为网页的背景？

**回答:**
将图片制作为背景的命令是:
<body background = " image.gif ">其中 image . gif 将是图片的路径。

#### 14.框架是什么？

**回答:**
通过使用框架，我们可以让网站的导航变得更加容易。

#### 15.HTML5 是什么？

**回答:**
HTML5 是 [HTML 语言](https://www.educba.com/what-is-html/)的第五个版本，是 HTML 目前的[运行版本。](https://www.educba.com/versions-of-html/)

让我们转到下一个 HTML 面试问题。

#### 16.HTML5 中包含了哪些新标签？

**答:**
<视频>和<音频>是 HTML5 中引入的新标签。它们基本上是用来替代 flash player 和 Silverlight 来播放网页上的多媒体项目。

#### 17.哪个浏览器支持 HTML5？

**答案:**
谷歌 Chrome、苹果 Safari、Mozilla Firefox、Opera 都支持 HTML5 的大部分功能。

#### 18.HTML5 中的

<figure>是什么？</figure>

**答案:**
对于表示自含流内容，我们使用一个<图>标签。

#### 19.HTML 中的 Canvas 元素是什么？

**答:**
为了在网页上表现图表、2D 图像、图形，我们使用了 Canvas 元素。

#### 20.HTML5 的存储类型有哪些？

**回答:**
html 5 的两种存储类型是:
**会话存储**——它会存储与当前相关的数据。
**本地存储-** 当浏览器关闭时，该数据不会被擦除

### 第 2 部分 HTML 面试问题(高级)

现在让我们来看看高级面试问题。

#### 21.HTML5 中的应用缓存是什么意思？

**答:**
应用缓存是指 web 应用被缓存。因此它可以在没有互联网连接的情况下访问。

#### 22.解释 HTML5 对于表单的输入类型？

**答案:**
日期，日期时间-本地，日期时间，月份，邮件。

#### 23.什么是小程序？

**答案:**
这是 HTML 面试中经常被问到的 HTML 面试问题。小应用程序是一个小程序，它可以嵌入到网页中，执行一些特定的功能，如计算、动画。

#### 24.小程序是用哪种语言编写的？

**答案:**
爪哇

#### 25.如何在 HTML5 中获取用户的地理位置？

**答案:**
通过使用地理定位 API，我们可以检索到用户的位置。

让我们转到下一个 HTML 面试问题。

#### 26.HTML 属性是什么意思？

**答:**
与元素一起给出的附加信息称为属性。例如
<font size = " 10 " color = " red ">

#### 27.HTML 页面的扩展名是什么？

**答案:**
。htm 或者。html 是 HTML 的扩展

#### 28.HTML 支持哪些不同类型的标题格式？

**答案:**
HTML 标题是用来突出 HTML 文档内容的。HTML 中使用的标题标签是< h1 >到< h6 >。

#### 29.我们如何在 HTML 中使用 JavaScript？

**回答:**
通过使用脚本标签我们可以将 JavaScript 与 HTML 一起使用。例如:
<脚本>
document . getelementbyid(" demo ")。innerHTML = "你好 JavaScript！";
</脚本>

#### 30.Get 和 Post 方法是什么？

**答:**
GET s 用于向服务器请求数据，POST 用于向服务器提交数据。

让我们转到下一个 HTML 面试问题。

#### 31.哪个编辑器用于创建 HTML 页面？

**回答:**
HTML 页面的编辑器非常多，像记事本、Notepad++和 Sublime Text editor。

#### 32.HTML 中的 Longdesc 是什么？

**答:**
Longdesc 是一个属性，允许你提供到另一个包含框架内容描述的页面的链接。例如 longdesc = " frame description . html "。

#### 33.如何用 HTML 创建表格？

**回答:**
通过使用< table >标签我们在 HTML 中创建表格。例如，如果你想创建一个 3 行 3 列的表格:
T0】

#### 34.什么是 CSS？

**答案:**
CSS 代表级联样式表。这是一个可以分配给各种 HTML 元素的规则列表。它不区分大小写。

#### 35.什么是 web 套接字？

**答:**
Web Sockets 是下一代 Web 应用的双向通信技术，它在单个套接字上运行，并通过 HTML 5 兼容浏览器中的 JavaScript 接口公开。

让我们转到下一个 HTML 面试问题。

#### 36.SPAN 标签的用途是什么？

**答案:**
Span 用于格式化 Span 块中的元素。它用于选择行内文本。

#### 37.什么是单元格间距和单元格边距？

**答案:**
单元格间距定义 HTML 页面边框宽度的属性，单元格边距是定义单元格内容间距的属性。

#### 38.XHTML 元素的国际化属性是什么？

**答案:**
XML，DIR，LANG

#### 39.你说的箱子模型是什么意思？

**答:**
页面上的每个元素都是一个矩形框，可能有宽度、高度、填充、边框、边距。盒子模型的每一部分都与一个 [CSS 属性](https://www.educba.com/what-is-css/)相关:宽度、高度、填充、边框和边距。

#### 40.为什么我们使用嵌入标签？

**答:**
Embed 标签用于在 HTML 文档中包含一段视频或音频。嵌入标记需要一个结束标记。它需要需要在页面上显示的视频或音频文件的来源。
语法:<嵌入>源文件</嵌入>
