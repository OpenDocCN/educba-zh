## Python 中算术运算符的介绍

与任何其他编程语言类似，python 中的算术运算符只不过是用于表示特定数学和逻辑运算的符号/元素，这些运算将在值或赋值变量上执行。python 中用于算术运算的各种运算符是“+”表示加法，“-”表示减法，“*”表示乘法，“/”表示除法，“%”是用于获得除法运算结果余数的模数运算符，“//”表示底数除法，“**”表示指数运算符。

python 中的算术运算符有:

#### 1\. ‘+.’

“+”运算符用于执行加法。可以使用“+”运算符将两个操作数相加。

**语法:**

```
x '+' y
```

让我们举一个简单的例子，在这个例子中，我们将使用“+”运算符将两个数字相加。这两位数是操作数。

```
x = 4
y = 5
print(x+y)
```

**输出:**

#### 2\. ‘-’

“-”运算符用于执行减法。两个操作数可以用“-”运算符相减。

**语法:**

```
x '-' y
```

让我们举一个简单的例子，我们将使用'-'运算符减去两位数。这两位数是操作数。

```
x = 5
y = 4
print(x-y)
```

#### 3\. ‘*’

“*”运算符用于执行乘法。使用“*”运算符可以将两个操作数相乘。

**语法:**

```
x '*' y
```

让我们举一个简单的例子，我们将使用' * '运算符将两位数相乘。这两位数是操作数。

```
x = 4

print(x*y)

#### 4\. ‘/’

“/”运算符用于执行除法。左操作数除以右操作数。两个操作数可以用“/”运算符相除。

**语法:**

```
x '/' y
```

让我们举一个简单的例子，在这个例子中，我们将使用“/”运算符将两位数相除。这两位数是操作数。

```
x = 4
y = 2
print(x/y)

#### 5\. ‘%’

“%”运算符用于在左操作数除以右操作数时找出除法的余数。两个操作数的其余部分可以使用“%”运算符找到。“%”运算符称为模数运算符。

**语法:**

```
x '%' y
```

让我们举一个简单的例子，当一个数字被另一个数字用“%”运算符除时，我们将找出两个数字的余数。这两位数是操作数。

```
x = 4
y = 2
print(x%y)
```

**输出:**

![ using ‘%’ operator](../Images/f54f045ea6460922d7b48723bc01a46a.png)

在上面的例子中，x 和 y 是操作数，' % '是操作符，0 是输出。

#### 6\. ‘//’

“//”运算符称为地板除法运算符。通过使用基数除法将数字行中的数字向左调整，除法运算的结果被转换为整数。

**语法:**

```
x '//' y
```

让我们举一个简单的例子，在这个例子中，我们将两个数字相除，并应用 floor division '//'运算符。这两位数是操作数。

```
x = 5
y = 2
print(x//y)
```

**输出:**

![two digits ](../Images/bf86d647d85c997735c7b008a72be0e6.png)

<noscript><img class="alignnone wp-image-277924 size-full" src="../Images/bf86d647d85c997735c7b008a72be0e6.png" alt="two digits " width="71" height="44" data-original-src="https://cdn.educba.com/academy/wp-content/uploads/2020/01/Arithmetic-operators-in-python6.png"/></noscript>

在上面的例子中，x 和 y 是操作数，'//'是操作符，2 是输出。该操作的实际输出是 2.5，但是由于我们应用了 floor division，小数点前的数字被调整到数字行的左侧，因此输出是 2。

#### 7\. ‘**’

“**”运算符称为指数运算符。一个操作数的幂

另一个操作数是指数' ** '运算符的功能。它的左操作符提升到右操作符的幂。

**语法:**

```
x '**' y
```

让我们举一个简单的例子，在这个例子中，我们将两个数字相除，并应用 floor division '//'运算符。这两位数是操作数。

```
x = 2
y = 2
print(x**y)
```

**输出:**

在上面的例子中，x 和 y 是操作数，' ** '是运算符，4 是输出。

### 用 Python 实现算术运算符的示例

让我们举一个简单的例子，通过将 python 程序中的所有算术运算符都包含在下面的单个程序中来理解它们:

**代码:**

```
x = 15
y = 4
print('x + y =',x+y)
print('x - y =',x-y)
print('x * y =',x*y)
print('x / y =',x/y)
print('x // y =',x//y)
print('x ** y =',x**y)
```
