## SVG 墨水匣简介

以下文章提供了 Inkscape SVG 的概要。Inkscape SVG 是一种文件格式，这种 SVG 代表可缩放矢量图形，这意味着您可以编辑该文件的对象，以改变它们的大小和属性，而不会损失其质量。

### 过程

在这篇文章中，我们将看到如何将 Jpeg 或光栅图像转换成矢量文件，并保存该矢量文件，以便在其他设计平台上使用。这个过程包括参数调整位图描摹的特点，该软件的菜单栏，我们也经历了一些其他参数。
