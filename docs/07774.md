## CSS 光标手简介

在 HTML 中设计页面时，您可能希望根据上下文更改鼠标光标的样式。例如，当页面加载时，光标符号将是一种类型，当帮助部分悬停时，光标将再次改变，等等。为了实现这一点，光标属性允许根据所需的场景(如“等待”、“帮助”、“移动”等)来指示应该向用户显示哪种类型的光标。在您的 HTML 代码中。在本文中，我们将在一些例子中看到如何使用不同类型光标的设计。

**语法:**

<small>网页开发、编程语言、软件测试&其他</small>

```
cursor: value;
```
