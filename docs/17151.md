## Vaporwave Photoshop 简介

Photoshop 是光栅图像编辑软件，它有不同的技术，通过它我们可以制作一些旧的视觉效果。Vaporwave 是 20 世纪 90 年代后期的一种视觉效果，它与 glitch art、音乐视频和不同类型的艺术品一起出现。借助 Photoshop 软件的不同功能，我们可以在我们的图像上创建这种效果。在本文中，我们将学习如何使用该软件的功能创建 vaporwave，并分析这些功能的不同参数。那么就让我们开始关于这款软件中的 Vaporwave Photoshop 话题的讨论吧。

### 如何在 Photoshop 中创建 Vaporwave？

我们应该遵循一些简单的步骤在我们的图像上应用这种效果，但是我认为在我们开始学习这个主题之前，我们应该看看这个软件的工作屏幕，这样我们就可以在这篇文章中更好地理解这个主题。

在顶部，我们有一个菜单栏，它有不同的菜单，如文件、编辑、图像和其他一些菜单。这些菜单帮助我们在使用该软件的过程中管理该软件的不同参数。在这个菜单栏下面，我们有一个活动工具或图像的属性栏，通过它我们可以管理这些工具或图像的参数。在这个栏的下面，我们有一个工具面板，左侧有一个不同类型的工具，使我们在这个软件中的工作变得容易，在中心我们有一个显示窗口，我们可以在其中看到我们的工作，在右侧我们有一些面板，帮助管理这个软件的不同功能。您可以根据自己的选择在不同的位置重置所有部分。

![vaporwave photoshop output 1](img/cff79cf0abac072f0ba265085367578e.png)



现在让我们有一个图像，我们将应用蒸汽波效应。要获得图像，请转到菜单栏的文件菜单，然后单击它。一旦你点击它，一个下拉列表将被打开，点击“打开”下拉列表的选项。

![vaporwave photoshop output 2](img/db7ec5145a1ad76fa430b5b60ac643ed.png)



一旦你点击打开，一个放置图片的对话框将会打开。进入你个人电脑的目标文件夹，点击选择你想要的图像。选择后，按下此对话框的打开按钮。

![vaporwave photoshop output 3](img/f9ecab18c9235b5564533f9c0f5b06e0.png)



一旦你点击你的图像将作为背景层打开，使其可编辑，进入图层面板，双击这个图层。一旦你双击一个“新层”框将被打开。根据你的名字命名这个图层，然后点击这个框的 Ok 按钮。

![vaporwave photoshop output 4](img/17ffab0f688ad9eb3b96a49642202ada.png)



现在按 Ctrl + J 按钮来复制这个图层。

![vaporwave photoshop output 5](img/6348b31f3874c7ccd72e7df498ce49ab.png)



现在选择复制层点击它，并按 Ctrl + I 按钮，这种类型的影响你的形象。

![vaporwave photoshop output 6](img/7937368e8ae81d16cc6dc7227f4887e5.png)



现在点击这一层的模式选项的下拉箭头。通过单击从列表中选择一个颜色选项。

![vaporwave photoshop output 7](img/6508a4073bec8c9a43871e3ab7e1d753.png)



现在它看起来会像这样。

![vaporwave photoshop output 8](img/fd5ae4303d6a4349161d24e43be92632.png)



现在到图层面板的底部，点击调整图层按钮。将打开一个下拉列表，从这里选择“通道混合”选项。

![vaporwave photoshop output 9](img/56d7d61d264d571352c18011301b57db.png)



在打开的对话框的输出通道选项中选择红色，使红色值为 0，蓝色值为 100。

![vaporwave photoshop output 10](img/8a54107a9a8b0b191a12a9fa3930ffec.png)



在打开的对话框的输出通道选项中选择蓝色。

![vaporwave photoshop output 11](img/7a99cb530f12e69a01d7aa095e8f7953.png)



并将红色值设为 100，蓝色值设为 0。

![vaporwave photoshop output 12](img/d1705e168cee7f62699875c5a5416c9a.png)



现在右键单击图层面板中的通道混合器图层，从下拉列表中选择“创建剪贴蒙版”选项。

![vaporwave photoshop output 13](img/a03da5c814e1eacb1ad9ecaed5efde25.png)



现在再次点击图层面板的调整图层按钮，从列表中选择色调/饱和度选项。

![vaporwave photoshop output 14](img/1cc75556150137dca6867553843a5a61.png)



右键点击色相/饱和度层，再次点击列表中的“创建剪贴蒙版”选项。

![vaporwave photoshop output 15](img/848d5a72c6747a928b9ee73fe4671036.png)



现在从色调/饱和度框的列表中选择红色选项。

![vaporwave photoshop output 16](img/741e4c4ef410f7352a503ea2ddb44180.png)



将红色的色调值降低到-40。你可以看到对你形象的影响。

![vaporwave photoshop output 17](img/0003600e93b9bd0c9855ae77f0e72ac7.png)



现在从色调/饱和度框的列表中选择黄色选项。

![vaporwave photoshop output 18](img/57441fa76ec24c49897f1ce78e2cb38f.png)



将黄色的色调值降低到-45。你可以看到对你形象的影响。

![vaporwave photoshop output 19](img/188ab7e6cd758b4a06ecc08ef9f0ef8d.png)



现在从色调/饱和度框的列表中选择蓝色选项。

![output 20](img/1c8ecf093530b685f12c7eae230f500d.png)



将蓝色的色调值降低到-60。你可以看到对你形象的影响。

![output 21](img/6f21525830abb2441034f00ce1b0d7a2.png)



现在在这个软件中放置一个旧电视效果的图像。我已经从网上下载了。

![output 22](img/5a35d27b342fe0873f6b0bc2fa9b70ea.png)



调整这个图像的大小，因为它像这样覆盖了你的背景图像，然后按下回车键。

![output 23](img/f0aca6ed7d7aa7a2e9721564a2e7cba9.png)



现在点击模式选项的下拉箭头，从这里选择柔光选项。

![output 24](img/a0bfaac68d33daa1ecc0964696e1f7fe.png)

现在转到菜单栏的过滤器菜单，并点击它。从列表中选择液化选项。

![output 26](img/489a509e87b01c370ada7c514eab15c5.png)



现在一个液化选项对话框将被打开。根据你自己调整画笔的大小。

![output 27](img/778490d98f0d8b7dec58d21c50fc4fe7.png)



用这样的笔刷扭曲这张图片，然后点击对话框中的 Ok 按钮。

![output 28](img/97340dddc4093afce847aa93756aff32.png)



一旦你按下确定按钮，你会看到你的图像扭曲的效果。现在点击图层面板按钮上的新建图层选项。

![output 29](img/4729675b435d0613d2ba5b930fdbe280.png)



转到编辑菜单并点击它。从下拉列表中选择填充选项。

现在从填充框列表中选择 50 %灰色选项，并按下此对话框的确定按钮。
