## SketchUp 图层简介

SketchUp Layers 是该软件的一个非常有用的功能，通过它我们可以将任何模型的不同组件组织在每个单独的部分中，您可以将这些部分称为层/标签。这个软件中有许多图层面板的特性，通过使用它们，我们可以很好地管理任何模型的组件。在这个软件的最新版本中，你可以找到带有名称标签的图层，它比以前版本的图层面板有更多的功能。你知道，几乎每个设计软件都有图层面板，但在 SketchUp 中是以不同的方式工作的。让我们来看看它在这个软件中是如何工作的。

### 如何在 SketchUp 中使用图层/标签？

如果您开始在此软件中使用图层面板或标签，您可以以更明确的方式管理您的工作。因此，您可以在默认的托盘面板中找到标签/图层面板，它位于工作屏幕的右侧。

![SketchUp Layers Output 1](img/d1dcc8e238bfc1ac5c873d2725bab2d9.png)



如果没有默认托盘，您可以在窗口菜单的下拉列表中找到它。然后，只需点击它启用它。

![SketchUp Layers Output 2](img/9d20862fce62bff2651a1a9ec8d9ea2c.png)



还有一种方法可以让您拥有一个可移动的标签/层面板，为此，单击查看菜单下拉列表中的工具栏选项。

![SketchUp Layers Output 3](img/33d1bc69e17c8df2c9757e607a113419.png)



并启用“工具栏列表”对话框的“标记”选项的复选框。

![SketchUp Layers Output 4](img/a10dc0e1de0aea094abd053f82b75cee.png)



您可以将标签/层面板移动到工作窗口屏幕上的任何位置。

![SketchUp Layers Output 5](img/eddce6405c9733368c3440490a8722b6.png)



我画了这个物体。您可以在“标签”面板中看到有一个未标记的标签/层(在之前的层面板中，它由层 0 表示)，并且所有绘图对象都将位于默认标签/层中。您可以在实体信息面板中看到它。

![SketchUp Layers Output 6](img/6cb022c0469c0bad9ebc7a7bd2aaea3d.png)



让我们创建一个新的标签/层，为此，单击该面板的加号按钮。

![SketchUp Layers Output 7](img/f8efaf46ea40a7c5af8ee1e7497bb489.png)



我将把它命名为地板。您可以双击任何标签/层进行重命名。

![SketchUp Layers Output 8](img/809220c290ad533159e843e7dff660b9.png)



你必须记住，你必须从实体信息中分配层给任何组件，以便组件将从未标记的层移动到分配的层。因此，我将选择该组件并转到“实体信息”面板，然后从该面板的“标记”选项中选择“楼层标记”。

![SketchUp Layers Output 9](img/fbfc06c9fe0dfbb74c5c5612875efdcd.png)



您可以看到此构件现在已指定给楼板标记/层。

![SketchUp Layers Output 10](img/1a25c61d6bebb7a7b47a84c2e1686c58.png)



如果我点击它的眼睛按钮来隐藏这一层，那么这个组件也将隐藏。

![SketchUp Layers Output 11](img/c5bd48c63dc58e0f866d6cdcfca18c0e.png)



现在我将添加一个标签/层，并命名为墙。

![SketchUp Layers Output 12](img/5bc93a61ae13d65178389730977d49f1.png)



现在我要画一面这样的墙。绘制完成后，我将从实体信息面板中将其分配给墙标签/层。

![SketchUp Layers Output 13](img/4eb720c259efe35df657cd4fa57d3fa3.png)



现在，我将通过单击它的眼睛按钮来隐藏墙标签/层。您可以看到一些未隐藏的部分，它们也会影响楼板层构件。

![SketchUp Layers Output 14](img/365cea4f5f01c3b95d5e6f08766fdce3.png)



因此，在处理标签/层或 SketchUp 时，您必须对任何组件进行分组，以便在您对一个组件进行任何更改时，其他组件不会影响它。因此，我将首先选择地板组件，右键单击它，然后单击“Make group”选项，将 I 作为一个组件。

![SketchUp Layers Output 15](img/01b42ccd19b59ca4f568377026d31022.png)



现在，我将再次绘制墙层中的这面墙，方法与之前相同。

![SketchUp Layers Output 16](img/a2befb28af189629dbb3b7b5d4d0c925.png)



现在，这一次如果您隐藏此墙层，那么此标记的组件将会隐藏，而不会影响其他层组件。

![SketchUp Layers Output 17](img/f1d8792e5158f893f18466676765bebd.png)



现在，我将制作一个屋顶组件，并将其分配给一个新的标记/层，这是一个屋顶标记。

![Roof tag Output 18](img/34c0e225af64a51193d728fc9cf40d57.png)



您可以只显示您想要更改的层，并通过单击这些层的眼睛按钮来隐藏多个层。我想告诉你，你必须双击编辑任何组件，因为我们已经将它们单独分组，使一个组件。

![SketchUp Layers Output 19](img/cd4cec26e2b0f26b46ff6bbb7954cb6a.png)



有一个按钮是详细信息按钮，所以单击它。

![SketchUp Layers Output 20](img/5e83e48eff1dee3191a131c3a4e6b2ce.png)



现在点击开放列表中的“标签颜色”选项。

![Color by Tag Output 21](img/01b10d10c20b865f61f0b8a709798a07.png)



并且每个组件将根据分配给它们的标签/层的颜色转换成彩色对象。

![SketchUp Layers Output 22](img/ea582f237a0e5280d05b33edd31baad7.png)



你也可以通过点击特定层的颜色框来改变任何层的颜色，一旦你点击它，一个编辑材料对话框将会打开，你可以从中选择你想要的颜色。

![Color option Output 23](img/3a07d7e0baff168a043cbb2100e0a362.png)



您也可以将组件的线条更改为虚线，以便分别标识每个组件。为此，单击此面板的虚线按钮的默认选项，并选择您想要的线型。

![SketchUp Layers Output 24](img/d4e0e45a45995d98b20183e2e171fcf4.png)



和组件将具有您指定的线型，如下所示。

![SketchUp Layers Output 25](img/08fe7f374d304a85691e6db5c44c35a3.png)



让我添加一个空层，我将它命名为椅子。

![Add empty layer Output 26](img/2a93e47bc12fc884b1166bddf571930a.png)



现在，单击“详细信息”按钮，然后单击下拉列表中的“清除”选项。一旦你点击清除按钮，它将自动删除所有空层。

![Purge option Output 27](img/6c7a29eb9ddee0a08455a9310f4ae09d.png)



您也可以创建一个与我们创建标签相同的文件夹。因此，要创建文件夹，请单击该面板的“添加文件夹”按钮。

![Add Folder button Output 28](img/2491c0fcd4a497953208c15961d6afff.png)



我将把它命名为外部。

![Exterior Output 29](img/0e466c13f59ed3d32c1435f5a35bb335.png)



您可以将所需的标签/层移动到任何文件夹中，以便任何对象的所有层都在一个文件夹中，并且它们的管理变得容易。

![SketchUp Layers Output 30](img/8a6988d68fc949664ab0d0794b20ec1e.png)



我在这个文件夹中移动了两层，当我单击下拉箭头时，我们可以看到这个文件夹中的可用标签。

![Tags Available Output 31](img/876608f1dbdfa9b2609b3f722250df7f.png)



您也可以删除不需要的层。因此，选择该层，进入细节面板，然后点击删除标签选项。

![SketchUp Layers Output 32](img/13c3d4214252774d0465dc32e19b864b.png)



一旦你点击删除选项，它会问你是否要将被删除层的组件转移到任何其他层。您可以从该列表中选择要将该组件移动到的层。

![Assign another tag Output 33](img/a50e261e0a08dbc05a6b765caf7e60d1.png)



您也可以通过选择“删除实体”选项来删除该层及其组件。

![Delete the entities Output 34](img/79a41d5bd278a6f1312c4a4e70cb2efb.png)
