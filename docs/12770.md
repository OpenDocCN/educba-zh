### 实现 Scala for 循环的示例

Scala 中有很多方法可以迭代元素。在这篇文章中，我们将触及所有这些方法。所以让我们从 Scala 中基本 for 循环的语法和例子开始我们的旅程。

开始之前，让我们定义一个将在下面的示例中使用的数据结构:

`val name_seq= Seq("eduCBA", "is", "good")
val num_seq = Seq(1, 2, 3)`

#### 示例# 1–基本 for 循环

**语法:**

`for(item <- List){
// Inner loop code
}`

在语法中，我们遍历列表中的所有元素。对于每一次迭代，其中一个元素的值被顺序放置在 item var 中。然后我们可以使用这个项目变量对它进行操作。

**代码:**

`Print values of items in a list
object Demo {
def main(args: Array[String]) {
val name_seq= Seq("eduCBA", "is", "good")
val num_seq = Seq(1, 2, 3)
for (item <- name_seq) println(item)
for (item <- num_seq) {
println(item);
}
}
}`

**输出:**

![Scala for Loop Example 1](../Images/5a99ca2dcf373a6d548742d28a0432a8.png "Scala for Loop Example 1")

<noscript><img class="alignnone wp-image-326716 size-full" title="Scala for Loop Example 1" src="../Images/5a99ca2dcf373a6d548742d28a0432a8.png" alt="Scala for Loop Example 1" width="84" height="108" data-original-src="https://cdn.educba.com/academy/wp-content/uploads/2020/03/Scala-for-Loop-Example1.png"/></noscript>

**说明:**

输出由两次循环执行的打印语句组成。前 3 行是 *"* name_seq *"* 序列的输出，之后的 3 行来自“num_seq”。

#### 示例 2–带条件/过滤器的基本 for 循环

接下来我们有一个带过滤器的 for 循环。这是第一种 for 循环的变种/扩展。

**语法:**

`for(item <- List
if filter1; if filter2; ...)
{
// Inner loop code
}`

**代码:**

`object Demo {
def main(args: Array[String]) {
val item_seq = Seq(1, 2, 3, 4, 5, 6, 7, 8)
for( item <- item_seq
if item < 5; if item > 3 )
{
println("Filtered number is : " +item);
}
}
}`

**输出:**

![Scala for Loop Example 2](../Images/3e665425c2c293f00ed9f6b69b9e7c28.png "Scala for Loop Example 2")

**说明:**

从 1 到 8 的序列中筛选出的数字是 4。这是因为过滤器。即该项应小于 5 且应大于 3。唯一满足此过滤器的数字是 4。

如示例所示，我们可以向 for 循环添加多个过滤器。

#### 示例 3–forEach

这是 Scala 中用来迭代条目的主要原语之一。

**语法:**

`List.forEach()`

**代码:**

`object Demo {
def main(args: Array[String]) {
val name_seq= Seq("eduCBA", "is", "good")
val num_seq = Seq(1, 2, 3)
name_seq.foreach(println)
name_seq.foreach(e => println(e.toUpperCase))
name_seq.foreach {
// A multiline example
name => println(name.toUpperCase)
}
}
}`

**For 循环中的其他迭代方式**

*   **直到:**

可以使用 until 来指定循环的终止条件。让我们看一下这个例子

**代码:**

`object Demo {
def main(args: Array[String]) {
val name_seq= Seq("eduCBA", "is", "good")
for (index <- 0 until name_seq.length) {
println(s"Position of ${name_seq(index)} is $index")
}
}
}`

**输出:**

![Scala for Loop Example 7](../Images/85c3b185c086653d9e291308b4a0b72a.png "Scala for Loop Example 7")

<noscript><img class="alignnone wp-image-326741 size-full" title="Scala for Loop Example 7" src="../Images/85c3b185c086653d9e291308b4a0b72a.png" alt="Scala for Loop Example 7" width="233" height="53" data-original-src="https://cdn.educba.com/academy/wp-content/uploads/2020/03/Scala-for-Loop-Example8.png"/></noscript>

*   **拉链子指数:**

向序列添加计数器的另一种方法是使用 zipWithIndex。zipWithIndex 用从 0 开始的计数器值压缩列表/序列。让我们看一个例子来更好地理解它，

**代码:**

`object Demo {
def main(args: Array[String]) {
val name_seq= Seq("eduCBA", "is", "good")
for ((name, count) <- name_seq.zipWithIndex) {
println(s"Index: $count | Item: $name")
}
}
}`

**输出:**

![Scala for Loop Example 8](../Images/7155b53b8ef834274fcc077b973cdbe1.png "Scala for Loop Example 8")

<noscript><img class="alignnone wp-image-326745 size-full" title="Scala for Loop Example 8" src="../Images/7155b53b8ef834274fcc077b973cdbe1.png" alt="Scala for Loop Example 8" width="218" height="61" data-original-src="https://cdn.educba.com/academy/wp-content/uploads/2020/03/Scala-for-Loop-Example9.png"/></noscript>

从输出中可以看出，我们得到了两个值 name 和 count。Count 是使用 zipWithIndex 生成的计数器。这里需要注意的一点是，zipWithIndex 将始终有一个从 0 开始的计数器值。
