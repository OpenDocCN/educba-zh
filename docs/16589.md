## Adobe Indesign 使用简介

Adobe Indesign 用于多种类型的文档以及视觉传达设计工作，这些工作包括传单设计、杂志版面设计、小册子、名片设计、明信片、漫画、贴纸设计以及许多更专业的营销设计。它还涉及不同类型的行业标准，这些标准与设计的出版以及广告和营销专业人员使用该设计相结合。作为桌面出版和排版应用软件，我们还可以为报纸文章和其他一些东西进行一些创造性的设计。因此，让我们分析这个软件和它的使用有更多的了解。在本主题中，我们将了解 Adobe Indesign 的用途。

Indesign 与 adobe 的其他设计软件配合使用，如 adobe Illustrator、adobe Photoshop，这意味着我们可以从该软件中获取图像和插图，在 Indesign 中设计不同类型的布局，因为图像和插图通常不是在 Indesign 中创建的。我们在 Indesign 中准备的布局可以在印刷和数字媒体平台上使用。

Indesign 特别适用于设计包含大量文本的文档布局，所以让我告诉你这是一个在 Indesign 中设计报纸布局的例子，并向你解释它是如何实现这一目的的。

### 如何使用 Indesign 创建版面？

在本节中，我将告诉您 Indesign 在专业文档设计工作中的一些用法。为了解释该软件的一种用法，我将创建一个报纸版面，这样您就可以了解 Indesign 在这类工作中是如何发挥重要作用的。

首先，转到 Indesign 软件的欢迎屏幕，单击“新建”按钮，为报纸版面选择合适的文档尺寸。您可以单击欢迎屏幕上的任何“新建”按钮，这两个按钮都用于相同类型的工作。

![Adobe Indesign uses output 1](img/f1de479705740ef180a4410fb53b0d0a.png)



一旦你点击这个按钮，你将有不同类型的文件预置。我会选择这份小报预设为报纸版面设计。

![Adobe Indesign uses output 2](img/300a831ddb36410c58f630300030ddec.png)



在文档的单位中，我将从单位列表中选择英寸。

![Adobe Indesign uses output 3](img/0a4ab6bf6deb6805978761527a9beadf.png)



你可以看到它的标准尺寸是 11 x 17 英寸。它可能在你的地区不同，所以你可以根据自己的情况来选择。我只是向您解释我们如何使用 Indesign 来创建报纸版面。现在点击创建按钮。

![Adobe Indesign uses output 4](img/d4e3309e4e41363beabee3788e6d820f.png)



文件大小，我一直在考虑，它有空白区。你可以根据自己的选择设定你想要的利润。现在，我将在这个文档上划分一些部分，这些部分将成为报纸的专栏和部分，并在其中包含特定的内容。为了划分部分，我们使用水平和垂直标尺。

![Adobe Indesign uses output 5](img/067fe94affb19d5e9a591e6cf8ccb44f.png)



我将这份文件分成这样，以便对报纸的基本版面有一个概念。

现在我将采取类型工具，这是非常重要的工具，这个软件。

在第一部分拖动一个矩形框，我用引导线把它分开了。

![Adobe Indesign uses output 8](img/72ad5cfe1e017a7619f1b880f91ef3c5.png)
