## GIMP 中的线条工具介绍

线条工具指的是一种工具，通过它你可以画出不同类型的线条，但是 GIMP 并没有提供任何特定的名为线条工具的工具。代替这个特定的工具，我们有许多其他的工具，通过对它们的参数进行微小的调整，我们可以用许多方法画线。GIMP 中用于绘制线条的工具有画笔工具、铅笔工具、喷枪工具、墨水工具、路径工具和矩形选择工具。那么让我们看看如何使用这些工具来画线？

### 如何在 GIMP 中画线？

首先，创建一个新文档，然后使用我们的第一个工具。

#### 画笔作为线条工具

现在拿画笔工具或者按 P 作为快捷键。

![GIMP line tool output 1](img/c70decc32aae12c7089bcae0c86af244.png)



现在，单击文档区域中的任意位置作为线条的起点，然后按住 Shift 键向另一个方向拖动鼠标光标作为线条的终点。当你把它往另一个方向拖，按住 shift 键，它会给你显示这样一条指引线。

![GIMP line tool output 2](img/6b5c45c17f0b10f6a88bd155263ff547.png)



现在，在你想要的方向和距离上点击另一端。这样做你会得到一条线。这条线是一些软边，因为我们选择了一个软刷。可以选择 100 %硬度的笔尖。

![GIMP line tool output 3](img/9909ee7227f4a66ac190c8dc1be6f915.png)



现在你可以用和上一步一样的方法画出硬边缘线。

![GIMP line tool output 4](img/ccfe3507f122c08fcc9855fc537bcfcf.png)



#### 铅笔作为线条工具

使用铅笔工具或按 N 键作为快捷键。

![GIMP line tool output 5](img/11264ec341ecf47f1b8af66a41c594fd.png)



使用与我们使用画笔工具相同的方法绘制线条，你将得到这种类型的线条。

![GIMP line tool output 6](img/ac10c16e0849cdb7a3192ff7b7d1a5c0.png)



#### 画笔和铅笔画的线的区别

这两种工具绘制的线几乎没有区别。使用这两种工具一个接一个地画一条线。上面的线是用画笔画的，下面的线是用铅笔画的。

#### 作为线条工具的空气画笔工具

现在拿画笔工具或者按 A 作为快捷键。

![GIMP line tool output 9](img/b90ea0ce807942ab0f0e0aa809b7d5a1.png)



用这个工具绘制线条的方法也是一样的，但是用这个工具你会得到一条不透明度更小的线条。在线的末端也会有点。

![GIMP line tool output 10](img/c4e67f98bb7035a1b22c0887f10d9080.png)



线条末端的颜色强度取决于您将此工具的光标放在该点上多长时间；这意味着点击起点，并保持点击几秒钟，你会得到这种类型的黑点。

![GIMP line tool output 11](img/fe43fc628d35c2f8693d089006e7770d.png)



您可以使用此工具通过使用线条来制作这种类型的形状。

![GIMP line tool output 12](img/e62d0a018741764c93e6ae21a193ec4f.png)



#### 作为线条工具的墨迹工具

现在使用墨迹工具或按 K 键作为快捷键。

![GIMP line tool output 13](img/f7aa1e70273ad23ac1e4297cab2a560d.png)



现在画一条线，用这个工具你会得到一条两边粗细不同的线。

![GIMP line tool output 14](img/d017b9928af61fe4147caa8f789a823c.png)



您可以将此工具用作线条工具来绘制这种类型的形状。

![GIMP line tool output 15](img/fb8532df04ba1a571cdb63917783f5f1.png)



如果要更改线条起点和终点的形状，请从该选项中选择一个形状。这次我会选择菱形。

![GIMP line tool output 16](img/ac1a4ca010738757112601c71ff64ee1.png)



现在，再用它画一条线，你可以看到变化。

![GIMP line tool output 17](img/62b8a2f661c40e123de8d260fca34dc6.png)



也可以通过此选项更改线的透视平面。在白色平面上移动正方形以改变透视平面。我会像这样改变它。

![GIMP line tool output 18](img/0bb116606c0cf392886445753579e7a4.png)



现在再次画这条线，你可以看到它会画在你选择的平面上。

#### 矩形选择工具作为线条工具

使用矩形选择工具或按 R 键作为快捷键。

![GIMP line tool output 20](img/ac40453a0d523c6391af314f8df58bf9.png)



用这个工具画出这种矩形，然后右击它。点击用 FG 颜色填充来填充矩形选区的颜色。你可以根据自己的选择改变它的颜色。

![GIMP line tool output 21](img/3709779cda47652a8e18be85fd29f1df.png)



当我们选择这个填充选项时，它将填充前景。

![GIMP line tool output 22](img/429fc4174aeb2b1b75bc72ae84576dfa.png)



现在通过点击选择菜单中的“无”选项来禁用选择。

![GIMP line tool output 23](img/5cb8721b727a08dc2d3db07ff1bd83ad.png)



你的线准备好了。

![GIMP line tool output 24](img/c4c6ce7b7c658c4f40dc041393f1c904.png)



#### 作为线条工具的路径工具

使用路径工具或按 B 键作为快捷键。

![output 25](img/504b3fc59ba030e7004461efc3e437fe.png)



现在点击文档区域的任意位置，并再次点击您想要的方向。你将有一个符合这个工具的路径。

![output 26](img/d4c4b7da0c3c865f3424d9ffbadc2d66.png)



现在点击路径工具的描边路径选项，让 stork 在这个线条路径上。

![output 27](img/95f4754e69c9521352ed202910793ff5.png)



点击纯色选项，在笔画中填充纯色，然后选择线条的宽度，点击笔画按钮。

![output 28](img/95b56ddf9b2788923c2cfe4ce4f3fb83.png)



你会有你的台词。

![output 29](img/c0138bae5d017f96819a4ab725d115e8.png)



现在让我们来看看用这些工具可以画出的不同形式的线。

#### 直线

您可以使用上面讨论的任何工具来绘制直线。我将用画笔工具向您展示它，所以单击线条的起点，然后按住 shift 键绘制线条，按住 Ctrl 键绘制直线。当你按住 Ctrl 键的时候，它会把你的直线拉向你选择的方向。

![output 30](img/e3e965d57cb8ee598304a56370052989.png)



#### 倾斜线

为此，您也可以使用上面讨论的所有工具。画斜线的话，只要按住 Ctrl 键和 shift 键，把光标往斜的方向移动就可以了；然后，你将有自动倾斜的线，每次移动相差 15 度。您可以在文档区域的按钮上看到角度的变化。

![output 31](img/c7daa5e4dc7ced07982e3db13207715b.png)



![output 31.2](img/3eeaa2a9d54f762ec69c5d85e248d4cd.png)



通过将路径工具用作线条工具，可以绘制任意角度的斜线。

![output 32](img/2eac01487eb8ec9b3479e52e9ba1b3ef.png)



![output 32.2](img/9378cb7503f04565a9966bc55fa836ea.png)



#### 曲线

路径工具是绘制曲线的最佳选择。只需单击起点，然后在另一个方向单击下一个点，然后拖动鼠标而不释放鼠标按钮。对下一条曲线做同样的操作。这样，你就可以画出一条曲线的路径。

![ output 33](img/bb4a91f3f9c63616894c1905ea86ee9d.png)
