## Blender 中的纹理绘制介绍

Blender 是一款免费软件，可用于创建 3D 动画、运动图形和计算机图形工具设置和属性，可用于所有项目方面和要求。有许多归因和工作方法，今天我们将看到纹理绘画工作工具的细节，它的性能，它的性能和它的多种用途。纹理绘制工具有点类似于 ms paint 选项，并且创建的颜色将应用于已经具有默认 Lambert(模型的默认颜色是灰色)的模型。纹理绘画是 Blender 中可用的传统纹理方法。

### 如何在搅拌机中使用纹理绘画模式？

纹理绘制类似于 ms 绘制工具，其中该工具的使用略有不同，因为该工具与 Blender 动画软件集成在一起。我们可以直接使用纹理绘制来绘制模型选项，也可以使用 UV 窗口绘制选项，我们将在下面看到细节。在初始过程中，我们将[使 blender](https://www.educba.com/blender-plugins/) 软件在我们的设备上可用，然后打开它。

**第一步:**使用线、多边形、顶点和形状用 Blender 创建一个模型，在使用纹理绘制工具之前根据需要创建模型。创建一个应该使用它的模型。在这个演示中，我们将拍摄画框的肖像。我们将使用纹理绘制工具在肖像的正面绘制图片。

**第三步:**将默认为 Blender 渲染的渲染引擎改为 Cycles 渲染。为了更快地执行动态，UV 中应用的颜色将会反映在模型上，如果我们在模型上应用颜色，它会很快反映在 UV 纹理贴图上。速度也代表了系统配置，所以使用性能良好的系统。为了更好的体验，让事物看起来更棒，看起来更真实。

**步骤 4:** 启用左下角基于节点的编辑工具中的**使用节点**选项。这在模型和 UV 纹理贴图之间创建了一个基本的纹理汇。无论何时涂上颜色，它都会在两个地方的水槽里。我们可以使用基于节点的编辑器进行更多的编辑；我们将在下面的对话中看到它们。

**第五步:**除了默认的纹理 Lambert(文件)，我们可以创建自己的纹理文件，以基本的背景色作为背景。使用应用程序右侧的材质选项，如下图所示，并通过单击材质类型上的+按钮创建一个新文件。新材质创建为“Material1 ”,使用鼠标选择完全重命名，并输入所需的纹理名称。

**步骤 6:** 我们将使用 UV 智能项目选项展开模型。为此，我们将前往模型窗口。将编辑模式更改为对象。每当我们打开模型，我们需要改变模式对象的完整对象展开。然而，如果我们在编辑模式下展开，只有选定的面会展开，而剩余的面必须再次展开，这可能会导致重叠，或者每次展开都需要时间，这不是展开的最佳做法。

将模式设置为对象，然后从右边的面板中选择阴影和 UV。点击 UV 贴图下拉菜单，选择“智能 UV 项目”，这有助于平滑地展开面部，只需很少的基本设置。

步骤 7: 现在，我们将在 UV 贴图面板上工作。点击**+新建**按钮创建一个新的默认纹理。如果我们已经有一个纹理，我们可以使用它旁边的打开选项来添加它。单击+New 后，将打开一个新窗口，显示以下详细信息。为纹理创建一个名称；根据需要设置宽度和高度；如果我们想给纯色背景，可以选择颜色；我们可以选择黑色来搭配它。然后单击“确定”将其反映在 UV 贴图中。
