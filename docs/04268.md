## 如何在 Windows 上安装 MySQL？

以下文章概述了如何安装 MySQL。MySQL 是一个关系数据库系统，最棒的是，它是开源的，而且完全免费。它是由“迈克尔·维德纽斯”和他们的团队开发的。现在由甲骨文公司维护。

### 在 Windows 上安装 MySQL 的步骤

下面给出了在 windows 上安装 MySQL 的步骤:

<small>Hadoop、数据科学、统计学&其他</small>
