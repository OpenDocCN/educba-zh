### HTML 标记的示例

下面是 HTML Strike 标签的例子。

#### 示例#1

**代码:**

```
<html>
<body>
<p>Welcome <strike>To My Domain</strike> Thanks</p>
</body>
</html>
```

**输出:**

![html strike tag 1](img/df9bcc26b88df1ff21ab42b3e79160ce.png)



#### 实施例 2

**代码:**

```
<html>
<head>
<style>
.sample{
color:blue;
}
body {
text-align:center;
}
</style>
</head>
<body>
<div class="sample">
<strike>Welcome</strike>
<strike>To My Domain</strike>
</div>
</body>
</html>
```

**输出:**

![html strike tag 2](img/5d77835b7909366475cf666f966f7860.png)
