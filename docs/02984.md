## 云计算和雾计算的区别

云计算是通过互联网使用远程服务器或计算机来执行数据操作、存储和管理数据，而不是使用本地计算机或服务器的过程。云计算直接通过互联网提供递送服务。云计算提供的服务可以是任何类型的，如存储、数据库、软件、应用程序、网络、服务器等。“雾计算”是思科创造的术语，意思是云计算之外的服务向企业需求的延伸。它由一个分布式计算环境组成，其中的基础设施提供存储、应用程序、数据和计算。雾计算也被称为雾联网或雾化。

### 云计算和雾计算的主要区别

以下是云计算和雾计算之间最重要的区别:

1.云计算架构有不同的组件，如存储、数据库、服务器、网络等。而雾计算具有与云计算相似的所有功能，包括一些额外的功能，如高效、强大的存储以及系统和云网络之间的性能。

2.[云计算架构](https://www.educba.com/cloud-computing-architecture/)系统可以分为前端和后端两个部分，两者以网络的形式连接，而雾计算通过提供网络边缘的功能来扩展云计算。

3.云计算的前端部分被称为用户界面，其中终端用户或客户使用云计算的服务，后端是云计算网络的云部分，而雾计算的目标是提高效率并减少来自和去往分布在不同位置的远程网络的数据或数据操作的转换。

4.客户端可以通过云计算的前端部分访问不同类型的服务，其中用户可以像本地计算机一样正常访问服务，但将通过连接到网络来访问，而雾计算由一个名为 Open Fog Consortium 的大型开放集团联盟提供支持，该联盟由思科、戴尔、微软、英特尔、ARM 和普林斯顿大学等公司于 2015 年 11 月成立。

5.在云计算中，后端部分包括服务器、不同的计算机、存储和数据库系统，它们相互连接以形成分布在不同位置的云网络，而雾计算通过从部署在远距离或远离中央服务器的不同位置的不同设备收集数据来处理中央服务器中的数据。

6.在云计算中，对存储空间的要求更多地是为了让客户端访问他们存储的数据，几乎存储空间将是已存储数据的两倍，以提供高速访问，而雾计算数据操作和计算发生在设备的中央集线器中，以减少来自和去往中央服务器的数据转换。

7.云计算中有一个中央服务器来管理彼此连接的不同计算机或服务器，它们的交互和机制将受到控制和管理，而雾计算通过提供更多的合规性和更轻松的迁移来支持物联网中的大多数设备。

8.中间件与中央服务器一起存在，以在多个服务器之间建立通信协议，并以安全可靠的方式相互通信，而[雾计算通过处理大量数据和各种设备来支持](https://www.educba.com/fog-computing/)大量物联网应用和大数据服务。

9.存储在中央数据库服务器存储中的所有数据将作为备份可用，以使其在少数服务器故障的情况下高度可用，在这种情况下，该过程被称为冗余，而雾计算通过有效地支持网络上的大量用户而在地理区域中具有更大的分布。

10.云计算的主要核心组件是互联网/网络，没有互联网/网络，整个网络就会崩溃，无法连接到云服务器，而雾计算有不同的应用，从物联网到人机交互，应用范围广泛。

11.大量终端用户可以使用称为虚拟机的虚拟设备接口从远程机器连接到云服务器，其中的概念称为虚拟化，而每当在铁路、船舶、车辆和道路等极端边缘收集大量数据时，就可以考虑雾计算。

12.云计算是利用不同的可用服务，如存储、软件开发应用程序、服务器和数据库。云计算为操作服务器或应用程序提供了更多的可访问性，没有任何限制。

13.雾计算主要利用本地计算机资源，而不是访问远程计算机资源，从而减少了延迟问题和性能，进一步使其更加强大和高效。

14.云计算服务是基于服务器应用程序提供的，它允许用户从任何位置通过不同类型的设备(如电脑、手机和平板电脑等)访问服务。,

15.雾计算有许多好处，例如它提供了更大的业务灵活性、对安全控制的更深刻的见解、更好的隐私和更少的操作。它有一个额外的边缘层，支持并类似于云计算和物联网应用程序。雾计算主要通过在与彼此互连的设备一起工作时提供即时响应来提供网络中的低延迟。

### 云计算和雾计算对照表

以下是要点列表，描述了云计算和雾计算之间的比较。

| **BASIS FOR****比较** | **云计算** | **雾计算** |
| **等待时间** | 云计算具有低延迟，但无法与雾计算相比 | 雾计算在网络方面具有低延迟 |
| **容量** | 在发送或转换数据时，云计算不会减少任何数据 | 雾计算减少了发送到云计算的数据量。 |
| **带宽** | 与雾计算相比，云计算节省的资源更少 | 雾计算节省了带宽量。 |
| **响应度** | 在雾计算中，系统的响应时间较低。 | 在雾计算中，系统的响应时间很长。 |
| **安全** | 高，但比雾计算低 | 高度安全。 |
| **速度** | 根据虚拟机的连接情况，访问速度很快 | 与云计算相比更高 |
| **数据整合** | 可以集成多个数据源。 | 可以集成多个数据源和设备。 |

### 结论

与云计算相比，可以获得的主要好处来自雾计算。与云计算相比，雾计算延迟低，响应速度快，是最值得推荐的计算方式。与云计算相比，它支持物联网。

在大用户和广泛分布的网络方面，雾计算是首选和推荐的，以获得更高的效率和高生产率。
