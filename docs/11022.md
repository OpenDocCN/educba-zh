## 网络安全基础介绍

网络安全基础词可以分为两个不同的词，即网络意味着“与计算机直接或间接相关的东西”，安全意味着“防止各种东西被滥用”。计算机安全或 IT 安全等术语是网络安全基础知识的别名。

通俗地说，网络安全基础知识就是采取措施保护重要信息，防止硬件、软件和其中包含的信息被窃取或损坏。重要信息可能被黑客(即实施黑客行为的人)窃取(即在没有任何特权的情况下获得访问权)。这个黑客可能是你认识的，也可能是匿名的。有两种不同类型的黑客，即白帽黑客和黑帽黑客。

<small>网页开发、编程语言、软件测试&其他</small>

白帽黑客通常为您所知，让您知道系统中漏洞和弱点。他们也被称为道德黑客。一家公司通常雇佣这些人来寻找他们系统中的安全漏洞。

相比之下，在网络安全基础知识中，黑帽黑客是指那些为了从重要信息中获利(或为了娱乐或报复)而实施黑客行为的人。这些重要信息可以是个人机密信息，如银行信息、电子邮件访问等。或公司机密细节，如访问安全服务器等。

实施的黑客攻击可分为不同类型，例如:

*   社会工程
*   秘密的
*   拒绝服务(DoS)攻击
*   偷听
*   电子欺骗
*   网络钓鱼

社会工程是一种技巧，黑客只需与受害者进行良好的沟通就能获得重要信息。例如，在与朋友交流时，您泄露了电子邮件凭据，而一个陌生人正好在您身后听到。

在现实场景中，后门意味着允许重要资源的秘密进出点。后门是任何绕过计算机系统正常认证或网络安全基本控制的秘密方法。这些后门可能由于多种原因而存在，例如系统的不良设计或设计者对自己保密的有意添加，或者可能是授权方后来添加的，以允许对系统的合法访问。

< lang=”EN-IN”>拒绝服务(DoS) < lang=”EN-IN”>攻击是指授权用户无法访问机器或网络资源的攻击。例如，黑客可能会多次输入错误的密码，从而导致帐户被锁定，或者机器/网络可能会过载到所有授权用户都无法访问资源的程度。在这种情况下，黑客从一台机器(即单个 IP 地址)进行攻击。这可以通过使用防火墙阻止特定的 IP 地址来解决。但是，可能会出现黑客从多台机器上实施攻击的情况。这被称为分布式拒绝服务(DDoS)攻击。

### 个人安全提示

以下是一些个人安全提示和技巧，可以帮助你免受黑客攻击。

#### 1.无线家庭网络安全的重要性

*   Wi-Fi 在安装的初始阶段总是有默认密码。更改 Wi-Fi 的默认密码。密码应该是强密码，包含字母(大写和小写)、数字、特殊字符和至少八个字符。
*   始终为 Wi-Fi 打开兼容的 WPA2(无线保护接入)/ WEP(有线等效保密)加密。最好使用一些[加密](https://www.educba.com/encryption-process/ "The Process of Encryption")而不是什么都不用。
*   更改默认网络名称。这是因为默认网络更容易被黑客攻击。
*   启用 MAC 地址过滤。这种机制允许 Wi-Fi 只在注册的 MAC 地址上工作。
*   不要自动连接到开放的 Wi-Fi 网络，因为这样的网络更容易被黑客攻击。
*   <lang>长时间不使用时，关闭网络<lang>。</lang></lang>

#### 2.社交媒体网络安全的重要性

*   单击来自未知发件人的邮件中的链接时要小心。这是因为链接可能会重定向到钓鱼网站。
*   了解您发布的关于自己的内容。有些人会在脸书、推特等社交网络上公布个人联系电话或地址等机密信息。这可能很危险。

![Social Media Security](../Images/4ae5f63d6e121feec9f68e789b12efe1.png "Social Media Security")

<noscript><img class="alignnone wp-image-29600 size-full" title="Social Media Security" src="../Images/4ae5f63d6e121feec9f68e789b12efe1.png" alt="Social Media Security" width="679" height="353" srcset="https://cdn.educba.com/academy/wp-content/uploads/2015/10/8.png 679w, https://cdn.educba.com/academy/wp-content/uploads/2015/10/8-300x155.png 300w, https://cdn.educba.com/academy/wp-content/uploads/2015/10/8-420x218.png 420w" sizes="(max-width: 679px) 100vw, 679px" data-original-src="https://cdn.educba.com/academy/wp-content/uploads/2015/10/8.png"/></noscript>

*   不要相信一条信息真的来自它所说的那个人。它可能是声称是原始发件人的欺骗内容。
*   不允许脸书、Twitter、LinkedIn 等社交网络服务。，扫描您的电子邮件地址簿。这也可能为阅读电子邮件内容打开一扇门。
*   直接在浏览器中键入您的社交网站地址，而不是点击链接，因为它可能是一个等待获取您的机密信息的钓鱼网站。
*   选择你在社交网络上接受的朋友。如果你在社交网络上加了一个不知名的人，可能就是黑客攻击的切入点。

![Cyber Security Basics](../Images/f2330ea9af7706b138f729a60b1ae0a9.png "Social Media Security")

<noscript><img class="alignnone wp-image-29601" title="Social Media Security" src="../Images/f2330ea9af7706b138f729a60b1ae0a9.png" alt="Cyber Security Basics" width="700" height="340" srcset="https://cdn.educba.com/academy/wp-content/uploads/2015/10/9.png 1227w, https://cdn.educba.com/academy/wp-content/uploads/2015/10/9-300x145.png 300w, https://cdn.educba.com/academy/wp-content/uploads/2015/10/9-1024x497.png 1024w, https://cdn.educba.com/academy/wp-content/uploads/2015/10/9-420x204.png 420w, https://cdn.educba.com/academy/wp-content/uploads/2015/10/9-738x358.png 738w" sizes="(max-width: 700px) 100vw, 700px" data-original-src="https://cdn.educba.com/academy/wp-content/uploads/2015/10/9.png"/></noscript>

*   仔细选择你的社交网络。了解隐私政策。
*   在您的网站上安装附加应用程序时要小心，因为该附加程序可能是能够访问您系统的[特洛伊木马](https://www.educba.com/viruses-function/ "Viruses and Trojans: We Live in Your Wires")。

#### 3.网上银行网络安全的重要性

*   定期监控您的帐户。
*   避免点击浏览电子邮件。这样的电子邮件可能是一个网络钓鱼陷阱，会给你带来很大的麻烦。
*   定期更改密码，并确保始终使用强密码。

![Password](../Images/f17c4921ab28388dbde8ffb6606f69dc.png "Password")

<noscript><img class="alignnone wp-image-29602 size-full" title="Password" src="../Images/f17c4921ab28388dbde8ffb6606f69dc.png" alt="Password" width="467" height="76" srcset="https://cdn.educba.com/academy/wp-content/uploads/2015/10/101.png 467w, https://cdn.educba.com/academy/wp-content/uploads/2015/10/101-300x48.png 300w, https://cdn.educba.com/academy/wp-content/uploads/2015/10/101-420x68.png 420w" sizes="(max-width: 467px) 100vw, 467px" data-original-src="https://cdn.educba.com/academy/wp-content/uploads/2015/10/101.png"/></noscript>

*   从安全的位置访问您的帐户，例如使用 Firefox 而不是 internet explorer。
*   如果你收到一封电子邮件或短信或任何电话承诺提供你的个人信息的奖励，不要被诱惑，因为这可能是黑客的社会工程方法。
*   确保你只使用官方银行认可的应用程序。

#### 4.手机网络安全的重要性

*   始终使用模式、PIN 或密码来锁定您的设备。如果设备被盗并且没有密码保护，那么黑客可以滥用该设备。

![Mobile Phone Security](../Images/d6d650ff61eb108e93dd1997730eab07.png "Mobile Phone Security")

*   自动锁定您的手机，以保持它的安全。
*   数据保护软件应用于设备数据保护。
*   让你的应用和设备软件保持最新，并始终从正版网站/来源进行更新。

#### 5.孩子们关心网络安全基础知识

*   告诉你的孩子保护他们的个人信息，留意“免费”的东西，使用强有力的电子邮件密码。
*   建议孩子使用安全软件扫描任何下载的程序。
*   监控孩子的 P2P(点对点)活动或文件共享习惯。
*   通过演示向他们介绍网络钓鱼诈骗。
*   向孩子介绍他们安装的应用程序。
