## CorelDraw 的用途介绍

Coreldraw 是一款基于矢量的设计软件,用于创建徽标、flexes、宣传册、邀请函以及任何基于衬里的矢量设计。这是一个非常老的软件，发布于 29 年前的 1989 年 1 月 16 日<sup>。Corel Corporation 开发了它，因此它也被称为 Corel 的图形套件，因为它可以处理基于矢量的图像，所以它被用于编辑二维图像，如徽标和海报。</sup>

CorelDraw 的最新版本于 2018 年 4 月 10 日<sup>发布，增加了一些新功能，如将信封应用于位图，交互式调整照片视角。CorelDraw 是为微软 Windows XP、Vista、Windows 7、Windows 8 和 Windows 10 开发的。他们还推出了 Mac OS 和 Mac OS x 的版本，但由于销售不佳，他们停止了进一步的版本。Corel 公司还开发了 Windows 3.1X、CTOS 和 OS2 的版本，直到第 5 版。</sup>

### Corel Draw 的主要用途

它用于设计不同的艺术品，如海报，标志，卡片和任何矢量相关的图形。我们在下面讨论它的用途:

#### 2.为常规需求创建一个基本形状

这表明，无论我们在日常生活中需要什么样的基本形状，如大表情符号、用于制作贴纸的卡通草图、动物贴纸、汽车贴纸等。在 CorelDraw 中绘制基本形状要容易得多；我们只需拖放一些形状，如圆形、矩形、椭圆形、直线、胚珠等。所以我们可以说，使用 CorelDraw，很容易画出基本的形状。我们可以根据需要导入图像。

#### 3.我们可以用它来做一个小的邀请卡。

如果我们知道如何制作基本的形状，我们可以在家里使用 Coreldraw 制作基本的小型请柬。这对我们非常有用，因为我们不必去任何商店买它。

![How to Use Corel Draw - invitation card](img/6069a9483ddc7ee512eb0852f2425b77.png)



#### 4.将位图转换为对象

从位图到对象的转换有点复杂，但 Corel 提供了一种更简单的方法，因为 Corel 有一个很好的工具集，通过它转换变得更容易。

我们只需右击位图，选择我们想要做的事情:

*   quick race 是一个将位图转换为矢量对象的命令。
*   中心线追踪是一个试图提取位图中心线并移除其余部分的命令。
*   轮廓跟踪是一个尝试提取位图轮廓的命令。

![How to Use Corel Draw - Bitmap to objects](img/1e35265b4e96154c2739e285fa76bc15.png)



#### 5.在其他程序中使用 CorelDraw

CorelDraw 的设计也适用于其他软件。Coral 公司设计的另一个软件是“Coral Photo Paint”，它可以与 Coral Draw 结合使用。因为设计的图形可以用珊瑚颜料进行编辑和再造。当我们保存图形时，它可以保存为与其他软件兼容的其他格式。它可以保存为 PNG，CDR，Jpeg 等格式。这些保存的图像也可以在 Adobe Illustrator 中打开。所有这些保存的格式与其他设计者建立了联系，以更容易的方式对设备的设计进行修改。

### 关于如何使用 Corel Draw 的提示和技巧:

以下是不同的提示和技巧:

*   创建并保存文档:当你第一次打开文档时，你最好按时保存它。要保存文档，应该先点击文件>另存为文件名。一旦你保存它，然后继续按 Ctrl + s，以保存每隔一段时间。
*   **玩混合:**[混合工具](https://www.educba.com/blend-tool-in-illustrator/)用于通过形状和颜色的渐变将一个对象转换成另一个对象。这个工具是最古老的工具之一，也非常有用。
*   **使用文档调色板:**这显示了如何在文档中使用调色板。在软件的 CorelDraw 和 Corel Photo-Paint 中，都为对方创建了一个定制的调色板。
*   **使用高斯模糊镜头:**这是本软件使用的特效。它允许您在不影响图像质量的情况下模糊图像。
*   **网络图形工具像素视图的使用:**该工具允许您编辑图形，因为您可以用实际像素单位绘制，从而提供更加精确的演示。因此，该设计以增强的方式出现在网页上。
*   **使用多边形工具:**正如我们之前讨论过的，我们有大量的形状可以合并，使我们的设计更容易。在这里，我们有三个工具:多边形工具，选择工具，和形状工具。
