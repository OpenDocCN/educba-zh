## Photoshop 中的颜色替换工具介绍

在本文中，我们将学习 Photoshop 中的颜色替换工具。每个图形专业人员都非常熟悉 Photoshop，因为这个软件使他们的工作变得非常容易，并将他们工作的准确性提高到了一个完美的水平。Photoshop 是 Adobe 家族的一部分，由他们维护。他们不时地向用户提供该软件的更新版本，以避免其绘图包。借助这个软件非常有效的工具，我们可以在这个软件中做很多编辑工作。颜色替换是它最好的工具之一。使用颜色替换工具，我们可以改变图像中任何选定对象的颜色。

### Photoshop 中的颜色替换工具怎么用？

在本文中，我们将非常有趣地看到这个工具的所有方面。让我们开始了解这个工具:

<small>3D 动画、建模、仿真、游戏开发&其他</small>

**第一步:**去互联网浏览器下载一张高分辨率的图片效果更好，也可以自己拍摄高分辨率的图片。或者转到保存的图像文件夹，将此图像拖到 Photoshop 中，然后放入软件中。

![image folder](img/2531f33a006d8e7b9b80f94b4a5d28c6.png)



第二步:进入工具面板部分，点击笔刷工具图标。从这里选择颜色替换工具，或者你可以按键盘上的 **Shift+B** 键三次来激活这个工具。

![tool panel section](img/37b52385436c4875cea799b9d8756268.png)



**第三步:**拿到这个工具后，只要点击这个工具带加号的区域就可以了。一旦你点击任何特定的区域，它将捕捉该区域的颜色，并用你的颜色面板的前景色改变该区域的颜色。例如，我在颜色面板的前景色中有棕色，然后当我单击如图所示带有此工具加号的苹果时，它会将苹果的绿色更改为棕色。

![color panel](img/7c6d4bfb950f07cc9d246912777b8a2e.png)



第四步:我们在工作区的顶部有一个这个工具的属性栏。我们将逐一查看该工具的所有属性，并使用它们。

![property tool bar](img/1234346872885218ed4fa9efa5b1d99d.png)



**第五步:**对于第一个属性，点击笔刷大小图标。将会打开一个下拉框。在这个框中，我们有画笔大小的值、画笔硬度的值、画笔间距的值、画笔笔尖的角度和圆度的值以及其他一些属性。

![brush size icon](img/240f5af3a178d5125e322ed4dd029968.png)



**第六步:**你也可以在工作区的任意位置点击右键打开这个属性框。

![property box](img/8701d73e287bb84f8297da6627a183be.png)



**第七步:**要改变笔尖的大小，只需移动大小值的滑块或在该属性的值框中用键盘手动输入任意数字即可。我们可以根据图像的选择区域来改变画笔笔尖的大小；也就是说，对于精细的工作，我们减小画笔的大小，而对于粗糙的工作，我们增大画笔的大小。

![brush tip](img/9c5eadf88cbee26fc6828f5b0a5a417a.png)



**第八步:**对于下一个属性，我将通过在这个软件中放置图像的相同方法来拍摄另一个图像。

![Color Replacement Tool Photoshop-8](img/8d98490bb256183e4ee4544d239304d7.png)



**第九步:**接下来是硬度。将硬度值增加到 100%以查看其效果。

![hardness 100%](img/7d97fbcb5470c5cdc9781eacb794c7c7.png)



**第十步:**现在，我将在图像上画一条简单的线。

![simple line](img/05bd411959a2e67f73e14153d625f398.png)



**步骤 11:** 再次将硬度值降低至 50%。

![Color Replacement Tool Photoshop-11](img/b6ec6323638f7f2a99b92dcb1f0e48c5.png)



**步骤 12:** 再次，在前一条线附近画一条简单的线。你可以看到硬度值变化的不同。当您增加该值时，颜色会产生锐利的边缘，当您减少硬度值时，颜色会变得平滑。

![Color Replacement Tool Photoshop-12](img/bb4c094b4ef1d57c28f342977c72a242.png)



**第十三步:**接下来是间距。

![Spacing](img/1b7e40cdc608cc9bddc3d70ea06b90d5.png)



**步骤 14:** 当你增加间距的值时，结果会是这样的

![Color Replacement Tool Photoshop-14](img/8b5e6b35aab6676700590c8fffa64482.png)



**第 15 步:**间距必须达到 10%，这样着色的速度就不会因为你的个人电脑内存少而受到影响。

**第十六步:**接下来是笔尖的角度和圆度。你可以根据你的使用目的来改变它。

![Angle and Roundness](img/763ee16ece2eb3b6078a1df12d68eeac.png)



**第十七步:**接下来是模式。在模式中，我们通常会在颜色选项上工作，因为当我们选择任何颜色来用该颜色选项替换颜色时，该工具会用所选颜色替换该颜色。

**第 18 步:**色相、饱和度、光度模式会对选择的颜色给出一些不同的结果。当你熟悉这项工作时，你就会理解这种变化。

![Color Replacement Tool Photoshop-16](img/f5a015c0864d0dce19cbce3bc0478682.png)



![Color Replacement Tool Photoshop-17](img/db0e1c9c617ab78b49411d78ec4ebbc1.png)



**第十九步:**接下来是采样。我们将在该属性中选择“该工具如何从图像中提取样本”。第一种方法是连续取样。

![Color Replacement Tool Photoshop-18](img/408a4abac7fc9a98a7958893ef0f7056.png)



**步骤 20:** 在该方法中，工具将随着该工具加号位置的改变而改变样本颜色。例如，当我们单击半个橙色区域时，工具将改变这个橙色的颜色，当我们向前移动工具，到达叶子区域时，它将选取叶子颜色作为样本，并用相同的橙色改变叶子的颜色，如下图所示。

![Color Replacement Tool Photoshop-19](img/80305c9bffffc72fcac3c7b0411ee53c.png)



**步骤 21:** 接下来是“一次采样法”

![Color Replacement Tool Photoshop-20](img/f2871f4c193bd49b483c89d1432ff945.png)



**步骤 22:** 在其中，工具将只改变第一次点击所选取的颜色作为样本颜色。

![Color Replacement Tool Photoshop-21](img/bf01538865534625bf1addc85f49968d.png)



**步骤 23:** 另一个是背景样本采样。

![Color Replacement Tool Photoshop-22](img/8cac8f41e7125548c32cf30d180c713c.png)



**步骤 24:** 这个工具会把那个颜色作为样本，它在颜色面板部分的背景色里。

![Color Replacement Tool Photoshop-23](img/a4953fafe464f648b37c213b2f76f42f.png)



**第 25 步:**在极限选项中，为了得到更好的结果，我们一般使用 Contiguous。连续使选择变得容易。

![Color Replacement Tool Photoshop-24](img/b90cfb116ebda76d8e9543aadd4e388e.png)



![Color Replacement Tool Photoshop-25](img/122557606a06c1516d74c6adb413c8db.png)



**第 26 步:**接下来是公差。当您增加容差值时，该工具会将所有颜色视为与您选择的颜色相匹配的样本颜色。

![Color Replacement Tool Photoshop-26](img/3657bca5a62678c6e23a562d045334b7.png)



第 27 步:为了获得更好的结果，公差必须在 30%到 40%之间。

![Color Replacement Tool Photoshop-27](img/1d1c913e86dccac363cd987eda7a0eff.png)



**步骤 28:** 接下来是反走样。

![Color Replacement Tool Photoshop-28](img/f1b0a8881d997f21fe7eb37d1f13ac1c.png)



步骤 29: 这里有两种形状的笔刷在这张图片中给出。你很容易发现这两者的区别。当您启用“消除锯齿”选项时，颜色会随着平滑的边缘而改变，而当您禁用“消除锯齿”选项时，颜色会随着边缘的扭曲而改变，如图所示。左边的没有抗锯齿，右边的有抗锯齿。
