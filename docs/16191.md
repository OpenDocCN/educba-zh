## CorelDRAW Powerclip 简介

在本文中，我们将学习如何在 CorelDraw 中使用 powerclip。使用 powerclip，我们可以将任何对象放置在任何其他矢量对象中，这些对象通常是矩形、椭圆形甚至是文本。唯一的条件是用于 powerclip 的对象应该是一个向量。我们还将看到如何从 powerclip 中删除对象，并在剪辑完成后将创建的帧转换回矢量对象。除此之外，我们将看看一些优化技术，这将提高工作流程的速度。

**步骤 1:** 让我们从文件菜单或按 Ctrl + N 开始创建一个新文档。另一种方法是从欢迎屏幕创建它，在欢迎屏幕中有一个选择新文件、最近文件或模板文件的选项。
