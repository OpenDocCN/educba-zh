## DOS 和 Windows 的区别

*   DOS 或磁盘操作系统是计算机世界最原始的操作系统。这是一个计算机没有发展到图形化的时代。曾经有一个黑色/蓝色背景的普通屏幕，上面写着白色的文本命令。这就是磁盘操作系统。
*   Windows 操作系统是最先进、最现代的版本。它拥有所有精美的图形，使视觉效果更具吸引力。文本命令已经被彩色的交互式图标所取代，这些图标可以响应鼠标点击、触摸和语音命令。

### DOS 和 Windows 的主要区别

让我们讨论一下 DOS 和 Windows 之间的一些主要区别。

*   DOS 和 Windows 的主要区别在于界面。如上所述，DOS 是一个简单的界面，在这里你可以对任何事情发出文本命令。这类似于现代 Windows 操作系统中的命令提示符终端。
*   Windows 使用图形来提供更具视觉吸引力的界面。鼠标的出现使它变得易于使用。不再需要记住命令。你所要做的就是点击鼠标。图形也使得在电脑上播放电影、图像和游戏成为可能。
*   当然，还有许多关于存储系统、安全性、网络等方面的增强。Windows 是计算技术的高级版本，集成电路和紧凑型内存芯片的进步为其提供了支持。

### 有趣的事实

*   通读完整篇文章，有没有让你疑惑“为什么叫[磁盘操作系统](https://www.educba.com/what-is-dos/)”？之所以这样命名，是因为从外部磁盘(软驱、硬盘等)传输命令被认为是一项巨大的成就。)到计算机存储器(RAM)用于执行。这是对它的前身磁带操作系统(TOS)的一个根本性的进步。于是，DOS 这个名字就产生了。今天，任何现代操作系统都隐含着磁盘操作系统，但没有明确地这样命名。

### DOS 与 Windows 比较表

让我们讨论一下 DOS 和 Windows 之间的 11 大区别:

| **比较的基础** | **DOS** | **窗户** |
| **界面** | **命令行界面**–在 DOS 下，每一个动作都是通过命令来执行的。例如， *cd* 我的文件夹会放在一个名为“我的文件夹”的文件夹中。 | **图形用户界面**–Windows 遵循 WIMP(窗口、图标、菜单、指针)交互风格。鼠标指针的点击/滚动事件执行预期的动作。 |
| **初级输入** | 键盘–DOS 使用命令来做任何事情。这些命令是通过键盘输入的。因此，键盘成为 DOS 的主要输入方式。 | **键盘和鼠标–**Windows 在屏幕上引入了一个指针和一个控制该指针的操纵杆/鼠标。现在，指针点击/滚动事件执行了大多数与文本无关的任务。 |
| **文件系统** | **FAT16**–DOS 使用 16 位文件分配表(FAT16)系统。这意味着它使用 16 位标识符来唯一标识每个文件的内存位置。这些标识符然后被存储在一个表中，因此命名为文件分配表。这张桌子非常重要。如果此表损坏，将无法检索文件。 | **FAT32/NTFS**–Windows 在早期版本中使用 fat 32，后来用新技术文件系统(NTFS)替代。在 NTFS 中，有关每个群集的信息与数据一起存储在群集中，从而消除了文件分配表。它还使用二叉树目录系统，以便检索和存储速度更快。 |
| **注册和交换系统** | **目录系统**–DOS 为其注册表信息和交换框架使用简单的目录框架。整个信息都包含在特定的索引或子目录中，这使得它更容易受到攻击。 | **复杂**–Windows 使用复杂的基于目录的系统来存储注册表信息。这是一个错综复杂的信息存储系统，很难抹去这些重要的信息。 |
| **多任务处理** | 不，DOS 是一个单任务操作系统。你必须等待一个命令完成后才能发出另一个命令。 | **是的**–Windows 是多任务的。您可以在播放音乐和复制文件的同时键入文档。 |
| **多用户** | **否**–DOS 一次只能支持一个用户。这意味着一台计算机一次只能由一个用户使用。 | **是**–使用终端服务，Windows 可以同时支持多个用户。这方面最好的例子是 Windows 服务器。 |
| **安全** | **受限**–DOS 的安全性非常低，因为它不符合用户角色的概念。任何可以登录系统的用户都是管理员用户，因此拥有完全的访问权限。 | **广泛的**–Windows 通过授权的概念克服了 DOS 的安全缺陷。要获得对计算机的完全访问权限，用户必须具有提升的权限(管理员权限)。只有管理员才能执行内核级任务，如注册表修改、程序执行、软件安装等。 |
| **联网** | **外部功能**–DOS 下的联网是一项附加功能。计算机很少连接到网络。这使得在计算机之间共享文件变得困难。 | **内置特性**——Windows 推出了电脑内置网络连接。这有助于连接共享磁盘存储和共享设备(打印机、扫描仪等)。)甚至共享内存。 |
| **存储大小** | **小于 2GB**–由于 16 位分段格式，DOS 的磁盘存储容量有限。16 位分段不能识别超过 65536 个簇。因此，对于 32 kB 的集群，您可以存储最多 *32*1024*65536 = 2GB* 的数据。 | **兆兆字节**–由于内存芯片和分段技术的进步，现代 Windows 操作系统已经克服了这一限制。 |
| **操作系统的大小** | **小(以千字节计)**–由于其高度简单的界面和有限的功能，DOS 是非常轻量级的。一台典型的 DOS 机器需要大约 512 kB 到 1MB 的空间来存储操作系统、应用程序和数据。 | **大容量(千兆字节)**–Windows 提供更多图形丰富的功能，如图像、视频、游戏等。这使得 Windows 10 操作系统的最低要求是 4GB 内存和 500GB 磁盘空间。 |
| **易用性** | **低**–在 DOS 下，你必须记住各种命令来执行操作。 | **高**–Windows 肯定不是这种情况。无需记住 Windows 中的任何命令。 |

### 结论

DOS 不再用于个人计算。Windows 占据了大部分市场份额。尽管如此，即使是突破性的 WIMP 概念现在也受到了更多进步的挑战，如基于触摸的操作系统、基于语音的界面、手势命令等。随着技术的进步，人机交互也在进步。
