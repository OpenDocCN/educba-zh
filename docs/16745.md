## Revit 键盘快捷键简介

以下文章概述了 Revit 键盘快捷键。Revit 的字面意思是修订和编辑，所以为了使工作流程更快，并创建更多的修订，我们应该精通所有的快捷键，至少是默认的快捷键。这种快捷方式不仅节省了你的时间，还省去了浏览多个菜单的麻烦。现在想象一下，如果你一遍又一遍地使用同一个命令，那么使用快捷方式是正确的，因为它肯定会节省你的时间。

### Revit 键盘快捷键的步骤

下面是提到的步骤:

<small>3D 动画、建模、仿真、游戏开发&其他</small>

**步骤 1:** 要学习 Revit 中的快捷键，首先从快捷键开始，查看 Revit 中可用快捷键的列表。KS 是查看可用键盘快捷键的快捷方式，您可以在其中根据菜单进行搜索和过滤。可以通过分配新的键来改变快捷键，但是在本文中，我们将坚持使用默认的键。

![Revit Keyboard Shortcuts 1](img/ce93baeadf641b13f45d940e979a76e8.png)



**第二步:**先从墙面开始。WA 是墙的快捷方式，我们创建一个矩形墙，如图所示。

![Revit Keyboard Shortcuts 2](img/53ffaebee59cd666ac3c4f92fa17f3aa.png)



**第三步:**然后让我们创建一些线墙，使结构如图所示。

![Revit Keyboard Shortcuts 3](img/3d18539bae51802f72024b5d1cb087a7.png)



**步骤 4:** 接下来，我们添加一些门。快捷方式是 DR。如果您得到没有门加载的错误，那么您可能需要下载库并从那里加载。

![add some doors](img/2f9bbf078494023c4184217693e3b8c9.png)



**步骤 5:** 您可以使用鼠标滚轮进行缩放，但如果您想要平移，则可以使用滚轮，该滚轮可以使用 Shift + W 进行切换。

![use the mouse wheel to zoom](img/a1307e7cc1ea1f58e9ce4f82b76dca49.png)



第六步:你可以使用快捷方式 WN 来放置一些窗口。同样，如果它没有加载到项目中，那么您可能希望从库中加载。

![place some windows using shortcut WN](img/760981942df917ee915f82dec3aa151e.png)



**步骤 7:** 同样，我们可以快速添加如图所示的所有门和墙，因为 Revit 意味着修改和编辑，您可以轻松地在快捷方式之间切换以完成任务。

![Revit Keyboard Shortcuts 7](img/53418b51587ad488c3b1df5e3a51e88f.png)



**步骤 8:** 然后使用快捷键 GR 放置一个网格，如图所示。

![use shortcut GR to place a grid](img/0866a098108cab3d1b9a4b586af97137.png)



**步骤 9:** 现在我们使用快捷键 CL 放置一些结构柱，但是由于它们没有被加载，我们将从库中加载。

![Revit Keyboard Shortcuts 9](img/ffcdcd05f98b3d46fc77d04e5a020f97.png)



**步骤 10:** 导航到库中的列。

![Navigate to the column in library](img/bbc02b48ab0e2d548a626e24f7f86b55.png)



**步骤 11:** 使用 CL 放置柱时，将选项设置为高度。

![Revit Keyboard Shortcuts 11](img/785bb05c3db3a25f07aeb571a2b64d99.png)



**步骤 12:** 接下来，我们使用 DI 作为快捷方式添加尺寸，并如图所示放置。

![we add dimensions using DI](img/9bbeb7df014711408eb0daacbe4cf803.png)



第 13 步:然后，如果你点击中间一列的 EQ，那么这两列将被等间距排列。

![Revit Keyboard Shortcuts 13](img/2fb28f8bbe282b94eb713133685ccdd4.png)



**第 14 步:**我们可以在整个墙壁上添加一个尺寸，但这个值太大了，因为它是以毫米为单位的，我们可以改变尺寸。

![add a dimension across the entire wall](img/eb4e8f3332a71ba22b85fc8181cf77cf.png)



**步骤 15:** 要更改项目单位，按 UN 将长度单位更改为米。

![To change Project Units press UN](img/2a22b1ce5b5481e422b97fc900d5a63e.png)



**步骤 16:** 如果你想从墙的边缘而不是中心开始标注尺寸，按 tab 键。

![Revit Keyboard Shortcuts 16](img/16b098992ef616879460da8b62ca6d4e.png)



**步骤 17:** 然后创建一个结构楼板，捷径是 SB，我们创建一个矩形，如图所示，直到柱子。

![Revit Keyboard Shortcuts 17](img/a9500dd359cb088a0cfd92b0d1eff458.png)



**步骤 18:** 我们可以使用 al 将楼板对齐到结构柱的顶部，然后单击柱，然后单击楼板进行对齐，如果需要，您可以将其锁定。

![Revit Keyboard Shortcuts 18](img/ac0ecac19cd82561e969c0ca3db3b07a.png)



**步骤 19:** 我们可以创建一个不同形状的板，使用分割工具或 SL 分割它，然后点击分离零件。

![Revit Keyboard Shortcuts 19](img/89d1d3b8fd7a481ad9956876350b9646.png)



**步骤 20:** 然后使用 TR-Trim/Extend 工具，我们可以移除或修剪任何我们不需要的东西。

![Revit Keyboard Shortcuts 20](img/186f5946751563657053f6b8a7dc5d56.png)



**第 21 步:**然后你可以通过按压进行偏移，可以将选中的元素垂直于其长度移动或复制到指定的距离。

![you can do offset by pressing OF](img/44497d455683fb9691524fca05d14735.png)



第 22 步:但目前我们不需要它们，所以我们删除了它们，并做了修剪/延伸或 TR 快捷方式来连接线条，如图所示。

![Revit Keyboard Shortcuts 22](img/9cd8129aaae5504ef4cedd5b1412fd41.png)



**步骤 23:** 然后我们添加一条跨度方向线，如果没有加载符号。

![Revit Keyboard Shortcuts 23](img/808a3052b0f6a6b1f99aac754f1fc446.png)



**步骤 24:** 在库中导航到跨度方向。

![Revit Keyboard Shortcuts 24](img/9c600083ae04c89ac3dacc00abf5d416.png)



**步骤 25:** 这样我们就添加了跨度方向。

![Revit Keyboard Shortcuts 25](img/6b76894973ea81dd9854c0617b7e5b6f.png)



**步骤 26:** 现在，我们可以使用顶部的图标或项目浏览器来查看默认的 3D 视图。

![Revit Keyboard Shortcuts 26](img/68337307ea69b1704f88c4ca1956a5aa.png)



**步骤 27:** 假设我们想暂时隐藏墙，那么我们使用快捷键 HH 来隐藏墙，然后 HR 来重置隐藏/隔离。

![Revit Keyboard Shortcuts 27](img/771566a20dfb15906d8f5b6442b4dae7.png)



第 28 步:如果你想改变一些视觉风格，那么你可以进入图形显示选项或者按 GD，在这里你可以给一些阴影和一些背景。

![Revit Keyboard Shortcuts 28](img/20d93c9b084356830190a6e64e3f0636.png)



**步骤 29:** 更改图形显示选项后，输出如图所示。

![Revit Keyboard Shortcuts 29](img/18a367cbe77ef49c1e0387917f64185c.png)



**步骤 30:** 现在让我们再次进入楼层视图，以厘米为单位放置一些组件。要载入族，请定位到“家具”部分并添加一张桌子。

![Revit Keyboard Shortcuts 30](img/758c4ab89b4463f0ef9191952f55e9e9.png)



**步骤 31:** 您可以使用快捷方式 MV to move 来捕捉或对齐对象。

![Revit Keyboard Shortcuts 31](img/1db9399b0e8da950480adfeffaac000b.png)



**步骤 32:** 也可以使用快捷键 CO，然后复制到想要的地方。

![Revit Keyboard Shortcuts 32](img/72683f42e049de8a2997affa93ed5ea7.png)



**步骤 33:** 也可以用快捷键 RO 旋转到指定角度。在选项中，您可以在旋转后制作副本或更改旋转中心。

![evit Keyboard Shortcuts 33](img/8d2a542d423801e81b6e5a85e4faac63.png)



**步骤 34:** 现在我们将使用快捷方式 DL 创建一条详细的线。

![Revit Keyboard Shortcuts 34](img/7bd6c905d8bd870800ae9aaf1f894e83.png)



**步骤 35:** 然后你可以选择所有的线，按 RE 快捷键缩放。但是大多数元素不能以类似的方式缩放。

![Revit Keyboard Shortcuts 35](img/770664c2df59775f5729a398e894fbe6.png)



**第 36 步:**然后你就可以使用 AR 通过选择桌子放置多个同类物体，然后将 number 设置为 5，移动到最后。

![can use AR to place multiple objects](img/29f41288bedbe07add2e986ffded6df7.png)



所以通过移动物体和改变数字，我们可以做出一些很酷的安排。当您选择其中一个对象时，它们都被链接，因为它们在一个组中。

![we can make some cool arrangements](img/f050f57a38a7c3b880e94cc9b0c6235a.png)



**步骤 38:** 如果你想创建一个组，那么让我们进入 3d 视图，选择墙壁、门和窗户，使用快捷键 GP 创建一个组。

![Revit Keyboard Shortcuts 38](img/f76b46aeb2f995882c07e301dc982019.png)
