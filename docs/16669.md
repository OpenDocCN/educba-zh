## 2D 和 3D 形状的区别

画在二维上的物体，即 X 和 Y，叫做 2D 图形。在绘制形状时，当第三维或方向进入时，Z 方向和向 Z 方向投影的形状称为 3D 形状。在这个主题中，我们将学习 2D 和三维形状。这些形状绘制在同一个平面上，与方向无关，显示出不同的外观。这两种形状的使用是特殊的，并证明在同一领域是独一无二的。大多数想学习 3D 形状的人必须通过 2D 形状，然后他们必须更进一步。2D 是 3D 的基础，因为学习者必须积累所有的 2D 技巧，所以很难画出 3D 形状。

形状不是问题，但是它们的平面决定了它们的维度特征。如果我们检查任何 2D 结构和 3D 结构之间的差异，那么我们可以注意到 2D 和 3D 形状之间的差异。假设我们想知道正方形和立方体的区别。正方形有 4 条边和 1 个面，如果考虑立方体，那么它有 6 个面和 12 条边。这将是解释差异的正确例子。通常，2D 形状是从我们小的时候开始的，那时我们习惯在画册上画小草图，当我们变老的时候，我们坚持 3D。在图形设计中，我们看到，在学习时，我们最初必须积累所有的 2D 形状，如画圆、六边形等。我们利用这一点开始画一朵花或任何其他小动物。

### 2D 和三维形状的主要区别

以下是描述 2D 与 3D 形状之间主要差异的要点列表:

*   2D 图形和三维图形之间的主要区别是一个轴与其他两个轴成 90 度角。
*   不显示对象或结构的深度或宽度的形状是 2D 形状；相反，在 3D 中，我们可以看到深度或高度。
*   在工程制图中，2D 异形图非工程人员很难读懂；相反，许多人可以阅读 3D 形状的图纸。
*   与 2D 形状相比，3D 形状中的表面和边的数量更多。
*   当我们想通过 2D 图形解释任何产品时，我们必须画出视图的数量，如前视图、俯视图、侧视图和仰视图。有时我们必须根据产品绘制更多视图，但在 3D 视图中，我们可以在一个视图中清楚地看到客户的客观视图。
*   尺寸，在 2D 形状或图纸中解释零件的每一个边缘的详细尺寸要容易得多。在三维中，只有外部尺寸被清除；很难展示所有细节。
*   基本上，当我们绘制一个截面图或任何部分的详细视图，小功能和隐藏的功能是突出使用 2D 形状。在 3D 形状中，这种细节是不可能的。
*   在 2D 图形绘制中，任何画在图纸上的物体的阴影是不可能显示出来的。由于显示反映在另一个角度或第三轴，即 z 轴。要显示阴影，我们需要的是绘制在三维总是。
*   2D 形状显示出原始形状的基本效果，可用正方形、五边形、六边形等。，这有助于在绘制任何 2D 形状时绘制软件中的任何复杂形状。相反，在 3D 中，我们可以利用这些正方形和五边形，但不能全部累加在一起。

### 2D 和三维形状的比较表

下面是 2D 和 3D 图形之间的比较:

| **Sr 号** | **2D 形状** | **3D 形状** |
| One | 它有两个维度，即 X 和 y。 | 它有 3 个维度，即 X、Y 和 z。 |
| Two | 正方形、圆形和三角形是用 2D 图形绘制的。 | 立方体、球体和棱柱体是使用 3D 形状绘制的。 |
| Three | 2D 形状用于在工程绘图中显示俯视图、侧视图、仰视图和正视图 | 3D 形状用于绘制等轴测和正交形状。 |
| Four | 雕刻，兰格利斯是绘制在 2D 形状，使设计看起来更加美丽，因为它是在颜色的帮助下绘制的。 | 兰格利斯和版画很难绘制成 3D 形状，因为它可能不会像 2 D 形状那样完美。 |
| Five | 当我们测量二维图形的长度和宽度时。 | 在 3 D 形状中，它有长度、宽度和高度。 |
| Six | 当我们按照 2D 的形状画出一个结构的平面图时，它会给出一个总览并解释所有的细节。 | 绘制等轴测视图(3D)时，它会给出结构的实际视图。 |
| Seven | 这些 2D 形状广泛用于绘制简单物体的简单图像或草图。 | 3D 形状用于定义对象的建筑视图。 |
| Eight | 尺寸很容易解释。 | 只能解释外部尺寸。 |
| Nine | 在 2D 形状中，所有使用的边缘都清晰可见。 | 在 3D 形状中，一些边缘是隐藏的，有时我们必须隐藏一些边缘来获得 3D 形状。 |
| Ten | 2D 形状很容易画出来，因为人们在观察它们的时候不需要想太多。他找到了一个更好的方法来画它。 | 3D 结构中的一些复杂形状看起来很难，用户发现很难绘制。最初，他必须考虑从哪里开始；该结构可能会变成奇怪的形状。 |

### 结论

在考虑 2D 和 3D 形状之间的差异时，我们必须了解它们的特征和用途。从上面可以得出结论，2D 是绘画或形状的鼻祖，一个学习者必须首先学习这个，然后他必须更进一步。2D 绘图由一些规则的形状组成，这有助于我们提高我们的动手绘图，也提高了我们的准确性。最后，可以得出结论，这两种形状都有独特的品质，因为我们必须选择特定的图纸。因此，学习 2D 和 3D 的所有复杂功能使学生成为绘画的完美候选人。
