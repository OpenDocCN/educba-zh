## 矢量图形编辑器简介

照片在计算机图形学中有两种查看方式:矢量图和光栅图(或位图)。前者将所有基于数学计算的图像描述为一组像素。第二种使用道路、角度、弧线、曲线、形状和多边形来达到同样的目的。光栅图像是基于像素的，因此会失去透明度，并且可以在基于矢量的图像上永久缩放而不会退化。换句话说，独立于分辨率的矢量图形能够保持最高程度的一致性。

### 矢量图形编辑器

让我们来看看矢量图形编辑器。

#### 1.Inkscape

Inkscape 是一个矢量图形编辑器开源软件，类似于 Adobe Illustrator 和 CorelDRAW，但它使用一个开放的 XML W3C 标准，可缩放矢量图形(SVG)，因为原始格式将它与其他格式分开。Inkscape 支持各种复杂的 SVG 功能(克隆、标记、alpha 混合等)。)而且一个流畅的 GUI 设计的非常用心。在 Inkscape 中，编辑节点、全轨迹操作、监控位图等等都非常简单。Inkscape 的命令行中有一个多功能的 GUI，可用于许多功能，包括导出和格式转换。

#### 2.Xara Xtreme

该软件基于 Xara Xtreme 的 Windows 图形程序(最快的可用图形程序)。Xara Xtreme 是一个通用的多功能图形程序，包括 Linux、Unix 系统、FreeBSD 和(开发中的)Mac Os X。新的更新是 0.7。Xara Xtreme 对于 Linux 来说非常简洁易用。它包括一些最强大的可用图形工具。Xara Xtreme 的学习曲线有限，可以选择大量优秀的图形。获得大量学习内容、视频、电影、技巧，以及 Xara Xtreme 背后高度参与、充满激情且不断增长的用户群。

#### 3\. Skencil

在 GNU 宽松通用公共许可证中，Skencil 原名 Sketch，是一个自由矢量图形编辑器。对于图形、图表和其他用途来说，它是一个通用而高效的工具，可以在 GNU / Linux 和其他 UNIX 兼容系统上运行。Skencil 很少用 Python 实现，Python 是一种非常高级的解释语言。Python 是强大的、面向对象的，并且易于使用。剩下的软件用 c 语言陈述，app 0 . 6 . 17 版本现已推出。它有 SPARC，i386，M68，DEC Alpha，PowerPC，以及与 Linux 兼容的 FreeBSD，Solaris，IRIX64 和 AIX 版本。AIX 也支持它。

#### 4.静脉反射反应

VRR 始于捷克共和国布拉格的卡洛斯大学，当时是数学和物理系的一个学生实验。2005 年 9 月成功维持了这一判决。但是，在作者的业余时间里，还在创作。VRR 项目旨在构建一个专门为数学论文设计的矢量图像编辑器。它的主要特性包括各种几何图形及其稳定性、与 TeX 的协调、方案脚本、实际尺寸以及对各种文件格式(包括 EPS、PS、PDF 和 SVG)的支持。VRR 0.9.4 是最后一次更新。用户手册也可以在这里免费获得。

#### 5.ZeusDraw

ZeusDraw 是最新的矢量绘图软件，具有快速、优雅的界面、宽画笔和许多其他功能。需要 Mac OS X 10.4 或以上版本。ZeusDraw 是一个通用的二进制文件，可以在 Intel 或 PowerPC 上执行。目前是 1.3.1 版。可可和石英让 ZeusDraw 教你怎么做。当您移动或旋转对象时，您看到的是对象移动或旋转，而不是其边界框。ZeusDraw 有一组矢量画笔(阴影、平滑、多毛发),它还有一个实体画笔，可以用矢量或位图比特来绘制。ZeusDraw 使贝塞尔曲线更平滑。

#### 6.MagicTracer

MagicTracer 是一个基于 Windows 的光栅到矢量转换器。包括光栅和矢量编辑软件。图像扫描仪的辅助需要 MagicTracer。如果您是一名艺术家或建筑规划师，需要将扫描的蓝图或图纸导入到您首选的 CAD 软件中，或者是一名标牌制作人，需要一台乙烯基切割机将图片文件转换为表格，MagicTracer 就是您的理想选择。MagicTracer 提供了三种处理光栅和矢量信息的关键工具:图像、光栅和矢量。

#### 7.节点盒

NodeBox 是一个应用程序，可以使用 Python 编程代码(静态、动画或交互式)生成二维图形，并将其导出为 PDF 或 Quicktime 电影。NodeBox 是有据可查且易于访问的。NodeBox 可以创建矩形、椭圆形、星形、箭头和一般的贝塞尔曲线方向。它支持图片(包括 PDF)和文本(换行)。NodeBox 是一台“状态计算机”，但是没有编程经验的人很容易理解它。

#### 8.Tplot(赢)

Turbo Plot 是一个免费的 GUI 应用程序，可以从二维或三维绘图中生成特征和数据，无论是 Windows XP 还是更高版本。TPLOT 可以像任何绘图程序一样保存图像和矢量。TPLOT 还可以保存 TPLXML 文件，其中包含由文本命令生成的文本文件，以便于手动加载或修改。在动画矢量文件格式中，TPLOT 可以保存为 SWF (Adobe Flash)格式。你可以制作出高质量的插图，未经雕琢的图形。
